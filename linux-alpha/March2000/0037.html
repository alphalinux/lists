Trying to run 2.3 on a DEV XL(T)300
(Alcor, Bret, BIOS 5.70, MILO 2.0.35).

It looks like this machine has problems with the new alpha
irq code.

Being pretty new to alpha stuff and unable to decode the
CIA PCI error messages (pointers appreciated) I just
searched for which kernel introduced the problems.

2.3.49-pre2, 2.3.50, 2.3.51 crashed with
CIA PCI machine check: err0=... (more below)

I made a minimal working generic kernel 2.3.47.

2.3.48 seems to have incompatibilities regarding
irq_stat for UP and init_irq_proc for sysctl,
seems like a old/new mix.
