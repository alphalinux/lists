[...]

> SCSI 2 tape drive:
> File number=-1, block number=-1, partition=0.
> Tape block size 512 bytes. Density code 0x24 (DDS-2).
> Soft error count since last status=0
> General status bits on (5010000):
> WR_PROT ONLINE IM_REP_EN

Use automatic (variable) blocksize, as in:

Marvin:~/ftp$ mt setblk 0
Marvin:~/ftp$ mt status
SCSI 2 tape drive:
File number=0, block number=0, partition=0.
Tape block size 0 bytes. Density code 0x24 (DDS-2).
               ^^^^
Soft error count since last status=0
General status bits on (41010000):
 BOT ONLINE IM_REP_EN
