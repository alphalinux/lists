Has anyone but me experienced Linux 2.2.15 stopping suddenly in the middle of
the boot process (for instance after starting at)?
Full halt - no error messages anywhere.
The halt point seems fixed, but seems to be influenced by some factors
like how much RAM the machine has (I pulled out half the RAM, and it stopped
at a slightly different point in the boot process)

My 2.2.14 kernel sometimes stops in the same way, but rarely (once every 2
weeks or so).
Architecture is Noritake - Alphaserver 800.
