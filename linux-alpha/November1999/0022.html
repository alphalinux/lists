Hi All,

For EV6 based systems, booting from SRM is all this is recommended
by Compaq. The publicly available installation guide (which is updated
very regularly) only talks about SRM installs and booting.

You can find this doc here, but be sure to check back before any
new install to be sure you have the latest. It contains install info
for various distributions:

http://www.digital.com/alphaserver/linux/install_guide.html

And for hardware docs, check here:

http://ftp.digital.com/pub/Digital/info/semiconductor/literature/dsc-library
.html

Enjoy,

Tim

Compaq Computer (realize I am an engineer who enjoys linux, but
am not part of the official group who oversees any of these systems
or docs.)

PS: Be sure to get the RH 6.0 Alpha updates or various pieces will
not work on your EV6 system (like some network stuff, etc).
