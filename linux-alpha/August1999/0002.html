Hi

There were a couple of comments about linux on a 1000A 4/266 in
the archive but I couldn't find any mention of anybody actually
having linux running and if so which version.

We have a 1000A 4/266 with a mylex raid controller and qlogic
scsi. Looking at the red had hardware listings this looked like a
possible goer but not supported. I tried installing Red Hat 6 but it
comes up a with a double halt error 6 when it tries to load the
kernel from the installation floppy.
I tried going back to Redhad 5 as they had separate images for the
SRM console models (6 just has the generic image). The kernel
loaded ok with this version but things froze when it tried probing the
scsi devices, I am pretty sure that version five has no inbuilt
support for qlogic cards.
Any body had any success with this sort of hardware
configuration? Is it worth persuing, or I am I flogging a dead horse?
