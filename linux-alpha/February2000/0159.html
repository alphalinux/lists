On Fri, Feb 18, 2000 at 06:41:43PM -0500, Matthew R. Pavlovich wrote:
> I seem to be missing something here.. what is the difference b/w
> PC164 and Cabriolet?

Cabriolet is the code name for the PC64. It's IRQ subsystem
was reused for the EB66+, EB164, PC164, and LX164.

The PC64 is a 21064A with a Saturn SuperIO chip for peecee isa widgetry.
Uses 21071 (APECS) for PCI I/O.

The EB66+ is a 21066 evaluation board. It appears to be otherwise
identical to the PC64.

The EB164 is a 21164 evaluation board, also using the Saturn.
Uses 21171 (CIA) for PCI I/O.

The PC164 is similar to an EB164, except with a 21164A
and a 21172 (CIA-2), and a CMD646 IDE controler.

The LX164 is similar to the PC164 except with a 21174 (PYXIS).
