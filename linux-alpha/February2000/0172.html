> original 7631627, wh64 9230262 diff -1598635, other 7579267 diff 52360

You may have a broken assembler. At one point wh64 was mis-encoded,
and the effect was that it didn't do anything. The correct encoding is

  { "wh64", MFC(0x18,0xF800), BASE, { ZA, PRB } }, /* ev56 una */

in binutils/opcodes/alpha-opc.c.
