Hi, all;

I haven't seen the SX MVI patch in Alan Cox's 2.2.16pre summaries
(haven't looked in code). Should it be?

Ivan's patch (with codycyl) is working great for me. For autoconf
stuff, with patch applied, config.guess reports the correct machine
specification:

with patch applied:

./config.guess
alphapca56-unknown-linux-gnu

without:

./config.guess
alphaev56-unknown-linux-gnu

And just a side note, my SX runs SRM v5.5; Ivan's original note to the
linux-alpha list had said/implied this was a problem for v5.6/5.7, but
it would seem to be relavent to older SRM versions as well...
