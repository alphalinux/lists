What's Cooking at MandrakeSoft?
----------------------------------------
Mandrake Linux 8.1 Beta 2 "Raklet" is available for download & testing.

Changes and improvements since Beta1:
* DiskDrake (the graphical disk partitioner) now supports Samba and NFS
  mount points
* New Mandrake Server Wizards: professional configuration wizards for
  quick & easy server setup
* Prelude 0.4.2 (Intrusion detection system): see
  http://prelude.sourceforge.net/ for more info
* Gnomemeeting 0.10 (Video conferencing software H323 compatible)
* Libsafe (Stack Overflow protection), for increased security
* Bug fixes for KDE 2.2, i810, and the Mandrake Software Manager
  (rpmdrake)

Mandrake Linux 8.1 New and Killer Features:
* Draknet (network configuration tool); test and enjoy this thoroughly
  reworked version
* Updated URPMI (package installer) and Software Manager
* The reworked Mandrake Control Center provides a convenient embedded
  root console as well as new tools such as Logdrake (a graphical
  frontend to the system log files) and Drakinst (for easy setup of an
  auto-install disk)
* Updated HardDrake (hardware configurator)
* New version of PrinterDrake to get the most out of your printer

See http://www.linux-mandrake.com/en/81beta2.php3 for the complete
announcement.

To get involved as a beta tester, please download the files from
http://www.linux-mandrake.com/en/ftp.php3#81; then install & test on
your machine and report any bugs to:
https://qa.mandrakesoft.com/cgi-bin/index.cgi

Also, please subscribe to the Cooker mailing list by sending an email
to sympa@linux-mandrake.com with the words "SUB cooker" (no quotes) in
the subject of your message to report and discuss any problems.

Business Case of the Week
----------------------------------------
Air Force public uses Internet computers with Mandrake SNF.
A system administrator at the Rhein Main U.S. Air Force Base in
Frankfurt Germany offers Internet access to the public with
MandrakeSoft's Single Network Firewall:
"The install of the OS was a snap. I could not believe that getting
Squid with Squidguard working would be such a no brainer... Mandrake
has played a helping hand in improving quality of life for American
Airman in Germany at Rhein Main AB."
http://mandrakebizcases.com/article.php?sid=145&mode=thread&order=0

Read many more examples of Mandrake Linux in the workplace at:
http://www.MandrakeBizCases.com/

If you use MandrakeSoft products in your business, please share your
story by submitting your own personal "Bizcase".
