Hi:

    This is my first experience with SuSE Linux and I must say that in
general it is excellent. Top marks go to "robustness".. Especially the
installer, it never crashed for me and I gave it a serious workout on the
Jensen. The install on my Alphastation 4/255 233 mHz was simple and
the machine is running almost flawlessly using kernel 2.2.19.

    Another area that you have done well in is supplying a usefull variety of
applications and utilities in a very clean, well integrated way.

    I attempted to install it on one of my three DEC Axp 150 machines in
"expert" mode, YaST. That went very well once I was acustomed to the way YaST
works. Unfortunately the Jensen kernel that you supply does not have compiled
in AHA-1740 disk controller support. RedHat and Debian distributions are
making the same mistake. If it was not for this one item the Jensen install
would be as flawless as the other Alpha family machines.

    Please fix this and I will definately buy your Alpha Edition. Remember
too that the Jensen was the first Alpha machine to run Linux.

    I would also check to see that linux/arch/alpha/kernel/traps.c has been
patched. I saw "awk" running 98% of the CPU time, which is a sure sign that
you need to apply the patch.

    The only bugs that I see on the Alphastation are some "unaligned access
error" coming from X (Matrox driver) and a couple of the "KDE Control Center"
utilities crashed with SEGV faults.

    Overall I think you have an excellent distribution and I look forward to
using it. Please fix Jensen support and I'll be a very happy customer.

Cheers,

</Duane>
