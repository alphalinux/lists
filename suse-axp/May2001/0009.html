Hi!

I want to report some bugs I found in the SuSE Linux AXP 7.1.

Fist I want do describe my hardware:
- mainboard: DEC AlphaPC 164
- RAM: 8x16MB = 128 MB
- CPU: 21164 500 MHz
- Graphic: Elsa Winner 2000 AVI PCI
- Controller: Adaptec 2940AU PCI
- HD: 2x Quantum 2GB (I set the 2940 to 8MB for these devices because of
scsi timeout problems.)
- cd-rom: Pioneer
- Network: Via Rhine II Network Card PCI
- Sound: (ALS4000 (Avance logic PCI))
        Soundblaster 16 pnp ISA

First bug: This bug was reallly annoying: The milo for my machine isn´t
working. What happened ist that the milo just stops loading with the
message: "switching to PAL at 0x0" (I hope I remember correctly). The
previous milo switched after this message from the blue ARC screen to
the black milo screen with the bios messages of the graphics card. So I
had to use the old milo. But with the old milo it wasn't possibile to
boot from CD. It seems to be impossible to load the kernel from a fat
partition or a too big ext2 partition! So I had to make a disk with the
right kernel (wich by the way ist the right one?). And a second disk
with the initrd for installation. I HOPE YOU'LL FIND THE BUG, BECAUSE
THE 2.4 KERNEL CAN USUALLY NOT BOOTED WITH THE OLD MILO!!!

Second bug: If you set the file permission to secure in yast1 some
programs won't start (for example Netscape, mc)

Third bug: graphical installation doesn't work: I get a screen wich
looks like cutted into peices, so that you can't recognize anything!

Fourth bug: some programs under KDE2 (for example komqueror) crashes
with SEGV after a time. Also some Problems with Netscape: I can't sent
any e-mails because Netscape says: There's no user name for the smtp
server.... But I set a username! After some time it seems that I cannot
start some new programs. It looks like they can't connect to the
x-server, but there aren't any error messages.

Fifth bug: You fogot the alsa drivers. So I can't use the ALS4000
soundcard and I have to use the Soundblaster wich ist supported by the
kernel.

But the other expierience I made was quite good. The system is running
stable and all programs are working fine.

Have fun
