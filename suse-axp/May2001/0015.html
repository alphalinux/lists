I have some problems booting an AS600 5/333 for installing SuSE 7.1
(the beta version).

When using the new milo, the machine always halts at the point
"Swapping to PALcode at 0x0". This seems to be the case because
the graphics is always recognized as TGA (the old milo I am using
now says:
Built against linux version 2.0.30 ...
Video is Digital BIOS Emulation (VGA), Zlxp (TGA)...

I tried upgrading the firmware using as600_v72_1.exe, which I found

"Opcode Reserved To DEC occured..." or "Fatal machine check, CIA

Fortunately, there is an old redhat install with its milo already
on the disk, so at least I have a working milo (I thought).

"cannot open root device 00:00"
