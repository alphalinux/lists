On Fri, May 21, 1999 at 10:17:11AM +0200, Baptist Trappist wrote:
> Hello can somebody help me
>
> I'm seeking an mail-client that can read mail from multi - mail accounts and
> that's can filter mail. I prefer an KDE or X-window program

If you are used to use text-mode Un*x programs and to write configuration
files, I'd really recommend mutt as MUA and procmail to do the filtering.

mutt has support for more than one inbox, threading, pgp and many more. On
the other hand, you need to print the docu and spend some hours to learn how
to use it's nice features.

procmail can be automatically called by sendmail on incoming messages.
Basically you give regular expressions like ^(To|CC): linux-kernel.* and
write those messages into the appropriate mailbox file.

If this does not sound too hard for you ...
