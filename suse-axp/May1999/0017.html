Baptist Trappist wrote:
>
> Is there a way to open Ms Access 97 documents in linux.
>
> I have a very large database in access so I don't want to
> retype it in a new linux database. An I heared that wine
> doesn't work wiht Office 97.

If you need the Alpha for running Linux by all means ...
maybe the best solution for you would be hook up a cheap Intel
machine to the network, run native NT there, and then access
the MS database via ODBC / JDBC.

If you need to work with Access under Linux on the same machine,
for some reason, forget the Alpha, forget WINE, and give VM-Ware
(www.vmware.com) a try on a Intel box instead.

I played around with VM-Ware a bit and found it to be almost
rock solid. I managed to run the X-Com game in a DOS window
on Win95 running on VM-Ware on RedHat 6.0.
