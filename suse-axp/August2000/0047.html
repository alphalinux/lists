I was not able to get the TGA configured with SaX. I had
to use xf86setup to get a working config file, and then
had to set a broken symlink from X to the installed TGA
server. It's a pain, but it was the only way I could
get it working. It works fine once it's installed though.

There does seem to be a graphics bug in SaX though. On
non-TGA cards, the graphics on the display configuration
screen (where the screen is shown in the canvas) are
garbled.
