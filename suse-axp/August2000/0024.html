Hi,

to be able to put milo-2.2 on the next axp-release, we need some more
testing on that.
So if you've got a alpha that needs to be booted by Milo, please download

ftp://ftp.suse.com/pub/people/sf/axp/7.0/milo.tar.gz

unpack it and put the corresponding files on a FAT-formatted floppy.
then boot from that floppy, and mail me the results.

Even better would be to to insert CD1 and use following OSLoadOptions:

boot scd0:/etc/vm_full.gz root=/dev/fd0 initrd=/etc/initdisk.gz
