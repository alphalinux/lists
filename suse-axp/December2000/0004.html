Observe the diffrence between root and boot! I use a 5 MB boot partition
using FAT for MILO, and placing the kernel on it. This should be the best
solution, especially since MILO requires a filesystem to reside on.

I'm not sure wheter aboot requires a boot partition, but from a safety
POV a read only boot partition should ALWAYS be used.
