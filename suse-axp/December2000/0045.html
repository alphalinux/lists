Do you have the same graphics card? Did 6.3 work for you, too?

Is there anybody with a running X on SuSE 7.0 and a ruffian system?

I tested the 6.3 milo without any success and upgraded to the 2.2.18
kernel, but nothing changed. So the initializiation steps seem
not to be the problematic ones.

I even assume, that it has nothing to do with the X server directly,
since i am not able to successfully use the SuSE 6.3 X versions
on my 7.0 machine, though it once worked perfectly on 6.3.

The only things, which can effect X when started by XF86_Mach64 should
be XF86_Mach64, /etc/XF86Config and libc.

I changed everything but the last one - but i shudder to replace libc
(at least i have a working non-X system, now).
