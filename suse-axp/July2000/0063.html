There two differnet things here. First is the X server. Voodoo3 cards are
support by XFree (both 3.3.6 and 4.0/4.0.1) so you should be OK from that
perspective (I've been running one in an SX for months now w/o any
trouble).

GLIDE is another story. Porting GLIDE to Alpha is a difficult task. There
are people working on it but it's going to take some time, however this
doesn't stop you from using the card.

--rdp
