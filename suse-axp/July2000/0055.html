Hello Justin;

configuring a Mylex DAC960 on Alpha is real problem.
The configure-tools from Digtal works only with Firmware 2.x
Linux supports only Firmware 3.x
If you have firmware 2.x, you have to update to 3.x with two new
FlashProms which
are available on Mylex hardware-partners.
Myles has only tools for Intel.

I do the following to configure my DAC960 in a AS1000 4/266;

- downloaded the config-tool for DOS and copy it to a DOS-Boot-Floppy
- opend the side-panel of the server
- build the DAC960 in a DOS-PC which was standing on top of the Server
- the DAC 960 was conected with the drives in the AS1000
- swiched on the AS1000 and stoped booting (only for powering the
HD-drives)
- booted the PC from the floppy
- configured the RAID-Systems with the DOS-Tools
(It worked only if Ib configured the RAID5 first and took about 5 ours
for 17 GB)
- switched of the computers and build the DAC960 bak into the AS1000
- now i was able to partition the RAID's and install Linux

attention:
if you want to boot from the DAC960 you need a special-kernel and you
have to boot over SRM/aboot.

Please mail me for further questions.
