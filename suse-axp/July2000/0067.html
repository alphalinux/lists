Hi Werner!

I have not tried that, but from what I learned from the list (redhat I
believe) recently it does not work. The reason for that is, that generic
kernels do not work on jensen. So the kerneltype that should work on all
alphas is not so generic afterall. It seems the compaq folks simply forgot
about that old beast.
        But the SuSE 6.4 distribution does work (to some extend) and boot
on the jensen. have a go with this one. On the other hand the jensen is
quite obsolete and slow by all means. So one might ask, why go for one of
the fastest architectures if the chosen modell is beat to pieces by any
recent computer. (Given that it is quite expensive to equip the jensen
with sufficient memory to run e.g. X and an application)
