Though I do not run an Alpha, I have experience the same thing with large
HD's with 6.3. fdisk is squirrelly and cannot recognize the proper drive
geometry. Why, I don't know.. I can only suppose some of the changes to the
install program affected fdisk also. (darn those new fangled graphical
installers)

I suggest using the fdisk on 6.2 as an option and restart your install with
6.3 and choose "do not partition" when linuxrc queries you, and move on to
indicate the mount points as needed.

Cheers,

John @ DataEFX
