Dear Alpha users,

I just released another version of the Milo 2.2 source tree.

Changes are:
 * initrd is working
 * several Kernel 2.4 fixes, i.e.
   - correct L1 page allocation
   - kernel start address is taken
     from elf headers
   - Milo size limits changed to handle
     both old and new kernel start addresses should work.
 * support for multiple program headers
 * build fixes for recent kernels and binutils
 * removed machine vectors from relocate
   and stub code (reduced size)
 * regular milo crashes since 2.2-14 fixed
 * ...

The prebuilt binaries are built against Linux Kernel 2.2.16.

You can get the source and binaries from

        http://www.freiburg.linux.de/~stepan/Milo/

Success stories and bug reports are welcome.
