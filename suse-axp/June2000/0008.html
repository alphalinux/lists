Hi, Jim

The best solution is to look through mailing list archives at SuSE.com.

You find that there is an issue with 6.3 and X. KDE (anything that uses
QT) won't work and GTK will work but the widget text is all wacky and you
can't read it. This is because some parts of the distro were compiled for
a only for the next generation chip after the one (21064) in the
Alphastation x00's.

You may also find that your scsi adapter is not supported by the kernels
available for selection during the installation process. Install
everything and then use the boot disks (make floppy's for the least amount
of hassle) to start a system. You then copy the vm_full.gz kernel from
/etc/ (? ) on the first CD rom to your boot partition and go.

This has been fixed in 6.4 (an evaluation iso image) is available at
http://www.alphalinux.org and at the SuSE ftp site.

All the detail can be found in the archive.

Enjoy the hack.

Justin
