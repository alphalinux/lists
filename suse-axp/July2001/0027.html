> When I press the '-' key I get an inverted "B" character and several

> setup and I can't see to get this in KDE. I'm using a stock 7.1 install.
> I tried several permutations (101, 101/deadkeys, 101/ISO, 104) and

sounds like for some reason german keyboard mapping was activated. Could
you please send me any system information or starting point to debug your
trouble? The /etc/X11/XF86Config would be very useful. Are for example Y and

any problems in mapping. You can even start sax2 on a running X server and
change the keyboard settings on the fly and test them using the "apply" button.
