builtin ethernet controller. During the install I'm asked which eth0
module I want to load. I tried DE4*5 and Tulip.

causes a segmentation fault and stops the boot process!

Again, I don't recall what I did to get this working in 6.4 but I thought
that
DE4xx module worked.

  - No support for very common ATI RagePro (Mach64). This worked in 6.4.

  - All this and I've only had it for 2 days.
