Hi,

i have compiled samba-2.2.1a without errors (some warnings) on my
Alpha XP1000 SuSE Linux 7.0
Compaq C V6.2-505 on Linux 2.4.4 alpha
Compiler Driver V6.2-505 (Linux) cc Driver
Installed as ccc-6.2.9.505-1
SuSE Linux 7.1 (Alpha)
VERSION = 7.1
Using /usr/lib/gcc-lib/alpha-suse-linux/2.96 (4).

When configuring do something like this:

bash: export CC="ccc -ms"
t/csh: setenv CC "ccc -ms"

"-ms" select the Microsoft language mode.
See man ccc
