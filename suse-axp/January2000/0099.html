Thanks, Cor! That did it! I don't know why I didn't think of that before.
I guess I thought that the disk needed to be the old ramdisk image format
and not a ext2 fs formatted floppy. Anyhow, once I changed the root= parm,
I was able to read MILO and the kernel from the CD. The kernel then booted
and prompted for the ramdisk. I had the disk with the install image in the
drive and the kernel read it and then YAST started up. Being wary of the
CD change prompt issue that you seemed to have encountered, I installed a
minimal install + some minor packages (egcs, basic X, perl and few others)
all from CD 1 and it's now up and running.

Thanks again!
Alan
