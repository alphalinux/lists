To Igor and Duane:

I found a very good book on alpha assembling:

ALPHA RISC ARCHITECTURE for programmers
by James S.Evnas and Richard H. Eckhouse
Prentice Hall PTR
ISBN 0-13-081438-5

2nd book on algorithms written in assembler:
The Art of Computer Programming Volume I to III:
by Donald E. Knuth
Addison Wesley

My plan is to write website for alpha assembler. The contents should be
implementations for alpha of Knuths algorithms (written for the MIX), a
tutorial on alpha assembling, a downloadable manual page (gcc, gdb and
assembler).

I would be very happy, if you´d like to join me - even if it´s only for a
part of this "journey"!!!

Yours Rudi
