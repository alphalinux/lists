Hi,

With the kernel source 2.2.13 of SuSE 6.3 beta, bttv.o cannot be
loaded because of a missing symbol "init_mm".

It has to be exported in arch/alpha/kernel/alpha_ksyms.c:

--- alpha_ksyms.c.old Tue Jan 18 21:15:00 2000
+++ alpha_ksyms.c Tue Jan 18 22:59:29 2000
@@ -109,6 +109,8 @@
 EXPORT_SYMBOL(alpha_write_fp_reg);
 EXPORT_SYMBOL(alpha_write_fp_reg_s);
 
+EXPORT_SYMBOL(init_mm);
+
 /* In-kernel system calls. */
 EXPORT_SYMBOL(__kernel_thread);
 EXPORT_SYMBOL(sys_open);

With this fix, tv and radio programs using bttv.o work fine.
Maybe this is already fixed in 2.2.14, maybe not...
