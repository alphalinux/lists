On Mon, 24 Jan 2000, Iwao Makino wrote:

> Hi,
>
> I finally got SuSE6.3 installed on the UP2000(w/ lots of headaches)
>
> Now, I went to recompile the kernel and have one question.
>
> How(what) is different between selecting Generic and DP264 as machine type?
> And will there be UP2000 in the future?

If you select Generic it compiles in all the platform dependant code, so
that kernel will boot on anything from a Multia to an ES40. If you compile
as DP264 that kernel will only work on a Tsunami based machine. There will
probably never be a UP2000 specific kernel because there's very little
point in going through the effort. After all the UP2000 is just a DS20
with 4 less memory controller chips (and slot-B instead of the DS20
daughtercards).
