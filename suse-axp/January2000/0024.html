Hello,

I've two problems with SuSE 6.3beta AXP.
The new driver for aic7xxx scsi controllers doesn't work here. It
crashs while installing with "Unexpected SCSI reset". After
reboot, the whole partition is corrupt.
I installed the kernel 2.2.13 from the 6.3beta on my working 6.1
system and compiled it with the old aic7xxx driver. That works.

When I want to use ISDN MPPP, no DNS is working. I used exactly
the same configuration, which works fine on my Intel system. But
on the Alpha, I can access everything with IP addresses, but DNS
names are not resolved. There is an answer from the DNS server,
but the resolver doesn't understand it.
I configured another ippp device in the same way, but without
MPPPP support. It works.

My system:
164LX, 21164, AVM Fritz classic, Adaptec 2940UW.

Ciao... Ralf
