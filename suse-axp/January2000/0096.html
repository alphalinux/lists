Hi..

I have an AXPpci33 (noname). I also had problems (kernel panic, no root
fs or something like that), but after creating an 'install' diskette and
changing 'root' in the boot-line to

  boot scd0:/etc/vmlinux.gz root=/dev/fd0
  
the installation program started and I could install linux.

Unfortunately after CD1 I did not get a prompt for CD2, 3, and 4 and
changing CD's in between is not possible.. I tried to install 'almost
everything'... I will try to install a minimal set this evening and then
upgrade to everything, a suggestion I read somewhere in this list.
