> The most powerful set of tools on the planet is in SRM.
> You should lern it thourally
> such as "set ewa0_mode fastfd" for example.
>

  This tells SRM to set the mode of the eth device - good
  for a network boot, but Linux doesn't do anything with ewa0_mode.
  The tulip driver resets the device and sets the "mode"
  via ethernet options from the boot args or "options ="
  on the tulip line in conf.modules.
