On Monday 17 September 2001 11:09, Posselt Werner wrote:

> As I see from the Suse Website the current version for AXP is 7.1.
> Has development of the AXP distribution been stopped ?

I'm not sure about that, but SuSE announced that it will try to release all
dists for all plattforms at the same time. The question is, whether they
managed to do so or not. We (the Alpha people) might have to wait a bit
longer than the others (although the PPC people are also suffering).

> If so, can you still recommend Suse Linux on Alphas,
> or should I consider switching to something else?

I'm using AlphaLinux on my XP1000 quite a while and works well, not
perfectly, though. I experienced some problems with newer kernel versions
(>2.2.19) and stability under KDE2. Seems to be an Alpha-specific problem,
the SPARCs around here are doing a great job. I recommend using gnome and/or
Enlightenment.
Another point is, that you are bound to Compaqs future development efforts if
you're programming Java or Fortran. The offer great support, but their latest
SDKs still have some weaknesses. C/C++ proramming is fun, they have nice free
binary tools and optimizers.

Neertheless, I'm very satisfied. SuSE produced a well working distribution
that is worth buying (and it's quite cheap: approx. Euro 50,-, Red Hats
AlphaLinux costs about Euro 150-200,-).

Bye,

Steve.
