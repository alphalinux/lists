On Tru64, you have the option to make post-link optimisations based on
execute profiles.

In essence, you compile, link and run the program, collecting profiles, then
re-link the program, specifying that profiles should be used for additional
optimisations. You can then re-run the program, collect new profiles,
re-link, etc, etc.
