Hi!
Yesterday i wanted to get my ISDN to work on Alpha.
My problem is, that it only works, if i compile
the kernel with modules, since otherwise the hisax driver
insists on interrupt 10 (or 12, i don't remember exactly),
which doesn't work.

I would like to compile the stuff directly into the kernel.
I tried these possibilities with the kernel without modules:

All of them weren't successful.

- from milo prompt issue
  boot sda9:/boot/vmlinuz-2.2.13 root=/dev/sda9 hisax=5,2,5,0x300
  (result: kernel hangs)
  
- from milo prompt issue (-> milo howto)
  set BOOT_STRING hisax=5,2,5,0x300
  boot sda9:/boot/vmlinuz-2.2.13 root=/dev/sda9

- from ARC console (Ruffian system):
  generate an environment variable
  BOOT_STRING with value hisax=5,2,5,0x300

- from ARC console (Ruffian system):
  generate an environment variable
  hisax with value 5,2,5,0x300

Now i have no further ideas.
How the hell can i pass the hisax command line arg to the kernel?
