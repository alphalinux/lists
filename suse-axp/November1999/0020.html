On Wed, 10 Nov 1999, Rich Payne wrote:

>
> Can somebody please tell me what is special about the SuSE install
> kernels. I'm trying to create a kernel that will install SuSE for the New
> API UP1000 and it always fails after loading Yast with a failure to load
> libncurses.so.4
>
> any ideas?
Sounds like the ramdisk-size problem. We patch our kernels to use a
ramdisk-size of 20480 as default. For an unmodified kernel you'll need
the parameter "ramdisk_size=20480".
