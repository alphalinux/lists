Hallo !
I'm posting in english 'cause international users are reading on this
list.

Citing from redhat's site:
     * Dec 3000 Series Workstations containing TURBOchannel busses.
If you want to use this machine you will probably have to install a BSD
variant on it. There has been a question on the redhat-axp-list on porting
Linux to TurboChannel bases systems, so if you are interested check the
archive of that list under
              http://www.lib.uaa.alaska.edu/axp-list/
for details.

For further information about supported systems check
                       www.alphalinux.org
AFAIK SuSE supports a subset of the RedHat supported systems, so if it's
not listed under RedHat it won't run. Unfortunately there is no
information about supported systems on any SuSE site (at least havn't
found any).
