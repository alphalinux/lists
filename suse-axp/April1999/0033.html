Could you try the following workaround:
while you are in linuxrc, go to "Settings"->"Debug(Expert)" and
set "Force Rootimage" to "No". Then the installation-system will
be mounted from CD instead of loaded to a ramdisk.
AFAIK this should work above 32M, lower memory would need another
(ugly) hack.
