OK, due to popular demand, here is what I picked up from the RedHat mailing
list some time ago. It was posted by Luke Deller (luked@cse.unsw.edu.au) at
1999-10-21. All credits to him.

The patch was apparently created for 2.2.11, but I have sucessfully applied
it to the SuSE 6.3 2.2.13 kernel.

The patch emulates some (all?) of the extended instructions added to the
alpha architecture so far, which enables older alpha versions to executed
code compiled for later cpu, albeit at a performance penalty (compared to
code compiled for the older cpu), but I prefer slower than not at all.
