You can try the /etc/vm_full.gz kernel from CD1 for booting, it has all
common SCSI drivers compiled in, but it is too large for preplacing the
one on your MILO bootdisk with it. If enter the MILO command line manually,
MILO should give you the chance to insert a floppy(with FAT or ext2fs) with
this kernel on it and load it. Then continue with the install floppy.

I can try to make a special kernel for you if this does not help.

I think your X problem may be a PCI problem which has not been tracked
down yet. There was another graphic card which has shown a similar problem.
Checking with an S3 with 2MB RAM or so would be interesting.
