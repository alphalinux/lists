I tried to run the Tru64 Acrobat reader on my systems, and it failed with
"Illegal instruction".

On speculation that acroread was compiled for at least EV55 (I have only
EV4, i.e. 21064 and 21066), I tried an "instruction emulation patch" which I
saved from the RedHat mailing list some months ago.

It worked wonderfully! I can now run acroread.

If there's interest, I can repost it here.
