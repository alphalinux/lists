good day all ...

thanks to the all the help from this list, i've had success in installing 6.3
onto my AS250. i used the 6.1 aboot and install disks to install 6.3. since
the diskettes had kernel 2.2.9 i built a new 2.2.13 kernel and boot it without
any difficulty. network works, scsi works, so far so good.

i did have a bugette in the kernel build. i could not 'make bzImage' or make
'zImage' i had to make an uncompressed kernel.
