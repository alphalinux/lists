If all you're wanting to do is move the default root, it's set in
...arch/alpha/kernel/setup.c around line 315. I _don't_ know if
it's legal to change it, and my boxen are all set up for sda2 anyway,
but that's where I found it when I went looking a while back. Of
course, it's just as easy to set it at boot time.
