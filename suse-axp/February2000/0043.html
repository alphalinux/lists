<Personal opinion warning ON>

The Alphas have always had way better *system* performance. No bottlenecks.
I bet there are several on this list who, like me, are running the latest
versions of Linux/NT/VMS/whatever on their *old* Alphas, and they're still
happily running at about the same perceived speed as ever. Only the absolute
top-of-the-line x86 systems of comparative age are still able to boot, let
alone run, NT at any usable performance levels.

Witness the Athlon. It owes its sucess to the Alpha bus technology, licensed
from Digital, which gives it system performance and load resilience.

Where would we be if Intel had been the sole player in the field?

Of course I would like to have newer Alphas, but the ur-models I'm using
(AS200, Multia) are still giving me adequate performance for the simple
tasks I'm using them for, at (I believe) a rock-solid reliability that I've
never got from any other platform (except my Vaxen...) that I've used. My
only problem is getting the applications I need, native on Alpha, under any
Alpha-OS. Thus I applaud being able to run Tru64 applications on AlphaLinux,
since this broadens the available software spectrum by orders of magnitude.
