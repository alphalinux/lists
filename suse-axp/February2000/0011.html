Hi,

First, thanks to Alex for sending me a TGA and as well to all the
others for offering. I did some testing today and it seem that
a kernel with support for VGA-console and TGA-console fails to
work right on TGA equipped machines.

I've put two kernels (together with System.map and their configs)
at /mirror/SuSE/ftp.suse.com/pub/suse/axp/update/6.3/kernel/TGA

The smaller one (vmlinux-tga.gz) should fit to replace the kernel
on a bootdisk and work with the modules on the install image.
The other one, vm_full.gz has support for some SCSI-controllers
compiled in.

Please report if this helps or fails for you.
