Disclaimer: This is ugly and not elegant at all. It will create
complaints from the box. It _has_ worked for me.

cd /
touch now
find . -newer now -exec touch {} \;
rm now

This doesn't exclude /proc which gripes a lot, but it does get your
dates back to the present. The 20-year offset is an ARC-ish artifact.
You would obviously want to set the OS date before doing this.
I use /sbin/clock -u -w -A to set the h/w clock, so I can't comment
on the other method. YMMV.
