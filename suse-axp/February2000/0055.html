On Sat, 29 Jan 2000 20:38:56 +0100, Cor Slot <suse@csoft.nu> wrote:

Cor> Unfortunately after CD1 I did not get a prompt for CD2, 3, and 4 and
Cor> changing CD's in between is not possible.. I tried to install 'almost
Cor> everything'... I will try to install a minimal set this evening and then
Cor> upgrade to everything, a suggestion I read somewhere in this list.

This was on an AXPpci33 machine (21066 processor) and 32 MB memory...

Today I upgraded this machine with 64 MB RAM to a total of 96 MB memory...

I reinstalled SuSE Linux 6.3 AXP, and to my surprise I got a prompt to
switch CD where I didn't get a prompt previously...

My conclusion: the installation program YaST doesn't ask for CD
switching due to memory problems on an AXPpci33 (noname) with 32 MB
memory... This is probably true for other types of machines too..

I think the minimum or 24 MB RAM memory stated on the box is too low,
maybe this should be 64 MB?
