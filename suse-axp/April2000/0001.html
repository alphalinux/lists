I wondered whether the ruffian mb has temperature sensors for cpu and
fan sensors to get the cpu and powersupply fans. And if so, are there any
ways to use them in Linux?

Why do I want to know? The CPU gets very hot when running, see
http://shayol-ghul.hemmet.chalmers.se/mrtg/load.html for load stats, and
I would like to try to cool it a bit using liquid cooling. Thus I'd very
much like to know how much the processor is cooled. Also I'd like a
statistics page over the cpu-temp to be able to plot the load-temp
diff.

If no temp sensors exist, does anyone know where I can find such addon
card, preferably PCI, but ISA is ok, with the temp-specs of Si (around
40-500K).

For building the liquid cooler I thought I'd use a standard 21164
heatsink, but I can't seem to find any on the market. Does those babies
only come with the cpu, or have I missed something?

I'm not out to overclock my computer, I'm just trying to make it run
more silent. Making it cooler is just a (good) side effect, and thus
extending it's lifespan.
