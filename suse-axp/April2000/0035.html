On Thu, 13 Apr 2000, Werner Posselt wrote:

> Hello,
>
> I know this is a FAQ but I cannot find proper information on how
> to install linux on the axp150.
> Is it possible to boot from CD-Rom ?
Yes the Jwensen is able to boot from cds, only the generic kernels do not
work on jensens. But I guess using the kernel from khubla should allow you
to run the installer, but I remember that there is one option in debug or
something where you can disable, the the ramdisk gets reloaded from cd. As
the khubla aboot-kernel-disk sports too small a ramdisk, you have to stick
to the ramdisk on floppy.
> I have a copy of Suse AXP 6.1 but a readme says that jensen is not yet
> supported. True ?
Yup, no support, well Rüdiger sure is helpfull, it is just no real support
out of the box.
> www.khubla.com/jensen.html contains some information but I had no success so
> far.
>
> Does the boot Partition have to be of type boot (1) or ext2 (8) when
> partitioning with minlabel ?
I would recommend to create a first partition of around 64M as home for
the /boot tree of the filesystem starting around cylinder 4. That leaves
room for aboot and an initial kernel before that partition.
> The instructions provided by Compaq/Digital disappeared at the gatekeeper
> ftp-site.
> There is just a note that the procedure is being improved.
>
> Is there any other documentation available ???
Have a look at http://tuddv2.ced.tudelft.nl/jensen, there is more recent
documentaion, although regarding rh 6.2.
> Appreciating any hints,
>
> Werner Posselt
