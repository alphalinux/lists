Hi Wolfgang Kutscherauer,

>
> totay i updated my AS1000 4/266 with a Mylex RAID DAC960P to firmware
> 3.x (2 new flash-Roms).
> Now the DAC960-driver find the controller.
> But there is one big trouble:
> The old Config-Tool from DEC couldn't work with the new firmware.
> Do someone know anything about a configtool for the DAC960P with
> firmware 3.xx like the
> swxcrmgr.exe for the old firmware 2.x
>
> Mylex mailed that there is no configtool for Alpha-AXP.
>

The last infos I have is that Mylex and Digital (Compaq) use different
firmware for the DAC.
The latest Compaq release is about 2.7 and is different from the one
Mylex uses. The configmanager
ra200rcu.exe works only with the Compaq releases (I guess so). It can be
found on the firmware CD.

There is also a new tool on:

http://www.compaq.com/support/files/alphant/drivers/index.html
