Hi

I am having great difficulty deciphering the Suse manual for 6.1 axp. Can
someone give me a quick howto for doing an install using <<both>> ARC and
SRM ?

SRM:
I have made the aboot disk from /disks/aboot and the install from
/disks/install. The instructions say to 'boot dva0' at the SRM prompt
which I do and the boot sector is found but then fails.

ARC:
I make the milo boot disk from /milo/pc164 and the install from
/disks/install. I set up the ARC bios to use the settings from the manual,
i.e.:

SystemPartition=multi(0)disk(0)fdisk(0)
OsLoader=multi(0)disk(0)fdisk(0)\linload (there is no ".exe" )
OsLoadPartition=multi(0)disk(0)fdisk(0)
OsLoadFilename=\milo
OsLoadOptions=boot fd0:/vmlinux.gz root=/dev/fd0

MILO boots fine and then requests the 'root floppy to be installed into
ram' and I insert the 'install' disk. the system finds a compressed image,
commences loading and then FAILS saying that it has reached the end of the
file or disk or something like that. I have remade this disk several times
and gotten the same results.
