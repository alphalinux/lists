Use either a DP264 or a generic kernel and you should be fine. You'll need
to boot from SRM, not AlphaBIOS, and I think you'll find that RedHat does
not supply a MILO for this machine ( at least I'll be rather surpsrised if
they do).

I can tell you it does work, I have installed SuSE 6.1 on a UP2000 very
similar to the one you have there. If you have any trouble let me know (I
also happen to work for API).

--rdp
