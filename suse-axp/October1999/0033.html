AS250 4/266 is moving along. I still cannot run SaX on the machine
however. Since I can't use the vga server I have to start SaX with the
option --server /usr/X11R6/bin/XF86_TGA which starts off alright
showing the Kummer surface and claiming that it is loading. Then the
first window of SaX appears then disappears with the Kummer curve
showing again and SaX claiming that is is loading again then the raster
appears and SaX hangs. The StartLog it leaves behind is attached below.
Any ideas of how I can get SaX running would be appreciated.
Alternatively, it would be okay for me to run xf86config. However, for
that I need the monitor specifications for the digital VRT17-HA. If
someone has the same monitor, could you please send me your XF86Config
file? That would really help. Thank you very much in advance, Alex.
