On Tue, 19 Oct 1999, Iwao Makino wrote:

> full, but seems Xservers were not installed, so sax did not even work(using

The problem here is that sax expects the vga16-server, that doesn't work.

>
> reboot w/ MILO using 2.2.7 kernel from Samsung(because suse kernel installed
> by CD gave me 264 not implemented error).
> system works ok, not X

You will need to install the xglint-server, then start sax as follows:
sax -s XF86_3DLabs

then sax should work.
