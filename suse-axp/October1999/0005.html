Strangely enough, I found an installation guide on Compaq's site for the
DS10/20 etc machines. The instructions work very well for the AlphaPC164.

One thing though -- the "c" partition that is created for BSD disklabels is
not at all problematic for Linux. This is mistakenly asserted in the
Compaq guide. The only issue is the addition of the "-f3" switch when
using swriteboot.
