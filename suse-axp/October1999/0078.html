On Mon, 25 Oct 1999, Waite, Michael wrote:

> Also...............
>
> The standard boot line is:
> boot dva0 -file vmlinux.gz -flags "root=/dev/fd0 load_ramdisk=1
> bootdevice=/dev/fd0"
>
> Note it is LOAD_ramdisk not BOOT_ramdisk.
well, if it's the original suse-kernel, root=/dev/fd0 will load&prompt for the
ramdisk if just "root=/dev/fd0".
If it isn't, you will need "load_ramdisk=1 prompt_ramdisk=1 ramdisk_size=20480"
