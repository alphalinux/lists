On Thu, 7 Oct 1999, Lennart Börjeson wrote:

> I've sucessfully installed a basic SuSE Linux 6.1 on my Multia. I've
> installed the "SuSE default system" + the xsrv package.
>
> When I try to start sax, it says:
>
> sax: searching card...done
> sax: scanning failed switching to VGA16 server
> sax: VGA16 server not installed abort
>
> Any hints on how to proceed?
Hi,
sax doesn't find the graphics-card by itself in this version
(an ugly little bug...).
You can tell SaX which server to use by running it as
sax -s s3 (for s3-based cards for example)
