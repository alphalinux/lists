Well, I can tell you it is NOT an aboot problem, but rather the kernel you
are booting. Can you copy the kernel off the CD and try that? It seems
that the default kernel SuSE has installed is having trouble with the
Cypress (this is the IDE, keyboard, mouse, ISA etc... controllr) chip.
