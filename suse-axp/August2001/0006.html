I resolved the problems with getting the ATI Xpert98 (RagePro)
and the 3DFX Voodoo3 setup with SaX2 on an XP1000.

The trick is to use xf86config4 to configure the cards *and* install
the 3.3 X related RPMs. I don't know *why* the older 3.3 modules
are required, but without them I could not get X to start. It seems
that one of the 3.3 packages contains the X wrapper needed for
startx to work properly.

I now have these cards working well. Unfortunately, 3D acceleration
is not supported on the Voodoo3 because the glide package is not
included for Alpha.

I'm now much happier with SuSE 7.1 but I think SaX2 needs a bit
of work. It's buggy and should have more intelligence in detecting
which packages are needed for a working X configuration.

Good job with XawTV too! It worked out of the box after running
the v4l setup tool.

-Michael
