The 6.1-modules are on the install-disk belonging to 6.1. You should be
able to install 6.3 over FTP with this 2 disks, but you *might* run in
trouble, when it comes to a reboot during the installation-process.

Sorry, that's all I know.

Perhaps you should try again to start from the 6.3-aboot-images. I was
definitely able to start the installation on an AS 255 from 6.3-CD. So the
6.3-aboot-image works in principle.

Bye.
