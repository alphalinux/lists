Actually the OFFICIAL starting point is http://www.compaq.com/linux (which
redirects you)

As for building the kernel with CCC, unfortunately no. The kernel contains
highly specific GCC conventions (ASM statements for one) which are not
supported by CCC, the readme documents some of these.
