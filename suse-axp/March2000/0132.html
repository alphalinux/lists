Hi,

as I see in the patch there is an option -mev6 , which will select ev6
cpu's (as I understand it).
Remember that you should usually build things for the smallest cpu (ev4)
or keep it generic.

This is also the same for Tru64 Unix, there you suffer serious problems
running an optimized
programm for ev6 on an ev4 ,it should work, but in a bad case up to 1000
times !!! slower than for
the processor optimized , because the optimized instructions get
emulated.

I really don't know how linux does optimize for the different
processors, but I think that the behaviour is
even worse.
