You would probably be better off using the
boot dva0 -fi "" -fl 0

If memory serves me the SuSE initrd actually requires a larger than standard
ramdisk. I think the boot option is "ramdisk=2048" in addition to all the
others... using a flags of 0 should set all these for you already. FYI -
sometimes a prompt_ramdisk=1 is also required.
