There were several changes since v3.63beta, see
http://cvs.sourceforge.net/cgi-bin/cvsweb.cgi/lame/?cvsroot=lame
Some of them affect speed, so I think the latest version from CVS should
be slightly faster. AFAIK my patch was included after v3.63beta, so
there should be fewer compiler warnings with the latest version.

If you have CVS (included in every(?) distribution), checking out the
repository only requires two commands (as mentioned on
http://sourceforge.net/cvs/?group_id=290).
