On Tue, 7 Mar 2000, Craig Allen wrote:

> Hi.
>
> Which Disk has the root filesystem.

If you are doing the install then it would be the floppy (or CD-ROM). You
boot commmand might look like this:

>>boot dva0 -fi vmlinux.gz -fl "root=/dev/fd0 load_ramdisk=1"

If this is after the install, then it is whatever you setup during the
install for your root fs.
 
> This seems to be my problem.
>

possible.
