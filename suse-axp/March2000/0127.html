If you compile the whole qt-package with optimization (-O2) you'll get
that misplaced icons in the menu
and font problems. If you keep it away, it works fine afterwards. The only
problem is that you get unaligned
kernel traps. I don't know how to get rid off them, there seems to be no
compiler flag for avoiding it.

Another thing is that if you build the kde with a new qt without
optimization, you 'll get always unaligned
kernel traps and the most applications work, but not kvt. Also kmoon gives
a floating point exception.

It' s really annoying !!
