Iwao MAKINO wrote:
>
> Has anyone tested 6.4 beta for Alpha?
> Any comments? I am about to give it a try, but would like to know outside
> source opinion.

I'm running it in an AS 400 4/233 (ev4 chip). 6.3 had X built with ev6
extensions that made it unusable in my machine when I wanted to run KDE
as my desktop manager. 6.4 beta had this corrected and runs mostly fine.

The GTK development packages were not included, so you can't build GTK
based packages. QT devel was included. It doesn't include TeX/LaTeX.

I've experienced some freezes while running Emacs in X. They leave no
apparent error message in the log.

Since I need to use LaTeX in this machine (will use the 6.3 sources), I
really would like to find out what is causing the freezes.
