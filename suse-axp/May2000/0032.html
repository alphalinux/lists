You don't need to install AlphaBIOS (isn't it rather ARC on the 255?), you
can boot Linux from SRM. Also called the "aboot" way.

I'd say it is definitely worth the 46EUR to buy the original distribution,
then you'll get all files in the right place without any hassle at all. Plus
a manual, which describes how to install and boot from SRM...

BTW, are all the various problems we had with installing on AS25X in 6.3
corrected in 6.4 ? Especially, can you install 6.4 on an AS250 by booting
from the CD:s?)
