Your problems may be caused by using a low color resolution X server. If
X is set to 8 bit color (256 colors) it is very likely that you'll get a
missing or "unable to allocate" colors. In this case gv (or to be
accurate, ghostscript) is trying to use antialias to avoid jagged font
shapes (what you call jiggling?), that needs several gray scales and it
can't get them.

Set your X server to 16 bit color or more. To find out what color
resolution you are using run the 'xdpyinfo' command and look for the
'depth' entry.
