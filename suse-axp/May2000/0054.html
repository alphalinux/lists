Now that Compaq has made their own Alpha-optimised C and C++ compilers
available, I'm sure more Alpha-users than myself have toyed with the idea of
building a complete Linux with these tools. I'd guess most of the
applications and much of the kernel would compile without changes, but
ofcourse, that's unknown until tested. I know there are assembler parts and
inline code which probably would not compile, but these might possibly be
left to Gnu tools.

Has SuSE any plans to try this (or have you already done so)?
