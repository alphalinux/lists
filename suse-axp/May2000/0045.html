"G. Hugh Song" wrote:

> > No, the adaptec stays enabled. The machine doesn't select the boot disk,
> > you do via the bootdef_dev SRM variable.
>
> Would you please write the exact SRM command to change that variable?

Hi!
Make a "show dev" then you see a list of the devices. You see e.g.
"dkb0" where your disk is connected. Then type in "set bootdef_dev dkb0"
and as next command "init". Then you can boot to this disk.
