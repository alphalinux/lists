> I'd like to install Linux on a Alpha255 (DEC). On the system is OpenVMS
> installed, so I have to remove this first. In the machine is only a CDROM Drive
> and no Floppy.
> Is it necessary to install the AlphaBios first? If the answer is yes, how can I
> do this from a CDROM, and where can I get the files? I tried to download the
> Alpha distribution from SUSE, but how can I create bootable CDROMs with an
> Windows NT machine, and wich files belongs to CD1, CD2 .....
Since you are using OpenVMS you will probably already have an SRM console
and this is sufficient to boot linux (IMHO even easier than the AlphaBios
variant...).
Try to get an iso-image since it's absolutely non-trivial to split a ftp-version
on several CDs, you'd end up having to re-write all of YaST's configuration
files...
Maybe you'll want to try the image presented on www.alphalinux.org, it's
a SRM bootable CD.
