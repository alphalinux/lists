Compaq has released JDK 1.3.1 for AlphaLinux
(http://www.compaq.com/java/download/jdk_linux/1.3.1/index.html).

The page says this beta is supported on RedHat 6.2 only, but I've
successfully installed and run it SuSE 6.4. (Just had to ignore the
dependency on glibc.)

I've also successfully used it to install and run Sun Forte 2.0 CE. Download
the "All Platforms" distribution to get a pure java installer.
