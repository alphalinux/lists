Hi guys (and gals certainly)!

I just managed to put together a (very first) package of kde2 for SuSE/AXP. I
compiled it using Compaq's CXX and CCC. It is way fast (faster than gcc kde1
on my machine: ev56 sx164 533MHz).
The desktop is not extremly usable yet (why - see below!) but I would like
some people to help me finding the bugs. That where produced by cxx.

Some things I already know, that don't work:
1. konqueror does not like http connect. It renders html pages correctly when
loading them via ftp).
2. konsole dies away once in a while with a segfault.
3. Desktop appplet in statusbar once in a while looses track of reality.
4. when logging in KCMinit issues a sigsegfault. (Doesn't seem to have any
consequences though.

If anybody is interested in it I will upload a complete binary (and patched
source) package to our company's ftp server.
