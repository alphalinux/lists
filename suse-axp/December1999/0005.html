I recently started using SuSE-AXP and tried to find information about my
installed programs

What might be my C library?

$ rpm -q -i -f /usr/lib/libc.a

Version : 99.5.28 License: Copyright (C) 1992, 93 etc.

What the fsck is version 99.5.28? And isn't the glibc License LGPL?

$ rpm -q -i -f /usr/bin/gcc

Version : 990408 License: 1999 - not specified

Again the same questions. License clearly is GPL. What is version
990408?

$ which gdb

which: no gdb in ... (no further comment)

In short: How do I find out what I *really* have installed?
How do I compare it to commonly used version numbers?

looking at /usr/include/bits/types.h, I see that time_t is defined as
long int.
We do have glibc2.1, right?
