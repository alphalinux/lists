Is this a 21264 based machine? If so you need to use the
DP264 kernel, not the generic kernel. I had the same problem
on an XP1000. I could not boot with the generic, but had no
troubles with the 21264-specific kernel. I think it's boot
option 3 (could be wrong...try every number from 0-4 and you'll
hit it eventually).
