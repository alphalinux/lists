Jaap,

This hang is normal- don't panic! Just reboot using your MILO floppy, then
from MILO boot <drive>:vmlinuz root=/dev/<root partition> and the install
process will pick up where it left off. After this you should be able to go
back into the AlphaBIOS OS selection function and set it to boot the hard
disk directly.
