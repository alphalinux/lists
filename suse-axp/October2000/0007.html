> > Hello,
> >
> > i would like to install SuSE Linux 6.4 on my Compaq XP900.
> >
> > >>>boot dqb0 -fl 0 (SuSE Linux 6.4 CD 1)
> >
> > ...
> > ...
> > aboot: loading initrd (2213148 bytes/2161 blocks) at 0xfffffc0017c4a000
> > aboot: starting kernel /etc/vm_full.gz with arguments rw root=/dev/fd0
> >
> > this is the last message, then the computer hangs.
> > What can i do?
> > SuSE Linux Version 6.3 works fine.
>
> Try to use the 6.3-bootdisk and the bootstring:
>
> b dva0 -fi vmlinux.gz -fl "root=/dev/scd0 initrd=/etc/initdisk.gz"
>
> with CD1 of 6.4 inserted in dqb0
>
> Maybe the new aboot makes trouble on this machine.

After i do this, the computer crashes with many PCI Errors. :-(

But i solved my problem with the "jumpstart"-CD from
Compaq. http://www.linuxalpha.compaq.com/jumpstart
First prepare a harddisk with a valid BSD disklabel is very important.
The original disklabel from a Compaq TRU64 operating system
MUST be destroy.
