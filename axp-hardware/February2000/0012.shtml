Hi, I'm still looking around for more memory for my XL300, probably
256MB (4 x 64MB SIMMs).

Prices range from about $99-$235 each for generic true parity 64MB
SIMMs, all the way up to $1200 for 256MB from Kingston (specifically
listed as for the XL300). *wow* I've seen entire 433MHz alpha systems
for less.

According to some of the specs for the memory for sale, the SIMMs are
true parity but *not* ECC. Is regular true parity memory adequate for
an alpha, or does it need more? I'm a bit fuzzy on the exact
distinction between parity and ECC.

The docs I've read only say that 70ns 36-bit SIMMs are needed. I'm
assuming these are not EDO. Are all 72-pin parity SIMMs "36-bit"?
Any comments?

Thanks,
Wes.
