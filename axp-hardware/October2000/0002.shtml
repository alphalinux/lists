Hello;

Someone already recommended Debian, so I'll second that.

Give Debian GNU/Linux a try. The inital install may be slightly more
time consuming then say RatHat, but it's well worth it IMO! I believe
the base install for the lastest release of Debian GNU/Linux is less
then 50MB. After you have a base install completed you can then
configure 'apt' for your local/favorite mirror. 'apt' can then install
whatever package(s) you want as well as remove those you don't.

I recommend subscribing to both debian-user and debian-alpha mail lists:

Send mail to "debian-user-request@lists.debian.org" (with subscribe in
the body). This list is excellent for general Debian and Linux issues.
If you need to know something that is not platform specific, you ask
here. There is also a digest version of this list (it is high traffic).
just send your email to "debian-user-digest-request@lists.debian.org"
instead of the address above.

Send mail to "debian-alpha-request@lists.debian.org" (with subscribe in
the body). This is list is for debian on alpha. If you have a question
about something specific to debian on alpha, here is the place.

When installing you will be asked to define a configuration for your
system (i.e. workstation, server, etc) just select the base install
option.

If you require help installing, feel free to contact me.
