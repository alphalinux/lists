Hi!

Well, I've also went through documentation for UX and didn't find
anything about serial console support. However in Alpha/Linux FAQ
question 12.4 is:

<quote>
12.4 Can Linux/Alpha Run Without Graphics Card?

The firmware can be controlled over a serial line terminal at 9600 Baud
and it is possible to boot Linux/Alpha in that manner, but
Linux itself does not currently support headless boots. There are kernel
patches floating on the net that add this functionality,
however.
</quote>

What about this?

The point is, that I'm having few machines without keyboards and monitors
and hooking them up every time you need to change something is a pain.
Possibility to control BIOS over serial line would be a great advantage.
There is a solution (www.realweasel.com) but it kind of costly (the price
for the card is 250$ plus shipping).
