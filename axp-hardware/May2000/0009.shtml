On Sun, May 07, 2000 at 05:28:00PM -0700, Ron Farrer wrote:
>
> Hello;
>
> Does anyone remember what the memory bus width (bits) was on the
> AXPpci33 motherboards? For some reason I seem to recall it being either
> 128-bit or 256-bit if all banks were filled with the exact same
> size/speed memory. Anyone remember for sure?

The board has 4 PS/2 slots for memory. I *think* that you can use them as
2x2, but maybe you need all SIMM modules to be of one type (1x4). However,
you have to use full-parity modules (which are unfortunately *very* expen-
sive).
