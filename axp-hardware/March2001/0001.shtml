Martin,

Sounds like you are a MILO/AlphaBIOS booter, you will note there is some
difficulty in getting into the SCSI BIOS setup program with most cards,
probably a limitation of SCSI BIOS emulation in AlphaBIOS. There should
still be a tool at Compaq in the Alpha/NT section for modifing NVRAM
properties from an ARC console prompt. I don't have the name of the tool
anymore, but I do know the Adaptec SCSI ARC console app is definitely still
there.

Worse comes to worse, yank the card, set it from a Wintel/DOS machine stick
it back in. The nifty NVRAM dump/modify utils are DOS realmode only, and
well worth the extra fuss.
