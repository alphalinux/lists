I try to get informations about RAID controller currently supported by the
SRM console, i made some tests with recent controllers on diffrent
(old) systems without having success :

Controller tested on 164LX and 164SX :

DPT SmartRAID V and ADAPTEC 2100S
( same board in fact )

SRM 5.7 not found.
SRM 5.8 don't detect anything.

SRM console. ( same chipset is using on 29160 board ).
