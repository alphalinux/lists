I am new to the Alpha platform and to Linux. So just to make it hard I
thought I'd try putting Suse Linux 7.0 for Alpha on an Alpha box. Here are
the spec's:

ALPHA Clone PC 600MHz 
* 600MHz
* 256 Megs of Ram (TWO 128meg DIMM's)
* 4.3 Gig Hard Drive (seagate ST34501W)
* 3 1/2 Floppy drive
* 32 meg PCI Video card (OXYGEN 402)
* 32X IDE CD-ROM
* SCSI PCI CARD
* NIC CARD 10/100

System ALPHA PL 164LX, ALPHA BIOS VERSION 5.62-1A, DIGITAL ALPHA 21164
PROCESSOR

So I follow the instructions in the Suse book and set up everything. I
create the boot disk and boot up with the floppy and the CD in. The first
set of error messages said: unknown file system on on device sr0. And the
second one was: failed to load the kernel. So I formatted the hard drive
with the built in formatting app and changed the OS options path from:
boot scd0:/etc/vm_full.gz root=/dev/scd0
to
boot hda:/etc/vm_full.gz root=/dev/hda
After I did that I got these error messages:
Warning: unable to open an initial terminal. And
Kernel Panic: no init found, try passing init= option to kernel

HELP!...
