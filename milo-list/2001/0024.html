You can pass the boot parameters to MILO from ARC Bios. If you look at the
boot selection you've setup, I think they're called the Boot Options, but
it's been a while since I've used ARC Bios.

Also MILO by default outputs everything to the first serial port so you
should be able to control it from there as well.
