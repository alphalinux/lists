Hello world,

It's time for a new release of Milo-2.2, the Alpha-Linux boot loader for
ARC/AlphaBIOS machines.
This version is based on the Milo-2.0.35-c7 sources of Nikita
Schmidt , which are available on ftp://genie.ucd.ie/pub/alpha/milo/

Please note: These sources are currently only tested on Ruffian. Several
other platforms compile cleanly, but are not tested.

Currently the following platforms are at least compiling:
        * Alcor
        * Cabriolet
        * EB164
        * Miata
        * Mikasa
        * Ruffian
        * SX164
        * Takara

Please send me all positive and of course negative test reports.

The source, binaries and patches to older Milo-2.2 releases can be found at the
following URL:
        http://www.stefan-reinauer.de/Milo/

Regards,
     Stefan Reinauer
