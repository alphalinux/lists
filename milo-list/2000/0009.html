On Friday, 7 Apr, Damian Maxwell wrote:
>
> When i issue a 'show' command at the milo prompt I get the information
> abount bios emulation, device drivers: SCSI (a few), IDE and Floppy.
>
> MILO then loops after 'Please wait while setting up devices....'
>
> It just sits there printing dot after dot.
>

Firstly, you can see what's actually going on by entering
        set VERBOSE 1
before the 'show' command, but it's not going to be of much help, I guess.

Secondly, I had this problem myself, but I couldn't diagnose it because
Linux also hung in the same way MILO did. Try booting Linux using an
IDE-disabled MILO (ftp://genie.ucd.ie/pub/alpha/milo/XLT-without-IDE/milo)
to see if Linux can recognise your IDE disk properly.

Nikita
