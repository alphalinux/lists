On Wed, 22 Dec 1999, Raffaele Cicchese wrote:

>
> Thanks to Rich Payne and to all that helped me to install RH6.0!!!
> Now the alpha is running the pinguin!!!
> Anyway there's a problem :-( I cannot boot from the hard disk and need
> to use the boot floppy with the command:
>
> boot dva0 -fi vmlinux.gz -fl "root=/dev/sda1"
>
> I also recompiled the kernel with the latest 2.2.13 an replaced the two
> files /boot/vmlinux and /boot/System.map, but it doesn't work.
> Any clues?

You need to put the vmlinux.gz file onto the floppy. After you compile the
new kernel the file will be in /usr/src/linux/arch/alpha/boot (if you're
kernel source isn't in /usr/src/linux then substitute the correct
directory).

As for the harddisk, is this SCSI ot IDE. It's a DS10 so I'm assuming IDE?
Did you partition the disk with BSD disklabes?
Have you used swriteboot to install aboot?

If not, take a look at this:
http://talisman.alphalinux.org/faq/srm-3.html#ss3.3
