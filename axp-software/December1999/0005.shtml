> > This is the X server from the 3DFX webite? How do you feel about sharing

> Fine - although I'm not sure how much you'll need - I'll try and take it
> off on tape and upload a built version to an ftp site next week sometime.

> However building it isn't that bad. use rpm --install to install the
> source RPMs. Then rpm -bc /usr/src/redhat/SPECS/DRI.??? to build it.
>
> It will fall over building the Mesa section.
>
> cd /usr/src/redhat/tdfx_dri??? find the Mesa subdirectory

>
> Then cd into the top of the Xserver code and make that then make install;
