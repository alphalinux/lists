Michael Kane schrieb:
>
> Hello,
>
> I'm having trouble installing version 2.2rev0a of debian using cds. I'm
> able to get milo to start up without a problem. When I pass
> boot hdd:/boot/linux root=/dev/hdd/boot/root.bin load_ramdisk=1 to milo
> the boot process beings without a problem, but the I get the messages:

Did you try to go the "normal" way for ARC/AlphaBIOS systems,
that is to install using two boot floppies and to boot into the
installation process using

boot fd0:/linux.gz root=/dev/fd0 load_ramdisk=1

as milo command?

A few days ago I tried to boot from a potato CD on a ARC based
system (PC164) and I was not successfull. I found another way
to avoid the need for boot floppies, which I described in

http://lists.debian.org/debian-alpha-0011/msg00109.html

So booting from floppies or from HD should work, but I'm not
aware why booting directly from CD does not work using
ARC/AlphaBIOS and milo.

Good luck,
