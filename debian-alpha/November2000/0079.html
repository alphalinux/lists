> What was the exact commands you used to create the disks?

I created both floppies under RH 5.1 on the same machine
on which I'm trying to install potato.

The FAT formatted floppy:
mkfs -t msdos /dev/fd0
mcopy root.bin a:

Results in (using load_ramdisk=1 at Milo prompt)
RAMDISK: Couldn't find valid RAM disk image starting at 0

The unformatted floppy:
dd if=root.bin of=/dev/fd0 bs=1024 conv=sync; sync

Results in (again, using load_ramdisk=1)
ufs_read_super: bad magic number
Kernel panic: VFS: Unable to mount root fs on 02:00

So should I give tar a try? Or ext2fs, minix, ...
formatted floppies?
