"Rob B" <robbyrnes@ozemail.com.au> writes:

> Hi all,
>
> I need to build a kernel on my Alpha (Debian 2.2 with kernel version 2.2.15)
> for an i386 target, with the intent of making a netbooting image. I know
> I'll need the kernel sources, but is there anything else that wouldn't be
> installed as part of a general Debian install? I looked around the LDP, but
> couldn't find anything there. What are the steps needed to build and
> cross-compile a kernel? Or am I getting too far ahead of myself :)

You need to compile gcc and binutils for crosscompiling. I don't think
debian has any of them (only i386 to m68k crosscompile stuff) so you
have to fetch the source and read the documentation therein.
