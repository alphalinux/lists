Thanks again for patching those programs.

mpg123 is working, but xmms stopped working. I was getting a dialogue
box with the title: "Couldn't open audio"

Please check that:
1. You have the correct output plugin selected
2. No other programs is blocking the soundcard [sic]
3. Your soundcard is configured properly

Upon inspection, the output plugin was set to OSS instead of esound.
I'm not sure how that got changed. Changing it to esound fixed it.
