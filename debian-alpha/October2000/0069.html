On Thu, Oct 19, 2000 at 01:24:50AM -0700, Jakob 'sparky' Kaivo wrote:
>
> I seem to be having the exact opposite problem. I have a PC-164 with a
> SCSI disk (Seagate Barracuda) and Tekram controller (DC-390F), but
> while installation goes fine, SRM is *not* happy. When I run "show
> config", the controller is shown, but "show device" shows only the
> floppy drive (DVA0). Does anyone have any experience with SRM not
> liking Tekram controllers?

SRM is not guaranteed to like *any* card that it wasn't tested/sold
with. That's not to say it won't work with an arbitrary card, but
that card best "look like" (PCI vendor/device IDs) one that it was
tested against.

If you don't have the latest SRM console version, try updating to it
and then try again. If it still fails, I guess you're out of luck...

Good luck.
