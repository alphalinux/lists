"George A. Dowding" wrote:

> dowdingg@ayes ~/src/openssh]$ dpkg -l libglib1.2-dev libgtk-dev
> Desired=Unknown/Install/Remove/Purge/Hold
> | Status=Not/Installed/Config-files/Unpacked/Failed-config/Half-installed
> |/ Err?=(none)/Hold/Reinst-required/X=both-problems (Status,Err: uppercase=bad)
> ||/ Name Version Description
> +++-==============-==============-============================================
> pn libglib1.2-dev <none> (no description available)
> ii libgtk-dev 1.0.6-4 Header files and static libraries for the GI

That's the old 1.0 gtk-dev. You need libglib1.2-dev, and probably libgtk1.2-dev.

There should be a Build-Depends line in the .dsc file of the openssh source package, listing the packages needed to build it
successfully. If there isn't one, file a bug report.
