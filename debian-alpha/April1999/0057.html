I responded yesterday, but my email system was all screwed up so I guess
it didn't get through :(. It took me 2 weeks to get the jensen to finally
take debian. First of all, the jensen SRM console can't access the floppy
so you have to boot into the nt side and run copyboot.exe from a floppy
to copy a floppy image to the hard disk. For some reason <at least with
my jensen> aboot wont boot linux if it has tried to boot a kernel that
dosnt exist. What i ended up doing is compiling the aboot utils for i386
<don't have any other alphas> and created my own boot disk to write to the
HD to boot off of. you sould read the Jensen-HOWTO, it is at ftp.dec.com.
If you would like a copy of my custom 'boot disk' ill send it to you.
Just rember that a jensen is quite a challange to install to.
