Greetings,

There's a "framebuffer howto" on http://www.alphalinux.org/ which might
help. Matroxfb is somewhat broken on Alpha so it won't boot in graphics
mode, this is a known problem with no known solution. :-(

But it will boot in text mode and then switch nicely into graphics. The
howto describes how to do this in all the consoles automatically (on
Redhat), IIRC.

You don't get the penguin, but everything else seems to work.
