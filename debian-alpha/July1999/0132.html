Yes the debs are there. Christopher C. Chimelis has put together the deb
packages. They work great, except a litle problem with the shared library of
the cpml. He already corrected this bug, but I am not sure if the new files are
already uploaded to the compaq server. But if you can not wait just get the
missing .so lib file from the corresponding .rpm file and copy it to the
correct directory and set the symbolic links yourself. Then it worked for me.
