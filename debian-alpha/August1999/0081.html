I dont use xcdroast, I just use dd to create an image and if it's an
important cd, I'll use diff -r to verify the file stucture against
the original. Then I burn the image with cdrecord...

i.e.

dd if=/dev/scd0 of=cd.iso
mount /dev/scd0 /cdrom
mount -o loop cd.iso /cdrom2
diff -r /cdrom /cdrom2
umount /cdrom
umount /cdrom2
cdrecord -v speed=4 dev=4,0 cd.iso
