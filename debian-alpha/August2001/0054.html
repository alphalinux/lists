I tested on Ruffian, so that wouldn't be as intresting as the other
archs.

I'm uploading the set to ftp://134.2.15.134/boot-floppies/disks-alpha

But you can build your own working set:
- Use the mklibs.py instead of mklibs.sh in rootdisk.sh
- Use Falks fake_get patch if you hit that sysvinit problem
  (It only appears when you didn't already downloaded some packages)
- Compile sid's ash (If you get a linker failure, the symbol is an
  alias, just change the one occurance to what the alias points to)

There you go, "make" should work.

For "make release" you need some small patches concerning file names,
but you don't need "make release". Gotta catch Falk to talk them
through and commit them.
