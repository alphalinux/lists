On Wed 07 Mar 2001, mel kravitz wrote:

> i have a fully operational
> system, however i don't seem to have lpd in /sbin or /usr/sbin?
> Something i did?
> I assume the print daemon is installed as part of the 'base.tgz' binary?

No, the base system is just that, very basic.
If you want a printing daemon, you will have to install the lpr
or lprng package. If you have apt setup correctly,
"apt-get install lpr" will be enough. Setting up apt correctly is
basically putting the correct things in /etc/apt/sources.list.
I'm assuming you have net connectivity on your system; then
you should have a line such as the following:

deb http://http.us.debian.org/debian stable main contrib non-free

After that, run "apt-get update" to update your system's info about
what's in the archive.
