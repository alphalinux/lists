Sebastian:

I finally got everything working , the patch you suggested did the trick! Thanks
hugely!!

The bad news is that when I tried to rebuild kernel-2.2.17 or 2.2.18 with
CONFIG_ALPHA_AVANTI
set (which also sets CONFIG_ALPHA_EV4) in the config file and the patched traps.c file,
I had other
problems. The ncr53c8xx scsi driver had timeout problems and I could not boot the
machine with the modified
kernel.. So, I set CONFIG_ALPHA_GENERIC and manually removed "ifdef CONFIG_ALPHA_EV4"

statement from the patched traps.c. I realize that this is not the best way to handle
the problem, but
it worked.

Thanks to you and all the others for the help.
