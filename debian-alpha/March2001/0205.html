This file seems to be missing in debian. Just do the following:

cd /usr/X11R6/lib/
ld -shared -soname libXxf86dga.so -o libXxf86dga.so --whole-archive \
libXxf86dga.a
and

ldconfig -v

do the same with all missing libs.
