Hi. I installed potato rev2 on Ruffian, then downloaded rpm packages from
a Compaq ftp site and installed Compaq C/C++ Compilers as follows:
alien -c -k *.rpm
dpkg -i libots_2.2.7-2_alpha.deb
dpkg -i cpml-ev5_5.1.0-3_alpha.deb
dpkg -i ladebug_4.0.62-16_alpha.deb
dpkg -i ccc_6.2.9.506-1_alpha.deb
dpkg -i cxx_6.3.9.7-1_alpha.deb

I successfully compiled a C program with ccc, but I can't compile a C++
program with cxx.

cxx -o test test.C -lcpml

After my answering the question about License (no commercial license),
it returned
/usr/bin/cxx: Permission denied .

ls -al /usr/bin/cxx
