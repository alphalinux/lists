Hi,

I wonder, if anybody ever did some monitoring on Alpha's using the lm-sensors
package.

On my UP2000, there is an i2c Bus, apparently using a PCF8584 controller with a LM75
sensor, capable of doing thermal monitoring ( I hope the four fans are being monitored too ;-)

I now have the i2c bus up and running, using kernel 2.4.2, but i run into some strange
compile-problems wrt the lm-sensors package. Before asking specitfic questions on the
compilation of lm-sensors, i want to dig a bit further.
But since there is no alpha-deb of lm-sensor, I wonder if anybody ever managed to get
the whole thing going on Alpha.

I'd be happy to hear about it. Since the newer Alphas need *a lot* of reliable cooling and are
expensive enough ( gasp, the hole in my pocket is still big after 1.5 years ;-), I would assume,
that this sort of thing is of interest to other linux/alpha users as well....
