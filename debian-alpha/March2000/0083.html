"Waite, Michael" <Michael.Waite@compaq.com> writes:

> And the "console" install works?
> This system doesnot have a videocard as it only has one pci slot used by the

You have to put 'console=ttyS0' in the boot flags in order to get
Linux to use the serial console.

Unfortunately the installer that's on the FTP site right now won't
properly set up the base system so that you can log in via the serial
console after installing the base system. This has been fixed and
will go in the next revision of the install disks (later this week or
early next week, probably). If you like I can build a new tftpboot.img
with the fixes in it.
