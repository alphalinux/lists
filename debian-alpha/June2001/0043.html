Rob Byrnes (robbyrnes@ozemail.com.au) wrote:

> > ifconfig eth0 <address> <netmask> hw ether media 10baseT up
>
> and everything works, but I still have to add a default route by hand.

Edit /etc/network/interfaces so it includes something similar to:
---cut-here---
auto eth0
iface eth1 inet static
address 192.168.1.1
netmask 255.255.255.0
broadcast 192.168.1.255
gateway 192.168.1.1
---cut-here---

If your ethernet is loaded as a module then edit /etc/modutils/aliases
so it has a line similar to:
---cut-here---
alias eth0 tulip options=4
---cut-here---

Change as needed.

If your ethernet is NOT a module, then you will have to pass arguments
to the kernel at boot time. The commands depend on which firmware you
have (SRM or ARCS/ARC/AlphaBIOS+MILO).

That should do it..
