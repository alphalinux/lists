> Also, is there any news about 2.2r4? How about
installing SuSE on an
> alpha? Is it worth while or are there hidden pitfalls?

I'm using 7.1RC since it was available on ftp and I've
found it to be the most stable *nix for Alpha. (I had more
crashes when I used Tru64 5.1 as a workstation). Much more
stable than RH7.0, FreeBSD 4.3 and Debian/Woody (6 months
ago)!
Remember to use XFree86 v3.3.6 and Kernel 2.2.19! 4.0.x and
2.4.x are very unstable on alpha. I haven't tested XF 4.1.x
yet though...

I really want to go back to using Debian though, once it
gets a bit stable again (maybe it is already?)
