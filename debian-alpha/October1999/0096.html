On 13 Oct 1999, Ron Farrer wrote:

> Conflicts: libgnome0 but 0.30.1-5 is installed

Check to see if libgnome 0.30.1-5 is installed. If so, remove it like so:
        dpkg --purge --force-depends libgnome0

After that, try:
        apt-get -f install

That should handle everything ok (fingers crossed).

It faintly looked like a conflicting package was installed and it was
having a problem with that first and foremost. It didn't trace
dependencies all the way through, though, so it never saw the libgnome0
problem while installing the other stuff.
