On Wed 03 Nov 1999, Ron Farrer wrote:
>
> I can use rdate to sync the kernel time to the correct, but how do I update the hardware time to the same? I've looked into ntd but it's not what I'm looking for. Basically I've already used rdate to get the correct time and now I want to update the hwclock.

Basically it's "hwclock --systohc". Add --utc if you run the hardware
time on UTC (as you should :-).

Note that debian does this automatically when you shutdown...
