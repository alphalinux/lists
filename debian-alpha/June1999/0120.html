>
> waiting for a reaction to my aplication. Should I send it again?

If you haven't had *any* response, you can send a gentle message asking
whether your application was received OK, and if a time for processing
can be indicated. My research indicates that you have to treat the
new-maintainer people gently or you'll never get through :-) Reminder
messages, if polite, are accepted cheerfully; if harsh ("why is this
taking so long!!" - type messages for example) will cause an indefinite
delay :-).
