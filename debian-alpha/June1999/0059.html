Ettore Aldrovandi <ettore@ubik.fm.sissa.it> writes:

> Third: how much is memory speed important? That is, suppose the
> old memory is 10ns and the new one is slightly faster, like 8 or
> 7. Is this important?
>
>
If you have different speeds for your memory, the bios will probably
take one speed or the other. Just order the rams so that the slower
one is chosen and you are fine. If you can`t find out what speed is
used, try if it crashes and then change the order of the rams.
