Loic,

I guess this is either a boot-floppies or base-files problem, but as Jeff
says, the SCSI cdrom devices are missing in the default install (I noted this
then forgot about it). This is a bug and should be fixed.

Jeff, you need to do:

# (cd /dev; ./MAKEDEV scd0)

which should give you a /dev/scd0 which is the same as /dev/sr0.
