I had the same problem but on DEC2000 Model 500.
If you have installed linux simply copy linux from
http://http.us.debian.org/debian/dists/potato/main/disks-alpha/current/image
s-1.44/root.bin
and kernel from
http://http.us.debian.org/debian/dists/potato/main/disks-alpha/current/jense
n/linux
to your /boot partition
Then from aboot you can load as any other kernel
linux root=/dev/ram initrd=root.bin

and it should work. I did not check whole instalation, but at least the
beginning worked
