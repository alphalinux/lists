Try the following command to see if it will bring up the kernel for you:

boot fd0:linux root=/dev/fd0 rw ramdisk_size=5120 load_ramdisk=1
prompt_ramdisk=1

all on one line...

version on my PC164 (which has sense been upgraded to Potato)...
