On Tue 06 Feb 2001, Andrei A. Dergatchev wrote:

> Can't locate Exporter/Heavy.pm in @INC (@INC contains: /usr/lib/perl5 /usr/lib/p
> erl5/5.6/alpha-linux /usr/lib/perl5/5.6 /usr/local/lib/site_perl/alpha-linux /us
> r/local/lib/site_perl /usr/lib/perl5/5.005/alpha-linux /usr/lib/perl5/5.005 .) a
> t /usr/lib/perl5/5.6/Exporter.pm line 15.
> BEGIN failed--compilation aborted at /usr/bin/dpkg-source line 17.
> Unpack command 'dpkg-source -x glibc_2.2.1-1.dsc' failed.

Looks like you've been selectively updating packages from unstable,
and not doing a general "apt-get upgrade". Exporter/Heavy.pm for
example is in the current perl5.6 package, whereas Exporter.pm is
in perl5.6-base.

dpkg-dev should perhaps have versioned dependencies...

Install perl5.6, although if you have more stuff that isn't
upgraded yet, you may run into lots more of such troubles.

Tip: download Contents-alpha.gz from your favourite debian mirror,
and if you see an error such as the above, you can do:

zgrep Exporter/Heavy.pm Contents-alpha.gz

to see which package contains it.
