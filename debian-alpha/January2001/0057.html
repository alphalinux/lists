> kabarga:/home/gnana# apt-get install netscape-base-476

> never uploaded, has been obsoleted or is not available with the contents
> of sources.list
> E: Package netscape-base-476 has no installation candidate
> kabarga:/home/gnana#

I assume that you are trying to install on an Alpha? If so Netscape is
only available for x86 systems. You can grab the Tru64 UNIX version of
Netscape in RPM from Compaq (which include the libs needed to run it)
and use alien to convert it to a deb. Otherwise you could try a recent
mozilla (NOT the one in Debian for Alphas) and try building it
yourself. Hopefully this will be fixed sometimes soon. If you need
help, let me know.
