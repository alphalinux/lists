I recommend going straight to potato (2.2, almost stable :-), whose boot
floppies can be found at:

http://debian.midco.net/debian/dists/potato/main/disks-alpha/current/

The generic kernel worked for me. The instructions in the README worked fine.
I did an NFS install, which just required three floppies: rescue, root,
drivers-1; http should work fine too.
