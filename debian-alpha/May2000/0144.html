Yes, the rev of glibc that debian is using doesn't have the patch needed
for
system type Nautilus (I can probably find it i somewhere), so you need to
make the symbolic link for alpha_systype, from the / dir type:
ln -s Tsunami /etc/alpha_systype

(just like it says in the UP1000 user manual), now there is no file named
Tsunami, and that's OK.
