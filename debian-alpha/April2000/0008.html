Roland Rosenfeld wrote:

> Did anybody try out to burn a CD using cdrecord on an alpha?

cdrecord's default shared memory size is 4MB, which is larger than the
Linux kernel supports on the Alpha. You can edit the kernel source and
recompile, or pass cdrecord the option "mem=3.5M" (I think that's the
syntax).
