Cormac McGuinness <cormac@BUPHY.BU.EDU> writes:

> I recently got a DEC Multia and would like to install debian on it.
> Is there any way I can install (or at least prepare) a hard disk
> with all the necessary alpha binaries while the hard disk is sitting
> connected to an i386 box ?

You can prepare the partitions, and put base2_1.tgz and the
drv1440.bin on a partition with some space. This saves you from having
to put these onto floppies. You then need only root1440.bin and
resc1440.bin on floppies.
