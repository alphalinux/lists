>
> With 2.2.16 kernel system normally shutdown,when i type "shutdown -h" or
> "shutdown -r", but cannot out to SRM. It print:

Is this the stock installation kernel on the boot disks? If so, it
shouldn't be 2.2.16, IIRC (should be 2.2.17pre-something). Give the one
on the rescue disk a try and see if you get the same problem.

> With 2.4.0 kernel system out to SRM, but cannot reboot.

This isn't that surprising. Several of my machines that I'm running
2.4.0-test kernels on die while trying to reboot (sparc, mips,
alpha). The 2.4.0-test kernels are still far from being reliable on archs
other than i386, so caveat emptor.
