The generic kernels DO NOT work with jensens (I don't think they ever
did). Therefore you need a special jensen kernel aboot combo (for 6.2 the
kernel HAS to include support for modules, otherwise the installer just
generates aborts due to signal 11...) I have not been successful with 6.2
myself, that is I coulsd install it with the options text and expert, but
can only boot it in single user mode. As I am having suse6.3 running on
that machine I am not really working hard to make redhat run.
