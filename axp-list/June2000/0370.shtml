On Thu, Jun 29, 2000 at 08:45:35AM +0300, Erdal MUTLU wrote:
> Hello!
>
> I have installed RedHat 6.1.92 on Alpha Personal Workstation 433au. But I
> cannot run X. I looked at the graphivics adapter it is Cirrus Logic GL GD
> 5429-86QC-C Ramdac BE486AKHJ170. The output of /proc/pci file is :
>
> Bus 0, device 11, function 0:
> VGA compatible controller: DEC TGA2 (rev 34).
> Medium devsel. Fast back-to-back capable. IRQ 28. Master
> Capable. Latency=255. Min Gnt=8.Max Lat=64.
> Non-prefetchable 32 bit memory at 0xa000000 [0xa000000].

This is a TGA2, which it NOT supported currently. There is a Cirrus dumb
VGA to provide console text mode; you may be able to run the Mono server
on that, but there's no server that supports the TGA2 accelerator.
