> Does anyone know/have good information on using DLT auto loader on RH or any
> other Alpha/Linux?
>
> I would like to use NetVault type of application for Linux, if available.
>

I know that there is a linux something for Legato.
It's probably only a bear bones client, not a server (and therfore doesn't
handle the tapes), but I don't know.

It was on Legato's ftp site. ftp.legato.com

Legato has a "single server" mode whereby the host with the drive can
backup itself (not just any "single" host). This is bundled with Tru64
Unix, and is the same kit as is used by the clients, you just skip
installing the server components when installing the client.

[Legato was Digital's NSR (Network Save and Restore) product. Digital did
the 64 port and sold the code back to Legato. So it definately runs on the
Alpha under Tru64.]
