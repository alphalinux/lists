On Mon, Jun 26, 2000 at 03:16:17PM -0600, Gordon Schumacher wrote:
>
> I'm trying to get RH6.2 installed on an AlphaStation 200. I can use the
> SRM to boot off of the CD-ROM, and it gets as far as starting the X client
> - and then it stops. I've checked the consoles and there are no obvious
> error messages.
>
> If someone can either tell me what's wrong or what I should look for I'd
> appreciate it! Thanks all...

It always helps to tell a little bit about the peripherals that are in
the machine when you have a problem, but there's a couple of things to
remember:

1. you can always bypass the GUI install and use the old style by
   adding the word "text" to the command arguments, ie

        boot dva0 -fi kernels/generic.gz -flags "root=/dev/scd0 text"

2. if your machine has a TGA2 in it, it won't run X; if it has a
   Trio64 in it, you may need to add:

        Option "no_pci_disconnect"

   to your XF86Config file Device section.

In any case, use tip #1 should allow you to get a complete install,
and then you can play around getting X configured.

Good luck.
