        You can go to the API website and get a list
        of resellers in your geography.

        The UP1000 and UP1100 are both based on the
        same core logic chipset, the AMD 751. The
        UP1100 comes in one flavour, 600MHz 2MB cache.
        You can get a UP1000 with 600/2 or 700/4. Both
        the UP1000 and UP1100 use a 64-bit wide memory
        bus.

        The UP2000 comes in a number of flavours. Check
        our website for the latest info. The bus runs
        at 83MHz or 75, depending on the speed of the
        processor. (667=83 750=75) However, its width
        is much greater at 256bits so the memory bandwidth
        is in the 2+Gb/s range.

        Fastest is a function of the application. What
        type of app are you looking at?

        These motherboards all use EV67 processors. See
        our web site for benchmarks and exact specifications.

        http://www.alpha-processor.com
