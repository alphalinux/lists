Lukas Pustina wrote:
>
> > Also, I think there's an environment variable for it,
> > CDR_FIFOSIZE.
> Yeah, that's right!
> I start XCD-Roast with the following "script"
>
> export CDR_FIFOSIZE=4000k
> xcdroast

That is one way to do it. But for quite some time 'cdrecord', which
does the writing, has described in a manual page a file 'cdrecord.conf'.
It is enough to put there a line

CDR_FIFOSIZE=4000k

and this will work with any Alpha kernel as long as your CD writer is
supported.

Specifying fifo size elsewhere will override this setting.
