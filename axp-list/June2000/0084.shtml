Robert Williams wrote:
>
> I've tried to build the apache-ssl-1_3_4-1_31-1_src.rpm

It is way too old anyway. It does not include, in particular,
various important security fixes.

> I'm using the latest open-ssl libraries.

Version?

Use instead apache-mod_ssl from http://www.modssl.org/
You will find there source rpm for the latest released Apache
version and they compile on Alpha without any problems. If you
have open-ssl installed then 'rpm --rebuild ....' should be all
what is needed unless you plan to make some configuration changes.
