Champigny, Michael wrote:
>
> Not as much memory use it seems, and faster
> interactivity in X, probably due to the bloated window manager
> (Enlightement) in GNOME. Use sawmill (now sawfish) with RH6.2 instead
> and you'll notice a big improvement in speed.

You may noticed that in my earlier posting in this thread I advised
against using E with Gnome. This is one of reasons. While KDE is indeed
tied up with its kwm window manager (and that is one of my beefs with
it) there is no similar situation with Gnome even if one of window
managers comes normally as a default in distributions. Gnome is really
a framework and not a desktop. AFAIK E is not a default window
manager in the latest Gnome releases (at least not with Helix sets
which are using sawfish).
