        Much of this is in the archives. I used Google and
        got alot of stuff.

        In a nutshell, from AlphaBIOS, go to CMOS setup and
        change the console to SRM. Then you can get the
        latest SRM for the 3300/AS800 from Compaq. (May as
        well be running the latest)

        Then you can boot the Red Hat CD directly.

        If your 3300 is using an IDE CDROM and a SCSI disk,
        you can boot using this command:

>>>b dqa0 -file "kernels/generic.gz" -flag "root=/dev/hda text"

        That'll start the install RH 6.2 will take care of the
        BSD disk labelling that SRM understands.

        Enjoy and welcome to Alpha,

                                                                mike
