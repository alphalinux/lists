Alan Keane wrote:
>
> Is there any way to increase the maximum stack size on Alpha Linux without
> recompiling the kernel?

Yes. You have to edit /etc/security/limits.conf and this has
__nothing__ to do with Alpha but with how login program, and PAM
libraries, work. At least this worked the last time I tried.

Reading docs before you start with editor is highly recommended even if
a header comment in the aforementioned file gives quite a bit of
information.
