On Thu, 15 Jun 2000, Konrad Podloucky wrote:

> Hi guys!
>
> Wanted to check before I start downloading:
> Is somebody successfully using Helix-Gnome on an Alpha?
> Any problems or gotchas?

Works a treat. I have it working with all packages built.
(Although I seem to remember gnome-utils was a bit touchy lately).

1) There is a problem with cuttign and pasting large ( >4K) things
  - this is a 64 bitism - a patch has gone in for it.

2) Figuring out the build order is nearly impossible.

3) You may find that you don't get any plugins in gimp if you build from
SRPM. The trick is to build it and do a make install from the build
directory to force it to actually install the plugins properly.
(Something like that?)

Helix gnome is nice.
