I *think* that the 3300 uses AlphaBIOS. If so, you can switch to SRM
as follows:

1. When the console threatens to boot NT, press F2 for "setup"
2. From the menu, select "CMOS setup"
3. Press F6 for "advanced"
4. TAB to the "console selection" item
5. Use UP or DOWN arrows to select UNIX SRM
