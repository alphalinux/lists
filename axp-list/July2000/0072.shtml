Well, either you use the ramdisk floppy and and choose the NFS install, or
I think it something like this:

boot dva0 -fi vmlinux.gz -fl "root=/dev/nfs nfsroot=IPaddress:/mnt/cdrom"

If you want to take this further, and SRM reckognizes your NIC card, you
can build a bootp-able kernel and not use any floppies (you'll also need
to setup the bootp/DHCP server for this).

boot ewa0 -fi vmlinux.bootp -fl "root=/dev/nfs nfsroot=IPaddress:/mnt/cdrom"

This is actually what I use at work to install multiple machines (well,
that and kickstart).

Seeing as you're having the boot parameter problem, either use -i and
enter it all or set the env variables.
