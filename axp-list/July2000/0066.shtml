It turns out the reason the install was freezing was that X-windows was, ahh,
intermitantly screwing with the cdrom, for want of a better description. That
was compounded by the fact that the SRM conveniently was dropping all but the
first parameter to the -flag switch, so that (accursed to me by now) gui kept
loading. I used the -fl i (for interactive I suppose) to get the expert text
parameters in and RH 6.2 installed with no problems using the "old" text-based
installation. Thanks again for your suggestion Mike.
