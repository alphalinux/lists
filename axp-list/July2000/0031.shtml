dirt cheap way to fix it:

-login to X windows
-close all the windows you don't want to apear
-click on logout in the menu
-when the box comes up prompting you what you would like to do (the one
with are you sure you want to log out) click on save configuration, then
click yes (or ok.... slips my mind at the moment which one it is)
-log back in
-there should be none of those pesky default windows there any more
