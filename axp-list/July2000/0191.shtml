On Fri, 14 Jul 2000, Rich Payne wrote:

>
> Martin,
>
> A 3c509b? or a 3c905b?
>
> I've done a hefty amount of testing with the latter and it works fine in
> the UP1000. As for the 509b, I don't know what that is.

It sounds like the 509b, which is the pnp ISA card. I've never used one
in an Alpha before, but I've used quite a few in x86 Linux systems. I've
never had problems on x86 as long as the card wasn't in pnp mode, and it
sounds like Martin has already verified that.
