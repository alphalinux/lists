On Wed, Jul 05, 2000 at 09:53:49PM -0500, Wynne Anderson Hexamer wrote:
>
> I've gotten further along in my attempt to install Red Hat on my AS
> 800 with 6.2 than 6.1. However, though I've managed to boot the
> x-windows gui install, it just freezes (help button and mouse still
> move but CD quits) about 1/6 of the way into the install of the RPM
> packages, usually on termcap. It isn't BSD disk labels, as I've
> tried both ways of partitioning, nor the CD drive which I swapped
> out, nor the ISO which I downloaded twice. I would greatly
> appreciate any suggestions from someone who has solved a similar
> problem or anyone with an idea of what I might try next.

I'd say try bypassing the GUI install by adding "text" to the flags, ie

>>> boot dka400 -file kernels/generic.gz -fl "root=/dev/scd0 text"
