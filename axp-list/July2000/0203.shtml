On Sun, 16 Jul 2000, W Bauske wrote:

> /proc/mdstat:
>
> [root@wsb46 /root]# cat /proc/mdstat
> Personalities : [1 linear] [2 raid0]
> read_ahead not set
> md0 : inactive
> md1 : inactive
> md2 : inactive
> md3 : inactive

This seems like you are using un-patched kernel ('old-style' raid) with
new raidtools ('new-style' raid).

You should get a new RAID patch (patches for 2.2.14-2.2.16 are avaiable at
http://people.redhat.com/mingo/raid-patches, along with most recent
raidtools), patch the kernel and try again.

However, what is 'raid-level -1' supposed to mean? AFAIK, raid-level can
be 0 for linear, 1 for mirroring and 5 for RAID5. Since you have four
disks, I expect RAID5 is what you want and you should carefully consider
option 'chunk-side' as it is supposed to be imoprtant for performance.

   D.
