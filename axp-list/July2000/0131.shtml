On Wed, 12 Jul 2000, J Bacher wrote:

>
> Does anyone have the URL for a listing of the SRM console variables,
> options and descriptions? I am using 5.5 and 5.7.

There's an SRM reference manual available
http://ftp.digital.com/pub/Digital/info/semiconductor/literature/archives.html
but it's a little old. Other than that you'd have to refer to the docs for
your particular machine.
 
> I need to change ether duplex, boot device, and auto boot mode in addition
> to some other options. Thanks.

boot device is: set bootdef_dev (new device name, use show dev to get the
names)
auto boot is: set auto_boot (halt or boot)
ether duplex: I think that's 'set ewa0_mode'
(or eia0_mode for Intel cards).
