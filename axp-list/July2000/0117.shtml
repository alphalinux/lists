Malcolm Caldwell (malcolm@it.ntu.edu.au) wrote:

> Is there a HOWTO on how to move an alpha from ARC to SRM? I would like to do
> this on my avanti machine.
>
> Is this possible/easy?
>
> Perhaps someone should write one. From what I understand, now that ARC is no
> longer supported, and SRM has more features, this may be a common thing to do.
>
> There are two sides to this - 1) actually moving a particular machine type
> over, and 2) the linux spcificic stuff.
>
> I think 1) is probably already out there. I don't know about 2).

They both get asked quite a bit. The SRM HOWTO on AlphaLinux.org answers
most of 2, IIRC. As for 1, how many people want it? It's pretty easy,
check the list archives.

Contact me *off* list if anyone would like to see a new HOWTO with
instructions for as many systems as possible. If there is enough
interest and no one has any objections, then I'll go ahead and do it.
