SteveCraft (scraft@nothinbut.net) wrote:

> One day my Multia (no monitor, kybd, mouse, it's "headless") stopped
> responding to pings, so I hooked up a monitor and the screen wasn't
> showing any video. I powercycled it and before the ARC menu came up it
> complained the keyboard had failed or somehting. So then I shut it off,
> added amy keyboard, turned it on, then it started to boot and then froze
> up again.
>
> Am I the victim of a heat problem? This baby has been sitting for a year
> (in its vertical stand) without any problem, and now this. Help?

It kinda sounds like it, but you may also want to check the RAM. Make
sure it is clean. I've seen RAM that's been in a system for serveral
months or more without a single problem, then all of a sudden lots of
problems. Thanks to a suggestion in the list archives I pulled the RAM
out and cleaned it, then put them back in and that fixed it.

Other things to check would be both the CPU fan and the PS fan. If that
doesn't help, I would let it sit in ARC and see how long it will stay
there before it goes (rule out software problem).
