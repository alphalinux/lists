> One other little known fact -- Alpha Linux and Tru64 are binary
> compatible.

Details are in the FAQs, even.

We cross-compiled everything for the FSL bid, because the Compaq compilers
hadn't been ported to Linux yet. But do note that if you don't have a Tru64
license for all your machines, you shouldn't be moving static linked
binaries around. But you can install the Netscape RPM, which includes some
of the Tru64 shared libraries...
