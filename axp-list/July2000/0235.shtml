Hi!
Pls check if you have installed the gnome package or you can boot the system with the RedHat CD and
upgrade the packages. Make sure that you upgrade these packages:-

System Environment Libraries - check the packages by pressing F1 and choose what's relevant to GNOME

Development Library -check the packages by pressing F1 and choose what's relevant to GNOME

Application/ Multimedia - check the packages by pressing F1 and choose what's relevant to GNOME

Application/System - check the packages by pressing F1 and choose what's relevant to GNOME

continue checking the packages and when you are done the system will tell you about the
dependencies, install the dependencies to be on the safe side.

I hope this will work for you. All the best.
