OK, the 509B is a decent card....we've used a lot of them and they seem to
run very well. However......it does not show up in SRM, so you can't net
boot from it. If you're running AlphaBIOS I would suggest you get the beta
SRM from API's website:
http://www.alpha-processor.com

follow the links for support, ftp downloads, customer support,
up1000. Follow the docs in the package. The reason being that AlphaBIOS is
dead, you have the last version that will ever be produced and there's no
updates to AlphaBIOS or APB at this point!

Now, if you go with either the Tulip or the Intel series you'll be able
to use the card in SRM and boot via the network...which I think is really
neat feature.
