> Please tell us if it looks like a child drawing with Crayola or whether
> the font server and renderers have it looking as good as a WINDOZE
> screen?

Actually it looks quite good. At the moment I have helix gnome using a
theme with the customized font being MS verdana. I can scale the font
8-72 pts and gnome handles them quite nicely. I have netscape using the
TTF fonts though it will only display them at 12 pt. I got around that
by adding the following lines after some testing to my .Xdefaults

Netscape*documentFonts.sizeIncrement: 10
Netscape*documentFonts.xResolution*iso-8859-1: 85
Netscape*documentFonts.yResolution*iso-8859-1: 85

I've noticed though that in netscape wont display the font in bold or
italic, it just looks normal.

I believe the xtt backend works in XF-4.0 as well. Note you need the
mkttfdir tool from the perlftlib to generate the fonts.dir files.
