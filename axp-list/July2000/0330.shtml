>The licensing problems are the same as with Tru64 products for education--
>DEC/Compaq did not write the basic code.
>
>One other little known fact -- Alpha Linux and Tru64 are binary compatible.
>
>I haven't had an opportunity to try it, but I've been told that any
>statically-linked Tru64 binary will run under Alpha Linux, and vice-versa.
>Run-time libraries are (obviously) likely not the same, hence the binaries
>need to be statically linked.
>
>I don't know if this is version dependent or not, but if it is, I would
>assume that Tru64 5.x will work, but 4.x won't.
