Here's a copy of a message M. Jaegermann sent last September:

------------- <snip>

From: Michal Jaegermann <michal@ellpspace.math.ualberta.ca>
To: axp-list@redhat.com
Subject: Re: DUnix Binaries on Alpha Linux

Stuart Midgley wrote:
>
> When I run Mathematica 3.0 it runs fine, except for these error
> messages:
>
> Warning: Cannot convert string
> "-*-Menu-Medium-R-Normal--*-120-*-*-P-*-ISO8859-1" to type FontStruct

This one is merely annoying and it will show up when you try to
run other programs from DU which use X-windows. When you will
get fed up sufficiently enough you can kill that message adding
to fonts.dir in some directory where you have Type 1 fonts a string
like that:

n019003l.pfb -urw-menu-medium-r-normal--0-0-0-0-p-0-iso8859-1

Use some other font if you prefer and you have to have a file like
'n019003l.pfb' obviously enough. Do not forget to bump up a font count
at the top of 'fonts.dir' and also copy results to 'fonts.scale'.

Other errors - this is a totally different story. :-)

  Michal
