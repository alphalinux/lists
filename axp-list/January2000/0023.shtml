i'm rather new to alpha-linux, since i got my Miata box about a wee ago, and
i was quite frustrated with the lack of native netscape which is taken for
granted on i386 boxes.

i didnt want to try the emulator so i went for OSF/1 netscape with OSF/1
libraries since i have acces to 5 OSF/1 4.x boxes.

i just wish to add some info that extends the document found on
www.alphalinux.org about installing netscape 3.01 gold.

i installled communicator 4.7 and aditional libraries were required in order
to run.
the list of the libraries i have right now in /usr/shlib is :

libICE.so
libSM.so
libX11.so
libXext.so
libXm.so
libXmu.so
libXt.so
libc.so
libcxx.so
libdnet_stub.so
libexc.so
libm.so
libots.so

all libraries have been taken from OSF/1 4.0 box

loader is copied to the folder specified in the document.

to add ot the original document, Java and Java Script support is working
perfectly, and i still have to check if the mailer works with no additional
modifications.

ps. i'm building 2.2.14 kernel right now and i will report its performance
as soon as i boot it :))
