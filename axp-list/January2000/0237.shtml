yep.. should work without problems.

Also does anyone know what the LVD means?

Low Voltage Differential.

Normal SCSI uses (plain??) signalling, ie one wire, and logic value is
determined by the voltage level on the line. 3V = binary 1, 0V= binary 0.
However that type of signalling is very sensitive to distortion.

Enter LVD, this uses 2 wires (+,-). Logic level is not dependent on an
absolute voltage, rather it depends on the polarity with respect to the
other. Ie forward polarity might be 1, reverse polarity would be 0.

This is a far more robust method of signalling, and hence clock speed can be
much higher and/or length of bus can be longer.
