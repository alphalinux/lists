Hi,

Does anyone know if the Castlewood internal SCSI drives work on a UX
box ?

I'm running RH5.2 on my UX2 and my Iomega Jaz 2GB drive just quit on
me after working just fine for about 11 months. I can no longer mount
the disk and attempting to run something like:
        mkfs -t ext2 -c -v /dev/sdb1
results in numerous SCSI errors in /var/log/messages like this:

-----
Jan 12 19:26:58 xyz kernel: scsi0: MEDIUM ERROR on channel 0, id 6, lun 0, CDB: Read (6) 1e c0 52 02 00
Jan 12 19:26:58 xyz kernel: Current error sd08:11: sense key Medium Error
Jan 12 19:26:58 xyz kernel: Additional sense indicates Unrecovered read error
Jan 12 19:26:58 xyz kernel: scsidisk I/O error: dev 08:11, sector 2015252, absolute sector 2015314
Jan 12 19:26:58 xyz kernel: EXT2-fs error (device 08:11): read_block_bitmap: Cannot read block bitmap - block_group = 123, block_bitmap = 1007626
-----

I tried a new cassette with no better luck, so it does seem to be a
drive failure. I'm wondering if anyone else out there has had similar
problems with the JAZ 2GB (I know they've had numerous quality problems
with their 1GB drives).

Are Castelwood drives any better ?
