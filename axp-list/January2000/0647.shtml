Using the SRM console boot from the CD..(never tried it from milo)

boot dka0 -fi /kernels/generic.gz -fl "root=/dev/hda"

The Install will run from the CD (partitions will be dos style)
you can switch back to ALphabios after the install. ("set os_type NT" at the
SRM console)
