On Wed, 29 Dec 1999, nm wrote:

> I recently acquired an AlphaStation 200 4/233. The system
> has an ATI Mach64 video card in it. I connected a PS/2 mouse
> and keyboard to it and powered it on. It just sits at a blue
> screen, the keyboard's numlock responds as expected as does
> caps lock. I suspect that the firmware is set to talk via the
> serial port. I do not have the right serial cable to connect
> this system to a terminal or pc, so I am wondering if there is
> a way to force the system to graphical mode. Anyone have similar
> experiences?

Press the enter key a few times, you should get the >> prompt (CTRL-C can
also be your friend here). Once you get the prompt type is:
>> set console graphics

and restart the machine. You should be OK after that.
