Sharing mailboxes between two machines is uaually considered a bad idea.
Solaris gripes if the mailbox is nfs-mounted. How about declaring one
machine the actual mailhost and doing a .forward for all the accounts
on the other one? That lets the mail flow to one box, which you could
access via nfs from the other.
