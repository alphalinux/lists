>>> Michael Werner said:
>
> Has anyone tried to use the CTTZ instructions with GCC?
> ECGS 1.1.2 didn't like this instruction in an xtended asm inline macro.
> I passed the ev6 flags to gcc and for assembly stage.

I think Richard Henderson had GCC generating these instructions a while ago,
but unfortunately it wasn't until the EV67s appeared (667MHz is the most
common) that there was actually silicon devoted to the counting
instructions.

EV6 has the FP <-> int conversion instructions but *not* the counting ones.
There is an error in the older docs that state that the conversion and
counting would be in EV6, as only the conversions ones could be done in the
time that was available.
