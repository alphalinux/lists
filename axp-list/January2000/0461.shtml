Am Son, 23 Jan 2000 schrieb Peter Petrakis:

Hi,

I managed to compile XEmacs 21.1.8 on my 164LX with ccc ( and a little help of
gcc :-)
The problem was that the dump form temacs to xemacs didn't work. Like I read in
the Debian-archieves this is because ccc miscompile _ONE_ single file. This
file was "unexelf.c" on my system.
My work-a-round was to compile this single file with gcc. This worked.

My subjectiv opinion is, that XEmacs speeds up around 25%, depending what you
are doing.
