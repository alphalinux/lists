Wade Blackwell wrote:
>
> Good morning linux fans,
> I installed my 6.0 system with the stock ide components and now I
> want to add an aha 1542cf to my system with a jazz drive hanging off
> of it. This may sound like an incredibly stupid question but how do I
> add this hardware in linux? Do I have to rebuild the kernel or is
> there another way? Thanks for humoring me, I have looked high and low
> to no avail. Thanks!
>

Not sure if you're using an Alpha box from your question.
If so, I think someone a while back said those ISA cards
won't run on an Alpha based Linux box.

If you're using an Intel/x86 box, it should work but this
is the wrong mailing list to ask on.

But as to what to do, assuming an Intel box, install it
and boot and see if it shows up in the boot messages.
The RH6.0 kernel is modular so you may have to load a
driver for it. Try "man modprobe" and "man insmod".
The driver is called aha1542.o.
