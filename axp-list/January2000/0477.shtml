larrys@lexis-nexis.com said:
> I doubt it will ever get to the point where you stuff in a cd and type
> "setup" :-)

Just FYI, NT is pretty broken in this regard, too. If you don't partition
it just right, you're screwed. It won't boot. Sometimes, it won't even
install. At least in the Linux world there are competing distributions
and you have a choice of installer.

(In order to set up a dual boot Linux--NT system, I boot the Linux CDROM,
used the fdisk tool to partition the disk, boot the NT disk and use the
partitions so made, then install Linux afterwards to get dual boot. You
cannot install NT second and get a dual boot, and the NT disk partitioner
is pretty bad. And this is Intel. NT/alpha is long since gone from my machine.)
