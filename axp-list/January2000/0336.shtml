> My question is whether this chipset supports ATA-2 DMA transfers. My current
> understanding is that it supports ATA-2 DMA at 33MHz (not the newer 66MHz), and
> that this functionality is supported by the IDE kernel driver on the Alpha.
> Is this correct? Also, do I have to jump through any hoops to get this
> working (e.g. hdparm) or will the driver configure itself?

it supports dma 2 mword transfer. if you've got 2.3.x kernel you only
have to enable the support for generic pci ide and then the support
for Cypress 82C693U, then rebuild and reboot. now use hdparm -d1 to
enable dma mode for your disk. you may also have to use hdparm -X34
to set your disk to dma-2 mode.

if you're running a 2.1.x kernel you can get the ide patches from
ftp://ftp.kernel.org/pub/linux/kernel/people/hedrick/
or one of it's mirrors.
apply the patch, the rest is the same as with 2.3.x.
