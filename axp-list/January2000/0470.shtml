I started out w/ a RH6.0 AS200 (SRM) and attempted to load up
SuSE6.3. That was a bad idea. After I found a kernel on the cd
that would support the scsi host, I built the box. I couldn't
load a kernel from anything but a cd or a dos-ish floppy. That was
after I'd thrown in the towel and flashed ARC on it. I ended up
reloading from scratch with RH6.1 using the same kernel parms you
mentioned...something like 'expert text root=/dev/fd0' or thereabouts.
(thanx to Praveen for getting me that far). The Alpha's are
absolutely killer boxes, and the support for them here is excellent.
I'm glad they're not being ignored by the distro folks,
but it would sure be nice to see a bit more support from them.
Like Michal said, a newbie wouldn't stand a snowball's chance in
Hell getting an axp up with the current distros. Sorry about the
rant, but it took way too long to get that box back up.
-ls-
