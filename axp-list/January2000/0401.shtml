> "Jakma, Paul" wrote:
> >
> > > Qlogic appear to be unable to drive CD-RW drives, a Yamaha 4416
> > > in my case. Even with newer kernels. They run disks fine.
> > >
> >
> > The problem is most likely with the 4416, which has had some amazing
> > firmware bugs. Upgrade to the latest firmware and it will
> probably work.
> >
>
> It works fine on a Tekram DC390F on the same system.
>

Like I said, the Yamaha has some really weird firmware bugs. Eg certain
firmware rev's will lockup on wide or ultra negotiation. Maybe your DC390F
is set to not negotiate u/w with the yamaha, in which case it will work.

You need to upgrade the 4416 to the latest firmware, which iirc is revision
G something. I doubt it's the SCSI controller, as both the Qlogic and Tekram
are Symbios/NCR based.

I had the exact same problem with my 4416. To get it to work i had to power
it up *after* booting linux, then tell the linux symbios driver not to
attempt u/w negotiation for it before doing the old 'echo "scsi
add-single-device ...' trick.
