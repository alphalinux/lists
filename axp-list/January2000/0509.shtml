Hi !

> Is it doable? Is there some way I can convince my jensen to proceed to
> load my linux kernel after a reboot?

I guess, it could be(in srm):
set os_type = vms
set bootdef_dev = dka0 (or whatever you have)
set auto_action = boot
set boot_osflags = 0,2
(sda2,line 0 from aboot.conf)
... I hope :)
