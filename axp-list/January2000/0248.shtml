The UX uses ARCSBios, which isn't ARC or AlphaBIOS. It's different to both
of them. The latest version is available from the API website:
http://www.alpha-processor.com/support/ftp.asp?folder=/products/downloads/customer_support/UX

It is flashable, but you can't flash AlphaBIOS or anything like that in
there!
