With regards to your message at 11:01 AM 1/18/00 -0700, Michael W. Folsom.
Where you stated:
>Folks:
>
>Quick question - can Linux for Alpha's support 4 GB of RAM?
Yes, as long as the motherboard supports it.
For example a DP264 does support it.
And yes, kernel 2.2.14 with AA patches supports 4GB ( possibly more) and
works well.
We a rerunning a machine in our shop for a week now with this, and all the
testing we have done so far works well. Mcalpin streams test gives
virtually identical results with Linux and Tru64. Actually, when we get
close to or into swap space Linux is quite a bit faster on this benchmark.

>If so, when you compare Linux to Compac's True64 Unix how good is
>Linux's performance?

Excellent. Less system resources used. Typical install shows me 38MB more
free RAM with Linux and similar services running..
