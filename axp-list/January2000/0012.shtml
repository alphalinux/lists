Working from Mark Abene's earlier patches to libpcap-0.4a6 and
tcpdump-3.4a6, I kludged up the sources to the latest CVS snapshots
to build and operate cleanly on Linux/axp. Some coders picked up
libpcap and tcpdump after Vern P. left LBL, and have added some
new featurisms to keep it fresh - see http://www.tcpdump.org/
if/when their server comes back up.

Integrating the modifications into the official source should be
fairly simple; add a definition for Linux/axp, and offset the
custom timeval code with hefty #ifdefs.

Grab the port of the 30 Dec sources from:
ftp://ftp.alphalinux.org/people/dfrasnel/
