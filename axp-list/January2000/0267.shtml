Jakma, Paul (Paul.Jakma@compaq.com) wrote:

> Would API consider releasing a flash-writer to flash MILO in?
>
> With the work stefan has done to port MILO to 2.2, UX flashed with MILO
> would
> be able to boot from any controller supported by 2.2, eg Mylex RAID.
> ARCSBIOS is dead, and MILO as firmware would be preferable.

While you're at it why not ask for SRM? SRM, IMO, would solve a lot of
problems with this board... If there was a way for soemone outside
Compaq/API/Samsung to develop SRM, I'd jump on it in a second and learn
to be a SRM-dev master! These boards have been prone to several
problems, most notibly the ethernet, but I still think they are the best
OEM pre-21264 boards out there!

I would imagine flashing MILO over ARCSBIOS would probably result in the
loss of certian features as it does with most newer Alphas flashed with
MILO. YMMV. This is just from what I've seen on the list, check the
archives.

Just my $0.02
