On Fri, 7 Jan 2000, Thomas Weyergraf wrote:

>
> Which leads me to the - probably very stupid - question of how to unpack the *.zip file
> under Linux. Yes, I feel a bit dull not knowing this, but I don't really use/used gzip and compress.

It actually not gzip-ed, but rather pkzip-ed. Try using the unzip command
from the command line, if it's not installed there are RPMs on the CD for
zip and unzip just install those and you'll be OK.
 
> Has anybody done the flashing yet ? Any success or horror stories, that can be told before
> I fry my UP2000 flash-mem ;-)

I've flashed it about 20 times so far....no problems. As long as the power
doesn't go out while you're flashing, you won't have any trouble.
