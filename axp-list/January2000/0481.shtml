Well, you seem to have mentioned that you tried a text install but it just
went into X mode anyway? I've done several text installs with 6.1 w/o any
trouble. How are you starting the text install? All you should need to do
is add the 'text' flag to the end of the boot command, so from MILO,
something like:

MILO> boot hda:kernels/generic.gz root=/dev/hda text

also, if you have another machine you could try a serial install, then you
won't get X!

MILO> boot hda:kernels/generic.gz root=/dev/hda console=ttyS0

--rdp
