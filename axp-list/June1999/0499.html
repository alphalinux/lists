The DS20 can be ordered using three basic part numbers with reflect the
OS-Less Linux-Ready configuration, The part numbers reflect the supplied
memory size of 128, 256 and 512MB respectively, The buyer then must either
add memory to the order to bring the box up to 2GB if they wish, or purchase
and install the memory from another source.
