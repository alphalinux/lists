There's a good chance that most of the first four problems are related to
the same thing: the xfs server under RedHat 6.0 for axp. There are
several notes about it if you want to search for "xfs" on the archive
site: "http://www.lib.uaa.alaska.edu/axp-list/". Basically, it is only
necessary to edit the XF86Config file to list static fonts instead of the
font server: FontPath "unix/:-1".

There is also some notes on the ping problem if you search in the archive.
I believe there is a patch available for ping. I haven't had a chance to
try and fix this problem yet. You can probably recompile your own ping or
wait for a whole new netkit to come out.
