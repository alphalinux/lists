Hello,

Mark Lin wrote:
|
| Has anyone gotten RedHat 6.0 installed on a DS20? So far I've only been
| able to get RedHat 5.2 installed on our DS20.

I have two DS20s and I have successfully install 6.0 on them.
It was a much easier install than 5.2, as network install works fine.

| I've used the bootdisk/ramdisk that the page above suggests and even making
| my own bootdisks but everytime that I boot, when loading the ramdisk,
| something goes wrong (I can't exactly tell what since nothing is shown) and
| cpu halts and I get thrown back to the SRM prompt.

I think the key is to make sure you don't have any unsupported hardware
in the DS20. One of our DS20 was a very "nice" model with two SCSI
controllers (one of them a fancy raid thing). We had to take out all
of the "excess" controllers. We took out the graphics board that
came with it, and replaced it with one of the old ones from our other
Alphastations. Then it installed fine.
I only get 640x400.. (not using the console, so I don't care yet)

I used the bootdisk/ramdisk that came with 6.0.
The 6.0 that I used was ftp'ed from gatekeeper.dec.com, as I
doubt the official 6.0 for Alpha is available in Japan yet..
