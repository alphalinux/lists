>>> Bernhard Gschaider said:
>
> I tried to install RH 6.0 on a DS20 the way it is described in
> http://www.digital.com/alphaserver/linux/install_guid.html
>
> Everything works fine up to the point when the ramdisk-disk
> has been inserted and uncompressed: I am thrown back to the SRM
> console and I am told that the processor halted
>
> Or is there something that isn't mentioned in the Installation Guide?

I think this is something wrong with the floppy install. Did you ask RH?

Anyway, there is hope, since one can boot and install directly from CD
without the use of any floppies. Try doing:

[NOTE: this assumes a SCSI CD; if you have IDE, use dqa0 and hda as approp.]

>>> set boot_file kernels/generic.gz
>>> set bootdef_dev dka400 # assumes CD is unit 4
>>> set boot_osflags "root=/dev/scd0 bootdevice=sr0 bootfile=kernels/generic.gz"
>>> boot

Remember to reset those boot_ envvars when the install is done.

Good luck.
