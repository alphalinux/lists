I think this is something wrong with the floppy install. Did you ask RH?

Anyway, there is hope, since one can boot and install directly from CD
without the use of any floppies. Try doing:

[NOTE: this assumes a SCSI CD; if you have IDE, use dqa0 and hda as approp.]

>>> set boot_file kernels/generic.gz
>>> set bootdef_dev dka400 # assumes CD is unit 4
>>> set boot_osflags "root=/dev/scd0 bootdevice=sr0
bootfile=kernels/generic.gz"
>>> boot

Remember to reset those boot_ envvars when the install is done.

Good luck.
