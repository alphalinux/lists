> > I notice there are unsupported binaries for PPC, so maybe if we lobbied
> > netscape we could get them to provide an Alpha binary aswell.
>
> When the DEC CC for Linux/Alpha is released, it has the
> "-taso" option, and the compilation becomes possible.
>

 Compilation may work, but.....

 The DEC C option for making 32bit pointers is -xtaso. -taso is
 a DEC linker option. I doubt very much that just the native Linux
 DEC C compiler will allow you to make a "taso" image on Linux/Alpha.
 There's header file assumptions about pointer size that need to be
 addressed, shared libraries also need to be loaded below 4GB, and
 other issues.
