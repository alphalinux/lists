We expect our LX164 alpha server to provide almost all of our network services
including: dns, nfs, intranet web serving, NIS/yp, samba, netatalk, cvs, email.
We had been running 5.0 with quite a number of rpm upgrades, and this was not
all peaches and cream.

When we upgraded to 6.0, there were several surprises:
1) How some very fundamental network services were broken and this was
"missed"? i.e. nfs
2) How well almost everything else ran in general. It was relatively easy to
patch the broken packages by downgrading to older versions.
3) Most of the previously unresolved mysterious problems we had disappeared
with the upgrade.

So, maybe 6.0 isn't perfect, but it is certainly usable.
