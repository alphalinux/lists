Hi, all;

Ok, so I finally got all my parts (SX mobo from Ebay) and put it
together.

The machine has AlphaBIOS on it; it came with v5.62 (which I've since
upgraded to the v5.66 from DEC's ftp site). I want to go to SRM,
though. I realize this has been discussed in the past on this list,
but it looks like things might have changed since then, and I want to
make sure I get it right. But I'm confused as to the correct part
number... maybe I should just wait until Monday and get on the phone
with someone, but I thought someone here might know the answer.

In the past, the cheapest solution was to get the Alpha Motherboard
SDK, part number QR-21B02-03, which contained the goodies you need to
install SRM "from scratch", among other cool stuff. However, this
part number appears to have been "obsoleted" by a new SDK.

It looks like the new SDK (v4.0) no longer contains the SRM
console firmware images - the SDK product brief and "readme first"
docs kindly tell the reader that

     Alpha SRM Console firmware images that were formerly supplied
     in the Alpha Motherboards SDK are now available on a separate
     CD-ROM from COMPAQ:

     Alpha Systems Firmware Update CD-ROM: QY-003AA-A8

This seems to be the annual subscription service, which I'm not really
interested in at this time - I just want the CD and no subscription
for quarterly updates. I did find the above part in a pricelist:

QY-003AA-A8 ALPHA AXP CNSL/FW ANN UPDT SV

and at the same time, I saw this:

QZ-003AA-E8 ALPHA SYSTEM FW CONSOLE

which sounds like it could be what I want, but as I could not find
further documentation...

Also could not find references to QT-5VUAA-XA and QB-5VUAA-SA, which
are the parts list on the firmware web page.

Aaargh.

So - what part number do I want? Just a CD, no subscription.

And am I to understand that the quarterly firmware update CDs contain
the necessary stuff to install SRM "from scratch"? Has it always been
so? I've never had to do this before, and have never used the
firmware CDs before, so I don't know.

Any help would be appreciated.

Thanks in advance,
Craig
