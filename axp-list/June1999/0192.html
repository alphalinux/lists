On Thu, Jun 10, 1999 at 01:30:18PM -0600, Neil Carlson wrote:
> I've got a 164LX (Aspen Durango II) that I'd like to stick 2GB of
> memory into. It's got 4 dimm sockets. Is it possible to get
> 512MB dimms now? Sources? Would this amount of ram be a problem
> for linux (2.2.x) to handle?

        That amount (2GB) of RAM is no problem at Alpha Linux, larger
        amount might be a bit difficult due to issues at PCI-bus to
        main memory access. There are ways to handle even larger main
        memories, but the effort to use scatter-gather PCI/memory-bridge
        mappings has not yet begun..

        After that, the limit will be -- high :)
        ... but it requires complete workout thru all PCI, and possibly
        even ISA device drivers to use that mapping layer.
