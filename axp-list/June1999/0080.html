I just places 6.0 on a virgin 164SX and have found a number of problems.

General config:
        128Mram
        6Gb IDE HDD
        ATI 3DRage video
        PCI ne2k
        SB16 clone

1) under Gnome(I havent run anything else yet) the curser in an Xterm
session seems to display a very strange overlay when the cursor
dissapears. If I move the mouse the overlay dissapears and the regular
cursor comes back.

2)Gnome seems to generate a high number of core dumps.

3) I have had gnome effectivly lock up so tha I was unable to create new
windows. I could still move around between open windows but I could not
open any new ones.

4) xfm seems to die.

5) ping gives me:

wrong data byte #8 should be 0x8 but was 0xb1
