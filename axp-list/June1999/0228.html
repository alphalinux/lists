>Just wondering if anyone tried 2.2.9-ac5 in a heavy loaded
>single CPU Alpha yet? Does it need additional patches to
>be more stable?

>I tried 2.2.9-ac4 and it froze up on me, twice in a day. After
>I went back to 2.2.5, things seem more stable and did not experience
>any lock up so far (up for 20 hours now under moderate load.)
>But ypbind-mt does not work so well in 2.2.5, sometimes cannot
>get user info under heavy load.

Can you describe exactly what is happening with 2.2.9-ac4? The reason I
ask is that I've found 2.2.9 (ac1,ac2,ac3, with and without d-axp-229 and
others) to be very unstable. I was testing a machine at work and it would
run between 5 minutes and 1/2 an hour and then it would lock solid. I
should say that I was testing a new machine so I was keeping the load
between 3 and 5. No OOPS, no info in syslog. The only thing that functioned was the ALT-SYSRQ
keys. I went back to 2.2.5 and all was fine, I think that 2.2.7 was also
fairly reliable.
