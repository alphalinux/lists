The "other installation", as far as installation process is involved,
is basically the same even if your screen graphics looks somewhat
different. That means that you start with milo and you load from
there a ramdisk from which /linuxrc starts installation procedure.

The problem with README was that it was not only in German but was
also incorrect so it is to your advantage if you cannot consult it. :-)
An installation, once started, is pretty obvious. Like Martin said
gobs of things in a resulting system simply do not work.

I tried that, but this was a while ago, so I am bit fuzzy on small
details. Yes, I wrote and submitted pretty lengthy report. Let us
hope that the upcoming release will be of a better quality. Unless
you are testing and have some experience with Linux and Alpha in
particular you DO NOT want this beta. It had a "very early beta" feel.
