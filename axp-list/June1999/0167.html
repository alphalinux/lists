There are no drivers for TNT for Alpha/NT. The main reason being
companies like Nvidia write these drivers in assembly code so it's not
portable and a bitch to port. Now that the source code for the Linux X
server has been released some one who's knowledgable in both windows and
*nix could write the drivers using the Xserver as a rough template but
I don't see anyone even mentioning it on the AlphaNT list. I have a
question. Is anyone working on the TNT driver for Alphalinux? Diamond
does make a PCI version of the V550 and so do a few others.
