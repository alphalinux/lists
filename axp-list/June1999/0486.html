On Fri, 18 Jun 1999, T.Hval wrote:

> I have been assigned with some simulation work which demands a quite
> powerful computer. And I was wondering if somebody could supply me with
> some essential information.
> What is the largest amount of memory an alpha box running Linux can make
> use of?

Linux has a memory limit of 2Gig currently. This is not an Alpha
limitation but rather a Linux limitation.

> How quick is the quickest Alpha chip available and how does it compare
> to the quickest intel can offer (a PIII 550)?
>
> (Unfortunately the software I am gona use is not parallelized so I am
> stuck with a single processor machine)

You may want to look at the new motherboard from Alpha Processor Inc
(UP2000) it's SMP capable but will run fine with only one.

It will support processors at 667MHz (2 and 4MB cache) and 750MHz (8MB
cache).
