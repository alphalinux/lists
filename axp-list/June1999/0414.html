Hello everyone,
 
I am trying to use a non-compression option, when backing up to an
Exabyte tape drive, attached to a Linux system running Redhat 2.2.5-15

When I use:

tar cvf /dev/st0 file.*

compression is assumed.

If I was to try using:

tar cvf /dev/st0m files.*
tar cvf /dev/st0l files.*
tar cvf /dev/st0a files.*

/dev/st0l: Device not configured
/dev/st0m: Device not configured
/dev/st0a: Device not configured

I also did the following:
              mknod -m 666 /dev/st0l c 9 32
              mknod -m 666 /dev/st0m c 9 64
              mknod -m 666 /dev/st0a c 9 96

which still did not help.
