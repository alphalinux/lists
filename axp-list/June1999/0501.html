Certainly the one I use seems to run with 2gig; I sent mail to see if
there is some real limitation, or just a mistake on the web page.

Note that extrapolating from part numbers to supported memory is silly:
the number of different part numbers (basic machine configurations) are
deliberately kept small, as each one ends up costing money in the ordering
system, in manufacturing, and confusion in the field.
