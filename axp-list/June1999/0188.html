Neil Carlson wrote:

> I've got a 164LX (Aspen Durango II) that I'd like to stick 2GB of
> memory into. It's got 4 dimm sockets. Is it possible to get
> 512MB dimms now? Sources? Would this amount of ram be a problem
> for linux (2.2.x) to handle?

Someone else would have a more authoritative answer on this, but I belive
that the Linux kernel can only handle 1GB of physical memory, but the rest
(up to 2GB) can be used for cache. This may be a limitation of the Intel
architecture, though.

As for where to get this memory, the best place to start looking (IMO) is
http://www.pricewatch.com/
