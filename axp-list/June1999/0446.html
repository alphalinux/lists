On Sun, Jun 20, 1999 at 07:27:00PM -0400, Uncle George wrote:
> according to gdb /lib/libc.so.6.1 is linked to my JVM. My simple 'c' pgm
> (which works) also shows /lib/libc.so.6.1.

That doesn't count.

The symbol version are bound at link time. So the creation of libjava.so
must explicitly include an `-lc'. To see that you have done that,

        objdump -p <library> | grep NEEDED

would report an entry for libc.
