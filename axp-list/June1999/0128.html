RaymondInFinland wrote:
>
> Hi,
>
> I recently decided to do a clean upgrade from RH 5.0 to 6.0 but i'm
> stuck at the install process..
>
> I installed the new milo that ships with 6.0 (no problem at all), i
> boot with 'boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1
> prompt_ramdisk=1'. The kernel boots fine and prompts me for the
> ramdisk. When i insert this it says: compressed image found at block
> 0. Starts loading and then exits with a kernel panic, Unable to mount
> fs on 01:00.
> Whatever i try(i also tried the new updated image), i cannot seem to
> get it working. Does anyone have a clue about what is going on?
>

What kernel are you booting? If it's from the RH6.0 cd, it
should work. The new ramdisk is a romfs, not ext2 so you have
to have a kernel built to support it if you don't use RH's.
Try make a new floppy and see if that works. I had trouble
with the first floppy I made for it.
