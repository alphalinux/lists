> try KXicq ( www.caiw.nl/~herwinjs ) it is tested on my own Alpha ;-)

KXicq looks very promising, but I have some problems with it. First let me
say that I compiled 0.3.1 against kde 1.1.1 libs, so if this is the problem
you can stop reading and tell me what works ;)

Anyway, I've got the following problems:

* messages sent through the www interface or through mail
  (<uin>@pager.mirabilis.com) seem to get silently ignored by kxicq: other
  clients display them immediately, kxicq doesn't do anything

* messages sent from other icq accounts seem to be received fine, but when
  i click on the name icon to read the message kxicq crashes immediately
  after it opened the window displaying the message

* when i sign up to a new account using kxicq this account seems to have no
  info (name, gender, favorite color :-) etc) stored on the server.
  Corollary: When I sign up to the icq network using kxicq, people with
  other (wind*ws) clients get a bogus error message ("disconnected") when
  trying to add me to their contact list. I agree that the error here is
  probably inherent to the icq system, but some part of the error also seem
  to be with kxicq

Please tell me if I can do anything else (like trying to gdb kxicq or
providing more detailled infos)!
