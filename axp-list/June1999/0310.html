To me this sounds like /dev/ram? which is a device for ramdisks. You can
make any filesystem that is supported by linux on it and mount it somewhere
For details on doing so look at the Bootdisk Howto which can be found with
most i386 linux distributions (I am totaly new at Alpha/Linux but i think
that ramdisks works similar to i386). I did not found an mfs Filesystem
while configuring a kernel for alpha so i think there is no such filesystem
available. But again , i AM new at alpha-linux, so i might be wrong.
