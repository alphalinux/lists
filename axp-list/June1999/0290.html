smbmount used to be seperate, but has been made obsolete.

There's now code in samba to do the same thing, but it's not included in the
redhat samba rpm by default. You'll probably need to download the src.rpm
and add the configure option to the spec file and then rebuild the rpm.

Or you might be able to find it seperately as smbfs or something like that,
not sure.
