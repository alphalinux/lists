>>> Rich Payne said:
>
> Linux has a memory limit of 2Gig currently. This is not an Alpha
> limitation but rather a Linux limitation.

But SPARCs do run with more than 4! This is primarily because they have
total controller of their drivers, so that they can be made to do what
the generic LINUX kernel drivers don't do yet: request/release DMA addresses
for memory whose physical addresses are out of the 32-bit range.
