Adam C. Powell, IV wrote:
>
> "warp@xs4all.nl" wrote:
>
> > 2nd try, don't forget to add something like:
> >
> > -fomit-frame-pointer -funroll-all-loops \
> > -finline-functions -ffast-math \
> > -Wall -O6 -DUSE_MMAP
>
> Doesn't -O6 -mcpu=ev4 (or whatever cpu) include all of these?

AFAIK, no, but I could be VERY wrong ;-)

> Are all of these optimizations helpful for mpg123?

Yes, cpu usage went from 15% (with plain -O2) to 8-9%
using the above CFLAGS.. when I linked it to libffm
it decreased another 1-2% (about 6-7% total cpu usage).
A 200% performance increase is not bad /me thinks ;)

BTW, this is on my 21164a/366MHz EV5 based box.
