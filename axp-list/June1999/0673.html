Ta-da.
Metrolink gave me the go-ahead to distribute a modified
'Xmetro' binary (against 4.3.1 patch 3).
If you're running MetroX:
- You need 'metroess-4.3.1' and 'metroessp-4.3.1-3' installed.
- Grab the binary 'http://www.op.net/~rbowles/axp/Xmetro.bz2'
- Copy it to /usr/local/bin
- Execute the following commands (as root):
        cd /usr/local/bin
        bzip2 -d Xmetro.bz2
        chmod 4755 Xmetro (optional)
        cd /etc/X11
        mv X X.orig
        ln -s /usr/local/bin/Xmetro X

If you'd care to save bandwidth, snag the patch from:
        http://www.op.net/~rbowles/axp/Xmetro-4.3.1-3.patch.gz

Don't forget to use the '--binary' and '--rtfm' flags. :)
