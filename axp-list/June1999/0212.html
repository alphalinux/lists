I understand. Have you tried fdisk? I was able to use fdisk to
partition the drive I wanted to install to without using the Win95
disk. I just did the Win95 boot disk to see if Disk Druid was looking
for primary dos partition, and apparently it was. I had this problem
with a disk that had Suse 6.1 on it, and I had used fdisk to partion it
for Suse. When I loaded the RH 6.0 CD and booted it all went fine up to
the point where I tried to use Disk Druid to repartion my drive, then it
dumped with the same error you had. I then rebooted and used fdisk with
no problems and the install proceeded as normal. I have two other disks
in this machine and neither one of them have a dos partion on them but
they didn't interfere with Disk Druid during the install, it was just
the drive I was installing to that gave me problems.

Craig Oettle
