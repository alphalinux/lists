Are you sure needed to do all this?

I use the rp-pppoe (on my intel) with out any kernel modifcations and it works great.

Roaring Penguin even has alpha binaries that ready to go.

The steps I use are:

1) download latest from http://www.roaringpenguin.com/pppoe/#download.
2) install the rpm.
3) as root run adsl-setup.

Done. I'm now online.

There are not any kerenl modules needed. It was my understanding that you only
needed the kernel modules if you wanted a use kernel level ppp.
This gives a performance boost I think, but you might try the easier method first.
