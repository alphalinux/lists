Have you loooked at www.debian.org. There you will find a "Debian on
CD" link, "Port Specific Pages" with further links to Alpha and it's
sublinks. You could lie about "Official CD" status and get the iso
images yourself. Also, there are debian packages that can create CD
images from local/remote archieves. Personally, I don't do CD
installs, but use net installs and net updates. Remember, once you
have an initial install, updates come easy with Debian. There really
is no need for a CD (see apt-get). It is a matter if getting used to a
superior technology.
