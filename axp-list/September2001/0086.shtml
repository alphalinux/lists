The libcpml libs are with the "ccc" or the compaq C compiler. There is a linux
version somewhere on compaq web site.
libXm is the motif library, which are not generally on compaq's site. There is
lesstif ( www.lesstif.org ? or was that www.hungry.org ). I think that there is
a open-motif somewhere. And there is metrolink's motif software for the alpha
computer.
