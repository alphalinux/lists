"Terry A. Bowling" wrote:

> James and Bryce - and anyone else who wants to try building openoffice
> for me,
>
> Here's the requirements and links for doing a full build of
> OpenOffice638. Everthing is in alpha-rpm form except the stlport and
> gpc.
>
> The gpc is just a c source file and a header file, so you will need to
> make the object file with the following command:
> gcc -c gpc.c
> This will make the object file gpc.o - I assume you just need to put
> this with your other object files. The gpc231.tar.Z file does not
> include any instructions.
>
> Openoffice requires:
> stlport 4.5 available at http://www.stlport.org/download.html
> gpc 2.31 available at ftp://ftp.cs.man.ac.uk/pub/toby/gpc/gpc231.tar.Z
> JDK 1.3.1
>
> Compaq's Alpha-Linux JDK available at
> ftp://ftp.digital.com/pub/Digital/info/java/jdk-1.3.1-1-linux-alpha.tgz
> JDK requires:
> libots-2.2.7-2.alpha.rpm available at
> ftp://ftp.digital.com/pub/Digital/info/java/libots-2.2.7-2.alpha.rpm
> libcpml.so available at
> http://www.compaq.com/math/download/download_.html
> lesstif-0.92.0-2.alpha.rpm - on the RH Powertools cdrom or rpmfind.net
>
> OpenOffice source at:
> http://a2024.g.akamai.net/7/2024/2064/OpenOffice638/anoncvs.openoffice.org/download/OpenOffice638/oo_638_src.tar.gz
>
> View build instructions at:
> http://www.openoffice.org/dev_docs/source/build_linux.html
>
> I'm eager to hear how it goes.
> Terry
