-> now my question: is there a way to configure the milo to boot from for
-> example kernel sda1:vmlinuz in system with root=/dev/sda2 (sda1=fat,
-> sda2=ext2). To boot without entering at the prompt ?

Hi!
I had alpha bios and milo on my machines for a long time
and i know that you can do this on alfha bios.
After entering the alpha bios menu select "Utilities" --> "OS selection setup"
then edit the entry for linux and make the changes you need...
On the "OS options" line you have to write the line you want to pass to MILO
"boot sda1:vmlinuz root=/dev/sda2 ... ro"
Don't forget the "ro" option or you'll see funny message telling you that you didn't
tell to boot your root partition as read only but "someone" will do this for you :-)
You need to have milo on your disk!

This was on alpha bios 5.something for a pws 600a. I hope you'll find something
similar on your machine.
