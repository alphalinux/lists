Typically on my system to boot from the cdrom to do an install I first
boot into the srm console.

After which I do:

boot dka100 -flags 0

Check "show device" to see what numbers are assigned to what
hardware. On my system the cdrom drive appears as dka100. This worked for
booting the RedHat 7.0 CD.
