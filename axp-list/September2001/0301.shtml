On Mon, Oct 29, 2001 at 10:13:09AM -0500, Aaron Gee wrote:
> ext3 is supposed to be included in the kernel "real soon now". Check out
> your favorite kernel archives for the discussion. FYI ext3 is included in
> the RH 7.2 and is the default install.

In some sense it is already "included". It is in current kernels from
"ac" series.

If you want precompiled kernels then if you will install
kernel-2.4.9-6.alpha.rpm from current updates to RH 7.1, which requires
updates to some other packages as well, then you will find out that ext3
support is there (as well as reiserfs support but RH 7.2, or 7.1 for
both types, does not give you an option to choose ReiserFS during an
installation; you will have to create such filesystems later).
