The >>> power-on prompt is srm, do a "show dev" see what the device
name for the CD rom is, it's usually dka400 or dka500

then simply put the cd in and type BOOT DKA500 -fl 0 (using your device
name thought)

It is useful to pass a boot flag (-fl) of 0, if it was a unix system it
may be set to 0,1..
