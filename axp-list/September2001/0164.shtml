>
> What is SRM?

Alphas have (had) two consoles, one for running NT and one for running
Tru64/VMS/Linux. The latter is called SRM, the former ARC or AlphaBIOS.

If your AS1000A was used to run NT, you'll nned to shift to SRM; this
should be possible via the menus you get with ARC/AlphaBIOS. If it is
not there, you can download firmware upgrades from the COMPAQ website
to add SRM to your machine.

And there is no MILO for AS1000A; you MUST use SRM.

Good luck.
