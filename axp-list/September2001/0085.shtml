On Sat, Sep 15, 2001 at 12:11:02PM -0400, tony.icuc wrote:
> Does anyone have the following libs:
>
> libcpml.so

CPML (Compaq Portable Math Library) is a part of a compiler suite
from Compaq for Alpha linux and a link to that is prominently displayed
on www.alphalinux.org. It is

  ftp://ftp.compaq.com/pub/products/C-CXX/linux/compaq_c

You will find there two versions of cpml (for ev4 and ev5) packed
in an rpm format.

> libXm.so.2

This is not a math library but the basic Motif library (and quite
possible you already have it installed but maybe not in path you are
searching by default). If this is not the case then you will get it by
installing either 'lesstif' or 'openmotif' packages. Both you will find
in "Powertools" set for 7.1. Make sure that /etc/ld.so.conf includes
relevant directory on its list.
