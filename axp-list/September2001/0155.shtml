Hello there - this is a preliminary success message.

With all my troubles earlier this week, I ended up determining that one
of my CPUs was bad. I'm still in the process of testing the memory
chips, but so far I haven't received any errors like I had before.

So, now running on one processor, I installed gcc-3.0 so that I could
recompile the kernel and hopefully not get anymore unaligned trap
messages. I also put 2.4.12 on there, and I have seen a huge jump in
performance for both high memory and low memory use applications. I'm
talking 15-20% faster than RH 6.2 w/ kernel 2.2.16. This is similar to
the increase I saw with kernels 2.4.9 and 2.4.10 with low memory use,
but with high memory applications (~1 GB on my 2 GB machine), the
machine swapped all over the place, which obviously slowed it down
considerably. So far, I am very happy with 2.4.12. Unfortunately,
I still see a few unaligned trap messages, but I don't mind that much...

We'll see.
