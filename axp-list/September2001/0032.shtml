So it's automatically booting to Tru64 on power up? Use the halt button,
if it doesn't have one (I think that PWS machines do) you can always take
the brute force approach and disconnect the disk!

Once you get to SRM's prompt ( >>> ) type:
>>>set auto_action halt

then you'll be able to do as you wish.

--rdp
