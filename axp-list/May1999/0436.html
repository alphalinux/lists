All,

So that there's no confusion, the 164UX/BX is not supported by Compaq,
as are, e.g., the APC 164LX/SX. The firmware used in the 164UX/BX,
ARCSBIOS, was developed by Deskstation, not Compaq, so using the MILO
for 164LX and 164SX should not be expected to work in the 164UX/BX.

Cheers,
jwl
