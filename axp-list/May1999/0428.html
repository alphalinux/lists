You're close...

I'm assuming you're booting from a CDROM (at dka0, for example.) From the
SRM console, you need to use a boot command that looks something like
this:

boot dka0 -fi /kernels/noritake-s.gz -fl "root=/dev/fd0 load_ramdisk=1 prompt_ramdisk=1"

If you're getting to the 'aboot' console, then you've loaded the secondary
loader correctly, but you're boot line was not loading the kernel
correctly.
