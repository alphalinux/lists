Dan,

 you want "adduser" - check the man page ("man adduser") or try
"adduser -h" fo the details, it'll default most everything, i.e.
"adduser fred" will create a passwd/group entry and a /home/fred.
You'll have to set his passwd with "passwd fred" before he can login.
NOTE: this all must be done as root.
 There's also a gui interface under control-panel, which also must be
run as root.
