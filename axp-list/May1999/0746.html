Not sure just any ATX PS will do. My PC164 has a
400 watt ATX power supply in it. Not many ATX boxes
use that size of PS. The form factor is ATX though.
I recall some discussion about it and the issue was
that you had to have a quite large PS to supply the
current needed to drive the processor.

I don't use SRM so I can't say whether it works but if
you can find a description of the SRM CD, you can look
to see if it's there. I think it is available as some
sort of SDK from Compaq, under $100. Search the archives
and you can probably find the part number. It's been
discussed previously.
