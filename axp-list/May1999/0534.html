> In order to run mozilla M5, what other libraries or anything
> are necessary, apart from what is in the above M5 directory ?

If you're running Rh 6.0 you're all set. just grab the nspr and the
libIDL rpms.
If not you'll need gilbc-2.1, glib-2.1, gtk+-2.1. Here's a complete
list.
