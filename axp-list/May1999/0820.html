We had a production sun server that had 4Gig of ram and about 16 of
swap. The process memory limit was 2Gig. We used to have an app that
_leaked_ about 256 meg a day, and so used to get killed and restarted
about twice a week.

And yes, this was a large bank.

The moral of the story is: Anything Slowaris can do we should be able
to do.

Sean
