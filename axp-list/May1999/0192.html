The DS10 is brick? If so, there are no daughter cards, just one mainboard.
The DS20 is called DP264 and XP1000 is called Monet. They are two seperate
systems and don't use the same daughter cards.....
ES40 = Clipper.

Brick - Uni, no daughter card 466MHz CPU 75MHz bus
DP264 - Uni/Dual, daugher cards have CPU and cache, 500MHz CPU 83MHz bus
Monet - Uni, daughter card has core logic, CPU and cache. 500MHz CPU 83MHz
bus.
Clipper - 1-4 CPUs, daughter card has CPU/cache, CPU=500MHz 83MHz bus.
