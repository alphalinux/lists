> 2) You can definitely install Red Hat Linux using just the CDROM.
>
> If you are installing Red Hat Linux 5.2, and assuming the CDROM is at
> SCSI ID1, To do so from SRM, you would type:
>
> boot dka0 -fi /kernels/noname-s .gz -fl root=/dev/scd0
>
> and from ARC/AlphaBIOS, you have to select 'Run a Program' and run:
>
> D:\milo\arc\linload.exe d:\milo\udb-noname (check this one, I haven't an ARC machine in front of me to verify this)
>
> This should get you started...
>
