On 20-May-99 Danilo Godec wrote:

> I had similar problems with the same combination (SX164/S3 Virge/DX) when
> I was using XF86_S3V server (the S3 Virge specific X server).

Odd, that. I had the exact opposite results on my AS200, though the video card
was an S3 virge VX. The SVGA server never did work correctly, but the S3V
server was rock solid. I guess the moral of the story is - try them both.

> BTW, these lockups with XF86_S3V only happen on Alpha... i386 version
> works nice.

Depends on the card and software you're running. fvwm2 used to hard lock a
virge DX on my old intel box on exit - every time. I had to write a script that
would forcibly kill X when exiting fvwm2 to prevent the lockup.
