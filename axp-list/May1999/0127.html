On Wed, 5 May 1999 mgraffam@idsi.net wrote:

> I was wondering if anyone has gotten ide_cs.o (from the pcmcia package)
> to work on a Multia?

Under linux 2.0.35 and 2.0.36 I have the PCMCIA package working with my
PCMCIA modem. I have not been able to compile PCMCIA 2.0.9 under the 2.2
series kernels, and currently cannot compile anything because my binutils
package is seriously hosed.

IIRC, all I did was install IDE support, removable IDE support (PCMCIA),
fix the pcmcia startup scripts from redhat (they were doing something
wierd waiting for interfaces (pcmcia network adaptors apprently) to come
up), reboot, and then compile and install pcmcia from the running kernel.

You may want to try turning on kernel symbol versioning, though that won't
eliminate unresolved symbols. Be sure to follow the pcmcia instructions
because some two or three modules will load to get pcmcia working, and be
sure to run "depmod -a" after the installation.
