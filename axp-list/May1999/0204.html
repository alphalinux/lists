> At 06:53 AM 5/6/99 , you wrote:
>
> Do you have any info on what changes must be made to the system to operate without xfs?

to run the system without xfs remove it from the services list and change
XF86config by adding the fontpath arguments again. that's it - since then
redhat 6.0 seems to run fine (at least it survived the night with X and
several apps running).
