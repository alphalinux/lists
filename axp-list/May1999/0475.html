Jurgen Kramer wrote:
>
> I upgraded my sx164 to RedHat 6.0 with a fresh install. Now telnet,
> rlogin and rsh do not work anymore (from a remote machine).
>
> When I telnet, rsh or rlogin to my Alpha box it says:
>
> [telnet]
>
> telnetd: All network ports in use.
>
> [rsh, rlogin]
>
> rlogind: Out of ptys.
> rlogin: connection closed.
>
> I find this hard to believe, I even compiled in support for 1024 ptys.
> Am I missing something here?
> Is there a special configuration file somewhere?

You need to make sure that /dev/pts is mounted. Put the following in
/etc/fstab

        none /dev/pts devpts mode=0622,gid=5 0 0

and then type "mount /dev/pts".
