I have run my UDB at 32 bit colour with my S3 Virge DX 4MB card using the
S3V server but I currently use the SVGA server which only supports up to
16 bit colour. The commercial AccelX server will support 32 bit colour
though and is speedy but expensive for my tastes.

I currently run a 533SX with S3 Trio 64 2MB card at 16 bit 1152x900 and it
works fine.
