Hello

I have had exactly the same problems, when installing RH6.0 . I wanted to
upgrade from 5.2. I tried 5.1 and 5.2 disks in almost all variations, but did
not succeed. Because with Version 5.1 and 5.2 we had the same problems in
January and December, which took us a lot of time to resolve with no help from
Redhat, we have now decided to switch to Debian. And for a Ruffian system
Debian 2.1 seems to be a better choice. It installs right away wth NO problems.

Although Redhat might be the better and more mature distribution, with Debian
you need less time to install on a Ruffian system and the response time to
questions is much shorter with Debian.

Peter Biechele,
Germany
