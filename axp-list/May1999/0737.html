make dep
make clean
make boot

Use make boot instead of the Zimage stuff. Then just copy the kernel from
/usr/src/linux/arch/alpha/boot to either / or /boot . RH 6.0 keeps your
kernel in /boot so I suggest you put it there but don't destroy the
existing kernel, rename it or something. Then reboot and tell milo to boot
that kernel.

There's some System.map stuff that I'm not so sure how to resolve. Does
anyone know what you be linked to what in /boot?
