Kevin's right -- xcdroast rocks!

Couple of notes from my installation:

Depending on which kernel config builder you use, 'Other SCSI' may be
referred to as SCSI_GENERIC (as it is in the .config file that
make [whatever]config outputs). Also, I never got a good answer why
the shared memory parameter for alpha is so small. I left it alone,
and tweaked xcdroast. It can be done at compile time, at the command
line (for cdrecord), or with an environment variable. I'm running
just fine with a 3 meg fifo. Also, use the newer ncr53c8xx driver,
rather than the 53c7,8xx one. Ever since I made those changes, my
coaster:cd ratio went to 0!
enjoy!
