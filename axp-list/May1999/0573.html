On 17-May-99 Peter Petrakis wrote:

[Changing resolutions on a Multia's TGA...]

> I think the UDB's can to 1280x1024 as long as it's in 256 colors. To
> change
> resolutions press ctrl+alt+ (+ or -).

You can't do this with the TGA server - it only supports one modeline.

If you're running Redhat, the easiest way to deal with this is run
Xconfigurator. When it prompts you for the resolutions you want, make sure the
resolution you want *and only that resolution* is selected. Note that, while
Xconfigurator presents you with some 16 and 24-bit modes, you can't use them on
the Multia's TGA. Select an 8-bit mode.
