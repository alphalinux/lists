export the filesystem from the unix system, then nfs mount them to your
Linux system

example, you would add a line to your Linux /etc/fstab similar to
unix.xxx.xxx.xxx:/r10/linux/linuxroot /tst nfs
rw,rsize=8192,wsize=8192,bg,intr

where:
> unix.xxx.xxx.xxx: = node where the exported filesystem exists
> /r10/linux/linuxroot = name of the exported filesystem (/etc/exports on
> Unix)
        /tst = your local mount point.
        nfs = filesystem type

        The rest you can get from man pages..

        Otherwise try UFS filesystem you will have to build it into the
kernel and I'm not sure it will allow you to mount the disk. NFS is the only
sure way.
