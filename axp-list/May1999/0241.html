The DS20 has 2 256 bit memory busses. Thats 512 bits/cycle. That is how
they say 5.2GB/s bandwidth. The DS10 has 1 128 bit memory bus. This is 1/4
the peak bandwidth of DS20. Thus the 1.3 GB/s peak bandwitdh advertised.
Actual measured bandwitdh will vary on application. The crossbar switch
means that there is a path from each CPU to each memory bus. The bandwidth
per processor only remains constant if the number of memory busses goes up
at the same ratio. So, with DS10 and 1 CPU, and DS20 with 2 processors, you
can see the number of memory busses scales the same (1-2). They both have
crossbar switching architecture.

All info given here is from
http://www.digital.com/alphaserver/products.html

Please go there if you wish to get more public data on the great systems.
