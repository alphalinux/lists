Jason.Pincin@networks.net wrote:
>
> drivers/block/block.a(DAC960.o): In function `DAC960_DetectControllers':
> DAC960.c(.text+0xf9c): undefined reference to `ioremap_nocache'
> DAC960.c(.text+0xfa0): undefined reference to `ioremap_nocache'

grep through kernel sources reveals immediately the following line in
include/asm-ppc/io.h:
#define ioremap_nocache(addr, size) ioremap((addr), (size))

So, I guess, that in your place I would add the same thing to
include/asm-alpha/io.h (or to the offending DAC960.c file). True
'ioremap_nocache' can be found in include/asm-i386/io.h.
