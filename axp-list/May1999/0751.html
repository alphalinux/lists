No one mentioned modules, so if anyone is confused on those, something
like:

make (old|menu|)config
make dep
make boot
make modules
make modules_install

should do the trick. Make sure that ELF, ext2fs and your boot devices
/ busses are built into the kernel, not modular, as those are REQUIRED to
boot, run and mount on a typical system.
