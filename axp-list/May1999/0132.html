Along these lines, there are two very important
things you need to make sure of on your SCSI bus.

1. Use active terminators
2. If you install a 50 pin device, make sure you handle
   the high 8 bits termination correctly. You do this
   by either terminating with a wide device internally,
   or by using Adaptec's 68/50 pin cable adapter externally.
   Many 68/50 pin cables do NOT terminate the high 8 bits.

Adaptec seems to be touchy if you get the termination
wrong.
