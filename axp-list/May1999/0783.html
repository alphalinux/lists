I run the "new" software raid card off a symbios scsi card, which
works very well on my alpha. If you do this, the easiest way is
probably to go for redhat 6.0 and start by using their patched kernel.
It autodetects the raid personality from the raid's superblock, and
(unlike the old software raid code) you can simply mount up the raid
device in the fstab. I know some very big installations which have
gone for the software raid solution.

I've heard very good things about the mylex raid cards under linux,
but couldn't say for sure. On i386 I have used a DPT smartraid IV
card, but that needed to be set up under windows first. Once I had
done that, it worked very well, but didn't report much state to my
box. It just looked like a very big, very fast scsi disk. This was a
long time ago, though, and I believe that they may have some linux
tools available now.
