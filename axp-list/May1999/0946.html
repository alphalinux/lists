On Fri, 28 May 1999, Mark Arrasmith wrote:

> Does anyone have an ATAPI cd-writer working under Alpha/Linux? I've
> got one today and I would like to set it up on my alpha box, but all
> the help I've read is for x86 boxes (lilo and such).

I do. I have a Traxdata CDRW2260 (or something like that) ATAPI burner and
a Teac CD-532E ATAPI CD-ROM together with IDE disk. The procedure for
setting these things up was the same as for x86 box.

> Mainly, is it possible for me to use ide-scsi for my cd-rom's and
> generic ide for my hard disk drives. Or should I just compile in
> ide-scsi and not ATAPI?

I compiled both IDE disk, IDE/ATAPI CD-ROM and IDE-SCSI emulaton (the last
two as modules). On boot time I pass 'hdb=ide-scsi hdc=ide-scsi'
parameters to kernel so only the CD-ROM and the burner use ide-scsi
emulation.
