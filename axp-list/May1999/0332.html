Don't forget that from the NT console you should do
setup the system
and set the floppy type to 1.44MB
you can also do it by setting an environment variable FLOPPY=2
( inside NTsupplementary menu I think)

For me the environment variable worked over multiple switches between
arc and NT console.
b.t.w. the aboot boot image was better for me than the one from the
compaq/digital site
