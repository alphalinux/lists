The W means wide, 16 bit connections, instead of 8 bit
ones. If you want to add 8 bit devices to a 16 bit bus,
you need to pay attention to what happens to the other
piece of the wide data path. In particular, if you terminate
the scsi chain with an 8 bit device, you had better have
terminated the the rest of the data path with some other
connection. I use Adaptec's 68/50 pin connector which
properly terminates the top half and allows the bottom
half to continue to the narrow device. If your disk is
an UW device, then using it's termination should work.
If it's not, then you need to figure out how to get 16
bit active termination.
