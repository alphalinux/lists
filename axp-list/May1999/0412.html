If you create the partition on the destination disk and mount it you can
always
do something like
tar cvf - 'list of directories excluding mnt' | ( cd /mnt ; tar xpfv -)

beware that tar follows the tree - so make sure only root is mounted !

or use cpio ( but i don't know how to preserve links )
