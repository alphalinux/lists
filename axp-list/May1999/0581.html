On Tue, 18 May 1999, T.Hval wrote:

> How to get the righ term type, when using rlogin or telnet to other
> machines, such that it will be possible to use programs like pine through
> the rlogin/telnet connection.(I just want to set it to vt100, now I am
> getting a message that linux or kterm(depending on wheter X is used or
> not) terminal is not supported(or correct).

That depends on shell used. If you're using bash, type this:

# export TERM=vt100

If you're using tcsh (or other csh based shells), type this:

# set term=vt100
