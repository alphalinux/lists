Jaap Hogenberg wrote:

> The X server starts but exits with : No supported TGA chipset found,
> aborting.

IIRC, the problem is that with the TGA chipset you have to explicitly
tell it tha FB address. This is what I have:

Section "Device"
MemBase 0xa000000
    Identifier "TGA"
    VendorName "DEC"
    BoardName "TGA"
    VideoRam 2048
    # Insert Clocks lines here if appropriate
EndSection

I think you can find this address from the output of dmesg, but you
can't if you have it compiled as a frame buffer device (as I do).
