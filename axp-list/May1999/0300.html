You can try:

 boot dva0 -fi vmlinux.gz -fl "root=/dev/scd0 load_ramdisk=1
prompt_ramdisk=1"

BTW, under Linux, you can easily made a bootable cd by making first its
iso image (with mkisofs, distributed as rpm) and then making it bootable
with the aboot package (also a rpm package). Please, see the
/usr/doc/aboot-x.x docs for details.
