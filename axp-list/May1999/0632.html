I just burnt one today. I was able to upgrade using it on
my XP1000. Tried to boot from it using SRM but it gets hung
trying to mount my root partition, whether I select install
or upgrade. I suspect there is an issue with BSD labeled
disks. Booting using my own kernel disk and the RH5.2 ramdisk
was how I got it to upgrade correctly. Didn't bother to try
the RH6.0 ramdisk since I had the RH5.2 one laying on top of
the box. (lazy...)

Currently downloading the SRPM and I386 trees so I can also
burn those.

So, it can be done.
