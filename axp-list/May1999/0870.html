Wes Bauske wrote:
>
> Uncle George wrote:
> >
> > ANYBODY out there can create a file >2gig on linux/alpha ?
>
> Sure.
>
> -rw-r--r-- 1 wsb pds 15039538080 May 16 23:01 junk
>
> Just having troubles lseek()'ing into it past 2GB.

How about using ext2fs_llseek()?. From
/usr/include/ext2fs/ext2_io.h:

/* llseek.c */
ext2_loff_t ext2fs_llseek (int, ext2_loff_t, int);

The header is a part of e2fsprogs-devel package and, presumably,
a library which supplies this function belongs there as well.

BTW - manpage for llseek, at least for RH 5.2 distribution, is obviously
out of sync with headers.
