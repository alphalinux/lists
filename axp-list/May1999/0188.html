On 06-May-99 Shaw Terwilliger wrote:
> On Thu, May 06, 1999 at 07:15:01PM +0200, Kai Schmidt wrote:
>> I have a little problem with one of my multia's. I got the machine two weeks
>> ago, I installed the OS everything was ok, suddenly the machine was
>> dead, no
>> ping no telnet. After rebooting I was able to connect via telnet but after a
>> few keystrokes it turned dead again. Only this time a reboot didn't help.
> I've had zero problems since. The machine has been up for 7 days.
> I think it was the reseat of all the connectors (power especially)
> that did it. I should mention that I had this machine for a good
> 6 months before these problems, and not once did it not work as
> expected.

This is a well-known problem with Multias. It was almost certainly the power
connectors that were the problem. I've known folks who actually re-tinned the
fingers on the board with solder.
