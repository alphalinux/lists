On Thu, 20 May 1999, Jurgen Kramer wrote:

> I am having some difficulties with my Alpha (SX164) while running X.
>
> When I work at the console it works just great but as soon as I start X
> it locks up
> solid after a random time. The only thing that helps to unlock is
> pussing the reset button.
> Could this be a problem with my videocard? It's a S3 Virge/DX 4MB PCI.

I had similar problems with the same combination (SX164/S3 Virge/DX) when
I was using XF86_S3V server (the S3 Virge specific X server).

I since use the 'standard' SVGA server (SF86_SVGA) that seems to support
S3 Virge/DX quite nicely.

BTW, these lockups with XF86_S3V only happen on Alpha... i386 version
works nice.
