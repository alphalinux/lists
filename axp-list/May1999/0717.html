I just installed AW on RH 5.2. The only problem I've noticed so far is
that it won't launch from the menu on AnotherLevel. This is a minor
inconvenience. I haven't taken it out for a serious test drive, but the
word processor and spreadsheet work fine when building and printing some
simple test documents.

If you have printing problems, there's a section in the book "Red Hat
Linux 5.2 Unleashed" with a pretty good tip. I don't have the book here
or I would paraphrase it for you.

I had 0 problems with installation (on a PWS 500a).
