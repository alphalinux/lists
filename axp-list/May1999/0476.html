> My question is: how do I get the installation setup to use the "sym53c8xx"
> driver instead of the "ncr53c8xx" driver?

Why aren't you using the on-board SCSI? If you use that while
installing, then you can use whatever you want later in the game.

If you really want to use a different driver, you can build your own
kernel and use it to install. I'd bet that RedHat *still* isn't
distributing the config files they used to build the install kernels.
