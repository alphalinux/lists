Editing the passwd and group files by hand (Especially in a shadowed
environemnt with lots of users) is probably not a good idea. If you do
this you should always make a back up of both files before writing. I can
tell horror stories about such mistakes. If you still want to edit the
files by hand use vipw and vigr to edit. This ensures the proper locking
and helps prevent bad things from happening.

I would use the useradd script. This performs well and will prevent
catastrophic mistakes. You can change the script as you see fit for your
uses.
