Hi all,

To follow up on a previous post, has anyone else had problems with the xfs
included with RedHat/Alpha 6.0? By default, RH 6 now sets the X server up
in the font path, so that the font path is simply "unix/:-1". When we
start up an xterm (either displayed remotely from another client, or
running on the same machine as the display), xfs crashes, Enlightenment
freezes up, and xterm never starts.

If we change the font path to simply use manual paths rather than the font
server, all is well.

Is this just us, or can others confirm this situation?
