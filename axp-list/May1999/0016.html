On Mon, 3 May 1999, Nikolaos Ntarmos wrote:

> I wondered whether there is some kernel command line parameter for
> the cypress chipset. The generic ide support doesn't recognize this

http://www.dyer.vanderbilt.edu/server/udma/ - there is a patch, that adds
support for Cypress chipset. I use it with 2.2.5 kernel and I get 5.99
MB/s with a fairly modern Seagate ST34321A disk (I think, the disk could
go faster than that).
