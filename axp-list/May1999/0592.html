hi franco,

You have little time work to do. Epoch is 00:00 01/01/70. It's a fairly
standard UNIX thing. The linux time system call returns the time in seconds
since epoch, just as SGI does. There is a more general time_t structure in
the C library which should be fairly identical between sgi and linux.

see the strftime strptime C library call's for more info on how to convert
and format time.

regards,

paul.
