Jon 'maddog' Hall, USG Senior Leader wrote:
> jrc@art-render.com said:
> > Out of curiosity, will these compilers use Linux style ELF or retain
> > the ECOFF of Tru64?
>
> ELF. When we started the Alpha Linux project, I was fairly animate about
> having ELF shared libraries, rather than ECOFF. Intel Linux was ELF, and it
> seemed the way the world was going. I figured that we would get more debuggers
> and other types of tools if we could at least have the same symbol tables and
> linking information that Intel Linux had.

Yet more curiosity. Which debugging format(s) will be supported? Will
a debugger be included with the compilers?

> And (of course) when Richard Henderson gave the Alpha Linux project ELF
> shared libraries, I was elated......and it is something that I have never,
> ever forgotten. It is for this reason that Richard will never, ever have to
> pay for a beer in my presence.
