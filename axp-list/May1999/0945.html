the PWS 600au is part of the Miata family... hence you
will need the miata MILO if you are booting via
AlphaBIOS.

The basic sequence of thing is like this:

use AlphaBIOS to load MILO
use MILO to boot Linux/Alpha kernel
then... begin install or whatever.

I don't know if the "au" series has the ARC console
option.. I get the feeling not. There is plenty of
info on www.alphalinux.org.... I would read whatever
you can find there on the Miata first.
D.
