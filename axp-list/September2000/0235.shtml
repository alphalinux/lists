examples. They all exhibit the same behaviour (segfault on exit, but they

configs/linux-* files and remove '-O2' for Alpha. Seems that we run into
an optimiser bug when compiling. I'm *not* discounting the source,
though, WRT that bug. MANY times, I've found that there is a problem in
the code that causes gcc to exit with an ICE. I'm just trying to exhaust
all options before looking at the code (since it's quite a few lines to
