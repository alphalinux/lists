also in case the list cares, to set the boot device so that the machine will
actually boot, do this:

>>>set bootdef_dev dka0

(dka0 = the drive you want to boot)

Just in case the list cares,

to boot automatically in SRM, at the console type:

>>> set auto_action boot

to break out of it if you don't want to get back to the SRM console, press
<Ctrl>-<C> when SRM reads something like "Booting CPU 0..."
