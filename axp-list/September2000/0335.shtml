Eric,

Both projects are on SourceForge.

DRI: http://dri.sourceforge.net
Glide: http://glide.sourceforge.net

Both projects have pages on how to download, compile and install
the software. The glide cvs command is a little tricky as you
have to checkout a specific tagged branch of the tree. You
can do this with by adding -rGlide-64bit parameter to the glide
cvs checkout command. First, build and install glide. After
building glide you will need to build the X DRI server and
install that.
