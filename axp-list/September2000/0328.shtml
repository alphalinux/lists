Ok here's how to do it (it's not as easy on Linux as it is on solaris / DU)

first off, if you're running RHat 6.2 you need an updated autofs rpm
(the one I have here is autofs-4.0.0-pre7-11) because the one that came with
6.2 for alpha doesn't work

set the config file /etc/auto.master to have a line like this:

/home yp:auto.home

(/home is the directory you want to automount to, then the yp:auto.master is
the config file that's exported via NIS)

next you need to start the svc (this one is obvious but I'll mention it
anyways)

see if that's any better. I had it running at one point and I broke it
somehow (I know not what I did... but again this is RH7 beta 3 that I'm
running, so anything could have caused it)

lemmie know how you make out
