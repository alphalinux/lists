On 01-Sep-00 Jonas Gustavsson wrote:
> Hi people!
>
> I just have a quick Q for the Alpha Linux crowd here(myself included I guess):
>
> How many would be interested in commercial games for the AlphaLinux platform?
> (as in moving Intel Linux games over to Alpha Linux)

i am also very interested in games for linux/alpha.
that is the reason why i started to port the glide driver
(for voodoo cards from 3dfx) from linux/i386 to linux/alpha.

i would not buy shooters like quake, but i am sure, that many people would be interested
in it ... but only if you can say that the framerate on linux/alpha is better
then on linux/i386 ... that might be difficult (you cannot compare the
machines really).
i prefer games like myth, monkey island, ... games in which you have to think ;-)

it would be cool if - one day - you buy for example myth, and you get a
linux/i386 and a linux/alpha version - if a windows and mac version are on
the cd too, then it would be very very cool.
... but i think that there is a long way until that will happen.

maybe if the glide driver is working (the /dev/3dfx device driver is already working
- had to port that beast too!) it will be a tiny little step to the aims above.
