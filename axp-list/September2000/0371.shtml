A larger (than the prior version) performance kicker tarball is in
the process of migrating to our external web site. When it arrives,
it will be at:
        http://www.alpha-processor.com/products/downloads/developer_support/

And called perfpatch.tar.

As a patch, it would be huge, so I'm delivering it as a tarball.
I started with linux-2.4.0-test6; it covers
the following files:

                linux-2.4.0-test6/arch/alpha/lib/Makefile
                linux-2.4.0-test6/arch/alpha/lib/checksum.c
                linux-2.4.0-test6/arch/alpha/lib/clear_user.S
                linux-2.4.0-test6/arch/alpha/lib/copy_user.S
                linux-2.4.0-test6/arch/alpha/lib/csum_ipv6_magic.S
                linux-2.4.0-test6/arch/alpha/lib/csum_partial_copy.c
                linux-2.4.0-test6/arch/alpha/lib/divide.S
                linux-2.4.0-test6/arch/alpha/lib/memchr.S
                linux-2.4.0-test6/arch/alpha/lib/memcpy.c
                linux-2.4.0-test6/arch/alpha/lib/memset.S
                linux-2.4.0-test6/arch/alpha/lib/strchr.S

I've been running 2.4.0-test6 with these changes for a while now, so I'm
pretty confident in their correctness. I still have to tune the
remaining str*S routines, and then I'm going to look at the various
bits of __asm__ stuff in kernel includes.

        Enjoy,
