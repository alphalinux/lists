>Okay, seems the consensus is that this isn't really a steal-of-a-deal for

As the controller in the UDB is simple FAST SCSI, and is 50 pin (narrow)
interface, you would want to match that interface spec.
These are getting a bit scarce nowadays, as all the new stuff is usually
wide 68 pin, Ultra2 LVD or U3 LVD. Good news is that Ultra2 and 3 are
downward compatible.
However using a 68 pin drive on a 50 pin interface is not advisable, due to
termination issues that arise.
Some examples of available (new) 50 pin narrow SCSI drives:
IBM DNES-309170 9.1GB 7200rpm 1" for $265
Seagate ST39216N 9.2GB CUDA 18XL 7200RPM for $255
Add a case and cable, which generally runs for about $100

Hope this helps..
