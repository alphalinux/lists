jmd wrote:
>
> Hi all..
>
> A few months ago there were some people (Dr Gilbert mainly) on this list
> working on using the Promise Ultra 66 controller on Alphas. I was given one
> the other day and thought I might try to use it. I searched the archives and
> did not find anything conclusive as to whether it works or not and how well
> it works. Has anyone had any luck with these? What kernel was required? I
> tried putting in in my Cabriolet but it was not recognized at all in ARC.
> This is a single disk box so using one disk to boot and then initialize the
> Promise thru the kernel is not an option.
>
> The box i'd like to put it in is a PWS 500a. Alpha Bios 5.70 Any
> suggestions??
>

I run them in XP1000's and UP2K's but none of them are
bootable. Also, you need the IDE patches to get the
kernel to recognize them. Lastly, I found putting more
than one drive per channel is unstable, at least with
a patched 2.2.15 or so kernel.

Heck, you can buy a 60GB Maxtor for around $220 now and
an 80GB Maxtor for a little over $300. Prices so cheap it's
hard to see why you can't use a separate disk for it.
If you want real cheap, you can go with a 20GB drive for
around $85. I use the 20-30GB drives for the built-in IDE
to boot from, or, on the XP1000, leave the small SCSI drive
as the boot device.
