Doh! I thought about that after I went to bed last night... Anyhow, I
downloaded the needed files for the server and fired it up. It seems
to be pretty good. Nothing to kill, but I don't see any critical
problems. It does run faster if the X video mode is set to 800x600.
However, there's a small problem that it runs in a window and the server
sometimes put the window in a bad spot. The window can get paritally
off screen. And there's no mouse to move the window and the keyboard
gets tied to the game... You might be able to work around it if your
window manager lets you position the window before it starts.

Anyone remember how to bring up the frame rate counter in quake...?

Alan
