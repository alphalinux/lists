Go to

http://www.compaq.com/alphaserver/technology/index.html

Checkout the ES40, you can get it with 1-cpu and up to 32Gb of memory.

It's also sold as Linux ready but with a max of 2Gb, this is only because
the stock 2.2 kernels only support a max of 2gb without the large memory
patch or going to a 2.4 kernel..

For Linux ready goto

http://www.compaq.com/alphaserver/linux/

For Linux ready, they should let the order go through with 8Gb, But if
not,Just order the memory needed to populate the box with 8GB have 2Gb
installed in mfg and the rest as spare and install yourself. DO NOT
ordered the Linux ready and just order spare memory. You may not have slots
to install it.. depends on what they plug in to make the 2gb.
