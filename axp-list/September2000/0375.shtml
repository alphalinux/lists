I'd recommend OpenSSH from the OpenBSD folks. You can find it at
http://www.openssh.com . It has a real Open-Source license and
the latest versions support both the ssh1 and ssh2 protocols.
However you'll also need the OpenSSL package
(http://www.openssl.org)
