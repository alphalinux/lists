On Thu, 21 Sep 2000, Ivy Wang wrote:

: what does "cat uses obsolete /proc/pci interface" mean? Thanks.
:
It's a warning that /proc/pci is going to disappear in some future kernel
version. You can ignore it for now. IIRC, /proc/pci is replaced with
/proc/bus/pci/* which is not really human readable. Check out the lspci
command from the pciutils package.

Reinier.
