Yes, -O2 makes a difference:

popov-289% cat > denormtest.c
#include <stdio.h>

int main()
{
  double x = 1.0;
  for (x = 1.0; x != 0.0; x /= 2)
    printf("%e\n", x);
  return 0;
}
popov-290% gcc -Wall -mieee -O2 -o denormtest denormtest.c
popov-291% ./denormtest | tail

this gets rid of the problem :-).
