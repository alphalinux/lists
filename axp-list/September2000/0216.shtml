Too bad you know so little about it. Possibly next you
might also improve your vocabulary since it reveals
quite a bit about your lack of imagination or rational
argument. Rather than rely on the work of others in
news groups, the place of lowest signal to noise
ratio in the universe, I made my own measurments.

The seti@home client for the alpha's is optimized code
that is compiled using the the compaq ccc compiler.
The comparison between athlon and pentium is done
using the same client. I am simply using the selfsame
numbers published by the nongraphics clients. The
computations are computing number of ticks per flop.
Because of the large amount of memory (the entire
data set is stored in memory and is always being
chirped and FFT'd), the L2 write through performance,
interrupted frequently for disk writes, is put to the
serious test. The alpha whips both the athlon
and the pentium in these comparisons but the pentium-ii
beats the athlon . The pentium-iii beats the both and
the pentium-iii xeon is the big winner as should be expected.

Bob
