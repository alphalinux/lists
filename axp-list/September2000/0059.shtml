Nikolaos Margaritis wrote:
>
> On Mon, 04 Sep 2000, you wrote:
> > > Does anyone know which system type I should choose when I configure the kernel
> > > for a UP2000? I need both SRM and SMP enabled.
> > >
> >
> > Generic.
> >
> Would this not inhibit performance?

No. Your kernel will include some unused code to perform setups
on other platforms.

Besides if you worry about such details you should not be touching test
kernels with a ten foot pole. These are NOT production kernels in your
wildest imagination. Garbled files, for example, are much bigger
performance impediment and 2.4.0-test8-pre2, for example, does that.
If, OTOH, you would like to contribute (even just by testing) to a
kernel development then, by all means, go ahead but inform yourself
beforehand what you are doing and keep good backups.
