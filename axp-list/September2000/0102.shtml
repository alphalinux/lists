Well, it's the same as a UP1000 (Nautilus system type), the only catch is
some dists don't know the Nautilus system time so you have to make a
symlink in /etc (RH 6.2 is fine, Debian Potato needs the link).

ln -s Tsunami /etc/alpha_systype

this is for Xwin, hwclock etc.... The onboard NIC works with the tulip
driver just fine, the sound will work with the Trident drive in 2.2.16 and
later (does require a small patch that's be put into 2.2.18pre1). We've
had the IDE going at Ultra66 w/o to much trouble....
