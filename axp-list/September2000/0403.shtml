> >Can anyone tell me how I can set the system date and time on 
> my PWS 433au?

> temporarily, but when
> >I reboot it goes back to thinking that it's October 2068...
> 
> 
> I've got a PWS433a and asked the same question ;^)
> 
> The "date" command sets the *software* clock but doesn't change the 
> *hardware* clock. The "hwclock" command sets the hardware 
> clock. Run "man 
> hwclock" for details. Once the hwclock is set, you should be 
> fine when 
> rebooting.
