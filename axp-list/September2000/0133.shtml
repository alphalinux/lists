The stock kernel has a bug, you need a kernel from

http://ftp.digital.com/pub/Digital/Linux-Alpha/TESTING/

The up-2.2.16-16jun kernel should work fine. Boot a floppy with the kernel
image, and set the root to /the CD..

example: (device names may be wrong for your boot)

boot dva0 -fi vmlinux.gz -fl "root=/dev/scd0"
vmlinux.gz=the kernel name - you can rename the kernel you copy or enter the
actual name..
