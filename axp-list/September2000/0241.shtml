On Sun, 10 Sep 2000, you wrote:
> What is the state of the JDK for AlphaLinux, does it work? is it stable?

Java 1.1 and 1.2 are available from ftp://ftp.alphalinux.org/pub/java/ . Done
by Uncle George http://www.voicenet.com/~gatgul/Java.html and they require
lesstif (or openmotif) . I've used 1.2 for a few applets and it worked well
for me. I've never used on a 'production' level like as part of jserv or
something. kaffe works though I don't know how well.

> I'm planning on setting up a personal web/data server (apache/oracle

> (SUSE/Debian/RedHat). If I'm doing any work with Java should I get a copy
> of Tru64 and run that instead (I'm running an XL266 with 96M RAM).

It shouldnt matter which distro you happen to use. I use RedHat, no problems
here. Tru64 on that sort of machine won't be the fastest thing in the world
though not what I would call the fastest thing in the world.
