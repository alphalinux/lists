Alvin Starr wrote:

> The Alpha and othre RISC CPU's trade off instruction complexity and code
> size against instruction speed. In the xx064 the cpu would execute an
> instruction on each clock cycle so that MHZ == MIPS. In the xx164 the
> number of instruction was pushed to 2 and with the latest incarnation the
> xx264 MHZ == 4MIPS. So an 600MHZ Alpha will actually run at 2400MIPS.
>

Just to clarify further...The 21264 is capable of quad-issue, but does not
necessarily always _have_ to issue four instructions. Things like ICache misses,
branch mispredictions, etc. can all contribute to a less than ideal instruction
stream even for the 21264.
