On Sat, 16 Sep 2000, Alexej Chupahin wrote:

> How do you do!
> May I ask anybody?
> 1. Is anywhere a Xserver for QVision?
No, there is not. XFree86 4 promised to incorporate jensen support in the
svga server, but it has not been done yet (at least not in 4.0). If
started in a jensen it says if you want jensen support, plaese contact the
developers... (I am a lousy coder, so I did not pick up that challenge)
> 2. If still no, can anybody send me old S3-based card worked in DECpc Jenen
> and supported by Xfree ?
I can tell you that the ELSA Winner 1000 works (at least up to ROM
version 1.7 of the Jensen), but unfortunately I can not send you my card
(it is the only one I have). This card is supported by the S3 server in
XFree86 3.3.5. In 3.3.6 the server binary XF_S3 (or something
similar) will hang the card for good, only a power down will reset the
card!. It seems possible to use the XF_S3 from 3.3.5 with the rest of
3.3.6 though.
> 3. Does anybody knows a quake version that doesn't require Xwindow?
Sorry no.
