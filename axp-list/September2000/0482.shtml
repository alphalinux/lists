Here is a .pl script that was written for use here in the shop. Use it,
alter it, do what you like. If you execute it, it should make all the
changes necessary for you.

The only thing you will need is to add the following command in SRM:

set boot_osflags "root=/dev/sdx console=ttyS0"

where x is your partition. You have to have the "" or SRM will take
everything up to the first space. Hope this helps you out.
