I've had this problem before. Use the failsafe boot option on the motherboard
and make sure the alphabios floppy is in the drive. From there you should be
able to use the upgrade menu selection to switch from srm to alphabios more
permanently.
In general, I've been frustrated at the flakiness of srm trying to boot linux
on this machine.
