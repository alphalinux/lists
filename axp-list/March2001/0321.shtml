Hi!

I have a problem with a recently purchased Alphastation 250 4/266:
Every morning when I try to boot the machine, I get beep code 1-2-1 and LED
code D6, which I translate to "RTC interrupt did not occur".

The only work-around seems to be to unplug the system, remove the battery
from the mainboard and wait a reasonable amount of time. After reinstalling
the battery and replugging power, the machine runs fine except for the lost
boot defaults.

Has anyone experiences something similar or even found a better solution
than inserting a switch in the battery circuit?
