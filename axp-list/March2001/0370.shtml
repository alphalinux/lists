On Wed, Mar 28, 2001 at 09:34:32AM +0900, J. Hart wrote:
>
> I have finally been able to successfully install RedHat 6.2 on
> an AlphaPC > 164 that I rescued from the trash....:-) My Alpha
> uses a S3 (Virge ?) Trio 64 graphics card with what seems to
> be 2 mb of video ram.

Some of the cards had sockets for a second 2Mb, but few were ever filled.

> I have set my X configuration for 1024x768 resolution. I have
> noticed that X is very stable at 8 bit planes, but very
> unstable at 16. At 16 bit planes, it will lock up when the
> mouse is moved. It takes about one minute for this to
> happen. Is there a fix somewhere for this which allows
> operation at 16 bit planes under 6.2 ?

Try adding:

        Option "no_pci_disconnect"

to the Device Section of your XF86Config file.
