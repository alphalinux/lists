I've finally set up a multia running RedHat 6.2 (gave up trying to use SRM
and reverted to ARC). Now I'm trying to get the IDE drive working a little
faster. I've played with hdparm and following something I read tried

    hdparm -c 3 -m 16 /dev/hda

but don't seem to see any performance improvement. In particular after
running this command and re-entering

    hdparm /dev/hda

I still seem to be doing 16 bit I/O.

Has anyone had any success in getting more than 1.3Mb/s out of an IDE drive?
