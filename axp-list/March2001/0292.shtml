Hi,

Like already seen in the subject line, the problem is that netscape runs
only for root. I have checked the permissions of all files in the netscape
package (4.7-3) and they're OK . If a user tries to run netscape, then it
gets a complain about a missing font and nothing more. No window, nothing.
The CPU time is next to null. I tried to strace the executable and it
seems like it blocks in some osf_gettimeofday calls.

If, as user, I try sudo netscape, then I get a window, but nothing
more. If a click the Home icon it hangs up in "Looking up host ...". Stop,
Esc, nothing helps anymore.
