Thanks for all help, but i found the problem with caused this error..
The machine did not recognize the CDromplayer as Master which gave it dqb1,, if you don´t have anything on dqb0,, the system sures tells you in different ways ;).
When i tried to make the system to know the cdromplayer as Master it got dqb0, which solved the problem at once..

I then tried to use Alpha Jumpstart CD and my own Redhat CD-R and installed Redhat without any problems..
