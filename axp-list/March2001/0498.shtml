Someone else posted this cookbook fix for the ftp passive mode problem
in alpha rh 7.0. Here it is again:

# fix ftp requiring passive mode transfers

rpm --recompile ftp-0.17-6.src.rpm
rpm --erase ftp
rpm -Uvh /usr/src/redhat/RPMS/alpha/ftp-0.17-6.alpha.rpm

rpm --recompile wu-ftpd-2.6.1-6.src.rpm
rpm --erase --nodeps wu-ftpd
rpm -Uvh /usr/src/redhat/RPMS/alpha/wu-ftpd-2.6.1-6.alpha.rpm
