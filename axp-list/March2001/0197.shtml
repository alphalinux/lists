PWS's which shipped in Nov-Dec 97, did not have SRM, This was because
testing was not completed yet for the SRM/Unix version of the product.
However when the updated firmware was available in very-early 98/very-late
97, all PWS's could receive the update and have both consoles present. The
only difference between the "a" and "au" are the options offered. an "a"
could be ordered with options only supported by NT. an "au" could be
ordered only with options that would run on all three OS's (NT, Unix and
VMS)
