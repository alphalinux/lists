On Sun, Mar 11, 2001 at 11:03:01AM -0500, Eric Christy wrote:
>
> I recently did a clean install of RH7 on my LX164 box with a
> Voodoo3 card and I am trying to get DRI going. I downloaded
> XF4.0.2 Alpha binaries with the understanding that these were
> built with DRI. I tried testing this with glquake from quakeforge
> but it always starts in indirect mode. Any ideas?

IIRC, various releases of X from RH didn't include the tdfx_dri.so
module which makes direct rendering possible.

If you decide to build it yourself, you need to activate both
currently commented-out Glide3-related #defines from xf86site.def
when creating your host.def.
