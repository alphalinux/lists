gshamov wrote:

>Is there a way to reinstall Linux (i have RedHat 6.2 distribution) on
>DS10L machine with broken CD-ROM drive?
>
>May be, it is possible via network installation from another DS10L
>machine? If so, how can i do it?

If you have another identical machine, I would say the easiest way is
to install the linux onto the other machine, and clone its disk, and
put it into the machine with the bad CD-ROM. Of course, you should
correct the individual node's information after you've cloned the
disk. In most UNIX systems, the easiest way is to use dd.
