There was an unfinished patch to Linux 2.2.16 that created a /proc/srm
directory, and you could modify various SRM settings (os_type, bootdef_dev,
etc) that way. However I've since lost the patch. It's in the kernel on the
JumpStart! CD however, so you could use that. If you want, I'll make the
kernels available separate from the CD.

I'm in the process of reconstructing the patch so I can finish it, probably
rewritten as a kernel module. First I have to finish JumpStart 1.4 and go on
vacation though.
