An excellent suggestions at that. FWIW, most of the testing I've been
doing has centred around installs, so I can tell you right now that if you
burn a CD, you can't use it as the root fs for install, you'll need to
make the ramdisk floppy (which does work). If you're installing over the
network use either ftp or nfs installs.

I've done some testing with GNOME, though by no means every application.
Haven't even tried KDE.
