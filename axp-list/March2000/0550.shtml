On Sun, Mar 26, 2000 at 09:00:43PM -0800, Stephen Williams wrote:
> Does you really mean that the *object file* is limited to 64K, or that the
> .got subsection for each object file is limited to 64K. You must mean the
> .got subsection is limited.

Yes of course the later.

> What things go into the .got subsection? External symbols?

Currently any symbol. Though static symbols wouldn't have to
if I got off my ass and fixed gas then gcc.

> If I make some of the symbols static, will that keep them out of
> the .got subsection? As I said, if this is the case, I can arrange
> for my symbols to be made local to the file.

Can you arrange to put them in a structure? E.g.

        static struct
        {
          int foo, bar, baz;
        } G;

Now you've got one symbol instead of three. Gcc can even generate
a bit better code with input like that.
