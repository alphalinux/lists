> Is this a bug in the linker? Smaller programs link without problems.

Looks like you exceeded ld's guess at how many program headers
your code would need. Do a "info ld", go to the index, and look
at the ELF program header section and it talks about PHDRS and how
to change them. Must be an unusual piece of code.
