I could successfully install RedHat6.1 on AS200 except x-windows
configuration setup. After installation finished without x configuration, I
booted from "/boot" partitioned drive (in my case sda3). But a error
message showed up as follows during booting procedure even adding text flag
with the command. I don't know how to pass "init" file on booting.

kernel panic: No init found. Try passing init=option to kernel.

I think the point of this problem is x-windows configuration. Linux
installer indicated X probe results "Video card: DEC chip 21030[TGA] X
server: TGA". On the x-windows configuration process, I tried almost all
combinations of "Monitor" (I have a DEC VRC21-Hx 21" monitor, H-Frequency:
77.17KHz V-Frequency: 72.56Hz, Maximum Resolution: Up to 1280x1024), "video
memory" without "clock chip configuration". But the configuration procedure
failed. I couldn't find ICS1562 programmable clock chip which is installed
in AS200 in the list of clock chip selection menu.

Does anyone know how to fix the problem?
