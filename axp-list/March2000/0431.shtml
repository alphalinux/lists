On Fri, 17 Mar 2000, Ian Davidson wrote:

> Is anyone successfully using the dump & restore utilities from the RH6.1 CD.
> I am having serious problems trying to use these programs. If I install
> everything from the CD I can backup the / partition to an Exabyte 8505-XL
> that is on the system. If I try to then append the /usr partition to the
> tape the dump program crashes. If I try to restore a file from the first
> dump file on the tape it always fails with a segmentation fault.
>
> Can anyone recommend a reliable backup/restore package to use on RH6.1.

The version of dump/restore in 6.1 is a bit on the older side.
Unfortunately when I entered a bug report about it in bugzilla.redhat.com
and mentioned that my particular problem was fixed in the most recent
version of the package (http://dump.sourceforge.net), the followup from
Red Hat indicated that a more recent version would be included in 6.2--but
implied that no update would be released.

The current version of dump/restore is 0.4b16 at the above url. I
currently use it without any problems on my as200.
