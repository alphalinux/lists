One thing which I think could be very interesting to increase the user base
is an alpha/Linux port of MATLAB, a script language very widely used
for numerical and visualization tasks. This could work as a gateway for
people which are interested in alpha FP performance but not
so much in alpha programming. (Does anybody know if the True64 Matlab runs
under Linux ?)

As Matlab / Linux-x86 and Matlab / True64 already exists,
perhaps Mathworks is a bit susceptible to this, especially
under the aspect that Numerical Python in some time could
become serious competition as visualization capabilities are
increasing quickly ...
