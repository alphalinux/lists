Fatal server error:
xf86EnableIOPorts: Failed to set IOPL for I/O

(rooted on hdd3),

and three different kernels, 2.2.14, 2.2.15, and 2.3.99pre3.

boot Suse 6.3 (rooted on hda5).

Has anyone else seen this, and is there a fix?
