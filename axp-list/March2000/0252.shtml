  I have placed a set of prebuild Linux/Alpha XFree 4.0 binaries on

ftp://zenii.linux.org.uk/pub/people/treblig/XFree4.0-linuxalpha

These were built on my LX164 under RedHat 6.1 and are provided in bzip'd
tar format - so there is a fair chance of them working on other
distributions as well. I built them under Linux kernel 2.3.49.

Please feel to download these and tell me how you go on.
