OK Here's my MySQL story =)
went to the MySQL site and downloaded the latest stable version of it.
Compiled it and tried to run it.
The compiling went fine, BUT, as soon as I tried to run it, it would
SEGfault. Went back, found an alpha RPM and installed that. No problems.
might be 64bit issues (again) w/ MySQL on alpha

PHP:
downloaded PHP3, compiled it and installed it, no problems... BUT it didn't
work. Turns out RedHat only compiles the apache moduels for PostGresSQL
(which comes w/ RH6.1). Recompiling it as apache modules and re-installing
did the trick.

This is on an AS3300 w/ RedHat 6.1 (w/ latest RPM unpdates and kernel
2.2.14)
