Here's a patch against 2.3.99-pre2-pre2 + stuff. It should
apply with fuzz to any kernel in the general area. I'll work
on a version for 2.2.15 this weekend.

Try it out like

        unsigned long xbus_base(int bus, int dfn)
        {
          return syscall(376, 2, bus, dfn);
        }

        int main(int argc, char **argv)
        {
          unsigned long x = xbus_base(atoi(argv[1]), atoi(argv[2]));
          
          if (x == -1UL)
            perror("bus_base");
          else
            printf("bus_base %lx\n", x);
          return 0;
        }

00:08.0 Ethernet controller: Lite-On Communications Inc LNE100TX (rev 20)
01:08.0 SCSI storage controller: Adaptec AIC-7881U (rev 01)
[fidel:~] ./a.out 0 64
bus_base 80000000000
[fidel:~] ./a.out 1 64
bus_base 80200000000
[fidel:~] ./a.out 1 65
bus_base: No such device
