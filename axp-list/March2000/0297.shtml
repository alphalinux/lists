> >Check to see if the scsi bus is properly terminated.
>
> 1. The scsi bus 0 is properly terminated because there is a terminator at
> the end of the scsi cable.

This is neither necessary nor sufficient. You also need to check that (1)
the drive doesn't have on-drive termination enabled, and (2) that you have
an active terminator, if needed. Termination problems can be extremely
subtle; sometimes they only show up with particular usage patterns.
