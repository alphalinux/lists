Hi

I have an LX164 with 2 Matrox Millenium 2's and I'm trying to get
multihead working.

I am using kernel 2.2.12 and I have compiled matroxfb support into the
kernel and am sending the kernel "video=matrox:vesa:0x108" as a parameter.
It works fine if there is only 1 card in the machine, but when the other
card is in it stops booting as soon as Milo passes control to the kernel
(this is the point where it shows the Matrox BIOS screen again with only
one card in the machine) The screen blanks and the machine halts at this
point - no more noises from the hard drive, and it never shows up on the
network.

Is it possible to get this working with this particular setup? If so, can
somebody help me out here?

It's not so critical to have kernel-level multihead working, but I need to
be able to use Xinerama with XFree86 4, and at the moment I can't even
boot up the machine if there is more than one video card in it.
