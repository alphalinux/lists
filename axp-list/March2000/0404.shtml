Hi,
I am successfully running 2.3.99-pre2 on a DS10.
To get it to compile remove the madvise syscall in
arch/alpha/kernel/osf_sys.c
and copy the following defines into include/asm-alpha/mman.h

#define MADV_NORMAL 0x0 /* default page-in behavior */
#define MADV_RANDOM 0x1 /* page-in minimum required */
#define MADV_SEQUENTIAL 0x2 /* read-ahead
aggressively */
#define MADV_WILLNEED 0x3 /* pre-fault pages */
#define MADV_DONTNEED 0x4 /* discard these pages */
#define MADV_FREE 0x5 /* (Solaris) contents can be
freed */

No guarantee these changes are correct of course.
