hello, Gary

> boot dva0 -file vmlinux.gz
  ...
> Kernal panic: VFS: Unable to mount root fs on 08:02
> ... and it just hangs here
  depends on which partition you made root (/) during the install
  the above happens if you specify a partition that is not actually the
root partition

  try boot dva0 -fi vmlinux.gz -flag root=/dev/XXaN
  where XXaN is dva0, hda0, hda1, sda0, sda1 -- wherever you actually
loaded /
  or dva0 if you are at the pre-install phase

  hope this helps,
