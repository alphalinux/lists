The multiplier between crystal and CPU-speed is fixed on the PC164. If the
crystal is 36.6 the CPU will always run at 366MHz. The jumpers are used to
set the divisor from CPU-speed to 'front end bus' (memory, PCI). If you've
tried jumpersettings foor a 500MHz setup, you've likely running those parts
at too low a speed. This is usually less dangerous than too high, but can
still cause problems
