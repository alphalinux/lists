On Wed, 22 Mar 2000, Stephen Degler wrote:

> Hello,
>
> I have a miata and I wish to test a 2.3.xx kernel. What version + patches
> should I be working with?

In general use the latest available version in
ftp.??.kernel.org/pub/linux/kernel/v2.3 - all the versions are now
2.3.99-pre? where ? is about 3 now I think. If you want even less
stability you can then apply the patches on top of that from
/pub/linux/kernel/testing - but I wouldn't unless you really feel
adventurous.

> One problem I have is that I haven't been able to get a voodoo 3000 pci
> running under X. Others have reported success with other alpha models, and I
> wondered if this wasn't some pyxis issue. An elsa gloria synergy works fine.
> I tried using both the pre 4.0 stuff and the server distributed with piglet.

Do you have another video card in the machine (on board?) - if so that
could be causing some problems. To diagnose this I need you to try to
startx with the voodoo in and send em a copy of /var/log/XFree86.0.log
where all the important messages go. Also send me a copy of /proc/pci.
