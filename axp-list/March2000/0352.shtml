Hi,

Try the following boot_osflags:

set boot_osflags "console=ttyS0,9600 serial root=/dev/fd0 prompt_ramdisk=1
load_ramdisk=1"
I used this to install a DS20 via a serial connection.
Also make sure the kernel you are booting has serial console support.

Hope this helps!
