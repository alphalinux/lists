I'm having a problem installing Redhat 6.1 on a UP2000 system.

During the execution of linuxrc you're prompted for the
language -- I chose English. Then you're prompted for the
device type to install from {ftp, nfs, disk, cdrom, etc}. I
choose 'disk'. I'm told my "system apparently has no disk drives".
If you try the option to configure the disk drives, you are
asked "Which driver should I try" and are presented with an
empty list. However, on the <ALT>-<F4> screen you can see that
the kernel's boot time messages found the two disk drives
on the Symbios 53c895 controller.

This machine appears to work fine with the RedHat 5.2 that came
loaded on it.

Are there any obvious things to try?
