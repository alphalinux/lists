Just a comment. I installed a UP2000 with SRM and RH6.1
last week and it worked fine. Even did an NFS based install
because the system doesn't have a cdrom. Try new floppies
and see if that helps. Floppies are quite flakey but once
installed, you shouldn't need them much. (unless you have
no network card)

The above is what I booted with.
