I've got a DS10 I'm trying to install 6.1 on. I am planning on doing an
ftp install.
This is what I've done so far:

1.) downloaded iso image and mounted it on an intel Redhat box

2.) made two disk using:
dd if=generic.img of=/dev/fd0 bs=1440k
dd if=ramdisk.img of=/dev/fd0 bs=1440k
from the iso image

3.) set boot params:
set bootdef_dev dva0
set boot_osflags "root=/dev/fd0 prompt_ramdisk=1 load_ramdisk=1"
set boot_file vmlinux.gz

4.) booted the box

The box goes through the loading of vmlinux.gz and then says it is booting
it, but never gets past that.

Is there something obvious I'm missing? The box doesn't have a video card
and I'm using the seriel console. I've downloaded and looked at the guide
Compaq made, but it doesn't say much about booting from floppys.
