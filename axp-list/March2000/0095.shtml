On Sat, Mar 04, 2000 at 10:16:07AM -0500, Steve Craft wrote:
>
> Does anyone out there use qmail? Has anyone succesfully compiled it? I
> looked all over for an AXP binary RPM and saw none so I guess I'm going to
> have to do it "the hard way" but I wanted to know if I am even wasting my
> time and should go back to sendmail.... TIA.

I use qmail on Alpha and intel, and would never go back to sendmail.
It is very fast and lightweight, but the configuration is a little
different to most things you've probably used. Its designed with
security from the ground up, and the code is very readable and small.

The "Maildir" format of the mailboxes is very clever, and allows it to
work seemlessly over nfs etc and handle multiple parallell deliveries
without any locking.

Compilation is easy, if you just read the documentation that comes
with the package. OTOH, if you're expecting something that's like
sendmail, you will be surprised.
