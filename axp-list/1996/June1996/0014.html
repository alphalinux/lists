Red Hat Linux/AXP 3.0.3 Now Available via FTP
============================================

Red Hat Software is pleased to announce we've made Red Hat Linux/AXP
release 3.0.3 available for FTP from:

ftp://ftp.redhat.com/pub/redhat-3.0.3/axp

This release is distributed under the terms of the GPL and we
encourage other CD-ROM vendors to include it in their products!

With Red Hat's ease of installation, system management, automatic
upgrades, and preconfigured tools, Red Hat Linux is easier to use
and manage than ever.

New Features
------------

. RPM 2.0: A NEW VERSION OF RPM
Provides many new features and dramatically faster performance:
- complete C implementation (no longer requires perl)
- over 10 times faster
- RPMs are PGP signed, for authentication and security
- backwards compatible with RPM 1.x

. EVEN EASIER INSTALLATION
- Fine grained package selection in text install (with package
descriptions)
- Disk space required now calculated in text install during
package selection (to help avoid selecting more packages
than you have disk space to accommodate)
- You can boot from the CD (using MILO) and install without
touching a floppy disk (on most Alpha hardware).

. XFREE86 (X11R6) 3.1.2 X SERVER
- Red Hat Software continues to support the XFree86 X server
as being a critically important piece of the freely
distributable Linux OS. Among other important features
XFree86 gives you the ability to install Red Hat on as many
machines as you chose without additional cost or restriction.

. LOTS OF UPDATED PACKAGES
Including:
- Apache WWW server (1.0.3)
- DNS Server (bind-4.9.3)
- libc 0.43
- gdb 4.15
- HOW-TO's and LDP Documentation
- kernel 1.3.90 with patches to allow one kernel to work on
TGA and VGA machines
- many security fixes

. 50 NEW PACKAGES INCLUDED AS PART OF RED HAT LINUX
From X tools to games for a total of over 380 packages:
- Multimedia programs (playmidi, aumix, tracker, etc...)
- popclient
- mawk
- and many more...

. EASE OF UPGRADING
- Red Hat's RPM packaging technology allows for trouble
free upgrades. We have tested 3.0.3's upgradeability
against Red Hat Linux/AXP 2.1.
