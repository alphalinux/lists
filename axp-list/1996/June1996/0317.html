Hello:

I am new to the list. I've got an Alpha XL 233, with 32Mb of RAM, a 2 Gb
drive, CD-ROM, and a SCSI bus. The machine was purchased with intentions
as a Windows NT file & print server.

With much prodding, I was able to convince the "powers that be" at my
organization to let me try out Linux on Alpha, so that I may attempt to
prove, over the summer, that it is an *free*, attractive alternative to
NT. We have been using Linux on an Intel box as an Internet server since
October, and I must say it's worked splendidly.

The purpose of my test is to demonstrate the Alpha as:

- a file & print server using SAMBA.
- a network FAX server
- an Internet server

All tasks should be ideally suited to Linux.

I have the go-ahead for this for the beginning of July, but I have a few
questions before I begin. The most worriesome to me is that the machine
comes with a Matrox Millenium card, and I have since heard that Matrox is
unwilling to provide programming information for the card without a
non-disclosure agreement. I had intentions of running X, to show that
Linux can be just as "user friendly" as the next GUI.

What are my options as far as the video card goes? As long as I can run
the card in 256 colour mode at least 640x480 it shouldn't be a problem.
BTW...exchanging the card, nor buying a commercial X-server are *NOT*
options -- heck, I can't even convince my organization to spring the $100
on the Red Hat CD. :-( As I said, I am going to prove Linux's worth as a
*free* alternative.

I want to install the Red Hat 3.0.3 release from FTP. From what little I
know about Alpha Linux, I create some kind of boot floppy, and MILO
floppies, and then from there I can perform the entire installation via
FTP. If so, this is ideal!

I have visited the Alpha Linux home page at
http://www.azstarnet.com/~axplinux/FAQ.html but it seems to be a little
out-of-date as far as specific information on Red Hat goes.

Any advice would be greatly appreciated. I'm really looking forward to
trying out Linux on this beast. Actually, it's keeping me awake at
night. :-)

BTW...any chance the JE extensions are included in the Red Hat release?
