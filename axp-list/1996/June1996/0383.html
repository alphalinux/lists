This program is using intel specific assembly instructions. You can't use
this on non-x86 without changing the program to eliminate this (or adding
equivalent AXP asm instructions).
The best thing to do IMHO is contact the maintainer for this program and
ask him if he can do away with it in a future version or (even better) do
it and send him the patches.
