Traditionally, Linux NFS performance hasn't been all that great.
Other than that, the Linux/Alpha NFS server works fine (at least
AFAIK---if you know of any bugs, please let me know). We use NFS to
access four Cabriolets in the department and at home we use it all the
time as well. So, for moderate uses, it certainly does the job. I
don't know how it would keep up with high loads though. Olaf Kirsch
has an experimental kernel nfs daemon that is supposed to be able to
saturate a 10Mbps Ethernet. I'm downloading it as I write this. If
you're interested, I'll post some info on how knfsd is doing on the
Alpha.
