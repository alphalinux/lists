Lance Ware <lware@voxel.com> asks:
> do you know an easy way to ftp all those recursive directories

Aside from Erik's suggestions, there are three other ways I know of:

1) Create empty directories by hand, matching the hierarchy on the server.
Then an "mget *" will actually get everything.

2) Use the mirror package (written in Perl). Be forewarned that by default
it doesn't preserve the file permissions. I used this approach, and it
took me an hour or so to figure out everything I needed to set execute
permissions on in order to get my NFS installation to succeed.

3) Use ncftp. I haven't tried this one.

Cheers,
Eric
