> I really need help with this, how do I install a new kernel on a jensen?
> I've rebuilt the kernel but I don't know how to mkae the machine boot to it.

Get aboot installed. From the SRM console, do a
boot -fl "i"
and type
3/vmlinux-new.gz
where 3 is the partition number of your root fs.

The kernel is built by doing a
make dep; make clean
make
quickstrip vmlinux
gzip vmlinux
mv vmlinux.gz /vmlinux-new.gz

If you don't have aboot, but have a second SCSI disk with space free
at the beginning, look in /usr/src/linux/arch/alpha/Makefile how to build
a SRM bootable image.
