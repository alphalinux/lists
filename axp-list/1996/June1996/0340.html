Sorry to have the reply at the top but I am at work using a terminal
that is brain damaged.

I bought my UDB from DCG Computers and it came with a very comprehensive
manual including full info on all jumpers as well as the answer to the
question below about the serial port. It is true that you should be
able to access the first port using a normal cable if you are using
async devices but when I hook it up to my Bitsurfrusing a 25-pin cable
some of the other lines hung up the adapter. I built a "Y" cable
using the information in the manual (DEC will sell you one for about
$75 :-)) and everything is working OK with both ports.

The manual also listed DEC part numbers for just about every
component as well as detailed procedures for field-replacing
everything from the fan to the mainboard.
