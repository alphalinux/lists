Would somebody be willing to sponsor some diskspace for me? In the
past couple of weeks, I already bought a 1GB drive and more memory,
and this is starting to get expensive for me. I'd need another 1GB
drive (the bigger, the better, as usual) to be able to make a smooth
transition from ECOFF/oldlibc to ELF/glibc. I also started to work on
fixing the Linux kernel so that it is able to support files >2GB.

have a disk with a huge partition.

Not to give any false impression: glibc/ELF is going to happen soon
whether there is a new disk or not, but it would make life quite a bit
easier with some extra space and it certainly would be greatly
appreciated.
