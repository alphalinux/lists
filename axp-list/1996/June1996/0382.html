While we are at that topic:

the NFS installation script is a bit annoying - ie., all defaults may be right
for redhat.com, but not for other domains. I mean, I have to type in *every*
network/broadcast/... address, since all defaults are for a class B network,
but I (and I suspect most users) am on a class C subnet.

Furthermore, If I make a typo, I have to re-type *everything*, not just
the wrong line.

This is no big problem, but it's a bit annoying.

Apart from that, 3.03 runs absolutely fine now on my Jensen - I mounted my home
directories, installed the Jensen X server and my local null-client sendmail.cf,
and I can really work on the machine, just like it is supposed to be.
