Can anyone explain why I should need to explicitly specify extern
strlen() after including <string.h> for the following simple example?

fresco@banshee:/tmp $ cat a.c
#include <string.h>

/*extern size_t strlen(const char *ptr);*/

size_t z(const char *y)
{
return strlen(y);
}
fresco@banshee:/tmp $ cc a.c
a.c: In function `z':
a.c:7: warning: type mismatch in implicit declaration for built-in function `strlen'
