A patch for zftape-1.05 is now at:

ftp://ftp.azstarnet.com/pub/linux/axp/misc/zftape-1.05axp.patch.gz

This patch will make zftape compile for the Alpha and with recent
kernels. I'm using zftape together with taper-6.6 and they work
pretty well together. I'm not sure yet I'd trust them for all backup
purposes, but if you are not a fan of tar and/or cpio, you may want to
give taper a try.
