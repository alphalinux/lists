Yes, InfoMagic shipped bad disks. All hope is not lost!

One work-around is to go into the second virtual console, mount the CD
manually, and then from "install options" select a mounted hard disk
partition and give the location of the source RPMs on the CD-ROM.

Hope this helps. (I think this works... someone else discussed it)
