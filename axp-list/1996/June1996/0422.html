> I cannot access a Exabyte 8200 on a UDB (Multia) running 3.0.3 and kernel
> 1.3.89. The kernel seems to recognize the device :

Your SCSI card is seeing the device, but the kernel doesn't have
SCSI_TAPE compiled in. There is no code to handle the tape drive
in your kernel...you need to rebuild with that option enabled.
