The Multia/UDB is certainly a great deal for the money, as long as you don't
expect that just because it's an Alpha it will blow the doors off everything
else. I know several people who have been disappointed in the performance,
but what can you expect out of an LCA (Low Cost Alpha).

I have two 233 MHz Multias and am quite happy with them. I use one for my
web server (http://www.brouhaha.com/).

If you get the 166 MHz with the 340 MB disk, you'll definitely want to add
more disk. And I'd recommend adding more RAM, though on the 166 MHz model
all the SIMM sockets are full so you'll have to replace some. Once ELF and
shared libraries are stable the memory requirements should ease up a bit.

The TGA will be fine for your 17" monitor as long as you don't want more than
8-bit color. I'm reasonably happy with it on my 20" monitor. The TGA X
server is still in the early stages of development. If you decide you need
more colors or a more stable X server, you can stick an S3-based PCI video
card in the slot (as long as it is a short card).

I'm planning to stick a DEC 21140-based 100 Mbit/s ethernet card in the PCI
slot of mine, and run a point-to-point link to my Pentium Amateur box to
improve my X performance.

I just want to know when DEC is going to offer to sell me a 400 MHz 21164A
motherboard upgrade for the Multia/UDB. :-)
