> I need to follow-up on this... my AlphaStation 200 4/166 won't
> autoreboot or halt... The keyboard lights start blinking but nothing
> happens...
>
> Anyone has some info on this?

Yes, and no.

Rebooting currently works fine when using MILO; the kernel exits back to
MILO correctly, and MILO has kept some info re: what it lasted booted, and
so it all works. Almost everyone with a UDB uses MILO, so it works for them.

Now, as I remember, you're booting via SRM; I don't believe the kernel
actually exits back to the SRM console correctly, and thus you're left
with an (essentially) dead machine upon shutdown.

I don't know how hard this would be to fix; anyone else?
