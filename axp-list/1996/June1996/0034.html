Vareck Bostrom <bostrov@teleport.com> writes (Re: the UDB):

> Oh. 8-bits color is quite a limitation these days.

There is a short PCI slot, so you could put in a better video card.

> Does the off-chip cache run at core speed or bus speed?

> I'd heard 21164 based machines ran the level-3 cache (I think 21164's have
> 96kb L2 on the die itself?) at full core speed. Is that true?

No. Even the L2 cache doesn't run as fast as the L1. If it did, they would
have made it part of L1. The reason for a hierarchy of cache is that it is
impractical to make (and interconnect) a very large amount of very fast memory.

> What about 21066/21064's? Do they have L2 cache on the die?

21064/66/66A/68 21064A 21164/64A
--------------- ------ ---------
L1 instruction cache: 8K 16K 8K
L1 data cache: 8K 16K 8K
L2 unified cache: 0 0 96K

> I thought both could go to 1 MB cache?

The 166 MHz UDB has the 256K of cache soldered to the board. The 233 MHz UDB
has 512K of cache on a replaceable module, and is upgradeable to 1M.
