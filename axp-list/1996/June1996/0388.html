Joy! Joy!

Thank you for this driver!

I compiled it up tonight and tested with my Archive Python DDS-2 DAT drive.
The only way to use this tape drive is with disconnect enabled. With the
old NCR driver, I could only run in single user mode to use the tape
drive. With the new driver, I can access my disks and the tape at the
same time without locking up or screwing up the disks.

I didn't turn on the command tag queuing yet, but I'll try it out after
reviewing my hardware documentation. I think everything I have on the
SCSI bus supports it. I haven't run any performance comparisions yet.
