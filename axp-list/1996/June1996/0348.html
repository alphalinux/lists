Kyle,

Yes I have had the same problem with InfoMagic RedHat 2.1.

I got around this problem by creating a small hard disk partition to
which I copied the ROOT floppy and then at the MILO prompt typed

MILO> boot fd0:vmlinux.gz root=/dev/sda2 load_ramdisk=1

This means that ONLY the boot floppy is needed.

Hope this is of some help.
