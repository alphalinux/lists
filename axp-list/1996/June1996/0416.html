A new kernel patch has been uploaded to starnet:

ftp://ftp.azstarnet.com/pub/linux/axp/axp-diffs-2.0+3.gz

It comes with all the caveats of the earlier patches and a new set of
qlogic isp1020 patches. With the latter, I now get pretty decent
performance out of the qlogic scsi card (thanks to qlogic for sending
me docs so promptly!). Read throughput is better than with an ncr810,
write throughput is slightly worse---it probably could still use some
fine-tuning, but it does appear to work reliably now. No guarantees,
of course, but I have been using the driver for two days now and no
problems at all. Also, Michael Griffith confirmed that the fixes also
work on the x86 and boost performance there as well (by a factor of
2-10, depending on what you're doing).

Enjoy,
--david
