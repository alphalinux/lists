We have two Alpha systems. One is based on the 21066-166Mhz, running NT plus
Linux and the other is based on a 21164-266Mhz running Linux.
We have observed that when running both systems with Linux, all sources
that we generate with the gcc compiler, will run approximately 4-6 times
faster on the 21164. But when we compile the same sources under NT, the
21066 is twice as fast as the 21164.
It should be noticed that the sources referet to mainly consists of
floating-point operations.
Can the speed degradation under linux, be due to the fact that the
the compiler used is gcc-2.6, or is there another explenation???
