>recognize this scsi-2 cdrom i grabed of my sgi workstation?
Sorry to say, many of the CD-ROM drives used on Suns and SGI's use weird
ROMs, that is to say, non-standard...
I truly hope that is not the case, and that indeed, you are merely slightly
brain-deficient...
So, let's start with the basics.. If it's normal SCSI CD, it will just work.
Setup parameters: Is the SCSI bus termination on both ends, and nowhere else?
Is there an active terminator on the CD drive that need to be turned off (or
on) with a jumper? Is the SCSI I.D set right ( 4 or 5 are usually good safe
numbers).
Is it in an external box, and if so, is the termination power jumper on the
drive turned on (sometimes needed). If you post make and model number here,
I can probably post all the settings for it...
What model Alpha box are you running, as in what is the motherboard, and /or
SCSI interface card or chip in it??
As for ARC settings, they should see it if the hardware is set right...
