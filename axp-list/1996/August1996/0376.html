I can't help with the MILO problem, but for the above problem - I had the same
thing happen. What finally fixed it was using NT's arcinst.exe program to set up
a system partition on the hard drive. It doesn't need to have anything in it,
but apparently it does need to be there for some reason. If you don't have NT, I
think there may be other free programs to do the same thing, or I think
arcinst.exe itself may be freely available from Microsoft from their FTP site.
