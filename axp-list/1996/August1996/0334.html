I put a new patch and binary (version 0.3) on my Alpha Page today:

http://www.comnets.rwth-aachen.de/~ost/alpha/

Changes are:

* Optimized MemToBus() routine, which uses the same algorithm
as my optimized memcpy routine.

* Option "nolinear" is enabled by default, lowering the risk
of hanging the machine with a bad XF86Config.

* The patch includes work on SuperProbe. It works, except
detection of the memory size, which has been disabled.

* The binary has Harald's fix-font patch applied.

This is probably the last version of the patch from me, unless there
is sufficient interest to merge this with the main XF86_S3 source.

I won't fuzz around with the linear framebuffer mode, since I neither
expect a big improvement, nor own a card which supports this.

Greetings, Martin
