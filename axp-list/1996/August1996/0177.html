jestabro@amt.tay1.dec.com wrote:
>
> >>> Niel Balsino said:
> > I set up the UDB TGA X Windows using Stephen J. Gaudet's instructions from
> > this mail list. The instructions were great, X Windows loads with the
> > startx command, but my mouse does not work. Any suggestions?
>
> Make sure there's a /dev/mouse; if not, create it via:
>
> ln -sf /dev/psaux /dev/mouse
>
> It's probable that /dev/mouse appears in the XF86Config file as the mouse
> device name; if it's something different, change it to /dev/mouse and restart
> the Xserver.
