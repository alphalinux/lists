> Who had successfully installed Linux to Alpha Server 1000 4/233,
> please give me tips. What is the way of installing milo from Digital Unix
> SRM console? What boot and root images should I use?

Here is the way I finally got it working on the same machine.....

1) Get the kernel
ftp://ftp.digital.com/pub/Digital/Linux-Alpha/Kernels/mikasa-srm-2.0.10.img

2) dd or rawrite it to a boot floppy

3) from the srm console type
>>> boot dva0 -fl i

4) When you get the aboot prompt type
aboot> vmlinux.gz root=/dev/fd0 load_ramdisk=1

5) Once you get linux installed you need to compile and install aboot
