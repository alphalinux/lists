>>> Max Hadersbeck said:
> Hi
> I want to overwrite my NVRAM of my UDB with WinNT settings to boot
> directly linux. But its not possible
> I can write the changes in arc mode, but after power down it forgets my
> new settings.
> My aim is to bring the UDB to start LINUX immedeatly. Without any input
> from the console. Are there other possibilities ?

You want to continue booting ARC to MILO to LINUX, correct?

1. in the ARC "Setup" menu, select "Autoboot" and set a timeout value.

2. set the OSLOADOPTIONS line in the boot selection of MILO to:

boot sda2:vmlinux.gz

(or whatever the partition/kernel is that you boot). MILO looks at
that variable as its first command, so it'll start you up automagically.
