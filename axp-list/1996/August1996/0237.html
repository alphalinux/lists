Hi
After a lot of xdm - tests I found the right settings for the VRT19HA
Monitor on an UDB Box.
For debugging you start xdm -debug 1
then you see the selection of the monitor mode, done by xdm.
It also reports errors if it doesn't find the right entry and makes
sometimes (?) suggestions, what it needs if your values fit nearly.
You just have to set the right Sync Values and then the monitor works:

HorizSync 78.86 # VRT19 HA

VertRefresh 74.11 # VRT19 HA

I commented all the other sync values out.
You should not change anything with the Modeline Options, because they
belong to the monitor card.
