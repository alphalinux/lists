It sounds like your kernel is not configured for scsi tape support.
A tape drive will be "detected" at boot even if this is so since all
the boot sequence is doing is querying each address and parroting
whatever the device says that it is.

You need to have scsi tape support either compiled in or loaded as
a module (st.o) and you should be OK.
