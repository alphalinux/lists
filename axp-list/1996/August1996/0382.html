Since you did say "urgently" I take it to mean that you'd rather not
want to build libc either. Erik says you can get glibc (and related
RPMS) at:

ftp.redhat.com:/pub/devel/axp/RedHat/RPMS

Since this is not officially released software (note the "devel"...)
the usual disclaimers apply. In particular, you'll need to get the
latest rpm, though (it's up there somewhere; rpm-2.2.2 is what you
want, I think).

Note that the above contains ELF binaries, so you'll need the binutils
RPM etc as well. It's probably best to install these inside a
chrooted environment so there is no danger of messing up the existing
ECOFF system.
