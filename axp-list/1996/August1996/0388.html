>>> Olle Hammar said:
>
> I have a UDB 233 with a Toshiba 2.5" 548Mb disk, during the install the =
> fdisk cant find the cylinders, sectors and heads for the disk. I know =
> that the x option i fdisk can set them, but they wont stay to the values =
> i set them to. Question:
> 1. Does anyone have the correct values for this disk ?
> 2. If i want to use the NT arcinst.exe utility, how do i start it ?
> 3. Is there another solution to this problem?

Without having to resort to running an NT util, you should be able to
do it this way:

1. run "minlabel" on this disk, only so far as to get the
tracks/sectors/cylinders numbers usually printed out right
at the top, then quit out of "minlabel" without writing
the partition table.

2. run "fdisk" and use the "x" (expert) option to put in those
numbers you got from "minlabel", noting that "heads" refers
to the "tracks" you got from "minlabel".

3. write the partition table on exit from "fdisk", then REBOOT!!!

Good luck...
