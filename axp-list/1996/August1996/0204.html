>>> Max Hadersbeck said:
> I want to connect a VRT19HA monitor to my UDB - BOX.
> This is a 1280x1024 fixed frequency Monitor with
> lowscan : Horiz : 70.7 HZ, Vert 66,5 Hz
> highscan: Horiz : 66,5 Hz, Vert 72,6 Hz
> Does anyone know the settings for the xdm in
> /etc/X11/XF86Config

If you could boot directly into X you'd be fine; problem is, TGA support in
the LINUX kernel will attempt to setup (roughly) VGA-like video (640x480@60Hz).
Also, the ARC console will attempt VGA-like settings as well.

You could avoid the latter by booting via SRM console, which (I believe) all
UDB's have available as well as ARC; SRM reads some jumpers on the UDB MB and
sets the desired resolution/refresh from them. You still have to solve the
problem of the LINUX console code, which is not too hard but somewhat involved.

Let me know if you (or anyone) is *really* interested in taking this on, and
I'll make the information needed available.
