On Tue, 6 Aug 1996, Niel Balsino wrote:
> I set up the UDB TGA X Windows using Stephen J. Gaudet's instructions from
> this mail list. The instructions were great, X Windows loads with the
> startx command, but my mouse does not work. Any suggestions?

Some obvious, simple things to check:

* Did you have the mouse plugged in at power-up time? Sometimes machines
don't recognize the mouse unless it's there from the start.
* Does your current kernel have the PS2 mouse driver built in?

Don't be insulted if you've already tried these; I mention them because these
are the sorts of things that get me all the time.
