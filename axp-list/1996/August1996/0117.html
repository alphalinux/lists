If you use the XL.img from the images/1390/ directory, you need to "dd" that
onto a floppy; if you use the XL.gz from the kernels/ directory, you need to
put that file onto either a DOS floppy (the one with MILO/linload.exe works
fine), or onto an EXT2 format floppy.

If you try to boot from a DOS floppy, tell MILO:

boot -t msdos fd0:vmlinux.gz load_ramdisk=1 root=/dev/fd0

If the kernel is on an EXT2 file system (which is what the XL.img is from),
tell MILO:

boot fd0:vmlinux.gz load_ramdisk=1 root=/dev/fd0

There are two XL-related files in:

gatekeeper.dec.com:pub/DEC/Linux-Alpha/Kernels

namely:

milo-1.3.77-27mar1525.xl
xl-1.3.77-27mar1525.gz

These are older than the ones on the Red Hat CD or ftp site, but should work
OK, I think.

Good luck.
