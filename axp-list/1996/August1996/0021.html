> is it possible to compile a linux alpha kernel on an intel linux machine?
> I noticed that gcc does not have an alpha-linux target, only an alpha-osf1
> target.. can that be used? if not, how do I go about compiling a kernel?

No, the OSF target is not what you want. Get and apply the alpha-linux
gcc patches from ftp://ftp.azstarnet.com/pub/linux/axp/misc/.

While I once built an Alpha cross-compiler on a 32-bit machine, and it
seemed to produce correct code (only visually inspected), several
optimizations are disabled on the smaller machine. So while you might
be able to use this technique to bootstrap a kernel, you wouldn't want
to use it on a regular basis.

If you do go and try this, let me know how/if it works.
