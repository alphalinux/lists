Bill,
you will need a partition that the WNT ARC console can read
(as it has to read Milo). This means either DOS or NTFS. As Linux NTFS
support is not yet all there - use DOS. Then you can mount the DOS "boot"
partition from Linux to change the Milo that you use (if you wish). Note
that as Milo can also read DOS, you could put your kernel there too.
Generally, I'd recommend:

1st Partition: DOS ~10Mbytes
2nd Partition: EXT2 everything but partition 1 and partition 3
3rd Partition: SWAP ~2*memory size
