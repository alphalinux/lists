On Tue, 13 Aug 1996 14:11:46 -0400 (EDT), "Joshua M. Thompson"
<invid@optera.com> said:

> Ever since I moved my mail server to my UDB, I've noticed that my mailbox
> file seems to be randomly corrupted. Specifically, the "From " headers at
> the top of the messages get mangled (commonly it turns into "FFrom " or
> "rom "), causing messages to become concatenated. This really worries me,
> because I'm in the habit of skimming the subjects of my messages and
> deleting mailing list posts I don't want, and I'm thinking I may have
> deletd important mail stuck onto the ends of those messages.

This has been a known problem with RedHat's procmail binaries for some
time now. The default RedHat binaries enable a procmail "feature"
which actually causes much more harm than good, and is only ever
useful on nfs-mounted mail spools.

I've got a fixed set of rpms (and the srpms for the new configuration)
on

ftp://alphapc.dcs.ed.ac.uk/pub/linux/utils

I have been using them on my own RedHat boxes for months now with no
problems.

Cheers,
Stephen.
