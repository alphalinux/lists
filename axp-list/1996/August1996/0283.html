To all whom concerned

I put a source of mixer command to http://antenna.infocity.co.jp/file.html
This is a part of snd-util package, the entire package is some hundred
byte long and all we need is this few K byte file.

You can stop the noises from internal speaker like: mixer -line1 10

I too can not solve loud BEEP problem with it's console. so I always start
X and use "xset" command to stop the beep.

hope this helps somebody.
