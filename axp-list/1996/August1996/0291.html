An updated kernel patch is now available at:

ftp://ftp.azstarnet.com/pub/linux/axp/axp-diffs-2.0.12.gz

There are a few new bug fixes:

- profiling should work again (this involves increasing the timer update
frequency from 100Hz to 1024Hz; this is what used to be there and the
performance loss is hardly measurable according to my tests)

- kernel support for ELF coredumps has been fixed (you'll need a new gdb
fix to make that end working, too; stay tuned for a patch...)

- a few more kernel-header fixes (completely trivial stuff)

Again, this patch should be considered experimental (though there are
no known problems with it, at this point).

Enjoy,
--david
