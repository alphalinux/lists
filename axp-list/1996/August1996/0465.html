Robert Hendley wrote:
>
> Help, I am new to the axp I am trying to install RH 3.0.3 for the AXP
> Please assume in responding that I am new unix user.
>
> Question #1
> During the first part of the install script am getting an
> error "wrong fs type or superblock bad on /dev/fd0" .
>
> The call "mount -t ext2 /dev/fd0 /bootdisk" is causing the error.
> I have tried both udb and noname images using both rawrite and a
> gerneric dos copy.

Robert, I'm confused. Are you doing this from the milo prompt? If you
are trying to install 3.0.3 try:
mount -t ext2 fd0:vmlinux.gz root=/dev/fd0 ramdisk=1440
If however you are trying to boot back into the system after install,
try:
mount sda2:vmlinux.gz root=/dev/sda2
This should have been in the documentation if you got any.
