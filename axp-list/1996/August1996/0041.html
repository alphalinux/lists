> > So what target should I configure for then, if any?
> > And can I install the patched gcc to compile the kernel?
>
> You should configure both binutils and gcc with "alpha-linuxecoff"
> to get the standard ECOFF compiler. If you want to do ELF you'll
> need a completely different set of patches for gcc. But if you're
> just trying to cross-compile the kernel, ELF doesn't buy you anything.
