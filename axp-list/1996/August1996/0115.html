> Should i try the 53c8xx driver? If so, what should options should i use?

I had problems with the 53c7,8xx driver. Since I switched to the 53c8xx
driver, I've had no problems at all. I have disconnect, synchronous mode,
force synchronous and enable SCSI parity enabled. I disabled tag queuing
and PCI parity checking. You have to set the force I/O mode for
controller chip access.
