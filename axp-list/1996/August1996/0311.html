I got the following answer from C&P Hotline Technical Support<support@librt6.ENET.dec.com>
when I asked DEC:

----------------
The memory type for the Alpha or Intel based multia is
JEDEC Standard 36-bit, SIMMS in 4 slots 70ns or faster.
It is basically a standard PC SIMM, the Alpha system
board was originally spec'd to accept ECC memory or
error correcting so it will take either. However, you
should use a true parity SIMM.

Regards,

Multia Technical Support
