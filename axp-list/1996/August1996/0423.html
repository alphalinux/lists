I too thought startup message show the SCSI drive's geometory, but I
remember that I had to calculate CHS manually (because I did't have a
spec sheet for that drive) last time I add a externel scsi drive to my
UDB box, because kernel didn't report the CHS for that drive at that
time. The driver get geometry information from very first sector of
the disk and that disk was never formated for any PCs.

anyway, if it were new drive, that is, you don't have exist
partition(s) in that drive, you can try:

C H S
1096 16 64
or
548 32 64

I think both works.
