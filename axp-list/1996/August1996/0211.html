I think it *is* possible to blow it, depending on the particular monitor,
of course; some would simply ignore bad signals, others might try to do
something with them... :-(

If the monitor you would like to use does one of the valid resolution/refresh
modes as specified in the sample XF868Config for TGA Xserver, then I can help
with info about how to program the clock to get that mode. If it does *not*
support one of those listed, I really don't have enough *general* info to
allow an arbitrary rate to be programmed; meaning, I have a table of mode
settings for the clock that handles only those modes listed, and don't know
anything about how to determine the setting for anything different.
