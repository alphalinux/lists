>>> "Thomas E. Black" said:
>
> Does anyone know what the default refresh a UDB spits out? I have a
> 15" CTX multi-synch, and I want to make sure that my UDB is driving
> it within its normal operating range. The image looks good... but
> I don't want to fry my monitor in a few weeks.

Default text mode console is 640x480 @ 60Hz; this is good for almost any
monitor, except the fixed frequency ones.

Resolution and refresh are choosable under the X server; read the README
associated with the XF86_TGA at gatekeeper.dec.com:pub/DEC/Linux-Alpha/X11.
The sample XF86Config file has *only* the available choices.
