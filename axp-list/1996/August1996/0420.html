The file /etc/sysconfig/clock, for a system run from ARC console, needs to
contain the line:

CLOCKMODE="ARC"

This is used by the rc.d/rc.sysinit script for parameterizing the "clock"
call when reading from the hardware and setting the kernel time.

To ensure that a changed time is stored to the hardware in the appropriate
format, "clock -A -w" will write the kernel time to the hardware in ARC
console format. Or, use the ARC console facilities to change the hardware
time.
