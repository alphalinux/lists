On Thu, 1 Aug 1996, Robert Williams wrote:
>
> How well does Red Hat Linux run on
> Alpha 21164 based systems, particulary
> the Alpine Durango board?
> I see that Linux is reported to run on
> several 21164 based PCI systems, but
> representatives at Alpine tell me that
> it is not stable on the 21164.

Linux _is_ stable on the 21164 (I have one as my main computer, and haven't
had a single glitch), but the kernel images on the CD's aren't really recent
enough to be stable.

I don't know what the Durango board looks like, so no idea if Linux works
on that, but I'm very happy with the AlphaStation 600 I'm using ("alcor"
support as far as Linux is concerned).
