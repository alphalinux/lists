All,
there are all new test-images and sources now on gatekeeper.dec.com in the usual
place. The major change is that Milo now supports ELF format object files such as the kernel.
Right now, those sources do not build under ELF (unless you take David Mosberger-Tang's patches)
as the Digital BIOS emulation code is released as an ECOFF library; once that's moved to ELF then
I'll release an ELF release of Milo.

Just like Linus, I love it when you find bugs/problems or just report that all's well.
So, please test out these images; so far Cabriolet, EB164, PC164 and XL work fine. I'd like to
move these images over to be the stable images by Friday (I go on holiday then) so your input is
needed. Oh, all of the images include NCR810, Qlogic ISP and Buslogic SCSI drivers and they're all
built against 2.0.12 kernel trees.
