Bryan W. Headley wrote:

>>Actually, I'd like some books on Alpha assembler. Where do you send off for
>>this type of literature?

There are 2 (well, 3 I suppose) sources of this information:

1. Read some source code and figure it out for yourself

2. The Alpha Architecture Handbook, order number EC-QD2KA-TE

3. The Alpha Architecture Reference Manual (2nd Edition) by Dick Sites

The Handbook is a subset of the Reference Manual. It's pretty comprehensive
and has everything that most people will need. Note that is does *not* have
platform or CPU-specific information it it.

The Handbook is available from Digital Semiconductor and its distributors. Look
at David M-T's FAQ for phone numbers/addresses. This handbook is/should be
free.

The Reference Manual is a book and you can order it from any book store. It's
priced like any other textbook (ie it's rather expensive).

Neal.
