Well, it all depends. Even with production-quality SMP, you still need
software to use it. If your FORTRAN code is parallelized (say by
using PVM), or if you typically run several programs at once, you win
big; however, if you just run a single program, all those other CPUs
sit idle. For most people a SMP workstation is a waste. SMP is great
for multiuser/server machines, but may be an overkill for personal
stuff.

This would change if automatic parallelizing compilers were available,
but those are few and far between, and there are no Linux ports that I know of.
