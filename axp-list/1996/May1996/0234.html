strip 2.6 (from Red Hat AXP 3.0.3) breaks [some] binaries.
quickstrip doesn't show this problem and produces smaller binaries.
example:

# echo "main() { while(1); }" > idle.c
# cc idle.c
# ll a.out
-rwxrwxr-x 1 root root 72128 May 14 19:13 a.out
# ./a.out
^C
# strip a.out
-rwxrwxr-x 1 root root 40960 May 14 19:13 a.out
# ./a.out
Segmentation fault (core dumped)
# cc idle.c
# quickstrip a.out
# a.out truncated at 32768
# ll a.out
-rwxrwxr-x 1 root root 32768 May 14 19:13 a.out
# ./a.out
^C
# strip a.out
-rwxrwxr-x 1 root root 40960 May 14 19:13 a.out
# ./a.out
Segmentation fault (core dumped)
