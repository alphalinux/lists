> I've got one of the later kernels, and it built cleanly. How
> do I use it with the Jensen. I already roached the partition
> once and had to reinstall. Not fun. I've tried aboot-0.3x,
> but it won't compile cleanly. Anyone who has the magic cookie
> to make this work would be appreciated.

Well, if you're *not* using aboot and you have a separate mini-partition
already set aside for the kernel (as was suggested in the Jensen-HOWTO),
then you can simply do

make BOOTDEV=/dev/sda srmboot

in the kernel source tree and everything *should* turn out okay.

This assumes that your bootable kernel is on /dev/sda1 If it's on /dev/sdb1,
say "make BOOTDEV=/dev/sdb srmboot", and so on.
