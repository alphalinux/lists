OK, by popular request (as they say), here is the latest version of
aboot:

ftp://ftp.azstarnet.com/pub/linux/axp/aboot/aboot-0.4.tar.gz

Updated documentation is available at:

http://www.azstarnet.com/~axplinux/srm.html

What's new?

- it actually builds against recent kernels

- aboot can now boot Linux over the network (thanks to Dave Larson)!
If you have lots of Linux systems and are getting of having to
copy a new kernel to a dozen machines, this might be the solution.
It's particularly useful if you're a kernel hacker and need to
frequently try out different Linux kernels; network booting allows
you to boot a new kernel without having to deal with messy floppies.

- iso9660 support---yes it's true! Now we're all looking forward
to the first CD-ROM distribution that can do floppy-less installation
with the SRM console; who is it gonna be? :-)

Michael, I read you're planning on some other aboot improvements. I'd
recommand getting 0.4 since the source organization changed quite a
bit. It should still look familiar, though... :-)

Enjoy,
--david
