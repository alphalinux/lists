> BTW: I have installed the Linux distribution from gatekeeper. No sweat

BLADE ? . It was an quick hack. There is a real distribution now. RedHat.

> Did you have the same problem and what did you do to get rid of them?

I didnt install from a Distribution. I built the system by hand. BLADE didnt
even exist then. I used a OSF system to partition the disk in the first
instance. Booted off floppy and build the root fs buy hand, coping the base
system via floppy.
A lot of debugging messages were in those early kernels, of which BLADE
is based.

Once I got the system to the point it could build it self, I enabled
fdisk style partitions , some one had just gotten the IDE driver working,
so I then used an IDE disk swaping between an Linux/86 system and Linux/AXP
to get increasingly large archives of utilities accross.
Those days you also had to apply patches to the source to get it to compile
under gcc. Linus was building AXP kernels under OSF/1 with DEC's compiler
at the time.

BTW, my order for cache RAM took 4 months to be filled....
