yesterday I had my alpha machine at the university and connected
to our LAN and thus I had internet access. I tried to load the FAQ html files
using lynx but the following command failed (can't remember the error message, sorry;
but this was no problem with lynx on other platforms):

lynx -source "http://www.azstarnet.com/~axplinux/FAQ-2.html" > file

with the following binary (probably from RedHat 2.1):

-r-xr-xr-x 1 root root 942080 Mar 18 00:06 /usr/bin/lynx*
