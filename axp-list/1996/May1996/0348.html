OK, the latest glibc patches are now available at
ftp://ftp.azstarnet.com/pub/linux/axp/glibc. In that directory you'll
find:

-rw-r--r-- 1 davidm other 11811 May 21 18:28 axp-extras-960521.tar.gz
-rw-r--r-- 1 davidm other 47573 May 21 23:09 libc-960516-alpha.diff.gz
-rw-r--r-- 1 davidm other 6639 May 21 23:24 pre2.0.4-kheader.diff.gz

The first file contains additional include files that, strictly
speaking, don't belong either into the kernel or into glibc.
Eventually, these files may become part of either the kernel or glibc
sources, but it's something yet to be resolved. For now, just get
that tar file, edit the destination directory in the Makefile, type
make, and that's it.

The second file is a patch relative to the glibc snapshot of 960516.
Since this not an officially released version, you'll either have to
know where to get such a snapshot or contact me.

The third file, finally, is a set of patches for the kernel-header
files as distributed in pre2.0.4 by Linus.

BIG FAT NOTICE: if you are not planning on glibc and/or ELF
development, don't bother getting the above files.

Furthermore, to compile glibc with the ECOFF gcc/binutils, you'd need
some additional patches. If somebody planning on glibc development
wants them, let me know.

You may want to take a look at Linux.Migration in the root directory
of the glibc source tree after applying the libc patch---it lists some
of the changes that are often necessary when going from the old libc
to glibc.

Enjoy,
--david
