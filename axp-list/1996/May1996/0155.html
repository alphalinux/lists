Hi,

are there any electrical differences between a normal PCI slot in a PC
and this in the XL266 ? e.g. 3.3V instead of 5V power supply or similar ?

I'm asking because I've severe problems with one S3 card with Trio64V+ chip
from SPEA/V7 (Diamond in USA) which only show up in the alpha box.
I get lots of corrupted pixels when I'm jut moving a window around with fvwm
using non-opaque moves.

in my PC (ASUS SP3G main board, everything 5V) I see this problems only
*very* rarely with the default BIOS settings and when I enlarge the
"-RAS precharge timing" signal for the EDORAM on this card (clear CR68 bit3)
they're completely gone but for the alpha box this (and other DRAM timing
settings) doesn't help.
here I have to select a "2-cycle EDO mode" for the DRAM (set CR36 bit 3).

one more problem I only have with this card and only in the alpha box
even with slow DRAM settings and with and w/o MMIO and/or hw-cursor
is that when moving the window long and intensively enough the machine
completely locks up with no message at all (this does happen much more
often/faster with fast DRAM settings which will first show up severe
display corruptions and lock up after some 5-10secs).

any idea ?

Harald
