handle PALcode resides at

ftp://atheist.tamu.edu/pub/richard/linux/axp/newgas-960423.tar.gz

looking at) contains C++ style comments that need to be converted
as well as form feed characters that, for reasons I have yet to look
into, need to be stripped.
