Just uploaded a newer version of mpeg_play. It's in:

ftp://ftp.azstarnet.com/pub/linux/axp/misc/axpsrc-mpeg_play-2.3axp-2.tar.gz

The only change compared to the earlier version is that dithering
algorithm "ordered4" is now implemented (something I have been using
for a while in experimental code). That algorithm is optimized for
the Alpha. The only disadvantage is that it always wants its own
colormap (this allows to save one table lookup per pixel). On a
Cabriolet with a TGA, this increases the framerate on
RedsNightmare.mpg from around 30 frames/second to about 55 fps.

To get the highest possible decoding speed, do something like:

mpeg_play -dither ordered4 -framerate 0 -controls none filename

Enjoy,
--david

PS: Please no complaints about the ugliness of ordered4.c. Beauty was
not an issue---raw speed was.
