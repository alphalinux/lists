All,
there are new sources for Milo (1.3.95) on gatekeeper
together with all new /test-imges. There was a bug in memory allocation
that meant in some circumstances a guard band would be written over
some dangerous place in memory. This manifested itself as an inability
to load from floppy. All of the new images are built with 1.3.94
or above and so contain all the latest PCI-PCI bridge support.
