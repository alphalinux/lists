I've been reading this list for awhile now and I, myself, have been trying to
decide whether to get a Alpha 21164 family board, or a dual, tripple, quad, etc.
Pentium Pro motherboard.
I am , like yourself , using Scientific programs like FORTRAN, as well as
Engineering Programs like CAD Packages, FEA Analysis and so forth.
I need to be able to run both Linux and NT on the same machine, and I was
very happy when I heard Alpha boards were running Linux.
But at the same time I still need Win NT to run my Windows applications ( I
know, I know but Microsoft Word , Excel, etc. are a must for me)
But the motherboard you've mentioned above is one hell of a deal.
4 P6 CPU's running at 200 Mhz and since the Linux has SMP support now, that
sure does beat Alpha. Right? (NO offense , I am a great Alpha fan but like
everyone, I too have limited resources, and need to spare money for the RAM,
Hard Drive and other extras like a really nice Imagine 128 Video Card)

I was in the brink of getting an Alpha board but that certainly changed my mind.
And unless something really shocking comes up I think I am going to go with
Intel again.

Does anyone have anything to add to this.?

Thanks for the very informative info Kendall

I'll be looking forward to people's respond to this.
