I believe that most of the difference is in the math library. The following
numbers come from an EB164 333MHz. Alpha with Linux 1.3.94. The only
difference is the math library used. Both were compiled with gcc 2.7.1.3:

latest gcc libc.a 1040 sec
old libdpml.a from OSF 1.3 220 sec

Again there is a factor of 4/5 times. I have tried to link the latest DEC
libdpml.a with programs under Linux since I believe they are better
optimized for the 21164 but ld returns error 6.
