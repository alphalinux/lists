As far as I know, the sound driver does not setup UDB's mixer setting
collectly at boot time. The mic input gain is TOO large and causes bad
humming noise.

I'm using simple mixer command came along with snd-utils-2.0.tar.gz.

antenna:~$ mixer
Usage: mixer { pcm|speaker|mic|mix|igain|line1|line2|line3 } <value>
or mixer { +rec|-rec } <devicename>

xmixer is just a overkill.
