Hello,

after trying around with Jim Paradis' ARC-based installation tools for the
Jensen, I have made available an aboot-based installation toolkit. This
consists of an aboot-0.2 (newer to follow) image and a standard kernel (1.3.89
currently).

I have just installed RedHat on a fresh, new 730MB disk and it seems to
work fine. The necessary files and updated installation docs are available
from
ftp.azstarnet.com:/pub/linux/axp/jensen/install.

You will have to make the dummy-install-disk needed for the redhat installation
yourself (mke2fs a floppy and copy vmlinux.gz to it).

There seems to be a problem with copyboot which wants to copy more than
1440 block on my 2.88MB drive - a workaround is described in the installation
docs.
