Bob,
the best thing about Red Hat is the ability to upgrade.
Either buy the latest CD or go to ftp.redhat.com and pull over the
latest .rpm files. Either way have all of them somewhere handy.
Then either use the nice windows control panel to upgrade the packages
or do it by hand using the rpm command. For critical stuff like the
passwd file it backs the old ones up and so there's a little post
installation tidying up.
