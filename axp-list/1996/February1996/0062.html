You should be able to connect the power supply and a serial port to test the
board (that is see the console prompt) right out of the box. I would suggest
that you disconnect all of the other cables and concentrate on this very basic
configuration.

If you are sure that you have these two cables correct, the only other
possibilities are a bad board (unusual, but within the realm of possibilities)
or bad memory. If you get to this point I would most likely suspect the memory
SIMMs.

There have been several situations where people were confused and were not
using 36 bit SIMMs. Note that you must use 36 bit SIMMs with the AXPpci33
card. The more common 72 pin, 32 bit (non-parity) SIMMs will not function.
The newer "parity" simms that have parity detection logic on the SIMM will not
work.

I will be curious to hear what you find...
