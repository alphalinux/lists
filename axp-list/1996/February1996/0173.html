Frank,
is WNT running on this system? If so, then I suggest that
you create a linux sub-directory of the "os" directory in C and put
linload.exe and eb164.arc into there from WNT. Failing that, make
a DOS floppy up with those two things on it. The worst thing that
should happen is that linload.exe will hang or that eb164.arc will
crash. I don't know much about the system that you describe but
does it have the Alcor chipset or a different chipset in it?
