>Maybe this is too general for Alpha, but the sendmail.cf included
>in the Red Hat release is a tad too simple for my needs both at home and
>at work. I've built myself a new sendmail.cf (from sendmail.8.6.12
>m4 sources). However, I'd be interested in seeing the source files that
>produced Red Hat's sendmail.cf - any possibility. Also, for someone new
>to sendmail/mail on Unix/Linux is there an easy way to marry user names
>with pseudo user names? For example, my external mail address is
>
>david.rusling@reo.mts.dec.com
>
>and reo.mts.dec.com is a gateway that knows that david.rusling is really
>rusling@rdgeng.reo.dec.com. What software would I need to accomplish
>the same thing on Linux?

check the bat book. you need to define the MASQURADE_AS macro in your
.mc file. there is a complete list of the m4 macros and their use for
just about anything in the bat book. in general a leaf node only needs
4-6 lines in the mc file: smart host and masqurade, localhost and domain
and a few true/false items.
