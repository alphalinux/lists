Hi, everybody.

I am quiet new to this list and would like to know if there is anybody using
Linux on a Raptor system from Deskstation Technologies. I am currently working
in a company here in Germany that would really like to get a Unix running on
these machines. Some Infos about the Raptors:

Alpha 21164 processors 266Mhz and up
2 MB ext. Cache
AM53C974 SCSI Ports
PCIset (Digital) + ISA Bridge
Diamond Stealth 4MB VRAM
ARCS BIOS (should be an ARC clone)

Currently I am having trouble to even get the linload.exe to work. I think i
did a wrong installation of the image-disks, but I don't know whats wrong
exactly.

I used the "cat image >/dev/fd0" to get the data to the disk. Then I copy it on
to the SCSI-Partition 0 (FAT), where the WinNT Loader resides. I setup up a set
of environment variables as mentioned in the faq, but if I run the linload.exe
from the Bios-Loader, the screen blanks, and a blink of an eye later I am back
in the boot menu.

I would really appreciate to get some help.

Bye, Frank...
