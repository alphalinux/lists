Howdy!

As I promised one week ago, I should clean up my patches and
make 'em availble.. Ofcourse, maybe the best solution still
is a kernel solution...The ircII is a "cleaner" solution with
a if defiened (linux) && defined(__alpha__) in one of the
header files. The more patch is just a "quickhack" which
replaces termio->termios, TCGETA->TCGETS,TCSETA->TCSETS...

I didn't make an srpm/rpm for util-linux avaible due to the
fact that the "original" one at ftp.redhat.com *does not* compile
straight out of the box (atleast not on redhat-2.1/axp "release"),
and I didn't have time to fix thoose errors..

I've made the following availble at: nova.campus.luth.se

ircII:
/pub/linux/alpha/patches/ircII-2.8.2.patch
/pub/linux/alpha/SRPMS/ircii-2.8.2-5.src.rpm (Complete source+patch)
/pub/linux/alpha/SRPMS/ircii-2.8.2-5.src.comments:
Wrote a small patch for fixing the termio(s) problem.
Added a symlink for the manpage.
/pub/linux/alpha/RPMS/ircii-* (precompiled binaries)

more:
/pub/linux/alpha/patches/util-linux-2.5-axp.more.patch (Small "quickhack")

Enjoy! If you have any comments about the above stuff, plz mail me!
