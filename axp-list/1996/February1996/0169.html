All,
yet more Milo images and sources are winging their way
towards gatekeeper:

(1) milo-1.3.64-021996.tgz are the latest sources and these
will build from 1.3.48 up to 1.3.66 (and hopefully beyond).

(2) test-images - all new images built against 1.3.57 and
1.3.66. All the images bar the eb164 images contain the
latest SCSI drivers but with OPTION_DISCONNECT removed. This
does mean that you'll need to use those drivers in the linux
kernels that you boot.

I've fixed a minor bug in the AUTOBOOT stuff so that the
boot string is now correctly passed and I've tidied up the
flash management utility so that it's almost what I really
want.

Dave
