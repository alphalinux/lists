> (2) Fully rebooting the system is somewhat tricky. Where we booted via
> WNT ARC, that would mean reloading and restarting it (most processors
> do not have reset under software control).

On the i386, one way to reset the chip is to load a null
interrupt vector table and generate a software interrupt.
I do not have any of the alpha chip-specific documentation,
but is there anything similar that can be used?

Another way to reboot PC's is to ask the keyboard controller
to pull the reset line down. Since by and large the alpha
systems out there seem to use PC-compatible hardware, does
anyone know if that line is still connected on the alphas?
