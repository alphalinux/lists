> binaries would run under Linux/AXP.
>
> WIll some/all programs do this? Netscape would be the most usefull.

Netscape *will* run, but it is built with shared libs. So, you need to
copy your netscape binary and shared libs over. This should only be
done if you have a Digital Unix license for the machine you are on as
you are using Digital Unix code.
