> I've been told that Fast-Wide SCSI drives are not supported by Linux/Alpha.

The 53c8xx driver doesn't currently do wide. Even so, the drive should work
fine as long as you set the ID in the range 0 to 7 (but typically not 7 as that
is usually the host adapter).

It shouldn't be too hard to add wide support to the driver, but last I heard,
Drew wasn't working on it because he didn't have any wide devices.
