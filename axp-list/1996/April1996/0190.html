> I am thinking about getting a box with the Alpha board to use it as a
> conventional number-crunching workstation and for any other normal workstation
> use.
>
> At the current stage, is it really usable? Can I get a decent FORTRAN
> compiler, for example? [Sorry, I still use FORTRAN].

Yes, they run -- they can even run OSF/1 shared libraries,
but that is stretching things a bit -- if you have an OSF/1
license at the machine, you are unlikely to run Linux at it..

At the moment the best Fortran compilers are still DEC made,
as well as the best C-compilers are.. If you do a statically
linked binary at some near-by OSF/1 machine with its compilers,
then you can run it all right at your Linux machine.
( f77 -O4 -non_shared -o program.exe program.f subrout.f ... )

At the moment the only Fortran for Linux alpha is f2c with
"-mieee" compile option. (It must be made with that option!)
The GNU-Fortran (g77) does not produce workable code at Alpha
yet.
