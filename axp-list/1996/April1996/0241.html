Sounds like a locking problem. I haven't seen this my self, but since
I get mail through popmail only and never run more than one copy of it
at the same time, there is no reason I'd see such locking problems.

One thing you might want to check is that your kernel has the flock
syscall. Just search arch/alpha/kernel/entry.S for "flock". It was
added relatively late, and I believe the first Red Hat Linux CD-ROMs
didn't have that syscall.

Since I'm not sure how well locking has been tested, it may be
worthwhile to write a small test program to make sure it really works
as advertised.
