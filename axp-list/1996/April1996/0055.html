The 233 MHz Multia 24 MB configuration uses a pair of bizarre 12 MB SIMMs,
and comes with a warning that they can only be used in the first bank.
Other than that, Multias use standard 36-bit SIMMs, which must be installed
in pairs. The memory bus width is 72 bits, with ECC (error correcting code)
done inside the processor, so they have to be "real" 36 bit SIMMs, not
"logic parity SIMMs".

The SIMM you describe sounds *very* bizarre. It doesn't sound like either an
8 MB or 12 MB SIMM. Are you sure it wasn't 16 514400 and 8 511000? That
would make sense for an 8MB SIMM.
