> I have a 1.08 GB IBM DPES-31080 Disk (scsi-2) in my system. Fdisk asks me
to
> change the sectors, cylinders, and heads of this disk.
>
> As i don't have any information about this figures of the disk how can i
> figure them out ??
>
> Thanks for all the help in advance.
>
> wolfgang
>

Use cfdisk insted of fdisk, it detects the HD params from BIOS.
If write fails, try using cfdisk -z /dev/sdX, this wont read the
original partition table.

BUT: I have tried to install the 2 GB version of the IBM DPES and
it NEVER worked (errors when making mke2fs). Changed to a
Quantum and it was PnP. So there is probably a problem with
the DPES series.
