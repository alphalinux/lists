> Does DEC UNIX 4.0 differ so much from Linux AXP?

Linux/Alpha has no shared libraries yet. Some of the administration and
file system layouts are a bit different too. Digital Unix has full kernel
threads, Linux's thread support is not well documented yet.

> Is Linux AXP radically different from Linux 1.2.13, or can I expect the
> same learning curve coming from DOS/Windows/Netware to 'regular' Linux?

Linux/Alpha is just like Linux 1.3.92 for the most part. There is no
DOS/Windows compatability mode software. Quite a bit of source code
available software that runs on Linux/x86 will run on Linux/Alpha.
Occasionally there are 64-bit porting issues to contend with.

> Does anyone out there have experience/knowledge of Oracle 7 on DEC
> Alpha's, running either UNIX 4.0 or Linux AXP they would share?

No, we just have Oracle on Sun. If I were to run it on an Alpha, Digital
Unix would be the only way to do so and receive support from Oracle.
