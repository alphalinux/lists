OK, I just uploaded the file:

ftp://ftp.azstarnet.com/pub/linux/axp/misc/axpbin-vrweb-1.1.2e.tar.gz

It's about 1.3MB and contains the vrweb binary, a couple of READMEs,
and some sample VRML files. The README also explains how to teach
Mosaic to run the vrweb browser on VRML files and you can find a lot
more sample scense at the source site in Austria (there is a pretty
cool scene showing the environs of the city of Graz, for example).

Anyhow, I should mention that I don't plan to have time (or knowledge)
for bug fixing vrweb. It's provided as is. Basically, I don't want
to hear about bugs, but if you like it, I sure would love to hear
about it (hey, getting some positive feedback once in a while never
hurts... :-). Having said that: the basic things seem to work quite
well. I'm not sure whether it deals with TIFF files correctly and I
wasn't able to get vrweb to send commands to Mosaic (I may have done
something dumb). The reverse direction (Mosaic->vrweb) seems to work
fine though.

Enjoy,
--david
