> I have a 1.08 GB IBM DPES-31080 Disk (scsi-2) in my system. Fdisk asks me
> to change the sectors, cylinders, and heads of this disk.
>
> As i don't have any information about this figures of the disk how can i
> figure them out ??

I've run into this a few times, and usually do the following:

1. run "minlabel", which always seems to get the disk specs right; just
write them down, then quit without writing the disk label.

2. run "fdisk" and use the expert mode to enter in the specs from 1.

Good luck.
