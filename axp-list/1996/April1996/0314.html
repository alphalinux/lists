On Tue, 23 Apr 1996 netman@szem.szeg.matav.hu wrote:

> Is anyone has rpm'ised TGA X-server? Or is it easy to istall on redhat?

There is no RPM for it available, but installing it on Red Hat is trivial.
Just:

1) untar it
2) put the binary in /usr/X11R6/bin/XF86_TGA
3) do this: "ln -sf ../../usr/X11R6/bin/XF86_TGA /etc/X11/X"
4) read the README to create /etc/X11/XF86Config
