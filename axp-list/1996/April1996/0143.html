I've been playing around with Fortran myself... here's a hint
for both you and Erik:

Get the f2c sources, add "-ieee" to the CFLAGS, and rebuild.

You should also use the "-ieee" flag on any resulting C files
that use floating-point (which you're likely to have because
this *is* Fortran).

The -ieee flag causes gcc to generate code that can recover from
imprecise floating-point exceptions. The code to handle this has
been in the kernel for some time now, but you still have to ask
for the compile-time support.
