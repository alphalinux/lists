Boris,
the supported graphics cards (for all alphas running Linux) are
described in David Mosberger-Tangs excellent FAQ at ftp:://ftp.azstarnet.com
and on his web site at http://www.azstarnet.com/~axplinux. Generally
the following applies:

(1) Any Number 9 card (ISA or PCI)
(2) MACH 64 PCI cards,
(3) Digital Zlxp (21030) aka TGA cards

Here, by supported I mean supported by X, if you boot Linux via
Milo then any card supported by Digital in that system will work in
character cell mode (Milo includes Digital's BIOS emulation code for just
this reason).
