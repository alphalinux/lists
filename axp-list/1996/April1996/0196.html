Wolfgang,
hmm, just read your latest message. OK, so you are using the
latest. From memory make sure that the WNT ARC parameters are set up
like:

OSLOADER multi(0)...fdisk(0)\linload.exe
OSLOADFILENAME multi()...\cabrio.arc

or you can put milo onto a DOS floppy, call it fwupdate.exe and use the
"Upgrade firmware" option from WNT ARC. Note that all of this is in the
Milo HOWTO...
