Date: Wed, 6 Mar 1996 19:15:17 -0500 (EST)
From: Erik Troan <ewt@redhat.com>
To: redhat-announce-list@redhat.com
Subject: RPM 2.0 is now available

Thank God we are able to announce:

RPM 2.0 is Now Available!

RPM 2.0 is a vast improvement over the 1.x series, and incorporates
many suggestions from our users. Some of the new features are:

. Complete reimplementation in C - *much* faster, no Perl required :-)
. Much more flexible "header" structure makes new features easy to add
. librpm.a - write your own apps that understand RPMs and the
database!
. Much faster than RPM 1.x
. Finer grains of --force
. Improved --verify output
. PGP signatures to guarantee integrity and origin of packages
. Fine grained control (via spec file) of verification process
. Vast speed improvements over earlier releases
. Improved database integrity
. Additional spec file syntax:
Summary - short one-line description
Description - (same as before, but meant for longer text)
Packager - name, email of packager (this way when a
package is rebuilt, the original "author"
of the spec file and patches is maintained).
URL - place for URL with more info, docs, etc
Serial - serial number of package (to alleviate version
ordering problems)
Exclusive - build exclusvely on this architecture
%clean - executed after all packaging is finished
%changelog - place to put change information
. "Subpackages" can have their own names using %package -n foo
. Did I mention that it was fast?

To upgrade to the latest RPM on an Intel machine:

1) Upgrade cpio-2.4.1-1.i386.rpm
2) Upgrade to rpm-2.0-1.i386.rpm (with rpm -U)
3) run /usr/bin/rpmconvert if you've never used RPM 2.0 before
4) Upgrade to the python, pythonlib and glint

All of this is summed up by:

rpm -Uvh cpio-2.4.1-1.i386.rpm
rpm -Uvh rpm-2.0-1.i386.rpm
rpm -Uvh pythonlib-1.1-1.i386.rpm
rpm -Uvh glint-2.0-2.i386.rpm
rpm -Uvh python-1.3-4.i386.rpm

If you are using an Alpha, you should do the following:

1) Upgrade to rpm-2.0-1.axp.rpm (with rpm -U)
2) run /usr/bin/rpmconvert if you've never used RPM 2.0 before
3) Upgrade to the python, pythonlib and glint

All of this is summed up by:

rpm -Uvh rpm-2.0-1.axp.rpm
rpm -Uvh pythonlib-1.1-1.axp.rpm
rpm -Uvh glint-2.0-2.axp.rpm
rpm -Uvh python-1.3-4.axp.rpm

All of the necessary packages for these upgrades are available from
ftp.redhat.com in /pub/code/rpm/update.

Now that RPM 2.0 is officially released, the redhat-list is an
appropriate
place for discussion of RPM 2.0 issues.
