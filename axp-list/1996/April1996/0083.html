All,
I've put a new version of the Milo sources onto gatekeeper (1.3.83).
These include a couple of changes:

(1) Better cross-compilation support (Matti Aarnio),
(2) Better HWRPB setup. This results in /proc/cpuinfo that looks something
like the one at the end of this mail. It turns out that there are architected
ways of figuring out what sort of CPU you are running on. I've implemented
part of that and I'll do the rest some other time. To build it you will need
these two extra definitions in hwrpb.h:

#define EV56_CPU 7 /* EV5.6 (21164) */
#define EV6_CPU 8 /* EV6 (21164) */

Dave
