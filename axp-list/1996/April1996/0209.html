I had to laugh when I read this, it's so typical! I was in a group of
physicists doing relativity simulations on supercomputers, all in fortran.
Since I left the group and am on my own, I discovered C and C++ and have
been using that. I think one of the reasons for the fortran-centric world
of science is that the fortran compilers on the late 70's early 80s Crays
were very efficient and so we were trained to use only them. Even after
good C and C++ compilers came out people were used to fortran and stuck
with it (in fact, I specifically remember learning to poo-poo _any_ code
which used pointers, these are not vectorizable, I was taught).

Sorry, this has nothing to do with Linux/alpha (except that I'm still
wondering about glib.c ... anytime soon?, other than that, my system is
running fine!)
