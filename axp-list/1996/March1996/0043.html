Ok. When the system comes up to the blue screen it should display something
about the firmware release number. The most current is 4.42. you may have
something like 3.52 which comes on all motherboards.

The one with MILO you may want to flip it back to ARC and check it out.
This can be done by placing a dumb terminal on com 1 with 9600 baud, 8 bits,
1 stop bit, no parity, Xon/Xoff, and pull Jumper sp_bit7 off at J2 you
should then see AlphaPC64>.

Type in "romlist" and should see 3 images debugger, NT and MILO. Type in
bootopt nt to get back to arc. Install the jumper sp_bit7 again and it
should come up in ARC. From here you'll see firmware release number. If
you need 4.42 let me know and I'll get it to you.
