Find on gatekeeper.dec.com later today or tomorrow:

pub/DEC/Linux-Alpha/X11/xl-1.3.75.gz
pub/DEC/Linux-Alpha/X11/milo-1.3.75.xl

This MILO and kernel *will* work on a true XL, as opposed to an AVANTI.

Note that a true XL has a *different* motherboard than AVANTIs, and
consequently the AVANTI MILOs and kernels will *not* work correctly.

You can tell a true XL by the SIO (PCI<->ISA bridge chip); its a 82379AB
and *not* the AVANTIs 82378IB/ZB. Either look on the motherboard at the
chip, or, under ARC console, display the hardware configuration and look
for, under the "PCI slot information" display, vendor ID 8086, device ID 484.
If the revision ID says 88 or larger, then its an XL.

If your box is labelled XL but does *not* meet the above criteria for XL-ness,
then it's really an AVANTI, and should work with those MILOs/kernels...

This pair solves the generic XL floppy problems, as well as the keyboard
problem also found with XL's.

I've successfully installed, from CD and NFS, the full RedHat 2.1, and run
X on an S3-968.

IMPORTANT LIMITATION: don't run with more than 64 Mb of physical memory!!!

This *may* be removed in the future, or upped to 96 Mb, but for now and that
MILO/kernel, don't run with more than 64 Mb installed.
