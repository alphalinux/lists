> Hi All,
>
> I'm trying to boot Linux from the ARC console on an alphastation
> 200. I'm looking for the avanti kernel on the redhat CD-ROM and there
> doesn't seem to be one (there exists an avanti.img file but does this work
> with the ARC console?). What do I do? I have a tga video card.
>
> Thanks,
> David Bernstein

The RedHat 2.1 Alpha distribution doesn't have a TGA server; that you'll have
to get off gatekeeper.dec.com:pub/DEC/Linux-Alpha/X11 as:

axpbin-Xtga-BETA-0.2.gz
and
axpbin-Xtga-BETA-0.0.README

Note that the BETA-0.2 requires NO 'Option "sw_cursor"' in the config file,
an earlier version of which you'll find along with important notes in the
README file.

Now, of course, that server will *ONLY* work with 8-bit TGA, ie ZLXp-E1, and
*NOT* one of the 24-bit versions (E2 and E3).

However, the Linux kernel *does* support all versions of the TGA in text mode.

Have fun...
