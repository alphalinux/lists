"Is Milo the same as Lilo?"

The short answer is yes and no. Consider Milo as freeware firmware/BIOS.
Like Lilo, it's only task is to load Linux.

"What happens if I get a power dip when I'm flashing Milo?"

The flash probably goes horribly wrong and you end up with broken
flash. However, the Noname supports a boot block format floppy boot
(milo.dd) and so long as you don't blow away the Debug Monitor
in the other systems, then you should be fine. Generally, I'd
stick to WNT ARC + Milo but the AlphaPC64 and EB164 both have enough
flash for Debug Monitor + (WNT ARC | SRM) + Milo.
