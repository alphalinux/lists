Firstly, you have a very old Milo (I can tell from the memory
cluster description - I took that out ages ago). Please grab a
new one from gatekeeper.dec.com ~ftp/pub/Digital/Linux-Alpha/Miniloader
/test-images/noname. "milo" will boot from WNT ARC and fmu.gz is
a flash update utility that you can run from Milo to blow a new
milo into flash. Secondly, it looks like Milo is not initialising
the video device correctly. Again, this is cured in the newer milo
which now contains Digital's video BIOS emulation technology.
