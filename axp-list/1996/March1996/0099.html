Sorry to keep posting, but I feel like hacking at C code I don't understand
apparently.

I just got imap compiled for Linux/Alpha, and it's in the devel tree as
imap-3.6.BETA-3.axp.rpm.

For those who don't know, imap is a TCP/IP mailbox protocol like POP,
but much more flexible. Red Hat uses it for both its IMAP server and
its POP server.

This hasn't been tested a hoot, and I'd *really* appreciate it if someone
could try it out for me. This is the last big piece Linux/Alpha needs
to let ISP's use it for dialins.
