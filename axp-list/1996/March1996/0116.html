I've just finished my preliminary testing of the 3.1.2Dc XFree86 Mach 64
server. It appears that a problem I had with the shared memory interface
has cleared up. I'm not sure if it is due to changes on the server or
library side. The xkbd extension appears to work, but now the backspace
key really gives a backspace instead of a DEL (this is easily changeable).

The number of drawing glitches (seen mainly in programs that do a lot
of drawing like xgalaga or Maelstrom or rapid repeated posting/unposting
of menus) seems to have decreased. However, I don't know if that is just
coincidence.

Most of the other X programs (only compiled to 3.1.2D) appear to work.
Xterm runs without changes, but still requires 'stty sane' to get things
setup properly.

I'm now able to track the patches, and when the non-beta version of 3.1.2D
(or whatever it will be called) is available, I'm going to try and make
binaries available, if others would like.
