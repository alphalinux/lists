JK,
you don't have to flash milo, but if you want to, put fmu.gz
for your machine onto an EXT2 formatted floppy and reboot to Milo.
Then choose the "run" command with something like:

MILO> run fd:fmu.gz

Like Milo, it has help, so type "help" for the commands. "program"
is the one that you want and it will give you a get out (do you
really want to do this?".
