I was able to boot and install the redhat CD on my jensen. It wasn't
easy but I followed the instructions given in the FAQ. You need to use
the kernel provided with the CD rather than the one pointed in the FAQ.
The FAQ points to a special aboot set, ftp that and follow the readme in
the aboot-0.2/jensen-starter dir. but make sure you follow the new kernel
(from CD). I now have linux installed on a 1GB scsii HD and I can boot
with with: boot -fl "root=/dev/sda2" dka100
that brings the machine up and everything runs great. I am having problems
getting aboot installed. When I do the
swriteboot /dev/sda bootlx jensen.gz
It seems to install it fine, but then when I reboot, the system hangs on
Loading new kernel

I am also having lots of unalligned kernel something or another stuff.

I hope that helps and if you need more info please email me. Meanwhile,
if anyone can help me with aboot installation I'de really appreciate it.
