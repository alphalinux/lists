David (et al)
if you haven't already read it, please read the Milo HOWTO
on gatekeeper.dec.com ~ftp/pub/Digital/Miniloader/docs. That explains
how to use Linload.exe plus Milo to load a Linux kernel on an Alpha.
Then you'll need a kernel to load. There isn't a TGA Avanti kernel
handy on gatekeeper right now but in Miniloader/Kernels there is
a standard VGA kernel. If you have a VGA card, you could use that to
get you past the install to a point where you can build a TGA version.
