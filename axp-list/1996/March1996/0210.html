Larry,
first, use the latest Milo in /test-images/eb164/milo. This
has both VGA (BIOS) and TGA drivers in it and it contains the latest
kernel sources (these are much more stable for the EB164). As for
co-existing, WNT disk partitioning tends to have a DOS primary followed
by an NTFS partition. What's happening is that Milo doesn't have
support for NTFS (the latest has support for MSDOS, ISO9660 and EXT2).
What you need to do is to provide Linux with two (primary) partitions -
one for EXT2 and one for Swap and then use the Red Hat boot mechanism
to get you to the Red Hat installation so that you can format and use
them...
