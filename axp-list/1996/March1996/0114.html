On Tue, 12 Mar 1996, Martin Ostermann wrote:

> Does the aboot loader work with that image? Has anybody done that
> before? I allready installed the BLADE_0.2. Is this any help in
> bootstraping Redhat?

It's *lot* of help. Here's what you do:

Grab the jensen.gz kernel image from ftp.redhat.com or from the CD. Make
that your kernel using aboot. You should still be able to boot your blade
partition.

You should now be able to install by using the ramdisk and a Jensen bootdisk.
The only reason you need the later is to keep the install process happy, which
expects a bootdisk.
