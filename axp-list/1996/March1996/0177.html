> Hi All,
>
> I'm trying to boot Linux from the ARC console on an alphastation
> 200. I'm looking for the avanti kernel on the redhat CD-ROM and there
> doesn't seem to be one (there exists an avanti.img file but does this work
> with the ARC console?). What do I do? I have a tga video card.
>
> Thanks,
> David Bernstein

Grab the latest MILO for Avanti from off gatekeeper.dec.com in:

pub/DEC/Linux-Alpha/Miniloader/test-images/XL/milo

This MILO supports *both* VGA and TGA, whatever it finds on the system.

The CD file for Avanti with TGA is: ava-tga.gz.

You should be able to boot that directly off the CD, telling MILO:

boot -t iso9660 sr0:ava-tga.gz load_ramdisk=1 root=/dev/fd0

Good luck with your Mustang... :-)
