On Tue, 5 Mar 1996, Stephen Ollis wrote:

> I got the multia to boot from ARC and got to the prompt asking for where the
> linux kit was .. I entered Hard Disk, and partition 0 (only 1 HD in this
> box)..
> and got No Joy..

I'm not sure you can do the install from a FAT partition on the Alpha. It
needs files with long names, which is going to really confuse it. Is there
any reason you can't:

1) use a CD
2) ftp the files to another machine and nfs mount them from there

Whatever you do, you should copy the packages into RedHat/RPMS, not
RedHat/series/Base. We divide them up into different directories to make
downloading them easier (the RedHat-HOWTO at http://www.redhat.com explains
this better). You also need to get the RedHat/base and RedHat/instimage
directories. Be sure not to lose the exec bits in RedHat/instimage.
