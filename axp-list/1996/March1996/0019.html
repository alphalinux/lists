On Mon, 4 Mar 1996, Federico Noferi wrote:

> With Blade 0.3 I do: boot fd0:vmlinux.gz root=/dev/fd0 ramdisk=1440

Try using a Red Hat boot disk (ftp.redhat.com:/pub/redhat-2.1/axp/images)
instead, and use the command:

boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1 prompt_ramdisk=1

I don't think you need the prompt_ramdisk=1 part, but it won't hurt. If
that works, you can use that kernel to install either blade or Red Hat.
