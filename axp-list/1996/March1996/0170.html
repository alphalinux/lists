I just tried to compile the 1.3.75 kernel with TGA support. There is an
undeclared variable called __scrollback_mode in tga.c which prevents that
file from compiling. Removing the one refernece to that variable from
tga.c fixes the problem, and the resulting kernel does run on my AlphaStation.
It looks like this is left over from the console cleanups that Linus
recently did.
