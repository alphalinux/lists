> Last time I checked you have to set Memory size to 24, Linux will not run on
> anything less under Alpha. I know I read it some place change your setting
> to 24 and all should work ok. Do you know how to set it? How are you
> booting up Linux MILO or ARC?

Not true; I've run X on a 16Mb Alpha machine with no problems except swap
disk noise... :-)

Latest linload.exe and MILO should communicate well enough to be accurate
about the amount of memory in the machine; try getting the latest of *BOTH*
and see.

If, however, you've FLASHed MILO, it may have a harder time determining for
itself, as it normally depends on ARC and linload.exe to pass along this info.
Right, Dave Rusling?

> Should be able to change the mouse type in the configuration table.

Unless he's booting a canned kernel, ie one from the RedHat CD, where the
normal configuration is what's been chosen, and that means PS/2 mouse. And
that driver (psaux.c) doesn't seem to do much probing during bootup (as far
as I can remember), so it may say it's there when it's really not.
