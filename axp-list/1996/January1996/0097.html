Robert,

We're playing with an AlphaStation 250 4/266 here with 128MB of memory,
currently running 1.3.43 from Red Hat. Things seem to be working okay.
In terms of actual operation, if your memory size is indicating that
you're interested in servicing a large number of users, I believe you
want to modify linux/include/linux/tasks.h to increase NR_TASKS (usually
set to 512 tasks, but I have it set to 4096 -- 16 tasks * 256 users).
