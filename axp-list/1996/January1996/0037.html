A number of new packages are now available for anonymous ftp form
ftp.redhat.com in /pub/redhat-2.1/axp-beta/updates/RPMS (and SRPMS).
I suggest installing all of them.

binutils-2.6-1.axp.rpm
fvwm-1.24r-5.axp.rpm
gccaxp-2.7.1-1.axp.rpm
gccaxp-c++-2.7.1-1.axp.rpm
gccaxp-objc-2.7.1-1.axp.rpm
libcaxp-0.40-1.axp.rpm
libcaxp-debug-0.40-1.axp.rpm
libcaxp-include-0.40-1.axp.rpm
libcaxp-profile-0.40-1.axp.rpm
nfs-server-2.2beta4-4.axp.rpm

The debug and profile versions of libc are not needed for most people.

Please install all or none of gccaxp, libcaxp, and binutils. If you mix
and match them, you'll probably encounter problems.

Let me know if you have any problems.
