One feature I have discussed with Dave already, but not got going yet (so much to do and so little time...).

I need the ability to boot off the first available of a variety of devices. I (after a suggestion from Dave) am going to do this by implementing another env variable (BOOT_CMDS maybe); which takes a list of boot command lines - probably semicolon separated. The first command to succeed wins!

I need this so a "black box" product boots normally under most circumstances, but boots the CD-ROM if one is inserted (no keyboard so playing with the setup is tricky :-) ).

I am intending to do this very soon, but if anyone needs this functionality, or can contribute... (it looks like a small job), then get in contact.
