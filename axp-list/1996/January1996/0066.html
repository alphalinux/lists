After executing the BIOS initialization code, the emulator currently
tries to set video mode 3. Normally, the call should end up executing
code in the on-board BIOS, but for some reason, this doesn't seem to
happen with your card. You could try disabling the "#if 1" in the
milo file x86_sysenv.c (it's around line 399).

I don't understand why executing the emulator from the floppy vs. the
rom should make a difference.
