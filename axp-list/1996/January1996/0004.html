If you're using Red Hat/AXP please upgrade to the new rxvt immediately. The
shipped version had a major security hole.

To upgrade from any Alpha box on the internet, use this command:

rpm -Uvh ftp://ftp.redhat.com/pub/redhat-2.1/axp-beta/updates/RPMS/rxvt-2.10-3.axp.rpm

The same fix is also available for Intel machines.
