On Tue, 16 Jan 1996, Craig Groeschel wrote:

> I wanted to add a second scsi hard drive to the system.
> I used minlabel to set up the default label, wrote, rebooted,
> and the kernel would not recognize sdb2. I tried minlabel
> again, giving it this configuration, wrote, and rebooted:
>
> 2 partitions:
> # size offset fstype Size(Kb) Size(Mb)
> 1: 512 1 boot 256 0.2 # (Cyl. 0*- 0*)
> 2: 1037792 513 ext2 518896 506.7 # (Cyl. 0*- 506*)
>
> Erik mentioned mkbb. Should I try it? Where can I get it?

Red Hat ships mkbb. Try "mkbb /dev/sdb /usr/lib/lxboot"
