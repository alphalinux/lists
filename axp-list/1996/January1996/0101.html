Subject: SECURITY FIX: New dump RPM for Red Hat Linux

A security hole exists in all shipped releases of the "dump" package
for Red Hat Linux. Users should immediately upgrade:

Intel users:

rpm -Uvh ftp://ftp.redhat.com/pub/redhat-2.1/i386/updates/RPMS/dump-0.2e-4.i386.rpm

Alpha users:

rpm -Uvh ftp://ftp.redhat.com/pub/redhat-2.1/axp-beta/updates/RPMS/dump-0.2e-4.axp.rpm
