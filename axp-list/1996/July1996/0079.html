This is just a warning that my new-and-improved IP copy+checksum routine in
2.0.6 had some problems. I fixed a few bugs in my test-suite, and 2.0.7
should be fine (and if somebody wants to instruction-schedule the damn thing,
you're welcome - I've done as much as I can in C and inline assembly without
going totally crazy from looking at all the byte operations, now I'll have to
take a short vacation from that particular file for a few days ;).

Anyway, Linux/axp users should definitely upgrade from 2.0.6->2.0.7 (the new
2.0.7 also contains support for the Alpha XLT's)
