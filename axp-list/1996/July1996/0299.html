trouble@cybertsx.com wrote:
> 1)
> Is there a 2.0.??? kernel for the alpha yet... this 1.3 redhat kernel
> doesnt compile for anything... where are better docs for linux on the
> alpha...?

Yes, there is a 2.0.x kernel for the alpha. It is distributed
with all the rest of the architectures.

See http://www.azstarnet.com/~axplinux/ for the Linux/Alpha FAQ.

> 2)
> i have a multia desktop (UDB) small disk 340M i know its small, can
> someone direct me to the best place for axp/linux binaries ie. netscape....
> or once ELF is compiled in will it run most elf binaries around or does
> it all have to be ported.

There is no Netscape for Linux/Alpha. It'll be up to Netscape, Inc.
to decide if there ever will be.

As for ELF, it is a format for laying out machine-specific executables
and object files, not a byte code such as Java. Thus you can't just
run any "ELF" binary you find laying about, it has to be specifically
compiled for the alpha.

David M-T has a decent sized collection of binaries for anon-ftp on
azstarnet. FWIW, you can find Mosiac there.
