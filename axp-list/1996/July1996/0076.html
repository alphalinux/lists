On Sun, 30 Jun 1996, "Brian E.W. Wood" <beww@intac.com> wrote:
>
>RISC architecture is "reduced" instruction set computing. It can therefore
>take more instructions to perform a specific task than a "CISC" (complex
>instruction set computing) processor. Steve Gaudet at DCG told me he figured
>a 166 Mhz. alpha to be about equivalent to a 75 Mhz. Pentium. I have seen
>other reports which seem to think it is a lot faster than that.

Yes indeed !!
I did some neural-network simulations on a 166 MHz Alpha which mainly consist
of a big floating-point matrix (approximately 2000x2000 in size), and found it
to be between 6-8 times faster than a pentium 90 MHz, using the same OS,
source-code and compiler.
