On Wed, 17 Jul 1996, David Andrew Bushnell wrote:

> Hello All,
> I was wondering if it was possible to run Digital Unix binaries
> under Linux for alphas. If so what do you need to do to get a program
> to run. The reason I ask this is I tried to run denzo (a crystallography
> program) compiled on digital unix under linux and I got the error command
> not found. I made sure denzo was executable, in my path and the file
> type was alpha (denzo: ECOFF alpha demand paged not stripped - version 2864.)
> but I still got the command not found error. I am running Red Hat Linux
> version 3.0.3 on a Multia.
>

You must have /sbin/loader -- without it there is a file not found, even
for starting with absolute file path. After starting the load can crash if
you don't have the shared libraries for non statically linked objects
(/usr/shlib).
Some programs need more info (i.e. /etc/svc.conf)
