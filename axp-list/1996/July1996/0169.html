Harald> I'm looking for a gcc/axp running on Liinux/x86 and vice
Harald> versa incuding bintutils etc. to build a kernel for the
Harald> "other" box. if possible I'd like to have cross compilers

Building 32-bit binaries on a 64-bit machine should be no problem
(haven't tried it myself though). The other way round is calling for
trouble. Last time I looked into it, the binutils, gdb, nor gcc
handled going from a smaller to a larger target well. Jim Paradis at
one point had patches to build Alpha binaries on an x86, but I don't
think they were ever integrated into the main-sources.
