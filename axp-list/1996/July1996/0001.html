Earlier I asked how to find the dependencies of a dynamically linked
object file or executable on dUnix (for the purpose of copying[1] said
files to Linux/ALPHA, thus making dUNIX/osf binaries executable).

It is possible to do this on dUnix with the 'odump -Dl objectfile' command.

HTH.

dmd@3e.org Daniel Drucker

[1] For this to be legal, you must have a dUnix license on the machine.
