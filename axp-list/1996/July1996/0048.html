> I've read the relevant FAQ which states, more or less, that the XL-366 is
> fast and may someday run Linux, but that it doesn't yet do so, not at least
> without some kernel hacking.

True; but I hope to submit (untested) XL-366 (XLTurbo) patches to the 2.0.x
kernels within the next few days...

> My company is going to buy one of these beauties for me to run NT, but I
> want to run Linux too. Anyone have any plans to work on these machines?
> Any work already in progress?

As mentioned above, I've got untested patches I'm going to submit to Linus.
I've had no access to any of the XLT machines (300 or 366 versions) so I
can't tell how close I am, but they are very similar to the ALCOR (AS-600)
which support is already present in 2.0.5, although *not* for booting via
ARC and MILO, which is the *only* path available on the XLT... :-(

> I'm particularly worried about the chances that any of the graphics options
> for these machines will ever be supported by Xfree (or Xinside or Metro) in
> order to run X on this system. I'll probably be going with the PowerStorm
> 4d20 graphics option; but I'm pessimistic if it will run X.

Yes, you should be worried... :-(

There's no *current* plans to support the 4D40 (24-bit color), although it
will prolly work fine in console text mode, as there's a VGA controller on
board that card, I believe.

If and when for X support of that card is not even being contemplated at
this time. I suspect the 8-plane version, 3D30 (aka TGA2), support may stand
a better chance, being more of a commodity (read cheaper... :-) card.

> Thanks for the help. I'm trying to become another NT to Linux convert, but
> I've got to do it on the machine that the company will buy!

I don't know what the other graphics options are, esp. for 24-bit color.
Going with the MATROX Millenium is problematic, for LINUX, at this time as
well, as there's no announced third party X servers for Linux/Alpha, and no
planned XFree86 support at all...

Keep your eye on this mailing list, I'll announce support for the XLT when
it happens...
