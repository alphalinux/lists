Hi,

> Can any of the non-x86 CPUs that linux runs on be virtualized? It might be
> really nice to be able to run Linux under Linux as a user process for
> debugging purposes.

- MIPS can be virtualized though it's not easy for kernel/supervisor
mode code. Saying it's ugly fits better ...
- Motorola 68k can be virtualized for all CPUs from the 68010 on.
68000 and 68008 both have an unpriviledged "Move from SR" instruction
which would makes it possible for a program to detect whether it is
being traced/really in supervisor mode etc. Also the 68010 enhances
handling of bus error which were unrecoverable for the 6800[08].
