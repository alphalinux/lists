I'd like to thank everyone for their help. I got it working and installed. The
problem was that the RedHat kernel assumes that the root partition is partition
2, which was my swap partition. In general, the kernel seems to get very upset
if the root partition isn't part 2 (even if you tell the kernel that root is on
part 1).

After I did that, everything works fine.
