OK, why don't you try the stuff on starnet:

ftp.azstarnet.com:/pub/linux/axp/misc/axpelf-povray-test.tar.gz

It contains everything you need to run the test. It's an elf binary,
but it's statically linked, so as long as you have a recent Linux
kernel and ELF support enabled, it should work. I'd be interested in
hearing what numbers you get. I'm curious since povray looks quite
memory intensive (tons of tiny functions invoked via function
pointers, lots of variables accessed through memory and stuff like
that; occasionally, it does do some fp ops, though).

And no, I certainly don't mind if you share the Alcor results with
others. That machine is indeed quite fast (understatements are not
reserved for English folks, are they? ;-). Just a caveat: the sources
I have are nothing official. Every couple of months I tinker with the
sources a bit. Those may help performance somewhat, but I don't think
they explain the large difference we're seeing (unless my changes are
dramatically more important to the ev4 than to the ev5, which is
possible, but not likely).
