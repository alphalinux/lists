Early> I'd like to thank everyone for their help. I got it working
Early> and installed. The problem was that the RedHat kernel assumes
Early> that the root partition is partition 2, which was my swap
Early> partition. In general, the kernel seems to get very upset if
Early> the root partition isn't part 2 (even if you tell the kernel
Early> that root is on part 1).

Early> After I did that, everything works fine.

The kernel defaults to /dev/sda2, but you can specify anything you
want. With MILO, "boot vmlinux.gz root=/dev/sda1" would have done the
trick. Oh, yeah, this is all documented in the FAQ
(http://www.azstarnet.com/~axplinux/FAQ.html).
