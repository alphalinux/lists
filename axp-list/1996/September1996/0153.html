My ISP was having problems once again, so I finally bothered to
compile and install kermit for ELF (no, I really don't like minicom or
seyon or what not). A tiny patch is at:

ftp://ftp.azstarnet.com/pub/linux/axp/glibc/ckermit-190glibc.patch.gz

A (not so tiny) binary/man-page is available at:

ftp://ftp.azstarnet.com/pub/linux/axp/misc/axpelf-kermit-190.tar.gz

While not tiny, that tar file is 253006 bytes long (compared to about
1MB for the source tar file), so it's not bad at all. The wonders of
shared libraries. Obviously, you need a recent set of shared libs for
this to work.
