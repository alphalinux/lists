Sorry, I'd forgotten that all new platforms running NT were also running the
new AlphaBIOS console; MILO can't (yet) run from there. Assumptions were made
in the AlphaBIOS code that make it impossible to boot MILO as we've always
done from ARC (via the "boot selection" under ARC or "OS selection" under
AlphaBIOS).

We're investigating booting MILO via the "Run Program" path, but, as yet,
have made no progress.

Again, I'm sorry about forgetting about AlphaBIOS, and sending you off on a
wild goose chase... :-(
