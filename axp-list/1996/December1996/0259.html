According to Chris Adams:

> when I try to mount the CD (and yes, there is a CD in the drive). I
> have tried multiple cables and active and passive terminators. Is this
> just not going to work?

Probably. :-( If you have proper active termination, have Force I/O set
to YES when building a kernel, and selected the BSD NCR driver, then
that's about all you can do. This is problem that I had with a Philips
X8 CD-ROM. I bought a X2 IBM and haven't had a problem since. The UDB
likes it best when you attach cheap, slow peripherals...kinda like the
UDB itself. ;-)
