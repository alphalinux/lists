In case there are other LaTeX users out there interested in using
their own PostScript fonts: I purchased a couple of Adobe TypeOnCall
fonts a while back and finally took the time to install them properly.
By "properly" I mean that they are now installed in /usr/local with a
minimum of changes to the files that come with tetex (as distributed
by Red Hat). This makes it easy to keep the files around even when
upgrading to newer Red Hat versions.

The fonts are commercial, so I can't make them available publically,
but I do have a README file that should help greatly in setting up
your own local fonts. If interested, just give me a holler.
