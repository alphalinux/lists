Actually, your best bet might be:

XF86_TGA-3.2-bugfix.gz

in the same place as above. It is rock solid compared with its older
counterparts. I had server crash problems while running some apps in
the past, and after picking up the 3.2 release I haven't had a single
crash. The console-switching code isn't bullet-proof but subjectively
a little better for quick dashes to virtual console. The resolution-
swapping code isn't available here either, but at least now it vocally
complains that it can't do it. XF86_TGA-3.1.2-bugfix-EV4.gz has a
newer timestamp, but appears to be an older version of the X server.
