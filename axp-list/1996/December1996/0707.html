I did finally get it going with help from this list. I'll assume you
are working with the 1.06 distribution for Alpha. Unzip quake106.zip.
You will get a file called resource.1. Type:

lha -x resource.1

to unpack the DOS archive.

This will generate a .pak file in the directory id1.

Make sure the quake binary is placed in the parent directory of id1.

Kill as many background processes as is feasible to give the processor
less to do, then run a minimal window manager such as twm. Startup X in
640x480 mode. Be careful not to overload your colormap. Open only one
xterm or rxvt and type:

./alpha-ELF-xquake

Go into options and set the brightness higher and don't bother running
in full screen mode. Just use the default for a speedup.

You'll be good to go!

Thanks to Jay and Matthew for getting me going on this.
