I've been using NCR now for some Intel boxes, and I also don't have
problems with it using a HP streamer. My problems usually are cabling and
termination, and that's something where the ZIP is problem: It uses a
25-pin connector, and this is, according to different sources, not
acceptable for Fast SCSI bus, because it has to few Ground lines. The
right way would be to use a internal ZIP (which for me, has never produced
ANY problems), and if you need it portable, use a good external box.
