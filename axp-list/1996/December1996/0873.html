I seem to remember people having similar problems, but that was quite some
time ago. I think it happened when installing RedHat 2.1 from the InfoMagic
CD set. InfoMagic had put the distribution in a subdirectory, but the
install script expected it to be in the rootdirectory of the CD. The
workaround in that case was to mount the CD from VC2, select harddisk
install, and point the script to the right subdir.
