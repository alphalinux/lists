On Sun, 22 Dec 1996, Robert Allen Barker wrote:

> Does anyone know where I could find kernel sources that support the pc164
> AND are in either rpm version 2 format or tar.gz format?

Just get a plain tar file from ftp.redhat.com:/pub/mirrors/kernels/v2.0. That
should build out of the box.

> How to upgrade my redhat 3.0 rpm-version-2 to rpm-version-3?

Don't do this. All of the rpm-version-3 alpha packages really want things
like glibc. Getting the packages to install is easy but the chances that
any of them will actually work is virtually nil. You need to upgrade
to Red Hat 4.0 if you want to use recent packages.
