So, I'm trying to compile some C++ code under Red Hat 4.0, and the
streams library doesn't seem to match libio.h. Here's a simple example
program:

#include <stream.h>

int main(int argc, char *argv[]) {
cerr << "Hello World\n";
}

/usr/include/g++/streambuf.h:384: warning: implicit declaration of
function `int _IO_putc(...)'
