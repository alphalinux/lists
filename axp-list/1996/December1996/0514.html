Here's a technique that has possibilities...

------- Forwarded Message

Subject: Re: ARC, SRM, and the PC164

One way around the "only 1 firmware in the flash at a time"
problem is to flash your PC164 with the SRM console, and
put the ARC firmware .ROM image on a floppy diskette (formatted
with a FAT file system).

When you want to run NT, insert the floppy diskette into the
system and type the following command from the SRM console:

>>>arc

This command loads the ARC firmware ROM image off the floppy
and into memory and then executes it.

------- End of Forwarded Message
