> > Are any of the following 3D graphics cards supported under redhat linux, and
> > therefor have documentation available for writing drivers?

docs for the S3 ViRGE and ViRGE/VX are available from S3
if you're interested in 3D programming

> > 3. Diamond Stealth 3D (Virge VX chip)

Stealth 2000 3D is ViRGE (DRAM), Stealth 3000 3D is ViRGE/VX (VRAM)

for the Diamond Stealth 3000 3D a small patch is needed for
the intel server binary; all other ViRGE/VX work fine.

> > 4. Velocity 3D (Virge VX chip)

> Since then, I've ported the Millenium code to the Alpha, and Harald Koenig
> has gotten the Virge code to work, I believe, but I don't know the details
> about *which* Virge chips are supported, nor where any servers might be
> located. Harald?

my patch adds support for both ViRGE and ViRGE/VX.
please let me know if you're needing an AXP binary and/or source patch
(to be released RSN).
