For some silly reason, Redhat 4.0 on alpha will nto boot up unless it is
hooked to the netwrk it is set-up to connect to. If you specify a
gateway, it will not boot unless the gateway is active. This is really
silly to me, but its just how it is. You'll have to boot into single-user,
and chaneg the network config before it will work.
