Ooops, forgot about this one...

This is likely the same for all 4.0 distributions, surely for axp one,
and is immediately fixable.

/usr/lib/perl5/man entry for MANPATH is missing from /etc/man.config

Another nastie: updates clobber /usr/info/dir - which is likely
customized. No backup is made.

Other things to watch for:
If you had your own code in /usr/lib/emacs/site-lisp/ (say
site-start.el, or the whole auctex tree - for example) then this
has to moved now to /usr/share/emacs/site-lisp/.

/var/spool/at should be owned "daemon:daemon" and not "root:root".

Probably other things I forgot for now. :-)
