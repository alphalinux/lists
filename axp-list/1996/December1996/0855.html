I just put in ftp://ftp.redhat.com/pub/Incoming three files

e2fsprogs-1.06-0.axp.README
e2fsprogs-1.06-0.axp.rpm
e2fsprogs-devel-1.06-0.axp.rpm

These are ELF and shared libraries (e2fsck is static :-) current
e2fs tools for axp. Spec file in a distributed sources can be used,
after minor modifications, to recreate all RPMs, so I did not bother
with anything but binaries.

e2fsprogs-1.06-0.axp.README includes patch file applied to avoid
"unaligned trap" messages and other minor quibbles. You can use it as
well if you run 3.0.3 (ECOFF) distribution and would like to recompile
e2fsprogs on it.

Oh, in case somebody wondered; yes, I did install these programs on two
different Alphas on hand, a while ago, and they did not blow up. :-)
