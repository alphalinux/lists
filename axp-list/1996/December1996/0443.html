There was a problem with "mailx" not getting the screen size right on
Linux/Alpha. It turns out the real reason it didn't work properly is
because GNU libc doesn't seem to emulate the BSD ioctls (as some of
you suspected). In the "old" libc, this emulation happens in
libbsd.a(sgtty.o). This is at least true for the glibc version that
comes with RH4.0. If this has been fixed already, just ignore this
mail.
