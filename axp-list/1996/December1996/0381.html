Hi all.

> I've NFS mounted the CD on a Digital UNIX box, and it is successfully
> exported, but the installer says "That directory does not seem to contain
> a Red Hat installation tree." A listing of the directory follows. Is it a
> problem that all the filenames are capitalized and have strange ;1's?

I know nothing about Linux per se, but I would expect that the ISO9660-style
filenames are your main problem...

> If
> so, is there any other way to NFS-mount the CD to avoid this problem?

Yup. Use '-o rrip' in the OSF/Digital Unix mount string when mounting
the CD-ROM, _then_ export it via NFS.

This will give filenames that look like *nix instead of VMS.
