>>> John Banghart said:
> Is it possible to use the Powerstorm 3D card in my AS255 in a PC? It
> looks like a standard PCI bus, but I'm simply not sure.

The Powerstorm cards (aka TGA2) have a (rather dumb) VGA+BIOS onboard,
useful for running text mode console but not much else.

It *is* standard PCI, but it may be impossible to find an X server that
will work on the VGA part, if my sources are correct. Let me know if you
want to try a 640x480 monochrome server which might have a chance of working.
That'd be quite a come-down for a card whose cheapest implementation (3D30)
lists for $795... :-)
