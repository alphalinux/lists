Richard,

I assume that you are compiling Xaw3d-1.3L for XFree 3.2. I am trying to get
this to work as well and have discovered the following:

Comment out "#include <widec.h>" in XawI18n.h. Since the defines have
-DNO_WIDEC_H, this must have been an oversight that should have been #ifdef'ed.
This should get you past the "make depend".

There are several 32bit assumptions (warnings) about pointers/integers that need
to be fixed. Anything that I have run linked to the code segfaults out. I am
currently trying to debug it on my own. I will let you know if I have success.

David
