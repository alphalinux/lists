Franco Tassone wrote
>
> Well, probably this is partially off-topics, but i'm trying to configure
> sendmail for multi-domain email on my alpha xl-266 running RH4.0.
> Is it possible to keep distinct domain e-mail addresses on the same machine ?
> i.e : john@mydomain.com, john@yourdomain.com, john@newdomain.com.
> What's the best way for doing it.
> Thanks in advance to anybody who will help me to do this.
>

I am doing something similar using different hostnames.

1.) I defined my mailhost to be mail.ping.net.au
using the mx record in the dns for each of my
servers.

e.g:

dingo.ping.net.au IN A 203.1.176.242
IN MX 10 mail.ping.net.au
....

2.) In the sendmail.cf I added the following line:
Fw/etc/mynames

3.) /etc/mynames contains the fully qualified host-names
of all servers for which the mail-server is accepting
mail.

e.g:

mail.ping.net.au
hotdog.apana.org.au
fox.satech.net.au
....

Well this works for me. Correct me if there is a better way... ;)
