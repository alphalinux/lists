> Is there any way to accelerate the Xterm in fvwm?

Don't use xterm or nxterms...they are much bigger than they need to be.
Does anyone really care about Tektronix graphics? I use rxvt which
builds fine on the UDB. I combine that with AfterStep and the reduced
8-bit color icon set and I get reasonable performance. Some
applications still like to take all your colors though (ie. UAE).

Also, make sure you are using the Digital proprietary ELF X server.
