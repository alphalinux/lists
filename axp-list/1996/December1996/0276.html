On Fri, 6 Dec 1996, Iwao Jack Makino wrote:

> I was wondering if anyone has done Linux-Alpha with Macintosh
> networking, I would like to know how is works.
> I have several options. AppleTalk, EtherTalk, MachTen as unix
> connection through my Macintosh Ethernetcard...
>
> Any suggestion will be appriciated.

I recently tried networking my UDB with my father's PowerCenter 150 over
thanksgiving break. After working with it for about 2 hours, I was
finally able to open a telnet window to the UDB from the Mac. I was using
just straight ethernet and OT on the PowerCenter. I was unable to make
the UDB recognize the appletalk network, but granted, I wasn't trying real
hard. I WAS able to make a connection to the UDB from the mac, but only
after I made the gateway=NONE in the UDB ethernet setup...yes, it
complains about not recognizing NONE as a gateway, but it works. I'm not
sure what you are trying to do, but I was trying to use the mac's ppp
connection to setup a connection to the internet for the UDB. In the end,
the only REAL problem I had was that I could not make the mac run on ppp
and ethernet at the same time, so I was unable to use the mac for a
gateway, although, if I set up the modem on the UDB, it was fairly simple
to turn it into a gateway for the Mac.
