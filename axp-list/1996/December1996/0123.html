OK here is the scoop, I've done it, with both Linux and NT 4.0 and I can
tell you the following.

1. You need to go out and buy a new cartridge. This is because
You cannot remove the write protect from the cartridge that comes
with the JAZZ Drive. (No matter what Iomega says, or alludes to)

2. Save the receipt, return the original cartridge and say that it was a
gift, and was the wrong size cartridge type (whatever). Get your
money back, and you now have a free pre formatted Cartridge!!

3. Win NT automatically recgonises the drive when you boot (turn the
drive on before the computer!)and you can write to it, use it, knock
your self out. This is also the only way to use a JAZZ with Linux
also.

4. Mount the drive as SDB with an msdos file system, for example
On my fathers system i used the following command.

mount -t msdos /dev/sdb1 /mnt/jazz

NOTE : I had already created the directory /mnt/jazz

5. Just remember all the usual stuff when dealing with a scsi device,
make sure it uses it's own unique device number (mine is set to 3)
An make sure that the last device in the chain is properly terminated.
The jazz drive autodetect terminator scheme has worked well in the
Configurations I have used.

NOW ON A MORE IMPORTANT NOTE :

I can vouch for the in-effectiveness, slowness, and basic stupidity of the
technical support offered by I-Omega (that you PAY for). When I ran into
this problem I called EVERY day for 7 STRAIGHT DAYS. Got a lot of WRONG
answers, a lot of I don't knows, and finally they said I had a deffective
drive. It took them another 5 days to fax me back a RA number, which
really didn't matter by that time, because I sent it back to
Microwarehouse where I got it and got my replacement by mail before I got
an RA number!! (I have to admit ONE person was of the same opinion as I as
to the problem on day 4, and offered to help by sending a IBM
preformatted cartridge, but this idea was nixed by a supervisor)

I figured the problem out myself, and the problem is that they do have a
program that runs under NT4.0 or with the alpha archetecture that allows
the drive to be "reclaimed" (ie remove the write protect and allow a
format of the mac part of the disk) If they do, they don't know about it,
or how to use it.

The product has been running great, it is fast and so far (2 weeks of
heavy use) is reliable. I cannot say the same for I-Omega Tech support.
