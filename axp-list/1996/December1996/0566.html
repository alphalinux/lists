I'm installing Linux on a new UDB, which I haven't powered up yet, since
I've just gotten the disk, RAM and other components recently. I'm putting
up RedHat 4.0

Can someone please give me the quick tour of what SRM, ARC and MILO are,
and the distinctions between them?

Here's my (mis)conceptions so far; corrections are solicited:

1. SRM is the default console;
2. ARC is another alternate console, but the documentation I've seen
leads me to believe that it's ordinarily used only with Windows NT, and
isn't applicable to Linux.
3. MILO is a loader comparable to LILO, but that also loads some sort of
microcode. It runs in conjunction with either SRM or ARC, but not in
place of them.
4. SRM or ARC is an either-or thing: you run either SRM or ARC, but not
both (can you even install both?).

Like I said, what I've written above is probably at least partially
inaccurate, but I figuer it's better to set out my misconceptions to be
specifically addressed rather than a blank "what are these?" kind of
question.

Thanks. Sorry to be the latest pain-in-the-ass UDB guy.
