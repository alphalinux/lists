On Tue, 17 Dec 1996, Aaron wrote:
> Also does any one have the mode lines that will work with a NEC
> MultySynx 3v. I can get 640x480 and 800x600, but no resolution above
> that. I tried the mode lines that were given for a sony, but to no avail.

Below is excerpts from my XF86Config file that works well for my Nec 3V
monitor at 1024x768 on my UDB. If you want to try another resolution,
just pick one with the horizontal and vertical frequency ranges
listed. -HCH

# **********************************************************************
# Monitor section
# **********************************************************************

Section "Monitor"

Identifier "PCXBV"
VendorName "DEC"
ModelName "PCXBV-KA/KB"

# HorizSync is in kHz unless units are specified.

HorizSync 31-50

# VertRefresh is in Hz unless units are specified.

VertRefresh 55-90

###########################

# 1024x768 @ 60 Hz, 48.4 kHz hsync
Modeline "1024x768" 65 1024 1032 1176 1344 768 771 777 806 -hsync -vsync

EndSection

# **********************************************************************
# Graphics device section
# **********************************************************************

Section "Device"
Identifier "TGA"
VendorName "DEC"
BoardName "ZLXp-E1"
MemBase 0x02000000
EndSection

# **********************************************************************
# Screen sections
# **********************************************************************

Section "Screen"
Driver "accel"
Device "TGA"
Monitor "PCXBV"
Subsection "Display"
Depth 8
Modes "1024x768"
ViewPort 0 0
EndSubsection
EndSection
