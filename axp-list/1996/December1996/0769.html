> Yes, they all do. I get the impression that you can't run ARC without SRM,
> since SRM boots first and then says "Loading ARC Console"...

I've experimented with that. The SROM loads the first console firmware it
can find, whether that is ARC, SRM, or something else. However, since ARC
can't load SRM, if you want both SRM has to come first.

I've been working on getting Milo into the UDB flash. (Actually, I've got
it there but it isn't working. I'd like to position Milo as the first image,
and add commands to load the ARC or SRM console, but I'm not sure I know
enough details about how to load console firmware.
