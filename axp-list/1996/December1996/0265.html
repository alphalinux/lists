On Thu, 5 Dec 1996, Franco Tassone wrote:

> Well, probably this is partially off-topics, but i'm trying to configure
> sendmail for multi-domain email on my alpha xl-266 running RH4.0.
> Is it possible to keep distinct domain e-mail addresses on the same machine ?
> i.e : john@mydomain.com, john@yourdomain.com, john@newdomain.com.

In this setup, would the email to any of these domains go to the same
john? If so, I have this setup now on my company's mail server. Just
make sure you have a Cw for each domain in you r/etc/sendmail.cf. Like:

Cwmydomain.com yourdomain.com newdomain.com
