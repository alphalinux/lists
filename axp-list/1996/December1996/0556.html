>>> Jonathan Nicholson said:
> Hi, I've been trying to install Linux on an Alphastation 500/333. I've
> been having some difficulty. I can get MILO installed without too much
> problem (booting off floppy) and when I come to use the 'alcor' kernel
> boot disk it fails.

I don't think you should be booting that kernel from MILO, as it was
probably built to be booted directly (via "aboot") from the SRM console.

Try setting up the floppy for booting via SRM console; that means an EXT2 FS
plus "aboot" installed as the bootstrap, along with the kernel. Then boot
from SRM via:

boot dva0 -file vmlinux.gz -flags "load_ramdisk=1 root=/dev/fd0"

Refer to the SRM Boot FAQ at www.azstarnet.com/~axplinux/srm.html, IIRC.
