>
> I'm a little confused with what I've read on the TGA interface. I gather
> I _can_ use a VGA monitor, right -- as opposed to getting some sort of
> TGA monitor? What about SVGA?
>
> When shopping for a monitor, are there any UDB- or TGA- specific gotchas
> I should keep in mind, or should any old VGA monitor work okay?
>
If you look at the pinout it is standard vga, the only real gotchas are
the sync frequencies in the XF86Config file. At this point, the TGA
server cannot be programmed with arbitrary modelines so check the file to
see if the monitor that you are considering can sync at those values. Most
multisyncmonitors should have no problem. There are a few hitches if you want
to use a single frequency monitor, you may have to adjust the monitors sync
frequency to match the frequency in the XF86Config file. Also, at this
point I din't believe the TGA server supports sync on green.
