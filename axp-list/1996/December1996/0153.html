>>> Hans Kuhn said:
> I am trying to install redhat 4.0 and am having trouble selecting the images.
> The Redhat-HOW-TO states that a description of the images available for
> installation are in images/1213/image.txt. This directory/file doesn't exist,
> and I can't seem to find the answer in other documentation. I need images for
> a DEC Alphastation 255/233 so I can install Alpha Linux and be free from
> Digital UNIX 4.0. TIA.

You need to grab from gatekeeper.dec.com:

/pub/DEC/Linux-Alpha/Kernels/as255-srm.img

This is an image file ("dd" it to a floppy) containing a bootstrap and
kernel appropriate for being booted from the SRM console on an AS255. You
must install and use SRM console for booting, if your AS255 has AlphaBIOS for
booting NT. If it already has SRM, simply put the floppy in and say:

>>> boot dva0 -file vmlinux.gz -flags "load_ramdisk=1 root=/dev/fd0"

This ought to start the normal RedHat 4.0 installation procedure (make the
RAMdisk floppy ahead of time... :-).

Good luck.
