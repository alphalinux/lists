> > I just installed the RHS 4.0 on UDB166. I then get TGA server and

> > type 'startx', ghost image appears. I guess the parameter I use in

> /usr/X11R6/bin/xf86config or something like that which will prompt you to

That may only work with the TGA server from release 3.2 of XFree86. I'm fairly
sure it will *not* work with the 3.1.2 version from gatekeeper.

You *must* check, in the XF886Config file:

1. that the monitor specs for vertical and horizontal rates are correct for
the monitor you are using.

2. that the resolution/refresh rate chosen in the "Modes" line is appropriate
for that monitor as well.
