>>> Daniel Benson said:
> After realizing that the price/performance boost with running Linux
> on an Alpha is very large I am in serious consideration of buying one of
> these newer 500mhz based systems. Here is the exact configuration
> Enorex ships out:
>
> Ultra PC 500s $5599
> Digital 500MHz Alpha CPU

This is an EV56, has some new instructions (byte/short load/store) which are
about to be supported by GCC... :-)

> Digital PC164 Motherboard w/1MB Cache

Made by Digital Semiconductor, my employer... :-)

> 64MB Parity Enabled DRAM
> 4.3GB 9ms Fast SCSI-2 Hard Drive w/512K Cache
> Adaptec SCSI-3 Ultra Wide PCI Controller

Most probably a 2940UW, supported on Linux-Alpha since 2.0.23, IIRC.

> 3.5" 1.44 MB Floppy Drive
> Matrox Millenium 64-Bit 3D Video Card w/4MB VRAM

Supported under XFree86 3.2 when built with some Alpha-specific patches,
and run with kernel 2.0.26/2.1.13 and above.

> MAG 17" .25dp Technitron Digital Monitor
> 3COM Etherlink lll Combo Ethernet Adapter

Um, maybe a 3c590 or 3c509? Both have been ported to the Alpha, IIRC.

> Toshiba 6.7X 135ms SCSI CD-ROM Drive
> 104-Keyboard and MS Mouse
> Windows NT 4.0 Workstation
> FX!32 Binary Translation Software

A *really* neat bit of work, done by others in my group; translates
binary INTEL Win32 apps into Alpha binaries, among other things. See

http://www.service.digital.com/fx32/

for more enlightenment.

> 3-year Limited Warranty
>
> Now, I have gone through the FAQs and searched the web etc...The only
> problem is that I can't seem to find any information on the Linux support
> for this kind of machine. I'm not sure about the model number of
> the adaptec card, but I am guessing it is the 2940UW. Anyway, I have
> one simple question, will Linux run on this machine?

Yes, but...

The RedHat 4.0 distribution's kernels were built *before* the 2940UW driver
was ported. I put a working PC164 kernel, based on 2.0.21 plus a patched
2940 driver, in:

gatekeeper.dec.com:/pub/DEC/Linux-Alpha/Kerenels/pc164-axp-2.0.21-2940.gz

Also, for 2940 support, you'll need the latest MILO for PC164 from the
/pub/DEC/Linux-Alpha/Miniloader/ directory tree.

Good luck!
