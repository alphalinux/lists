Ray <ray@ray.ray.camelot.de> writes:
> > able to borrow a card from an old PC with good chances that it works.
> But obviously not good enough :-/ Could there be an error in the EISA
> setup? Do I have to set up the card somewhere? For some reason I can't do
> very much with the EISA setup tool, as it wants a "System Configuration
> Disc" when I add or remove a card, and just won't proceed without this
> disc (I tried to use the EISA setup disc, but it doesn't accept
> it).

Two reasons for this come to my mind. First, the disk needs to be
writable. Secondly, it might not have been copied correct. Try a 'dd'
copy of the original.

> But
> I was told that I don't have to set up an ISA graphics card in the EISA
> setup, so I have no idea what else could I do to get a working VGA setup...

Try different ROM versions, earlier and later ones. But make sure
you got v1.7 disks to switch back, because aboot doesn't work with
later versions.
