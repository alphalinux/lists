I would like to run big jobs on big multiprocessor AlphaServers.
(After all, isn't that the point of using Alpha instead of p5/p6?)
Please help me figure out if Linux/Alpha is up to the task:

. Does Linux support the full 64 bit virtual address space?
. What is the maximum RAM that Linux will support?
. What is the most swap space that your process can use?
. What is the largest file that you can create?
. Does the kernal fully support multiprocessing?

Please don't speculate. Only reply if you are very confident in your
knowledge, eg., you've actually created a file larger than 2gb or 4gb,
actually declared a 4gb swap partition, actually run a 4gb process on
your machine, or really brought up Linux on your Alpha SMP machine and
watched different jobs dynamically run on different processors.
Thanks,
