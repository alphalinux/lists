> Hi, I've been trying to install Linux on an Alphastation 500/333. I've
> been having some difficulty. I can get MILO installed without too much
> problem (booting off floppy) and when I come to use the 'alcor' kernel
> boot disk it fails.
>
> I'm booting from the SRM console
> | 'Digital AlphaStation 500/333 Console V5.0-13'
>
> Milo loads, but gives an error message:

The first thing I would suggest is not using MILO. You dont need MILO if you
are booting from SRM, and using it only complicates things. You will need
a kernel compiled for SRM instead of MILO to do this. Unfortunatly I dont
know where to get one, but they are around. Hopefully someone else can
inform both of us about that. Also, take a look at:

http://www.azstarnet.com/~axplinux/srm.html

Hope this helps
