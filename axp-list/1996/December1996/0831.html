I have no idea how to get back to a console in
order to do an orderly shutdown. Whenever I exit my window manager, I'm
thrown back into the login widget. Also, how do I jump into the "text"
mode console and back into X again?
...
I tried logging into a virtual console with the function keys, but
the TGA server doesn't like this idea.

I don't have a TGA, so I don't know whether it is indeed impossible to
switch back to the text console (Alt-Ctrl-F1); however, if it indeed
is the case, you will have to use the shutdown account; most
distributions have it preconfigured:

shutdown:*:6:0:shutdown:/sbin:/sbin/shutdown

You just need to change/set the password; you'd shutdown by logging in
as user 'shutdown'
