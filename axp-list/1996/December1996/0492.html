> . Does Linux support the full 64 bit virtual address space?

Well, it supports the full 43-bit virtual address space
supported by the current CPU implementations.

> . What is the maximum RAM that Linux will support?

There is no maximum. It will use everything you can fit
in the box.

> . What is the most swap space that your process can use?

The kernel is currently limited to 128MB per swap file, and
defaults to a maximum of 8 swap files. If I remember properly,
this later can be extended by editing <linux/swap.h>.

Assuming no resource limits are in force, an application can
use it all.

> . What is the largest file that you can create?

The ext2 file system only supports 2GB files, sorry.

> . Does the kernal fully support multiprocessing?

Not on the Alpha, sorry. I don't think anyone with the time
and skill has been able to lay hands on such a beast.

But it is supported on the Pentiums and Sparcs, so all the
groundwork is there, all that's needed for the Alpha is the
bits to actually control the hardware.
