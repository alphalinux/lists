> I'm not sure what I did, but when I updated to the latest TGA X-server
> (for the UDB) I was unable to run X windows unless I was root?
>
> That seems rather strange. The only files I changed were the actual X
> server and the /etc/XF86Config file. What permissions do I need to set to
> allow me to run X windows from any-body that is logged in. (Like it used
> to be!!)

The X server must be run as root, or be SUID to root. To fix this, login as
root and do:

chown root /usr/X11R6/bin/Xwhatever
chmod 4755 /usr/X11R6/bin/Xwhatever

Where Xwhatever is the name of the X server.
