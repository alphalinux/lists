>>> Brian Cruikshank said:
> What is the maximum memory size supported.
>
> I seem to be able to get 256 meg working, but not 512 Meg.
>
> This is on a PC164. I have to do a setenv MEMORY_SIZE 256 every time I
> boot. Is there a way to automate that? The OSLOADOPTIONS does not seem
> to be sent to Milo.

Hmm; the docs say 512Mb, with all 8 SIMM slots occupied. We had a spot of
trouble with 128Mb, but it turned out that using different SIMMs with fewer
memory chips aboard made it much more stable.

The fact that MILO is getting the memory size wrong usually indicates there's
a mismatch between linload.exe and MILO; make sure you're using the latest
version of both. linload.exe is found in the pub/DEC/Linux-Alpha/Miniloader/arc
directory on gatekeeper.dec.com.
