As stated, the patches don't work for me. The kernel hangs if I tell
it to use the serial console. Maybe that's because I had to cut my
cable down to three wires to make the SRM console work with my
terminal, and now it is missing the handshake lines -- I still need to
check details.

Anyway, even without the patches it is possible to get all outputs
except direct kernel messages to the console just by linking
/dev/console to /dev/ttyS0 and inserting an appropriate 'getty' entry
in /etc/inittab. So what you really need is a modified install disk.
I'll see if I'm able to build one for you, but it might take a few
days.
