Well, I tried my own suggestion:
boot RH linux, run fdisk via the "edit" button when choosing the disk for
Linux to reside on. Chose to edit the BSD disklabel, to get the head
positioned right where it is.
ALT-F2 to the other window, run dd (copied off
an ECOFF system) using "dd of=/tmp/sda", typing a mess on the keyboard,
ctrl-D to send it, and it overwrote the disklabel.
Then fdisked it, rebooted, and voila, it liked what it saw. Wow, dd is my
friend now. :) So, that about covers it.. If you ever run into something
of that sort ..
