>It might be easier (and a whole lot faster) to first try getting this
>list gated to a newsgroup in the linux.* hierarchy. There is already
>a linux.ports.alpha, but it seems to be dead. This would make for an
>easy transition.

True, but the linux.* hierarchy is not very well propogated. It isn't
available on my University's news server for example. I can't
seriously see a problem in getting a new group created. I think it's
best to try for a comp.os.linux.axp group, and carry on with this list
till it's created.

Can someone draw up a draft RFD and forward it to the list for
preliminary discussion? Since the concept of a c.o.l.axp/alpha group
seems to be agreed, we might as well start moving now.

For those who want to carry on reading by mail, a news<->mail gateway
shouldn't be a problem for someone with sufficient spare bandwidth to
set up.
