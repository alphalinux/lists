MILO _must_ build against a built alpha linux kernel. It assumes that the
kernel tree is in /usr/src/linux but you can pass the real tree name by adding
the KSRC onto the end of the make command, eg

$ make KSRC=/usr/src/bizarre/place/to/put/a/kernel/tree
