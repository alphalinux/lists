For those of you who wish to install Red Hat on Jensens: The easiest way
to do so is completely from CD-ROM. Unfortunately, the version of "aboot"
on the Red Hat CD cannot deal with the compressed image in kernels/jensen.gz

I solved this problem by burning a custom CD-R disc which has the
kernels uncompressed as well as some Jensen goodies from azstarnet.
I may also add support for installing firmware version 1.7. If anyone
is having difficulty installing Red Hat 4.0 on their Jensen, send me
your shipping address and I will send you a copy of this CD. Since
I only have a 1X CD burner, I would prefer to *lend* these CDs out
rather than give them away. It should be required only for initial
installation; once you're up and running, you can use the regular Red Hat
CD for installing new packages. I will include return postage with any CDs
I send out.
