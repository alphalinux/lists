system builds (and this popped up a week or so back) is in
the usage of some of the aux configuration things (like
menuconfig or xconfig). You all must realize that these
paths are unlikely to get regression tested for anything
but major releases, so if this kind of thing fails for
you, try the 'make config' standard thingie first....
