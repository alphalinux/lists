If you want to do this you must do:

format with "run a programm" from your windows NT cdrom ARCINST.EXE and
create a 5 MB sda1 system partition.

install your windows NT on one partition

than copy your linload and Milo to system partition via filemanager.

Add boot selection for MILO

partition your the left space from your harddrive as you want ( /, /usr,
swap)

install Linux

thats it. NO PROBLEM any time.
