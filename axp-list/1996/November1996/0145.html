> GASP, My father's UDB was running great, but somehow the Xserver got
> screwed up. I reinstalled, but forgot what I needed the answers where to
> the Xconfigurator questions. Whenever I try to run X windows (startx, xdm)
> it can't find the device, or path.

I have found it useful to use strace to figure out exactly what has
gone wrong. This works when there is a missing config file or things
have moved unpredictably.

If this doesn't work, you can always uninstall X and reinstall it,
reconfigure with the configurator, and you should be back up. Note
that this is often the only way to get MS Windoze to work when it gets
screwed up, so if it works for them, it should certainly work for you.
Probably, you can inspect the server and discover what has changed,
but sometimes it is faster to reinstall.
