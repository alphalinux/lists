Could people try out the new Linux/axp quake binary I just uploaded to

ftp://ftp.cs.helsinki.fi/
pub/Software/Linux/Kernel/testing/alpha-ELF-xquake-1.06.gz

The main difference between this and the previous ELF binary is just that
I got a source update to the 1.06 setup, so this binary should run the
newer world maps.

NOTE! It does _not_ run the older shareware world maps, because the newer
version thinks those world maps are modified versions of the newer 1.06
world maps! So unless you have the full registered stuff, you need to get
updated shareware maps.

I'm making just the xquake binary available for now, and I hope people
will test it out and verify whether it does indeed meet with your approval
before I start to look into the other quake stuff (xf86dga and i86
versions).

Linus
