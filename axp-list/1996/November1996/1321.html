> I am trying to get ppp working on my axp linux box and I cannot
> seem to find the ppp module. In fact, I cannot find ANY modules
> for axp kernel.
>
> [tanner@metal-forge tanner]$ rpm -q kernel-modules
> kernel-modules-2.0.18-5
>
> [tanner@metal-forge tanner]$ rpm -q -l kernel-modules
> /lib/modules
>
> That is it. Are there modules for axp linux? If so, where can I get them?
> If not, do I need to build a monolithic kernel?

I'm not sure if there are already prebuilt ppp modules in rpm's out
there. If there aren't, you won't need to build a big kernel, but you
will need to recompile the kernel, setting ppp as a module, and then
after you are done with the make boot, do

make modules
make modules_install

That should create the ppp modules for you.
