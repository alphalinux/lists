>>> Koef said:
>
> I have a DEC Celebris XL 5133, which has it's Intel CPU on a plugin board.
> If I swap the Intel CPU board for a AXP CPU module, will I be able
> to run Linux/Alpha on this machine?

Well, maybe soon; plugging the Alpha CPU duaghtercard in turns the box into
an Alpha XL 266, I believe. Unfortunately, that runs only AlphaBIOS, as the
console firmware, and we can't quite (yet) boot MILO from there, as we do
from the ARC console.

I've also heard that this is a rather pricey proposition, or at least it
*was* when it first came out. If you can't get a bargain price, it might
be better to into one of the faster and less costly Alphas now available.
