/usr/lib/libdl.so.2 is in glibc-0.960906-1. I'm surprised you'd
not have that if you'd installed libc at all.

> Also, why in single mode, ftp, telnet are all not working? Anyway to
> make them work?

They're not supposed to be running in single user mode. As that
is a maintanence mode and you don't want people logging in while
you're doing that.

However, you can manually start things up by running

/etc/rc.d/init.d/network start
/etc/rc.d/init.d/inet start
