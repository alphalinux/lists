>>> "T. Derek Brace" said:
>
> I have an extra buslogic BT958, would this work in my UDB (UDB not having
> arrived yet, so I can't try it out).

If it fits in the space available, it should work. That driver is known to
work under Linux-Alpha.

> Also, is it true the TGA graphics card in this box will only do 256
> colors?

Well, the TGA *chip* is capable of more than 8-bit depth, but whether the
other components (RAMDAC, clock, memory, circuits, etc) will allow it to run
other than 8-bit, I don't know. I've never heard of it being done. DEC built
TGA boards that did 8-bit, or did 24-bit, but not both, AFAIK.

I'd love to be proved wrong; get the TGA specs from www.netbsd.org and have
a go, as there's XFree86 code for a TGA server in the new release (3.2).
