Here we go again. Hopefully all of the problems associated
with last Tuesday's snapshot have been resolved.

BTW, even though this is a tar archive, you'll want to use
"cpio -i" to install, because cpio unlinks the old file before
extracting the new one while tar overwrites. The last means
that tar can corrupt the libc.so it is using to run itself.
