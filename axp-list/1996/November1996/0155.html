OK this sound like the ELF/ECOFF incompatability problem.

Basically make sure that the TGA-Xserver you are running is the ELF version
and that the Quake binary is also ELF. This is down to an incompatability in
the use of SHared memory.

If you don't want to use an ELF X-Server you can get the ECOFF (a.out) version
of Quake but it's missing the sprites and isn't really very stable.
