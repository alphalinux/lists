Since folks have been asking, I've built a binary snapshot of
the current glibc. It is rumoured that glibc release 2.0 is
eminent, so it would be nice to get a bit of feedback on this.

This build includes both the DES and MD5 crypt implementations,
shadow passwords, NIS, and LinuxThreads.

What else? Oh, the tarball expects to be unpacked under /.
