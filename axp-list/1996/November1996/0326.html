This is the problem I have had. Boot in single user mode and try
running the S10Network script. If you enable verbose reporting in
that file and in /etc/sysconfig/network-scripts/ifup, you should find
that it is haning in ifconfig or route.

My simple fix is to rename the /etc/sysconfig/network-scripts/ifcfg*
files so that the loopback configuration is read first (alphabetical
sort) and initialized first.
