> Can I install RedHat Linux AXP in DEC 3000 model 800 AXP workstation?

No. The 3000 series is based on the TurboChannel, not the PCI bus.
Red Hat Linux/Alpha only supports PCI/EISA/ISA-based systems.

If you must run a free OS on this box, I believe that NetBSD has been
ported to the TurboChannel machines.
