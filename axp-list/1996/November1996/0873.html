Hi Randy,
I noticed the HD of the Alpha chugging a little this
afternoon. Hope everything is working out for you. You now have an html
directory in your user directory. Feel free to build yourself a nice web
page. The server is up and there is a link to index.html in your html
directory.

I now have Secure Shell running on this machine. You might want to use
ssh as it also does compression. In fact remote X is quite tolerable
over a 56k line. Over T1 it should be pretty nice. Try it if you can,
just ssh to the machine and run xterm. Ssh tunnels the X protocol over
it's own secure compressed connection.
