> Can I install RedHat Linux AXP in DEC 3000 model 800 AXP workstation?

No. One of the BSD variants (NetBSD, I think) supports those
machines. It probably wouldn't be very hard to get Linux running (the
sources for the drivers are around, after all), but there doesn't seem
to be a whole lot of interest in doing so right now. Maybe later on,
when you can pick up those boxes for 200 bucks or so... ;-)

> Can I install RedHat Linux AXP from the CD-ROM or from Network?

The answer is yes to both. CD-ROM works fine and for the network you
can use either ftp or NFS installation. But since the DEC 3000
machines are not supported, this is probably not very helpful, I'm
afraid.
