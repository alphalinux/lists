>I installed RedHat-4.0 in a UDB. Every time a switch on the machine
>it comes the firmware prompt. To boot milo I am using a floppy.
>is there any way to install milo in the system without using a floppy?

normaly you make your first partition a type 4 partition only 1.44 M big.

trie it on the end of your disk.

then dd if=/dev/fd0 of=/dev=sdx (x is for your partition number)

add a new boot selection via the arcconsole

should run
