a pc164 (Aspen Durango) and starting
a Linux installation? Is there a HOWTO?

ARC firmware is loaded). I've configured
the boot menu to boot from the floppy disk
as follows:

LOADIDENTIFIER linux
SYSTEMPARTITION multi(0)disk(0)fdisk(0)
OSLOADER multi(0)disk(0)fdisk(0)
OSLOADPARTITION multi(0)disk(0)fdisk(0)
OSLOADFILENAME
OSLOADOPTIONS nodebug

Checking linux...
SYSTEMPARTITION cannot be found, value is:
multi(0)disk(0)fdisk(0)

and I get:
