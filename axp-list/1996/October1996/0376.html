At 02:01 AM 10-12-96 -0500, you wrote:
>Finally I get to a point where I boot linux and start the RedHat install.
>When prompted to select the partition to install to, naturally, I opt to
>edit it. However, fdisk does not recognize my RZ28M and wants me to
>manually fill in the geometry for the drive. That fine, if I had that info.
>I searched the archive for info on that drive. The info i got refreed to
>minilabel. I found minilabel but now do not know how to get that to the
>install disk or partition. Any direction would be appriciated. TIA
What you need to do is move the highlight to Done, press enter, then press
Alt F2, and cd /usr/bin
there you type minlabel /dev/sda (I'm assuming the disk you are installing
to is the first SCSI hard disk).
Reply Y to the query, then look at what minlabel says at the top of the screen.
Sectors per track and cylinders are shown. heads = tracks per sector.
Write this stuff down. then q to quit, reply that you do _not_ want to write
the label!
Now type fdisk, x for the expert menu and do the c for cylinders, h for
heads, and the s for sectors/track info entry.
Type r to return to the main fdisk menu, and proceed to make the partitions.
You will get lots of unaligned access messages. Ignore them. When you are
done type p to see your partitions. Lots more unaligned access messages,
ignore again, and type p one or two more times to get a clean partition
structure listing on the screen. Don't forget to type t and tell it which
partition (1,2,3 or 4, whatever you chose) is your swap partition, and
change this type to 82. If you have a DOS partition to create, this is type 4.
Type w to write the partitions, and then when bcack at the prompt, shutdown
the machine. Don't worry that shutdown doesn't work, just give the old
3-finger salute. Put the 2nd disk ( the boot one, as opposed to the #1 milo
or #3 root disk) in the floppy drive, and it will reload the linux boot and
you get to continue to the point where it asks if you need to partition your
disk. This, now is done, so you can tell it no.
Carry on with the build and be happy!
