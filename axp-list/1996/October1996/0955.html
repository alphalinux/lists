On Tue, 22 Oct 1996, Jeff Gehlbach wrote:

> In the hands of a capable but busy administrator, is Alpha Linux
> sufficiently stable to support a full newsfeed? How about a
> moderate-volume commercial http server? Any input is appreciated.

We have a 166 MHz UDB with 128 MB running a inn1.4unoff4 server without
any problems. We take three full incoming NNTP feeds plus a full
compressed UUCP feed, plus two full outgoing feeds, and we're running
smooth as glass. Hell the thing is even running a late September beta of
Colgate and it's running fine. :-)

One thing I _have_ noted, FYI, is that inn doesn't want to run as an ELF
binary. It'll run as ECOFF, but if you try to run it as ELF it'll
mysteriously die after about 10 minutes.
