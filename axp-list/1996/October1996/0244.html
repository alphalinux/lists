On Mon, 7 Oct 1996, Dave Mitchell wrote:

> This is on a udb-166 - I don't have any prompt like '>>>' anywhere.
> In MILO if I do 'show' it doesn't give an ethernet device, but I don't
> think that's the problem.. :(

>>> is the SRM console prompt. If you boot using ARC, you might not see
it. You need to select "Digital UNIX Console" from the setup menu and
then cycle power to get to SRM. Once you've gotten bored with SRM, you
type "arc" to change back to the ARC firmware.
