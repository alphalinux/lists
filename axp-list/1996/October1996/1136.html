I too have had many problems compiling kernels on my UDB. but have worked
through most of them.
My problem now is I can't get the MSS to compile into the kernel. I get
numerous errors about the MSS
when I run make boot.

But to help you compile a kernel here is how I do it.
Get the source you want to run and put in the /usr/src/linux directory, then
you need to get the axp-diff
to match the kernel version you got from ftp.azstarnet.com/pub/linux/axp/sys
then apply it to the kernel
by coping it into the /usr/src directory and then change into the
/usr/src/linux directory and run
'zcat <axp-diff.patch>.gz | patch -p1' after this you need to make config
or make menuconfig or make xconfig
to specify what you what in the kernel. After this is done run 'make dep ;
make clean' after this run 'make boot'
this will create the new kernel in the /usr/src/linux/arch/axp/boot
directory just copy it to the / direcorty.
I suggest you name it something different in order to save the one that is
already out there, like vmlinux2020.gz
That's all the info I have on it.
