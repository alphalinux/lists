Hmm. I thought you were replying to my questions on the list and so
knew what was going wrong.

First was the problem with the rc scripts crashing. The trouble was
that route and ifconfig were trying to resolve addresses via the name
server. The calls were hanging up while requesting information from
the name server. nslookup works fine, but these program use the
resolver library and fail. I worked around it by editing the
host.conf file to ignore the name servers and use the hosts file. Not
an adequate long-term solution.

Once the system would boot, I found that emacs without X doesn't
install correctly. I never did find the emacs executable. tcsh
completion is broken. I recompiled tcsh, but this did not fix
completion. At this point I was finding myself overcommitted in time,
so I dropped the project. I have seen some updates announced on the
list, but none have addressed the problems I'm seeing.
