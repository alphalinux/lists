you are missing nnswitch.conf in your /etc dir. This seems to be a
bug in redhat. Here is one that works for me. (gratefully stolen from
someone else :)

# /etc/nsswitch.files:
#
# An example file that could be copied over to /etc/nsswitch.conf; it
# does not use any naming service.
#
# "hosts:" and "services:" in this file are used only if the
# /etc/netconfig file has a "-" for nametoaddr_libs of "inet" transports.

passwd: files
group: files
hosts: files dns
networks: files
protocols: files
rpc: files
ethers: files
netmasks: files
bootparams: files
publickey: nisplus
netgroup: nisplus
automount: files
aliases: files
services: files

You want the local dns to look at files (ie /etc/hosts) before going
out and looking at dns.
