Dave,

Are you using Coax or Unsheilded Twisted Pair? Following is what I had to
do to get my machine to talk over Twisted Pair instead of Coax... Maybe
yours is the opposite?

Cold Boot.
>>> show ewa0_mode
ewa0_mode AUI
>>> set ewa0_mode twisted
Change mode to Twisted Pair
>>> show ewa0_mode
ewa0_mode Twisted Pair
>>> boot
__
Press that \/-shaped button.
Go into the firmware setup
