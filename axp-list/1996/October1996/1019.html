On Thu, 24 Oct 1996, RHS Linux User wrote:

> Hi, There,
>
> I am running RHS 3.0.3. When I tried to install some newer packages
> (e.g. rpm-2.2.5-1) I got he following messeges:
>
> > rpm -Uvh rpm-2.2.5-1.axp.rpm
> only packages with major numbers <=2 are supported by this version of RPM.
> error:/ rpm-2.2.5-1 cannot be installed.
>
> How can I upgrade my packages?

I'm assuming that you have not ever upgraded RPM before, and if that is
the case, then you are are still running a 2.0.x version of RPM and you
need to get a 2.1.x version. I found one on the redhat ftp site - I think
it was version 2.1.6 or something. I don't remember what directory it was
in, but trust me it is there somewhere. Once you get a 2.1.x version
installed, you can then install a 2.2.x version.

Hope this helps and makes sense, as I've just woken up.
