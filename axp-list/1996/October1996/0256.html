>> "Amit" == Amit Margalit <amitm@amitpc.iso.dec.com> writes:

Amit> I have a UDB with RedHat 4.0 installed. I changed the
Amit> initdefault in /etc/inittab to 5, and everything's OK. No need
Amit> to remove anything. Sometimes, after a few logins/logouts, XDM
Amit> doesn't start the X properly.

If you use the TGA server it seems to be necessary to put the
following in /etc/X11/xdm/xdm-config to ensure that xdm comes up after
any logout:

DisplayManager*resetSignal: 2

This works for me,
