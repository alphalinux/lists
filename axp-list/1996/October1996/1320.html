> Can someone please tell me where the second
> serial port on my multia is???

It may sound silly, but the second serial port is presented on the
same 25-pin connector as the first port. The trick is that it uses
rarely used pins and you need to make up a splitter cable to use
both ports. This is the infamous "Y-Cable" - near unobtainable, but
check out http://www.annex.co.uk/systems/udbycabl.html if you want
to make up your own cable...

One other point that I get asked very frequently is "do I need a
Y-Cable to use a modem" - the answer is a definate "NO" - using a
standard modem cable works just fine, however, you loose access to
the second port, for most folks this is not important.
