The components needed to use Digital UNIX are as follows:
1) a console on the machine - SRM
2) Media and docs set to install
3) A license to use it - Options: 1 user development, 2 user, or more.
Common is 2 user, as the 1 user is normally only available with the
developers toolkit which costs many $$ ( or guilders ;-} )
