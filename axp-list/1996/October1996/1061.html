OK folks. See gatekeeper.dec.com in ~ftp/pub/Digital/Linux-Alpha/Miniloader/src.
There are two current source tar files. milo-2.0.23.tar.gz and milo-2.1.4.tar.gz.
These are only a few differences between these and the previous sources but the
main one is a bug fix which allows 'shutdown -r now' to work again (it broke).
Also there are milo*patch.elf.tar.gz files for each of the two sources to
allow them to build in an all-ELF environment. Sometime today I'll update the
MILO HOWTO with some ELF information.
