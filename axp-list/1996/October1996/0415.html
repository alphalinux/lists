>>Does anyone know which axp.rpm these files can be found in ?
>
> You most likely made a mistake while patching your kernel with the
>AXP-diffs : you should have gone to /usr/src/linux and patched -p1,
>instead of /usr/src and patch -p0. This command fails with files diffed
>against a non-existent file in the mainstream source tree. You should
>have a /usr/src/386 directory containing all the files created that
>way, along with an empty .orig : just move all the files in the
>/usr/src/linux tree in the same place.

Thank you very much !

-p1 is the ticket ...compiling a new kernel now
