sheesh. does no one ever read past axp-list email? i'm currently
running 2.1.3 with modules and kerneld on a udb. the changes that need
to be made are minimal. here's what you need to do to get it to
work (this is with modules-2.0.0):

1) get rid of those bad syscall references in kerneld.c. you don't
need them anyway.
2) compile depmod with BFD support.
3) add the autocleanup and syslog bits that modprobe uses to
insmod-bfd.
4) strip --strip-debug the modules. insmod-bfd can't figure out the
correct module size for some modules if you don't do this.
5) install and live happily with autoloading of modules.

the only major thing that needs to be done for 2.0.0 is to BFDize the
insmod. bjOrn?
