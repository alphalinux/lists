Well, it was a somewhat longish wait, but I'm very happy to find that
the Alpha Architecture Reference Manual is now online. I found this
URL in a post on comp.arch:

http://ftp.digital.com/pub/Digital/info/semiconductor/literature/alphahb2.pdf

It documents version 3 of the Alpha architecture and seems to include
all the "new" instructions (IMPLVERS, AMASK, sub-word load/store,
multimedia instructions).

I'm also happy to see that the Alpha multimedia extensions use the
regular integer register set. That makes them _much_ more
interesting, IMHO. Come to think of it, it may not even be so hard to
make gcc generate (some of) them automatically. Now _that_ would be
an interesting addition.
