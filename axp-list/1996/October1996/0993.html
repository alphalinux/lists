> Can you tell me something about this card? Is it a frame grabber
> only?

Yes.

> Does it work stand-alone, or does it need a VGA card?

For displaying it needs an extra graphics card, preferrably one with
linear memory (Matrox recommends Matrox Milenium). A 4MB VRAM S3 card
works well here.

> Does it compress?

A collegue believes to have read something about motion JPEG, but I
don't know whether this was in hardware.

The FreeBSD/Linux driver doesn't provide compression.

For hardware details see http://www.matrox.com/imgweb/home.htm
