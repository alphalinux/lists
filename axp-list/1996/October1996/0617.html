> Flash management utility complains "this flash is not intel 27f020"
> Any suggestions?
>
Don't use FMU on the UDB boot via WNT ARC. If you're feeling like
delving into the code, you'll see that it checks for a couple of types
of flash device before proceeding. If it doesn't recognise the flash
device (it reads an id field) then it will not do anything else.
