Ok, someone was looking for the multia jumper settings, so here
they are:

DEC Alpha Multia Jumper Settings
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

System Board Jumpers
+--------------+------------+--------------------------------------------+
| VX40 | VX41, VX42 | Meaning
|
+==============+============+============================================+
| W9 | W12 | Mini console (Auxiliary)
|
+--------------+------------+--------------------------------------------+
| W8 (default) | W10 | Normal power-up sequence
|
+--------------+------------+--------------------------------------------+
| W7 | W11 | Attempt a floppy boot fail safe load (FSL)
|
+--------------+------------+--------------------------------------------+
| W6-W3 | W6-W3 | Graphics Display Resolution (see below)
|
+--------------+------------+--------------------------------------------+
| W2 | W2 | When connected to a switch, causes a reset
|
| | | or halt (see J8)
|
+--------------+------------+--------------------------------------------+
| J8 | J8 | pos 2-3: reset (default)
|
| | | pos 1-2: halt request CPU interrupt
|
+--------------+------------+--------------------------------------------+
| | W9-W7 | See below
|
+--------------+------------+--------------------------------------------+
