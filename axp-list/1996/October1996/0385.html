To install a binary rpm do an `rpm -i RPMFILE`.

Building a source rpm is a tad more complex. You need to start by
installing the rpm, just as you were installing a binary rpm:
`rpm -i SRPMFILE`. This will unpack the rpm to /usr/src/redhat/. To
build and install the rpm, you need to execute
`rpm -bi /usr/src/redhat/SPECS/name-of-rpm.spec`.

If something goes wrong with the build, you can edit the source code,
which resides in /usr/src/redhat/BUILD/name-of-rpm/. Then, when you
go to restart the build, you need to do a
`rpm -bc --short-circuit /usr/src/redhat/SPECS/name-of-rpm.spec`
to rebuild it and a
`rpm -bi --short-circuit /usr/src/redhat/SPECS/name-of-rpm.spec`
to install it.

For more information, check rpm(1). There's a howto somewhere on
RedHat's web site, also.
