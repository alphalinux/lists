Just thought I'd drop the list a quick note.

The November issue of the Linux Journal featured graphics, a topic I've
always been interested in. In it, they mentioned Blue Moon Rendering
Tools. This is a suite of RenderMan compliant tools for graphics that
lends itself for animation. (For those who are interested but
unfamiliar with RenderMan, it is a graphics language used by high end
tools such as those used to create Disney's Toy Story).

I loaded the "OSF/1" version and I was able to run it with a fair degree
of success. It complained slightly when it started but I was able to
render the limboing desk lamp successfully.

Hope this helps anyone who is interested in graphics and animation.
