Hello *,

Shane Anderson wrote:
} The other reason I bought the UDB is because I read in the Linux
} Journal (and on-line as well) that the Alpha platform is just as
} stable as the Intel platform. After 'lurking' on this list for the
} past week I'm getting the impression this isn't true.

The Linux/Alpha is as stable as the Linux/Intel: rock-solid.

Uptime on my NoName/266 (mail/news/fax-server, at a quiet moment):
10:22pm up 8 days, 1:21, 3 users, load average: 0.00, 0.00, 0.00
And rebooted only to upgrade the linux version

Only when you start dabbling with the interiors of Linux,
or when you are on the front end of the development, e.g.,
beta testing very new drivers, then you may see the odd
system instability. But this is the same for Linux/Alpha as
it is for Linux/Intel.

What _is_ true is that not all Linux/Intel software is also
available for Linux/Alpha. Most publicly available software
has been ported (when porting was at all necessary) and recompiled
for Linux, but the (semi-)commercial software is notably missing,
e.g., no netscape, no applixware.

} Just how impressive is Linux on a 433 Mhz (or higher) system?

I'd reckon: awesome. Just imagine: 428 BogoMips: Wauw. But
the fastest that I have heard of is the 333MHz Alcor Alpha
(at 331.35 BogoMips).
