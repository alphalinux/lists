I've successfully installed RedHat 4.0 on my EB66, and so far everything
works OK, except..

When I try to boot my newly rebuilt kernel 2.0.21 (built without errors),
milo (test image dated 11/10/96 downloaded from gatekeeper.dec.com)
reports:

Expected 1, not 2 program headers

and it fails to boot that kernel.

Kernel from original RedHat 4.0 distribution boots OK.

Luckily, on my second partition I have old RedHat 3.0.3 installation and
2.0.21 kernel built with it boots just fine. So for now, I use that
installation only to build new kernels.

How can I convince milo to boot a kernel built with RedHat 4.0 ?
