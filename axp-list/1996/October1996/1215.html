> After applying all the Colgate errata for AXP, I couldn't log in anymore. What
> do I need to do to fix it ?

It sounds like you didn't follow the intstructions carefully enough...

Try booting in single user mode and install the util-linux package. You
may need to use the --force flag.

> How do you bring Linux to run level 1 by the way ?

Pass the "single" argument to the kernel at boot time or run /sbin/telinit 1
after it's running.
