All,
the sources for Milo have been updated on gatekeeper.dec.com.
These can be found in ~ftp/pub/Digital/Linux-Alpha/Miniloader/src.
The new files are:

milo-2.0.23.tar.gz
milo-2.0.23.elf.patch.gz
milo-2.1.4.tar.gz
milo-2.1.4.patch.elf.gz (nothing like consistancy is there?)

I fixed a little bug which stopped 'shutdown -r now' working on
some systems. I have build all new 2.0.23 based images for
test-images - these will appear overnight (EST). There are more
systems built for and more images (eg mboot/bootm/milo.dd for
pretty much everything). Oh, and all of these images contain the
new, faster, BIOS emulation code from Digital.

I'll update the HOWTO sometime today or maybe tomorrow with ELF
details.
