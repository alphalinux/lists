Tony McNeil <tmcneil@abts.net> writes:
> kernel linux-2.0.21, when I run make config it won't let me change the
> processor type from and intel to axp, I've found info that pointed me to the
> axp-diff and have download a couple different versions from
> ftp.azstarnet.com/pub/linux/axp/sys/ I can't figure out how to apply the
> patch ( I tried /usr/src/linux/Makefile ) but it really messed things up.

Install a clean kernel source in /usr/src, then cd to /usr/src/linux,
and apply the patch with the same version number with 'zcat
<axp-diff.patch>.gz | patch -p1'.

This is just memory, but it should be OK.
