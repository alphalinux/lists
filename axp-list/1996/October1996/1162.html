Jay> OK, here's the breakdown in capabilities:

Jay> Jay> ZLXp-E1 8bpp ONLY, Bt485 RAMDAC, 2Mb VRAM
Jay> Jay> ZLXp-E2 24 bpp ONLY, Bt463 RAMDAC, 8Mb VRAM
Jay> Jay> ZLXp-E3 24 bpp ONLY plus Z-buffer, Bt463 RAMDAC, 16Mb VRAM

According to Brooktree's web site, the Bt485 can do 32 bit true color
mode (however, it _cannot_ do 24 bit/pixel packed mode---only the 485A
can do that). With 2MB of VRAM, it should be possible to just barely
support 800x600x32bpp mode (leaving just ~170KB of VRAM for off-screen
caching, which might not be so great). But even a 640x480 true color
mode might be useful for doing some image processing.
