Since this came up recently: the latest test release of glibc has now
NIS (aka YP) support. If you'd like to use NIS and want to live on
the edge and don't mind some development work or bug fixing, you might
want to give it a shot. Otherwise, you might just want to wait for
the next release of a Linux distribution.

Oh, if you're interested in glibc and ELF development, you might want
to subscribe to axplinux-libc@azstarnet.com (see FAQ for details).
