"Petr Vanicek" <vanicekp@vinnetou.site.cas.cz> writes:
> it can be install RedHat 4.0 Colgate in DEC 2000 Model 300 AXP?

Yes. The machine is usually referred as the "Jensen" (that's the name
of its motherboard).

> If yes, how is he install MILO loader?

That's not possible. You have to use the SRM console and Aboot. Refer
to the FAQ's Jensen section at

http://www.azstarnet.com/~axplinux/FAQ-9.html

or (since you're from Europe) my local copy at

http://www.comnets.rwth-aachen.de/~ost/alpha/FAQ-9.html

Details about using SRM and Aboot can be found in the SRM-Howto at

http://www.azstarnet.com/~axplinux/srm.html

Good luck, Martin
