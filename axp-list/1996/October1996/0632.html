Greetings all -

I have a noname box running RH 3.0.3, and would very much appreciate any
help with the following problems which vary from very important to just
sort of annoying.:

1. I have FTP running and although annonymous FTP seems to be working
well, no user aside from root can log in (from another machine) for FTP
transfers. All users who are already logged in to my machine can use FTP
without any problems, but when trying to FTP to the system from the
outside, only the root account is allowed in. Aside from root, FTP
doesn't even ask for a password:

Connected to biophy2.phy.tufts.edu.
220 biophy2.phy.tufts.edu FTP server (Version
wu-2.4.2-academ[BETA-11](1) Fri Apr 19 10:33:45 EDT 1996) ready.
Name (biophy2.phy.tufts.edu:dalbert): dalbert
530 User dalbert access denied...
Login failed.
Remote system type is UNIX.
Using binary mode to transfer files.
ftp>

Is this simply a case of my having to alter permission settings
somewhere, or is it something more sinister?

2. When using GLINT, I occasionally get an error message saying that there
is a version problem (>= 2). Is GLINT telling me that _it_ is not up to
date, or is this a case of my needing a change in RPM?

3. In some applications (such as netscape run on a different machine
but operated from the noname by issuing an 'xhost +' command here, and
a 'setenv DISPLAY biophy2.phy.tufts.edu:0' there) and windows within XF86,
the Backspace key seems to be a little wierd. In a standard xterm window
it operates normally (at least what I consider 'normal' - the character to
the _left_ of the cursor is deleted), however in others it is reversed.
Characters to the _right_ of the cursor are removed. Is it possible to
change this, perhaps with a modification of the keymap that I am using?
(Or is there a keymap other than .us that I should use?) Also, in some
applications run within an x-term, the backspace key results in a '^?'.

4. Here is a semi-silly one... In plain text mode (when first logging in
to the system before issuing 'startx'), there is no cursor at all.
Commands can still be issued, including full use of the backspace key
(operating normally!), but there is no block or underline cursor. How
can I rectify this?

Thanks very much in advance for all help!
