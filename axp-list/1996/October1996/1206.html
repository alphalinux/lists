I've compiled and am running 2.1.5 on an Intel platform, and it went
without problems. When I went to compile it on the Alpha, I had to make
the following patch to get it to work. The fs/namei.c compile blew up
because of this.

Sorry if this has already been posted and I overlooked it.

-- Michael

--- segment.h.orig Mon Oct 28 12:53:27 1996
+++ segment.h Mon Oct 28 12:37:23 1996
@@ -293,7 +293,7 @@
long __sfu_len = (n), __sfu_ret = -EFAULT; \
if (__access_ok(((long)__sfu_from),__sfu_len,__access_mask)) { \
__sfu_ret = __strncpy_from_user(__sfu_to,__sfu_from, \
- __sfu_len, &current->tss.ex); \
+ __sfu_len, &current->tss.ex); } \
__sfu_ret; \
})
