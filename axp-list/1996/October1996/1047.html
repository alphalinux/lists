I have installed redhat 4.0 on my two UDBs: 166Mhz and 266Mhz. I ftped
redhat/axp 4.0 release from ftp.redhat.com:/pub/redhat/Redhat-4.0/axp
directory.

The 4.0 release seems uses less hard disk space than 3.0.3 - I wonder
what is not included in the release. The axp directory in ftp.redhat.com
has only about 190 mb data. (/pub/redhat/Redhat-4.0/axp directory).

Some other directories are useful also:
/pub/redhat/Redhat-4.0/SRPMS/* --- sounds like all the source code
/pub/redhat/Redhat-4.0/updates/axp --- new updates for axp ?

The installation options I selected does not include samba. (I don't
know why. I selected everything except X development, X games, Tex, X
emacs). I had to use rpm to install the samba from
.../axp/RedHat/RPMS/samba*.rpm. Now I can transfer files between Win95,
WinNT and Linux machines. It is not very clear in RPMS directory how
many packages are installed already. I haven't find a detailed
description about the new redhat (4.0) installation process.

The X interface(like win95) is good but it does not help me on finding a
usable www browser. I wonder if there is any gui browser in
ftp.redhat.com or anywhere else in the world. I used lynx a few years
ago. It is cool but kind of too classic now.

I installed Linux in the internal 340mb and 520mb hard drive for two
machines. I wonder if it possible to install WinNT in an external hard
drive. It seems to me the Digital firmware can support multiboot. But I
don't know if milo will affect the installation and the boot of another
patition (sdb1).

Except to find a good GUI browser for redhat/axp Linux, the other major
task for me is to find a good uses for my two UDBs. I always wanted to
have a dedicated Unix box at home. Now I have two not very fast but good
RISC boxes. Plus a few WinNT, Windows PCs I may well be connect them
into a small network. I think I definitely need some network games at
home. Anyone have any good suggestions?

I used to play a text based Netware game which I don't know the name. I
believe it is from Netware 2.1.5 (?) diskettes and I was in a Netware
training class. we used to play about 15 minutes between the classes.
Most things I learned from the class I have forgot. But I still remember
the feeling of playing that game.

Happy linux.
