On Fri, 18 Oct 1996, Robert Klein wrote:

> I've got a NoName 266 mhz AXP. I currently have craftworks installed and
> I'm tring to install RedHat via a boot floppy & FTP. How do I get the
> floppy to boot? If I boot the computer and hit ESC I get a ">>>" prompt that
> has a boot command but not in the format of 'boot fd:lmlinux... etc..'.
> I have MILO installed on the computer but it boots automatically from the
> HD. I can interupt the boot sequence and get into a set of menus that give
> me a choice of booting NT, OSF, or Linux (Of course Linux is the only OS
> installed) but not floppy boot option.

You're using SRM :-) You need to be using the ARC/MILO combination (or
make your own boot disk). See the MILO HOWTO for details.
