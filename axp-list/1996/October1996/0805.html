On Sun, 20 Oct 1996, Eric Jones wrote:

> I just used the new ramdisk to install redhat onto my system. The install
> looked quite good, but I have developed a really strange network problem.
> When I use ppp to connect over a serial line, I can ping and ftp and do all
> the standard things one would expect to be able to do but ONLY as root.
> If I try to resolve anything as a normal user, I get unknown host errors..
> hosts that I can reach by numeric IP.
> Any ideas on what could cause this?

There's a good chance that /etc/resolv.conf is mode 600, which would cause
this (see the Errata).
