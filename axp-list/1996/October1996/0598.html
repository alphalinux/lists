Note that the inline asm example for "put_user_64bit()" that I sent out was
not complete: looking at it it was missing a input operand of "0" (0) which
just sets the error to zero before the asm (it's then set to -EFAULT if an
exception occurs), and also makes sure that the error register is not
allocated for anything else on the input side.

(It was also _completely_ untested, but with the above fix it should at
least be close to working if not perfect).
