Is the CD you created in ISO9660 with RockRidge extensions? ( I suspect
not. ) Find the utility called "mkisofs". It is a Unix/Linux utility that
has been ported to the Win32 environment. With it, you can build a disc
image that is ISO9660 w/RockRidge extensions and also Joliet format ( so you
can read it on a M$ machine ).

Good luck.
