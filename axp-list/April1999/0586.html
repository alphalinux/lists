On Wed, 21 Apr 1999, Gyenes Istvan wrote:

> > Why wouldn't X run on an ATI mach64? I am using an ISA ATI mach64-GX on
> > AS300 and it works very well under X.
>
> Because my machine is a Jensen (DEC 2000 model 300,EISA based). And it
> only supports S3 based card under X.

 What does it mean: "Jensen only supports S3 based card under X?" AFAIK,
X servers are model independent, e.g. the XF86_Mach64 supports all Alpha
types and maps card's I/O and memory spaces appropriately for each Alpha
model, depending on current '/proc/cpuinfo' contents and falling back to
'/etc/alpha_systype' if the former method fails. Is it written anywhere
within X docs/sources that Jensen is not supported or you just were unable
to get it to run (just as I was unable to use mach64 in AS300 under X
until I found out that an aperture is needed to be set in XF86Config).
