I'm not using a Cybex Commander, but a Rose ServeView 4 port box.

In order to run milo, I have to power cycle the Rose box while holding
down the 'Next' button. It is quite inconvenient, but luckily the
Alpha box doesn't need to be power cycled very often. You might try
something similar.... I ended up having to talk to Rose tech support,
as the Alpha box worked just fine with a real keyboard attached.
