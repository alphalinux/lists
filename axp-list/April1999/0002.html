AS4100, aka RAWHIDE, is well supported, and, with a few tips, easy to install
to and run.

1. use the rawhide-s kernel from RH 5.1 (5.2 is broken in some way), or the
   uniprocessor generic kernel available on gatekeeper (see below).
2. use the 5.1 RAMdisk (updated) to install RH 5.2, as the latter doesn't
   install correctly on machines booted from SRM.
3. use BSD-stype disklabel (partition table), via the "b" option to fdisk.
4. leave the first cylinder or 2 empty for bootstrap install, by starting the
   first partition at cylinder 2 or 3.

For SMP, grab the generic SMP 2.2.3 kernel available on gatekeeper.dec.com
as either:

        /pub/DEC/Linux-Alpha/Kernels/generic-smp-223.gz
or
        /pub/DEC/Linux-Alpha/Images/generic-smp-223.img

But it's safer to use a uniprocessor kernel for installation, so:

        /pub/DEC/Linux-Alpha/Kernels/generic-up-223.gz
or
        /pub/DEC/Linux-Alpha/Images/generic-up-223.img
        
are alos available.

Good luck, and let me know of any problems.
