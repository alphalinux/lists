I added the following file to /etc/sysconfig/network-scripts

eth1

I simply copied eth0 to eth1 and then edited eth1 with the correct
information. Just make sure that there are no IRQ/Memory conflicts.
once I got the bloody card on a non used IRQ and memory space, everything
worked great.
