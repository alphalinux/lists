On Wed, 21 Apr 1999, Brian Rerecich wrote:

> Here what I did to install Redhat on my lx164. I made 3 floppies, one with
> milo, one with the boot kernel, and one with ramdisk. You'll need to make
> these again. Now, use the AlphaBIOS boot from the first floppy (milo +
> linload). Then once you get the milo prompt, switch to the second disk
> (boot kernel) and type 'boot fd0:vmlinux.gz root=/dev/fda0 load_ramdisk=1"
> and then switch disks to the ramdisk one when it asks for it.....
>
> Thats all....enjoy.
>
> -Brian
