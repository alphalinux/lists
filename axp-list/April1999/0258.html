Hi!

I had some problems with xntp myself. I wasn't trying to run server
on the machine, but ntpdate didn't run and stuff like this. Since
I'm lazy, I solved it by installing

xntp3-5.91_export-3.alpha.rpm

which is available at

ftp://ftp.sunet.se/pub/Linux/distributions/redhat/contrib/libc6/alpha/

and, probably, some other places too. xntpd from this package
works fine on UX (just tested it myself). As of a unreasonably big
tick, I would recommend to look at tickadj program. Never had to
use it myself, but seems like it's purpose - to adjust "tick value"
(whatever it is).
