On Fri, Apr 02, 1999 at 08:29:12PM -0500, Uncle George wrote:
> does the alpha version of gdb recognize pthread debugging. from the
> website http://pauillac.inria.fr/~xleroy/linuxthreads/
>
> he says that gdb will now debug linuxThreads with redhat 5.2. Is this
> accurate for alpha?

The best way to debug pthread problems on Linux today is using glibc 2.1.1
which will be out soon.

You may want to install glibc 2.1.1pre1 from ftp://glibc.cygnus.com/pub/glibc
and the Linuxthreads snapshot from

ftp://ftp.lucon.org/pub/linux/glibc/linuxthreads-1990331.tar.gz

Then recompile gdb 4.17 + H.J. Lu's patches from

ftp://ftp.varesearch.com/pub/support/hjl/gdb
