On Tue, 20 Apr 1999, Brian Rerecich wrote:

> Can someone point in the right direction? I am going to try running DEC
> UNIX Netscape 4.51 this afternoon, but want to know if there are any
> other choices. I have been to www.mozilla.org, but I can't find anyone
> who has compiled it for Alpha!!! Please help....

Well as you work at Compaq I will assume you have a Digital UNIX err Tru64
UNIX license. If you do then the Tru64 version is your best bet. If not,
then get the mozilla from ftp.alphalinux.org and give it a try. There is
also KFM, the KDE file manager - but I've never tried it. And last let us
not forget lynx!

I'm afraid that's about it unless:
1. Netscape (AOL?) makes a 64-bit clean browser - Ha! yeah right!
2. Compaq releases the Tru64 C/C++ compilers with that handy 32-bit flat
for Alpha Linux... And Netscape compiles a version of their browser using
those tools.
