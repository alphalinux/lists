I also bought one of these boards and already bought normal PC100 SDRAM DIMMS
for
it. Although the AlphaPC 164SX Motherboard Technical Reference Manual only shows
x72 configurations (ECC mem) it also states:

"Each DIMM can be 72 bits or 64 bits wide, with 100 MHz or faster"

So I am betting my 64 MB PC100 SDRAMs work, if not then my PC will have 256
MB...

About the EPROM on the DIMMs most modern DIMMs will have these especially PC100
DIMMS.

Jurgen
