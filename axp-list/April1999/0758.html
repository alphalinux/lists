No, what I meant is that you use tar at the FTP server. Some servers
provide this, e.g. you can download everything in directory 'xyz' by
getting 'xyz.tar' instead. The server packs the directory while you
during the download, thus on-the-fly. The advantage of this is, that
it will preserve the special device nodes of the /dev directory. To
make you CD image, you have of cause to upack the tar archive.
