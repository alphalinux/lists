On my LX board with 2.0.36 kernel, I put "ether=0,0,eth0 ether=0,0,eth1" in
my boot command and that fixed everything for me. When I upgraded to 2.2.6
kernel, it no longer worked. I think instead of using 0,0,ethX I need to
use <irq>,<i/o addr>,ethX.
Try it out and let me know what happens!!!

-Brian
