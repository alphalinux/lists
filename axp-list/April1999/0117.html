On Sat, 3 Apr 1999, greg meno wrote:

> just a few questions regarding the SuSE 6.1 AXP
>
> 1. anyone have a working installation ?

Did for a while, had to nuke it though. I might try installing it again on
another machine if I get the chance though.

> 2. What kind stuff did you run into ?

Docs are in German. Setup is still somewhat i386 oriented (like it makes
LILO references). The installer wasn't very intuitive, at least I didn't
think so. Unlike the RH installer there didn't seem to be a clear path
through it, though if I had understood the docs that might have helped.

> 3. do you like it ?

Ummm. Yeah, yast looked good, and looked similar to LinuxConf. I was
really impressed with their X-setup tool (SaX I think it's called).
However this was just my impression for using it for a couple of hours.

I was I had more time to play with it, as I've never even used their i386
distribution before.

BTW: This was on an LX, and it was pretty painless.

As usual, YMMV.
