On Sat, 17 Apr 1999, Robert J. Kochen wrote:

> Installed redhat Linux 5.2, being a novice I'm having difficulty getting
> the modem to be recognized. I suspect more questions will be forthcoming.
>
> Modem is on com2.

That will be equivalent to /dev/ttyS1, try starting minicom on that device
and giving the modem the command:

AT

it should return OK or 0 - if it does that then the link to the modem
itself is OK. If not then do:

setserial /dev/ttyS1

 and tell us what happens
