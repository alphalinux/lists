> I'm not sure about the LVD discs; but my underdstanding is that these
> will
> work with normal SCSI adapters safely - but it is definitly worth
> checking first!

It works this way:
If the controller and all attached devices support LVD (and it's not
disabled by a jumper or so) each drive will run in DVD mode.

If any device is not LVD compilant, the whole SCSI-Bus will fall back
to SE mode. (Because there is no LVD CD-Rom drive available one should
use a two channel card or a "bridging" card to avoid the fall back.)

Note that LVD also affects termination. Multi-mode (combined SE/LVD
terminators) should be used.

> I'm not sure if the modern LVD controllers also support the old
> Differential - I'm guessing
> they dont.

Correct.
