> Anyone succesfully using cdrecord on alpha (especially on IDE drives with
> ide-scsi emulation) ??

Like a charm... we have a sony CDU928E IDE writer, which although wasn't
supported by cdrecord until I got in touch with Schilly, now works with no
probs... worked from about 2.0.32.

The system is an SX164 with IDE disk and cdwriter.

The only slow thing is making disk images using dd which is _VERY_
processor intensive... but it still works.

It is a cheap and _reasonably_ easy option.
