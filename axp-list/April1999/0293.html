OK.. to start, since you mention this is your first
Alpha/Linux install, when you say you copied insmod
from another Linux box, would this be a linux box on
intel? If so, the intel binary won't run,.. and I
think won't be recognized as an executable. .. the
typical error is that it will say file does not
exist... at least, that's been my experience.

As for burning your own CD, what CD format did you
use? It probably at least needs to be a format that
the install kernel can recognize... the default I
think is ios9660 w/ rockridge ext.

One problem that I've encountered every now and then,
is that the ramdisk.img wasn't written correctly to
floppy... or the floppy had some bad sectors. Try
using the latest updated ramdisk and a new floppy
disk.

However, I've never encountered your specific problem,
so I don't know how much that applies.

D.
