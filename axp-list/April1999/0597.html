On Thu, 22 Apr 1999, William wrote:

> OK, I am going to install RH5.2 on an Alpha 200 4/233. I have installed it
> a couple times to get the "feel" of it, no this is the real thing. My
> question is this, are there any issure I need to be aware of useing an
> external Hayes-Acura 56k modem on com1? Are the serial ports limited in
> speed (I have herd they are)....

I would put it on com2 so milo/kernel or something else wont get confused
having something attached to com1 (where milo/kernel messages get echoed).

I don't remember what the max speed is on a AS200, but I would assume it's
the same or similiar to the rest of the systems from that time period; in
that case it should be able to do 115K, if you have problems using 115K
like I and some other have, you have to use 'setserial /dev/xxxx spd_vhi'
(where xxxx is, is your serial port) and set your software to use 38.4.
Read the man page for setserial for more information.

FYI, I use an external USR Sporster x2 (upgraded to .v90).
