  In what sense ? All software that is available in source form for
the ia32 should be usable also at Alpha -- modulo some stupidities in
the code where 64-bit wide values (like pointers) are stored into

but for these mentioned MTAs that should not be an issue.

what Cyrus is.

in an array of other platforms.
