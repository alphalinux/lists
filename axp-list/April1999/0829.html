The answer is VGA Text Support......

I noticed when using make xconfig instead of make config, the option for
VGA Text support was not settable and therefore since I don't have TGA, no
screen! You must use make config!!! I would do make xconfig, then only use
make config to change the vga text setting to make it easiest. Anyway, do
what you wish, but remember to set VGA Text mode with make config!!! Enjoy.
Let me know if this isn't the problem....
