Jack Wathey wrote:
>
> First a specific question. I just downloaded egcs 1.1.2 from:
>
> ftp://rpmfind.net/linux/rawhide/1.0/alpha/RedHat/RPMS/

'rawhide' is using glibc2.1 - so if you did not convert your system
already to that you are bound to get into troubles.

> When I run rpm --checksig egcs-c++-1.1.2-12.alpha.rpm I get:
>
> egcs-c++-1.1.2-12.alpha.rpm: size !!! NOT OK
>
> Is this something serious?

Quite possible

> Is there a better place to get rpms of egcs?

egcs-1.1.2 compiles without any troubles on alpha. If you insist
on rpm you are likely be better of by grabing 'src.rpm' and doing
'rpm --rebuild'. Or just './configure --prefix=..... ; make ; make
install'. Set "prefix" to keep it out-of-way for the future.
