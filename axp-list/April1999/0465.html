William wrote:
>
> Is it possible to use an internal pci modem on an alpha 200 4/233 ??

There are some PCI modems which work with Linux but they seem to
be rather rare birds at the moment.

> I have access to a USR currior v everything that can be set useing
> jumpers....how would I do this....or is it even possible?

PCI modems do not have jumpers, of course. If doing 'cat /proc/pci'
you will see 'i/o <some address here>' then you can use this address,
and an interrupt which will show up, as parameters to 'setserial'
to initialize properly the whole port. If you see only 'memory regions'
then you are in trouble.

This is not Alpha specific by any stretch of imagination.
