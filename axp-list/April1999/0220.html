That should be the right disk, (ramdisk.img), you may want to download the
updated ramdisk.img file from RedHat as there was a slight problem with
the first one. (http://www.redhat.com/errata)

Also try this as the command line in MILO:

boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1 prompt_ramdisk=1

that should work.
