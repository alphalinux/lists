I should add that my initial goal would be to get a turbochannel alpha
to boot (headless) from SRM and be able to boot over the network with
BOOTP. Video, ISDN, audio, etc. I would like to ignore until I at
least got that far. I guess this means getting the following hardware
to work:

1) AMD 7990 LANCE ethernet
2) Zilog 8530 serial chips
3) NCR 53C94 SCSI adapters

It seems there is already lance ethernet driver code in the linux
kernel, something is there redarding the zilog, and the NCR adapter
seems to be there, as well (at least for MACs). Any comments
regarding this hardware and the state of linux support for it would
also be most welcome.
