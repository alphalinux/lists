I would think u would also need to be more accurate. In order to correct 1 bit out of 64 or detect 2 errors out of the 64 bits you would need ~8 extra bits per 64 bits.

If u want to correct 2 or more bits out of 64 data bits then u would need more 'ecc' bits for each memory fetch/store to carry out the detection & correction.
