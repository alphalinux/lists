jmd (jmd@infinet.com) wrote:
>Has anyone ever tried the browser MMM with any success? I have tried to
>compile the required components without much success. Has someone else been
>successful? [...]

Urm, sure. As long as you have TCL 8 and Tk 8, as on RedHat 5, it
just requires a few 'configure's and 'make's. I run it on ARM Linux
and Alpha Linux (RH4 with a few RH5 bits copied over).

Perhaps the subtlety you missed is that you need to build it with
ocaml 1.07 and not 2.x since there were some changes in the language.

Build and install these in order:

  ocaml-1.07.tar.gz
  ocamltk41-R200.tar.gz
  mmm41-src.tar.gz

The first two are from:
  ftp://ftp.inria.fr/lang/caml-light/

and the last from:
  ftp://ftp.inria.fr/INRIA/Projects/cristal/Francois.Rouaix/mmm/

Then set your preferences (proxy, image loading...) and go!

Some pages display badly or are unreadable but most are OK.
