Stefan Schroepfer wrote:
>
> Paul Jakma wrote:
> >
> > how advisable is it to overclock a 533MHz Samsung 21164? anybody tried
> > it, and with what results?
>
> My 533 is running at 600 MHz for 17 months now.
> No problems so far.
>

Interesting. I switch my LX board to 600 MHz some time ago and yes it
seemed to work nice for some hours using "regular" programs. However,
when tortured with the usual repeated "kernel compile test", I got
several "Sig 11" errors that went away when going back to 533.

So I wouldn't trust overclocking a 533 to 600 on an LX board (even if it
HAS a very impressive CPU fan as in my case) , tooo bad
