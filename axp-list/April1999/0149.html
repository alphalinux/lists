On Wed, 31 Mar 1999 11:54:02 +0200 (MEST), Stefan Reinauer <stepan@suse.de> said:
> Can anyone give me a hint whether em86 does not work with newer kernels
> an/or glibc versions? Is there someone working on this?

Have you enabled kernel support for em86? If not, try to do so, even
though the docs may state that it isn't neccesarry with the foreign
executable option. But I believe you still need the em86 support to
enable limitation of addresses to a 32bit space.

Recently there were rumours on comp.os.linux.alpha that there will be
a updated version of em86 available soon. The current version isn't
really capable of running much useful stuff. Netscape works, but
neither StarOffice nor WordPerfect do. Also, v0.4 doesn't support
current x86 glib versions.
