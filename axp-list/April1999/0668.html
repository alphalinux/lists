On Sun, 25 Apr 1999, James McCollough wrote:

> I have an alpha running redhat 5.2 on a UX2 board with an ensoniq
> audioPCI sound card. It plays cds fine, but when I try to run _any_
> mp3 app, it hard locks the system. has anyone else experienced this
> and/or has any solutions?

The AudioPCI drivers (es1370) are broken on alpha. I have put some time
into trying to fix it, and it comes down to a hardware problem. There are
some bus timing issues, and possibly issues in initializing the alpha's PCI
controller. Both the kernel drivers (OSS/Free), and ALSA drivers have the
same problem (i.e. it's a hardware problem, both the drivers work on Intel).

If people want to work on fixing this, mail me privately, and I'll give you
all the specific info I have.
