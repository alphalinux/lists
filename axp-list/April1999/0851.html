I am running a MGE Pulsar EX7 on an PC164UX2. It came with serial cable
and various software on CD ('Solution Pac') but none for Linux. So I
looked on their webserver (www.mgeups.com) and found binaries for i386.
This won't run with em86 since they are using some inter process
communication which isn't supported by em86.
        Now I run mgeupsd (0.4) from "Stanislav V. Voronyi"
<stas@cnti.uanet.kharkov.ua>. MGE are using their own protocol to talk
with the ups (U-Talk). I phoned them about releasing the source of their
software and/or documenting U-Talk. The response was positive--but I have
no definitive answer until now (some weeks). I will phone them again next
week.
        mgeupsd works fine, but it doesn't use all the possibilities of
U-Talk and EX7.
