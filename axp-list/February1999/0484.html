On Mon, 15 Feb 1999, Uncle George wrote:

> So what is the app that IDLE's at max rpm ?

It's Apache JServ (Servlet Engine) v1.0b2 running on kernel 2.2.1.
Since Servlets are basically Java CGI handlers, the Servlet engine spends a lot
of time idling when the web server isn't being hit. (Which is most of the time,
since I'm the only one that ever uses it. :)

With your JDK 1.2, the java process owned by nobody associated with the apache
process is at around 96% of the CPU time; rc5des drops to 1.5% (though of
course it runs at lowest priority.) With your JDK 1.1.7v2, the java process
drops to 1.5% and rc5des goes up to 96%.
