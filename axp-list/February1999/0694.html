I run an ATI Rage II card in my 164LX. Only issue is it's
sluggish on the initial boot displaying the spinning ATI logo.
Once it get's thru that, it's fine. It takes maybe 45 secs
to get thru the logo.

Also run a Matrox Mystique in a PC164 without trouble. I did
have problems with the original S3 card I used.
