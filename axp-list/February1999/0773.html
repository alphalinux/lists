I've also some PC164 for 2 years now and they are stable with 2940UW.
I'm using axplinux-2.0.35-0.2 from gatekeeper with two modifications:
1) I'm using the old aic7xxx-driver (4.1.1) of the 2.0.33-kernel
2) I removed -Wa,-m21164a -DBWX_USABLE from arch/alpha/Makefile

I'm using several SCSI-disks at once or I'm using a large Raid-system for
realtime-vision-computing.
So the machines are stable unter "heavy IO-traffic".
