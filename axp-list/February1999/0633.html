I just found a feature of RPM that I thought was worth sharing.

Goes like this...

[[root@ns updates]# rpm -K --nopgp *rpm
XFree86-100dpi-fonts-3.3.3-1.i386.rpm: size md5 OK
XFree86-100dpi-fonts-3.3.3.1-1.i386.rpm: size md5 OK
XFree86-3.3.3-1.i386.rpm: size md5 OK

It checks the integrity of the file before you install it.

Found it here...

http://www.redhat.com/support/docs/rhl/kernel-2.2/kernel2.2-upgrade.html

Pretty handy, especially if you're downloading RPM's via a modem.
