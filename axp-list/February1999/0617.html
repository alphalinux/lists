This is an automated message.

You attempted to gain access to software distributed by MIT which is on
our export controlled server (PGP, PGPfone, ISAKMP and PFL [Jonah]).

You filled in a form with your e-mail address. This message includes
the password you now need to access the site. From the time of this
message (see the header) you have two days to use the enclosed password
after which it will expire and you will have to request another
one. Note: Once you use the password it is automatically invalidated, so
either download all you need when you use this password or request
another password at a later date (note: the process of mailing you the
password happens as soon as you submit the request form, the only delay
should be the time required for e-mail to go from MIT to you.

If you wish to retrieve the MIT PGP distribution go to:

        http://web.mit.edu/network/pgp-form.html

If you wish to retrieve the PGPfone distribution go to:

        http://web.mit.edu/network/pgpfone

If you wish to retrieve one of the ISAKMP distributions go to:

        http://web.mit.edu/network/isakmp

If you wish to obtain the PKIX Freeware Library (Jonah) go to:

        http://web.mit.edu/pfl/pfl-form.html

After the four questions (two questions for the PFL Distribution) there
is a space for you to "enter an e-mail password." In this area type:

7649

This should direct you to the distribution you requested. You will then
have 30 minutes to receive the distribution.
