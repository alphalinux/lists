With the help of Stefan Schroepfer and Peter Petrakis, I managed to compile
Chaos Analyser on my Alpha. It compiled when Stefan Schroepfer suggested
that I would add the flag -mieee. The compiling com_shell was thus altered
to:

gcc -I/usr/openwin/include -L/usr/openwin/lib -L/usr/X11R6/lib -mieee -O2
"$1".c -o "$1" -lxview -lolgx -lX11 -lm

Peter cleaned up the rotated.c file.

I am now about to test the program with my own data files.

I would like to thank the help I received with this problem!
