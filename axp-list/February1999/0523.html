set_program_attributes is a function in the linux kernel (and dec/osf) that
does memory setup before running a program. It's in
arch/alpha/kernel/osf_sys.c around line 70. Note that most kernels dump a
message to syslog when set_program_attributes is called. This is not an
error message. Anytime you run a Digital UNIX binary under alphalinux you
get this message...

If it *is* a dec unix binary running under alphalinux, don't forget you need
the latest kernels to run dec binaries (2.2...)
