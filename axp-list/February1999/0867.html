Set the os time with the date command, then run /sbin/clock -u -w. If
you're booting w/ arc, you may need a -A switch as well. I'd heard
that the latest versions of clock don't need the -A, but don't know
what ships w/ 5.2. If you need it but don't use it, things will get
REAL bizzare.
