Hello all,

AFAIK, the difference between an 'a' and an 'au' is the Tru64
UNIX license
and the options. The Firmware is the same, so all models have SRM and
ARC/AlphaBIOS that you can switch to. SRM sometimes does not accept
some PCI options but you can override that (set pci_options -1 or some
such).

You have to watch out for the options though. Tru64 UNIX does not
support
the same options as WinNT does. If you want Tru64 UNIX Support, you need

to get a T64U supported SCSI-Controller (Qlogic) and Graphics board,
but that's all. It's all described in
ftp://ftp.compaq.com/pub/products/workstations/dual-bt.pdf

I'm in the process of setting up a dual boot Linux / T64U Box. This cuts
down
your possible options list considerably ;-)
