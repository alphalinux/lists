On Tue, 2 Feb 1999, J.W. Pennington wrote:
> 1) Does anybody know any tricks for increasing speed on a UDB. (no
> comments from the peanut gallery :-)

a few ideas ...
- the VX42 (233MHz) can be overclocked to 300MHz. however, this is
probably going to produce an improvement far below linear, primarily
because of the slow memory subsystem. (such was my experience)

- i've heard that the pitifully small cache can be upgraded, but i can't
provide any specifics on that.

- if you've got the original 24 megs or less of memory, you might want to
get some more so that you avoid swapping.

- if you're using X, you might want to invest in a fast (but physically
small) vga card for it, since the TGA X server is minimally accellerated.

- supposedly 2.2 kernels run visibly faster on the UDB.

> 2) Does anybody know relatively inexpensive sources for Alpha hardware.
> Specifically, what's a good deal on a CPU/MB combo (nothing too fancy),
> and what kind of case do I need (AT, ATX, something strange)?

i've heard of good prices on SX motherboards and 164PCs. supposedly
DGCinc. is a good (=reputable & affordable) place, but there are many
others as well.
