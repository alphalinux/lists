>>> "greg meno" said:
> upon booting my newly built 2.2.1 kernel I get
>
> ......snip............
> VFS: Cannot open root device 08:02
> Kernel panic:VFS: unable to mount root fs on 08:02
>
> my root file sys is at /dev/sda2
> and i've upgraded all the particulars outlined in the 2.2 changes doc
>
> furthermore what is the kernel trying to tell me when it mentions
> "08:02"(above)

Any chance you might have multiple SCSI controllers of the same type?

2.2.x scan/process the PCI devices in the reverse order from 2.0.x.

I think there's a command line option, "pci=reverse", that will give you
back the old order of devices.

Good luck.
