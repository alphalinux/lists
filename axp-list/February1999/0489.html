Get the srpm and rebuild. never trust the binaries from rawhide.
ftp://rufus.w3.org/linux/rawhide/1.0/SRPMS/SRPMS/egcs-1.1.1-7.src.rpm

If you're not running RH-5.2 I suggest you get the binutils package from
there.

ftp://rufus.w3.org/linux/redhat/redhat-5.2/alpha/RedHat/RPMS/binutils-2.9.1.
0.15-1.alpha.rpm

Install procps cuz it depends on the version of binutils you have.
ftp://rufus.w3.org/linux/redhat/redhat-5.2/alpha/RedHat/RPMS/procps-1.2.9-2.
alpha.rpm

So install in this order,

rpm -Uvh binutils procps

then rpm --rebuild egcs-1.1.1-7.src.rpm

Walk away and get some coffee, it's going to take a long time on a multia.
If you run into anything let me know.

Peter
