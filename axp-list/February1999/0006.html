Just thought this might be a good idea... so I created a Yahoo Club for
Alpha Linux...

I love this list, but sometimes I wished some of these discussions were
more realtime... so, mainly, I hope that this Alpha/Linux Yahoo Club
will provide a forum for realtime discussions.... but I hope it serves
as just another source for information exchange about Alpha/Linux.

If you're interested, please join! Check out www.yahoo.com and goto
Yahoo Clubs --> Computer --> Operating Systems --> Linux --> Linux for
64bit Alpha. If you don't see it yet, please come back later... it might
take a few hours to get listed.
