Try "indent" - there used to be a program called something like "prettyprint"
for formatted hardcopy, but I have no idea how to get it now. "indent" can
re-format code to your tastes, you just have to describe what you like in
an ".indent.pro" file, which can be global (~/.indent.pro), or different for
each directory. It's "pretty" nice!
