This may be of help.

I have a Carrera LX164 with Samsung motherboard. It came installed with a 2940uw. I was randomly getting errors and hangs like this:

SCSI host 0 abort (pid 397694) timed out - resetting
SCSI bus is being reset for host 0 channel 0.
SCSI host 0 channel 0 reset (pid 397694) timed out - trying harder
SCSI bus is being reset for host 0 channel 0.
SCSI host 0 reset (pid 397694) timed out again -
probably an unrecoverable SCSI bus or device hang.

I first replaced the scsi cable and after sill getting errors I then replaced the 2940 with a card revision of v1.25 with a new one with a card revision of 1.32. Everything has been working fine for a week. After another week I'll send the old Adaptec card back to Carrera.
