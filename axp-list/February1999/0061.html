> Anyone have matlab running under NT-alpha or linux-Alpha? I have
> a faculty who wants a 21264 if he can run matlab.

We are running Matlab 5.1 on our axp-Linux server along with the license
manager for multiple users. Matlab is a core application for our
business, and octave does not quite cut it for our needs. However, you
need to run the Digital UNIX version of Matlab, and therefore you NEED to
have the Digital UNIX libraries. If you have access to these libraries,
then you can easily setup and install Matlab following some
well-put-together http instructions.
