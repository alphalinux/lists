I wish I knew. It seems that the premium for the 'au' also goes for a DU
licence, but then Maurice Hillarius claims it is just for the licence to
use the 'SRM'. I don't understand why anyone would pay 2,500$ extra for
the legal ability to use a boot loader, but hey, suckers are born every
minute.

The SRM on the 'a' units seem to be fully functioning, but I have never
seriously used mine. Linux and BSD can boot from SRM, BSD last I heard
doesn't boot w/o the SRM loader.

As for the End-of-life on the PWS series, doesn't the Compaq XP1000 use
the same case? For a price, there used to be an upgrade you can pre-order
to get the '264 motherboard for the PWS case. The '164 CPU is upgradabile
according to the maintainance docs, but you don't get the frequency switch
info unless you buy the new chip, so I think that might rule out a cheaper
Samsung alternative.

JRDM
