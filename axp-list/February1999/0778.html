On Wed, 24 Feb 1999, Geoff Shukin wrote:

> My system correctly resolves names from the shell but no name
> resolution while using netscape!

I'm not really sure what it can be, but perhaps your /etc/svc.conf doesn't
look like this:

aliases=local,yp,bind
group=local,yp,bind
hosts=local,bind,yp,bind
netgroup=local,yp,bind
networks=local,yp,bind
passwd=local,yp,bind
protocols=local,yp,bind
rpc=local,yp,bind
services=local,yp,bind

SECLEVEL=BSD # for backwards compatibility ONLY

(If not, try the above, which works great for me)
