Ron Farrer wrote:
>
> Is it possible to upgrade from RH5.1 to RH5.2 without rebooting?

Yes. It is. But you will have to do that in the right order
as there some interdepencies and some actions may be a bit hairy.
When you are running the "standard" procedure you do not
have to worry about that as you run a minimal system from a ramdisk
and doing updates on something else.

> I'm
> thinking no, but perhaps some else knows for sure?

"Yes" answer will be not correct between any two distributions
but between 5.1 and 5.2 you can pull that trick. Is this an
exercise in a difficult system administration? If you will
not do some pretty non-standard things while installing shared
libraries you better reboot anyway.
