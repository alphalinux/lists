in such a way that it can be loaded into different kernel versions
without generating "kernel-module version mismatch" messages on insmod.

"CONFIG_MODVERSIONS" would be required for this but is rather light on
details. There is more to be found in "Documentation/modules.txt", but
not much more!

as to the procedure?
