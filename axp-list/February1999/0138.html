José Pedro Sousa do Amaral wrote:

> What are the main differences between kernel 2.2.1 and kernel 2.2.0? Any
> relevant advantages of kernel 2.2.1 over kernel 2.2.0 for Alpha users with
> AS 200 machines? ;^)

Never tested it, but I've heard the big difference (which made 2.2.1
something like the "brown bag on Linus' head" release) is that "ldd core" for
any core file will bring down a 2.2.0 machine. If you're certain you have no
mischievous users, maybe you don't need to upgrade.

There are diffs also in the soundblaster driver code, but this didn't solve
the missing symbols problem in the sb.o module. :-( (No problem in
2.2.0-pre9, problem in 2.2.0/2.2.1.)
