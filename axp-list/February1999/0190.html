Well, after looking at the source code, it appears this
package has it's OWN version of <netinet/ip.h> that has
the wrong #ifdef's in it for Alphas. Fix the Makefile
as follows:

linux:
        @$(MAKE) all LIBS= XFLAGS=" -O2 -D__USE_BSD -D_BSD_SOURCE -DSYS_ERRLIST_DECLARED
-DAUTH_GID_T=int -g"

I added the -O2 just in case this code needs some optimization.

Didn't bother to run it since I don't plan on using it.

It built with this on my system but did bitch about things
a bit.

Good luck.
