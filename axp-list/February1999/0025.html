One easy way to use fdisk is to make a file with the values and have fdisk take
the values from the file.

For instance, lets say you have 2 partitions on your disk.. you can do
something like this in a text file
n
p
1
1
+1000m
n
p
2
256
+1000m
w
Of course the cylinder values are just out of my head and not real, just to
prove the point.
Name the file something like values
and do fdisk < values.

quick and dirty :)
