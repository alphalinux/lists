>Hello,
>Is it possible to compile sources on the Alpha and get Intel-Binarys as
result ?
>
>I have an old Intel 486 System which is verry slow when compiling the Kernel.
>So it would be nice to compile it on my fast Alpha.
>
>I know, I need the Intel Librarys, but is it possible to get Intel Code as
>result, or to build a gcc on the Alpha which produces Intel Code ??

Yes this should work. I've cross built from Alpha->ARM.
Build yourself binutils cross (just configure --target=?????)
then with that directory in the path build a gcc with the same options;
you'll probably want a set of Intel headers/libraries before you start.
