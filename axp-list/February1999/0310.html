> Dosemu emulated DOS. (Like wine provides Win32 API for windows apps). It

Not exactly. Dosemu emulates the BIOS of a pc thus providing DOS (or other
operating systems running in real mode) the possibility to boot. Wine on
the contrary translates all Win-API calls into native calls. Basically the
difference is, that Dosemu requires a licencse of DOS (or FreeDOS) and
every time you need it you boot a full OS, while wine just starts the
application and requires neither a licence nor a full OS.
