Phil,

Yep, go to the following URL and download modules-2.0.0.tar.gz and
pcmcia-3.0.6.tar.gz:

   http://www.sunsite.unc.edu/pub/Linux/kernel

Unpack both in /usr/src (same directory that contains your linux kernel
sources).

   tar xvzf modules-2.0.0.tar.gz
   tar xvzf pcmcia-3.0.6.tar.gz

Read the README in modules, compile, install and configure. Then Read the
README and HOW-TO in pcmcia, compile, install, configure, and pray! ;->
