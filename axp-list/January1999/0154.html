Hi.. I just thought I'd share some successes with getting a browser for
my Alpha/Linux machine...

With JDK 1.1.7 for Alpha/Linux (found at ftp.linux.org.uk) and HotJava
1.1.5 from www.sun.com... I'm actually able to surf the web and write
this email from hotmail right now. HotJava isn't the best browser, but
it beats using lynx... and with JDK117, it manages to run most Java
applets on the web.. although, I've also noticed a lot of java that
won't
run... go figure.. a java browser that doesn't run java.

If anyone wants to try it, and runs into problems, just email me.

Also, a question, I'm trying to run Digital Unix Netscape but I keep
getting core dumps with Floating Point Exceptions. (SIGFPE) I've got all
the DU 4.0 shared libraries and the loader... it starts up, then core
dumps. This is Netscape 4.5, I've also tried the other 4.x Netscapes,
and
I still get the same results. The FAQ on the net that describes how to
do this seems to refer to Netscape 3.x ... but I can't seem to find that
anymore on ftp.netscape.com... does anyone know where I can find those
older
versions of Netscape for DU?
