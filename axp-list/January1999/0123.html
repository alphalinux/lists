I recall reading in the past about problems with syslogd/klogd causing system
load to increase to the point of making the system unusable. I didn't pay too
much attention since I was not experiencing the problem.

Recent versions of GTK+ (starting about 1.1.6) have caused this problem to
manifest itself on my system, but only when running a program compiled against
GTK+. Earlier versions of GTK+ (1.1.4 for example) did not cause this trouble.

Is there some problem with GTK+ on the alpha or is it just triggering some other
problem that I should have fixed?

I have also noticed that control-panel no longer works since I went to the
1.1.x series of GTK+ but I never use CP so that's not really a problem.

System is an SX164, RH 5.0.
