AlphaPC164LX egcs-1.1.1 (PC164 config)

CONFIG_RTC now works (thanks Richard!) and so does hwclock.
(I did upgrade to XFree86 3.3.3.1, too, which might have affected
the mouse problems, but I don't think that was it 'cause the interrupts
reported by vmstat are sane again.)

One thing though: with RTC the BogoMips is always slightly low, and if I
ctrl-alt-del instead of reset then it hangs (sometimes) while calibrating
the delay loop.

SMBFS and MSDOS both randomly hang the system under high load when
configured as modules, but they both work OK when compiled in. (They get
automatically insmoded OK, but maybe the pages they live in get trashed at
some point.) Anyhow just don't use those guys as modules.

Interactiveness is good.
