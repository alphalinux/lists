Shing-Gene Yung <shing@cz3.nus.edu.sg> asks:
>
> Hi,
> What's the best way to convert 32-bit pointers to 64-bit? (to fix common
> warnings abt pointer-int casts....)

        Use complete ANSI prototypes, then the *default* return type of
        the undeclared function (int) won't hurt you.

        To help hunting them, use these GCC options:
                -W -Wall -Wstrict-prototypes -Wmissing-prototypes
