The one I run is used for single threaded number crunching. Hard to beat a
633MHz 21164 (a UX4 Ruffian based box, not a badged DEC machine) in terms of
single-cpu bang for buck!

Having said that, as a general workstation the only problems have been finding
a web browser (which will solve itself given time and has been worked around
for now) and some messing about with which compiler to use (native gcc/egcs or
compile static binaries on a DU box - usually a bit faster but less convenient)
