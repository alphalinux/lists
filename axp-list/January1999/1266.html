I thought what the heck I'll bug Netscape and see if they even know what
Alpha is, I mean they have a Linux SPARC port so why not Alpha? (I already
new they didn't want to do a 64-bit port but thought it would be
interesting to see what their response would be.)

-- Ron Farrer

Here is their reply:

---------- Forwarded message ----------

Greetings --
The reason that there is no port of Netscape for Alpha Linux is
threefold. First, you can actually run the Digital Unix version from
Alpha Linux. At this time this is the fastest method of running
Netscape on Alpha Linux. The URL for how to do this is below:
http://www.alphalinux.org/faq/FAQ-18.html
Second, if you don't have a license to run the original libraries, the
linux version of Netscape will run fine (albeit a little slower) through
em86. If you need em86 or info on it, please see:
ftp://ftp.digital.com/pub/DEC/Linux-Alpha/em86/
Finally, now that the source code to Netscape has been released a
port is being done. Work is going slower than some of the other
previously unsupported platforms because of some serious 64 bit
issues that cropped up early in the porting phase. For more info on
the native port, please see:
http://www.unix-ag.org/Linux-Alpha/Mozilla/
___________________________________________________
Thank you for using Netscape E-mail Technical Support. Please
let us know if this answers your question. If not, please respond
quoting this ENTIRE message to quoting this ENTIRE message to nav-support2@netscape.com. Or,
see the following page for your support options:
http://help.netscape.com/support/client.html
-- Travis
