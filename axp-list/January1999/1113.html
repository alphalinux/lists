We got into Alphas because we are developing software for 64 bit
processors and were using MIPS R10000s on an SGI Challenge box. We had
some money around and the owners of the SGI were complaining that we
were hogging a whole CPU with our experiments so we decided to get
ourselves a 64 bit machine. The Alpha took a long time to arrive and to
say sorry the company gave us a 166 UDB which I took home to replace my
tired old Sparc 1 and 486/33. Compared to those two it flies!

Contrary to other comments I don't have anything other than alpha and I
do everything on it. Code development, web browsing (Mozilla, kfm,
lynx), document preparation (LateX, LyX), firewalling, SAMBA (for a
demonstration Win95 laptop install which I never ever use for real work,
an OS which crashes when I try to print something while web browsing?
Please!), molecular modelling (Roger Sayle of Rasmol fame used to work
in my lab), apache and so on.

There are some programs which only run on Intel but that will change, I
just bought Applix for the UDB because my wife uses it for club
secretary work and I am very pleased with it. If other companies see
that Applix can sell commercially onto the Alpha then we should see
other stuff arriving.

The alpha is a platform that deserves Linux, Intel gets it by shear
weight of numbers.
