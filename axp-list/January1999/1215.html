Quoting Rich Payne (rdp@cheshire.net):
> > [> > [root@alphagen bjava]# java Welcome.java
> > Unable to initialize threads: cannot find class java/lang/Thread
>
> What if you try:
>
> java -classpath /usr/local/jdk-1.1.6/lib/classes.zip:. Welcome.java
>

You don't run the .java file, you run the class file. And you do that by
specifying the class name, not the file name. So to start a java program
called Welcome, which was compiled from Welcome.java into Welcome.class, you
say "java Welcome".
