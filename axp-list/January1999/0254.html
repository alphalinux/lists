Hi Peter.
setld is a tool on DU that unpacks some special archive types for DUNIX and
makes entries in the kernel configuration file. Then you run the linker to
link these and build a new UNIX kernel.
It is a shell script, refers to some other binaries and scripts and so forth.
I sincerely doubt it will be useful to you on a Linux kernel, as this is a
tool for doing kernel configurations and linking DUNIX kernels..
