> Before you rebuild the kernel, AFAIR that's the only thing that
> absolutely WON'T work as a module.

Technically you are wrong. You can boot into a ramdisk (your
installation set is doing that) which may have some other file
system on it and not ext2, load your ext2fs as a module and switch
root (root switching is performed, for example, by a freshly
installed Red Hat on Intel where 'initrd' is used).

I cannot now invent a reason for such games with ext2fs but
maybe that would be convenient in some situations. Anyway,
you **could** do that. Besides ext2fs for a root file system
is NOT a requirement. This file system was introduced when
Linux was in use already for quite a while.
