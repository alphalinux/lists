is to set this system up headless as a firewall/gateway for my home network).

The UDB on power up runs it's self tests and then talks down the serial
console. I can see 32Mb of memory and the disks (SCSI) one internal,
external CD. However, as soon as I boot from either the floppy drive or the
CD I see lots of the following errors:

Processor correctable error through vector 00000063.

ESR: EFF4F8F80000001B EAR: EFF4F8F801F50BE0
STAT0: 0000000200000002 STAT1: 0000007100000071
BC_TAG: 0000000000000000

Can anyone suggest how I might overcome these (I'm hoping it's nothing
terminal). I've reseated memory and checked cables. I have also installed
the latest firmware (from the v5.3 firmware CD).

If no one has any suggestions I'll try to install NT on it tonight to see
