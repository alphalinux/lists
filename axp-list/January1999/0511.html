Why instead of using abominable registry hacks you will not turn on
encrypted passwords support in Samba? It is one line in config.smb
in current versions (yes, you will need smbpasswd file unless
you authenticate from some other domain password server). As weak
as these passwords are they are still better than a plain text.
December issue of Linux Journal has an article on this topic.
Check it out.
