AFAIK an access to ftp.kernel.org is limited now on purpose but
there is already in place a network of "official mirrors". They
have names like ftp.xx.kernel.org where 'xx' is a country code.

ftp.us.kernel.org has address 204.201.36.6
ftp.us.kernel.org has address 207.179.39.12
ftp.us.kernel.org has address 206.100.251.10
ftp.us.kernel.org has address 204.192.112.175
ftp.us.kernel.org has address 209.249.146.48
ftp.us.kernel.org has address 144.92.245.22
$ host ftp.ca.kernel.org
ftp.ca.kernel.org has address 142.92.65.9
$ host ftp.uk.kernel.org
ftp.uk.kernel.org has address 129.12.200.128
ftp.uk.kernel.org has address 194.159.255.135

and so on. Pick one close to you.
