Ack. My fault. Pre6 had a compile problem in arch/alpha/kernel/process.c;
it needed the line:

static struct file * init_fd_array[NR_OPEN] = { NULL, };

Be sure to *remove this line* before applying ac2 (or any other patch).
(NR_OPEN is now defined as 1024*1024, so this resulted in a biggish kernel
image. ;-)
