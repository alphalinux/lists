What do you want to know ? ;-}
I was a whole week there (iX-Red) and I've set up this cluster.

Here in short:
1 Bootserver LX4 667MHz, 1GB RAM, 3x9GB HD, 6xeth (21143), 2xUW/SCSI,
  Millennium 8MB
50 Diskless/Headless clients LX2 667MHz 512MB Ram, 1 eth DE500 (21143)
1 HP ProCurve Switch 8000M 64Port (great to play with it...)

We used Linux 2.1.125 with minor patches, based on Redhat 5.2 distribution
with some modifications (recompiled important things using egcs-1.1.1,
a modified light rsh, rshd/rexecd), current MPI (mpich not LAM)/PVM
releases and btools. We used a LINPACK provided by the Los Almos people.

The same cluster was at WDR-Computernacht (CLOWN) but with half of memory
(server 512MB, clients 256MB).
(Note: CLOWN: 27,9 GFlops; iX-Red: 31,009 GFlops)
We reached the 30GFlops at iX-Red by manually adjusting the LINPACK
matrix (was a 51 17x3 nodes run); the 31GFlops by a (silly) script
that tried all possible Linpack-Runs using 48, 50 and 51 nodes (see the
nice chart in the iX-magazine.
