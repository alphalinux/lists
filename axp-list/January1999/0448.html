OK, some info -

How the power "should be":

============================================================================
======
= Ora Red Yel Blue Blk Blk || Blk Blk Whi Red Red Red
= -------------------------------||-----------------------------
=======
= J31 J29 |
= | CPU
= Blk | | Brn |
= Blk | | Brn |
= Blk | | Brn |
= J28 Brn | | Blk J27 |
= Brn | | Blk |
= Brn | | Blk |
=
=
Key:
Red = +5V
Yel = +12V
Brn = +3.3V
Blk = Grnd
I won't bother with the rest, this is the main part.

>A previous message suggested checking jumpers according
>to the manual. Can you give me an idea where to look for
>J16-2 and J16-3. I found the note about this jumper in
>the PDF manual but it doesn't give any hints were it is on the
>board.

Jumpers:
Between the 1st ISA slot ( near board edge) and the 2nd ISA slot is jumper
"VPP" or "JP16"
It should be jumpered between pins 2 and 3. If in doubt there is a number 1
silkscreened at the side closer to the 2nd ISA slot. So jumper between
centre pin and one closest to first ISA slot

J15: Between the CPU and the cache RAM DIMM sockets:
Four pairs of pins. Jumpers on the 2 pairs closest to the centre of the
board, like so:

===============================================

        =============== Cache DIMM
                          || ==============
                          ||
                CPU ||
                          ||
                          || 0 0
                          || 0 0 Cache DIMM
                          || IIII ==============
                          || IIII

Fianlly, a big row of jumpers and connectors on the corner of the board, by
the part with the name "Dallas" on it:
There is a set of 3 pins by themselves. - No jumper here.
There is a set of 20 pairs of pins. The first set, or pins 1&2 are closest
to the "Dallas" part.
Jumpers like this:

========|| o o o ||
 Dallas || I o o o o o o I I I o o I o o o o o o o ||
           || I o o o o o o I I I o o I o o o o o o o ||
========|| SP SP HD SPKR ||
                                        4 7 LED ||
=========================================================================

This is on my "running" board..
