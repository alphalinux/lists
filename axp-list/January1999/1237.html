Peter Petrakis wrote:
>
> and proceed to execute the install program as
> root and get this.
>
> [> [root@alphagen cdrom]# ./install-applix
> bash: ./install-applix: Permission denied
>
> How can it be permission denied?

If you have your CD mounted with 'noexec', like it normally should
be, then this permission will be denied (and this is a good thing).
Removable media on a general access machines should be also mounted
with 'nodev' and 'nosuid' options. See 'man mount'.

> This program must run from the CDROM.

If you are root then you can remount this CD with exec priviledges.
