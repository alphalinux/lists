On Thu, 07 Jan 1999, Alex de Joode wrote:
>Is it possible to have a dual boot alpha as200 ?
>
>Ie: sda and sdb hold the Linux OS, and sdc should
> be the NT disk. Is it possible to install NT
> on sdc whithout killing Linux ?
>
        There's a FAQ that explains how to install both NT and Linux and make
them dual boot on the AS200. At the moment, I have both NT Server 4.0 and RH
5.2 running on an as200. The technique involves first making a very small
partition on sda where the bootloaders for both NT and Linux can live, and then
letting ARC's boot menu know where the system partitions for NT and Linux can
be located. Works like a charm.
