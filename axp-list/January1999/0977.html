We use alpha linux here for scientific purposes. I generally feel that
we get more flexibility with linux, so for our limited budget, this is
ideal. Speed also seems to be improved, e.g., I ran a matrix
multiplication algorithm with PVM on DU and then on the same machine with
alpha/Linux and it was >2x faster....
