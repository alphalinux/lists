The memory sounds right....are they all the same manufacturer and
nanoseconds etc?

In my manual, there are some jumper setting for the debug monitor on
comm port 1. Refer to Starting and Using the AlphaPC64 section 3.4.2
This cover what Mark was talking about with the debug monitor. It's
possible your video output is going out thru comm port 1. Check these to
see if they are set right. This manual is available at digital's site in
.pdf if you don't have it:

http://ftp.digital.com/pub/Digital/info/semiconductor/literature/archives.html

If this is one of the boards called a PCI64 there may be some
differences. The website that sold these boards had a URL for that
manual.
