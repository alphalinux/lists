On Thu, 7 Jan 1999, Designer 7 wrote:

>However, I'm having trouble mounting SMB shares. I can access the share
>via smbclient... but when I attempt a smbmount, I get the following
>error:
>
>root# smbmount //WINDOWSMACHINE/SHARE /mnt/smbfs -U USERNAME
>Password:
>mount error: Invalid argument

Try a command like:

smbmount //host/share password -c 'mount /mnt/smbfs'
