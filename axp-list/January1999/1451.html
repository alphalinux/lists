To sum it up you can run anything that can be run on DU on AlphaLinux
providing you have the libraries and a few other files from a DU box. If
that library is using DECTHREADS at all it'll fail miserably since that
is an attribute of DU's kernel and there are no provisions for it in the
Linux kernel right now. Could you post the output of the nm -u?
