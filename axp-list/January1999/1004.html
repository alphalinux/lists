On Tue, Jan 19, 1999 at 09:18:26PM -0800, Designer 7 wrote:
> Ok.. so your tip worked great,.. I just had to recompile libffm as a
> shared library... however, I noticed that libffm gets loaded ALL the
> time, even if a program doesn't need libm/libffm.

Indeed. But you requested that it be loaded, so it was.

Note that at some indefinite point in the future, you'll see
a modest to severe performance penalty because of LD_PRELOAD.

We have plans to implement Irix-like shared library quickload
scheme. Preloading defeats that, and so quickloading would
be disabled for that process. If you have LD_PRELOAD set for
everything you run, you would see no benefit from the
quickloading at all.
