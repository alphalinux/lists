Hello all,

I was on the list a few weeks ago with several
problems with a Cabriolet that wouldn't boot,
not even the video. While I got off the list I didn't
give up on the problems.

Those that responded from the list were very responsive
to my problem and made very good suggestions and statements
of support. Thanks

Well I'm happy to say that one or two of the problems
was solved by taking the board over to where I bought
the memory and swapping out memory chips until
I got a good set of true parity memory sticks.

So I can boot now! Yeaa!

But while the skies have cleared a little, the day is not
sunny yet.

I trying to use a Mylex Buslogic Flashpoint SCSI adapter
to boot my system. I'm pretty sure I have the hard
drive set for the proper ID and terminated.

I'm using MILO to boot the "cabrio.img" file from the
RedHat 5.2 Alpha distribution.

The problem is that the boot file says that Buslogic
Flashpoint is not compiled into the kernel.

And I can't find a ReadMe file on the CD under "images"
or "kernels" that tells me if there is an alternative kernel
I can use that does have the Buslogic driver built in.

Does any one know if there is such a beast somewhere
and where I might go look for it?
