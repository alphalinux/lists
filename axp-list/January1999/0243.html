Linux 2.1 has been a bit rocky, and 2.2.0-pre4 doesn't compile without
some patches. But here I am.

I get deadlocks of unknown sort when I ran the 2.2.0-pre2 kernel, and
a previous iteration of this would hang on boot. However, this go-round
I have eliminated all modules and this monolithic kernel seems to be
doing a bit better. I'm not sure yet if the hanging is gone, it usually
takes a day or so.

However, it is true that the 2.1.102 kernel I have been using also does
not use any modules, and it's been working fine. fingers crossed.
