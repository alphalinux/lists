I just got hold of an S3 Virge DX card with 4MB which I managed to
shoehorn into my UDB. Currently running it at 1024x768 32 bit and it is
running very nicely. The card is actually about 1cm too long but I
managed to slide the end of the card between the floppy and HD!

Anyway, a couple of things. I have recompiled my kernel not to have TGA
support but when the machine boots it still comes up on the TGA and the S3
doesn't start until Xdm kicks in. Second, there is the problem of only
being able to have 20 or so windows open with the RedHat 5.1 driver, is
this fixed in 5.2? If not, I remember someone had a site with fixed
drivers for alpha but I didn't keep the URL (Doh!).

Still, nice to have 16 million colours on a UDB! Doesn't half make the
machine feel fast. Adding another 64MB of main memory this well along
with a 15" LCD panel, should make for a very sweet little machine.
