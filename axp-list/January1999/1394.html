On Fri, 22 Jan 1999, Bernard Chew, Lai Wan wrote:

>
> So far, Lynx is the only browser that works for me on Alphalinux. I
> don't really mind the text-only interface. However, there is a
> problem. My ISP requires that all external http access goes through
> their proxy server. Is there a way to get Lynx to point to a proxy
> server for http requests, rather than talking to the http server
> directly?

Besides modifying your lynx.cfg, you could set:
ftp_proxy=http://proxy-hostname:port/
http_proxy=http://proxy-hostname:port/

This could be used by other programs.
