OK, I was being stupid there for a minute. Here's what to do....

1) Don't unzip/untar the classes.zip file. It's supposed to be
compressed.

2) Compile your Welcome.java file into a class file
> javac Welcome.java

(BTW: this failed on my machine)

3) Then run the class file, but just give it the class name (no ext).
BUT you are
compiling an applet NOT an application. So you can't use 'java Welcome'
you're going
to need appletviewer or a browser. In this case you'll need to make an
html file (index.html)
that refernces the class.
> appletviewer index.html

 You may want to consider an application instead of an applet. Then you
don't need
appletviewer etc... Just extend Object instead of applet and then put in
a main() method
etc...

Hope that helps.

regards,
rdp
