Try the following.

Do a find for the file 'shmparam.h'. It should show up under your kernel.
Open that file with vi and change the variable SHMMAX. You will find a hex
number. Change the 3 to an f and recompile the kernel. It works fine after
that. Now if you ask me how much shared memory is allocated now, I couldn't
tell you, but like I say, the program works after that. I believe that you
need a minimum of 24 or 32 megs of memory shared (It should say in the
readme), but this should get you started.
