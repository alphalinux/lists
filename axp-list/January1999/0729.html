>Ok...

>wasn't planning to install Mozilla then, so I didn't follow it.

>works, but it fails to do dns resolution, so it can't move around...

It was me who mailed the same question a few days ago. Nobody answered
so I asked Goto via mail. He didn't anwer also. I don't know what is exactly
the problem but a workaround is given in the README of Goto's ftp directory:
         "you MUST use a proxy to resolve DNS"

If you use a proxy either by manual or automatic proxy configuration this
proxy will resolve the names for Mozilla. I don't know what to do if you don't
have access to a proxy :-(.
Don't forget to enter the IP address and not the name of the proxy server.

I hope this helps and that the problem will be solved in the near future.
