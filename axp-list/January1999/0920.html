I took out the Mesa support at the last minute soooo you don't need it.
Kdeutils and kdenetwork have bugs and just won't build, I tried two
different snapshots with no luck. The installation directory has also
changed to the default /opt directory so If you plan to use these you
should uninstall your old version.

Here's the list:

kdegames-990118-pre-1.1.alpha.rpm
kdemultimedia-990118-pre-1.1.alpha.rpm
kdegraphics-990118-pre-1.1.alpha.rpm
kdeadmin-990118-pre-1.1.alpha.rpm
kdebase-990118-pre-1.1.alpha.rpm
kdelibs-990118-pre-1.1.alpha.rpm
kdesupport-990118-pre-1.1.alpha.rpm

Requirements?:

I'm building on a RH 5.2 system so if you have a fully updated 5.1 you
shouldn't have any trouble. libstdc++-2.9.0-4 is required so you might
as well upgrade to egcs-1.1.1-4.

get them at http://math.stcc.mass.edu/petrakis/downloads/kde
