>>> Martin Lucina said:
>
> Just looking through the binaries on
> gatekeeper.dec.com:/pub/DEC/Linux-Alpha/X11, this all seems to be pretty old
> stuff.

Indeed - 2 years old or more, based on XFree86 3.1.2. At best built under
one of the RH 4.x releases, so not the new GLIBC.

> Is there a working accelerated server for TGA? Or are the XFree86 people
> working on one?

There've been a few people who've asked for TGA docs (see the NetBSD site
for those and many other old DEC products), but I don't believe they've
released anything to this point.

Also, DEC did contribute a lot of accelerated code to XFree86 more than a
year ago, but again, it appears that nothing has yet been done with it. The
current TGA server has little, if any, acceleration, AFAIK.

I'm sure that if someone volunteered to join XFree86 development, those
accelerated sources would be made available, if they aren't already filed
away somewhere on their site.
