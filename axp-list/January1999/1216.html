In case anybody cares, here is the setup script I've been using for a year or
so:

I have the following alias set up in my .profile:
sys=`uname -m`
alias setjdk=". $HOME/bin/setjdk_$sys"

And then in ~/bin, I have a setjdk_alpha that looks like:
# Set up environment for using JDK
JDK_HOME=/usr/local/
JAVA_HOME=/usr/local
SWING_HOME=/usr/local/swing
CLASSPATH=.:$HOME/java/:${HOME}/java/com/bls/SSF/crysec.jar:${SWING_HOME}/swingall.jar:${JDK_HOME}/lib/classes.zip
export JDK_HOME JAVA_HOME CLASSPATH
export PATH=$JDK_HOME/bin:$PATH

I have a similar setjdk_i486 that looks like:
# Set up environment for using JDK
JDK_HOME=/usr/jdk
JAVA_HOME=/usr/jdk
SWING_HOME=/usr/local/swing
CLASSPATH=.:$HOME/java/:${HOME}/java/com/bls/SSF/crysec.jar:${SWING_HOME}/swingall.jar:${JDK_HOME}/lib/classes.zip
export JDK_HOME JAVA_HOME CLASSPATH
export SWING_HOME
export PATH=${JDK_HOME}/bin:$PATH
export LD_LIBRARY_PATH=${JDK_HOME}/lib/i486:${JDK_HOME}/lib/
export JAVA_COMPILER=tya
