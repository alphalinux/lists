Hi !

(a) I haven't used a dual processor before, so how much performance can you
expect over a single CPU machine when working on a single job using an SMP
linux kernel. Can you configure the system to use both CPUs for an ordinary
job, or does that require a special compilation option/re-coding?

In general if you run only a single non-threaded application the performance
is not better than on a SMP system.
In fact your application may run slower, because there is some overhead to
keep the processors in sync (locks).
So SMP works on a thread base.

(c) Will code already compiled for ev56 CPUs run efficiently on an ev6 CPU?

The design is different, so for optimal performance you need to optimize for
EV6.
