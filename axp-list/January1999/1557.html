>>> Jaap Hogenberg said:
>
> I'm planning on buying a PC164SX MB with a 533 21164 CPU,
> with a Adaptec 3940 SCSI controller (2 ch Ultra/Wide) :-)
>
> Is anyone using such a controller with Linux, is it supported ?
>
> I would like to use 2 channels, 1 for slow devices like my cdrom player,
> and the other for fast UW disks.

It's a possibility that it will *not* work with any of the existing SX
binary kernels and MILOs available, because of the newer DEC PCI-PCI bridge
chip that is present on that board (DC21152). It would work with the 2.2.x
kernels, and I have patches for 2.0.35 that could also be used to build a
MILO that would work, but to this point I haven't done so... :-\

On the other hand it may just work; if the console (SRM or AlphaBIOS) sets
up that 21152 bridge in a compatible fashion to the older MILO/kernel, it
will be OK.

If there's any way you could try it before committing yourself, I strongly
urge you to do so.
