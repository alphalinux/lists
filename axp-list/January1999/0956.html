I had similar problems with xcdroast-0.96d (compiled with egcs 1.1b)
on a machine running RedHat 5.1, but I found a (rather inconvenient)
workaround: I called mymkisofs-1.12b3 and cdrecord-1.6 manually
from the shell using copy and paste from xcdroast.log after doing
the setup in xcdrecord. I didn't try recompiling it with egcs 1.1.1
so far. Warning: while cdrecord returned with an error message,
mymkisofs simply created different output if called from xcdroast.
Which version of xcdroast do you have? Which compiler?
