Hmmm... I thought that x86 was little endian, the same as Alpha. I think
this is the case as Windows NT uses little endian on x86 and Alpha systems.
I really don't know how many system types are big-endian, but I read that
HP-UX will be the main (only?) OS that will operate the still-vaporous
IA-64 in big endian mode. I'm still waiting for the '264 systems become
less vaporous (more common) so that I can buy a faster '164 chip in a few
months.

I'm getting th idea that everyone is reinforcing the idea that security
through obscurity is not a reliable notion. It should only be used as an
additional sandwall of an otherwise viable security setup.

Now can we _PLEASE_ kill this thread? Either the machine, if truly for
sale, has been sold or will not be for various reasons. As for the
security debate, I don't think there is much debate anymore, it seems that
there is little new to be said now.

Of course I could be off my rocker in any point that I gave here, so if I
am incorrect, I don't mind corrections.
