It should.

The important things are:

1. that the system type name is recognized by GLIBC, and
2. that that system type's PCI bus I/O and memory parameters are appropriate
   for the actual hardware present.

Nearly all EV4 platforms, whether based on LCA, EV4, EV45, are identical
WRT #2 above (JENSEN and SABLE are exceptions). Same is true of the EV5 boxes,
again with exceptions such as SABLE, RAWHIDE, and RX164. And so far, no EV6
box exceptions, as we're only supporting TSUNAMI-based ones (TurboLaser-6
would definitely be an exception :-).
