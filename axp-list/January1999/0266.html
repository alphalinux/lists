I have still some troubles to understand how the alpha-cpu
issues instructions.

Maybe some kind soul could help me with this example?

/* M[0] = 0xaaaaaaaaaaaaaaaa; M[1] = 0xcccccccccccccccc; M[2] = 0xf0f0f0f0f0f0f0f0; */
/* M[3] = 0xff00ff00ff00ff00; M[4] = 0xffff0000ffff0000; M[5] = 0xffffffff00000000; */

        lda $0,-1
        zap $0,0x0f,$6
        zap $0,0x33,$5
        zap $0,0x55,$4
        srl $4,4,$3
        stq $6,552($30)
        xor $4,$3,$3
        stq $5,544($30)
        srl $3,2,$2
        stq $4,536($30)
        xor $3,$2,$2
        stq $3,528($30)
        srl $2,1,$1
        stq $2,520($30)
        xor $2,$1,$1

How many cycles would this take (on a 21164) and which instruction
do get paired together?

How would you optimize it?
