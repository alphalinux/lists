Hello *,

contrarily to using pre-build RPMs, I just like to download
the original sources, and build those. For Ghostscript 5.50
I did just that, and they compiled fine (no serious warnings).
It runs fine also when presenting some PostScript on the X11
window, but it dumps core when using the cdj550 (or other
deskjet drivers) for my HP DeskJet 600.

RedHat Linux/Alpha 5.1 on Alpha PC164SX/533
egcs-2.90.27 980315 (egcs-1.0.2 release)
unix-gcc.mak or unixansi.mak as makefile

Using driver djet500 does work problemless. Anyone commenting?
Was any special work done for those sources in the RPMs
currently being mentioned on this list?
