The url for the library is
ftp://ftp.digital.com/archive/private/netscape/CXXREDIST600V01.tar
per netscape's support pages.

Assume untarring to ./CXXREDIST600.kit/

I extracted it by doing:

[[root@mother larrys]# cd CXXREDIST600.kit/
[[root@mother CXXREDIST600.kit]# ls
CXX.image CXXREDIST600 INSTCTRL instctrl
[[root@mother CXXREDIST600.kit]# file *
CXX.image: ASCII text
CXXREDIST600: compress'd data 16 bits
INSTCTRL: POSIX tar archive
instctrl: directory
[[root@mother CXXREDIST600.kit]# mv CXXREDIST600 CXXREDIST600.Z
[[root@mother CXXREDIST600.kit]# uncompress CXXREDIST600.Z
[[root@mother CXXREDIST600.kit]# ls
CXX.image CXXREDIST600 INSTCTRL instctrl
[[root@mother CXXREDIST600.kit]# file *
CXX.image: ASCII text
CXXREDIST600: POSIX tar archive
INSTCTRL: POSIX tar archive
instctrl: directory
[[root@mother CXXREDIST600.kit]# tar tvf CXXREDIST600
-rw-r--r-- root/system 286720 1997-12-01 15:37 ./usr/lib/cmplrs/cxx/libcxx.so
[[root@mother CXXREDIST600.kit]#

You'd want to do the final step (with x instead of t) from /, or just
move stuff around and add the new name w/ ldconfig....

I have it on good authority that this works. I'm waiting for my licensed
copy of DU to try it here....I'd REALLY like to be running NS450
natively.
