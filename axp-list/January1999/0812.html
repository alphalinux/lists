>>> Robert Fridman said:
>
> Has anyone gotten a DPW500au (Miata GL) to boot RH5.2 from SRM? I have one
> without a graphics card so my console is a tty.

Yes; what exactly is the problem? You lose output to the tty once the kernel
is loaded and running?

You need to run a 2.2 kernel to get reasonable serial console support. Read
the document on it under Documentation/ in a 2.2 source tree.

Note that the setup is somewhat difficult from scratch; it may be necessary
to temporarily install a VGA to make it easier.

Good luck.
