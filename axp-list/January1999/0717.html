Thought I'd post this to the list.
        HotJava DOES work on the Alpha. I have mine running now.
        It's available at:
                http://java.sun.com:80/products/hotjava/index.html
        There is no version out for the Alpha, but there IS an "unsupported"
version that simply doesn't have a JRE element, and will require that you
install the JDK ( check through alphalinux.org (Great Site!!)).

        Mine runs great, but it's a memory hog. I only have 48MB of ram on the
Alpha (ouch!), and running HotJava uses it up, and into swap.... Gotta get
more ram...
        Would anyone be interested in swapping a 2GB SCSI-2 Fast harddisk for
some memory? heh heh...

        Ric
