I would recommend using the 'make xconfig' and configuring the kernel in X.
This not only gives you access to decent to good explanations for each choice.
It also allows you to move around in the config. Most of the kernel compile
(not bleeding edge stuff) is really done for us. Just the 'make clean' 'make
dep' 'make boot' 'make modules' 'make modules_install'
Nothing more to it.
