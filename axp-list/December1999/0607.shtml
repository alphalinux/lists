You should only need a 2 meg fat partition. On it goes the milo
image for the noname, not the kernel image. Dunno if you can
rawrite to it under NT, though. As long as it has a drive letter,
should be OK. If you are tied into the network, you might be
able to boot the kernel and the ramdisk off the network/ internet
once you get to the MILO> prompt.

Realistically, the best thing to do would be to take the drive to a
working Linux Alpha machine and set it up there.

My .02
