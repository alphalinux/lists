Well I'm presently having a hand-keyboard fight with the version released
by 3dfx; it is getting there. I have my matrox server running fine and I'm
trying to get the 3dfx up.

LX which I've told to run the VGA initialisation but not to run the x86
emulation. That always boots - but 50% of the time doesn't actually
initialise my Matrox - hmm.
