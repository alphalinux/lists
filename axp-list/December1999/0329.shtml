There have been some good replies referencing tools such as 'cp', which is
fine if you can/don't mind putting the drive from the new system into
another system (or vice versa).

However, I thought I'd mention another idea, which is relevant if you have a
copy of NT4 Workstation or Server lying around. We wrote a *free* utility
called "Stand Alone Disaster Recovery" which was written to allow NT users
to backup or restore an entire physical drive. Although based on NT and
written for NT, it will happily backup non-NT systems (and any type of FS).
What the tool does is allow you to create a set of 3 bootable floppies (5
floppies for Alpha systems), using files mostly from the NT CD. If you have
the smarts, you can burn a CD with these files on them (just make sure that
the files on the floppies exist in the \ALPHA directory on the CD). When
the program is invoked, you can backup from one physical drive to another,
-OR- you can backup/restore the entire physical drive to a tape drive (most
are supported, although spanning is not supported so it must all fit on one
tape). It is a 68KB download from our website, just click on "download".
