Hi, all;

I posted the message below to comp.os.linux.alpha; maybe someone here
knows, too. In short:

Is there a current maintainer of the aboot package "upstream" of the
Linux distribution outfits?

Thanks,
Craig

The post to comp.os.linux.alpha:

----------------

Greetings;

I have a patch for aboot which allows booting on
the DEC 3000 turbochannel machines, as I am
working (slowly) on getting linux to run on these
boxes. I've asked a few people to whom I should
submit this patch, and have been informed by some
that aboot currently has no maintainer upstream of
the various distributions.

It was suggested to me by David Mosberger to ask
here and verify if this is true or not.

Anybody know?

Thanks in advance,
Craig

P.S. In the event there is no known aboot
"upstream" maintainer for aboot, I should like to
volunteer. I can at least give the package a home
until someone more capable/willing than I wants to
do it... I can give it a web page, update docs,
review/accept patches, make any announcements,
etc.
