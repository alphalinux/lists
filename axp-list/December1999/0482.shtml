You need to use the command 'make boot' for compiles on Alpha, not the i386
'make zImage'
Just a small, but often troublesome difference.
James
then you can find the bootable file in the normal path
/usr/src/linux/arch/alpha/boot/ (I think this is it - I am doing this from
memory).
