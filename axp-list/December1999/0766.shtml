The Mustang is slow because it's...er, because it's an old box. You need
more RAM to get respectable speed. I have 96M and it works pretty well
with that. I'd expect 97% mem usage with only 32M.

The kernel is in the /boot directory, so load it with:

sda5:boot/vmlinuz-2.2.5-16

Make sure you install *all* updates for RH6.0 too.

For ethernet, edit conf.modules and set eth0 to be de4x5 instead
of tulip.

/Michael
