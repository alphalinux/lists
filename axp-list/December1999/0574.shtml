> Hello all. I have a Multia 166 and a SCSI CDROM and two SCSI HD's (9GB and
> 2GB). The 2GB is blank and I have installed NT4 Server on the 9GB drive
> since I couldn't get Linux going right away. I got as far as booting the
> kernel (generic.gz) and it finds the SCSI devices, video, etc (even with the
> Tux graphic) until I get the "Warning: Unable to open initial console" error
> mesage. All I can do from there is 3-finger solute back toMILO (which I am
> starting from ARC). I have been searching through the list archive trying
> to find a solution but all I can find is "use the floppies". Well, I don't
> have a floppy drive so that doesn't work for me. The 2GB HD is totally open
> to putting anything on it that would allow to me to install RH6.0. Any help
> is very much appreciated.

Use a floppy as the ramdisk instead of the CD. This should allow you to
install.
