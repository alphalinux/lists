I had linux running on an DS3305 (500 MHz). Yes it is an AS800 with
an ugly white paint job. Yes it has SRM. Yes you need to install using
SRM (no milo). I did find that RH6 would not boot off of floppies.
It did work to boot directly from the RH6 CD.

Needless to say, reading the alpha addendum regarding installing on
a system that boots from SRM will save a LOT of aggravation. An
incorrectly partitioned disk makes things very unpleasant.
