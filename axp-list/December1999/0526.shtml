> It's trivial to install (boot the CD and go). All the normal support things

It's extremely easy to install, if you are running a "stock" Alpha. If you
bought a motherboard and processor and started adding in components (as I
did), getting Tru64 running is going to be much harder. You basically
need SRM firmware and a very specific set of hardware: NCR53c8xx SCSI
controller, DEC Tulip NIC, one of maybe two video cards unless you like
640x480, etc. Of course, you can do without a particular component
(except without SCSI you can't install -- IDE isn't supported).

The good news is that it *IS* doable, if you stick to it.
