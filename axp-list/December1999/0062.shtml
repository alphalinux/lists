On Thu, 2 Dec 1999 mikeh@iuinc.com wrote:

> the built in video card only seems to be able to do 640 x 480 resolution..
> can somebody recommend a good graphics card? like a pci card, 12-16mb ram?
>

The built in video card should be able to go as high as 1280x1024... I'd
double check your configuration file :-) I've successfully used an ATI
Xpert98 card with the Mach64 server and a Permedia2 card with the 3D_Labs
server... Both worked extremely well with 16bpp @ 1600x1200. The author
of the Savage 3 & 4 driver is working on getting the Savage 4 to work with
the Savage 3 driver in the SVGA server. He's got it pretty stable with
8bpp and is now trying to get it working in 16bpp. You might want to take
a look at www.alphalinux.org to see what video cards are listed there as
well :-)

> Also, can I configure the udb to have 4 64mb simms? or only 4 32mb simms?

That one I don't have an answer to...
