I usually try to stay with the bleeding edge, which means regularly looking
for the latest RawHide version. I've never had problems with the binary
packages; the src ones did sometimes not compile out of the box.

Mostly this is just for the feeling of being ut to date; occasianally there
are security fixes, but those are also available in the `updates'.

Usually there is no need to generally use RawHide, just fetch those programs
you regularly use and want the latest and greatest version of.
