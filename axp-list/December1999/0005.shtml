On 30-Nov-99 Peter Rye wrote:

> I tried compiling the source with the -mieee flag just for fun, but this
> didn't solve the problem although the build completed with a "few"
> warnings along the way.
 
> If anyone has managed to run this could you let me know any tricks you
> used.

All I did to get a binary I could run was:

./configure --enable-optimize --disable-debug --disable-mailnews \
--disable-editor --disable-tests

And then run make. And wait. And wait some more.

This was on an AS200 4/233 that began life with a RH 5.1 installation but has
since mutated into something entirely other. ;)

Yes I know about the glibc version / stability issues. But I'm not ready to
change to RH 6.x for fear that everything *else* might stop working right.

Someone at someplace other than this list mentioned that having stuff around
from earlier Mozillas might cause a crash on startup.
