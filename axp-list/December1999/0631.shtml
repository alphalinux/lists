I just got RH6.1 going on my Multia and have been going crazy trying to
figure out how to enable support for PCMCIA. BTW, what a pain in the a*s
working around all the RH installer problems. (Had to boot in single-user
mode to create a /etc/sysconfig/network-scripts/network file containing a
value for hostname, plus had to manually paste the encrypted password in
/etc/passwd from a regular user account to the root password since the
installer decided to leave it blank - passwd wouldn't set it either). I'm
still getting a bunch of "Unary expected" errors in a bunch of the
initscripts (I thought the new version of initscripts - 4.70 - was suppossed
to fix these problems?). Anyway, back to the matter at hand, since I can
still boot past those errors, I am more concerned with getting PCMCIA going
so I can get the Multia going as the server for my LAN instead of a client
:) I have a possibly unsupported card (New Media Livewire 10/100). It's
not the Livewire+ which is definitely unsupported, but it's not listed as
supported either, potluck I guess. Any help is greatly appreciated (I'm
sure this is an easy one, I'm just not looking in the right place).
