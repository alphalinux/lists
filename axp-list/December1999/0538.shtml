I looked at the palcode source for the DP264 from the motherboard
SDK. Vector 0x630 appears to be for a processor corrected error.
Looks like it is correcting single bit errors, either in memory or
in the cache tags. Most likely a main memory problem but could be
a bad processor or a problem on the motherboard.

Kelvin
