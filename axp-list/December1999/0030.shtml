> Now the questions:
>
> 1) Is anybody planning to make MILO work with NetBSD*;
Chances are very good that folks using NetBSD/axp prefer
SRM over milo/alphabios. [personal opinion] I think it
makes more sense to use SRM on a box that will run only
*nix, and am more comfortable with it [/personal opinion]

> 2) Is Compaq planning to document the NT Firmware 'way';
AlphaBIOS is fairly well documented, and the source code
to the Linux loader is available. There is an SDK
available for developers as well.

> 3) Will Compaq release SRM firmware for AlphaBIOS based
> Alpha's like the XLT machines;
"Most" DEC-branded hardware has the capability to run
ARC, ABIOS, SRM, or provide a choice between two options.

> 4) Can I trade my Alpha's for free at the local Compaq
> store for SRM based ones :^)
Now that, I think, is wishful thinking ;)
