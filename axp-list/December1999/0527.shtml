> Designer Seven wrote:

I remember last year spending some time to get Merit Radius
working on AlphaLinux. It did not authenticate users due to
an MD5 problem similar to that reported by Designer Seven.

After strenuous debugging and hard thinking, it turned out to be a
trivial porting problem: all I had to do was to edit a file named 'md5.h',
turning
        #if defined(__alpha) && defined(__osf__)
        typedef unsigned int UINT4;
        #else
        typedef unsigned long int UINT4;
        #endif
into
        #if defined(__alpha) /* both OSF and Linux! */
        typedef unsigned int UINT4;
        #else
        typedef unsigned long int UINT4;
        #endif
and then it started working flawlessly!
