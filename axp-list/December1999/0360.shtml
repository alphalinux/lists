Sven,

I found this to work:
- start fdisk
- x (for expert menu)
- c, h, s (to set cylinders, heads and sectors)
- r (return to main menu)
- b (for BSD-style labels)
- create partitions
- w (write)
- q (quit)
- reboot
I have to do the first steps every time: it doesn't seem to remember the
geometry.
