On 2 Dec 99, at 18:40, Fernando Monera wrote:

> Hi,
> I finally installed RedHat6 on my 150 AXP. Very thanks to everybody,
> sincerely.
> I now have another problem: It doesnt boot!!!!.
> I write:
> aboot> 0
> And then starts loading the kernel 2.2.5 that RH6 installs. But when it
> tries to start "kswapd" (almost in the begining) it hangs (and says
> nothing).
>
> Tomorrow I'm going to remove the swap partition to see if it helps.
>

This has nothing to do with swap partition. This is problem with
gawk. gawk is broken for Jensen. boot in single user mode &
install mawk rpm. then change existing symlink /bin/awk to point
to mawk instead of gawk. & you should be all set to tackle next
round of problems :)
