I am trying to install RH6.0 on a PC164LX system with the SRM console. On
reading the SRM howto it says it is possible to boot straight from the CD. I
seem to have some success but not much. I am using the following line to try
and start the install.

>>>boot dka300 -file kernels/generic.gz -flags "root=/dev/scd0"

DKA300 being the scsi cdrom. It starts to boot with various initialisation
messages then I get some aboot messages and it reports that it is loading
compressed kernels/generic.gz. The last line says

aboot: ok, now starting the kernel..

After this message nothing else seems to happen. The system reports nothing,
doesn't crash or halt it just seems to have stopped dead in its tracks. Is
it possible to install startight from CD using the SRM console ?

Any help would be much appreciated.
