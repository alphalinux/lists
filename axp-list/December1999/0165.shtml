I don't know why it's crashing for you then. Perhaps a bad font selected
somewhere. My configuration is almost the same as you; RH6 on an AS200,
2.2.13, latest updated Kde and XFree86 RPM's, TGA video, frame buffer
console. I had problems at first, but they went away with the xfs update.
If you can't get it working, you could try hacking your way around it by
setting the fonts dir in your XF86Config file by hand, and running without
the font server. I did this until the xfs update solved my problem. Only
downside is that if you get a bad font render, the entire server might blow
away.
