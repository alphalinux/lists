I still have 4.0 D libraries running NS 4.6, I haven't bothered to get the
new non-commercial one yet. It is very easy to get the Tru64 libraries
installed, see http://www.alphalinux.org/docs/netscape_du.shtml. Besides
being able to run Netscape and Adobe Acrobat Reader you can run most any
Tru64 application like for example Mathematica (which costs as much as
the commercial version of Tru64).
