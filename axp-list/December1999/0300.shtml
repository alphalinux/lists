If this is indeed "something in that range" that it should be
good enough. Still you may want to check
ftp://ftp.varesearch.com/pub/support/hjl/binutils/
if you can find there something newer and if yes then if it makes
a difference. There are things there like binutils-2.9.5.0.22-1.src.rpm
which you can recompile with a simple command
  rpm --rebuild binutils-2.9.5.0.22-1.src.rpm
(this should make after a while binutils-2.9.5.0.22-1.alpha.rpm
which you can then install as usual but you better do all of this
from a 'root' login).

No, I do not know what PWS 500a is doing and what may really spook it.
