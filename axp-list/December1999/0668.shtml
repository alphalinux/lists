> > Does control-alt-backspace work?

> No, the keyboard is totally frozen, won't accept any keyboard input.

That sucks. I've never had that happan on my Alpha (knock knock) but
dead consoles were pretty common for my on x86 w/ DosEmu and X.

My hack to take care of this was a hardware mod (one of the clearest
indicators that you have an ugly hack is when you fix software with
hardware :) ...

I set an output pin on the parallel port high, and ran this pin to
one lead on my turbo switch, and ran the other lead to an input pin.

Then I just wrote some code that occasionally checks that input pin. If
it is high, we know I depressed the turbo button, so it ran some scripts
that tried to clean up the console. Sometimes the clean up worked, some
times it was so munged that killing X and/or DosEmu didn't fix it up.

In that case, I hit the reset button on my machine which was similarly
rigged to another pin -- when this pin is high, we run reboot.

It was a nice little project .. and it worked pretty good. But it was
the ugliest damn hack I've ever seen.
