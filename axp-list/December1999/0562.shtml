Andrita Constantin wrote:
> Can I build an alpha kernel on a x86 machine?

In theory, yes. I've never tried it though, and I've heard that
using gcc to cross-compile for a 64-bit architecture from a
32-bit host is dodgy at best (and the gcc 2.95 manual even
says that it's "problematic and sometimes does not work")
My advice: don't.
