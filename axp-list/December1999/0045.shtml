On Thu, 2 Dec 1999, Adam Chrystie wrote:

> i compiled my kernel and it made a VMLINUX file...but how do i make a
> VMLINUZ image file of the kernel..or do i set milo to just load the
> vmlinux file now???

Use 'make boot' and you should find in linux/arch/alpha/boot there is a
file called vmlinux.gz - DO NOT UNCOMPRESS THIS (sorry). Then you just
have to ask Milo to boot that.
