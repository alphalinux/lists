On 16-Dec-99 Tiels, Frank wrote:
> All
 
> Does anyone know if there exists a ssh for redhat alpha?
> I don't seem to be able to find it anywhere?

It definitely exists in source RPM form, which is easy enough to build on a RH
Alpha box (rpm --rebuild ssh-1.2.27-7us.src.rpm)

The latest US usable version (with the bug fix for the exploit recently posted
in a CERT advisory) appears to be ssh-1.2.27-7us.src.rpm.

Of course, you can always build without RSAREF, but I'm not certain on the
legality of those builds in the US. Silly math patents ...
