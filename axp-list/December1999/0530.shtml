It's quite a nice setup. However, the CDs seem to have been mastered in the summer
so by now there are more recent pieces (srm consoles in particular). Haven't
looked in depth yet though.

Doesn't seem to want to install on my PWS500a. Stops when enabling dynamic memory.
This box must have one of the memory impaired CPU chips (ie regarding the recent
discussion of NT only cpu chips with dynamic memory functionality disabled).
OpenVMS Hobbyist runs on the box so go figure.. (OpenVMS doesn't use dynamic
memory functionality of the CPU according to the aforementioned discussion). I've
a 164SX that I plan to run it on, & its on the supported platform list so should
be ok.

One caution. The install screens indicate that no changes will be made to the box
untill all questions have been answered. Well, I managed to trash the boot block
of my disk while just 'looking' so now I get to re-install whatever was previously
on the box.

Worth the price of admission. Definitely.
