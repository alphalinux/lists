        You should probably upgrade, but don't actually /need/ to.

        You need the latest gtk+, and glibc 2.1. RH5.2 ships with glibc
2.0, which will kinda-sorta function with mozilla (the shared library
loading functions in glibc 2.0 are not reentrant, so with Mozilla's
threading it'll sometimes cause a segfault), but you'll really want to
upgrade if you want to use it often.

        Upgrading to glibc 2.1 can technically be done w/o upgrading the
entire system, but I've seen enough explosions from trying to do that that I
just suggest a full upgrade at that point. I did an upgrade from RH5.2 to
RH6.0 on my box, and it was relatively painless. YMMV.

- Bob
