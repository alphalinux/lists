On Thu, 2 Dec 1999, Larry Creech wrote:

> Is there a work around or a fix in RH6.0 to prevent Netscape 4.X
> for Digital Unix from killing the Xfont server when ever you try to save
> an attachment from Netscape mail or a "shift left click" in the browser to
> download a file?

Yes - don't use the Xfont server! If you edit your /etc/X11/XF86Config
file to alter the font path to the normal none-font server version then
you should be fine.
