That makes sense. The instruction length should only tie
to the number of op codes and the number of registers that
a machine uses. Note load/store base their memory location
on a register plus offset so one only has to support a reasonable
offset length to access memory or, alternatively just change
the register value used to get anywhere one wishes in memory.
