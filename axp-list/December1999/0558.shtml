Well, since you asked:

I'm working on installing tru64 on an lx164. I have an Intraserver
3140 scsi card (53c875) which isn't supported, but works with some
complaints.
Also a diamond fire gl pro which uses the permedia2 chipset like the
elsa gloria synergy which is supported, that works too. my digital
dec450 nic, which is supported, doesn't work, even though it works
under both NT and Linux (triple boot box using a floppy to flip
between tru64 and alphabios, separate disks for each OS.)

so far it boots and runs X.

when I get the nic to run it'll be great, very nice utilities.

Read the documentation, especially chap 6. there's not much very
intuitive.

the compatible hardware list is VERY short, YMMV.
