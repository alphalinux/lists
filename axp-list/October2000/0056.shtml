At the same time when it will be available for x86 - we hope.

You can get anytime, from ftp.*.kernel.org (substitute your country
code for '*') a prerelease development version. Nobody guarantees
that it will work. Actually in this moment it does not even boot
on various machines including an Alpha which I have here. If it
does boot it may have assorted bugs - including stuff which can
be very detrimental to the general health of your system. If you
are playing with development kernels without a serious backup then
you were asking for it.
