On Thu, 5 Oct 2000, Savvas Michael Koushiappas wrote:

> I'm not familiar with the alpha hardware. The only information I
> have about a video card is that it is a "High performance PCI-based
> graphics accelerator." Does anyone know how I can go about configuring
> X?
> This is actually Debian linux, since Redhat doesn't support my
> alpha. Usually with intell pc's, I configure X by running xf86config and
> entering the video card information. Then the Xserver is selected based
> on this definition. Any help would be much appreciated.

For Multias (aka UDB) running Debian, pick up the xserver-tga deb and
install it. In /usr/share/doc/xserver-tga, there should be a
configuration that is very usable (so you don't need to run xf86config,
just copy the XF86Config from there to /etc/X11).

Let me know how that goes...if something goes wrong, I'll help you solve
it:-) (I've owned three UDB's, an AXPpci33, and an SX, so I'm pretty
familiar with those machines).
