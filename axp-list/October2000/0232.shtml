Hi guys,

Got a problem with a Jensen. Most people do! I was just wondering if anyone
has had any success using the Compaq Jumpstart cd on a Jensen. There are
instructions in the readme for use on a Jensen, but I just got a 84 Fail,
one of the great helpful error messages. I've worked out before in my many
attempts with this system, that an 84 Fail means there is a problem with a
boot loader, i.e. its not finding it. I wrote the Jumpstart cd myself on a
normal CD-R disc, reads fine in the PC, lots of files on the cd etc,
including the bootlx file needed to boot. I'm curious if this is a problem
with the cd drive, the cd I burned, or if I'm doing something else wrong.
Also, is there a way of starting the Jumpstart cd from aboot? I presume the
jumpstart has aboot in the boot bit, and just automates the bootup, so what
would the command be to start it up?

I've also tried using the other method, building the jensen.img and
ramdisk.img discs, etc, and can get into the RedHat 6.2 install program.
Things go fine, until it asks about the timezone. Then a message comes on
screen, /dev/port invalid file or directory. The menus still work, but when
I click the ok button, the menu disappears, and the machine does nothing, no
activity at all. I left it sitting just in case it was going to do
something, but after 10 minutes there was still nothing, so I just switched
it off.

Cheers,
