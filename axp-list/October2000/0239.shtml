This IS the wrong list for that, but here goes:

Installing and removing packages under Digital UNIX or Tru 64 UNIX

log in as root
Mount the distribution CDROM in the drive, and mount the CD filesystem. It
is UFS type.
cd to the mountpoint of the CDROM
If you don't know how to mount a CDROM filesystem try "man mount"
The command for package installation is called "setld"
setld with no parameters will show usage
For listing uninstalled packages try:
setld -l .
This will allow you to choose and install packages.
