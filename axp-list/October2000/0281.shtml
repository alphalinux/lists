I got all of the rh7 SRPMS from ftp.kde.org and with some very few changes to
some of the spec files it looks like this compiles on a rh6.2 box. I've only
compiled flex, htdig, libmng, and kdoc so far. And qt-2.2.1-5 seems to be well
on its way to finishing. I have the src.rpms and any compiled binaries at:

ftp://chaos.math.twsu.edu/pub/AlphaLinux/kde2-final/

So far only the kdoc and qt-2.2.1 src.rpms that I have posted are different
(only with the spec files). But, that could change as I compile the rest of
the applications. For compiling I'm using cxx aliased as g++ to see if I can
get some problems that were on AlphaLinux and not on Tru64 to go away.

This is running on a work computer so I will not get anything more than qt
compiled by tomorrow. There as some nice new stuff like the no_exeptions
compiling of qt in this release of kde2 final. So memory use should go way
down. Want to see how much kde2-1.93 uses with exceptions? Here is my top
