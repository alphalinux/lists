On Thu, 5 Oct 2000, Haniel Mercedes wrote:

> I have a MegaRaid controller (NetRaid) in a HP-LXr8500 server,
> I just installed Linux rh6.1 running 2.2.14 kernel,
> so I am trying to configure a Raid 0 using raid tools v0.90,

It sounds like you're trying to setup Linux software based RAID on a
hardware RAID device. If you have several disks hooked to the MegaRaid
controller and want to do something like RAID0 or RAID5, then you don't
need to worry about software RAID.

Hardware raid is pretty much transparent to the underlying operating
system, provided the OS has a working driver for the RAID controller. You
do the RAID configuration either in the card's bios setup, or using
software tools, and the card basically presents the array to the OS as a
single device. On the other hand, Software RAID is very much OS dependant,
and is only necessary when you want to do RAID but don't have a RAID
controller. A case where you might want software raid is if you have a
non-RAID SCSI controller with several disks attached to it.

In your case, you don't need to worry about the kernel RAID patches or
utilities.
