On Fri, 20 Oct 2000, S VDE wrote:

> A newbie question: how does one apply the patch...

Basically, get the source used to build the gcc on your system (on RH, get
the SRPM; on Debian, grab the .tar.gz and .dsc) and unpack it. Then, cd
into the source directory and type 'patch -p1 < {patchname}' (omit the
curly braces).

It should apply cleanly (or fuzzy). If you're a Debian user, I'm building
a new gcc now for upload. If all goes well, it should be in the woody
tree by tomorrow afternoon.
