Try disabling "SCAM Support" on the Symbios card. SCAM stands for Scsi
Configured Auto-Magically. It is supposed to automatically find a free SCSI
ID for devices that support it. That may be fine on a windows machine, but
Linux is looking for a specific ID, ( i.e. ID3 = sgc, 4 = sgd, etc. )
