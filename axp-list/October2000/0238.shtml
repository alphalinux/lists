I had quite a bit of trouble getting my PC164 to read CDR's with the
original SCSI 8x Toshiba drive. After talking to Toshiba the felt their
was
something wrong with the drive because it should be able to read CDR's.
Since I had two of these drives doing the exact same thing I knew they
were
full of it. After switching to an newer 36x IDE drive I was at least
able
to see the drive using milo, but I still couldn't read any CDROM's that
read
fine on the PC. I was using Adaptec easy cd creator to make my cdr's.
My
first mistake was I wasn't burning them using iso9660 format. The
second
mistake was under properties for the ISO9660 format there is a setting
for
mode 1 or mode 2, which sets the format to CDROM or CDROM XA. I was
using
CDROM XA by default and milo wouldn't read the disk. After changing that
to
mode 1 no problems. I am not sure about aboot since I haven't' used it
but
it might be a place to start.

I gave up on the jumpstart CD, I couldn't make it work. I am not
familiar
with a Jensen does it use arc or alpha bios?
