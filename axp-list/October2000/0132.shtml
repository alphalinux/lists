I've had qt2 and kde2 (1.94) running on a stock RH 6.2 box for awhile. You can
pickup the compiled alpha.rpms at . . .

ftp://chaos.math.twsu.edu/pub/AlphaLinux/kde2-1.94

The src directory has the src.rpms with the needed changes to the kdelibs from
Thomas Leitner (for netwm.cpp). From a newsgroup post I've read (on
comp.alpha.linux I think) I don't believe that netwm.cpp was fixed in this
release either, but I still need to verify that.

I get my src.rpms from http://people.redhat.com/~bero/ and he has new src.rpms
for the official rc2 at http://master.kde.org/~bero/rc2/rh6.2/srpms/

As far as not compiling I've had to clean the qt source after the compile dies
and then short-circuit to get it to finish. But overall these do compile and
run (with bugs in several apps). I haven't tried rc2 yet though. A couple
of things I'm trying now is a hint from the kde lists is to add
-fno-exceptions to the CXXFLAGS to speed things up and using cxx to see if it
fixes several major bugs with kmail from the 1.94 release.
