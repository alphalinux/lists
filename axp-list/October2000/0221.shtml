Luigi Cavallo <cavallo@chemistry.unina.it> wrote:

> Anything else that I could try ?

I use the following procedure on an ancient 2.2.5/tulip/RH6.0
Ruffian connected to a Kingston KNS800/WG switch:

- power on
  => machine will come up, no link
- after the system is up, reboot
  => machine comes up with link,
     switch says it's 100Mb/s full duplex,
     machine thinks it's 100Mb/s half duplex
- pull out network cable, put it back in
  => switch and machine agree on 100Mb/s half duplex

It doesn't always work like this, some additional fiddling with
reboots and pulling out the cable may be required, but eventually
it works.
