> Hi

> Well I finally got my hands on A/S 4100 today and decided
> to get on and install linux.
>
> I setup the boot to be SRM and duly started up the RH7.1
> boot disk.
>
> However after aboot has done it's stuff and kicked off the kernel
> the screen just scrolls and scrolls with what looks like panic after
> panic.
>
> It keeps on goint like this. I can't read the panic message as it's
> scolling pass so fast I can't read it :-(
>
> Are there any basic pointers for getting Linux to install on this box ?

First, make sure your BIOS stuff is up to date:

<http://ftp.digital.com/pub/DEC/Alpha/firmware/>

be careful to read up on interim updates as, depending on your current
SRM/Alpha BIOS versions, you might need to upgrade to an interim version
first - I had a tense half hour after trying to upgrade where I was just
getting SRM errors (luckily was able to break the error loop long enough in
SRM to get the upgrade process restarted on an older BIOS disk!!!)

other than that, we'd need a bit more info as to the first 'panic' message,
methinks
