Heid Oliver wrote:

> The recent 2.4.15-pre kernels (i cant compile the 2.4.14 out of the box, and
> I didnt' try 2.4.13) seem to have a problem with MILO on my PC164.

I have four pc164s and non of them run the 2.4 series reliably.
There was a recommendation on this list some time ago that
pc164 users should probably use 2.2.19, the current stable
2.2 series kernel for alpha. I've been running 2.2.19 very happily.
Even my network card problems (I and many others have
complained that heavy network loads crash pc164s) have gone away
and I'm running heavy loads at 100 Mb/s.
