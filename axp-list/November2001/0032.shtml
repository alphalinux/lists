On Mon, 5 Nov 2001, Davies, Huw wrote:

> I'm trying to install a 40G Seagate ATA-100 disk into my new PWS 500a
using
> a Promise Ultra 100 TX2 controller.

You can double check me, but last I looked promise only supplies precompiled
modules for X86. Not source code. Thus if you run Linux on anything
besides and X86 (like ALPHA or MIPS or SPAC) your SOL,

> On booting, Linux recognises the controller but it doesn't see a disk.
I've
> checked that the disk spins up and that the cables are sensibly attached.
> The disk is configured as a master. Unfortunately (or there again, perhaps
> not) I don't have an Intel box to test this on.

We had the same problem. We decided to goto a well supported SCSI solution.
MANY of the IDE raid solutions we tested were provided by companies that
seem to oppose open sourceing their drivers.

> Any suggestions welcome.

Ask Promise for an alpha compiled driver or ask them for the source. Baring
that you may have to get a different controller card.

If you do get some source code I'd be interested in your results. We've
tried to get different promise RAID chipsets working on a few different
platforms with varying degrees of success.
