> Can one purchase an update CD that automatically installs all updates
> for RH6.2 by clicking an "update" icon?

There is KRUD distribution which is really a subscribtion service
providing updated CDs for RH. No idea if they do that for Alpha.

I do not know about "clicking" but typing 'rpm -F *.rpm' in
a directory full of update rpms does effectively that so you can make
yourself an icon with this action. This will not work if you attempt
to install things like the same version of glibc both for "alpha"
and "alphaev6" at the same time. In such situations _you_ have to
choose.
