What is the difference?

Well, let's see, ...

RH7.1 comes with Mozilla 0.8.1-5.

When I try to save page source to a file, the file selector
window won't allow keyboard input into the file name box.
The only way to save page source to a file is to use the GUI
to overwrite a file that already exists.

I use xmodmap to make the key to the left of the 'A' key work
as CTRL. (Other users on the system want to use the nasty
"caps lock" function, so I need to use xmodmap locally within
my personal account.) Nearly everything accepts the mapping
and does the right thing. Mozilla is one of the very few
things I have tried that doesn't.

With due respect to the (presumably non-commercial) developers
who have made Mozilla available, if/when it becomes somewhere
near production-worty I'll take another look at it.
