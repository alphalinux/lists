>OK, I'm lazy... But is there a 2.4.14 kernel rpm available for
>redhat7.1 alpha? I've looked on rawhide, but see nothing.

The latest official stable release of any piece of software for
the distribution, including the kernel, can be found on
ftp://updates.redhat.com, or obtained from Red Hat Network via
up2date.

The latest developmental packages of all software are always in
rawhide, located at ftp://rawhide.redhat.com, and updated usually
once a week I believe.

If a particular package has not been officially released as an
erratum update, and also does not appear in rawhide, then it is
generally not available.

Hope this helps.
