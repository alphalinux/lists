On Thu, Nov 08, 2001 at 02:36:48PM +0100, Peter Bech wrote:
>
> Is there still no one out there having tried to install the official RH7.1
> Alpha CD on the AlphaServer 800?

I also have had problems installing RH 7.1 on certain Alpha platforms
that have:

1. QLogic SCSI controller built in (AS800, AS500, late PWS)
2. SCSI CDROM on that QLOGIC's SCSI bus along with SCSI HDDs

The problem is in the QLogic driver in the 2.4.x kernel RH used - it
no longer works with the above configuration (which did work under 2.2).

My solution: add another SCSI controller and take the CDROM off the one
with the HDDs.

Another solution (not useful for installs): replace the Linux QLogic
driver with the one ported from *BSD by Matt Jacob, found at:

        www.feral.com

Good luck.
