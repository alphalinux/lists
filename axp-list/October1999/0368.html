Indeed this is interesting. I know it is NOT the LX design as we have many
customers with very long uptimes on LX.
I think I agree with the suggestions to try some different memory, if possible.
Generally look in any machine check errors for lines mentioning "630" or
especially"670" which are often memory errors.

Also, do you access to a digital thermometer with a probe? If so maybe (
with the case closed and the system running for at least an hour) check
temperature of air coming off the CPU, and at the memory modules. I have
seen instances of the memory overheating, especially on larger ( 256MB)
modules when fully populated.

Power is also a possibility, although usually this show up on other things,
like, for instance, the monitor display being "wavy".
