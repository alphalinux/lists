if its compiled straight into the kernel typing the command 'dmesg | more'
will give you what the kernel found when it booted... this includes all the
drivers that are compiled straight into the kernel as well as any
pre-defined module loads (ie in /etc/conf.modules a line like 'alias eth0
3c59x')

lsmod will show you what modules are loaded into the kernel. this will also
tell you if it's running as a module.
