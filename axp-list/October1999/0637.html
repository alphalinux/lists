Is there a favorite? They are both good desktops but I'm not sure if the
world is leaning towards one or the other.. Any thoughts? Being able to use
the KDE Manager as a web browser sure is handy for Alpha users who don't
have a Tru64 registration for Netscape. The email client is nice too.
Gnome is little less "corporate" and I enjoy it's interface more..but it is
lacking the file manage/web browser combo and "GMail"..I can't handle PINE
or MUTT (using vi for email drives me nuts).
