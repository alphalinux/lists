On 10-Oct-99 pbrown@btechnet.com wrote:
> Hello All,
>
>
> Only Penguin Computing, VA Linux, and HP seem to have a good selection for
> what I'm looking for. Does anyone else have any other sites to check out
> before I make my decision?

Check out http://www.dcginc.com or Compaq itself for info on the DS-10 Alpha
machine. Can't be beat for the price (IMHO).

If the price is too high, check out http://www.thelinuxstore.com for info on
164LX-based Alpha server solutions.

Good prices on "slightly old" Sun machines can be found at
http://www.compsyscon.com. These can run Linux quite well (as well as Slowaris
which is now reasonably-priced).

CobaltMicro has some MIPS solutions that are quite well optimized for server
use. A 1.5K$ machine can totally saturate a T1.

Rebel.com has some very good Linux-based servers, in cute packages.

Power Mac machines can now run the PPC port of Linux quite well, G3 and (presu
mably) G4 machines just scream under Linux.

Don't overlook cheap PCs. A $400 E-Machine running Linux can do quite well.

Not that other vendors don't have equal or beter machines, these are just the
ones I'm aware of.

I'd do more research if I were you.

Not sure all this is topical though :-(
