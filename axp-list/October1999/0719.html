At 05:10 PM 10/26/1999 , you wrote:

To set this up properly, I think the directory should look like this...

drwxrwxrwx 2 root root 1024 Mar 11 1999 tftpboot/

So as root,

#cd /
#mkdir /tftpboot
#chmod a+rwx /tftpboot

It needs to be world writable & executable. Also, make sure you uncomment
tftp in you ined.conf, then

[root@pen /]# killall -HUP inetd
[root@pen /]#
