On Sun, Oct 24, 1999 at 12:26:58PM +0000, Mark L. Holloway wrote:
> Any hints, tip, tutorials on getting my Alpha 500a to support sound? It's an
> ESS1868 sound card that comes with the machine. I tried Sound Config tool in
> /usr/sbin but it didn't find the card. If I select ES1868 manually and "test"
> the sample sound file, I get a:
>
> sh: insmod: command not found
> sound: No such file or directory
> sound: No such file or directory (yes, it says it twice)
>
> However, if I go to /sbin the insmod file exists. Any thoughts? I appreciate
> the help..

/sbin is not in you PATH. You must have done 'su' to become root.
If you'd done 'su -', it would have been in your path.
