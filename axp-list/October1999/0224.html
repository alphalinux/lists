i don't use the ide-scsi emulation, but since i don't have an ide cdrom
i don't know if you really need them. maybe somebody else can answer
that question.

one other thing, somebody reported, was a problem with cdroms and an
incorrect ide bus speed parameter - have you set the idebus=33
kernel parameter, if not make sure to set it. cause otherwise the
timing parameters for the ide chip may cause problems.
