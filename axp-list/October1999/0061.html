Yes.
I used the RH 6.0 release together with the kernels that Sebastian
Moeller announced on this list (available at
ftp://ftp.lur.rwth-aachen.de/pub/jensen/) After downloading those, I
first "downgraded" my firmware to 1.7 with the files available from
digital, then I used the ARC utilities at ftp.digital.com to partition
my disk (with minlabel) and copy the boot image to the beginning of the
disk (using copyboot).
This allowed me to boot into the install program using:
boot -fl "i" dka0
and then repsonding to the aboot prompt:
0/2.2.12.gz root=/dev/fd0 load_ramdisk=1

After installing the RH distribution I copied the System.map and the
kernel from S. Moeller over the ones installed by RH. I also changed all
the init.d scripts to not use the "daemon" function to startup the
services, since this function seems to be broken.
Given all that I was able to make my Jensen (a Dec 2000 Model 300) run
Linux, except for the graphics stuff, since the graphics adapter on this
machine doesn't seem to be supported by Linux.

If you need more detailed instruction, please contact me.
