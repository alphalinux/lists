You did...so just edit your main partition and set the
mount point to /. You don't need to jump into fdisk/druid
since minlabel has already done the partitioning. Of course
you need to write out the partition info too in minlabel.
I think "w" will do it, going from memory.

I'd also make sure your boot partition is set to ext2. I
don't think it matters much what the partition type is for
the boot partition, but why take chances.

P.S. When you start for the first time, pass the "single"
flag to the kernel to start in single user mode. If you
want to use multiuser mode, see my hack in the post I made
today.

/Michael
