Jonas Gustavsson wrote:

> As we´re starting to move some of our alphas over to Linux/Tru64(from NT)
> it would be realy nice to have a Cross Platform IDE(Intergrated Development
> Enviroment), and I suppose I´m not the only one.
> It would be especialy usefull if the IDE was based on either GCC or the GEM
> stuff from compaq.
>
> Any ideas??

Hmm, there's Code Warrior, but that's only for Intel. Maybe Cygnus Code
Fusion, but that's for Linux and there's no indication it runs on other (OS)
platforms. On the other hand, combined with Cygwin libraries/tools, it should
be possible to build Code Fusion for NT. I would think they would have thought
of that and done it already.

For that matter, using Cygwin it should be possible to compile kIDE (KDE) or
glade (GNOME) on NT as well.

That's all I can think of.

FWIW, you really should not start a new thread on a list by replying to another
message. For those of us with threaded email clients, it sticks your post in
the middle of the other thread, so we never see it unless we open that thread,
even if you use a new subject. Just retype the email address, or set an alias
so you don't have to type the whole thing.
