On Sun, 31 Oct 1999, Larry Snyder wrote:

> Did you include romfs support? Reportedly, RH6 reqires it.

Dude! You're a genius. I am now running the Red Hat 6.0/Alpha install at
last. I'm surprised that I don't see any mention of the RH6 install
ramdisk being in ROM filesystem format in any of the documentation, but
it's true.

In the end, to start the RH 6.0 install on my Alpha with the 2940U2W,
I had to:

  - Make the MILO, generic.img, and ramdisk.img disks.
  - Build a 2.2.12 kernel on an Alpha, including support for:
      -- ramdisks
      -- The onboard NCR SCSI controller
      -- Adaptec AIC7xxxx controllers
      -- the ROM filesystem
  - gzip the resulting vmlinux, mount the generic.img floppy, and
    replace the old vmlinux.gz with the new one.
  - Proceed with the installation normally.

Thanks again for all your help!
