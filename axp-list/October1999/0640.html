> Gnome is little less "corporate" and I enjoy it's interface more..but it is
> lacking the file manage/web browser combo and "GMail"

"GMail" is called "Balsa" and on the first blush looks ok. Apart of
that nothing prevents you from using other mail clients from Gnome;
this includes kmail. If you want to go hog wild do that with
"Mahogany". :-) 'kfm' as a web browser also works from Gnome just dandy.

Personally I like Gnome better as well. It has more flexibility and
potential. KDE is more of a straightjacket. OTOH if you have space
on a disk you can install both Gnome and KDE and KDE applications will
show up on Gnome menus and also the other way around. :-) There are
no conflicts, although there is some duplication of functionality.
OTOH you may install only an "interesting" subset of KDE and/or Gnome.
Don't get into a rut.

The current version of gdm is IMO a preferred way to have a graphic
login screen - especially if you want an easy way to switch between
a number of desktops. I am talking here about gdm-2.0beta4 and
definitely not about gdm-1.0.xxx.
