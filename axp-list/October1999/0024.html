> you can have multiple OS with SRM but they have to reside on different

I did not try different OSes but 'aboot' allows for multiple boot images
which live on different partitions (this I tried :-) and a bootblock is
just a bootblock. So it is likely possible to lift that limitation not
only for different Linux versions/installations. You can boot from a
floppy for experiments before commiting anything to a hard drive.
