On Fri, 22 Oct 1999, Eric Jones wrote:

> I have a java program. I type javac program.java to generate a class file.
> I type java program.class and I get
>
> Can't find program.class

Try `java program`, not `java program.class`.

The ".class" file is the representation of the program; when you run it,
you're running the program, so you `java program` rather than
`java program.class`. Don't ask me, I only use it. :)

This is a Java thing; it's not an Alpha-related issue at all.
