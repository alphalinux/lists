--- Helge Kreutzmann wrote:
> I am not entirly sure about the SCSI specs. If I add
 an (internal/external) SCSI 1/Fast SCSI 2 devices on
an UW chain, will all devices run on SCSI 1/Fast SCSI
2 ? Maybe a FAQ on this somewhere ?<

The Linux Documentation project mirrored everywhere on
the 'net, as well as /usr/doc/HOWTO on some Linux
distributions has numerous HOWTOs, including one on
SCSI. Unless you set the speed of all the drives down
to the speed of the tape, you should not slow down the
entire bus. The only time it is slow is when
something is communcating with the drive, a drive or
the host adaptor.

I can be a bit wrong here, see the HOWTO, results may
vary somewhat by machine, adapter, software and
version level.
