Can I buy it in stores & what are the SPEC95 numbers?

The latest submitted AMD results are for the 650 Chip:

SPECInt95 of 29.4
(http://www.spec.org/osg/cpu95/results/res99q3/cpu95-19990914-03866.html)

SPECFp95 of 22.6
(http://www.spec.org/osg/cpu95/results/res99q3/cpu95-19990914-03867.html)

That latest submitted Alpha results are for the 700Mhz EV67

SPECInt95 of 39.1
(http://www.spec.org/osg/cpu95/results/res99q3/cpu95-19990913-03850.html)

SPECFp95 of 68.1
(http://www.spec.org/osg/cpu95/results/res99q3/cpu95-19990830-03820.html)

If everything scaled properly, (which it almost NEVER does), AMDs clockspeed
would have to be 3x that of the Alpha to beat it at FP. The int race would
be closer, but once again, assuming perfect scaling, the AMD SPECint95 score
would be an estimated (900/700)*29.4 = 37.8. The alpha is still in the lead.

;-)

BTW, API demoed a 1 GHZ room-temperature alpha running Linux, so if MHZ is
your game, alpha is still in the lead there too.
