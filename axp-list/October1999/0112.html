Many Tulip based cards have problems with autosensing speed... I recently
had trouble with a Kingston board using a Tulip in my K6-III system (and
Larry Sendlosky reported on various other trouble. As you can see, this
has everything to do with a Tulip card, and nothing specific to Alpha.

Suggestions:

1) get and install the latest tulip driver.
http://cesdis.gsfc.nasa.gov/linux/drivers/tulip.html

2) using the information there, you can
set an option in /etc/conf.modules to set the Tulip to the right speed.
There is a table of speed/media types to option needed

So, for example, my conf.modules contains:

        alias eth0 tulip
        options tulip options=3 debug=1

Which sets the board to 100mbit twisted pair half duplex. Hope this helps.
