I know a lot of people on this list hate Adaptec
controllers... so one might suggest getting a
controller that is known to work more smoothly.

However, I've been using Adaptec and have never had a
problem. But this might be because I always disable
the SCSI BIOS from doing any auto-detecting (for
termination, sync rate). You might want to try that,
and manually set all the settings. Although it seems
more of a hassle, at least you know for sure it's not
some auto-setting that isn't doing it's job.

D.
