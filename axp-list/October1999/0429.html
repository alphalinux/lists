Stuart Midgley wrote:
> How well does RedHat 6.0 handle a raid setup? I have a 164UX machine
> running RH6.0 with 2*9gig hard disks. I do a lot of computational
> physics stuff (with lots of data files) and I would like to join my two
> disk up to have just one virtual drive. That way, I do not have to
> bugger around with putting some different files on different disks.
>
> It means I have to reconfigure my machine, so I would like to get it right the first time.

Hi,

        I've done it with a RAID card (a Mylex), all that configuration stuff
is done via its integrated interface, all you yhave to do under Linux is
to add the kernel support, the name of the disks (looking
like/dev/rd/c0d0p1) at startup (I've moved a RAIDless system to a RAID
system qquite easily, but be careful and backup before ;-).

        For Mylex I know, under Linux you can give commands (I use it as RAID
1) asking to make online/offline a disk, do consistancy check, rebuild
an unit... see Documentation/README.DAC960 for the details
