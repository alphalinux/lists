Hi

I purchased some memory for my 500au workstation about a year ago her
ein Australia. The memory that I got was:

        64Mb ECC 100MHz(10ns) Hyundai SDRAM DIMM (with EEPROM)

        The RAM is made from 8x "HY57V658020 TC-10 9803A Korea"
                gold plated pins

        The Alpha 500au motherboard syncs at 83MHz

I have some documentation for the 500au on my server. You can get it
at:

http://canoepolo.pd.uwa.edu.au/docs/dpws_aau/Service/Aaudpwssg.htm

I found this on Digitals web site quite some time ago.

I hope it helps.
