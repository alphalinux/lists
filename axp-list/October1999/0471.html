"Jemiolo, John" wrote:

> The AS4100 is known as a Rawhide and runs linux just fine. The AS2100 is a
> different story, as it is a PCI/EISA hybrid Arch, and there has been little
> work done on getting it running (lack of hardware for debugging).

Is it completely not worth the time, or is it just "less than optimally
supported"? I'm about to move over to a group where I'll have access to a
bunch of these that I was hoping to get Linux up on... I do know, however,
that AS2000s and AS2100s do work, but specifically the AS2011As do _not_ -
totally different interrupt routing if I recall what Jay said correctly (which
lately I haven't...)

>
> On the AS4100, make sure you have a (linux) supported video card.
>

But you do only need the video card for install (at least with RH6.0 - haven't
tried the others). After the install, you can set up your system as
documented in Documentation/serial-console.txt and make sure you have a
mingetty running for /dev/ttyS?. That's how I run all of my test systems,
anyway.
