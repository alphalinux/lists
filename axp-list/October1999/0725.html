Mark Holloway wrote:
>
> Is there anywhere to turn off the feature that changes between desktops
> (1,2,3,4) just because my cursor moves past the edge of the screen?

Yes.

michal@ellpspace.math.ualberta.ca said:
> Use a different window manager. Nobody said that this has to be
> 'enlightement'.

That's not very helpful:-)

In the root window, button 2 brings up the enlightenment menu. Select
Enlightenment configuration. In the configuration editor, select
desktops and you will be able to arrange things how you would like.
You can even turn of edge flip and virtual screens completely.
