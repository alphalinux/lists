>The bigger problem I have is that it seems to be an intermittent
>problem. My uptime at the moment it 6 days. So far it seems to be
>fine. It seems like hangs are sort of clustered, i.e. I get several,
>then none for a while. I guess I could try building a modular kernel
>again and see if it's still reproducible. The only thing is that I'm

A few weeks ago, similar problems have been reported in this list.
No my uptime is 15 days, but that's probably the best I ever got since
I bought the LX164 in July 98. Usually the system hangs because the
clock stops. Give it a try. One it hangs, try to login from another
machine. It does not always work, sometimes a ping does not even
work anymore. But sometimes it does. The check if the clock is still
running. I bet it doesn't.
I never could figure out what the problem is (the motherboard got
exchanged by Transtec-Service warranty and nothing changed).
One thing I recognized: Clock stops usually exactely one minute after
the hour.
