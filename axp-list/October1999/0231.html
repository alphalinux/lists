"Vanco, Donald" wrote:
> I'd like to pull the source & compile under 2.2.12, but I'm not sure
> which RPMs comprise the source/headers - can someone send me a pointer
> (ftp://ftp.alphalinux.org/kernel/v2.2/ appears to have kernel images in bz2
> format - but what handles that compression?)

If you're looking to recompile, it's probably best to stay away from
RPM's. For the kernel source, you can get it in either .gz or .bz2
compressed formats. .bz2 is smaller (and transfers faster), but you
need a bzip2 uncompressor for it, which are still relatively uncommon.

Pretty much every Linux machine has gunzip, so go ahead and get the
.gz'd version. Same thing. :-)
 
> Also - what's the appropriate way to shut down this system? It

> box.

I'm a fan of "halt" or "reboot"... the "shutdown" command will also help
(see the man page).
