I'm sorry for my humor, but for most things of "source", one merely has
to do
1) get the rpm
2) rpm -i -vv xxx.rpm
3) go the redhat/SPEC directory
4) rpm -bc xxx.spec

This should build it for u
gat
ps xxx = the name of the rpm file that has what u need.
ALSO, if the RPM is from a newer release of RH ( like 6.0 ) you may have
to edit the spec file slightly

if there is no rpm, it is just as simple ( usually )
1) untar the file
2) cd to the base of the files
3) ./configure
4) make
5) AND if you are brave, make install.

gat
