There is an updated ramdisk in the errata section on RedHat's
web page. Try it. Or you could also try using the CD as the root
filesystem eg:

>>> b dva0 -fi generic.gz -fl root=/dev/scd0

if the cd is scsi, try /dev/hda if it's atapi. If you
have problems with it not being able to mount root from cd
verify the device name via the kernel message type out when
the cd is "discovered".
