Jamesvi@asu.edu (Jamesvi@asu.edu) wrote:

> I have tried the 'make zImage' command to compile a kernel on my alpha.
> However, it always says 'no rule to make target stop.' This is after I have
> done make dep and make clean. Am I missing something (obviously). Is the
> command different on Alphas than x86 machines to compile a compressed kernel?
> I realize that this is really simple. Sorry for the bother.

Here is what you want to be doing:
make config
make dep
make clean
make boot
make modules
make modules_install

Have fun!
