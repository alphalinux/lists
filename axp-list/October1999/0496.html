The host id cannot be zero or all 1s. For a class C address, this means
that x.x.x.0 or x.x.x.255 is not valid as a host id. For a class B address,
this would mean that x.x.0.0 and x.x.255.255 would be invalid. However, on
a class B network with a standard subnet mask, this would mean that x.x.0.20
and x.x.255.0 should both be valid addresses.
The reason is that an all zero host id refers to the "this host" address,
and all 1s refers to the broadcast address.
