No graphics, thats your problem, build a 2.2.* kernel with serial console
support, then change /dev/console as described in
/usr/src/linux/documentation/serial_console.txt

After the mod's, if a graphics card is present it will be used as the
console device, if it is not present, the serial port will be used as
console "automatically".

Do not forget to start a getty on /dev/ttyS0

I use this all the time.

JJ
