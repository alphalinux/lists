At 08:47 AM 10/5/1999 +0200, you wrote:
>I'm running RH6 on my DS-10 and I use the onboard network cards with the
>tulip (module) of kernel 2.2.5.-22 (from RH).
>SOmetimes (mostly on the weekend, when there is 'no' traffic on the net)
>the card changes to some different media (eg. form 10 to 100 Mbit) and
>in this case the machine is working.
>I'm trying to fix this as described in the kernel docu, but with no
>success.
>Has anyone the same problem?? Or an SOlution?
>This is an running mailserver, so I need to fix this asap. For the
>moment I'm pinging the machine every 15 min. once to keep it online.

In most unixes you can set the media type in ifconfig, I am not sure if
it's that way in linux. Yet I know you can set the media type with insmod
if you use the driver as a module. Read the soruce to the tulip.c driver
for more info. I'm pretty sure Becker put in nice comments.
