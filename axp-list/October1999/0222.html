Greetings problem-solvers!

I'm confused, I'm used to RH5.2 (admittedly on x86), and when I export /,
I export all the mounted partitions below that too as one virtual file
system. However, under RH6.0 (alpha, original CD no patches) I export only
/ and no submounts.
I've even resorted to buying the O'Rielly book on NIS & NFS, and I can't
see an explicit remark about this issue...

Personally I go with RH5.2, I prefer the virtual file system view.
(so, e.g., if a new disk is added and /usr/local is moved onto a new
partition, the NFS clients shouldn't need to change their configuration)

(this probably isn't architecture specific, but my alpha is my 6.0, and
my PC is my 5.2, so it could be)
