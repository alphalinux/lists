I'm having trouble using mpeg_encode the Berkeley MPEG-1 Video Encoder on
an LX164 with running 2.2.10. I've tried using the binaries from
mpeg_encode-1.5b-2.alpha.rpm and mpeg_encode-1.5b-4.alpha.rpm and I get
the same error:

mpeg_encode: main.c:568: CompileTests: Assertion `sizeof(uint32) == 4'
failed.
Abort

Seems like a 64 bit issue but I had the impression that there was a
version out there that was clean on the alpha. There is even an osf
binary on the berkley site but it gives me the same error. I've also
tried recompiling from the mpeg_encode-1.5b-2.src.rpm. Compliation was
successful but I got the same error from the resulting binary. I just
wanted to check to see if anyone was using this program before I try to go
into the source to find the exact problem.
