NT must be booted from ARC/Alphabios.
OSF and VMS must boot from SRM.
Linux can boot from either.

Also note that SRM doesn't understand partitions.

On an AlphaStation you can have both SRM and ARC/Alphabios loaded
simultaneously. There are other Alphas which can only have one at a time,
and some that can only have ARC/Alphabios, period.

Also note that the recent termination of NT on Alpha also means that any
further development of ARC/Alphabios has been stopped. This is *serious*,
since no other parties than Compaq & Microsoft have the sources... Since
Compaq has relegated Alpha to the high-end niche, only SRM will continue to
be developed, and only on a very limited set of high-end Compaq-brand
AlphaServers...

The Alpha community is in dire need of an Open Source firmware loader.
