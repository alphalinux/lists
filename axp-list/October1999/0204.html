One thing I haven't found one the web is a detailed description of
kernel bugs and which patches should be applied to fix them. Maybe
I'm just not looking in the right place, but short of collating a lot
of stuff from various email archives, I haven't see this info
available online. Since the 2.2 series has been nightmarish (IMHO) on
the alpha, I think a web page summarizing kernel problems and
solutions would be quite helpful.
