Wessels, Hans wrote:
>
> An other way is:
> void foo(char *p)
> {
> int alignment;
> alignment = ((int) p) &0x0F; /* Get a warning */
> ...
> }
>
> Anyone how to do this without getting a warning?

int foo(char *p)
{
    int alignment;
    long atmp;
    atmp = ((long) p) &0x0FL;
    return (alignment = atmp);
}

Actually a generated assembler will be much simpler than
you can expect. See for yourself. :-)
