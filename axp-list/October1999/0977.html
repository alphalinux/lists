This information was designed for the i386 platform, but it may also work on
Alpha...

Q: [ x:serve ] Diamond SpeedStar A50, or SIS 6326 based video cards.

    A: Edit the /etc/X11/XF86Config file and go to the section:

    # Device configured by Xconfigurator

    Insert under the "BoardName" line:
    option "no_bitblt"
    option "no_imageblt"
    option "sw_cursor"

    Save the file and try startx.
    Another line that can be added in the same place is:

    option "no_accel"

    Added July 21st, 1999 by ckloiber - Solution ID 32 - (edit)

Your mileage may vary.
