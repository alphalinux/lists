Further thoughts on "vaporware" clockspeeds on Alpha:
Fastest shipping chips readily available are 667MHz.
750MHz is easily done but:
    with current chipsets a chip at 750MHz has a 75MHz memory bus clock.
    To get a reasonable gain in performance and overcome this memory sped
issue:
            a bigger cache is needed 8 MB ideally.
            There is a problem with 8MB cache running reliably in the
current CPU packages
             Cache on SlotB runs at CPU clockrate, I believe.

In simple terms: If API or Compaq shipped a 750 with the same cache and
other design details as the 667, it would not be much ( if any) faster,
depending on which benchmark you ran.
We are now being told that nothing faster than 667 will ship in the near
future.
