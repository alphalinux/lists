On Mon, Jun 25, 2001 at 05:12:05PM -0400, Eric Christy wrote:
>
> Has anyone managed to successfully use the DRI with a pci Radeon on
> Alpha? If so were there any caveats to be aware of?
>
> Recently Jay Estabrook posted that he tested this card in
> EV56/PYXIS with good results but I have been unable to get DRI to
> use direct rendering.

OK, some of the things to remember about getting this code to work:

1. you must be running a 2.4.x kernel; I've been staying with 2.4.1
   since it was (reasonably) stable and clean, and I was more involved
   in the graphics side of things than the kernel side.

2. you must build the needed DRM kernel module(s) with the sources in
   the X tree against the 2.4.x kernel that you are running

I only have experience building the kernel, modules, and the Xserver
under RH 7.0-plus-updates, especially those to GCC and GLIBC.

Could you post the /var/log/XFree86.0.log for a failed load of DRI?
