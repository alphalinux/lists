I've got an lx164 with alphabios 5.60-3

I'd like to upgrade to the latest firmware release so I can use SRM and
I'm having some problems.

I downloaded relnote-lx164srm.rom-v1.txt and followed the instructions
for Updating to SRM Console from AlphaBIOS on AlphaPC 164LX/SX

I copied the file LX164SRM.ROM to a fat formated disk and from the
AlphaBIOS setup menu, selected "AlphaBIOS Update"

All I got was a pop-up that said to insert a disk with firmware update
utility.

So

I went to the web site
http://gatekeeper.research.compaq.com/pub/DEC/Alpha/firmware/

and got the firmware image: lx164_v5_8.exe
I copied it to a blank FAT floppy as: fwupdate.exe
I put the floppy in, and from the AlphaBIOS menu, selected
      "Update AlphaBIOS".

After some time the screen turned blue and I go the following messages:
eb.ea.e9.e8.e7.Invalid mode
searching dva:0.0.0.0.0 for option firmware files
"Enter device name .... or hit return"

I hit return and got a prompt
UPD>

UPD>list

device current rev filename update rev
fsb 2.2 fsb-fw 3.1
fw 5.60-3 nt-fw missing file

UPD>update

"Error: missing file nt-fw"

--------------------

Repeating the above process I now see

UPD>list

device current rev filename update rev
fsb 3.1 fsb-fw 3.1
fw 5.60-3 nt-fw missing file

--------------------

Help!

thanks,
Jonathan
