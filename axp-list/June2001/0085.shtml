Hi,
Did you see Kurt's mail??

He told about the kernel thing! That some alphas require a kernel
compiled for that specific processor. I have experience with alpha
au600 and xp1000.

I don't want to be rude or unpolite with the guy who first replied,
(the guy just said that there's a command to fix that "make xconfig";
this is to open the main menu to compile a kernel).
Kurt is someone that you can really trust (he helped me a few times).

So if you have some similar machine try to compile the kernel on that
machine. Or even simpler try to boot with a floppy disk, than see how
the machine behaves. If it works use the kernel on the floppy disk...
but try to compile a more specific kernel.

About the pci card there's no problem at all it simple don't work, but
nothing that should halt the machine, this i know one for sure because
i had one of those cards in an au600!

There's a kernel HOWTO that you should read before compiling it for the
first time. The worst thing that may happen is that if you do something
wrong you want be able to boot the machine :-) but this problem you already
have!
