Quote from Duke of URL

"Updated Chipset Support What's a new release of XFree86 without support
for some new chipsets? 4.1.0 features support for not only new cards, but
also strengthens the support for some cards. It also seems that a lot of
time, the Alpha platform gets ignored along with the *BSDs, but this
version changes that. With 4.1.0, Linux/PPC finally has DRI support,
FreeBSD has i810/i815 support, and Alpha/Linux finally has jumped on the
bandwagon with support for the ATI Radeon."
