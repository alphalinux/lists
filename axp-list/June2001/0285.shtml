I just want to understand some of this Compaq/Intel crap.

1) Compaq is selling all of their Alpha tech to Intel and then promising to
use Intel's future 64bit designs (whatever that may be) for Compaq hardware.

My problems with this are:
Is the govt going to allow this?

What happens to purely Compaq products? (SRM, OpenVMS, True64, Compaq's
compilers, etc)

How freaking stupid is Compaq's management? Intel can't even get a first
generation ia-64 product out the door. Does anyone really think they will
have a next generation processor available by 2004. Talk about being sold on
marketing.

2) How much of future Alpha design is inside of API now?

API seems to have been forced to be an all Linux hardware company with this
announcement. Which makes their work for gcc 3.0 improvements very
important. Personally I'm only buying Linux stuff, so Compaq leaving isn't
that big of a deal. UNLESS this really going to affect future design
improvements by API.
