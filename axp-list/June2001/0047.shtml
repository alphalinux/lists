On Thursday, 7 Jun, Jonathan Dubois wrote:
> I'm getting a signal 11 from inside malloc -- Is this a pretty clear
> indicator that I have some bad RAM?

I think it's much more likely to be an indicator of a mess-up with memory
allocation routines prior to the offending malloc. Something like freeing
memory that was never allocated or already freed. I heard that electric
fence could detect such problems.
