OK. here are some links.
First (114MB!!):
ftp://www.fyma.ucl.ac.be/pub/ljacques/XFree_alpha/XFree_4.0.3_axp_patched.tar.gz
This archive contains:
1°) the two patchings of the XFree sources (XF4.0.2->XF4.0.3, and the axp
patch);
2°) the compilation on my DEC PWau433 / RedHat 7.0 (glibc 2.2).
I was not able to do rpm files because I did'nt find any good spec files. I
don't think that a 'make install' can work on your computer (perhaps it's
even dangerous if you have to removed all that after (check if it exist a
'make uninstall' ?)

Second, you can obtain only the sources (~50MB) at
ftp://www.fyma.ucl.ac.be/pub/ljacques/XFree_alpha/XFree_4.0.3_axp_patched.tar.gz
but you have to compile again all the stuff (one night of compilation).

I hope it helps you.
