> To avoid of typing again and again the command 'passive' at each ftp
> connection, you can alternatively add these lines to the file $HOME/.netrc
>
> # BEGIN OF $HOME/.netrc
> default
> macdef init
> passive
>
> # END
>
> Be careful to put an empty line at the end. This terminates the macro init
> which is called at each connection.
