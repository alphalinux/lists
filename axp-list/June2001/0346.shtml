On Wednesday 27 June 2001 08:52 am, you wrote:
> So, does that mean I shouldn't install it on my LX's and SX? I finished
> burning the ISOs to CD-R just an hour before they pulled it.

I got it downloaded just in time as well. Actually my download of disk1 was
messed up and I found that download.sourceforge.net was running an rsync
server with redhat as a module. I was able to rsync my broken iso from
there. So instead of several hours to redownload it only took about 15
minutes to repair the file.

Anyway, I've installed it on an old as/200 box. I haven't seen any real
problems so far. Interesting features: kernel 2.4.3, XFree 4.0.3, kde 2.1.1
with 2.1.2 updates for kdelibs, and several other packages that the x86
version doesn't have (I think x86 updates are available though).

My problems so far: arts just doesn't cut it on a 166MHz box, kpanelapplet
crashes (which I wasn't having with RH 6.2), and that seems to be it. Note:
I don't use Gnome so I don't know how those packages work.
