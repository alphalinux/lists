Please refer to my prior posting to this list re: TGA/TGA2 "gotchas"
with 7.1. If you have a problem that is NOT solved by these workarounds
and suggestions, PLEASE post it!!

Note that the ELSA Gloria Synergy is Permedia-2 which has no 3D support.
Only the PCI 3DFx cards such as Voodoo Banshee and Voodoo3 2000/3000 are
supported for 3D under 4.0.3. The latest release, 4.1.0, adds 3D support
for PCI Rage128 and Radeon cards.
