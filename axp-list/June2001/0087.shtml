When I recompiled my AS800 I found a paper on RH site before trying. But I
cannot find it anymore. It specifically told, step for step, how to configure
with "make xconfig" and how to recompile with "makes".

So if anyone knows the link to this doc, it would be appreciated. I searched
the RH6.2 CD as well, allso with no success.
