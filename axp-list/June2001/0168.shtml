ok here's a slick trick that I've used that usually works...

put fwupdate.exe and LX164SRM.ROM on a floppy

*rename* the LX164SRM.ROM to LX164NT.ROM

there should be an option in AlphaBIOS to "update alphabios" or something to
that effect. Use that.

it should prompt you that it's replacing AlphaBIOS w/ SRM and if you want to
continue. (you obviously want to continue).

Power off the machine and when you power it back on you should have SRM.

Kurt
