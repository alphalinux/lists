I've tried everything I know how to try and am hoping someone can help
me out with installing RH7.0 on my Xl366.

Any of the supplied MILOs I found wouldn't boot (Redhat, SuSe,
Jumpstart). They all hang on the PALcode message. I've tried two
different video cards: FireGL 1000 Pro and a Matrox Millenium (came
with the system from DEC). I found an older version of milo built
against kernel 2.0.17 that seems to work fine. I have not been able
to get any ramdisk to work. It is like the kernel totally ignores the
arguments, although root=... seems to be passed ok. I have been able
to start a RedHat 7.0 install completely booted from floppy and then
FTP installed. If I try to install off the local CDROM, it looks like
the video mode gets switched and the system hangs with the video bios
bootup message on the screen. I can install in text mode by going to
install via FTP, but when it comes to configuring X the same thing
happens. After a reboot, I can boot into a shell and login. I can't
mount any MSDOS device (like /dev/floppy) and can't load the module
into the kernel.

Any suggestions?

Thanks for any help.
