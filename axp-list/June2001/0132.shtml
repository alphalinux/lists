On Thu, Jun 14, 2001 at 11:15:01PM -0500, Arkadiy Chapkis - Arc wrote:
>
> This is all nice. Is there a support for AGP and GeForce cards on
> AlphaLinux? If not - are there any plans to do those in the
> forseeable future?

If there are GeForce PCI cards, they should be supported so long as:

1. the console BIOS emulation can execute their code
2. you are running them in either an EV56/PYXIS or EV6 based machine.

If you can't get it by the BIOS emulation in the console, you are most
likely out of luck. If you don't have one of those class machines, then
the server will not work, as it depends on DENSE access and will NOT
work on machines that require SPARSE.

As for AGP, there's only the API UP1000/1100 and new UP1500 that are
Alphas with AGP (so far). AGP cards will work in there in PCI mode,
but AFAIK, none have been able to successfully activate the AGP GART
support required for the 3D features of cards like G200/400/450 and
R128/Radeon. I know, I've tried, and I have patches that almost work,
but there's some pieces missing WRT correct setup of the AMD Irongate
chipset that prevent it from working.
