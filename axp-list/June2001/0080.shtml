My experiences on my XL366 with Matrox Millenium II .

Mine has Three partitions

1. /dev/sda1 DOS boot partition where milo and linload.exe will be placed after install to facilitate booting from hard disk.

2. /dev/sda2       /   partition

3. /dev/sda3          swap

MILO: I have found using the milo image from Redhat6.2 works, RH7.0 milos don't work similar problem to yours.

You need to use the kernel and ramdisk floppies...RH7 doesn't support booting from the CD with Alphabios.

Kernel and Ramdisk from RH7.0 work on mine. You have to use these else it will not recognise the RH7 CD rom in the drive.

 I assume you are using

MILO> boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1

I remember starting out also having load_prompt = 1 and for some reason it wouldn't boot the kernel disk properly.

This gets me to the install no problem or you can put in "text" at the end for text mode. Can mount the CDROM drive too.

Do you have the latest formware for the XL366 think its about 5.69 from memory?

Have you had any version of linux installed before?
