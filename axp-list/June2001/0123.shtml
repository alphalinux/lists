On Friday 15 June 2001 08:26, you wrote:
> I try to install XFree86 4.1.0 from Rawhide (on my RH 7.0) and it seems
> that a library is missing from XFree86-libs : libXIE.so.6 which is required
> by Mozilla (at last 0.9).

I noticed that XFree 4.1 was in rawhide so I downloaded it to try and build
on a RedHat 6.2 system. Anyway, from the XFree.spec file:

<snip>
# Disable obsolete XFree86 stuff that is not used or actively developed
# in 10+ years. Will be disabled for all test builds, to see if it causes
# any problems. Our 4.1.0 RPM's will not support PEX or XIE for sure.
%define BuildXIE 0
%define BuildPEX 0
</snip>

So I would suppose that libXIE isn't going to be on RedHat distributions from
now on.
