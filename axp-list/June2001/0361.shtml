I believe the deal was:
Compaq licensed some Alpha technology ( i.e. the EV6 bus ) to AMD. In
return, everyone was hoping AMD would create Slot "B" motherboards that
would be able to accept a 21264 or an Athlon. What they did, was create
relatively lowcost chipsets that interface to the EV6 bus, primarily used on
Athlon/Duron MB's, now also on the UP1500.
