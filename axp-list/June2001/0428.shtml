On Wed, Jun 20, 2001 at 06:09:50PM +0200, Peter Mittermayer wrote:
>
> Is a refresh rate > 70 Hz possible with one of these cards on alpha
> linux using a resolution of 1280x1024 with 24bit depth? I tried a ATI
> Xpert98 which only gives 60 Hz ...

I'm pretty sure greater 70 Hz should be possible on either, but consider:

1. G200 PCI would not be 3D accelerated (that requires AGP, which LX is not).

2. Voodoo3 3000 PCI would have 3D acceleration, but only at 16-bit depth.

3. XFree86 4.1.0 has *much* improved 3DFx card 3D acceleration, and it works
   fine on the LX... :-)
