I discovered the following problem while trying the Compaq C++ compiler

their provided binutils. But i have already installed binutils 2.9.5 so
I thought everthing should be ok.

[maxl@extreme bin]# ld -v

ld --help
 ...
  --style {gnu,lucid,arm,hp,edg,gnat,compaq}
 ...

But i get the following error

/usr/bin/ld -L/usr/lib/compaq/cxx-6.3.6.9/alpha-linux/lib '-L/usr/local/l
