On 23-Apr-00 Rob Williams wrote:
> Ok, at long last, I was able to boot from the floppy!!
> Thank you!

> I logged in as root. Now what do I have to do to get X
> to start?

If you've already configured X (through the installer), you can type (don't
laugh) "startx".

If you haven't yet configured X, run "Xconfigurator" first and pray that you
have a supported video card. ;)
