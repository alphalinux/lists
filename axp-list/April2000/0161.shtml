On Mon, 10 Apr 2000, B. James Phillippe wrote:

> Has anybody had any luck with the latest mozilla milestone binary (M14) on
> a 21064? For me, mozilla-bin just spins at startup and never draws
> anything on the display (no X client appears). I'm guessing that my
> problem is that the binary from mozilla.org is ev56 and I'm ev45. It takes
> me over a day to build mozilla from source on this little beastie (AS200
> 4/233). Perhaps someone has built an RPM or tar of an ev45-compatible
> version? I saw it running on an x86 box and it looked quite usable.

There may be a problem with the distribution itself. I recompiled
it. Try getting M14 from the following site and let me know if it works
for you:

http://www.bcpl.net/~ryan/mozilla-m14-alpha.tar.bz2

When you untar the file, it will make a directory called
mozilla/dist/bin. You should probably extract this as a non-priv. user in
your home directory.

cd into mozilla/dist/bin and issue the following command: ./mozilla

(You might have to set some environment variables, see mozilla.org for
that info).
