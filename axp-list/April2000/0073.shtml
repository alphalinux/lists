In case some of you haven't noticed, the last few versions of Licq have finally
started to run correctly on Alphas. I've just built the latest version on my
AS200 4/233, and for those of you with similarly specced Alphas who don't want
to wait a whole day to build it (and the QT 2.1.0 toolkit it requires), I've put
some RPMs of licq 0.81 and QT 2.1.0 (beta 3) up on my FTP archive.

Built and tested on a Redhat 5.x machine. Probably works with RH6, but I can'r
be 100% certain. :)
