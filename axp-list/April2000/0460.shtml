On Mon, 1 May 2000, MACKAY, Lincoln wrote:

> Hi all,
>
> Trying to install RH 6.1 (or 6.2) on a Alpha 2100A 5/300. (DEMI-GAMMA-SABLE
> ?)
> I have tried booting off a RH CD, and a floppy created from generic.img
> In both cases, I get to an SRM prompt, and put a boot command along the
> lines of:
> P00>>>boot dva0 -file vmlinux.gz -flags "root=/dev/scd0"

Create the ramdisk floppy ("images/ramdisk.img" on the cd). Boot the
boot floppy with:
>>>b dva0 -fi vmlinux.gz -fl "root=/dev/fd0 load_ramdisk=1"
Insert the ramdisk when it asks you.

or, boot the cd directly:
>>>b dka??? -fi kernels/vmlinux.gz -fl "root=/dev/scd0 load_ramdisk=1"
substitute the correct device number of your cdrom for the ??? above.

Oh, is this a supported machine? I forget.
