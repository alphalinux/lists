from Paul Jakma's email Oct 7, 1999 that I keep around to reapply every
time I reboot.
<snip>
RE: cdrecord: shmget failed.
From: "Jakma, Paul" <Paul.Jakma@compaq.com>
To: "'axp-list@redhat.com'" <axp-list@redhat.com>

max shared memory is tweakable on 2.2 via syctl. For some reason it
defaults to a ridiculously low value on Alpha. Do:

echo 16777216 > /proc/sys/kernel/shmmax

to set shmmax to something sensible like 16MB.
</snip>
