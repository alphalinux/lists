Running one here...
w/ RH 6.1
first off... you need at least kernel 2.2.14 for it to be stable.
second off.. compile 2.2.14 generic, if you don't it'll drop the keyboard /
mouse at boot
(I'm not sure if 2.2.15 fixes these)

Pair of generic PC100? NO!!!!!
you Need EDO ECC DIMMs not SDRAM DIMMs
they won't even fit in the slots the right way (believe me I tried it)

mine came w/ 64MB of RAM and no disks, but I already had SCA drives to use
with it (a 4.3 and 2x 18.2s all 10k RPM Compaqs)

I added 256MB more of Compaq Server Memmory DIMMs (I think they were for the
5000 series line)

oh you also need to add the memmory in sets of 4! (256bit data path)
(so the 64mb that came with my server takes up 4 slots.... 4x 16MB)

Good Luck and it's a EXTREMELY fast server for 400MHz =) (it beats PIII
450's for performance),
Kurt
