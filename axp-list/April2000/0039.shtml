For all those interested in playing with glide on ALpha/Linux.
Hear ye, Hear ye.

I've put up the driver and library on the alphalinux.org CVS repository.

You can pull it down using the read-only anonymous account.

$ export CVSROOT=':pserver:anonymous@www.alphalinux.org:/home/axplinux/cvsroot'
$ cvs login
(there is no password, just hit return)

$ cvs get GlideDriver

$ cvs get Glide3-64

You must make the /dev/3dfx node, and insert the driver before trying to
run any of the glide3 tests.

mknod /dev/3dfx c BLAH 0

(I can't remeber which number it wants right now and my box is at home. You'll
have to check out the source)

insmod 3dfx.o

Bug reports/fixes are welcomed.
