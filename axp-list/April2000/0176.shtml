qmail is a good server, I have it installed on a box running as a list server.
It lacks imap support so you may have to use one of several alternatives.
I liked Cyrus but it really needs to run a closed box. I have UW-imap on a
production server with PHP4 and Horde/IMP for web access. There were several
things I had to fix however. PHP's imap module had a bug in the delete
routines and I also applied the imap-maildir patch to fix the user maildir
problems. With a little tweaking and experimentation I had the beast up and
running. The only other consideration of course is the use of a secure httpd.
This however, is another challenge that you need to overcome.

UW-imap: http://www.imap.org
UW-imap maildir support: http://www.davideous.com/imap-maildir/
PHP: http://www.php.net
Horde/IMP: http://www.horde.org
