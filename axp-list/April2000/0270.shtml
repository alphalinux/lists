Luke Lonergan wrote:
>
> > Sorry if this is slightly off-topic, but a quick question:
> > all y'all Alpha users out there, what inexpensive Ethernet
> > cards have you found to work well? This is for 533MHz
> > 21164 in the 164PC motherboard, if it makes a difference.
>
> Any old Tulip works well. Netgear's cards are all fine. Also, the Intel Pro
> 100+ is turning out to be my favorite.
>
> Remember that if you are using Tulip chipsets, do _not_ use the tulip.c
> driver. Use the de4x5.c instead.
>

tulip.c works just fine as long as you use a recent
version, say around v0.90 or higher. There now two
tulip drivers in the kernels and if you use recent
HW, you usually need the new one.
