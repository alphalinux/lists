I'm using a sym8751 in my sx, but it's not an SRM box. Nice thing
about this card is that I connect a pair of HDD's via the 68-pin
interface and the 8mm tape (and sometimes the scanner) via the 50-pin
apparently without any performance hits. This card has an external
68-pin I/F that I don't use. I ran an adapter to get the narrow
bus out the back panel for the tape/scanner. The part number is
sym8751X where X is a letter(s) I don't remember.

I don't know if that card plays well w/ SRM. IIRC, another version
of it is considered supported by Tru64 (but only with the right f/w
on the card). Steve/Maurice/Jay -- can you lend /dev/clue?
