You can delete, create, and format partitions from within Alphabios.
Simply press F2 when you reach the Alphabios and when it is done
initializing it will enter the setup. I forget the name of the line item
but you'll figure out which item to select to manage the disk. Once there,
delete it and confirm it.

> If so that would probably solve my problem, can i then go and copy my cd to
> the hard drive through the alphabios or how would i go about doing that. is

I'm curious...
How are actually installing this.
You must be able to initalize the boot process in some way.

The boot string for milo if you're using floppies should be

boot fd0:/generic.gz root=/dev/fd0 load_ramdisk=1

once it's finished booting the kernel it will ask you to insert the
ramdisk and press enter. Do that and after a minute you'll be in the
RH installer.

If you have a CD you could boot right from the CD and start the install
like so, assuming you IDE hardware. If you have SCSI use sda instead of
hda etc etc.

boot hcd0:/boot/generic.gz root=/dev/hcd0

and you'll be in the graphical X install, If for some reason you
want to do a text based install simply put " text " at the end of
the line.Good luck.
