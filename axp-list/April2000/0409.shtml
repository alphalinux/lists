 tru64 Unix.
 Currently the only way this works is:
 
 - If I remove all the tru64 unix disks, then and only then the
   linux kernel boots.
   But when I put the Tru64 Unix disks it goes through all
  the initialization and
   just when it starts detecting the devices (i,e my Tru64 Unix
      disks) it craps out and gives me the dreaded "HALT CODE 5"
