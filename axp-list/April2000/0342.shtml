Hi, I'm looking for memory for an XL300, but coming up against a couple
of problems.

After a tedious process which involved special ordering memory and
discovering that despite being the type specified (64MB 70ns true parity
SIMMs) they still didn't work (beeps on startup indicate "memory
error").

What is the relation between an XL300 and a "Celebris" machine? I'm
looking at some Kingston memory part # KTV-PZX/256, which I think is the
proper part for 4 64MB SIMMs, but the Kingston web page says it's for a
"Celebris XL300". Can anyone confirm that this is the right part?

I noticed that the Kingston memory already in the machine has tin
contacts, and the new ones I tried were gold. I hadn't heard of this
making a big difference, but then I guess alphas can be more sensitive.

So if I spring for the memory, how long do these machines last
(physically)? I've heard of capacitors and other parts failing in cheap
PC motherboards, but I assume one of these DEC machines should be able
to run for years. Part of me balks at buying new "obsolete" memory, but
otoh I don't feel the need to get a new machine, and although the memory
is expensive, it's not as expensive as getting a new machine.

So I guess the question boils down to: is the Kingston part number
correct, and is there a cheaper source for memory that will work, from
people who know about alphas? I don't want to go through that cycle
(which ended up taking a total of over 3 weeks) again! XL300's seem to
be reasonably common; where are people getting the memory?

thanks,
Wes.
