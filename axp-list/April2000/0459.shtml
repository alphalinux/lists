"MACKAY, Lincoln" <lincoln.mackay@contact-energy.co.nz> writes:

> Trying to install RH 6.1 (or 6.2) on a Alpha 2100A 5/300. (DEMI-GAMMA-SABLE ?)

It's rumored that Linux doesn't actually work on the 2100A. There
isn't any support for Sable-Demi-Gamma in the relevant kernel files,
only Sable and Sable-Gamma. A comment in sys_sable.c indicates that
the 2100A's PCI configuration is different somehow.

I've been trying to figure out the same problem with someone trying to
install Debian on one of these beasts. FWIW, we were actually able to
get a minimalistic kernel compiled for Sable to work, but it had
basically no device support or networking support, which probably
explains why.

If someone more knowledgeable on this part of DEC's product line could
comment on this it would be greatly appreciated.
