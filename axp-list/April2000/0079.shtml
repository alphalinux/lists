The way Kuro5hin.org works is that:
- any user can submit an article/story
- the said article/story goes into a holding que
- users (exculding the one who posted it) voite whether to post it; 1
  yes, 0 don't care, -1 dump it.
- the deciding factor in posting is when a certian percentage of
  registered users have voted positively on it. If not enough registered
  users vote positively it will stay in the que forever. Or in the event
  it's bad, it will get dumped.

If anyone wants to learn more they should see: http://www.kuro5hin.org
and http://scoop.kuro5hin.org
