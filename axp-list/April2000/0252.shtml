The biggest issue is making sure it is a 2100 and not a 2100A. The last I knew no one has worked up the necessary changes to get the 2100A to work. If this has changed please let me know.

The 2100 is the system board that has 8 EISA and 3 PCI slots on it. The 2100 works with either EV4 or EV5 cpus. There is no milo. You will have to use SRM and aboot to run linux. Please check the how to on these.
