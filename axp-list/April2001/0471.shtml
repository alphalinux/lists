Le Mon, Apr 23, 2001 at 10:07:33AM +0200, MILLET Mathieu a ecrit:
> I do everything from the CD. Should I create boot floppies ? If yes, which
> one should I create ? milo-xxx.img ? generic.img ? ramdisk.img ??
>

IIRW, you can't boot from the CD with every hardware, so you souhld make
the 3 DK :
dd if=milo-your_hardware.img of=/dev/fd0 bs=1024
same thing with generic.img ramdisk.img

I think you use ARC menu for booting with MILO, so configure it in order
to boot on the DK with MILO,

when you come to the MILO prompt, switch the MILO dk with the generic.img DK
and type :
boot fd0:/vmlinux.gz root=/dev/fd0 load_ramdisk=1 prompt_ramdisk=1
be sure to have the RH7.0 CD1 in place.
the installation process will ask you for the ramdisk.

hope this will help and have fun !
