I should point out that this thread is dangerous in many ways. The last
thing we need is to start up the generic "penis-size" contest of which is
better MS or Linux (which is bound to start when comments like "just like
microsoft" pop up). To help answer:

Linux is an unix-esque operating system. For helps in installing linux (and
or dual booting if you still want to run Windows along with anything else)
you can go over to: http://www.linuxdoc.org/HOWTO/HOWTO-INDEX/howtos.html .
This is the home of the linux documentation project howto's.

There are many distributions out there to choose from, they're all basically
the same (especially from a newbie standpoint), so just choose one and fool
around with it.

Hope this helps some...
