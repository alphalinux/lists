> The machine is an Alpha Server 1000a 5/300, `>>>show version` returns
> "V4.7-169, built on Nov 20 1996 at 13:32:50"
>
> I'm kind of new to the whole Alpha thing, is there anything I missed
> out ?

Wo thats old :-). First thing you'll want to do is upgrade your box
to the latest version available for it. Looks like it's 5.7. Before
we get that far it seems we need to figure out which revision of
alpha you have. gatekeeper (dec's ftp site) has two versions. One
for ev4 and ev5. At the SRM prompt do the following:

>>> show config | more

In the first screen scrolled you'll see mention of Processors.
Look for and note the type of alpha CPU. For example an EV5
would be an Alpha 21164. An EV4 is a Alpha 21064 . Find that out
and we'll go from there. It may be an EV5 judging from the speed...
