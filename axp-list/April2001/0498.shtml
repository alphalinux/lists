Ionut,

   I don't know the exact details, but Compaq and GCC use different
ways of specifying inline assembly. I would expect that cxx, uses the
same convetions as Tru64 Unix. (Maybe check the online man pages for
cxx/Tru64 to see how they specify assembly instructions.)

This is one of the main reasons preventing the compilation of the
Linux kernel by ccc.
