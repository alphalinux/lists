Can USB in a UP2000 be made to work, running RH7 with the
2.2.19-7.0.1 kernel?

The HOWTO says this machine should use OHCI, which I have put
into a kernel, with printer and scanner drivers as modules.
/proc/bus/usb/drivers shows hub and usbdevfs, but
/proc/bus/usb/devices is empty. Depmod finds a bunch of
unresolved symbols relative to various device drivers. In dmesg,
I see the following:

    usb.c: registered new driver usbdevfs
    usb.c: registered new driver hub
    usb-ohci.c: USB OHCI at membase 0x9010000, IRQ 255
    usb.c: new USB bus registered, assigned bus number 1
    usb-ohci.c: request interrupt 255 failed
    usb.c: USB bus 1 deregistered

Any suggestions on how to get USB to work? Do I need
the "hotplug" utilities from Redhat? Is UP2000 actually
an OHCI device, or might it be UHCI?
