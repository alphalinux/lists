On Sat, 28 Apr 2001, Uncle George wrote:

> Is it HARDWARE, or software problem. How can I tell.

 You may try to see if the error happens with the `scsiformat' program
(part of the scsiinfo package) as well. This would tell if that's the
problem with `scu' or with the drive. The `scsiformat' program is
available in the source form, so that may help debugging or looking at
actions that are performed.

 I've successfully formatted SCSI disks with `scsiformat' a few times.
