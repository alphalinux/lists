Actually, it may not be a CXX bug; merely a matter of what
the compiler chose in the absense of a specific declaration.
The -unsigned and -signed switches in cxx and ccc can be used
to set the default behavior when [un]signed is missing on a
declaration.

As a general performance tip on Alpha, using unsigned char/short
datatypes is going to be faster than signed char/short, particularly
when compiling for older Alpha machines.
