> Matthew,
>
> Do you know which, if any, support FCP mode 3, or are these limited to mode
> 1 (point to point) and mode 2?

I'm not quite sure I follow your terminology in this case.

If you're referring to FC Class of Service, the QLogic cards generally are
class 3, although you *can* get f/w for the 2200 that supports class 2
service. As far as I know, Class 1 isn't supported.

The supported topology for the QLA2100 is private loop (including PL_DA).
Fabric switches with aliasing can allow full fabric access, but it sure is
shakey.

The supported topologies for the QLA2200 are private loop (including PL_DA),
public loop, or point-to-point (with or w/o fabric, i.e, N-N or N-F port).
