To the alpha-linux folks,

As per several requests, I have put together a web page describing the
performance optimization issues with which I have dealt on my Alpha Linux
systems (XP-1000's). These issues include: mmap(2) problems, SCSI hardware
and software difficulties, kernel performance, and kernel panics, and
others.

I would very much appreciate feedback from the folks on this list who might
be interested in such performance problems. I should note as a caveat
that the target audience for this web page includes primarily computational
scientists and not Linux kernel experts. Nonetheless, I want to make sure
that the information I provide is at least accurate, if not complete.

http://orion.chem.vt.edu/alpha-linux/

Thanks in advance!
