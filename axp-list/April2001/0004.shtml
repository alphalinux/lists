NEED:
full ATX compliance. ( it would be especially sweet if the ports matched
one of the popular lay-outs )
4x AGP with the ability to use modern video boards.
low cost commodity RAM ( support for > 2Gig )
ATA-66/100
uses current EV67-EV69 processors

Would Be Nice:
ability to run OpenVMS, TRU64 and Linux
supported Integrated 10/100/1000Mb Ethernet on motherboard
supported sound integrated on motherboard
USB
some PCI slots to run dual monitors, SCSI, etc. ( is 66MHz, 64bit PCI out of
the question? )
