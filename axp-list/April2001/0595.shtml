Given that the linux 2.4.3 kern wont build on my po alpha, i would think
it would be prudent to inquire for the pro's and con's, particularily
when the machine is so expensive. I'd ask a lot more questions than just
hardware compatability. More meaningfull might be the effective
through-put ( computational, as well as I/O ) for 1, 2, 3 or 4 cpu's.
And maybe how does that compare with a beowolf(?) cluster strategy.
