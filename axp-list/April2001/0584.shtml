I use my original ECU floppy that came with the Jensen.

ECU SETUP:
==========

1. load ECU floppy from AlphaNT Firmware:
SRM >>> set os_type nt
ctrl-p (to power cycle the machine)
go to RUN ECU menu option

2. At the ECU menu, choose "Step 3: View or Edit details"

You will see all the devices listed, for instance:

Slot 6 - Adaptec 32-bit SCSI Host Adapter for Alpha XP (my AHA is slot 6)
         (with floppy)
Host Adpter Interface: Enhanced Mode Only
Host Adpter BIOS: BIOS Base Address CC000H
Host Adpter SCSI ID: Device Id 7
SCSI Bus Reset at Power on: Enable SCSI bus reset

3. Keep the highlight cursor on "Enhanced Mode Only"
   Press ENTER to Edit it.
   In my case, I see:

                (o) Enhanced Mode Only

   In your case you probably will be able to set it on/off with
   more options.

4. You can see also "Edit Resources = F6"
                    "Advanced = F7"

   ESC to cancel, or F10 to save and exit.

Good luck,
