Nope. No problem at all. You may want to stick with the same brand of
disk
that came out of it. Probably seagate or quantum. Only because
the connectors inside the SBB that hookup to the SCSI ID and FAIL state
on the disk should match up . Worse case is you hard set the SCSI ID.
