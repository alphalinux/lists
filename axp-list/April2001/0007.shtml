It would depend on whether you burned the CD correctly. If you
put the CD in another computer what do you see? A single file
or multiple directories and files? If you see only one file, you
did the burn wrong. The iso file is a complete image of the CD
and so when you burn it, specify it as such. I only use cdrecord
so I don't know what Winxx boxes need. For cdrecord, you would
do something like:

cdrecord -v speed=4 dev=5,0 6.2-i386.iso

Speed and dev are specific to your Linux system. See" man cdrecord"
for more info.
