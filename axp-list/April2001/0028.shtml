Where you stated:
>Tony,

Apparently, the AMD 761 Iron Gate can't do 66 MHz PCI (at least not from
>what I've read), though I suppose you could implement an external PCI bridge
>that could do the 64 bit/66 MHz.

The new AMD 760MP based dual CPU Athlon board from Tyan has 64 bit PCI with
( I think) 66MHz.
"Guiness" is code name for the Tyan board.
See some pics at:
http://www6.tomshardware.com/business/01q1/010329/cebit2001-04.html

A point to think about is that 64 bit is double the data rate, and there
are few cases where 66MHz helps. Also there are not many 66MHz cards to
choose from.
In most cases just the 64 bit bus should be sufficient for the extra
bandwidth people want.

As I believe that in this discussion thread we were looking at lower cost
solutions, I think we can hope for the AMD chipset to be adapted to dual
Alpha with DDR.
