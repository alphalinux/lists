Hi Luiz!
That is, you successfully using 18GB disk is fantastic!
And you do it still with "aha174x: Extended translation disabled"?

I'm not very well in English, but why you use aboot 0.5 ?
Where do you get it ? Installation "boot"-disks contains aboot 0.7 since RH
6.2. My bootable RH-6.2 CD-disk has aboot 0.7.
But if you wish I can send you URL where you can get "bootable" floppy for
Jensen with 2.2.15 kernel and aboot 0.7.
But if you've compiled aboot from sources, you may do a command (I get it
from aboot docs):
# cat tools/lxboot tools/bootlx vmlinux >/dev/fd0

You just get "boot" floppy. Then, you may use copyboot from ARC.
I send you aboot doc as attach file.
If you wish to use Jumpstart, please use
http://www.support.compaq.com/alpha-tools/

and please refer http://tuddv2.ced.tudelft.nl/jensen/redhat/6.2/

Henry does some work to make Jumpstart usable on Jensen.

regards,
Alexey
