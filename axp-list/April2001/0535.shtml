Hi All,
    The "ILLEGAL REQUEST" indicates the device doesn't like the
SCSI Format CDB sent. I think I've only used the interleave option
with SCSI floppies.

    The drive may allow you to change for Format Device Mode Page.
Use Scu's "set page" or "change page" command to alter mode pages.

    Also, if you wish to see SCSI CDB's issued, use "set debug on".

Regards,
Robin
