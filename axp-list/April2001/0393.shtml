Hi,

I collected two patches to use a Tru64 NFS server as boot
server for diskless Linux clients. If you are interested,
please have a look at:
http://www.scs.ch/~frey/linux/tru64nfs.html

The patches are tested with Tru64 5.1 on the server and
Linux 2.2.18 and 2.4.2 on the client. The patches handle
the different major/minor number encoding and the different
RPC authentication.
