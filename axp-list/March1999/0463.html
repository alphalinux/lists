Hi,
Digital Alpha XL-266 is finally up and running.
XL kernel absolutely refuses to boot, but Avanti
RH52 kernel seems to work almost OK. (BTW,
does anyone know why?) Even parallel printer
port now works.

One problem with Avanti kernel, though:
2.88M floppy drive does not work.

Trying to mount an MS-DOS floppy:
  end_request: i/o error, dev 02:00, sector 0
  mount: /dev/fd0 is not a valid block device

mdir a: gives this error message:
  init A: non dos media
  Cannot initialize 'A:'

/proc/ioports reports floppy 03F0-03F5,
which is also reported in the DEC user's
manual.

interrupts: nothing for fd0
dma: nothing for fd0

According to the manual, DMA setting
should be 2.

Does anyone know how to fix this
floppy drive problem?
