My original post mentioned finding extra copies of lpd running
and solving lpr problems by killing them (plus some other actions).

Since then I've realized the primary reason I've had problems is because
during my custom printcap installation I've tried to go too fast!

My custom printcap has an input filter (MagicFilter) which I keep on /user/local
(which I keep on a separate disk from my root disk). During installation
battles I don't load my /usr/local and thus any attempt at lpr gets foiled
because the 'if' parameter can't be found.

Moral of the story: Pay attention to the basics. reference to a non-existent
file will usually cause problems.
