lindahl@cs.virginia.edu said:
> I have a card which dma's to a buffer allocated in normal user
> programs. You mean I have to change all my user code (Fortran MPI)
> because of _your_ preferences? No, the solution should be general.

What are you talking about? Of course not.

If the driver does direct DMA to user pages, then the driver is compelled
to add any bounce buffers that are needed to do the job if the user page
is out of the range of the device.

The driver should be able to ask for buffer space in the first 2Gig if
that is all the device can handle.
