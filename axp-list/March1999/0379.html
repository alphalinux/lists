You seem to be assuming that zero is the "right answer". It is
if -mieee is not specified. But if -mieee is specified, then there
should be routines which return a denorm. I'm surprised that CPML
doesn't have such routines. (?)

If -mieee is specified, a preprocessor symbol is available. math.h
can use this to specify a different routine.
