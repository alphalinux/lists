Absolute easiest way would be an external solution
*independent* of any other machine - like a commercial
router or the new MultiTech Serial Proxy Server - it hooks
up to your back-bone and to a serial device (modem,
ISDN adapter, etc.) and dials up/connects on demand.

It wants a Win32 machine for initial configuration, but after
that you should be home-free.

Cost is $179 list.
