Hey all,

xntp3-5.93-4 (as distributed with Red Hat Linux 5.2) does not work
with Linux kernel 2.2 on the Alpha: xntpd abends, complaining about
the tick value being too large. Merely rebuilding the xntp3 package
results in a working (well, for the last 10 minutes, anyway) xntpd.

I've uploaded the package (alpha binaries for glibc-2 and repackaged
xntp3 sources) to in-rhcn.redhat.com. It should be available from
ftp.rhcn.redhat.com as soon as it is verified. The package is called
'xntp3-rhcn'.
