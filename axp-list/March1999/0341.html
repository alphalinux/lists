Okay, armed with my new 3/4" cable from Hard Data, the old 170M notebook
drive from my Dad's laptop, and the instructions from multimania.com, I
attempt to get my Multia IDE-enhanced! Bummer deal, though, as the ARC
console does not recognize it.

I currently have a 1G Quantum Empire on the SCSI chain to boot from. I
was going to install the IDE to see if it worked, and then find a larger
IDE drive so I could go totally internal.

On www.multimania.com/multia/miniide.html it says at the bottomw of the
page:

   Now that we have reached this point, if you have the latest firmware
   and if everything is plugged correctly, then there should not be any
   problem. Go to the ARC console, then do "Supplementary menu", and then
   "Display Hardware configuration". On the second page, you should see
   your new harddrive detected!

Well, all I see is

   multi(0)video(0)monitor(0)
   multi(0)key(0)keyboard(0)
   multi(0)disk(0)fdisk(0) (Removable)
   multi(0)serial(0)
   multi(0)serial(1)
   scsi(0)disk(0)rdisk(0) (6 Partitions) QUANTUM EMPIRE_1080S 1242

Where's the IDE drive?? I guess I should expect some trouble, 'cause the
instructions above say "if everything is plugged correctly". Maybe I got
stuff plugged in wrong. The latest firmware installation went without a
hitch and it says "Alpha Firmware Version 4.33" when it gets to the blue
ARC console, so I think I'm okay there.

Any ideas? There aren't jumpers that need tweaking are there?
