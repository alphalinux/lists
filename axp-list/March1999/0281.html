They really CAN'T. The old Netscape (i.e. Communicator 4.x and earlier)
was not 64-bit clean. The reason they had a DU version was, I presume,
because the DU compilers have that capability to grok 32-bit code that I know
next to nothing about.

The new Mozilla, however, which will become Communicator 5.0 as well as being
released independently as Mozilla, *will* be 64 bit clean. Whether Netscape's
actual distribution of Communicator (which will include their own proprietary
"value-adds" to the core Mozilla code (such as Full-Circle Talkback)) will be
released on Linux/Alpha remains to be seen. However, even if it's not, you
can still get Mozilla, which includes the browser, mail/news reader, and
editor/composer.
