I've had some interesting results with smbfs. I compiled the source rpm
version 2.0.2-6 (which I got off of rufus.w3.org). It builds and installs
fine but only works under kernel 2.0.36 for me. For 2.2.x kernels, it gives
the following strange error message:
"SMBFS: need mount version 6
 mount error: Invalid argument
 Please look at smbmount's manual page for possible reasons"

So, it does work sometimes!
