One thing that might be worth checking is for the use of shared memory; do
an ipcs -a and see if there are any big chunks allocated.

I've seen this happen with AVS on Irix boxes, if AVS crashes for some
reason or is killed unceremoniously it tends to lead big globs of shared
memory around. There may be other apps with the problem.
