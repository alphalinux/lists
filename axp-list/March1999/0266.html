This is a single 21264 board, made by Compaq.
The OEM single CPU boards debut in May with the new versions from API.
These will be released using Slot-B packaging for the CPU, and there is a
dual version releasing in June which might also be AMD K7 compatible.
This is all conjecture based on conversations, and I am NOT breaking any
non-disclosure agreements <grin>.
I refer you to an excerpt from an API data sheet on this where it says:

Product Part Number
Slot-B Alpha 21264 600MHz API264SB2-600
2MB L2 Cache

Slot-B Alpha 21264 600MHz API264SB4-600
4MB L2 Cache

Availability Q2 1999

For those interested I refer you to API for details beyond this..
