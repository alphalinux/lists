We have 2 8200s at Nasa/Ames. I started getting somewhere with a Linux
port a while back but dropped it. Instead we ported NetBSD to it- but
every now and then I think about finishing the Linux port.

You can have really substantial I/O connectivity with TurboLaser- not
only for PCI but for other busses as well- the I/O hoses come down to
outboard boxes which don't necessarily have to be the DWLPX that has PCI
cards.
