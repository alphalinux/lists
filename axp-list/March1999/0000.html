Amaya requires Motif libraries (lesstif, as long as a working
version, will do fine). I recompiled Amaya from sources and,
after some minor fixes, I am runing 1.4a on Alpha - no problems.
This is minus printing which is seriously broken in sources
and I have not a great urge to dive in it. If you will look
at the code you will find why. :-)

Oh, I sent my fixes to the authors.

The version I did for myself has the following requirements:

libICE.so.6
libSM.so.6
libX11.so.6
libXext.so.6
libXm.so.1
libXp.so.6
libXt.so.6

libdl.so.2.1
