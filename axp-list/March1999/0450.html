Many of you have presumably read the recent press release from Compaq at:

  http://www.compaq.com/newsroom/pr/1999/pr020399b.html

------------------------------------------------------------------------------
Compaq Offers Linux-ready ProLiant Servers,
AlphaServers and Professional Workstations

New Math Library Boosts Alpha Linux Computation
Performance by Factor of 10

[...]
------------------------------------------------------------------------------

It's great that Compaq is releasing Linux-related hardware and software
and I'd like to thank those responsible.

However the following statement in the press release is false as far
as I can tell:

>Compaq's active participation has been early and often, including:
>[...]
>* Releasing key Linux enablers under General Public License (GPL): MILO,
> em86 emulator, prototype Compaq Linux Smart-2 Array Controller Driver
> and IPROBE.

The source code files come with no obvious license at all. There is a
disassembler modified from GNU binutils which is marked GPL but that's it.

Most importantly, the source code that was made available only covers
a small part of em86. The crucial library libem486, which actually
does the emulation, was never made available in source-code form.

Thus there is no way that em86 is under GPL.

To those concerned: please either make it so, or withdraw the claim.

Bye,
  Rob.
