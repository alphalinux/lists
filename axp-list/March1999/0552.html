I've been trying to sneakernet a few files over to my Avanti, but when I try
to read the floppy via mtools (mdir, mcopy, etc.), I get a 30-line spew of
meaningless (to me) data beginning like this:

floppy driver state
-------------------

continuing on for a bunch of lines, and ending

floppy0: floppy timeout called.

What is this, and how can I fix it? I remember that on bootup, the system
thinks that the 1.44 MB floppy is a 2.88 MB floppy, so I've changed the
/etc/mtools.conf to

drive a: file="/dev/fd0H1440" exclusive
drive b: file="/dev/fd1H1440" exclusive

so that it *should* think that the floppy is a 1.44 MB floppy (which, in
fact,
it is).
