I have managed to install a 3.7GB 2.5 inch IDE drive in my UDB. My
firmware was up to date already so the machine recognised that there was
an IDE drive there when I booted it up. I didn't have to tweak anything
on the motherboard, just plugged my 44 pin mini-IDE cable in (got mine
from a company called Lindy, seems to be German) and stuck the IDE drive
in. Its definitely slower than the old 500MB SCSI but its fast enough for
my needs and having several gigs in the box makes all the difference along
with the nice S3 Virge DX 4MB video card. I've had no problems with the
machine running off IDE, a very good upgrade for Multias and if I need
speed I still have the SCSI available which is currently holding my JAZ
and CDROM drives.
