I would guess that a default RedHat kernel would have the correct fs
support in it. But it looks like it does not. You need to build a new
kernel and add this support to it:
DOS FAT fs support (CONFIG_FAT_FS) [Y/n/?] <-yes
  MSDOS fs support (CONFIG_MSDOS_FS) [Y/n/?] <-yes

After you have built the new kernel and added the correct fs support it
should work.
