Charles> How the $^#% did you get it installed? :)

Um, I downloaded a boot disk, told it where my local mirror was, and
off it went. There's a web page at
<http://www.alphalinux.org/docs/debian-udb-install.html> which explains
how I did it. Making the hard disk bootable is a case of mounting the
rescue floppy, copying linload.exe and milo onto the hard disk, and
entering them in th4e ARC console as a boot selection.

Charles> In all seriousness, though, I hope the Debian Alpha install
