| A system I administered in the past hired a "consultant". The only
| problem is this person erased a a text file that had all the users names
| in it. I had a script set up so that a users name was appened to this
| file when an account was created, and deleted when the account was
| created. Then a REALLY simple script allowed you to mail all the users.
| Since this change was done months ago (april 1998) ther are no backups!
| NOW I have to come up with another solution, since dumb/cheap consultant
| has dissapeared.
|
| Instead of re-inventing the wheel, I'd thought I'd ask if somone has
| already done this, parsed the users file and sent mail. If not, I'll do a
| perl script and post it to the list, perhaps somone else would need a
| hand :)

Assuming your users all have uids over 100 (which is typical):

        tr : ' ' </etc/passwd | awk '$3 >= 100 {print $1}'

I have tested that line and it's a good first cut towards rebuilding your file.
