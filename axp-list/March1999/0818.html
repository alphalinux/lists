Also, when compiling KDE, you'll get warnings about libc, etc, not
being linked dynamically. To fix this, you need to patch some file in
the acconfig (or similar name, don't recall exactly) dir, replacing
ELF-32 with ELF-64 prior to calling ./configure. You could also
replace this in libtool after running configure.
