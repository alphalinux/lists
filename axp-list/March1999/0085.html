When I was using ppp for dialup on my AS200, I couldn't get it working
without executing the SRM command script to install the fax/modem option
(covered in the manual). I don't know if there is a similar procedure for
ARC/MILO folks. For whatever reason I was not able to get shared IRQ's for
serial ports working on the original 2.0 installation (I'm running 2.2 now
and haven't tried).

Also, as a sanity check, you should make sure you have any update RPM's.
