Keeping in mind that this is the first cut.....

bochs builds and runs under 2.2.0-axp:
AS200 - EV45 h/w
egcs-1.1.1
2.2.0 kernel
glibc2.1

Messydos 622 boots ok, keyboard is a bit funky, and performance....well,
you don't wanna know. It's pretty bad. But it works. I need to spend
some time playing with it to see if there's a way to speed things up,
and get my app loaded into it. I'm still not sure where printer support
falls out. More later!
