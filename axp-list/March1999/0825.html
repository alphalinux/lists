I have a soundcard which uses memory mapped IO on ISA for data
transfer. This is terribly slow on my sx164, probably due to the
Cypress chip, which includes the PCI-to-ISA bridge AFAIK.
I used it on a PC164 once, and there it worked fine.

Does anyone happen to know if MMIO performance can be increased somehow,
by implementing specific support for this chip ?

Guenter
