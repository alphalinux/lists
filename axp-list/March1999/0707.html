>I just recently got my linux machine running on my internal network at work.
>I also have apache running and I have a static ip. I wanted to know how can
>it set it up to be able to get to the web page by typing http://linuxbox
><http://linuxbox> instead of typing the ip address. Is there any way i
>can do this with out having to bother the lan techs. The already dont like
>me because im not running win95 but who cares ahahahha =). thanks for your
>help guys.
>

  Ah, name resolving... There are two ways to resolve names - one is to have a
DNS server on your network. In this case I think you'll need to bother your lan
techs, 'cause the odds are they already have a DNS server(s) running (unless
LAN is MS-Windows strictlywith WINS server(s)) and have them include an entry
for your Linux machine. Therefore allowing everyone on your LAN to point to
your Linux box by name.

  Second - is just for you - to have a "hosts" file in your c:\windows (WIN95)
or C:\winnt\ssytem32\drivers\etc (WINNT) include an entry for your Linux like
this:

192.168.150.10 alpha.domain.net alpha

 Assuming the right IP address and the right name.
