ev4 like a multia. I haven't had the time to rebuild the latest egcs and
build for the lowest common demonmenator , the ev4. So if you first
uninstall those kde rpms you got from ALO and grab the ones from KDE you
should be fine. I'll be rebuilding all the rpms on ALO soon...
