On Mon, 15 Mar 1999, Durham, Kenneth J wrote:

> when i try mounting the dos partition i get an error that the file system
> was not reconized then i tried mounting it "mount msdos /dev/sda1 /mnt/dos"

I don't know if the syntax is ok, but if you get the message it seems like
it is.. (mount /dev/sda1 /mnt/dos -t msdos)

> but then i get an errror that this file system is not supported by the
> kernel.

I would recommend that you compile your own kernel and see to that it has
support for msdos; I had a disk kernel which worked with one computer,
while the "hd"-kernel didn't at all; the later didn't support my scsi
card. So probably it is like it says--the kernel does not have support for
dos compiled in it.
