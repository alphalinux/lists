>
> Yes, 'rpm -U xxx.rpm' will upgrade/install a package. But I don't want it
> to upgrade/install if a older/same version is not installed...

There is a package called autorpm that will look through a list of RPM's
and only install ones that are upgrades to the current ones. Most times
this works quite well but if there is a major change in a pakages format
like a package being broken into client/server portions autorpm will not
find those. But otherwise it is quite good.
