As per my posting earlier today, the S3 Trio64 on Alpha can be problematic
under X, unless the following are added to the XF86Config Device Section:

        Option "no_pci_disconnect"
        Option "sw_cursor"

So, if you have one of those, try this out.

If you don't, at least try to eliminate X from the equation by not running
it to see if the hang still happens.
