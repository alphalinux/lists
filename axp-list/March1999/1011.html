> blank itz a linux native partition..

The fact that it says "native linux" in fdisk does not mean that it
has a filesystem on it, just that the partition type is set to
"native linux" (82 I think). If you never made a filesystem on it you
will not be able to mount it. If you just did a RH install then you
would have had to check that you wanted a filesystem made on that
partition, also I don't remember if it will even ask if you don't
have the partition as part of the tree already. In other words, if
during the install you didn't pick this partition and say I want it
mounted to /home or where ever then I don't think RH install will
offer to make a filesystem on it.

If nothing is on it make a filesystem again anyway, it won't hurt.
The command you want it mkefs if I remember correctly. Use man or --
help (-?) and it should tell you what you need.
