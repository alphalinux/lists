Hi,

My configuration is

System 164 UX (Ruffian) 600 MHz
Matrox Millenium II (8MB)
Kernel 2.2.3
egcs-1.1.2

Has anyone managed to use the framebuffer support with this
configuration ?

I get the document available on www.alphalinux.org and followed
the sections specific for Mattrox.

That means:
1. I build a new kernel with added framebuffer support and installed it
2. Get the latest fbset sources and build it
3. Added video=matrox:vesa:0x108 to the boot line in MILO
4. Created the /dev/fb* devices (as described in framebuffer.txt)
5. Reboot

But when rebooting it switches to graphic mode - and never comes back
:-(

Only a Ctrl-Alt-Del helps

I also tried building loadable modules and load it with modprobe
after booting. But the result was the same.

Any Ideas.
