> Just one other question about this problem. I am really new at this if
> someone would like to give me some instructions on building ther kernel i
> would be very happy. ive been at this for hours. I have a stupid book
> infront of me telling me how to build a kernel but this shity book doesnt
> explain it that well so im realy stuck. Also can someone suggest a good
> book for redhat 5.2 thank you!

you should try the Kernel-HOWTO

or you can go to /usr/src/linux-YOURKERNELVERSION

and type make config (for plain text config)
or
make menuconfig (for text menuconfig)
or
make xconfig (for X menuconfig )

browse around in the kernel... and check what you wan't (to leave the
kernel small you should try using modules)

then do a 'make dep'

then on an alpha do 'make boot' to make the kernel
and a 'make modules' and a make 'modules-install'

the boot file is in the /usr/src/linux-YOURKERNELVERSION/arch/boot
it's called vmlinux.gz copy this to your root where your egsisting
vmlinux.gz file is (if your wise copy that to vmlinux_old.gz)

then do a reboot
that should do it.

If i left something out be free to add more info to this story
