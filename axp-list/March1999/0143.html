On Wed, 3 Mar 1999, Roman Martonak wrote:

> When I iconify an xemacs-20.4 window under fvwm2 (RH 5.1 on a UDB), it
> shows the same icon as for an xterm or rxvt. When I tried mwm from lesstif,
> it used the correct xemacs icon. Does anybody know how to get fvwm2 use the
> right xemacs icon ?

Roman,

  I don't use fvwm2 (I use WindowMaker) so the details I'm about to
give may be a bit vague. In fvwm2's configuration file (.fvwm2rc ?,
check fvwm2's man page) there should be a parameter something like
`PixmapPath'. Locate the directory which contains the xemacs icon you
want to use, and add that path to the `PixmapPath'. At least this is
the way it worked when I was using fvwm95 a few years ago. Good luck.
