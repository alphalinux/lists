Both ncr53c8xx and sym53c8xx controllers have a long list a parameters
which can be passed in a boot command and which affect how controllers
behave. Corresponding README files have a lot of details. Did you
try to boot using 'safe' option? What happens then? How actually
is configured your driver in your .config file? Also do you know
which is the first version of this driver which started to fail for you?
Last, but not least, did you ask the author?
