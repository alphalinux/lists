        The DEC 3000 series are Turbochannel machines, and Linux
        does not contain the necessary low-level support for them.
        On the other hand, NetBSD *might*. (Or was it OpenBSD ?)

        So the first question: Will Linux run at it ? NO.

        (This is a chicken/egg dilemma, Linux doesn't run at them,
         so nobody buys them to run Linux, and there are extremely
         few crazy enough to see the trouble thru to get Linux to
         run at previously unsupported hardware, and even they are
         unlikely to end up with such a box... ... especially as
         they would need another Linux/Alpha box to support the
         cross-port job. Been there, done that, and been success-
         full :) )
