Im not sure if this topic is resolved yet.

Just wanted to let you know that I got a non working ESS 1370 card to work
by using the 2.1.131 kernel (enable beta modules) on a PCI164.

With 2.2 kernels it didn´t work.
Additionally it always used to work on my SX with this kernel version.

I think that brokenness of this driver within 2.2.x was reported before.

Note, that probably something in the kernel changed and not the driver.
Haven´t checked yet.
