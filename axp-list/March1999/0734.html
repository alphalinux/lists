Bill,

Multia 166 MHz = Pentium 90
External CD-ROM can be purchased to attach to the built-in SCSI II
adaptor.
An external modem can be attached to the built-in serial port.
You can add a good video card if you choose.
You can also add an external backup device using the SCSI II adpator.
