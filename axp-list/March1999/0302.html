Sorry if this has been discussed before. I just recently installed
RedHat 5.2/axp on my EB164 machine, along with all the updates on the
RedHat web page, and found that a perl script I rely on (latex2html)
busts badly. It works fine on RedHat 5.2/i386, and also RedHat
4.2/axp, so I suspected there was a problem with perl.

I grabbed the sources for perl5.005_02 and did a build. The 'make test'
suite failed on lib/anydbm and lib/ndbm. I went over to
http://www.sleepycat.com/ and grabbed the sources for Berkeley DB
version 2.7.3, which built cleanly. A rebuild of perl5.005_02 now
passes 'make test'.

Latex2html is working much better now.
