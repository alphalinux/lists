> I need a second NIC in my Multia and would like some feedback on PCI cards
> that others have successfully used. Specifically, I have a spare 3Com 905
> that I would like to use. I would also need to know if I need to pass any
> options to MILO when I'm booting to have it recognize the card.

IIRC, the kernel only autodetects one NIC, whichever it finds first is
eth0, and then it continues with the rest of the boot process. You need
to tell it to look for eth1 with a kernel parameter. Check the
Ethernet-HOWTO .. for specifics.
