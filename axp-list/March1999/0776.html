No, No, No.... this looks like the older TurboChanel machines. This will
NOT run Linux, and it (or ones just like it) have been on Ebay for months.

If you get the machine you'll need to run NetBSD?? (One of the BSDs will
run on this, not sure which one).

X will run fine on any supported Alpha (not the 3000 series of course).
StarOffice is not on Alpha, though ApplixWare is, WP8 is Intel only, though I have
read reports from a couple of guys within Compaq that they have it running
under em86, I've also heard that Corel might port it to Alpha.

As for speed of the Multia (166MHz) it think a P90 is about right. I have
a P100 and it keeps up with that most of the time. Kernel compiles take
longer, but I don't see that as a problem. The Multia is a good box to
learn/play with, and the best part is they are cheap :)
