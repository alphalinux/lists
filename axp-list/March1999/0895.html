On Mon, 15 Mar 1999, Durham, Kenneth J wrote:

> Just one other question about this problem. I am really new at this if
> someone would like to give me some instructions on building ther kernel i

* Get the kernel source (I never use kernel-RPMs).
* Put it in /usr/src/linux
* Run make (config/menuconfig/xconfig)
(I recommend menuconfig)
* Choose what you want your kernel to support. The make config-stuff will
generate a file .config with you settings. I attach mine, if it can be to
any help--it is for an Avanti with ppp and such stuff.
* do the following:
make clean
make dep
make boot
make modules
make modules_install

make boot will generate your /usr/src/linux/arch/alpha/boot/vmlinux.gz,
which is the kernel to use.

* Try your new kernel, and _don't delete your old_. Make sure that you can
boot your old kernel if your new doesn't boot like you want it to.
