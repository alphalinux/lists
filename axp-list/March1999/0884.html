No the 166's do not overclock. The 233 mutia's can be overclocked to
266 or 300 MHz (at your own risk). There are several types of multias
on the market.
1. non upgradable 166 (VX40 models)
    This is the most abundant type on the market
2. upgradable 166 (VX41 models)
    This one can be upgraded to 233 but the 233 processors are hard
    to find and rather expensive.
3. "upgradeble" 233 (VX42 models)
    The upgrade processors for this type were never made. But the
    jumpersetting is available for 266 and 300 MHz. Mine is quit
    happely running at 266 MHz for a few years.
4. an Intel based one Pentium 100 (VX51 models)

each of this types can come with or without certain features, like a
spare pci slot, an external scsi connector etc. If you want to know
all the details you should download
   http://www.alphalinux.org/docs/udb-manual.pdf.gz

Gert Huisman
