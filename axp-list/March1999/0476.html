On Tue, 2 Mar 1999, Gyenes Istvan wrote:

> It might be a silly question, but how can I delete
> /proc/kcore?? I tried mozilla and I had to kill it. The result:
>
> -r-------- 1 root root 67117056 Mar 2 00:00 kcore

Wow, that's a rather large file, isn't it? I can't blame you for wanting
to delete it! Unfortunately, Linux doesn't support this operation, so if
you really want to remove the file:

1. Open up your system case. Some systems have quick-release knobs that
   allow you easy access. Others will require a screw driver.
2. Locate the memory chips. Likely they will be two long, slender items
   with lots of small memory chips embedded on them. These are called
   SIMMs or SDIMMs, depending on which memory architecture your system
   uses.
3. Lift the release tabs on each end of the items, and remove them.

Reboot (you may have difficulty getting a MILO prompt, however).
Congratulations, you've successfully removed the pseudo-file
'/proc/kcore'. :-)
