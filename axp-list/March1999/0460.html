Regarding > 1GB RAM...

I have a clone Alpha system (164LX MB) and it's maxed out at 1 GB RAM. I
always thought it was a bios or chipset limitation that I couldn't get more
RAM in my system. (As well as there being only 4 slots for RAM chips.) If
there are 512 MB chips for my system and if I thought it could use them I
would buy them in a heartbeat.

EN

> We have some subtle problems at exceeding about 1G RAM in
> Alpha systems. The subtleness issues come from the IO-
> subsystem, and how PCI bus-mastering access reaches the
> main memory thru the PCI/CPU interface.

Ah so - this is a topic of great interest to me. I would like to build
a cluster with 1 terabyte of memory, and the building blocks would be
alphas with 4-8 GB of ram. Unfortunately the Intel folks don't have
much motivation to work on these issues...
