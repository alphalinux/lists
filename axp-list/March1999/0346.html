It's called "bz2" and it's just a better gzip.  It's decompressing
sister program is bunzip2 (or you give the proper flag to bzip2).  It
does great compression, so we'd all save bandwidth if we make the
switch.  I think kernel.org even makes the bz2 source available in gzip
format, so you can get it and build it. 

Oh, and get the latest patch, ac4 I think.  2.2.2 doesn't buld on Alpha
out of the box.
