The XP1000 should have the same memory system -- the glossy literature
says "2.6 gigabytes/sec memory bandwidth", which is the same macho
number that yields 1 gigabyte/sec sustained performance on the 264DP.
Also, gcc should be able to do simple loops well, since stream
spends most of its time waiting for main memory; you can throw in all
kinds of bad compilers and get the right answer.
