Here is where the flame wars will probably start :(

*I* say this 233MHz machine feels like a P166 at some things and a P300 at
others. A test app I made shows that my AS200 4/233 is faster then a
200MHz Pentium Pro, but again, this is JMHO.

How much memory it has has a big deciding factor in this. If you have
32MB - 64MB or less it will probably feel like a P133 - P166, put in more
and it will act accordingly.

JMHO,

Ron Farrer
