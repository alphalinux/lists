Hi,
I now see that the DE500 (Tulip) card has been replaced by the DE600 which
is an Intel 82559 (EtherExpress Pro/100) based card. In our tests with this
card under Alpha Linux 2.2.X it has performed rather badly under load, with
lots of transmit timeout messages appearing on the console. e.g.
eth0: Transmit timed out: status 0050 0c00 at 2760762/2760790 command
000c0000.

We've also tried the Intel supplied source code ... this compiles fine (and
is also included in the RH7.0 Alpha 2.2.17-4 kernel) but it doesn't work at
all.
 
So is there a good Linux device driver for this thing anywhere ?
 
Cheers,
Addy.
