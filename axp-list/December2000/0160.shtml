Hi,

as some of you might recall I tried to set up an Avanti (4/233) as a
netfilter/iptables firewall. It's up and running with 2.4test12 now.

Alas I realized yesterday that it will not be able to handle our bandwith.
Even without any iptables rules and 3 active interfaces it maxes out at 3,6
MBytes/s and 99% kernel load. We "unfortunately" have a 100MBit/s line to
our ISP;-) And the traffic between our staging and our production web/db
servers goes through this box, so I do expect the bandwidth to be really
used...

Does anybody know if a Ruffian (533 MHz) with the same quad Starfire NIC
might be fit for this task or if I need a bigger iron? I plan to use 4
interfaces (eth0: Internet, eth1: dmz with mail/proxies, eth2: lab network,
eth3: link to interior router/fw. The filtering rules will not be too
complex.

Thanks,
Ingo
