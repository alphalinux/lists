>
> can anybody help me in upgrading rpm-3.0.4 to rpm-4.0

Don't!! Unless you _really_ know what you are doing. A database
format is different and is using new version of 'db' libraries.
Most likely you will get yourself into BIG troubles.

> Now I am getting a error message: only packages with major numbers <=3 are
> supported by this version of rpm. I want a detailed information.

Update to rpm-3.0.5-9.6x from updates to 6.2 and this will solve that.

There is also a version rpm-3.0.5-9.5x for 5.x distributions and
it also handles rmps with version 4 headers.

Whichever way - the first thing after updating rpm run
'rpm --rebuilddb' command.
