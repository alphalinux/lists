1. You can run some very sophisticated diagnostics on the hardware from it.
2. You can boot several OSs from it. (linux,Tru64,OpenVMS,*BSD)
3. You can netboot from it.
4. You can run diskless from it.
5. It's simple, efficent console.
6. You don't have to hop through ten levels of windows/screens to get to the function/information you want.
7. It can be ran through serial connection without having to remember some keycode to mimic navigation.
8. It is still supported and development will continue.
