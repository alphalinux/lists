There are lots of mailing list archives out there but they are usually inconvenient to use.
 
This is an archive tool you are going to like enough to really use! Here are some of the features that are way beyond the usual list archive capabilities:
 
• Show only messages since you last checked, or
• Specify how far back to look for messages
• Easy navigation
• Sort discussions by thread
• Each search match shows the whole thread
• Post to the mailing list from the web
• Define which mailing lists you want to look at and which ones to ignore
 
Check out "Last Day" in the upper left corner and the "New Messages" utility near the bottom of the left pane.
 
Add your comments about this mailing list browser or suggestions for new mailing lists to archive in the Forum Administration topic.
