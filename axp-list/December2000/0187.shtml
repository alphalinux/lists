Is there a command in Linux like the Tru64 disklabel command? The disklabel
command allowed you to copy the partition table of a disk to a file and
modify it. It could than be copied back to the disk.

This was nice if you had to partition alot of disks with the same partition
sizes or if you were building systems using scripts.
