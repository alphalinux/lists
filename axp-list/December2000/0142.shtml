I take it your trying to run the RCU utility?

try "run rcu" with the rcu floppy in the drive.(it should switch to arc
console and run the rcu program. Some firmware releases had this feature )

else type "arc" and select "run a program" from one of the menus..

The program name on the floppy is swxcrmgr.exe

This of course assumes this is a kzpac (dac960 based raid)
