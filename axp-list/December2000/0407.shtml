Yes it will install, but it does not

1. install a mingetty for ttyS0 in /etc/inittab
2. add ttyS0 to /etc/securetty (to allow root login over the serial port.
3. add console=ttyS0 to any "/etc/aboot.conf" entry to allow automatic
booting using a serial console.

So after a serial install I use the jumpstart "spawn a shell" command to go
and make the required changes after the install is completed.
