On Mon, Nov 27, 2000 at 03:03:13PM -0500, Don Russell wrote:
>
> I just received a 3DLabs Oxygen VX1 along with a note indicating that
> this card is going to replace the Elsa Gloria in XP1000s.
>
> Can anyone confirm this?

Yes, it looks like this will happen.

> If true, will older machines need a firmware update to use them?

Possibly, but we will only be equipping the EV6 boxen with them, so
it may not get tested in older ones.

> If I wanted to put one (or two) in a dpw433 or dpw500 would that be
> possible?

Two would only be possible if one had a VGA disable switch on it. Also,
I don't know if the XFree86 GLINT driver's VX1 support has been tested
for multihead...

XFree86 4.0.2 does support VX1, but the PCI version we have here
exhibits some color-related problems (that are not present under
the Tru64 server) that I'm trying to track down...
