> On his machine the OS is
> Redhat6.2
> with kernel patches to create files greater than 2GB.

First off, AlphaLinux supports files greater than 2 GB pretty much all the
time. So this other box must be an Intel box or something.

Second off, that Intel patch allows files bigger than 2 GB *if* the utility
uses special routines to access the big file. So that doesn't mean that the
ftp program, for example, uses such routines.
