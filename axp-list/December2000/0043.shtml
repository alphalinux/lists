I after several rebuilds of the server and eliminating each one of
the compile options I was specifying, I've found that compiling
with -mcpu=ev56 is causing the problems I've described. I'm not
sure where or why yet. I have found that it compiles ok if you
set in site.def or host.def these switches:

#define DefaultGcc2iAxpOpt -O2 -mcpu=ev5 -Wa,-m21164a -DBWIO_ENABLED
#define LibraryCDebugFlags DefaultGcc2AxpOpt
#define UseCompaqMathLibrary YES

I tried the 4.0.1e and g releases. The g snapshot was current ast
of this evening. I'm using GCC 2.95.2. I have not tried a later GCC.

Alan
