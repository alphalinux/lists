> 3. how do I single boot in Redhat Alpha?
> In linux(intel base), I only pass this paramaters... "linux
> single" upon prompt during the initial booting in linux. In Alpha,
> I supposed it would be in the SRM prompt, but how do we do this?
>
> Many thanks again for your advance help...

As shown in the SRM example you gave above, all boot parameters would
be included in the -fl switch, so if you boot dka (and aboot is set up
properly) you would simply enter

b dka -fl "single"

at the SRM prompt. The quotes can be left off if you only have on
argument in the -fl.
