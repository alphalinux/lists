Jeffrey Sturm <jsturm1@home.com> wrote 

There was a thread recently on c.o.l.a about cache architecture and page
colouring.  Search for "Tru64 vs. Linux" or something like that...

In short, when Tru64 and Linux run the same binary code, the only
difference lies in the kernel, specifically the VM subsystem.  Linux
probably causes more cache misses than Tru64 because its VM subsystem is
not optimized for Alpha.  That's part of the price you pay for
cross-architecture portability.
