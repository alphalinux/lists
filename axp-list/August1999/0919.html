Ken,
        With a standard kernel I've gotten a Trident NX based card (the
Hoontech NX) to work. With a hacked kernel I've gotten it to work better
(lowering the DMA window to 512 MB). It's a really nice card and
certainly warrants lowering the dma window, especially if you have 512 MB
RAM or less. Up to 1.5 Gig I think that you can just lower the first
window and set up a second window, though I've never bothered trying since
I only have 256 MB RAM. This card has a lot of nice features which work
under Linux/Alpha/ALSA (doesn't work under OSS).
        -Chris
