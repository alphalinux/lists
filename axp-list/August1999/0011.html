Yeah, I tried it this weekend... in fact, I've been
trying to upload my RPMs/SRPMs to
ftp.alphalinux.org... but can't seem to get in.

For me at least, replacing libstdc++ resulted in a lot
of unalignment traps (I guess I might rebuild those
apps).... also, most of the KDE stuff wouldn't compile
with gcc-2.95. With C apps, things seem to work very
well, but C++ code seems to have some problems here
and there... which I don't know if it is the result of
stricter C++ rules in 2.95 and poor code or just
compiler bugs.... there is already a list of known
bugs for 2.95 at egcs.cygnus.com... you might want to
check that out.

I've got two sets of RPMs built for EV56.. one
replaces RH's egcs and one installs in /usr/local ...
if you want them let me know... or if you want the
SRPM so you can customize to your liking ... let me
know. I hope to upload them to alphalinux.org soon.
