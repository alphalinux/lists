> Greetings -

Welcome to the Dark Side ;) I'm on the AlphaNT list as well as the
AlphaLinux list so it's good to see you asking questions.

> I would like to move the machine to AlphaLinux and forget about NT.

I've been using AlphaLinux as my main box on an AlphaStation 200
166MHz and then onto an SX 533MHz. Everything from word processing
to personal programs.
   
> Is there a Web browser - e-mail client (like outlook) - and basic Word
> processor for Alpha Linux.

Web Browser is the problem. Nothing as good as IE or Netscape on
AlphaNT yet. Right now Netscape 5.0 Beta (Mozilla) is 64bit clean and
coming along. But, the best browser I've found for me is in KDE. The
file manager in KDE is a nice fast and very usable browser.

Word Processing. ApplixWare is native on AlphaLinux. It has some key
binding stuff that will drive you a little nuts if you are moving from
Windows programs. (like no shift - end to select to the end of a line
etc.)

For $89 it comes with Word Processor, Spread Sheet, Mail Client,
Presentations Program, and Database Access.

Email. More applications than you can shake a stick at. I use KMail
(which comes with KDE). It is rather nice. It will read HTML, but not
compose in HTML (which seems to be the thing that most people in the
Linux world want).

> What are most people using their AlphaLinux systems to run?

I like Fractal Stuff, so I do graphics, movies, cd mastering, standard
Math programs and general use applications.

Samples:
POV-Ray (raytracer)
Blender (3D Modeler)
Various MPEG Encoders
Various MP3 Players and Encoders
ApplixWare
Scilab
Euler
Xaos
XFractint
Code Crusader (IDE)
Code Medic (IDE Debugger)
GIMP
the KDE suite of Applications
etc. etc. etc.
