>I have an AS200 currently running NT. Well now NT will be dropped and I
>would like to move to Alpha Linux.
>
>I have RedHat 5.1 for Alpha in the box.

>I would like to move the machine to AlphaLinux and forget about NT.
>
>Is there a Web browser - e-mail client (like outlook) - and basic Word
>processor for Alpha Linux.

lynx works well, when you do not need graphics. Amaya does not do frames
and distorts some images, but does well on most web pages. You have to get
used to double clicking on links. I played a bit with the KDE file manager
and it has promise. I never put it through a full set of paces. (It makes
a decent ftp client.) Now that some people have figured how to give
binaries a 32 bit personality, there is a widely communicated wish for a
netscape binary for the alpha.

I have used lyx for word processing for about two years. It takes some
adjustment, but I find the results very satisfactory. It has an sgml
document type (AKA linux doc) that converts to anything (smgl2xxxx). You
need the correct xforms library before it will compile. I have compiled 4
times on an alpha with never a single hitch (except the xforms library).
