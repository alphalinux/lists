On Fri, 13 Aug 1999 11:28:52 -0400, "Champigny, Michael" <Michael.Champigny@compaq.com> said:
> I get a couple startup messages from aboot (it's version 0.5) and
> then it complains about wrong type for partition 1. I tried with types

Well, this is OK. Aboot has a fixed location (determined at compile
time) on which partition and which file it will look for configuration
information. Since you haven't formated any partition yet, this is OK.

The question is: Do you get an aboot prompt ("aboot>"), and are you
able to enter anything?

If not: try skipping the "-fl i" part for booting.
