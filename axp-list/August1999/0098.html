Netscape requires the following Tru64 shared libraries:
libX11.so libXmu.so libc_r.so libots.so
libXext.so libXt.so libdnet_stub.so
libXm.so libc.so libm.so

Would it be possible to run Netscape with the GPL equivalents of these
libraries, all compiled under Tru64 unix. If so, then this would seem like an
effort that would be much appreciated by many people.
