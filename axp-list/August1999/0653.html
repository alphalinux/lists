   } Just wondering, i just compiled bogomips on my alpha and after

   } my AlphaServer 2100 5/250 (dual 250Mhz) bogomips = 60

Although BogoMips are rather bogus, they have there uses,
as is pointed out in the BogoMips mini-Howto, available from
the obvious Linux archives.

As listed there, all Alpha have a nearly 1:1 ratio to the clock
speed of the CPU, i.e., a 250 MHz alpha CPU should get say 245
BogoMips. Actually reported values are listed in the mini-Howto.

The actual value(s) for your system can easily be listed by
>> cat /proc/cpuinfo
When you compare these with the values in the mini-Howto, you'll
see that those kernel boot sequence based values give more
serious information than the standalone, CPU-load dependent
program run in user space.
