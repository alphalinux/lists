> to make a cd image that I can use to install from, and it seems to support
> making bootable CDs, but I need an image first. Can anyone help?

The only time you need a "bootable cd" is if you are installing from the
SRM console. For any other type of install it's only required that the
cd have the iso9660 fs + rockridge extentions and thats all. You can
find a iso image of the redhat-6.0 axp cd at
ftp://ftp.digital.com/pub/linux/redhat/redhat-6.0.iso .
