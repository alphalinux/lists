Alvin Tan Siong Joo,Systems Specialist,SCS-CSN wrote:
>
>
> Does anyone know where I could get those files so that I can configure using
> XF86Setup ?

For XF86Setup you need XF86_VGA16 server which does not exist for Alpha.
This is still not the end of the world as you can install Mono server
and provide a link XF86_Mono -> XF86_VGA16. This is good enough for
XF86Setup to work even if you will loose all colours which are there.

Saying that my past attempts when I tried XF86Setup on Alpha were
stymied by bugs most likely in Tcl. In the moment you tried to write
all your configuration to a file the whole thing was hanging. Not very
useful.

You may have a better luck with SaX utility from SuSE distribution;
but I did not try how it looks like, if at all, in a "final" 6.1 SuSE
distro for alpha.
