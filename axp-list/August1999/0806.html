>>> Xiaozhou Qiu said:
>
> Does anyone have ideas about it? I tried to boot in single mode, but in
> the single mode, the file system is read-only. I can not make any change.
> Anyone knows how to boot in single mode with r/w filesystems?

At the single user prompt, type in:

        mount -o remount,rw /

to get root remounted read/write.
