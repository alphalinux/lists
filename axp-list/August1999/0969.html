Hello

I have a UDB with a DEC 21143-PC based PCI nic and SRM sees it just fine.
ARC should work similarly -- if it knows about the card's specifics, it can
query them. As far as I know, this should only work with 2114x based
series nics.

ARC or SRM seeing the card should not affect it working though. If you
have the correct linux drivers for it loaded, it should be fine. I have
had 3Com nics in that slot and ARC did not see them properly but they
worked fine.

ciao
