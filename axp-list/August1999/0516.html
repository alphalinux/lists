I actually figured that one out - arch/alpha/kernel.c got modified to
ignore 4 of the 5 ranges supported by cy82c693, resulting in the
4*0x800 difference in I/O ranges.

So I looked at all the relevant differences between 2.2.12final3 and
2.3.15pre2 in both the alpha and the IDE code, and tried reverting the
changes one by one - no luck.

1. Is there *anyone* running a recent 2.3 kernel on alpha w/ IDE?
2. Is there *anyone* running a recent 2.3 kernel on a SX164?
