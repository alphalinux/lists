On Tue, 31 Aug 1999, nm wrote:

> I tried an S3 Virge (8c325). But I installed DU with
> my matrox in place so maybe the installation did not install
> the proper X server? Would it be worth my time to reinstall
> DU with the S3 card in place?

No. The S3 Virge is not supported. If you insert a supported card,
you notice this in the kernel boot messages. They change from:
"kernel console: vga0" to "kernel console: xxxxx" where xxxxx
may be "trio0" for the S3 trio or "comet0" for the Elsa Gloria
Synergy. BTW: There is some info on the Elsa Gloria driver
now on:

http://www.unix.digital.com/faqs/publications/updates/V50/comet.htm

It contains the expected PCI IDs and explicitly talks about
the Digital 4D10T and the ELSA Gloria Synergy. So maybe the
"original" Elsa version already works with DU 4.0E and higher.
