The simple answer is that the RPMs are made for RedHat 6.0, and depend
on some of the upgraded libraries. If you want RPMs for 5.2, you'll
have to roll your own. It's not hard though, just download the source
tarballs and do, e.g.,

rpm -tb gnome-libs-1.0.14.tar.gz

and it will build an RPM and put it in /usr/src/redhat/RPMS.

If that runs into trouble, e.g. a version problem in the spec file, you
can extract it with

tar tzf gnome-libs-1.0.14.tar.gz | grep spec

to make sure you get the right spec file name, then

tar xzf gnome-libs-1.0.14.tar.gz gnome-libs.spec

Then put the tarball in /usr/src/redhat/SOURCES and do

rpm -bb gnome-libs.spec

and it will build the RPM.

Hope this helps,
