Hi

Earlier today I wrote :
I know this is a FAQ , but I can't seem to get to the
mailinglists-archives.
I bought a CD recorder and installed it succesfully.
The only problem I have is that cdrecord bails out with a "shmget" error.
Anyone remembers how to fix it ?

Thanx to reactions on the list I now have a clear view:

cdrecord wants a shared memory segment bigger than the kernel allows.
This can be fixed by adjusting the SHMMAX value in include/asm/shmparam.h
and recompile.

As a work around, you can tell cdrecord to use less shared memory.
This can be done using a parameter to cdrecord (fs=2m)
or by setting the environment variable CDR-FIFOSIZE=2000k
