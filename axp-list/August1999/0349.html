>>> Rich Payne said:
>
> Second, I think that starting with 6.0 the ramdisk uses a different format
> and I remember somebody telling me that ROMfs had to be in the kernel. I'm
> not sure if Jay's 2.2.3 image has this or not.

Correct; that 2.2.3 image from off gatekeeper does NOT have the ROMFS
needed for the 6.0 RAMdisk to work. You could use either the 5.2 RAMdisk
or you could eschew RAMdisk altogether, via (assuming SCSI CD):

>>> b dva0 -fi vmlinux.gz -fl "root=/dev/scd0"

This should work on DS10, and is worth a try on other boxes when needing to
use that 2.2.3 kernel to install 6.0.

Good luck.
