On Wed, 11 Aug 1999, Wes Bauske wrote:

> Also, to use the disk as a single partition, does the
> ext2 file system block size need to be increased? I can
> easily generate a file that will fill the whole drive
> so that has to be supported.

I would probably do something like this:

mke2fs -m1 -b 4096 /dev/<superhugedisk>

That only reserves 1% instead of 5% space for the superuser, and uses 4K
blocks (very important for a single filesystem that large if you don't
want to waste half the day when you do a fsck).
