The problem appears to be a compiler problem. If you can build
your kernel on a RH5.2 system (with original egcs compiler) the
kernel should run on the DS20 - I haven't seen it fail yet, neither
has Jay E. However, some kernel configs will build and run on a DS20
with RH6.0. Below is a kernel config that works for me. Also please
send me you kernel config that doesn't work.

larry
