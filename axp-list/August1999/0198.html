On 10-Aug-99 Rob Byrnes wrote:

> So now to the questions .... Is paying AU$60 per unit, plus AU$75 per 16Meg
> of True Parity memory too much?

I'm not certain what the going conversion rate is at the moment, but if you
plan to actually run anything on the Multia, you should consider at least 32
megabytes of memory. Also note that the memory must be used in pairs, so each
Multia will require at least two SIMMs of whatever size you choose to use.
