> Question 1.
>
> I have installed RH 6.0 and booted the machine the second time.
>
> I need to run the file copy to copy linload.exe and milo to the DOS
> partition for booting without a floppy.
>
> I selected x Windows to boot with Linux.
>
> How can I exit x Windows and return to the prompt to copy files from
> diskette to the DOS milo partition?

Can you run Xterm in XWindow? If so, you do not need to quit X window.
Besides, you can enter Ctrl+Alt+( any key from F1 to F6) to login as a
console, you can switch back to Xwindow by Ctrl+Alt+F7.
