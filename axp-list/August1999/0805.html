        Well, what system do you have? I'm familiar with the pyxis system
because that's what I have, I'm not strictly sure what needs to be done on
yours if anything does, though it should be about the same if anything

> Also ... with this card under ALSA, do you get full duplex PCM audio?
Oh sure. Full duplex PCM audio. Also, it does mixing in hardware, so
that you can open up /dev/dsp with multiple programs simultaneously, and
they'll be mixed in the card itself. ALSA doesn't support software mixing
yet, so that's about the only way that you're going to get that sort of
mixing short of buying OSS and dealing with binary-only modules.
        Plus there are options for S/PDIF outputs, coax and optical
outputs, etc. (those options do cost more.) I think that the S/PDIF
output is in the process of being written.
