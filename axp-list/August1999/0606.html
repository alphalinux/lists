Glen,

I'm betting you have a hardware problem. First thing I'd do is remove
the Mylex, just to simplify the installation configuration, and reseat
all the memory SIMMs. You might also, depending upon the number of them
you have, try swapping them around. The error you're seeing, as you've
probably guessed, is the result of the error-handling being interrupted
by another error. Given the speed with which the events that cause that
must happen, the most likely scenario is some sort of broken hardware.

Cheers,
jwl
