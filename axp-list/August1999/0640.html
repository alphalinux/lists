you dont need to unzip the vmlinux.gz file.
use
 swriteboot /dev/ sda /boot/bootlx
to make your hard disk bootable.

then you can use
set boot_file "1/boot/vmlinux-2.x.x"

check what is the full file name in /boot directory.
now boot from harddisk.
