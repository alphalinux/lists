Hi,

I'm using RH6.0 on a pcLX164. I'm try to upgrade to
the 2.2.11 kernel. I did make menuconfig, make dep,
make clean, make boot, make modules, make
modules_install, then I moved the new
kernel(vmlinux-2.2.11.gz) and the new System.map
(System.map-2.2.11) to /boot. I linked System.map to
System.map-2.2.11. The old map file is named
System.map-2.2.5-22. Then shutdown -h now, and reboot.
I'm getting the following error messages the reboot:

"mounting local file systems insmod:
/lib/modules/2.2.11/fs/fat.o: unresolved symbol igrab"
The "insmod: ... igrab" error message repeats three
more times. Then:
"mount: fs type vfat not supported by kernel"
This repeats three more times. Then later in the boot
process:
"mounting other filesystems insmod:
/lib/modules/2.2.11/fs/fat.o: unresolved symbol igrab"
Again the "insmod:...igrab" message repeats three
times. Then the same "mount: fs type...by kernel"
message appears three times. Then sendmail holds up
the boot process until it times-out.

Any suggestions at this point will be appreciated.
Thanks in advance.
