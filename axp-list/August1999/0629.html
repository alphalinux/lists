Hi,

The problems that everyone is seeing are due to the fact that there are
Halts and SCSI bus resets ocurring when a non-SMP kernel is booted on an
SMP (or SMP-capable) machine.

There should be a boot disk image with an SMP kernel appearing on the
website and FTP site RSN.
