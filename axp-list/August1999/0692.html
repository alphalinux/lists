This is for booting.

I'm assuming your dos partition is sda1 change to match your system

First have the msdos File system built into the kernel then make a dir to
use as a mount point
mkdir /sda1
then
mount -t msdos /dev/sda1 /sda1
copy linload and milo into /sda1

Shutdown the system
boot ,then press the "KEY" F2 to get into setup.

on the menu go to "Utilities", then select "OS selection setup"

Add a new entry
Boot Name: Linux (or whatever)
Boot File: disk 0 partition 1 linload.exe
OS Path: disk 0 partition 2 \
note:(it will complain about OS Path, ignore any messages as this line is
not used but the system performed validation check on the entries, and is
looking for NT stuff)

OS Options: boot sda#:vmlinux.gz root=/dev/sda# (where # is the
partition where the kernel and root are.

Then save the entries, it may complain but its doing NT checking so save
anyway. Set the Linux entry as default, then boot.
