Anyone get redhat-6.0 to install on a alpha (even a ds-10) via serial?

I get close, as the log below shows. I compiled the 2.2.11 kernel (with
serial+serial console support), the updates redhat-6.0 ramdisk. Anyone
have some hints out there?

I also tried appending "console=/dev/ttyS0" and "console=/dev/ttyS1" to
the osflags as well.

The reason I'm trying so hard is a video card only works in the 32
bit slots (or else I get two beeps on boot), and I'd like to try
a 64 bit scsi card in the 32 bit pci slot.
