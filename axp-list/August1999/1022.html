Hi,

I've only been doing this for a little while, so I
expect more experienced hands will have better and
more complete advice, but I thought I'de give it a
try. So here goes.

While having someone else do the install might avoid
the particular problems you can run into with that
aspect of linux, it just puts off the inevitable if
you're going to test the linux waters. Check out
http://www.mainmatter.com/linux-faq-all.html . The
particular issues on the Alpha are: 1. MILO, which
replaces LILO on the Alpha.
http://www.suicidal.org/files/linux/howto/MILO-HOWTO.html
2. Rescue disk: No such thing on the Alpha, but , and
this is why you should learn the install procedure
yourself, using the AlphaBIOS(I've used 5.66-1 and
5.68-1)and the generic.img install floppy can partly
take care of this. 3. Browser: Netscape (Mozilla)
isn't very mature on the Alpha yet, though the KDE
file manager, KFM, is servicable as a WWW browser.
http://www.alphalinux.org is a very valuable resource.
Check out all it's links.
Good luck, and don't worry, not all web sites about
Linux on the Alpha have "suicidal" in thier
address.;-)
