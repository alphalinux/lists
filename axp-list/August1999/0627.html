HI everyone,

I've been working on some new kernels to fix some of the problems we've
been seeing with the 6.0 release on Alpha.

There should be a SMP boot disk appearing on the website RSN that enables
SMP-capable machines (specifically, the DS10 and DS20) to run the
installer. You will have to get the updated RAMDISK image and run the
installer in 'expert' mode to make sure the SMP kernel gets installed.

We're still not able to get any Noritake's to boot w/ 2.2.x w/o the Halt
Code 6 everyone's been seeing. That's my next dragon to slay...
