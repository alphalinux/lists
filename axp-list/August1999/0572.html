> On Tue, Aug 24, 1999 at 05:33:07PM +0300, Imran Siddiq wrote:
> > I am booting from the floppy using root=/dev/sda1 but I never get to the login

> Well there might be, but I never need to use them.
> One other thing that you might be able to try is to boot-up using your floppy
> again with the ramdisk (root=/dev/fd0 load_ramdisk=1) and then once you got
> to the setup, go to the other pty (alt-f2 or alt-f3) you should have a prompt
> there, then just mount the drive and run swriteboot.
>
> Also make sure that you're installing it into the sda1, since i'm assuming
> you are using bsdlabel, that would be bsdlabel a. Now if you're using first
> label as swap (which is most recomended) then you should do root=/dev/sda2
> That's what I use in mine. Just make sure that the label match the root
> device that you're trying to boot.
>
> Hope that helps.
