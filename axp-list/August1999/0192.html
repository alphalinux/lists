Hello all Alpha kernel developers!
Just downloaded the latest 2.2.11 diff from kernel.org, compiled a new
kernel an tell you what:
  This is the _first_ kernel that runs rock solid on my 164SX!

You remember me posting problems with programs that used threads and caused
high CPU load and random processes beginning to segfault? This remained up
until kernel 2.2.10.
But 2.2.11 now runs rock solid! I can finally play MPEG-1 movies with mtvp,
I can use xmms (which used to cause faults too) and so on.

I am so thankful!
Whoever found the bug and fixed it, feel invited for something. You have
(nearly) free choice ;)
Thanks again, great work.

So on the buttom line I can only recommend to everyone to upgrade to 2.2.11
as soon as possible.

It's so great to have a stable system now!
