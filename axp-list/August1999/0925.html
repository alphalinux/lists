IP Masquerading works under 2.0.x and 2.2.x kernels. The difference is
you have to use a different tool for each kernel.

Under 2.0 and earlier, you used "ipfwadm" to create the firewall rules for
masquerading.

Under 2.2, you use "ipchains" to create the firewall rules.

Both can accomplish the same thing though. Read
/usr/doc/HOWTO/mini/IP-Masquerade on a RH 5.2 box for using ipfwadm to do
masquerading, or on a RH 6.0 box for using ipcahins.

I might also suggest reading /usr/doc/HOWTO/Firewall-HOWTO and
/usr/doc/HOWTO/IPCHAINS-HOWTO as well.

All of these have the info you need to setup a masq. on whatever kernel
you please.

Oh yea, this assumes you installed the HOWTO documentation rpm :)
