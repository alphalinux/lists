"Segrest, Bob" wrote:
>
> Greetings,
>
> My CD-ROM writing resource is on an NT based machine. I have found that the
> most reliable way to write non-Windows CD-ROMs is to use a .iso image file.
>
> I have the RedHat 6.0 file and would like to add a couple of image files to
> it. Can anyone tell me how to create a .iso image file ???
>
> Bob Segrest
>
> --
> To unsubscribe: send e-mail to axp-list-request@redhat.com with
> 'unsubscribe' as the subject. Do not send it to axp-list@redhat.com

Install your complete CD tree on a directory on your Linux box, then
type makeisofs -o "name of the new img file, example: newcd.img" -R
"directory where you have the CD tree"
then if you what the CD to be bootable by aboot you have to type

isomarkboot newcd.img /boot/bootlx

ftp the img to your M$ box and burn the CD
