RH6 boot kernel has some problems with NCR53c8xx SCSI controller.
You'd better use booting image from digital.
You can find them in
ftp://mars.iol.unh.edu/pub/linux/images/

up-223 is for one CPU, smp-223 is for 2 CPUs. If you use either of them,
you need to use RH5.2 ramdisk or specify CD-ROM as root.
You can also try 2.2.10.img.

If you still have problem, please inform me.
Thanks.
