Michael Rich wrote:
>
> Is there a file image in the RedHat 6.0 AXP distribution that i can use to
> make a bootable CD from? I'm using some program called:
>
> mkisofs
>
> to make a cd image that I can use to install from, and it seems to support
> making bootable CDs, but I need an image first. Can anyone help?
>

The mkisofs boot stuff is for El Torrito style Intel
booting. I don't think it works for Alphas. In general,
you can make a bootable iso image using the aboot package
command "isomarkboot". See http://www.alphalinux.org/faq/srm-3.html
for info. This, of course assumes you're using SRM as
your system console.
