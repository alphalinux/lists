BTW: nVidia
Don't try it.
I've tried with both the Creative TNT-Blaster and Diamond-v550.
This card-bios contains 32-bit instructions which the AlphaBIOS
x86-emulator cannot handle (it only does 16-bit x86).
The result is a system-hang on boot. Using SRM, the system
will boot, but video-init fails (no /dev/tty0).
