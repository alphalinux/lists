I used to have framebuffer support, but it caused a lot of problems with the
X server, i.e. the machine hung when I tried to quit X, and
Ctrl-Alt-Backspace didn't work, so I'm not really sure if you want to do
this. Remember, your boot parameter from MILO should be something like this:
boot blah:vmlinux.gz root=blah video=matrox:vesa:0x108. I don't really know
if my kernel config file would help you or not.
