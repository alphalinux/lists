Hi,
  I've jus tmanaged to build and am running XFree 3.915 snapshot without
too many difficulties.

(I have the Matrox Mil 2 with 4MB).

   1) You have to edit the conf files to disable the modular X server - it
segfaults for me. Its fine when built non-modular.

   2) I get a corrupted text console when I exit x - mind you I always get
that with X I build myself.

   3) xterms work again - yes if you are running RH6 and have xterms which
do odd colour problems and lynx looks shit in them, building 3.915 solved
the problem.

It took about an hour to build on my LX.
