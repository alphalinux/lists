> So is compaq/API going to concentrate fully on alphalinux, and if so how
> soon can we expect a good browser and productivity platform, or atleast
> emulation that will let us run the intel ones good enough, though the
> thoughts of star office in emulation are almost sickening, its slow enough
> on its native platform.

Web broswers on AlphaLinux include amaya, grail, MMM, Lynx, KFM from
KDE, and mozilla. Once NS gets to release 5.0 you should see an
official AlphaLinux version. It compiles on Alpha already but it's
buggy since it's still under heavy development. Also if you has access
to Tru64 libraries you can run the Tru64 version of Netscape on
AlphaLinux and most any other program like Adobe Acrobat Reader. On RH
5.0-5.2 you can run NS through em86. There is an updated version of em86
to work with glibc-2.1 though it isn't there quite yet.

Productivity on AlphaLinux includes the Applixware Office Suite,
Koffice, KDE, and Gnome. BTW most of the developement of KDE is done on
Alphas.
