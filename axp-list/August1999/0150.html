Setup /etc/hosts.equiv.

Just list the systems in there you want to have access to.

So, on tjsys, in /etc/hosts.equiv, put the names of the
systems you want to have access that machine, one per line.
So, for example, if machine abc is to be access by machines
123 and xyz, on abc in /etc/hosts.equiv:

123
xyz

This is NOT something you should do on a system that is
connected to the external world. It can be dangerous to
allow rsh's into your systems in unprotected environments.
