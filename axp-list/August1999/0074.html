On Thu, 5 Aug 1999, Alvin Tan Siong Joo,Systems Specialist,SCS-CSN wrote:

> Does anyone know or where I could get the correct drivers for the ZLXp-E3
> graphic
> Card and the steps or how to install the drivers to boot in up in X
> environment ?

First check, what chipset this card is based on. Then see, if there is an
X server for this chipset (hint: a lot of different chipsets are covered
by XF86_SVGA server). Check http://www.xfree.org

If there is none, you can still use a frame buffer based X server
(XF86_FBDev). It's probably not the optimal solution, but it might get
your card working. Check out
http://metalab.unc.edu/pub/Linux/docs/HOWTO/mini/Vesafb

In worst case, use XF86_VGA16. It's ugly, but it really should work. :)
