This is for changing the color resolution in Tru 64 Unix. Can it be applied
to Linux?

To switch the default visual from PseudoColor to Truecolor, copy
your /usr/dt/config/Xservers file to /etc/dt/config/Xservers (creating
 /etc/dt/config if it does not exist yet) and modifiy the X Server
startup command in this file to include the options:

         -vclass0 TrueColor -depth xx

 where XX can be 16 or 24.
