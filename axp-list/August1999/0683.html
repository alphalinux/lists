On Thu, 26 Aug 1999, Stephen Hitchner wrote:

> First of all I just wanted to thank everyone for answering my previous question.
>
> The command below worked great....I believe I have completed a successful installation; however, I can't seem to boot the system after the install with the command:
> boot sda8 linux Disk Druid assigned my / to sda8

OK, If sda8 is your root partition then you're going to want something
like this:

boot sda8:/boot/vmlinuz-2.2.5-16 root=/dev/sda8

You should also be able to do this:
boot sda8:/boot/vmlinuz root=/dev/sda8

as /boot/vmlinuz is a sym link to /boot/vmlinuz-2.2.5-16

Starting with 6.0 RedHat has moved the kernel into the /boot directory
(like the i386 port).
