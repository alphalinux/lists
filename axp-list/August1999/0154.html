>>>>> On Sat, 7 Aug 1999 14:29:10 -0700, Richard Henderson <rth@twiddle.net> said:

> personality(PER_LINUX) = 8388608

( = 0x0800000)

> Whee. We've just reset 32bit mode.

> In the meantime, drop `personality(0x800000)' somewhere near the
> beginning of main.

Actually, from /usr/include/linux/personality.h:
        /* Flags for bug emulation. These occupy the top three bytes. */
        #define STICKY_TIMEOUTS 0x4000000
        #define WHOLE_SECONDS 0x2000000
        #define ADDR_MAX_32BIT 0x1000000
        #define ADDR_MAX_31BIT 0x0800000

so, that bit seems to be 31bit mode. But it makes sense to set it
back to what it was.
