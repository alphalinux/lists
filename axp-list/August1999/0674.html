>I've (finally) got RedHat 6.0 installed on my AS 200. Now my question is,
>how do i get it to start up with KDE instead of GNOME?

Depends on how you start X. If you use xdm, then you need to make a shell
script (mark it executable) called .xsession located in your home
directory. Have this execute your desired window manager or environment.

If you use 'startx' to launch X, then you need to do the above to .xinitrc
