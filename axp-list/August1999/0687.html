You can get it from ftp://ftp.digital.com/pub/DEC/Linux-Alpha/em86/
Look in the documentation there to decide what packages to download. Then
install them (as directed there). If you have RH5.2 or a glibc2.0 system,
you're fine. However, if you want to run it on RH6.0 or a glibc2.1 system,
then you'll have to append /usr/bin/em86 (the executable) with the
following each time you run it: env
LD_LIBRARY_PATH=/usr/alpha-glibc20-linux/lib
/usr/alpha-glibc20-linux/lib/ld-2.0.7.so If you have kernel 2.0.x, and em86
complains about the kernel not supporting 31-bit addressing, then you need
to apply one of the available patches. Otherwise, it should work fine.

Good luck,
Praveen
