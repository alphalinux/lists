Uncle George asks:
> Is there a way to change a physical block size to 1K?

SCSI protocol allows for that (you send required SCSI commands to
a disk) and older hardware actually **was** implementing that.
I have still an old WREN, used on NeXT, which is re-formatted that
way because this actually increased both disk performance and its
capacity (less sector gap overhead).

As MS-Windows cannot make any use of that facility manufacturers
**removed** that option from their disks; i.e. you can still send those
commands to your hard drive but they have no effect. The fact that other
OSes are possibly smarter was immaterial. I am not aware of any
hardware which still retains this capability but maybe there is
somewhere some.

I have no idea if (E)IDE specs allow physical block changes. I would be
quite surprised.
