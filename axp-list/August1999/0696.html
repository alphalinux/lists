On Fri, 27 Aug 1999, Xiaozhou Qiu wrote:

> RH6's default kernel image file is not /vmlinux.gz, but
> /boot/vmlinuz-2.2.5-16.
> You'd better boot from floppy:

Why boot from floppy? There kernel is still there and usable, you just
have to pass it the path.

boot sda8:/boot/vmlinuz root=/dev/sda8

> boot fd0:vmlinux.gz root=/dev/sda8 load_ramdisk=1
> Then copy the kernel file to the root:
> $cp /boot/vmlinuz-2.2.5.16 /vmlinux.gz
