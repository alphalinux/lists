Praveen Srinivasan wrote:
>
> Has anybody gotten em86 to work in kernel 2.2?

Sure. I am using it right now that way to run Netscape, so
somebody did. :-)

I think that you are really asking if somebody has a version
working with libraries supplied by Red Hat in 6.0 distribution.
This is totally different question and has **nothing** to do
with a kernel version. AFAIK the answer is no but you can still
change LD_LIBRARY_PATH for em86 programs and load that with
glibc2.0.

There is nothing special you have to do with 2.2 kernels to
run em86 beyond configuring them that way. Module works just fine
or you may configure that as an integral part of your kernel.
