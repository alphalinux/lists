> I guess there will be quite a big market for that nowdays :-)

Maybe. You would have to ask VMware.

If you really want to run some applications that require Intel processor
and billyware then likely the simplest and most reliable thing to do
is to add some cheap Intel box (could be a recycled leftover) which
will use a display and a keyboard of your main machine. There are
various X based solutions for Windows and this includes things like VNC.
The later for sure works on Alpha as I was playing with it some
year ago or so.
