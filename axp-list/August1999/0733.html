I have been trying for several weeks now to get Linux installed
on a Jensen and have lots of experience with other Alpha's. :-(
Still no luck.

You'll need a video card that can be recognized by the SRM
firmware. The following S3-based ISA/EISA cards are absolutely
known to work with the S3 XFree86 server.

Elsa Winner 1000
Number Nine GXE
Fahrenheit 1280

If you are only interested in text you can also add the Compaq
QVision card. No X server but at least the system will be usable.
Rumor has it that some very old 8-bit ISA cards are also recognized
though probably not usable under X.

If you plug a terminal into serial port 1 and disconnect your
keyboard you can at least get to the firmware with an unrecognized
card.

The latest XFree86 (4.0) dumped the S3 server and moved the sparse
bus access code into a more generic source area under the SVGA
server. So it looks like SVGA card (not just S3) that can be
recognized by SRM might work. It's no longer hardcoded to work with
just S3 chipsets.

Look at the docs on www.alphalinux.org for installation instructions.
They are *supposed* to work. I have had no luck with that method or
others I've tried. If you get a good install please let me know how
you did it.

/Michael
