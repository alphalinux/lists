Assuming the code at your site is the code you used,
keep in mind your loop style is a bit too straight forward.

One can change the coding style to substantially increase
the MFLOP rating by taking into account cache size/layout/
register reuse/etc. In fact, one can approach the theoretical
peak performance on many risc systems using code that is
aware of those effects, even with g77.
