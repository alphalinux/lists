On Thu, 05 Aug 1999, Bernhard Gschaider wrote:
> If I remember it correctly (sorry I can't verify it because I can't
> get to www.alphalinux.org right now) there is a way to make the
> Tru64-Netscape-binary run on a Linux machine.
>
> My question: is there a general way to make Tru64 binaries run under
> Linux? The application I have in mind is a simulation program (just
> numerics and file-i/o, no X-Windows).

If your code is static linked to any binaries then it will work straight off.

If it uses shared libraries, then you need to create directory /usr/shlib and
put all Tru64 shared libraries in there. You'll also need to install Tru64's
loader as /sbin/loader

Doing this I can run Netscape 4.08 quite happily under Linux/Alpha.
