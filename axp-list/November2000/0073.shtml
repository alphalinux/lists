and didn't see it.

http://www.alpha-processor.com/products/cs20.shtml
* 2 667 or 883 Mhz 21264's with 4 MB of DDR L2 cache each
* 2.65 GB/sec peak main memory bandwidth
* 2 64 bit 33 Mhz pci slots
* Linux in the boot flash rom
* Ultra 3 scsi on board (symbios 53c1000)
* rackmount 1U (!)

No idea on system price, but the motherboard is rumored to go for $2,388
(so says techweb).

Looks good to me. Not quite a ds-20 killer (1/2 the memory bandwidth),
but definitely a nice formfactor.
