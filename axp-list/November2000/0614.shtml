what is your 'console' variable set to in SRM?
I've seen systems where if the variable is set to 'graphics' and you have a
keyboard plugged into the system, it will disable the serial port.

if you set console to 'serial' with a keyboard plugged in, both will work.
You can also try completely unplugging the keyboard / mouse and see if that
fixes it. Even if the variable is set to 'graphics', without a keyboard it
will default to serial.
