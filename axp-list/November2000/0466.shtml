On Wed, 22 Nov 2000, Padiyath Kumar wrote:

>
> Iam trying to Upgrade the Redhat5.2 to Redhat6.2 on Alphastation
> 500/333
> machine. My cd device is dka400. I wrote the following command:
>
> boot dka400 -fi /kernels/generic.gz -fl root=/dev/scd0
>
> The system reads the CD and writes the information on the terminal.
> Then I was
> expecting a window for the selection of update. But Iam NOT seeing
> anything
> on the terminal - a black screen. Is there a way to see what is going
> on?

I assume you are running a serial terminal.

If this is the case you will want to do somthing more like:
boot dka400 -fi /kernels/generic.gz -fl "root=/dev/scd0 serial=ttyS0,9600"
