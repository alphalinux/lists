This has been covered most of today... bottom line.
use the floppy boot image at:

http://ftp.digital.com/pub/DEC/Linux-Alpha/TESTING/2.2.17-jj/
(floppy may be created on a wintel box with rawrite, also at the link)

and it will find the controller and the install will work.

Afterwards, installing the 2.2.16-3 rpm, will update the stock RH kernel and
everything will be fine. (booting the drive using the 2.2.17 kernel on the
floppy will create system.map errors during the boot process, annoying but
not harmful.)
