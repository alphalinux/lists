They label it the "street" price.

I don't see how a slot B board will fit in a 1U configuration.
Is this UP2K+ the board that will go into the CS20? Also, I
thought cache was on the cartridge, not the motherboard so how
does this new board help with currently available cartridges?
Memory bandwidth is still 2.6GB/sec. Are there now 667/750Mhz
DDR cartridges? The 1U config must be some sort of socketed
processor, maybe "socket B". The configs aren't clear to me...
Also, the CS20 has a single PCI slot available so if that's
what a UP2K+ board is, it's quite limited.
