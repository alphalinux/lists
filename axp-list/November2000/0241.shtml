> Spec wise, the DS20E should be faster.

But it isn't, because STREAM is The Right Way to measure memory peak form
the memory system. The wider bus doesn't get ANYONE much of a benefit, and
STREAM is the proof. While you are right that running your code is a good
idea, the statement that "The DS20E ought to have more bandwidth" is
provably false.
