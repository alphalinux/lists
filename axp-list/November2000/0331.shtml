On the PWS if you go into the AlphaBIOS at boot time (F2) there is an
option to change to the SRM. It doesn't say 'SRM' though. IT says
somthing like (I'm paraphrasing here) 'to boot into the UNIX boot loader
next time'. I have a PWS 433.
When you boot into SRM the first time. To keep it coming up in SRM you
have to set the 'os_type' variable to UNIX e.g. 'set os_type=UNIX'

Other will correct me if I am wrong. Once you set that variable it will
always come up in SRM.
James
