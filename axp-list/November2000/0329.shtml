If it's a PWS600, SRM is already on the box, go to alphabios setup, select
cmos setup
then select advanced cmos setup, the console selection is 2/3 way down the
screen.

To switch back from srm to alphabios, type "set os_type nt" at the srm
prompt >>> and power cycle.
