On Mon, Nov 13, 2000 at 02:35:44PM -0500, Jemiolo, John wrote:
> Question,
> Has anyone combined a bootpable kernel and a initrd image. Then
> successfully load the image over the network into an Alpha System, via a
> bootp load
>

Yup, I've been doing that here at LLNL for quite some time. On old
Miatas as well as ES40s. If you define 'INITRD = my_initrd_file' in
the top level makefile and then do a 'make bootpfile' it should build
you the combination that is usable for net booting.
