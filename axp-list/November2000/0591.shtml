> I changed from AlphaBios to SRM
> this weekend, also installing a new
> 40G Maxtor Ultra ATA 100 to make
> a fresh Linux install, as recommended.
> The Linux install was eventless and the
> system comes up on the Red Hat
> install 2.2.14 kernel. This kernel
> does not support dma, so I wasn't
> concerned to see that the new disk

> "hda lost interrupt"
