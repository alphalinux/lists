The gcc as shipped in Red Hat Linux 7/Alpha is based off the same CVS
snapshot we used for Intel, but contains some critical fixes for Alpha
such as the aliasing bug, the multi-inheritance thunking bug, as well
as some new features such as -mtune support for alpha.

I know at least the following bugzilla IDs are fixed in the compiler
as shipped for 7/Alpha:

  #12803, #17205, #17421, #17433, #18027, #18064, #18339, #18861,
  #19025, #19182
