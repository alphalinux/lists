Wes,

To learn more about the new 1U CS20 from API Networks please check out the
following link:

http://www.aspsys.com/pdf/CS20datasheet.pdf

This should clear up any questions you have. As for the DDR cache and 1U
size I have been reading about on the thread, the answers are:

1U = 1.75 inch height (the width does not factor in) Height = U factor X
1.75 inches
DDR cache is a Double Data Rate cache memory. In laymans terms, it is a
bi-directional memory chip. This means it can accomplish the same speed and
better latency than a system with twice as much non DDR cache.

We have been getting a tremendous amount of inquiries into these new 1U
units. They just announced this new at the Super Computing show as we read
:) Straight from API Networks, the CS20 will come primarily configured with
dual 833 MHz processors. A choice of 667s or 833s in dual or single
processor form may become available in the 1st quarter of next year. For
now it seems to be just the Dual 833s. As for the UP2000+ motherboard, this
new CS20 unit is using a scaled down version of this board. The CS20 will
use socket processors. We don't know if they will be hardwired to the
motherboard or not. Hopefully not so we can turn any single setup into a
dual in the future.
