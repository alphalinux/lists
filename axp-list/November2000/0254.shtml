The AS800 is also known as "Noritake". If you're still using a 2.1.x
kernel, I'd highly recommend upgrading to at least 2.2.17 (the 2.2.18pre
versions are much better still).

When configuring the kernel, it will prompt you for the Alpha system
type. If you type "?", it will show you a list of the Alpha system types
and Noritake should be amongst them. From there, just type "Noritake" and
proceed with the kernel configuration and building in the usual fashion...
