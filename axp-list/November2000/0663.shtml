On Tue, Nov 28, 2000 at 11:38:53AM +0100, Helge Kreutzmann wrote:
> Has someone been able to run XFree 4.01 on a ruffian ?

Yes, I did.

> We tried a matrox as well as an ATI Mach 64 card and both displayed a
> "garbled screen"

It happened to be Permedia2 (ELSA Gloria) in my particular case
but it run without any surprises and problems. Actually I found
on the same machine weird gotchas with 3.3.6 from beta to RH 7
(reported back in bugzilla) while 4.0.1 run like a clockwork.

What colour depths you tried? It does not say in a fragment you
included. Using nowadays less than 16 is probably counterproductive.
Not directly related but a Voodoo3 card I tested in some moment on
another Alpha simply did not work in 8 bits at all and was the best in
32 bits (with 3.3.6, this would be 24 for 4) with some garbled icons
when 16 bits were used.

I do not have this Ruffian anymore so I cannot tell you what would
be results with other cards and XFree 4.
