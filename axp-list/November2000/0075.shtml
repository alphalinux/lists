This is the neatest feature, IMO. SRM will become optional :-) I'm
anxious to see this feature in action, personally. I would probably keep
SRM around on my systems, but I think this will definitely help things a
bit for any server admins that install these that are currently having a
hard time with remote booting different configurations on their current
Alphas.
