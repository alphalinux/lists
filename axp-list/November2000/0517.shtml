Ok,

        I recently got Xfree4 working with my voodoo 3 on an
LX, satisfied with 2d I began to work on getting glide
and DRI going, ala P. Petrakis' mini-howto on the
lista while back. steps I've taken this far are:

1. compiled a 2.4.test11 kernel with DRI and tdfx
modules

2. CVS'd the latest Glide 64 and built it.

3. CVS'd Xfree4 and built it with DRI/glide support

4. built Mesa-3.4

everything still works but 3d is not accelerated.
looking through README.3DFX in the Mesa source I came
across MESA_GLX_FX
so I:
export MESA_GLX_FX=f
like the readme says and now running gears or blender
or and Mesa app gives...
Illegal Instruction(core dumped)
when I go back
export MESA_GLX_FX=disable
it will run the Mesa apps with software rendering

strange eh?

Is this a product of Glide not being up to par on the
alpha?

Is hardware acceleration possible on the alpha?

thanks
