On Sun, 19 Nov 2000, Greg Meno wrote:

> Hello,
>
> I recently purchased a PCI voodoo 3 3000 for use with
> my LX. The card shows it's video BIOS init then SRM
> comes up and a boot dka200 puts me into linux.

I'm running the Voodoo 3 3000 PCI in my LX with XFree 4; the only
difference I can see here is that I'm using AlphaBIOS not SRM.
I did have to get the most recent AlphaBIOS/MILO at the time for it to
boot.

I'm actually currently running with a development build of XFree 4; but it
has worked for a while. I've included my XF86Config; try uncommenting the
'option "XaaNo.....' lines and see if it helps; these turn off individual
accelerations.
