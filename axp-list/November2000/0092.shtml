Good, you are in Single user mode. To mount your filesystem read-write.
mount -o remount,rw /
Then you can mount any of your other partitions, and running bash should
give you a prompt and environment you're familiar with.
Then to start "network services" run this.
/etc/rc.d/init.d/network start
Anything else?
