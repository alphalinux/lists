On Tue, Nov 07, 2000 at 01:54:14PM +0100, zank wrote:
>
> i'm planning to install rh6.2 on an as1000a (noritake), generally the
> same way i did with some as255.
> i found the documentation on that machine a little confusing, so my
> questions:
> 1. are there any preferences using SRM or the AlphaBios?
> 2. if AlphaBios: can i use the MILO for as1000 (mikasa)? (didn't find it
> for noritake.)

You MUST use SRM console for AS1000 (MIKASA) and AS1000A (NORITAKE).

Yes, I know there's an available MILO for MIKASA, but it was a hack and
has proven to be incomplete/unstable under various conditions.

Please, use SRM on these boxes.

Also, note that the kernel under RH 6.2 doesn't work on AS1000A 5/xxx
models (4/xxx are OK). There's fixes in the latest kernel sources
(2.2.16 and later, IIRC), and an available kernel on gatekeeper.dec.com:

        /pub/DEC/Linux-Alpha/Kernels/TESTING/2.2.17-jj/

Good luck.
