> Correct me if I'm wrong, but doesn't Linux already use all the available
> swap partitions in a RAID like sceme? In other words it balances them out
> instead of waiting for one to fill, then starting on the next.

Not by a default. If you have many swap partitions then for the
effect you are talking about you have to set priorities properly.
Which is trivial - 'man 8 swapon' and 'man mount'.
