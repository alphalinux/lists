On Tue, Nov 14, 2000 at 07:46:20PM -0500, Speedracer wrote:
>
> Installing RH 6.2 on a PC 164. Using ARC to load a milo image.
> Couple of quick questions,
>
> 2. Does the latest SRM console for this board allow booting from
> IDE device?

Yes.

> 3. Can I add 2 more simms (both 32mb) or must I fill the last four
> slots. If I fill the last four slots Say 2 12mbs and 2 32s) can I
> then select the 256 bit memory options or do the last four slots
> need to be filled with the same simms?

PC164 memory system has 2 banks. If a bank is to be used, it must be filled with 4 identical SIMMs.

Both banks can be "clocked" onto the memory bus at the same time for
256-bit wide operation, but *ALL* the memory SIMMs must be the
identical size.

If the SIMMs are not all the same size, or only one bank is filled, it
will operate in 128-bit mode.

The mode is selected by a jumper on the motherboard. I can't remember
which mode is which WRT jumper IN or OUT; consult the motherboard
reference guide available at:

http://ftp.digital.com/pub/Digital/info/semiconductor/literature/archives.html
