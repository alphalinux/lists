It isn't, it's a PC164....an OEM board. Don't think it was ever used in a
DEC product.

If you need SRM try:
ftp://ftp.digital.com/pub/DEC/Alpha/firmware/v5.8/alphapc164/

put the pc164srm.rom file onto a DOS floppy, you should be able 'upgrade'
ARC console using that file and it will replace itself with SRM. (I know
this works on AlphaBIOS machines).
