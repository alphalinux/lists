OK, I really don't think that finding a MILO for a Tsunami system is going
to solve your problems. If it's only negoitiating at 10MHz then something
else is wrong. I have an LX with a 2940 in it and it negotiates at 40MHz
everytime, booted from SRM.

If you need to look at the Adaptec BIOS, you should be able to use the run
bios commmand in SRM V5.7 and later (I think), though I have yet to try
this myself.
