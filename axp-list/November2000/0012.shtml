On Thu, Nov 02, 2000 at 10:41:51AM -0500, smo9@cornell.edu wrote:
> I am having problems installing redhat 6.2 on an old alpha server. The
> video card is a compaq qvision 1024/e graphics controller. Besides not
> being able to run the installer in anything but text mode, it can't
> detect any of the video card settings and doesn't seem to be able to
> configure an x server. Does anyone know of any problems with the
> hardware, or if I need any special drivers to get it working?

The QVision support, AFAIK, has officially disappeared from XFree86,
as of the latest release (4.0) and probably happened sometime before
(during 3.3.x). Problem was, there was no maintainer.

If the server is a JENSEN, then try to find the #9 GXE ISA card, as
that was the only one ever supported on JENSEN (but is no longer being
maintained under 4.0, IIRC).

Good luck.
