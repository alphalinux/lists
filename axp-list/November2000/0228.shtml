Jeong Hwan Park wrote:
>
> Thanks for your reply.
>
> Is there any performance penalty when I use only one bank for dual
> processors and I run a two big memory I/O intensive process (like STREAM
> benchmark)?
>

4 DIMM's supply 2.6GB/sec peak bandwidth. Adding
additional DIMM's only gives you more memory to
work with. If you want more bandwidth, buy a DS20E.
Note adding more memory can make a system run faster
as more files can be cached in memory and more
simultaneous processes can run without paging/swapping.

I haven't bothered with STREAM in quite a while. I prefer
my own codes as benchmarks.
