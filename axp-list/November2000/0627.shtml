You can run with >2Gig if you apply the large mem patches:
ftp://ftp.us.kernel.org/pub/linux/kernel/people/andrea/patches/v2.2/

I think also the 2.4 series has large mem support in there. Some of the
dists might be compiling with this in already (RedHat enterprise
kernels?).
