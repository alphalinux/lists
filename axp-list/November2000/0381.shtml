I now have software raid working,
but not without some problems getting there:
I tried the pristine 2.2.16 kernel with
the Red Hat patches and the Red Hat 2.2.16
kernel and in both cases I ran into problems
when booting because I had already made an
/etc/raidtab file. This file should probably not
be there prior to running mkraid.
I had to rename /etc/raidtab to something else,
reboot, reestablish /etc/raidtab, and then mkraid /dev/md0...
fdisk /dev/md0....mkfs...mount....

I'm using the RedHat 2.2.16 aright now.
