Change the network name on the win98 machine to that of your Linux box
and get the IP address from DHCP there. Then use that address as a
static IP on your Linux box. This isn't a big deal since the dynamic IP
addresses from @home have infinte lease time. BTW - this is the workaround
I was given by a tech rep from @home who actually knew what Linux is.
The downside is that every once in a while @home will shake up the address
pool and you have to repeat the process. I've had to do this once in 1.5
years of service so it's not a big deal.

-- Blake
