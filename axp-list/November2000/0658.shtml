|-> Stupid question. I have a DPW 433au. Which image should I use:
|-> Alpha433a
|-> or digitalpw
|->
|-> or am I completely confused?

You've confused me :-) All the images in the digitalpw directory are
for BOOTP and MOP updates, no floppy. The Alpha433a has just the
fwupdate.exe
which is what you copy to a floppy. If I where to guess between the two I
would
pick the later. I'm not 100% positive either way.
 
|-> the machine works okay with a Voodoo 3, so do I need to
|-> upgrade the SRM?

Nope, You would only want to upgrade your SRM if you wanted to run a TNT2
card
or you wanted access to the new run bios command.
