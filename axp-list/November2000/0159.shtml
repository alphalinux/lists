> As for the UP2000+ motherboard, this
> new CS20 unit is using a scaled down version of this board. The CS20 will
> use socket processors. We don't know if they will be hardwired to the
> motherboard or not. Hopefully not so we can turn any single setup into a
> dual in the future.

Let me try this again. The CS20 is not based on the UP2000+
motherboard! The CS20 motherboard is different in size, shape
and functionality. Yes, it uses Tsunami like the UP2000(+), but it has
a different southbridge (ALi vs Cypress), different SCSI (LSI Ultra160 vs
Adaptec U2W), doesn't use SlotBs (the processors are soldered to the board
and will not be upgradable), and has dual 10/100 ethernet onboard.

I think I can speak with a little authority here....
