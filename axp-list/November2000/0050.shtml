  It's a WinModem Tom.
  I doubt you will have much luck with it!

  http://www.idir.net/~gromitkc/20001025a.html

  There has been some progress with making it a "LinModem" , but so far only a binary module for X86 kernels of the 2.2.14 vintage have been released. Company doing it is called Olitec, and if you read French you may find out more at:
  http://www.olitec.com/pci56kv2.html

  The Olitec is another implementation of the same Conexant (Rockwell) chip.
  If source gets released perhaps someone will port it for Alpha, but doing this is not for the faint of heart..

  If you have an ISA slot on the motherboard I would recommend looking at some of the 3COM/USR ISA modems still available. They are true modems, and have jumpers too..
  The 5687 is my favourite.
