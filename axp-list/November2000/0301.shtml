PC 164 came packed very nicely, install has been a bit of a challenge.
Got RH 6.2 installed over the network via FTP, but couldn't get it to boot
from the HD. Could load milo from Floppy and then switch to HD. Hmm
updated ARCconsole to the latest version, as the console would keep
freezing up. The new console still froze up if I tried to check what
hardware it detected. Grrr. Still won't load milo from HD, freezes up
when I try to boot from HD. Still have to load Milo from FD0 and then
swithc to HD.....

device, so here we go. Updated to the latest version of SRM. BANG, no
video. Great. I know the SRM update went OK, because I can type commands
and watch the FD0 light up :)
