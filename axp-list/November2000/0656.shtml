it's easier than you think. Simply copy the following file to a
MS-DOS formatted floppy.

ftp://gatekeeper.dec.com/pub/DEC/Alpha/firmware/v5.8/alphapc164lx/fwupdate.e
xe

insert said disk into alpha running SRM and then type.

>>> fwupdate

It will take a minute then sort of reset the system and start it's firmware
upgrade tool.
