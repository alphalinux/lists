I have MySQL running on my alpha (miata) now:
mysql Ver 10.8 Distrib 3.23.21-beta, for unknown-linux-gnu (alpha)
I didn't have problems with install exactly. I did have do a 'chown -R'
for the /var/lib/mysql directory. It orignally had root permission only
and so MySQL wouldn't work right. I always figured that it was of the
'stupid user' category. Maybe it has to do with the RPM. It could also
be that I am using 3.23.21. Hope that at least gives you the courage to
continue.:-/
