On Thu, Nov 18, 1999 at 01:32:51PM -0500, Charles E. Taylor IV wrote:
> And now, the showstopper. I wasn't able to enter text into any textbox into
> Mozilla. The URL bar wouldn't accept text input. Okay - I'm used to that (I
> usually browse with Mozilla 100898). Unfortunately, "Open URL" wouldn't take
> any input either. The only way I could specify a URL was to do so on the
> command line. Speaking of text boxes, I wasn't able to enter any text
> in HTML forms either. Or in any text box in the preferences dialogs.
> Or - well, anywhere in Mozilla. This makes using it as a web browser
> rather - challenging. :)

I had mostly the same experience, but I didn't have the above problem.
I can enter text into boxes just fine including the URL box. I'm
actually quite impressed with M11 so far. It's still not quite as
usable as Netscape 4.61 (over X from an x86 box), but much better than
previous milestones on Alpha. Not bad.

BTW, I am using a 164LX with Debian potato, so maybe the problem above
has to do with differing libraries?
