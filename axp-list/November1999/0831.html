        You imply one thing, then imply another. I'm confused. Are you
trying to set up a bunch of terminals tied to a single server, or a bunch of
separate machines, or just one machine? Have you already started this and
run into a snag, or are you wondering what will (in general) work?

        If you're looking to set up a single (or a few identical machines)
basic party machine(s), the hard drive(s) shouldn't matter. Nor should the
video card(s) beyond a certain point -- it depends how well you'd like them
to see stuff: a 2MB card will handle 1024x768 at 16-bit color (though I
prefer 32-bit). But this depends what you want to do: 32MB cards tend to be
designed for 3D, making them good for games.
