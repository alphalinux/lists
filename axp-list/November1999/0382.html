Well, first of all, I chose Redhat linux for Alpha
because at the time I got my first Alpha machine,
Redhat was the only one with a distribution for Alpha.
I think that is a big factor, because I've gotten use
to it, and not really in the mood to reinstall at this
point since I have a stable system that's been running
for over a year now.

On the intel side, I've used a variety of Linux
distributions. However, I am a fan of the RPM way of
packaging. It takes a while to know it well, but it
provides all the features I've ever wanted in a
Package Manager. So that is a big factor.

As an aside, I've heard that Tru64 is going to use RPM
in the future? Or do they already?
