You may have a better feel for the data, but from what
you've stated, the data is inconclusive. I don't think
downloading from some server on the net is a good
measure. Why not setup a server on a private LAN. Hook
up to the private LAN, and download from all those
machines. Collect several points of data and build a
graph. If only randomness is playing a factor, then
you should get some sort of Gaussian.

BTW, you don't mention anything about the Alpha... so
how is this relevant to this list?

D.
