> Please let me know if this is a previously solved problem, if I've

The Digital/Compaq Fortran compiler (are you talking about Fortran
unformatted files, aren't you..?) has a switch to read/write
big endian files rather than little endian. It works very well
when exchanging files with AIX, HP-UX, Irix and Solaris.

I suppose that you can enable the same switch on a file basis by
putting some parameter in OPEN , and therefore writing a conversion
program should be trivial under Tru64 Unix.
I am not sure that a Compaq Fortran unformatted file is entirely
compatible with a g77 unformatted file; but I would guess it is.
