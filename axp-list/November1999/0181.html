On Thu, 4 Nov 1999, Ovidiu Motz wrote:

> Did anyone managed to install RedHat 6.0 on an old Jensen machine (DEC
> 3000 AXP)? If yes, could you tell me what boot image and ramdisk image
> should I use with aboot? Thank you.

 If you have a DEC 3000 AXP TURBOchannel system then it's not a Jensen and
it currently does not run Linux due to some unresolved problems. You
might want to visit
http://www.phys.ufl.edu/~prescott/linux/alpha/dec3000-port.html to contact
a guy that was doing some works on the port. The page is somewhat
outdated, though.
