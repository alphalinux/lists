On Mon, 8 Nov 1999, Phil Carmody wrote:

> Shane,
>
> You sound like a man who would know... Where can I get a free
> (downloadable .html/.pdf/.doc/.ps for example) Introduction to alpha
> assembly language. Firstly I find the syntax alien (last asm I did was
> Watcom C++ inlines on x86), then secondly the opcodes look somewhat
> obfuscated. I found something on the compaq site once, which looked fairly
> complete, but it was too much a reference work, not for the novice.

I got hold of a bunch of pdf files when we got our first Alpha, these
include the 164hrm.pdf hardware reference manual and a few others which
discribe the architecture and instruction set. Its all pretty simple and
if you want to play with asm and see how things work it is probably
easiest to just write a bit of C and let the compiler turn it into .s code
which can be examined.

I am pretty sure the pdf files came from somewhere on the digital web
site, you might need to search around a bit, this was covered some time
back on the mailing list too. If not, ping me directly and I will forward
the ones I have to you. Try searching for the 164hrm.pdf file via google
or somewhere.
