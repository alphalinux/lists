You can try this way:

1st- you have to make to diskettes, one (called ramdisk) with the
ramdisk.img and one with the kernel image (I think you can try with the
generic.img)

2nd- When you have the prompt aboot> insert the kernel image disk and type:

generic.gz root=/dev/fd0 load_ramdisk=1

if this is the right way for you, system asks for ramdisk; insert the
last diskette and system goes up.
