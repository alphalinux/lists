Perhaps this will save someone some time.
I found that auto-sense for 100Mbit/sec operation between a DS10 embedded DE500
and a EEpro100 driven card in an Intel box did not occur. I needed to force both
drivers (tulip and eepro100) into 100Mbit/sec mode to see larger than
10Mbit./sec transfer rates. Both run RedHat.
On the DS10 in /etc/conf.modules I used
options tulip options=0,0x03
so that ewa0 is auto-sense and ewb0 is 100base-TX.
On the Intel box I used
options eepro100 options=0x20
