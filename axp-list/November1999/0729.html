> jeff

> >
> > Hi,
> >
> > I have a Cabriolet (aka AlphaPC64 ) and I have both SRM and ARC on the
> > system.
> >
> > The SRM version is: V 4.8-1 Feb 20 1997
> > The ARC version is: 4.49-7 961016.1359
> >
> > I don't know if there is a more recent version of the SRM for the
> > Cabriolet. In any case, I have not found any on the Firmware CDs.
> >
> > Currently, I boot Linux from ARC and Tru64 UNIX from SRM. To set the
> > console back to ARC from SRM change the os_type variable to NT and power
> > cycle the system.
> > To change to SRM from ARC go into the setup option described "Machine
> > Specific Setup" which allows you to change operating systems. It will
> > list Digital UNIX and VMS. Choose Digital UNIX, then power cycle the system.
> >
> > I have run Tru64 UNIX version 4.0E as well as Tru64 UNIX version 5.0 on my
> > Cabriolet. Keep in mind that you need a supported SCSI card like
> > the NCR53C810, a supported graphics card like the ATI Mach64 GX card,
> > and a supported network card like a DE500.
> >
