On Fri, Nov 19, 1999 at 11:58:45PM -0000, Matthew Clark wrote:
> Hi guys - I'm new to this list.
>
> I was wondering if there is anyone on here from the UK? I'm interested in
> prices (& any good/bad experiences) for the Alpha DS20 servers over here in
> the UK..

I'm from London. I don't have a DS20, tho...
 
> I'm also interested in good/bad experiences installing RedHat Linux 6.0+ on
> these servers with RAID 5 configurations.

I'm using 6.0+raid 5 on my box (a ruffian), however I merely upgraded
from redhat 5.2, which came preinstalled, so I can't comment on ease
of installation. Incidentally, if you have the disk capacity, kernel
raid 1 is _far_ faster than raid 5 (I find anyway).

> We believe the DS20 would make a pretty neat (Web) Server for our Internet
> Application - we need >2Gb files so we only really have two options.. one of
> which we're not interested in :-)

Bear in mind that a webserver is not (normally) cpu bound, so be
prepared to invest in decent I/O subsystem. Having said that, you
know your app much better than I do.

Do install all the updates. A lot of 6.1 packages will build from
source, so if you need to get going before 6.1 for alpha finally comes
out, do build strace and pidentd and a few others from 6.1 source
rpms. I never found the 2.2.12 kernel much cop on alpha, but didn't
use their heavily patched version, so YMMV. 2.2.13 has been solid as
a rock for me, however.

Feel free to mail me privately if you want/need any further advice
about what I've done.
