Just a heads-up to Alpha users looking for a web browser/editor. Amaya 2.2 is
out. It builds and runs on the Alpha. You can get the official SRPM from
rpmfind.net. Last time I checked, they didn't have an Alpha build. I've taken
this SRPM, modified the spec file so it takes the RPM optimization flags
instead of forcing -O2, and built it for EV4 Alpha. You can find the built RPM
and revised SRPM on my FTP site at orangesherbert.ces.clemson.edu in /pub/amaya/

Binary was built on a RH 5.x system. It should in theory work on a 6.x
system, though I'm extremely reluctant to move either of my Alphas to 6.x due
largely to the trouble people have heen having getting a stable 6.x system.

If you have an AS200 4/233 or Multia, use the binary. It takes a *long* time
to build on such a machine. ;)
