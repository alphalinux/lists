AFAIK, the only dual ev56 machine is the AlphaServer 1200 aka the Ultimate
Workstation which is still on Compaq's price list. This is the
DS20/DP264's predecessor, with 2 x 21164a @ 533MHz, 4MB cache per CPU, 1
CPU+cache per daughtercard. AFAIK again, API DOES NOT produce the board.
Now, if only your boss can convince Compaq to sell you a motherboard and 2
daughtercards sans CPU....
