I may have spoken too soon. While there is a refernce in my user manual
about 2 programs in the SROM this may well have been refering to Debug
Monitor and ARC or SRM. Here is a quote from the firmware upgrade
procedure for Digital 21164 and 21064 PICMG SBCs.

Section 2.1

          <snip>
          DIGITAL 21164 PICMG SBC and DIGITAL 21064A PICMG SBC
          systems contain a single FlashROM. This can be
          programmed with either AlphaBIOS firmware, for the
          Windows NT operating system, or with SRM console
          firmware for Tru64 UNIX and OpenVMS operating systems.
          See Tables 3 and 4 for compatible firmware and operating
          system versions.
          
          
          
                             Warning:
                                 
            It is not possible to store both AlphaBIOS and SRM
            Consoles on the same system at the same time. Each
            time a console is programmed into the FlashROM it
            will overwrite the previously installed console
            image.
           <snip>

the entire document can be found here:

http://ftp.digital.com/pub/DEC/Alpha/firmware/readmes/v5.5/doc/dmcc_v55_fw_relnote.txt
