To anyone:

I'm trying to load RH Linux 6.0 on an ES40. The problem that I'm having
has to do with the SCSI controller that came with the ES40 which is a
Symbios Logic Ultra2 controller using the 53C895 chip (I also changed
out that card and put in an IntraServer 6100 Ultra2 controller which
uses the same chip). After I install Linux, I try to boot off of the
kernel that comes with the normal distribution (2.2.5-16smp). It then
pauses when it sees the controller and just hangs right after that
saying that the controller has paused for 2 seconds.

I've experienced the same exact problem with the DS10, DS20, and 264DP
systems using an IntraServer 6100 Ultra2 controller using the same chip.
I did obtain a patch and applied it to those models and everything
worked just fine. But when I apply the patch on the ES40, I still run
into the same problem. It just will not continue to load after it sees
the SCSI card.

Any help on this would be appreciated.

Thanks to all,
