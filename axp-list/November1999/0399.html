I had a similar problem when I first got my PC164. The fans would spin up,
lights, camera, but no action. No console and it wouldn't boot.
The problem was a faulty ATX power supply. I replaced it and everything was
fine. You should make sure you're using a 300-watt power supply.
Anything lower, and you run the risk of blowing the power.
