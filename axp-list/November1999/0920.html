I have a DPWS433a/au. I could never get mine to boot from the CD. However, I
did get mine to boot from the SRM using the aboot floppies provided. IF you
are installing an equivalent to RH 6.0 then you need the updated ramdisk.img
floppy from ftp.rehat.com. IF it is equivalent to RH 5.2 then the images are
fine. The 6.0 command is just'> boot dva0' IF it is 5.2 based then you have
to enter '> boot dva0 -fi vmlinux.gz -fl "root=/dev/fd0 load_ramdisk=1"'

I hope that this helps.
