This is a similar box to the AlphaServer 800, You can load and run Linux
from SRM on the 3300. On power-up switch to SRM by typing F2 to get into
setup then follow the menus to advanced cmos setup (setup->cmos
setup->advanced (F6))-> 3/4 down the page is the console selection, select
SRM Unix Then save and power cycle.

Although you can select the correct console, these boxes will NOT run TRU64
Unix or VMS, but Linux will run just fine, and you get the latest pal code..

To get back to AlphaBios from SRM type "set os_type NT" at the srm prompt
">>>" and power cycle.

To install follow the guide at

http://www.digital.com/alphaserver/linux/install_guide.html

The quide is for xp1000, ds10 and ds20, "but" SRM Installs are all the
SAME. just use the generic single cpu kernel.

a note though make sure the graphics card in the box is supported by
Linux/Alpha.
