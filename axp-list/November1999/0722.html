On Fri, 19 Nov 1999, Matthew Clark wrote:

> Hi guys - I'm new to this list.

Welcome!

> I was wondering if there is anyone on here from the UK? I'm interested in
> prices (& any good/bad experiences) for the Alpha DS20 servers over here in
> the UK..

Yep, I'm in Manchester. The company I bought my Alpha off doesn't do them
any more (Transtec - or at least they didn't the last time I looked).
I bought some RAM for my Alpha off Compusys, and I've heard good things
about them.

> I'm also interested in good/bad experiences installing RedHat Linux 6.0+ on
> these servers with RAID 5 configurations.

My suggestion would be to go for an external RAID box; at work we have
some external IDE raid boxes which give out fast (ultra?) SCSI on the
back. I think the cost is about #4000 for a box loaded with 6x 34GB IBM
7200 RPM IDE drives. It just appears to Linux as an external SCSI disc.
We have these running off an HPUX box; although we tried them on a Linux
box and it seemed to be fine with them.
We bought a few of them and a spare chasis.

> We believe the DS20 would make a pretty neat (Web) Server for our Internet
> Application - we need >2Gb files so we only really have two options.. one of
> which we're not interested in :-)

Alphas have stonkingly good memory bandwidth (espeially the DS20) and you
should be able to load it up with a fast (64 bit PCI?) SCSI card (has
anyone tried 64 bit SCSI cards on Linux/Alpha?).
