In trying to get this machine up and running, things just seem to be
getting worse every day.

I install 4 64MB 36 bit simms, jumpered the motherboard for 128 bit mem
bus width, flashed in the most recent SRM firmware, and got very nearly
all the way through the Red Hat 6.0 install before it died with CIA
machine check 630 errors. Thinking it was memory, I put all four simms
through a simm tester, and all checked out fine.

I remember a message someone posted here saying that a flakey power
supply can also cause problems like this. Unfortunately, while we
do have some spare ATX power supplies available here at work, I'm not
about to do the necessary mods (stripping and connecting the black and
green wires together) to make it work on my own box.

Does anyone know how I might check to see if the existing ps is in fact
having problems?
