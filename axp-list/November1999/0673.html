The order for a compile

make sure your in the build tree

/usr/src/linux (typical redhat) I use /usr/local/src/linux as the dir for
my custom builds, that way I can leave the original RH sources alone.

then, make config (or if in x make xconfig)
make dep ; make clean
make boot

Then if you have modules
make modules modules_install

then copy the .map to /boot
and the kernel to /boot (RH6) or to / (my pref)

Combining the commands also works:
make config (or make xconfig)
make dep clean boot modules modules_install
