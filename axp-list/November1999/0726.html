On Sat, 20 Nov 1999, Jiann-Ming Su wrote:

> On Sat, 20 Nov 1999, Rich Payne wrote:
> > Take a look at /proc/cpuinfo look at the cpu model value, that should give
> > you a good starting point.
>
> cpu : Alpha
> cpu model : PCA56
>
> so I would use alphapca56-unknown-linux-gnu?

Probably not, actually. Only the most recent versions of GNU autoconf
understand "alphapca56" as a PCU type. If the ./configure file for Mozilla
was generated by a very recent autoconf, "alphapca56" might work.

However, the PCA56 is a superset of the plain old EV56, so use
"--host=alphaev56-unknown-linux-gnu" -- that's what I use all the time.
