> A UP1000 system with a 600MHz 21264 (Slot B) CPU, 2MB cache, 128MB RAM,
> medium tower, 400W power, 9.1GB 7200rpm UDMA66 IBM hard disk, 48X
> CDROM, 1.44 floppy, 8MB video card, keyboard, mouse, Linux
> pre-installed,
> no monitor, is $3,499.

So now we can see the new price point developing.

The old one was $7000 for one cpu (XP1000) and $12000+ for 2 (DP264).

The new one is between $3000 and $3500 for one cpu (Compaq DS10 special,
above price), up to $8000 for a 2 cpu system (I've seen many vendors quoting
2 cpu UP2000s at this price.)

The supply chain seems to be not moving that smoothly, however. I haven't
heard anyone complaining about being deluged with more boxes than they can
sell.
