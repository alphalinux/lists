On Mon, 29 Nov 1999, Mark Abene wrote:

> I've done some searching, and apparently there are no netrek clients for
> linux on the alpha. Anyone here play netrek? Would it be worth it to pursue
> getting a "sanctioned" client? The intel and ppc linux folks have sanctioned
> clients, why not us?

The various netrek clients do build; I've done it and used them on my own
Vanilla server installation (which also compiles and works. :)

The problem as you point out is that the various servers won't let
"unblessed" clients play, and there are no blessed Linux/Alpha clients (though
there is, I believe, a blessed OSF/1 client.)

FWIW, I'd also like a blessed Linux/Alpha client. However, until then you
can use the "JTrek" Java client. Unfortunately, it's almost a bit too slow
on my 533MHz SX164, so anything less and you'll probably find it unusable.
