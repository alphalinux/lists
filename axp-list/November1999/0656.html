On Thu, Nov 18, 1999 at 07:50:52PM -0500, mikeh@iuinc.com wrote:
>
> I've been getting a problem installing one of my machines.. when I
> do a custom install, and go to select individual packages.. the
> program exits via signal 11.. any thoughts/
>

What architecture? I couldn't use Redhat's install at all on my
ES40s. I had to roll my own on a Miata and net-boot the ES40s from
that, configure the local disk, then rpm --root install all the
packages onto the ES40s. I then install my own kernel (a 2.2.10
derived kernel) onto the local disk and did reboot to boot off the
local drive. After the first one was working I made a tarball of the
FS on the local drive and just unpacked that on the rest of the nodes
on my cluster. As for the Redhat install, it failed intermittently
during the install process. It had a really hard time with fdisk.
