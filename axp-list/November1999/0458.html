alternatively, the quick fix is to do

ldd <executable>

which will list all the shared objects you need. If you have the
libraries, put them in some other directory, and then set LD_LIBRARY_PATH
to that directory path. I haven't tried running x86 binaries on alpha
(yet) either, but if it is a library problem, then this is how I'd try to
solve it.

- andy
