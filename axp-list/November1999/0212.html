Yes, it does matter, but if you switched from AlphaBios and selected SRM
unix then it is set. (ARC is similar)

to switch between os's from srm you would:
set os_type unix (to switch from VMS to unix)
set os_type vms (to switch from unix to VMS)
set os_type nt ( to switch from either vms or unix to NT)

From NT
Alphabios->setup->cmos setup->F6 "advanced setup"->select the desired SRM
console&save
