> Waibel Niki wrote:
> > did anyone test linux-2.3.29 which came out this morning (or yesterday
> > evening)?
> > it was not possible to compile linux since 2.3.22.
>
> I'm running it on my SX164 now... seems ok so far.

Oh yeah, except that it doesn't see my PCI cards... forgot I was testing
it via serial console. :-) Specifically the PCI scan works (i.e. /proc/pci
looks fine) but drivers don't seem to actually find their cards. Maybe
a PYXIS-specific problem... I guess I'll wait for more reports.

Also, halt() isn't exported to modules even though the call path:
  __get_free_pages (include/linux/mm.h:381) -->
  alloc_pages (include/linux/mm.h:365) -->
  BUG (include/asm-alpha/page.h:113)
now calles halt(). So modules which use get_free_pages and friends
will not be loaded.

Back to 2.3.21 for now (sigh)
