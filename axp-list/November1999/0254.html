Hello,

I'd like it if this could find it's way to whoever is in charge of the Red
Hat distributions for various alpha platforms.

You see I've been having problems getting MILO to run and the BOOT disk to
be recognized. There's another guy posting to the axp-list who is having a
similiar problem with unrecognised file systems.

Basically from my point of view what has happened is this. Whoever decided
that there could be a generic boot disk for all the alpha systems was
wrong. With RH5.2 there is a seperate boot disk for all the different
platforms and one ramdisk. In RH6.0 there is a generic disk for the boot
disk and another generic disk for the ramdisk.

This is WRONG. It doesn't work. Please somebody organise proper boot disks.

I think one of the problems is some of the systems don't recognise the file
system being used. I don't have a linux box running at the moment so I
can't even see what's on the Red Hat boot disk.

Is there an official email address for send bug reports like this?
