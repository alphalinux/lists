This same problem happened to me on our ES40s. The problem was that I
put the graphics card in PCI bus 1 when it MUST be in PCI bus 0.
Also, I was completely unsuccessful in installing over a serial line.
You will probably want to use an SMP image, too. I don't think the
generic image will work with SMP machines.

In the end, I ditched trying to do an install with Redhat's installer
and rolled my own distribution and netbooted to single user mode to
set up the local drives and install RPMs. I found the Redhat install
to be completely unusable on the ES40s. Of course, your hardware is a
lot different from what I have.

BAPper
