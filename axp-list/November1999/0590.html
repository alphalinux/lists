On Wed, 17 Nov 1999, Michael Tibor wrote:

> On Wed, 17 Nov 1999, Steve Hitchner wrote:
>
> > Hi,
> >
> > I just installed RH6.0 on my XL266 and X doesn't work very well at all. I
> > startx and enlightenment loads but that is about it (same with
> > afterstep). When I logout the X servers crashes with a segmentation
> > fault. If I reboot right after when the system is shutting down all the
> > services, it says Font server failed which I assume means it crashed. I'm
> > still fairly new to the administration side of linux.
> >
> > I downloaded all the updates listed on alphalinux.org, my question is
> > should I uninstall all the packages that have updates, or just install
> > over top??? I plan on using just gnome so I imagine I don't have to worry
> > about the KDE packages.
>
> To update packages in Red Hat, regardless of architecture, do "rpm -Uvh
> <rpm file>".

I'd be careful with that, even though -U is upgrade it will install
everything in the directory regardless of whether it's install already. If
you just want to update only the RPMs you already have installed use:

rpm --Fvh *.rpm

the -F being freshen.
