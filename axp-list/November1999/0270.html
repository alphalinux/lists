On Sat, 6 Nov 1999, Mark Abene wrote:

> I'm going out on a limb:
> Is anyone on this list using an ISA sound card (like a SB16 PnP) which *isn't*
> locking up their whole machine with frequent use?

I have a SB16 PnP in my LX board; I've not had any problems with
it recently. I do remember quite a while ago (1 year+) that there were
some problems which were bought up when quake used it, but other than I've
not had problems. I configured its IRQ/DMA etc on a PC under DOS using the
Creative labs setup programs - it is not now running in the PnP mode but
instead has fixed values.

> > Does anyone CARE???

Yeh! But its a real sod to track down if your the only guy suffering! Try
and find a repeatable pattern which will cause the lock up - i.e. a run of
a few commands you can run in a script which will kill it - that might
help people try that on a wider range of machines. After all we can't
tell whether its your combination of hardware, the type of things you are
doing or just sods law which is affecting you!
