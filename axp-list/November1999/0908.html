Hi...

I made another version of Milo-2.2 ready tonight :-)

This time things are getting better:

* sample-.config files included
* Makefile and machine dependand code cleaned up
* Configure now lets you choose the machine type, since
* Milo builds against a generic kernel now :-)

A ChangeLog is included.

The only working platform is still Ruffian. None of the others
are compiling since I haven't adapted the Makefile completely.

The latest version 2.2-12 as well as a patch against 2.2-10b and
of course a Ruffian binary can be found at
        http://www.stefan-reinauer.de/Milo/

Bug reports and patches are welcome.
