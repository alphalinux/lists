On Tue, 16 Nov 1999, Secret Asian Man wrote:

> Okay, so I ordered a PC164 board the other day. And we all know the issues
> it has with standard spec ATX power supplies. I've read the archives, but

        We do? I'd never heard of any problems, but I'll take your word for
it. I really wish RedHat's mail archive server was up; they've been out of
the "quiet period" for quite a while now.

        Anyway, I have a 250W standard ATX power supply for my sx164 machine
(Microway Screamer SX; 21164PC). The power button no longer does anything;
I rewired the reset button to be power. As such, I no longer have a reset
button (and no, rewiring the old power button to be reset is not an option
with this case). It came with a 300W power supply.

> I was wondering, does the PC164 power supply picky? Inwin makes a nice
> case, and so does Acer, with 300W power supplies (is this enough? I won't
> be loading it down that much) but I know they're not the greatest power
> supplies (but I'll believe AOpen uses better components)...

        My 250W supply easily powers my motherboard, processor, 32x CD, IDE
HD, Matrox Millennium II, 3Com 3C905b, SB AWE64, and WinTV. The WinTV has
issues, so I rarely touch it, but the rest I do regularly.

        The fact that I use an IDE drive keeps power consumption down, but I
doubt the difference between it and SCSI would be significant enough to
choke a 300W power supply. I don't know how much power my system typically
uses, but it has to be <250W; if a SCSI drive requires more than 50W power,
I'd be just a wee bit worried. ;-)
