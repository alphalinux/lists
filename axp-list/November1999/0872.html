Hello!

I'm not using Alpha as a desktop, but knowing the problems with web
browsers for this architecture I wanted to point out, that there is
a new text-based browser out there. It is called "links" and home page
is

http://artax.karlin.mff.cuni.cz/~mikulas/vyplody/links/

It can render tables (even slashdot.org appears quite decently)
and has a nice user interface with pull-down menus.
Unfortunately, it does not run on Alpha "out-of-the-box" (it compiles ok,
but segfaults), but probably someone with porting experience could
give it a shot.
