The difference you are probably seeing is that the Hitachi machine
(and the other Japanese machines) are vector based large scale SMP
machines. American companies have gone a different direction in
super-computing recently by building MPP machines. The Japanese have
gone the route of optimizing SMP vector machines. Here at LLNL we
have found that the large scale SMP machines have quite a few
limitations that make it so they don't scale as well (at least current
versions, there are some that will fix these limitations in the next
2-3 years). MPP machines have a MUCH higher I/O throughput/CPU than
their SMP counterparts. The MPP machines also allow for finer
scheduling granularity and resource allotment between jobs. Of course,
some applications are much more suited to a large-scale vector machine
than for MPP machines, so there is a place for both types.
