Sigh; now I remember that I've seen a similar probelm with the one (!)
IBM drive I have here.

I had to run the SCSI BIOS setup/config utility and set the drive to slower
speed; it was trying to go ULTRA (40MB/sec) and if I set it back to
Fast/Wide
(20MB/sec) it would work fine.

If you are running SRM, you may need to switch to ARC/AlphaBIOS to get the
SCSI BIOS setup/config utility to run, but once set, you can go back.

Good luck.
