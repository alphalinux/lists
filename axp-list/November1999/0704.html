Matthew Clark wrote:
>
> We believe the DS20 would make a pretty neat (Web) Server for our Internet
> Application - we need >2Gb files so we only really have two options.. one of
> which we're not interested in :-)
>

You have more than 2 choices. Solaris/Irix/AIX/Tru64/AlphaLinux
all support file sizes gt 2GB. Most likely 64 bit SparcLinux also
supports them but I haven't used it. Also, most likely HP's Unix
also supports them, but again, I haven't used one. It's not an
unusual thing to support anymore. There are also patches around
to allow large file support under Intel Linux.
