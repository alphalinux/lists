HP's OpenMail Alternative
     Hewlett-Packard recently ported its OpenMail to Linux. OpenMail does
everything Exchange does, including calendaring support, without inventing
new standards and breaking old ones. (The server portion is not open-source;
you can download only the compiled version. HP offers a Linux client, too,
with source code.)
     The beta version is a free download. The real version will cost money -
although, HP sources say, probably a good deal less than the non-Linux
versions. It also might be free for small sites.
     The problem is that the server is available only on Intel-based Linux
right now. (The company hopes to have a Red Hat distribution version, but
beta testers have gotten it to work on others.)
     According to Richi Jennings, the OpenMail technical product manager, HP
would be interested in doing an Alpha version if interestwas sufficient. If
you want OpenMail for Alpha, visit the OpenMail Web site at
www.hp.com/go/openmail and vote by sending mail to linux@pwd.hp.com.
