Yes. I have Kingston KNE2031+ ISA card in my Jensen which is
NE2000 clone. First time around, I had to put it in windows
machine & check the configuration as reported by configuration
utility supplied by kingston. after that I have been booting with
following command line to aboot
0/vmlinux.gz root=/dev/sda3 ether=5,0x320,eth1
ether=10,0x340,eth0
eth1 is a DE204 set for IRQ5 & ioport=0x320. eth0 which is
NE2000 compatible is set to irq10 & ioport=0x340
