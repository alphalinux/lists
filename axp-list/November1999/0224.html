What does it say about it in /var/httpd/error_log? I've noticed
on RH6.0 (unlike 5.2) it quietly kicks ALL errors to the log file
and just sends a [FAILED] to console. If you want to know what's
happening, look in the error_log. Should have some slightly
more verbose info...
