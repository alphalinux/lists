> From the SRM console, i type :
> boot dva0 -file "generic.gz" -flags "root=/dev/fd0 load_ramdisk=1"

I believe that you want this:
boot dva0 -fi vmlinux.gz -fl "root=/dev/fd0 load_ramdisk=1
root_dev=/dev/fd0"

 - looks like you used the boot disk image name as they Linux kernel name -
they are not the same. The kernel _should_ be vmlinux.gz.
