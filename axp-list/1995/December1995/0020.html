Thanks to the folks who reported these problems. They don't fix all of
the reported bugs, but it is a start.

The following packages are not available on ftp.redhat.com in
/pub/redhat-axp-2.1-beta/updates/RPMS and SRPMS. You should use "rpm -U"
to upgrade your systems. Please let me know if you have problems.

setup-1.2-1 fixes HOSTNAME and PATH problems for [t]csh users
wu-ftpd-2.4-4 uses correct utmp location (/var/run/utmp)
dev-1.3-1 adds /dev entries for stdin, stdout, stderr,
random, urandom
hdparm-1.4-3 contains patch from David Mosberger-Tang for running
on SCSI disks
crontabs-1.1-1 removes /home from locatedb list

Additionally, the following packages are avaialble in the devel/axp tree
on ftp.redhat.com.

ncurses-1.9.6-3 adds /usr/include/ncurses.h -> ncurses/curses.h link
koules-1.1c-2 makes /usr/games/koules.svga SUID root

Note that only packages with bug fixes are in the updates tree. Enhanced
packages will stay in the devel tree. This is to make it easier for people
to get the minimal number of packages necessary to have all fixes installed.

We're on a T1 now :-) so you shouldn't have problems getting any of these
packages. The full AXP distribution is also available for ftp from
ftp.redhat.com if anybody would like it.

Let me know if you have any problems.

Erik
