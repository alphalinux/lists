AS1000 should have SRM 5.8 available. If so you should be able to
do a "run bios x" to access the mylex's BIOS and configure the disks from
there. On a related note, moving to version 3 firmware requires you to have
TWO flash chips. If the board already has v2x firmware it probably does not
have
a second chip.

Mylex does not sell the upgrade set of chips anymore. Coincidently, There's
been a major shortage of flash chips in the industry and especially so with
the chip they used for this now retired product. Which is Intel
N28F001BX-T120. I've found searching up and down the net no equivilent chip
from other manufacturers. Does anyone have one of those semiconductor cross
reference books handy? Supposedly AMD makes an equivilent part. I have in
my experimentation found the same intel chip but just a little slower,
T-150. My attempts to burn the chips ala rom burner have failed and
upgrading from 2x -> 3x has failed as well.

Good luck
