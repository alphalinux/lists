On Wed, Feb 28, 2001 at 10:37:03AM -0800, KirkE@paccessglobal.com wrote:

> The biggest distinction between
> UP2000/UP2000+ and the DP264/AlphaServer DS20 is the Slot B cartridge used
> on Alpha Processor Inc. (API NetWorks, more recently) motherboards as
> opposed to the proprietary CPU cards used in Compaq wares.

Given that exactly 1 company makes Slot B cartridges and exactly 1
company makes the proprietary Compaq CPU cards, I'm not sure if it's
really important that one has an open spec and the other does not.

> DP264/DS20 systems have a 5.2GBps main memory bandwidth aggregated between
> two 2.6GBps ports (256bit each). DP264/DS20 also has more PCI slots (6) and
> all of them are 64 bit. UP2000/UP2000+ uses half as many DSW, so its main
> memory performance is 2.6GBps (1.3GBps per 128bit memory channel).

Note that this is theoretical performance. The performance as measured
by the STREAM memory bandwidth benchmark is about 10% different.

But you did hit the major differences:

1) Higher clock processors available for UP2000+
2) Standard memory in the UP2000+ is cheaper
3) UP2000+ has fewer 64-bit PCI slots and fewer memory slots

And some others:

4) UP2000+ fits in 4U and 3U cases; the DS20E is 5U.

> I believe the XP1000 shares this architecture since it
> has a 256bit main memory port, is Tsunami based and may even use the same
> processor boards as the DS20 (can't verify that).

The XP1000 certainly doesn't use the same processor boards. And it has
only 1 main memory port, so it has a bit lower main memory bandwidth.
