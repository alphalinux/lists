On Tue, Feb 27, 2001 at 02:25:18PM +0100, Reinier Bikker wrote:

> How was this compiled? With gcc you have to specify -mieee explicitly both
> when compiling and linking. With ccc you use -ieee, but -mieee is mapped
> to -ieee.

This is all true. However, the math library will return denormalized
numbers when you don't specify -mieee, which most people consider to
be a bug. Do the new compaq compilers (I know that at least Fortran
has a new version in beta-test) fix this? It's a bit silly to have to
explicitly wrap math lib functions to avoid denorms which will cause
faults...
