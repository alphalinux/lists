Hello all,

I have installed netscape on my Redhat 6.2 / Alpha Power Worstation (au
433).
After very hard search I've found the good way to make it works. In
particular with nis/yp account.
To resume,
I have installed the Tru64 compatibility package, with the 4.76 netscape
packages (the common one and the communicator).
Finally, I had to remove the file /etc/sia/siainitgood and I had to put
this "/etc/svc.conf":

### /etc/svc.conf
aliases=local,yp
group=local,yp
hosts=local,bind
netgroup=local,yp
networks=local,yp
passwd=local,yp
protocols=local,yp
rpc=local,yp
services=local,yp

SECLEVEL=BSD # for backwards compatibility ONLY
###

All is working inside netscape: name resolution is ok, nis account can
use it.

Unfortunately, as I've seen in forums, it is now IMPOSSIBLE to sent any
email !!!
Does Anyone have a solution to this BIG problem ?

Thank you very much.

L. Jacques.
