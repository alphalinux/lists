On Thu, Jan 11, 2001 at 12:07:08AM +0100, Van den Bussche, Egbert Jan HEJ SSI-ISEP-4 wrote:
> I tried booting from the CD using 'boot scd0:kernels/vmlinux.gz
> root=/dev/scd0' but that complains that the file is not a gzipped file! Now
> I'm stuck.

As Red Hat Linux 7 uses multiple CDs, you can not do an install
without a RAM based / filesystem. So root=/dev/scd0 no longer works.
On SRM machines we use aboot to load an initrd. Since MILO doesn't
support this, you're forced to use the 2 floppy boot system.
