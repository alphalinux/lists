On Mon, Jan 08, 2001 at 10:28:52PM +0000, peterw871 wrote:
> Is it true that only SRM is supported with RH7.0?

Officially - yes. That means that if you bought RH 7 distro, you have
problems with milo, not after you booted that thing, and you call Red
Hat support line they will tell you "go away". OTOH a bunch of milos is
included in that distribution and they work, or do not work - as the case
may be, the same as before.

> If so would it be possible to install using milo from an earlier distro?

If you have a working milo on your machine, from the current, old, or
no distribution at all, then you may not want to replace it with another
one in any case. Usually there is no good reason for that. These
things have _nothing_ to do with libraries in the current distro.
They will not even see those.

> I own a Ruffian PC164UX and there is no SRM option.

RH 7 installs and runs on Ruffian without any problems. Just boot your
machine to an existing milo prompt and from there boot from CD. I know
because I did that when I still had this Ruffian around (it is gone).
Only do not try to use milo from any of 6.x distributions. At least for
Ruffian these were broken. I did not test Ruffian milo from RH 7 CD.
