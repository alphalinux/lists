Does anyone know if the following is valid C:

extern int a;
extern int b;

void f(int x) {
        (x ? a : b) = 1;
}

gcc compiles it fine, ccc fails with:

cc: Error: t.c, line 5: In this statement, "(x?a:b)" is not an lvalue, but occurs in a context that requires one. (needlvalue)
        (x ? a : b) = 1;
--------^

GCC contains similar constructs, so I'd think it's portable C. Where can
one submit bug reports for ccc?
