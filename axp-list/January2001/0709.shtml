If there's anybody who wants to network boot linux with initrd support
and don't want to compile kernel for network boot,
try this one.
You could use this for installation etc. without any media like floppies and
cdrom.

Usage:
Apply attached patch file to aboot-0.7a.
# make netabootwrap.
You'll get netabootwrap executable.
then run
# netabootwrap -t netboot.img -k vmlinux.gz -i initrd.gz -a "root=/dev/hda2"
then it will build netboot.img binary. It's aboot+vmlinux.gz+initrd.gz.
all of the arguments are optional.
To build an image for redhat installation, grab kernels/vmlinux.gz and
images/ramdisk.img from
redhat CDROM.
and run
# netabootwrap -i ramdisk.img
if all the files are in the same directory.
And boot it.
