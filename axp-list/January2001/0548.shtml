>On Wed, Feb 07, 2001 at 11:12:30AM -0500, Speedracer wrote:
>>
>> I have a PC164 with 4 slots open, I have 2 64mb chips and 2 32s
>> Will the system accept this odd combination or do all four slots have to
>> be the same? Can I just populate 6 slots? Anyone have any used memory
>> they want to seel cheap?

At 11:32 7-2-2001 -0500, Jay replied:
>
>Banks are 4 slots each on PC164, and the DIMMS in a bank MUST BE all
>the same size (at least).
>

Worse than that: all 8 SIMMs must be the same size. You can only utilize
all eight slots when you jumper the memory path to be 256 bits wide.
(That's on the PC164 I happen to have: not SX or LX.)
The SIMMs I can offer are 16Mb (4 of them) Let me know if they're any use
to you.
