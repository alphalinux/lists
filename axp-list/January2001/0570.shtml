On Thu, 8 Feb 2001, Niki W. Waibel wrote:
> what?!!
> really?
> is the tool available?
> what sort of x86 binaries does it convert?
> have you tested it?

it's called em86. it's about 5 years old by now. it's based on the
technology of fx!32, which compaq included with alpha WinNT, so that you
could run intel winodws binaries on alpha NT. fx!32 was successful as an
emulator (although there is some argument that it contributed to the death
of alpha NT because nobody ported any applications). em86 runs intel
linux binaries in alpha linux. em86 was much slower than fx!32 however,
because it didnt recompile the applications, but instead just emulated,
instruction-by-instruction. typically, it will run intel linux apps about
as fast as a 486 on a fast EV56 machine. for that reason, it was never
very useful. to make it work, you need a binary kit from compaq for which
source is not available. you also need to compile the right option into
the kernel and you need all the shared libraries from an x86 linux box.

> would be great if the source is free!

sorry ... no such luck. there are people within compaq that are working
to get it released, but since compaq owns patents on the technology, it is
an uphill battle with the lawyers. don't expect to see em86 sources any
time soon.
