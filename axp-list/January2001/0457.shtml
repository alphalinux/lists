We were working on this with Matrox, and supplied them with a UP1100 for
testing on.
It seemed that although they agreed to work on it, the concepts of 64 bit
computing was a bit beyond the reach of their people.
After a rather longish while they were effectively doing nothing, so we got
the computer back from them.
If anyone is interested, the file in particular is only one, but an
important one.
In the source trees is a subdirectory called HALlib.
It contains a CPU/machine type specific file called mgaHALlib.a
This is the specific part which needs to be ported.
I happen to know where a copy of the source is still sitting, and if
anybody were to feel up to the task...
