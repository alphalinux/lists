Anything that would work would be SLOW. You may be able to pick up an old
ISA IDE hard drive controller, but those all used Programmed I/O as opposed
to DMA to transfer the data to and from the hard drive, as a result, your
CPU overhead will be quite high. I don't know for sure that it would work,
but IF you can pick up an IDE controller card for ~$20 it might be worth
some experimenting.
