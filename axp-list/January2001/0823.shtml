Rich,

>EV69 soon, really? It's amazing what you learn on these lists :)

Little birdy, could'a been fluffing though. Concrete facts can be hard to
come by, as I have found out the hard way. There have been tales of EV68
cores extended to EV68"E" which was supposed to include on-die L2 cache
between 1.5-8MB (hard to believe the 8MB, but 1.5MB sounds like EV7). Even
Terry Shannon has talked about this (privately anyway). Also, a new die
shrink with a couple of fixes supposedly puts the EV69 at 1.0-1.2GHz, but
this isn't supposed to be available until the end of the year, debuted with
the new, hush-hush chipset to replace Tsunami/Typhoon. I'd be surprised if
you were willing to confirm or deny any of this.

>Aren't they doing an 833MHz 8MB DDR for the DS20E? At least I thought I
>saw a reference to that.

I haven't seen anything for DS20E published, but they are amping up the
ES40E. The Alphaserver report that was just sent today covered that upgrade.
The old timeline for 21264 was that we would see 933MHz and 1GHz+ parts by
now. Is this a yield problem, a thermal problem or the limitation of the
memory controller in the Tsunami/Typhoon chipset to keep up with these
faster processors? I can only hope that the new chipset supports 66MHz PCI,
my oldest UltraSPARC (an Ultra 30) had support for 64 bit/66MHz PCI way back
in 1997. A lot of impressive new Fibrechannel/SCI/Gigabit Ethernet wares
are starting to make good use of all that bandwidth. PCIX support for
GByte/ps slots would be even better (backward compatible) and jump over the
competition.

>Nope, they are actually quite different. All the Tsunami machines (DS20[E],
DS10[L], XP1000, UP2000[+], CS20, ES40 did I
>miss any?) use the same kernel type in Linux - DP264.

Having inspected the NT setup disk, I saw that both UP2000 and DP264 were
using the same HAL (DP264). I didn't want to include the DS10/DS10L/ES40 or
XP1000 since these are different markets and wouldn't be used in the same
roles. I didn't even want to compare them, Tsunami based or not. I don't
think DS10/DS10L ever supported NT (at least publicly), I've never seen an
AlphaBIOS for either of them.
