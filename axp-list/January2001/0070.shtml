I had CLIP with a FORE PCA-200E working in a 164LX. The Fore driver
should be 64bit clean due to sparc64 porting efforts. I don't know
about other drivers. I haven't experienced any 64bit issues with the
userland tools either (this is with ATM v0.78).

In general, setting it up was easy. I simply followed the
instructions:)

IIRC several people reported problems with (Intel) SMP machines and
some ATM cards on the atm-linux mailing list. I can't remember if that
has been resolved. So, be aware, that there might be SMP related
problems with your DP264.

The ATM/Linux is now part of 2.4.0. As Greg suggested, it probably is
a good idea to go with the latest version and try that kernel ...

Maintainer-ship for ATM/Linux recently passed from Werner Almesberger
to Mitchell Blank Jr <mitch@sfgoth.com>. For further info and a link
to the mailing list see: http://lrcwww.epfl.ch/linux-atm/
