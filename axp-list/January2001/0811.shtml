Well, the simpler the better. :-)
I'm not sure that either method will work in all possible PCI
configurations though...
IMHO ideally XFree86 should get all PCI layout info from OS - this
would greatly simplify PCI probing code. In 2.4 all necessary info
is in the /proc/bus/pci/devices (list of all devices with base addresses
and corresponding sizes). Unfortunately in 2.2 kernels /proc/bus/pci/devices
shows only base addresses...
