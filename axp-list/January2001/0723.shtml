>So, my problem is that ARC can not find anymore the OSLOADER.EXE,
>it does not recognize the partition anymore.

I hope you have a backup of your system because you are probably going to
need ARCINST.EXE to reformat the SYSTEM partition if you can't find or copy
the osloader.exe file from floppy. You'll also need to verify that the
correct HAL.DLL and *.PAL (Palcode) files are resident in the same
directory.

ARCDOS can be found in the ARC Development kit and is a sample ARC console
program. Works pretty well too (most of the time anyway).
