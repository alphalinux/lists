Starting with 7.0 on Alpha RH is using an initrd image, so you either have
to pass the initrd image to a MILO that handles initrd images
(initrd=/images/initrd.img ??) or build the ramdisk floppy, probably in
the /images directory of the CD. You can't use the CD as the root
filesystem.

--rdp
