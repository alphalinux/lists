Luiz

In the past I did lost also my NT partition and I did not preceed in it.
I downgrade it as you did to 1.7 with the the result of no booting
NT anymore. After that I noticed some problems with big disks, I upgrade
the rom to version 2.2
ftp://ftp.digital.com/pub/Digital/Alpha/firmware/readmes/dec2000.html
The problem is that the old aboot till 0.5 needs rom version 1.7 and
the new aboot 0.7 supports the new roms (not official)
So I think the need to have rom 1.7 for redhat 6.2 or 7.0 is not there.
What I know is that for NT 4 you need is minimal 1.9 ROM, so if you can
switch back to 1.9 ( the same as you had or the newer on )
I think that if you something under 1.7 on you NT Hard disk you probably
never get NT back.
The Linux install did not anything on your NT HD but probably you lost
some of your settings in your NT ARC menu.

I hope you able to get NT back ( as you had before ) and you see and feel
and maybe smell what is better NT or Linux.

Good luck again

Henry
