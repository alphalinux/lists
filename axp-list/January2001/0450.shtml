On Wed, 31 Jan 2001, Speedracer wrote:

> Whats the best way to do this in SRM? I currently have my SRM set up so
> all I have to do is type boot. Any one else had any upgrade expereince
> with 2.4.1?

P00>>boot -fi partition#/path/to/vmlinuz-2.4.1 -fl "any boot flags that
need to be added"

for example:

P00>>boot -fi 1/boot/vmlinuz-2.4.0 -fl "root=/dev/sda1 max_scsi_luns=16"
