I have also had problems with X since I upgraded (side-graded?)
from RH6.1 to RH7. A partial solution in my case was to
install the version 3.3.6 XFree servers from source. The
source RPM is on the source CD, XFree86-Servers-3.3.6 or
something close to that. After doing an rpm --rebuild, the
binary RPMs are stored in a path that shows up right at the
end of the output from the rebuild. Install all those
binary RPMs, one per server type. Then, do 'Xconfigurator
--preferxf3' to use the XFree 3.3.6 servers.

I may have had hardware problems complicating the RH7 software
issues. I had a Voodoo3-3K in my UP2K system, and today the
Voodoo died. Even in the text console, _and_ in SRM console,
every fourth scan line was blank, with its active pixels
shifted up or down (I can't remember which) one line. It
looked kind-of strange. I was fortunate I still had a Permedia
card, but the RH7 X stuff (4 or 3.3.6) doesn't play well with
that card, either.

Good luck with your system.
