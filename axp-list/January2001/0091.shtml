I have bought memory for my AlphaStation 200 from MemoryX
(<http://www.memoryx.com/>) and was pleased both times. Call them and
describe the memory and they will send it to you, because although they
don't sell memory specifically for Digital machines their true parity chips
will work on AlphaStations (at least they work on mine).
