> -----Original Message-----
> From: Steve Murdoch [mailto:steve@murdoch-technology.com.au]
>
> I am loading RH7 on pc164.
>
> I load MILO and from the prompt try
>
> boot scd0:/kernels/vmlinux.gz root=/dev/scd0

You have to specify the ramdisk image as your root filesystem - try:
boot scd0:/kernels/vmlinux.gz initrd=/images/ramdisk.img
if this fails, try it again, adding "root=/dev/ram" to the boot string.

Hope this helps,
