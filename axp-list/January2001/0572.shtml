I just asked received a reply from Oracle, asking if there is Oracle for
Alpha Linux. Below is the reply:

>>From hotline email:
could you please help me to get the following answers:
I'm currentely investigating into Redhad linux for AlphaServers.
1. I have Oracle for Linux (intel). Is there a special release for Alpha?
2. Does Oracle for linux installed on Alpha support 64-bit features.
3. How about our current license. Articles I found regarding Linux always
refer to the enterprise edition.
4. Redhad AlphaLinux ships a special Oracle 8i enterprise edition ($2500)?!
5. What is the best suited Redhad distribution for Oracle 8i on Alpha?
6. Are all Oracle8i options supported under Linux, or are there limitations,
like with Portal for OpenVMS?
Thanks in advance,
Markus
08-FEB-01 18:13:28
=== Ownership change ===
Passing to specialist.
08-FEB-01 18:14:09
=== Solution/Answer ===
1) there are no Oracle products for Linux on Alpha processors and there are
currently no plans to support this platform
2) no, see 1)
3) There are 2 distributions of Oracle Server: Standard and Enterprise
Edition
see "Getting to Know Oracle8i" in std. documentation for differences
contact your account manager for details
This also applies for Linux/intel.
4) I dont know nothing about it
5) none, see 1)
6) see respective documentation or Metalink availability and certification
matrices, Oracle Portal is a part of iAS bundle which is available on
Linux/inte
