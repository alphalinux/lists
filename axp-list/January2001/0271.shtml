hi ,
maybe try this :

first set your console in srm to serial :

>>> set console serial

>>> init

start your installation with the following flags :

>>> b <cdrom in console> -fi kernels/generic.gz -fl "root=/dev/<cdrom in linux> expert serial"

Install linux.

after the installation boot with :

>>> b <cdrom in console> -fi kernels/generic.gz -fl "single root=/dev/<your root
partition in Linux> console=ttyS0,9600"

after you've got you shell prompt in single user edit /etc/inittab and add :

s0:23:respawn:/sbin/getty ttyS0 9600 vt320

after the line that looks like :

6:2345:respawn:/sbin/mingetty tty6

if you've only got a vt320 on your alpha and no video you can delete all the mingetty lines from 1 to 6 and only
put your s0 line there.

edit /etc/securetty and add :

ttyS0

Shutdown your system and put your console variables right.

Put console=ttyS0,9600 somewhere in your boot_osflags
