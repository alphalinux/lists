Ross A Knepper wrote:
>
> On Thu, 8 Feb 2001, Peter Bech wrote:
> > Has anyone tried wine on Alpha?
>
> this won't work. wine emulates windows syscalls and such, but it does not
> emulate intel instructions. it requires an intel processor to run
> on. em86 will (slowly) run intel linux executables. in either case,
> you're much better off porting or rewriting the application however :)

True.

Dec/Compaq had/has a tool for converting x86 binaries to alpha binaries
I get the impression that it is a decompiler/recompiler type of
technology. It should in theory be possible to get that to work and
combine it with wine to make x86 binaries run on alpha. It should also
be able to work with regular x86 linux binaries.

All we have to do is figure out how to get Compaq to break the code
free.
