Now and forever - there is NO POINT in replacing milo on your machine if
the one you have loads your kernel. If it does not, there is another
story. For example, none of Ruffian milos in all 6.x Red Hat
distributions worked but as long as you are not trying to install on a
new machine from scratch this is even hard to notice. BTW - currently
Red Hat is not "supporting" machines with milos at all. It does not
mean that they will not work but only that it is up to you to fix things
if there are troubles.
