On Tue, Oct 31, 2000 at 12:41:24PM +0100, Elena Ortiz García wrote:

> With the bash command ulimit and csh limit I can do it as root,
>
> but it is impossible if you are a normal user.

What you need to do is either (1) use /etc/initscript to raise the
limits of all processes created by init, or (2) use the pam_limits
module. I think both are described in the searchable archive of this
list, assuming you can find one. Hm, I think it's

http://www.lib.uaa.alaska.edu/axp-list/

pam_limits is described in /usr/share/doc/pam*/txts/README.pam_limits

/etc/initscript is mentioned in "man init", but you'll need to read
the mailing list article to see how the arguments work.
