On Sun, Feb 18, 2001 at 08:31:18PM -0500, Bob Smith wrote:
>
> Could someone tell me if it's possible to disable the on-board SCSI
> controller in order to use a DPT PM2144W Raid controller as the default SCSI
> board?

AFAIK, the "BIOS" of the XL266 doesn't allow disable of any devices.

Use of the DPT from console may depend on whether AlphaBIOS has drivers
for it - I'm inclined to believe it does not.

> If not, is there a work-around that'll let me run it?

Once the Linux kernel is running, then you're only dependent on *it's*
DPT driver; I don't know if it has been "ported" to Alpha, but I believe
it has.

Good luck.
