> Thanks for your help. This has fixed my install problem. RH7 is now
> installed but unfortunately will not boot.
>
> I try the following line:
> boot sda1:/boot/vmlinux root=/dev/sda1
>
> straight out of the manual however no vmlinux file exists in the boot
> directory.
>
> The message comes back Failed to load the kernel.
>
what's in /boot?
(ls sda1:/boot)

> I can boot to text mode by entering boot sda1 from the milo prompt but
> really need the GUI.
have you installed X etc?

if you boot to text mode, and run
X >& /tmp/x.out
do you get X?

if it works, does /etc/init.d/gdm start work?
if so, then you just need to change the default runlevel in /etc/inittab
to 5
