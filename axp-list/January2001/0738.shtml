This is a problem in mpg123. The fix can be found in the archives:
http://www.lib.uaa.alaska.edu/axp-list/archive/2000-10/0141.html

Find the attachment link at the bottom. It's easy to patch by
hand in layer3.c in the mpg123 source, or in xmms, as it uses
the same file. Recompile, and problem fixed.
