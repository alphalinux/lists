You miss the point, Mike: It's increasinly frustrating, that most ports to
Linux goes to the Intel only world.

This is the Alpha list. We, the Alpha users, are missing a lot of applications
that are solely available for Intel Linux. I'm for instance searching for a CAD
package.

Therfore wine cold be an option, but even wine is not able to run on Alpha
Linux because it doesnt emulate the x86 code, it emulates the windows run-time
environment.

We are a lot of serious businesses, that are very willing to pay for good
applications. We did not enter the Linux world because of the free software.

We see the problem as "the hesitance of porting to Linux because the Linux
community resists to pay for software". If a market doesn't provide a business,
there is no market.
