Please help...
I have a 600au (Miata) workstation with RH7.1. I can't seem to be able
to copy audio cd's.

Here's my hardware:
/dev/cdrom -> /dev/hdc using ide-cdrom driver
/dev/cdrw -> /dev/scd0 using ide-scsi driver

Xcdroast will not acknowledge my ide cdrom at all. Does it only support
the ide-scsi driver? And if I put an audio cd into the CDRW, it keeps
saying no disk loaded. I can, however create an iso image from files on
my harddrive and burn a successfull cdr - so this tells me my hardware
is fine.

I compiled the latest koncd-0.9.2-1 from src.rpm just fine and it sees
both of my drives fine, but I get a fifo error with exit status 255
everytime I try to copy an audio cd. The full output from koncd is
listed below.

I installed the rawhide gtoaster for alpha but it segfaults immediately
when I try to run it.

Any ideas?
