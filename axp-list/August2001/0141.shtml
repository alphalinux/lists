On Mon, Aug 27, 2001 at 02:18:20PM +0200, Danilo Godec wrote:
> Hello, folks!
>
> I have a client that requires 'PC-NFS'...
>
> Just what is PC-NFS, what are the differencies to 'normal' NFS and is
> there an OpenSource implementation of the deamon available (a Linux
> src.rpm would be even better)?

AKAIK PC-NFS is the name of a NFS-client software for DOS PC's. For
authentication purposes it uses a 'pcnfsd' on the server that you must run
additionally to your normal nfs daemons.

I have a source tgz in my archive that I can send to you if you can't find it
anywhere. (about 55kb)
