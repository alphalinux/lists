On Mon, 3 Sep 2001, MILLET Mathieu wrote:

> I just buy an Firewire/USB combo card and I would like to make it works
> under my linux (RH 7.1 - PC164LX).
>
> This card is referenced to be supported well under linux for both USB and
> Firewire.
>
> When I get a new kernel to activate the Firewire, I found that I get no
> option to configure the IEEE-1394 part of the kernel.
>
> My question is : is IEEE-1394 yet supported under Alpha/Linux ?

I don't know if anyone has tried, but it's worth a shot :-)

>>From a brief glimpse of the config.in file, the directory isn't even
referenced by arch/alpha/config.in. You need to add this line to
arch/alpha/config.in to even try it:

source drivers/ieee1394/Config.in

After that, you need to enable the Experimental option and make sure that
PCI support is enabled before the questions pop up.
