My theory is that you made a "c" partition that covers the whole
disk and tried to _USE_ it as a partition. Don't do that.

The whole-disk "c" partition is for Tru64 alias DU alias ...
For Linux, you should never use the whole-disk partition. In
fact, if doing just Linux, you don't need a whole-disk partition.
However, if you have one, you must mark it unused and actually
use the other partitions.

Good luck.
