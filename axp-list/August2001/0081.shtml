On Tue, 14 Aug 2001, Michael C. B. Ashley wrote:

> Hi folks,
>
> I am trying to run my AS255/300 using an ATI Radeon PCI card.
>
> Have read that XFree86 4.x supports this card on Alpha, and having read
> somewhere else that SRM version 5.9 and above supports the card, and
> having downloaded the latest firmware (as255_v8_1.exe) from
> ftp.digital.com, I thought it would be easy.

Unfortunatly, at some point DEC/Compaq started versioning SRM over
again. So if effect the reference you see to 5.9 has no meaning compared
against the 8.1 you just downloaded ( 8.1 < 5.9 ).
 
> However, when I boot the machine with the Radeon installed (and my normal
> TGA2 removed), I get no video. Presumably my SRM firmware doesn't
> recognize the card.
>
> Help!
>
> Sorry for what is certainly a newbie question...

You're probably going to need a different card. You might be able to boot
the machine over the serial console and then have X start and init the
video card, but no guarantees.
