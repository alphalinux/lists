My PC 164 had been locking up whenever I passed a large amount of data to it
across the network (This started in with RH7.0): It was so bad that I went
back to 6.2! I had tried everything, recompiling kernels, swapping out the
memory and trying several ethernet controller cards. The solution, or so it
seems is to plug my 100mb ethernet card into a 10mb hub (maybe I should put
a 10mb card in the box?). No crashes, the machine downloads all day. I've
tried this with 3com cards, intel ether cards and realtek cards. So far so
good, we still have some locks ups when doing large network scp's, but the
machine seems more stable running now than it was. I'd like to get it
perfect any suggestions as to what can be causing this?

Running stock RH7.1 with included kernel. No log messages or oopses.

I hope this information helps someone else.
