> RedHat 7.1

>
> (--) SVGA: Video BIOS info block not detected!

> Please specify the correct amount in the XF86Config file. See the file

Hmmm; I don't know why the BIOS isn't being detected, but you can add:

        VideoRAM 4096

to the Device Section of the config file, to prevent it from trying to
size memory.
