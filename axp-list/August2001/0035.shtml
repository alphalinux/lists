Peter Mittermayer wrote:

> hi,
>
> i want to install an ide hard disk (using the onboard controller) in
> addition to the existing scsi disk on a 164lx. is there anything i have
> to pay attention to?

*scratch* well you'll be limited to UDMA2 ie roughly 8Mb/s by using the
onboard cypress controller
Unlike the intel arch, you won't have a bios booting order problem.

If you want faster than 8Mb/s then you'll have to look at getting an
ATA100 pci card. With a 80Gb drive, I can achieve UDMA5 and ~28Mb/s using
a promise ata100 card. With two 80Gb drives in alternative channels on the
same card, I can get ~50Mb/s using software raid0 (striping)
