>> > 0 7 * * * root /backup/reboot.sh >/backup/reboot.log
>
> You do not need "root" before the script. Your box is seeing that, not
> understanding it, and ignoring the rest of the line. Just make sure
> that your putting it in root's crontab.
 
If he's using the system wide crontab (/etc/crontab or files in /etc/cron.d)
then the root is needed. This crontab(s) have an additional user field so the
syntax is correct.

But putting it in roots crontab would work (use crontab -e and enter the line
without "root".)
