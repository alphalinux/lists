Peter,
I'm not familiar with your hardware so I don't know if you need to
adjust any pins - check the manual - you probably don't have to do
anything.

If you're running rh7.1 with linuxconf, it should auto recognize it. The
only thing you might want to do is adjust the mount point in your
/etc/fstab. But it's a pretty safe thing so just try it and it will
probably 'just work'.
This will probably work with any redhat since at least 6.x as well.
