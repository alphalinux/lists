As much as I like the Alpha architecture, a couple of points
need to be made regarding IA64. First, IA64 is _not_ X86 on
life support--it is a whole new architecture with elements
from VLIW and with no practical resemblance to CISC/X86.
Second, everything I have heard about the instruction set
architecture is that it was done by HP, _not_ John Crawford
or anyone else from Intel.
