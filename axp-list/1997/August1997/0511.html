>>> Sheer Panic said:
>
> I've noticed that when I use sound on my Avanti (255), it's beautiful,
> wonderful, and amazing.. but my multia has a continuous hiss, even when
> it's not playing audio. Anyone else have this problem, and what are your
> thoughts on it if so?

First, there's a command line mixer as part of the "snd-utils" packages
that you can find on sunsite, or tsx-11, IIRC. It's a single file, mixer.c,
that builds into an executable that you can use from /etc/rc.d/rc.local, or
some such. Make sure you get the latest snd-utils that's available, as only
the latest will work with the latest kernels.

Second, as has been discussed a while ago in axp-list, one of the "line"
devices needs to have a setting close to zero; this will greatly reduce the
distortion and noise you get out of the UDB soundchip. This can easily be
done using "aumix" which comes with RH 4.2; make sure to run it on a VC and
not in an xterm, as it uses most of the line-drawing chars to look nice.
Simply have something playing in the background, and reduce the settings
of the "line" devices until you start getting less distortion.

Good luck.
