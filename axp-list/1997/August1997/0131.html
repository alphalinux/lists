I suppose Red Hat should answer this, but I suspect a port
of CDE to the Alpha is a long way away if it will be done at all.
Motif under Alpha Linux is a problematic beast at best partly
because of shared library issues: currently, Motif-2.0 version
I am aware of from Red Hat (aka Metrolink) and X-Inside (not
a porno site but an X window develment house) only have static
versions. InfoMagic is working on a static version as well.
The effect of this is CDE for Linux Alpha at this point in
time would probably be

1. A very large in consumer of disk space (Intel client
version uses about 50MB and the development version
uses about 55-60MB build with shared libraries).
2. Prehaps somewhat dogged because of #1
Current infomagic attemps with MWM builds are
rather poor performers statically compiled.
3. Platform specific issues would probably show there
beady little heads to a greator degree than they do
now.

So, for what they could sell it for versus what end users would get
out of it may make it prohibitive to DEC Alpha Market at this time.
