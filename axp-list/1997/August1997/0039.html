>On Sun, 3 Aug 1997 csmith@stoneboro.uucp.cirr.com wrote:
>> It has to do with /etc/termcap, I think the ti= and te= strings.
>> Copying termcap across should make it work the way you want.
>
>I checked the files and they are identical.

It's terminfo, not termcap. Do this:
# infocmp >tmp.$$
# ${EDITOR} tmp.$$
(change the terminfo definition; see terminfo(5) for symbol values)
# tic tmp.$$

Or, just to copy from i386 to axp:
axp# ssh i386 "infocmp" | cat >tmp.$$ && tic tmp.$$ && rm tmp.$$
(Or use rsh, or ftp, whatever.) "tic" unfortunately must compile from
a file, no stdin allowed.
