On 5 Aug, Juergen Koller wrote:
> I have put a SMC Ultra in my PC164 ( Vobis IDE ). Now druing ftp to a fast
> local workstation my PC164 freeze some times if i want to transfer large
> files. Then there is no console switching or ctrl-alt-del. only the reset
> button helps. This was rare on 2.0.30 and is now very, very often on 2.1.48.
> Is this a well known bug, or ist this a feature ?

This is well known - I had this since quite some time on my Jensen,
although you are the first I heard this from on a non-Jensen machine.

I had this quite strong in 2.0.*, and it got a lot better in 2.1.*
(tried up to 42), although I can still reproduce the effect easily with
a local ftp transfer.

The SMC driver itself was not modified (at least in the kernel rev's I
compared), so it must be a problem elsewhere.
