I noticed this on mine when the sound card went through init. I
assume it is simply the gain being set to high. As I don't use external
speakers, and there is much fan noise in this room, the slight hiss from

command line mixer and set mic, line in and volume lower.

Intel), the cards came up with all settings at max.
