Hi!

> I have an Alpha, 500Mhz, 256MB, 2940UW controller, Digital DE-500 10/100
> Ethernet (and onter things).
> I have this reoccuring problem with my system. About once or twice a
> day my system hangs and I get the same message(s) over and over:
>
>
> SCSI: aborting command due to timeout: pid 217360, scsi0, channel 0, ...
> aic7xxx: (abourt) Aborting scb 0, TCL 0/0/0
>

I saw this message when I tried to upgrade my box from IDE-only with the
2940UW. Not just once, tho. The problem was that jumper J30 (system clock
divisor, see user's manual) was set so the board was clocked at 45MHz.
This seems to be too much for the Adaptec.
