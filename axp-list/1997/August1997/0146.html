>>> Stefan Schroepfer said:
>
> Use a newer milo from gatekeeper.dec.com, e.g.
> pub/DEC/Linux-Alpha/Kernels/milo-2029-pc164
> if you have a PC164 board.

Note that there are newer PC164 MILOs there, too, that may solve some
other problems as well. For example:

milo-2031-pc164

contains pre-patch-2.0.31-2 as well as the alpha-patches-2.0.30-0.1, so
the NCR53C8XX SCSI driver supports NCR875-based boards (Intraserver for one);
also, it has Dave Rusling's latest patches that allow MILO to support really
large disk partitions (this is the EXT2 "groups" problem).
