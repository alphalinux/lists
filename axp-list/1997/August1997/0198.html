You conveniently fail to mention which version of a driver is that
and where you got it (not even which of two alternative drivers is that
not mentioning release).

Anyway, current drivers for NCR (ncrBsd2Linux, not "7,8") can be found
at ftp://linux.wauug.org/pub/roudier/; the last changes are from
27th of July. Personally I am using this driver for a long time, both
on Alpha and Intel, and so far it works for me very well. If you happen
to have an Intraserver controller then using this driver is the only
option.
