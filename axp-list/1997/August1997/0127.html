I can't comment on the Zip disk on an Alpha using Linux, I'm in the
process of installing Linux 4.2 on my Alpha. However I can comment on
using a Zip disk under NT V4 on an alpha that may give you a clue to
what may be your problem. Iomega provide a number of utilities for the
Zip drive that cover things like salvaging a disk, formatting etc.
naturally, none of them will install on an Alpha.

The Zip disk is recognised and looks like a standard SCSI device,
removable hard disk. I can read, write and delete files, directories
etc. It works great. But when it comes down to fdisk, disk manager,
formatting, no way.

Has to be done on a i86 system that will run the Iomega utilities.

I hope this provides the answers you want, or need, regards.
