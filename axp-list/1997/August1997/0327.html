I was hoping someone would say that. Have you had problems using the
_rowid variable in selects? That seems to be where the trouble is.

where there is a greater than or less than comparison. Please see
<http://www.obscure.org/slow-net/msql-2.0.1-alpha-linux-rtest-failure.txt>
for the diff on the regression test results that concern me.

Are you even using _rowid in your code? I tend to write SQL that avoids
such constructs, so maybe it will operate without problems for me.
