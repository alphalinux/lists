> Hello!

> I am using a Vobis PC164 Alpha with Redhat 4.2, an I have really big
> problems to get the NIS/YP-Support running.
>
> I can start ypbind without any problems, but if I try to use any
> yp-client I get thousands of error messages.
>
> If I try to login on another shell as root (local passwd) or any other
> person (using yp) I get an memory violation error.
> If I start any other yp-client like ypcat or ypwhich I get the right
> context (for example the whole output of ypcat passwd) of the
> concerning yp-map and the following error message:
>
> yp_all: clnt_call: RPC: (unknown error code); s1 = 0, s2 = 0
> Illegal Instruction (core dumped).
>
> Any help is really welcome! Has anyone similar problems and/or
> solutions?
>
I use the same machine (Vobis Alpha, RH 4.2 with kernel 2.0.30
compiled from the original linux-2.0.30 sources with the
alpha-patches-2.0.30-0.1.gz from Jay, although I didn't have
NIS-problems when using the orig. RH4.2 kernel before upgrading,
256MB RAM, UW-SCSI)
and use the machine as a NIS-client (Solaris machines are master and slave)
WITHOUT PROBLEMS so far.
I distribute passwd, group, amd.home, aliases and a few more
and all these reach the alpha.
Every few hours
ypbind[2270]: unknown: RPC: Unable to send; \
errno = Resource temporarily unavailable
is logged in /var/log/messages ONCE and
the processes running after starting ypbind as shown below are:
rpc.yppasswdd, ypbind (master), ypbind (slave)
I removed the "+" from the files passwd and group as they don't seem
to be necessary and use the order files nis in nsswitch.conf for passwd and Co.

All I did was adding
NIS_DOMAIN=tfa.th-darmstadt.de
in /etc/sysconfig/network and sligthly changing
/etc/rc.d/init.d/ypbind.init:

# Source function library.
. /etc/rc.d/init.d/functions

> # eingefuegt und NIS_DOMAIN in network gesetzt (20.6.97, D.U.)
> # getting the YP-Domainname
> . /etc/sysconfig/network

# See how we were called.
case "$1" in
start)

> echo -n "Setting the domainname to: ${NIS_DOMAIN}"
> domainname ${NIS_DOMAIN}
> echo done
echo -n "Starting YPBIND services: "
daemon ypbind
echo "...done"
touch /var/lock/subsys/ypbind
;;
stop)
echo -n "Stopping YPBIND services: "
killproc ypbind
rm -f /var/lock/subsys/ypbind
echo "... done"
;;
*)
echo "*** Usage: ypbind {start|stop}"
exit 1
esac

exit 0
