Yes this can be done with samba. I have set this up for local printers
on the server running samba, and for printers that are attached to other
workstations that user only lpd. It works fine for Win95 and Winnt, I
have no idea about Apple stuff. If you want more specific info on setting
it up check out the help docs for samba. I found them very usefull, and
full of examples to make the setup rather easy.
