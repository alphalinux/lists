At 10:46 PM 8/26/97 -0400, vbajaj@portal.stwing.upenn.edu wrote:
> Following 4.2 upgrade, there are several minor problems with the
>system (UDB). First, the default XTGA server fails to operate;
>which accelerated server can I use from the Digital site? The
>original 4.2 upgrade version of XFree worked fine, and this problem
>developed only after I installed the updates.

I think you'll find (like I did) that the latest version of XFree moves the
base address of the TGA graphics adapter. I just changed the address to the
correct value (have a look in /proc/pci) but I understand that you can
remove the MemBasse option as well.
