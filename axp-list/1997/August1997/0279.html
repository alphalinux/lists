The unaligned trap is not fatal and can be ignored. Here are a few things
to look at.

Try starting ypbind then using ypwhich to poll it. Also check that your
/etc/yp.conf file exists and is valid. Mine looks like this:

% cat /etc/yp.conf
domain icarus.nis
ypserver icarus.icarus.com

Also make sure your nis domain name is set correctly.

% nisdomainname
icarus.nis

Make sure your /var/yp directory has the necessary things in it, including
/var/yp/binding. The ypbind command creates a file underneath that directory
a la: /var/yp/bindings/icarus.nis.2. That file should be there at least
when ypbind is running.

Finally, look on the redhat site for any updates that may apply to NIS
and apply them.
