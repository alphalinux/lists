>
>I am trying to find a source for an Alpha-powered notebook. Are there
>any available? We would need to be able to run RedHat on it.

I seem to recall reading somewhere about an Alpha-based Tadpole. I think
I ignored the rest of the item when I saw the price :-)

I'm pretty sure that Tadpole makes Sparc-based machines, and RedHat does
have a Sparc release.

RDI makes Sparc-based portable machines that are really incredible
(IMHO). Unfortunately the prices on the current units are incredible as
well (over $20,000 for some).

I use an old RDI "BriteLite", a Sparc based "laptop" (actually it is an
IPX, but I put a Weitek PowerUp CPU in it so it runs at 80 Mhz.). A
"Notebook" it is not, it is more like a knee-breaking "Laptop", about the
size of the smallest home VCRs these days), it IS portable, runs Unix
(SunOS, Solaris or Linux, as well as some flavors of free-BSD) and has
the most incredible active-matrix color display that I have ever seen (my
CRT looks dim by comparison).

Best of all, they are available with 32Mb RAM and a 1Gb. hard drive for
less than $1000.

Just my $0.02.
