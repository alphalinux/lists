Hi

My experience is only with an Alpha Server 2000 (Sable). I booted this machine
from CD once by mistake (it was a Digital Unix CD). by just specifying

boot dka4

on the SRM command-line. It started installing DU then, so I resetted the
machine :)

Three advices that may be worthless because you already did them:

Did you use:
boot dva0 -file vmlinux.gz -flags "load_ramdisk=1 root=/dev/fd0"
to boot?

Did you use the mikasa-s.img image?

Did you try to boot off the cd (boot dka4)
I don't know wheter it should work, but it's worth the try!

Good luck with it!
