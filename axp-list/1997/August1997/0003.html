Hi,

I'd like to upgrade my Jensen Board with 16MB SIMMs. I was told, that
it might be a problem to use modern SIMMs, because they often use a 2k
or 4k refresh scheme, whilst some motherboards aren't able to support
those.

Because the Jensen is an old design, I wonder if somebody knows which
kind of RAM is supported. BTW, the specs I found online say that 4, 16
and 32 MB SIMMs are supported, but I know at least one guy, who has been
able to use 8 MB SIMMs, too.

Greetings, Martin

PS.: I know that I need true parity RAM, aka ECC RAM.
