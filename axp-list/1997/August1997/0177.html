mSQL 2.01 compiles and runs fine on a UDB. As far as mySQL, this requires
a working threads implementation, since it is a multi-threaded server.
The last I checked on the porting status was a few months ago, they needed
glibc-2.0.1 or later, and ran into a few problems with conflicting system
header files. You may want to check the mySQL mailing list archives for
the status of the alpha port.
