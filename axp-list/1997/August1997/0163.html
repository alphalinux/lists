Brantley:

This almost always means that you have an error in your script and it is
not compiling correctly. Go into your cgi-bin directory and run the script
from the command line (e.g. perl email.cgi). Look for any errors that Perl
gives you. '

Also, if you use CGI.pm in your script, you can even enter parameters from
the command line for further testing.

Regards,

Jeff
