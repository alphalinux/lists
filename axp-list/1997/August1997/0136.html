On Thu, 7 Aug 1997, Rick Windham wrote:

>
> On Thu, 7 Aug 1997, Alex Krowitz wrote:
>
> > I'd like to buy an alpha to use as a networked computation server, and
> > don't need a monitor (except during linux installation, of
> > course). Someone mentioned that I might have trouble booting or
> > running without the video card.
> >
> > Does anyone know whether, after installation, I can run without video
> > card and monitor? Thanks for your help.
>
> I'm running a UDB 166Mhz with no keyboard or monitor (card is built into
> the motherboard) and it boots up fine. I belive most Digital systems will
> run with no keyboard or monitor.

I beleive that the Alpha firmware supports running without a video card,
but it is necessary to patch the kernel so it will boot and run that way.
If you are doing that, it would probably be a good idea to enable kernel
messages to ttyS0 with a terminal hooked up to it.
