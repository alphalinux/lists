Is it possible to compile xmgr on UDB (under RedHat Linux 4.2), if so,
where can one get the files that are missing, like ArrowB.h, ArrowBG.h,
ArrowBGP.h, ArrowBP.h, AtomMgr.h, etc. ?

Xmgr uses Motif; I compiled it with LessTiff once and it seemed to work
with few quirks as far as I can remember. You can purchase Motif for
Alpha (around $100); if you don't need to recompile, a statically linked
binary for DEC Unix should work on Linux.
