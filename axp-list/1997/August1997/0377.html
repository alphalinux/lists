You wrote :

/*

Hi!

Does anybody know if there is a X-server for a TGA2 card? At least this is the
name shown by 'cat /proc/pci'. The printed documentation calls it PBXGB-AA/CA
and I found a Digital site calling it 3D30.

Thanks in advance,

Carlos Vidal
carlos@tarkus.se

*/

As far as I know there is no support for those cards in any existing server.
They are also shown as ZLXP-E2 under console mode, and only the ZLXP-E1 cards
are supported (on multia/UDB and AS200, eg.).
Unfortunately, it seems there is also little luck to get this hardware suported
some day.
What you can do is to go get a supported PCI video card, install it and
accordingly set up your kernel.
Good luck !
