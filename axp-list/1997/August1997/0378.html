James Johnson <jamesj@chesapeake.net> wrote:
> Logical parity fools intel machines that are set
> up for ECC into thinking that they have ECC memory, but this doesn't work
> for alpha's.

Not quite. Logic parity will not work for ECC on either Intel or Alpha
machines. It is only useful for machines that do parity but not ECC.
AFAIK, all current Alpha machines use ECC.

Intel Pentium systems using the 430HX (Triton II), and Pentium Pro systems
using the 440FX (Natoma), 450GX, or 450KX chips are capable of doing ECC,
if you have 36-bit SIMMs (not logic-parity), and if you enable ECC in the
BIOS.

Alpha systems do the correct thing (so to speak) automatically. The
correct thing being ECC, of course. Thus true 36-bit SIMMs are required.
