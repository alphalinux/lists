I can't tell from your e-mail what distribution you are using, but the
problem is that your install script did not install the same kernel image
that it booted from.

You need to get a kernel, with advansys support compiled in, onto the
hard-drive and configure Lilo to point to it.

My advice would be to either re-install with a RedHat distribution
($3.99 from www.cheapbytes.com) or compile a kernel with the proper scsi
drivers on another Linux machine and then install it onto your machine.
(This is non-trivial if you are not familiar with Linux. You'll have to
boot with the boot.adv disk, get to a bash prompt, mount the scsi drive on
/mnt, get the new kernel onto the hard-disk using ftp,nfs,floppy disk,
or ppp or anyway you can, configure /mnt/etc/lilo.conf, then run lilo -r /mnt)
