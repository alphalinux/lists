Alex Krowitz wrote:
>
> Does anyone know which boards accept 64M SIMMs? I have two 64M SIMMs,
> but I had a hard time finding low-cost (Intel) motherboards which will
> accept them. If I can find a compatible low-cost Alpha board without
> the cache bottleneck, I'll buy that instead.
>

Are there any Alpha boards accepting *TWO* SIMMs?

I think 64MB SIMMs wouldn't be the problem at least if they
are fast page mode with parity.

But you need either 4 of them (for 128Bit memory access) or even
8 (for 256Bit memory access on some boards).
