Well the only real problem with GNAT is that you need GNAT to compile it...
I'm considering to do the port myself - I really need it.
As you said, we'd need cross-bintools. Other question is gcc itself -
if we're going to do new port the compiler should IMHO have latest patches for
alpha. We also should collaborate with ACT and get a prerelease for GNAT 3.10.

There is actually an official GNAT mailing list at chat-request@gnat.com,
but we could make gnat-axp-list although (I have admin access to a majordomo
server)
