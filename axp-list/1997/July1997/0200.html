Like all packages, you have to locate the <package>.src.rpm
file. Then:

rpm -ivh <pacakge>.src.rpm

Will install the source in:

/usr/src/redhat/SOURCES

Along with any seperately applied patches. If you want and
idea of how these patches were applied and the source was built under
RedHat, check /usr/src/redhat/SPECS for the spec file used to build
the binary version of the pacakges. Otherwise, I am personally not
aware of anyway to just extract the tar ball itself from the source
package (though I am sure there is a way).
