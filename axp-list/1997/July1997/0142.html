I have a 166 mhz UDB running NT 4.0 at the moment. The system has 64 meg
ram and 2 gig xtrnl drive. It runs very well, as to be expected. It is
probably comparable to a 90 or 100 mhz pentium system. I am very pleased
with its performance.

I am currently going to remove NT and run Linux since I now have a Intel
box to run NT. From what I've seen on other UDB's Linux is a much better
choice of OS's to run on the UDB.
