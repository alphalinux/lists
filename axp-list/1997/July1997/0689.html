Ferdy,

I don't know much about firewalls but I did read the manual that comes
with RH 4.2. It explicitly states modules are not supported. You must
use a monolithic kernel. I know some people have gotten modules to
work by using previous versions/patches of the relevant source code.
If you have those fixes great, but if you're using the standard kernel
source from RedHat, I don't think inserting modules works. Note this
is a Alpha specific problem.

Might try building your options into the static kernel. The other thing
is to figure out how to fix the problem. Someone on the list may have
suggestions.
