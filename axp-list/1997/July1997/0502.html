I was able to get xcdroast 0.96 running last night, but I haven't hooked up
the CD-R yet to actually try it. It compiled mostly fine, except for
two problems, one with the included "fdisk" which used _llseek, which
seems to be missing on my machine (probably on purprose, as _lseek takes
a 64 bit number, so llseek isn't necessary) and a badly set up vararg
function. For the first, I wrote an _llseek that converted the
arguments and called _lseek, and just fixed up the second.

If someone else has this working, they should probably send diffs to the
author.
