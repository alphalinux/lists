>>> "Steven St.Laurent" said:
> I installed XF86_TGA for the UDB w/Mac 14" MultiScan (supports SVGA & VGA @
> 800x600)
>
> When setting up the XF86_Config if i leave it unchanged, it clears the
> monitor (looks like its almost working) and then X server shuts down with
> the only error being No Valid Modes Found... If i edit anything it dies
> exactly at that modification

You must have edited in the monitor's specs, right? Because that error
message means that the Modelines you provided were out of the range of the
monitor's capabilities.

There's a set of working Modelines as part of the (old) Xtga README on:

gatekeeper.dec.com:pub/DEC/Linux-Alpha/X11

You should be able to find one in there that will work. If you're using the
new XF86_TGA from XFree86 release 3.3, IIRC, it is capable of arbitrary (?)
Modelines, so you'd just have to find one that fits with the monitor...

Good luck.
