>Has anybody ported Hylafax to the alpha running linux?

I have Hylafax-4.0pl1 running on an UDB running RedHat-4.2
It uses an Intel pcmcia modem to send the faxes.

You should compile and install tiff-3.4beta035 (I am not sure if there is
a newer version) before you compile and install hylafax.
You also should upgrade to ghostscript-4.03 + fonts ( I am not sure if
ghostscipt-5.0 works) if you plan using M$ Clients with WHFC.

Do not forget to change the ghostscript-fonts path during the
hylafax configuration. (/usr/share/ghostscript/fonts)

If you plan on using pcmcia modems in an UDB (like I do) I advise
to add a "sleep 5" in the /etc/rc.d/init.d/pcmcia script otherwise
"faxmodem ttySx" (which I added to the /etc/rc.d/init.d/faxq.init script)
starts before the modem initializes correctly.
