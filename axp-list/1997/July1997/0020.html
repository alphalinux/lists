There is a PDF version of the UDB manual.

Check out:

http://www.brouhaha.com/~eric/computers/udb.html

Quote:

Documentation

The UDB only comes with a few sheets of drool-proof paper showing how to plug
in the cables. More technical information is available in the 116-page manual
"Multia MultiClient Desktop Service Information", DEC part number
EK-MULTS-IN.C01. This manual is fairly pricey (about $85 in the US).
Fortunately it is now available in machine-readable form:

udb-man.pdf, Adobe PDF format for Acrobat, 2.3 megabytes
