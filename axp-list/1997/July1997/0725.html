> Before I go about re-inventing the wheel, I would like to know if
> anyone out there has made a rescue disk set for their Alpha box before?

In a sense. I was booting struggling Alphas from a floppy (with a root
on CD if hard disk was messed up) and also just from CD; the later method
does not seem to work on some boards, so YMMV. Milo documents describe
quite clearly how.

> Rescue disks are not too hard to build for Linux/Intel,

This aspect seems to simpler on Alpha so likely nobody bothered.
RAM disk and something like YARD should work as well. You may have
a hard time fitting some reasonable set on a floppy, or two, as
RISC 64-bit executables take more space than CISC equivalents.
