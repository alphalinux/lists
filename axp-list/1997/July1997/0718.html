I've had a PC164 about a month now and it was unstable the first two weeks.
That was due to the paging problems. After using milo to set MEMORY_SIZE
to 127 for a 128MB system and 252 for a 256MB, it stablized. That was my
biggest problem with it.

I'm working on a set of technical applications that routinely consume
100% of memory, typically 200MB of allocated memory, 1GB+ file accesses,
but no significant network access. I use a 100Mb tulip adapter to NFS
mount my SW tree and have no problems with the system. For the money,
I consider them a best buy.

The learning curve is an issue concerning boot procedures but after
that, if you pay attention to what's going on in this list and probably
the newsgroups, things go reasonably smoothly.

If you don't have Linux experience, I'd hesitate to recommend it but since
you've been running another Linux box, it shouldn't be too bad.
