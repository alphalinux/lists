hi rick,

go to the SRM console

at the >>> prompt, enter: >>> show memory

and see what it says ( xx Meg of System Memory)

_then_

at the >>> prompt, enter the following command: >>> MEMORY

if no error occurs within 30 seconds, there are no problems with the
memory

if an error occurs, then it is a process of exchange, starting with the
cache SIMM.
