Going to the web wanderer's list, there is ghostscript 5.01 and gv, which
is a viewer that together function like acrobat.

ghostscript 5.01 is at http://www.cs.wisc.edu/~ghost/index.html

gv is at http://wwwthep.physik.uni-mainz.de/~plass/gv/

For encrypted PDFs, check out

http://www.ozemail.com.au/~geoffk/pdfencrypt/

which has the replacement pdf file and an example encrypted file (which I
just viewed with gv on my UDB :)
