Before I go about re-inventing the wheel, I would like to know if
anyone out there has made a rescue disk set for their Alpha box before?
And if you have, how did you go about doing it?
Rescue disks are not too hard to build for Linux/Intel, as I just
finished doing so for my own systems. Of course, as soon as I got them
working fine, it dawned on me that I need a rescue disk(s) for my UDB as
well, if I wanted to be complete. And that the Linux/Intel rescue disks,
even the rescue disk HOWTO and rescue disk creation program (YARD)
would not do me a lot of good with the Alpha. I looked on the Linux/Alpha
(FAQ) page, but I did not find anything directly helpful.
I don't mind going through and figuring out how to make one,
getting the interactions with milo and the srm correct. I just don't want
to waste the time if some one else has already figure it out.
Basically, what I am looking for is a way to set up a floppy or
two, so the system could boot with out any help at all from the hard
drive. That way if the hard drive gets trashed, or dies and has to be
reformatted/replaced I can boot Linux on the Alpha, and restore its
filesystems from my backups. (Yes, I do make those on a regular basis).
So, anyone out there with rescue disks? Or do we all trust our
systems not to crash? ;) Thanks.
