On Jul 7, Stefan Kebekus wrote:
> Does anybody know whether the motherboard of an AlphaStation 255 has
> sound support built in?

Yes, it has.

> There seem to be connectors for microphones and
> loudspeakers, but I do not have any technical papers. If so, what kernel
> parameters should one set in order to use the sound hardware?
>

MSS/WSS sound system, port 0x530, irq 9, dma 0 & 1. FM synthesizer and
MPU-401 MIDI interface are not supported in the hardware.

The sound system is full duplex capable, but if you use 2.0.x you need
to patch the kernel to get full duplex support (otherwise it even won't ask
you for the second DMA channel, so say 0 when asked about the DMA).

You may wish to read ftp://genie.ucd.ie/pub/alpha/kernel/README.sound
for more details.
