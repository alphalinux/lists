I don't know this machine, but I guess the same method will work on
these...

> EthernetCard is Tulip Card（DECchip21142）.
> I used "RedHat4.2".

First, point your web browser at the URL given above and click on "driver
source code" (alternatively, enter
http://cesdis.gsfc.nasa.gov/linux/drivers/tulip.c in the URL field right
away) and save the file locally. Then, copy it (tulip.c) into the
drivers/net directory of your linux kernel source tree (there should
already be an older version of this file. Just overwrite it).

In the linux kernel configuration, you should activate "DECchip Tulip
(dc21x4x) PCI support" (somewhere in the "network devices" section).
Re-compile and install the kernel.

Then, you have to change your boot configuration (which will be
/etc/lilo.conf for your intel machine). Sorry but I can't tell you right
now /what/ you have to change -- I can't access my Miata and something
seems to be wrong with RedHat's search engine for the mailing list
archive. If noone else can give you the answer, ask me and I'll tell you
when I'm in front of my Miata again :)
