I'm using IP-masquerading from a Win95 box thru the UDB via PC-card modem to
Netcom (even as I type/speak). The insmod that came stock with RH4.2 didn't
work for me, so I replaced it with the insmod binary from 4.1. I have to
strip the modules in /lib/modules/2.0.30/xxx (partuclarly when xxx is
pcmcia). I compiled pcmcia-cs-2.9.5 from sources at hyper.stanford.edu. I
just made sure I had ip-masquerade compiled into the kernel and set up the
network ppp interface & routing thru the RH netwrok config tool in X. After
that it was just doing the ipfwadm commands right )which I'm not sure I've
done but at least it works).

Does anyone know how to use IP-masq with X remotely? (X tries to connect to
the UDB rather than the machines behind the wall.)
