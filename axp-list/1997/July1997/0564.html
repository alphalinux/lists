We've got a microway screamer 500MHz machine with redhat 4.1 (Vanderbilt). I'm trying to get
amd to work on it. I installed from the cd with the command

rpm -i amd-920824upl102-6.axp.rpm

but I get

cannot set owner and group for /.automount - No such file or directory

The files are installed, but amd goes in an infinite loop. I tried to
invoke it manually with the command

amd -l syslog -a /amd -r /net /etc/amd/amd.net

where /etc/amd/amd.net is

/defaults opts:=rw,intr,rsize=8192,wsize=8192;type:=nfs
sources rhost:=hoggar;rfs:=/usr2/sources

and I put a link /usr/sources -> /net/sources. It also goes in an
infinite loop. A direct mount works fine.

Is this a problem with redhat's distribution or with amd on an alpha?
The setup above works perfectly on intel boxes.
