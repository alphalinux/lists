I have already tried the following:
failsafe floppy boot -> no keyboard response
removed all hardware except floppy and video -> no keyboard response
switched video cards -> no keyboard response

Well, it does look like your keyboard controller is dead. This has
been known to happen if you hot-plug it---apparently the UDB keyboard
controller is much more static-sensitive than the average noname intel
motherboard. If this is true, it's a real bummer---the controller isn't
socketed, so it is at least a solder job, or a motherboard upgrade.
