On Mon, 30 Jun 1997, Nikhil Nair wrote:
> Is there any likelihood of being able to run a DOSEmu with EM86-like
> run-time compilation of DOS apps?

As I understand it, that will never happen. DOSemu depends on Intel
features that are not likely to be emulated by em86, which is designed for
Linux programs. Even if it happened, I'm sure performance would be abysmal.
However, I'm told a platform-independent Intel 386 emulator called "Bochs"
works pretty well, and can even run Win95 (in a kinda-sorta way :). And it
looks to be getting better. The webpage is:

http://world.std.com/~bochs

Hope this helps!
