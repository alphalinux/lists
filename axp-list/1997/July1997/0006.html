I would like to get xinetd running on the alpha. It is by far a
superior form of inetd. I can get it to compile properly on an alpha
pc164 running RedHat 4.0 with a few warnings, but when I run it I get
this error:

xinetd[24404]: CRITICAL: {init_common} Initialization of conf timer
facility failed. Exiting...

I think the problem is based on the 32 (x86) to 64(alpha) bit integer.
I am interested in working on a port for this (if anyone else hasn't
already!) Where would I start on the conversion? Is anyone else
interested in helping or offering advice. Any assistance in this matter
would be very helpful.
