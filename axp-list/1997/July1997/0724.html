Ryan Kirkpatrick <rkirkpat@nag.cs.colorado.edu> asks:
> Before I go about re-inventing the wheel, I would like to know if
> anyone out there has made a rescue disk set for their Alpha box before?
> And if you have, how did you go about doing it?

I use a rescue ZIP disk. It was really easy to create; I just did a
somewhat minimal install of Red Hat 3.0.3 onto a ZIP. I haven't got around
to doing a new one for 4.2; maybe I should. But I've never gotten any
kernel version newer than 1.3.90 to run reliably for more than half an hour
anyhow. I get tons of those "unable to handle kernel paging request"
messages.

Everyone tried to convince me that I had bad hardware, but 1.3.90 still works
great for me, with no paging problems or oopses. Nowadays it seems like lots
of people have this problem, and it is reported that it is being worked on.

Cheers,
Eric
