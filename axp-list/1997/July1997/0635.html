>i have a 83megabyte kcore file in /proc...does anyone know what this file
>is; why i can't chmod it; and why it keeps increasing in size?

It is the image of your RAM as seen by Linux. It is a virtual file and is
not "real" (as is true of all the files in the /proc directory).

It is a normal artifact of the operating system and is nothing to worry
about.
