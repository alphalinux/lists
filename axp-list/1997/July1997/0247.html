One warning: In my personal experience in setting up a number of PC164
based machines with NT is that the ARC utilities for making and setting up
partitions are not reliable.
If you want to use the NT tools, then the arcinst.exe on the NT disk under
\alpha\arcinst.exe is the only one that I found that can be trusted.
OTOH fdisk for linux works fine for me too. Bumping up freepages before
running mke2fs is likely advisable as well.
