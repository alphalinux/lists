Hi, I have recently installed RH4.2 in our newly bought DEC PWS 433a
(miata). After running the bench mark test against the previously
installed NT, we found that floating point operations drop drastically
under Linux.

Here's the benchmark figure for NT and Linux on our AXP:

BYTEmark (tm) Native Mode Benchmark ver. 2 (10/95)
Name NSrt SSrt BitF FPEm 4ier Asgn IDEA Huff NNet LUDe INT FPU
WinNT 5.03 9.14 6.34 4.48 5.08 7.22 4.89 6.79 9.26 7.96 6.098 7.210
Linux 4.45 5.89 6.30 3.80 1.71 7.07 5.66 4.73 1.93 3.98 5.309 2.360

Key:
NSrt = Numeric Sort Asgn = Assignment
SSrt = String Sort IDEA = IDEA
BitF = BitField Huff = Huffman
FPEm = FP Emulation NNet = Neural Net
4ier = Fourier LUDe = LU Decomposition

The INT and FPU index are based on (90 MHz Dell Pentium = 1.000).

Note that in Intel platform, Linux performance in FPU is lower than in
DOS/Win/WinNT, but the INT performance is better than in DOS/Win/WinNT.

I wonder if anyone who has done similiar comparisons could comment on the
results. If anyone has found any performance change when switching from
NT to Linux, I would like to learn about your experience.

Thank you!
