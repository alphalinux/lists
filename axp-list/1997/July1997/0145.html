>Third, does anybody know whether it is possible to install a redhat 4.2
>and WinNT on the same IDE DISK ?

Here are the answers.

No problem to install Windows NT and Linux on the same disk.

On the FAT partition make a directory /liniux and put there the milo and
linload.

Add a new startup entry and install Linux on the partitions you have
reserved for this.

One hint: partition your disk with the RH fdisk after you have setup a FAT
partition for booting NT with the programm ARCINST.

Reserve a extended Partition for NT.

Ther memory is only a problem at Install time. BUT BE AWARE: you must have
the same SIMMS in the fist and second bank. Don´t mix brand or size

get Milo and Linload from ftp.quant-x.com/pub/LINUX/PC164/linload_1.5 and
Milo-2029-pc164 witch should work out fine.

If you have any problems at installation time type in at milo prompt "set
MEMORY_SIZE 32" (cas sensitive) and your problems should be solved.

also don´t forget to remove the jumper beside the memory bank.

hope this helps
