NvRAM support
added by Richard Waltham <dormouse@farsrobt.demon.co.uk>
      The latest
version of the driver is able to detect and to read
      the Serial NvRAM
of Symbios compatible and Tekram DC-390/U/V/W boards.
      Patches for
stable linux-2.0.28/29/30 (27?) are available here.
      When this feature
is enabled, for each board using NvRAM, the driver 
      reads host and
targets SCSI parameters from the NvRAM.
      The boot order set up is
supported for Symbios compatible boards 
      if at least one board uses
NvRAM. Mixing Tekram cards with Symbios 
      (compatible) cards is
handled properly under Linux.
      This driver version seems very stable
and I am sure you will like it 
      if you are using recent SCSI
controllers based on the 53C875 chip.
      I recommend to enable NvRAM
detect option in the driver config.



--
The initial revision has been a
port from the FreeBSD NCR53C8XX driver. 
The driver has been incorporated
into the linux official source tree in 
June 1996 for linux-2.0.3.
The
original driver for FreeBSD has been written for 386bsd and FreeBSD by


Wolfgang Stanglmeier        <wolf@cologne.de>
    Stefan Esser
  <se@mi.Uni-Koeln.de>



--
The previous stable driver version is:
1.18i.
The current pretty stable driver version is:   2.3c.
Both support
the whole 53C8XX family.


Tagged Command Queuing and Fast Synchronous Data
Transfer are supported 
with all 53C8XX chips.
Other features depend on the
involved PCI and SCSI subsystem and so are 
supported and enabled only when
all needed conditions are met.


For example, the following features of
53C875 SCSI controllers are
supported by the driver:


- Wide 16 data
transfer.
- Fast 20 (ULTRA SCSI) synchronous data transfer.
- Synchronous
offset 16.
- Large bursting up to 128 dwords and 536 bytes DMA fifo.
-
'On-board RAM' for script instructions.
- Optimized PCI bus commands.
-
Script instruction Prefetching.
- Read user setup from NvRAM (version
2.3c).

ncr53c8xx-1.12d-to-2.3a-for-linux-2.0.29.patch.gz


ncr53c8xx-2.3a-to-2.3b.patch.gz


ncr53c8xx-2.3b-to-2.3c.patch.gz


If you need to use the driver under
linux-2.0.27 apply first the patch


'ncr53c8xx-12c-to-12d-for-2.0.27.patch.gz' and then the above ones.


For
pre-linux-2.0.31-2 and the future linux-2.0.31, you can use patch:


    ncr53c8xx-1.18i-to-2.3c-for-linux-pre-2.0.31-2.patch.gz


NvRAMs of
Symbios compatible and Tekram DC-390/U/F/W boards are supported.


If you
absolutely need to use the driver under linux-2.0.27 apply first the 
patch
'ncr53c8xx-12c-to-12d-for-2.0.27.patch.gz' and then the above
ones.
