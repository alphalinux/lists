>There was it with "I've finally found the solution to recognise the

I have been advised by fellow kind list members that it is important to
pass boot parameters to the kernel for the tulip card to work. We are
using the BNC connectors and I have to attached the following parameters:

ether=0,0,2,eth0

during boot time to enable the card to use BNC port for communication.
