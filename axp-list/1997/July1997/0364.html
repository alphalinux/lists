At 12:26 15.07.97 -0500, you wrote:
>Hello!
>at last! i have succesfully boot my alpha server 400 with redhat linux to
>begin installation, but now, when i reach the partioning stage (i've done
>it before on x86's), and i try to create a new partition, the program says:
>"set the number of heads and cylinders in the extended menu..." or
>something like that, i enter the extended menu, but i don't know which
>parameters to set.... i don't have information about my disks, i only know
>they are two SCSI drives, one with 1GB and the other with 4GB. What i do?
>Thanks in advance for any help.

First go to BSD Label (Type in B)

Bsd Label
write down the values for

tracks/cylinder
sector/track
Cylinder

quit
enter again
type in X for advanced mode

Type in the value for heads, sectors, and cylinders

In advanced Mode in BSD label

heads tracks/cylinder
sectors sector/track
cyl cylinder

TYpe R for return

make your partitions

Type w for writing

go further in install

hope this helps
