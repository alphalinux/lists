Hmmm, maybe worthwhile checking the parent process which started the
one you are trying to terminate.

You can do this by issuing the following command:

ps axjp <process id>

e.g. for process id 6824

ps axjp 6824

USER PID PPID PGID SESS JOBC STAT TT TIME COMMAND
squid 6824 6819 6816 4a441fc0 0 IW C0- 0:00.19 (dnsserver) (dnsserve

here ppid 6819 is the process which called process pid 6824....

Naturally you don't want to kill processes which are critical for the
running of the server 8)

hope this helps
