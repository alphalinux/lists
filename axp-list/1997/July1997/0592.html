Hi, Markus,

There should be no problem getting your miata running, as i am running
one myself, other than the fact that I still get unaligned traps, and my
CDrom doesn't play audio CDs though my sound card is working, and the
machine seems to run slower than it was under NT .... Nonethless, Linux
on miata works.

I used the miata-kernel-2029.gz (something like that). To enable your
ethernet card, you would need to put in some boot prompt option. We are
using the BNC port, so I have to add:

ether=0,0,2,eth0

to the boot prompt in order to get the ethernet card to work. After
which, should you need to recompile a custom kernel, you would need to
download the latest tulip.c to drive the 21142 ethernet card. I can't
remember the place to get tulip.c, but I could email it to you in private
if you need it.

For milo and loadlin, I've found that the ones come with RH4.2 works
perfectly fine for me, so you could probably stick to them.

After which, I guess you would be trying to get X up. After running
Xconfigurator, you would need to do this:

ln -s Alcor /etc/alpha_systype

Without which, X would never come up.

Since you mentioned you are quite new to Linux, I would imagine that you
would probably run into the same problems as I have, since I am pretty
inexperienced myself. I managed to my system up after many helpful advice
from fellow list members, so I thought I should share it with you,
hopefully to help save some headache in the process.

Good luck!
