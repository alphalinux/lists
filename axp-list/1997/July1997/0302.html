Craig,

I've used Linksys Etherfast LNE100tx/Netgear FA310tx/Cardware Labs CWL9512,
all Tulip based cards, all 10/100Mb in my PC164 (500MHZ) and PPro's.

Both the Linksys and Netgear are MII type cards that require little
special setup in Becker's tulip.076 driver. If you connect to older
equipment, watch out for n-way autonegotiation since old stuff won't
do it. I had to compile Enet support directly into the kernel since RH4.2
module support is broken with the SW provided on alpha. Modules work
on Intel where I'm using Slakware.

You can order these adapters from various catalogs for $55-$60.

Good Luck...
Wes
