I've hacked up "LyX" a bit to fix the problem with the label fonts: They
were too big to fit on pushbuttons and such on the Alpha (and on other
platforms too). -- There should also be an official patch for that (but I
don't know where and whether it has been released at all), but if you care
about getting good looking dialog boxes in LyX, just have a look at

http://stud1.tuwien.ac.at/~e9025461/lyxpatch.html

There is a small shell script fixing that problem that I've just made
available. (Drawback: you have to recompile LyX...)

LyX sources are available from the "official" LyX server:

ftp://ftp.via.ecp.fr/pub/lyx

BTW: Has anyone had any luck with running "fdesign" as it comes with
XForms 0.81 on Linux/Alpha? -- So far I've never managed to change
an element's attribute values in the "Attribute" dialog box without
getting a segfault. So fdesign is almost completely unusable on the Alpha
at the moment... :-( (and the executable comes stripped...)
