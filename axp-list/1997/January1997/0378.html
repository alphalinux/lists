> I'm running Linux on a EB164 for less than a week now, so please excuse if

> be stupid: Is there a way to get into the BIOS of the Adaptec 2940UW SCSI c

It's possible that the latest ARC console for EB164 has some kind of way to
do this; it has some support for editing the QLogic 1020-ISP NVRAM parameters,
so maybe the Adaptec has something similar?

Try getting the latest ARC console for EB164 from:

http://ftp.digital.com/pub/Digital/info/semiconductor/alpha/dsc-software-alpha.
html

...and see if it has such.

Good luck.
