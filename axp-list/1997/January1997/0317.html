IIRC, the Amiga uses some kind of GCR recording, at least on DD disks - it's
the low-level coding, not the number of sectors that cause the trouble.

The Amiga uses nearly-standard disk drives, so hooking up one of those to an
Alpha won't be of any use - you need a special controller, but a standard
PC-type disk drive.

Can't the Amiga be instructed to write standard PC-style MFM format while
still using it's own FS instead of MS-DOS?
