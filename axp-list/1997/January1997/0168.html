On Tue, 7 Jan 1997, David R. McGown wrote:

> May I be so bold as to ask a slightly off-topic question on how you have this
> system partitioned. The reason is I see a potential problem with booting the
> system if you are installing the root partition on /dev/sda1. If you want to
> boot off the harddrive, using MILO and the ARC console, then by convention the
> ARC console uses the first partition of the first drive
> (/dev/sda1) as the "operating system" and/or os loader. For this partition to
> be readable by the ARC firmware, it needs to be an MSDOS partition. Normally,
> you should create a small MSDOS partition and install linload.exe and the MILO
> image for your hardware (noname.img) on it. If you have /dev/sda1 as an ext2
> partition, you will always have to use the floppy to boot the machine.
>
