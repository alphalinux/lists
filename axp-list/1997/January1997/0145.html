Pablo Manalastas <pmana@tapa.admu.edu.ph> writes:
> updates/axp directory at redhat.com. Everything
> seems to be working fine, except that I can not find
> an xserver for the old Compaq SVGA controller card
> with 1MB ram that our Jensen comes with.

I am afraid there's no X-server for this card. The only server that's
has been ported to the Jensen is the S3 server, which is available
from my home page.

I read that the SVGA server has been ported to the Alpha to support
the Matrox Millenium, but I don't thing it will support your
card (even the standard XFree86/Intel doesn't support it). Maybe in
plain VGA mode (640x480x16). Then try to use it with the
'nomemaccess' option. It might work (but I don't think so), but it will
be very slow.

Search the mailing list archive, there has been a discussion about
this card before. Anyway, I'd rather suggest to get an S3 card.
