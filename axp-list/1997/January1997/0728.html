Is one of the boot possibilities to boot from floopy? If so try to stick
a floppy in the drive before booting. At least this was the solution to
my similar problem. Later I removed completly the boot from floppy
option,
i.e. made a small dos partition (1.4MB) an the hard disk for Milo and
loadlin and boot from there.
In my case I think the problem was that I had reference to a boot
disk/partition (in this case a floopy disk) that ARC could not find.
