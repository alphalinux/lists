here is the right and Digital conform answer to this topic.

If you buy a Board it depends on the processor Type if you can run
Digital Unix on this board. there is also a major pricedifference on the
two processor types. If you take the cooler from your processor and this
is labeled 21164-a Px you can only run Linux or Windows NT. There is a
rumor that there is comming up a SRM-Console programm witch is nessasery
for future releases of Digital UNIX witch looks if the right processor is
installed. If not there are stories abaout several actions taken by the
programm but at this time nothing confirmed.

These restrictions apply on all the motherboard and Processor types.
