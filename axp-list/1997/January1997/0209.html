Well, start with fresh kernel sources. I am not sure if the RedHat source
contains any patches, though you should be able to extract the clean source
from the source RPM.

Next, patch the kernel up to 2.0.21, then apply the axp-diffs patch. You need
to apply the axp-patch from /usr/src/linux, using
patch -p1 < axp-diffs-2.0.21

Then apply the rest of the kernel patches (again from /usr/src) up to 2.0.27.

Under "make config", select the Alpha platform you are compiling the kernel
for. I have a UDB, so I select the "Noname" platform, for an Alphastation, you
will have to select the appropriate platform for your hardware. If you do not
know this offhand, you may want to check the Alpha FAQ on www.azstartnet.com.
Continue with configuration options suiting your hardware. It is recommended
that you compile a monolithic kernel for Alpha, unless you have the new
modutils-010497 package, since module support is broken on AXP.

Once the config is done, make dep; make clean; make boot
^important, not zImage!

Then install the kernel residing in arch/alpha/boot to / or /boot, and reboot.
