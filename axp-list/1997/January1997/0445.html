In the _Alpha AXP Architecture Reference Manual_, see

Part III, Console Interface Architecture,
section 2.4, Interprocessor Console Communications
section 3.4.3, Multiprocessor Bootstrapping

as well as

Part II-B, DEC OSF/1 Software
section 2.2.16, Write Interprocessor Interrupt Request
section 5.4.3, System Entry Hardware Interrupts (entInt)
