The 433a and 500a have a 128-bit SDRAM memory system that cycles
at 61 MHz. McCalpin copy bandwidth is over 300 Mbytes/sec for the
cacheless machine, about twice the bandwidth of the AlphaStation
500/500.
The machine has a L3 cache socket and a cached version of the machine
is planned.

Not only is the memory BANDWIDTH dramatically better than our previous
AlphaStations, but the memory LATENCY is also dramatically better!
