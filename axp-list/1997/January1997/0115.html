Welcome to the club. As was explained to me just recently, the distribution
is missing the file /etc/nsswitch.conf.

Please look in this mailing list for a copy of such a file, boot your
system into single user mode (just add "single" to the MILO boot command
you used above), install the file, and reboot. It should work.
