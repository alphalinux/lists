OK, this is a good idea. I just ran a test compiling the gnuplot 3.5
sources on an alphastation 500/500 under Digital Unix 4.0. My installation
of gcc uses the DU assembler and (probably) their linker. I hacked the
Makefile so the autoconf script would not run each time. It looks like
the Digital Unix compiler runs in about 1/2 the time of gcc:

gcc:

cc:
