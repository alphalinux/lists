OK so I decided to Upgrade my machine, a UDB with 24 Meg of memory.
As has been suggested on this list I used bog standard 72 pin 36 bit simms, 2
8meg simms.

A quick install in the two spare sockets and the machine reported 40 Meg of
ram,
it booted fine, except some applications started to report Memory errors and
after a while these errors start to cause all sorts of problems which force a
reboot.

I ran a memory test from the SRM console, no errors were found after an hours
worth of testing. But when I try to compile any decent sized piece of code it
will usually crash out with variious errors.

I've tried reseating all the Simms and the Cache module but to no avail.

I'm lost, out of ideas and wondering if anyone has any help to offer?
