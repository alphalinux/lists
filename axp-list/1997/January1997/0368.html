tomas <tomasb@rainside.sk> writes:
> 2. i'm about to install redhat 4.0 on a jensen
> (finally) are there any serious problems
> i should know about?

Make sure you partition your disk with Jim's ARC minlabel.
Include at least a swap partition, and one ext2 partition.
Don't use the partition tool offered by RedHat, because it
will overwrite the boot sector of your disk. You could fix
this with 'copyboot' again, if you used BSD labels. But don't
use fdisk a style partition scheme on your boot drive, because
the SRM console can't boot from it.
