Only the VX-41 and VX-42's can be overclocked. These have ZIF sockets for the
CPU and speed selector jumpers. The jumpers are on the right edge of the
system board, labeled W7, W8, W9. According to the Multia Service Manual, the
jumper settings are as follows:

W9 W8 W7 CPU Speed (MHz)
1 1 1 66
0 1 1 100
1 1 1 133
0 0 1 166
1 1 0 200
0 1 0 233
1 0 0 266
0 0 0 300

0=installed, 1=not installed
