Ok, here's how to run Netscape (and other programs) for AXP/OSF on a AXP/Linux
box. A couple of notes. All OSF stuff is Copyright Digital. It is not legal
to use these libraries and programs on a Linux box without the appropriate DEC
license for the OSF. This is known to NOT work with DU 4.0x. I've tested and
had no problems with libraries from OSF/1 3.2, I hear 3.0 works also.

1. Get /sbin/loader from OSF machine and place in /sbin on Linux machine.
2. Next from the OSF machine directory /usr/shlib get the following and
put them in a /usr/shlib directory on the Linux machine:

libXm.so
libXmu.so
libm.so
libX11.so
libXt.so
libdnet_stub.so
libc.so
libXext.so

3. Get /etc/svc.conf from the OSF machine and put it in /etc on the
Linux box. chown and chgrp this to bin and chmod to 755.

4. Download from ftp.netscape.com the OSF/Alpha version of navigator
(ie. netscape-v301-export.alpha-dec-osf2.0.tar.gz)

5. Install it

6. Run it!!
