I'm sure I meant to CC this to somebody, but oh well.....

Got a new alpha running linux on campus! Only took me well
over a week to get the damn thing up up,(grumble) finally found
out via the redhat axp-linux list that I needed a newer version
of the kernel disk with 2940 support,(big thanks guys) then it
became relatively smooth sailing...also found that the linux kernel
didn't like me trying to put the root partition on a partition of
3GB...cut it back to 1GB, seems ok. Heard that minlabel would have
gotten around that, didn't feel like learning a new trick.

Anyways, it's an Enorex PC system based on the PC164 motherboard
and a 500MHz 21164 alpha CPU with 1MB of external cache.
Seems snappy, so far thrown ssh 2.0.17 and povray 3.0 on there..

Decided to run a quick benchmark for kicks...the numbers below are
from povray 3.0, ray tracing a frame for an animation for my new
webpage which will be around in a week or so. Specs for the systems
are listed below...looks like I'm not getting quite what I should
out of the alpha, but I haven't tried tweaking anything, yet.....

seconds)
