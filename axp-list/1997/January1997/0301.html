I would use IP Masquerading in this instance. Set up your home
LAN to use a private Ip addressing scheme, such as 192.168.x.0 for
you home subnet, and let the linux box do NAT (network address
translation.) This way you don't eat up a whole 30 device subnet
for 3 boxes. Also, doing it this way, you can handle dynamic IP
addressing on the PPP side.

-John
