I managed to install AfterStep on my UDB w/ Red Hat 4.0 with little difficulty.
The installme script ran fine, however, the makefiles that were generated
were attempting to link in a "net_stub" library if the architecture is an
alpha. Removing the references to this library made everything work just
fine. I believe that the net_stub library is a Digital Unix phenomenon, as
it does seem to be required to compile it under DU 3.2...
