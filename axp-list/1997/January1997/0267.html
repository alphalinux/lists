Check out:
http://www.workstation.digital.com/announce/dpw/dpwa_an.html

Basically:
12 specint 95, 15 specfp95, Faster then a 500 Mhz clone+1mb cache,
don't be fooled by vendors claiming 15 int/20 fp
with only 1 mb L3.
128 bit wide sdram memory system
Single chip core logic chipset (PYXIS)
0-2 mb cache
2 64 bit slots
Audio, ethernet+ultra wide scsi

Systems with 433 Mhz 21164, 64 mb sdram, 2.1 gb disk, matrox list $4,995.

Anyone seen any stream numbers? Know what speed the sdram is?

Linux work on it yet?

Will the boards be sold by reseller like the PC164? When?

Sounds like a pretty cheap motherboard (1 chip chipset, no cache on board),
and with the recent halving of the cpu costs sounds good to me....

Sounds like a killer linux box, more affordable to, seems like they
finally got a memory bus to do the alpha justice.
