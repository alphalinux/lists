At 11:19 AM 01-15-97 +0100, you wrote:
>I'm running Linux on a EB164 for less than a week now, so please excuse if
my question may
>be stupid: Is there a way to get into the BIOS of the Adaptec 2940UW SCSI
controller where
>you can change a lot of useful things when running it on a PC ?
>
>Thank you
As this is Intel code being executed from the BIOS chip on the Adaptec it
seems very unlikely. The driver compile options determine most of what you
need, except the termination setting, which is automatic configuration on a
2940UW anyway, so it shouldn't matter much. Worst comes to worst stick it in
an Intel box and configure it.
