With all this talk about putting PCI video cards into a UDB it makes
me wonder if a dual monitor configuration works well. For example,

TGA running the console... with 6 VT's ALT-F1 - ALT-F6 switches
between virtual consoles.. (like usual)

Run XF86_S3 for XWindows on some S3 chip.. ALT-F7 activates mouse
and keyboard focus for the 2nd monitor (with X-Win on it..)

My guess is that it doesn't work this way currently. The XServer
would probably want to restore the console mode on the second
monitor when you do and CTRL-ALT-Fx which it wouldn't need to do.
But it could be done with a little work. Would this be useful? The
monitor on the TGA could be pretty low end since the console is fixed
at 640x480 60Hz...

Just my 2 cents...
