Howie,

Yes, do a rpm -i for your src.rpms, it will put some files out in
/usr/src/redhat. After that go into /usr/src/redhat/SPECS and execute an
rpm build, ie rpm -bb xxx.spec. That should build and install your
application. Sometimes I've noticed that people insist on having
environment variables set to point to /usr/src/redhat/SOURCES (where it
actually puts the tar.gz file of source code) so you may have to look at
the .spec file and set that as well.

I basically had to figure this stuff out on my own, I could easily find
any documentation on it. Anybody know where there's anything official??

Gerald
