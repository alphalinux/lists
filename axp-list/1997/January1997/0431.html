Under kernel 2.1.20, I am able to run with the NCR53C8XX driver memory
mapped. My setup is a UDB/Multia with internal Toshiba 330 MB SCSI,
external Maxtor 540S FAST SCSI2 and external Wangtek ES1000S SCSI2 tape
drive. I am able to run with synchronous transfer rates of 10Mhz!

Bonnie reports 2.8MB/s on writes and 3.6MB/s on block I/O using the
Maxtor. IOZone looks even better.

Completely, 100% stable (so far).
