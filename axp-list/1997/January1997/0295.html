At 03:23 AM 1/11/97 +0000, John T. Croteau wrote:
>I am getting an AS255/233 with a ZLXp-L1 in a few weeks and will probably
>not be happy with Digital UNIX and will want to run Linux (been using
>Linux for years on x86 platforms).
>
>Is there any support in X for the ZLXp-L1 card currently? If not, is
>anything planned for the card?

Sorry, there's no current support nor any plan for support of those cards.
There are so few actually in a position to be used for LINUX, it doesn't seem
worth the effort, sorry to say... :-(

If you could get the hardware specs, perhaps you could try something along
the lines of the current XFree86 3.2 support for the TGA; ie use as much of
the "dumb" frame buffer support as possible with enough hardware-specific
code to get the card initialized.

Sorry.
