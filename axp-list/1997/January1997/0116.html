create the file /etc/nsswitch.conf
it solves you the problem here is mine

passwd: files
shadow: files
group: files
hosts: files dns
services: files
networks: files
protocols: files
rpc: files
ethers: files
netmasks: files
bootparams: files
netgroup: nisplus
publickey: nisplus
automount: files
aliases: files

hope this helps
