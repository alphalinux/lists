I have Windows NT server 4.0 running on my Mikasa AlphaServer 1000. It
runs absolutely beautifully. I have a number of nifty apps running under
FX!32 and it also runs beautifully. FX!32 isn't supposed to be able to
run compilers, but I'm not sure about Visual Basic (it's an interpreter,
right?) Besides, I bet there's a native Alpha port for it.

Words of advice when installing:

Make sure you have your EISA config disk if you have an EISA bus.

Remember to make a small DOS partition (also very useful for bringing
stuff between NT and Linux) for ARC to boot from. You can also use this
for MILO

Be careful when choosing partitions. NT lists them in reverse order. I
ended up acidentally erasing Digital Unix :(
