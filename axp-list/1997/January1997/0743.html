I am not aware of any office package available for AXP/Linux, however there was
some discussion a couple of months ago that Redhat would see about getting a
port of Applixware for AXP if there was a substantial interest.

Currently, LyX is a good choice for WP. For a generic draw program, you
probably already have xfig installed, which has the added benifit of outputing
PS and LaTeX figures, so it should work well with LyX.

Two spreadsheets I know about are sc and slsc. I know that slsc comes with
Redhat, so you probably have it installed. Of course, these are not as
"friendly" as the various Linux/Intel alternatives (Wingz, Xess, NeXs,
Applixware, etc.).

Hopefully, as Linux/AXP grows, and as ELF matures, the apps will come. An ELF
Motif is not availble yet, so this will hamper any porting efforts using these
libraries.
