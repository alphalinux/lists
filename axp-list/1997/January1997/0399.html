On Wed, 15 Jan 1997, Adam Biskobing wrote:

> I know this is not the right place to post this, but I am interested
> in setting up IP Masqerade for a few IP's but also have the machine
> running IP Masq act as a gateway for some "real" ip's, is this possible?

It should be. IP Masquerading will only kick in for the addresses you
specify. So, keep the fake addresses of the machines you want to
masquerade different from the valid IPs. You should probably use the
addresses that were reserved for this use. I forget most of the, but the
10.*.*.* A class should provide you with enough addresses :)
