>
> Mike Loseke <mike@bamboo.verinet.com> wrote:
> > I've been using a #9 Motion 771 w/4MB fora few months, both with RedHat

> > colors) and it's quite snappy.
>
> > The only problem I have is that the server does some strange stuff on
> > the right side of the display and Netscape displays it's stuff a bit screwy.
>
> >From the XFree86 V3.1.2 release notes for the S3 driver:
>
> Number Nine FX Motion 771
>
> s3RefClk 16
> DACspeed 220
>

>
> Invert_VCLK "*" 0
>

Doh! I forgot all about these when I upgraded from 3.0.3 to 4.0... Thanks!

> The Invert_VCLK solved the "strange stuff on the right side" problem for me,

I run Netscape off of an Intel Linux box and display it on my UDB. The
problems that I see are: the writing on the buttons, as well as the small
images themselves, become unreadable; many interlaced images show some
strange junk along with the image.

A few other apps (ical, xv when dragging in the schnauzer) show some
remenants on the screen as well. It's nothing that I can't live with, it's
just annoying.
