This is a common problem with the stock beta-0.2 TGA server. You can get the
latest Digital developed X-server from the same location, which fixes much of
the crashing under Netscape. I believe can be found at:

ftp://ftp.digital.com/pub/DEC/Linux-Alpha/X11/XF86_TGA-3.1.2-bugfix-EV4.gz for
the XF86 3.1.2 server, and XF86_TGA-3.2-bugfix.gz for an XF86 3.2 server.

I believe these are ELF binaries, so you need to be running RedHat 4.0 or
better.

David
