I found a great deal on a sdram Alpha.

433 Mhz, 2.5 GB eide, NT4.0, 32 mb ram, matrox NT under $5k.

Not bad, considering it's getting better specfp95 scores then the 500 Mhz
clones costing the same or more.

I BELIEVE it has a cdrom but not sure. I'm unassociated with the
vendor btw.

In any case this purchase is to run Linux. I've heard that EIDE, and
the on board ethernet are unsupported at this time.

I was considering buying a 4 GB scsi disk, Buslogic 958 scsi controller,
and SMC etherpower 10/100 Ethernet card. I was planning on using
the Matrox card (any matrox/alpha/linux users out there?).

Anyone using these with Alpha/linux? Any ideas on these working on the
new personel workstations?

I've heard that so far Linux will only boot with the SRM console thats
not available yet. Is that still the case?

BTW I've heard it gets 300-350 MB/sec on the stream benchmarks which
is approximately twice the alpha 500/500.

Any recommendations? Suggestions? Warnings?
