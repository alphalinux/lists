At 06:20 am 1/4/97 +0000, you wrote:
>Hello all:
>
>Just bought an alpha and I have a quick question bout src.rpm files. If
>I download a src.rpm file and type rpm -i filname, what does it do and
>where is either the source files that I can compile or a binary that I
>can execute? There is also a good chance that I am going about this
>wrong. If so, what the heck to do I do with these src.rpm files? Thnx
>fer any all help.

The src.rpm files usually get installed in the /usr/src/ directory
structure. Generally these should build w/o a hitch (if they are from
the RH dist that you installed from.)

All the src.rpm's that come with the RH dist are also there as binary
(xxxxx.rpm) rpm's. So unless there is something specific that you
need to change or customize, you really won't need to install them.
The only exception to this is the kernel source, which you will
probably want to rebuilt to support the options you want to run on
your alpha.
