Some more facts ( from Digital, passed on):
Part #: SN-B3AWW-EB List: $ 4945 U.S.
NO Cache RAM
IDE (no SCSI card)
IDE 2.1GB IDE H.D.
32MB RAM
8X CD-ROM (ATAPI IDE)

Now for a "bare-bones" variant machine:
Part #: SN-B3AWW-F1 List: $ 5145 U.S.
With 64MB RAM
No cache RAM
No SCSI
No Video
With 8x ATAPI CD-ROM
10/100T ethernet

Gives an idea what the RAM costs, eh?

Another more loaded system:
Part #: SN-B3AWW-FQ List: $ 6343 U.S.
64MB RAM
0K cache
2.1GB UltraSCSI (narrow) H.D.
Adaptec 2940UW SCSI card
No video
inc. floppy, 8X ATAPI CD-ROM

As you can see, these "inexpensive" variants all have lots of things either
missing or inadequate.
IDE drives, no cache, many pieces needed missing, no SRM or capability for
UNIX/LINUX, etc...

There are some pretty nifty models available, but you have to add lots and
pay even more....

A fully configured system with 64MB RAM, 1MB cache, MAtrox Millenium,
UltraSCSI, 2.1GB HD, ethernet, 8X SCSI CD-ROM, etc. will set you back about
$ 11,000 U.S.
