On Thu, 30 Jan 1997, Terran D Lane wrote:

> On Wed, 29 Jan 1997, Brian Walters wrote:
>
> > I remember seeing someone ask this question several months ago but have
> > yet to see an answer either negative or positive, so maybe I missed it.
> > Is there a way to change the console resolution in axp/linux like you can
> > in PC linux in the lilo.conf. I like to use a smaller mode like 7 to get
> > more info on the screen. Just thought I would ask.
>
> As nearly as I've been able to gather, the answer is that there's no
> way to set this through Milo or the kernel compile. You _can_,
> however, use SVGATextMode (I think is the program) to set it at
> runtime. Works ok (at least it has for me), but may require some
> twiddling to get it to do just what you want -- it uses an X-like
> config file to describe your video hardware.

Unless there has been some revolution in SVGATextMode, it won't work with
the UDB's built in graphics (TGA). However, it works fine with normal
VGA-based cards like S3.
