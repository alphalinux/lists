On Sun, 19 Jan 1997, Francois Mainguy wrote:
> I have a UDB (DEC Mutlia) running RedHat 4.0 Colgate distribution. I
> want to use the AfterStep window manager. Do you know how to install the
> files on the system. I tried out with the only downloadable file present
> on the AfterStep site. The "Installme" script file ran on my machine but
> the sub-Imakefile crashed. Help, I want to use this NeXTStep-like GUI...

The Imakefiles needs a patch to build on alpha's (the patch is available
at ftp://nova.campus.luth.se/pub/linux/patches/as-linux-alpha.diff).

I also suggest applying the as-linux-fsstnd.diff available at the same
place as the previos diff. This patch makes AfterStep be linux-fsstnd
compatible.

If you don't want the hassle of compiling it, I have a RPM made
available at
ftp://nova.campus.luth.se/pub/linux/RPMS/alpha/AfterStep-1.0pre2-5.alpha.rpm

You will need rpm-2.2.9+ to install this package (because of the change
of .alpha.rpm -> .axp.rpm)
