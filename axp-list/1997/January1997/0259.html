On Thu, 9 Jan 1997, Terry Carroll wrote:

> Actually, this reminds me of something I was going to bring up sooner or
> later: is thare any way to mount an Amiga-formatted floppy so as to be
> readable by Linux?
>

Well, this is not possible because the Amiga drives use such a way to
record tracks, as to be able to read only by Amiga drives. Alphas have
standard PC floppy drives, and they are *not* able to read Amiga-formatted
(80 tracks, 11 sectors per track) floppies.
Of course, hooking up an Amiga hard drive and mounting it (with affs)
should work. Oh, affs on alpha seems to be a bit broken, it returns length
0 for all files... I've already mailed the author about it. Seems like
some structure sizes use fixed numbers instead of sizeof() or something
similar.
