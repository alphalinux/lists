We are using fort77 on UDB's, last couple of months. It is a fortran to
c converter. We had no major problems with the converter except:
1) Performance of the codes were poorer than expected (I must admit that
we did not do a very serious comparison)
2) You can compile the c code with optimization but I guess that it is
different than fortran optimizations. In that case you may have to
hand-craft a c-optimizable fortran code :-).
