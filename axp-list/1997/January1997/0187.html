On Wed, 8 Jan 1997, Harry McKee wrote:

> After reconfiguring my machine to boot into Linux via the "Boot Menu" when
> I try to boot I get the message:
>
> Error: Bad device name/number or partition number, press any key to
> continue

After screwing around with the alphas boot menu, I found the problem. I
had to change the environment variables for SYSTEMPARTITIO, OSLOADER, and
OSLOADPARTITION to read disk(6) instead of disk(0).

> The only other problem I noticed is when initially booting into linux I
> got an error:
>
> mount: mount point /dos does not exist

Made a /dos directory.

problems. But, time for bed now...
