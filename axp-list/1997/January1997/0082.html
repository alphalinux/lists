Hello all,

I was perusing Dec's ftp site recently and found a directory under
gatekeeper.dec.com:/pub/DEC/Linux-Alpha/Freeport/. Here is the header from
the QUICKSTART file included in the distribution:

FreePort Express Quick Start

January 1996

This document provides the minimum information you need to
install FreePort Express and to translate and run a user
executable for SunOS Version 4.1.n (Solaris Version 1.n)
SPARC on an Alpha system running Digital UNIX[R]. All the
examples shown use the c shell. For further information
about installing the kit and for a list of known problems
and restrictions, see the FreePort Express Release Notes.
For all other information about the translator, see the
FreePort Express User's Guide.

Does anybody know anything about this? I've tried to make it work under
my Alpha under Linux v2.1.17, but so far, it's a no go. Why it was ever
included under the Linux portion is a mystery to me. I suppose someone
decided to give the Linux-Alpha hackers a wack at it. I was thinking that
this would very cool indeed since Netscape makes an executable for SunOS
v4.1.x. Of course, there exist other possibilities with what we could do
with it...

-d
