> > It sounds like the install failed to create the necessary device(s) for
> > console IO. Boot from the installation diskette and follow the
> > instructions through the point where it asks you to specify the location
> > of the CD. Then, switch VT's to the second console and mount your Linux
> > root partition by hand:
> >
> > # mount -t ext2 /dev/sda1 /mnt
> >
> > cd into /mnt/dev and look for tty[0-5] and console. If they are NOT
> > there, use mknod to create them, or copy them from the "live"
