Yes I use fortran a lot; FORTRAN 90. At the moment I'm stuck using NAG's
sun-based F90 compiler at work on a ratty dual processor Sparc 10. My UDB
running linux has no fortran compiler on it since there is no F90 compiler
for alpha linux. However, NASoftware (of Linerpool, England) do an
absolutely lovely pc-linux (and sun) fortran 90 compiler called
fortranplus. It has HPFS extensions and some F95 featues aswell as built
in graphics functions an f77 - f90 convertor and a debugging tool. Very
nice for about 150 pounds sterling. Now, hopefully someone on this list
will work at NASoftware, or know someone who does and they'll decide to
port it to alpha-linux - You can but dream :-)
