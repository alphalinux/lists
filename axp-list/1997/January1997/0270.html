I would really recommend you use an external SCSI drive. The performance of
the UDB is less than stellar, and a fairly fast SCSI drive makes a big
difference. The IDE interface is reported to be quite slow and uses 100% CPU
utilization during transfers. Remember, for a multitasking OS like Linux, there
is a performance gain with a busmastering controller like the NCR 53c810
built into the UDB. There are some nice compact SCSI cases out there for 3-1/2"
drives that will work nicely, or get a 2-bay case for the CDROM drive and the
SCSI drive.

David
