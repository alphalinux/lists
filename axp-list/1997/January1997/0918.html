>> >21A04-B2 PC164 Motherb. with SRM and 2 user base license.

>> You have it, 21A04-B2, and the 21164-IB. You can keep the B0 mb and reflash
>> in the SRM however, you need an SRM & Digital UNIX license in order to load
>> DEC UNIX. These come with the 21A0-B2 motherboard.
>
>ok ... so the only difference between the motherboards is what's in flash
>rom, and the licesnse to run srm. (i am running srm on a udb -- is this a
>violation somehow??)

You need an SRM license and a UNIX license. The UNIX license allows you to
unlock and load Digital UNIX. Without the license it will not load. As far
as the UDBs are concerned it comes with it so the license is built in.

>so is there any real difference between the actual cpu's? this seems very
>strange to me. somebody told me that DU alphas are somehow "faster" then
>the nt ones ... sounds like a myth to me, but now i hear they are
>different part numbers?

None that I know of. Digital UNIX is a hell of a lot faster than 32-bit NT.
