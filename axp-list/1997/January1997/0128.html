Well, this has been driving me nuts playing with it, and I finally got
it so here's the scoop on how to run DEC OSF/1 Alpha Netscape Navigator on
a Linux Alpha box.

All of the OSF files were taken from an AlphaStation 5000 running OSF/1
3.2. The linux box is a Cabriolet 275 with 64Meg, Kernel is 2.1.13.

Notes: I haven't played with anything else, but I really don't see why
a number of other programs would run also. Also, I've heard that Java
Applets still won't run, but I haven't tried it, and really won't miss it
that much (they're not very stable with an Intel Linux box).

Without further delay. . .

EVERYTHING MUST BE PERFORMED AS ROOT, obviously.

1. Get /sbin/loader from OSF machine and place in /sbin on Linux machine.

2. Next from the OSF machine directory /usr/shlib get the following and
put them in a /usr/shlib directory on the Linux machine:

libXm.so
libXmu.so
libm.so
libX11.so
libXt.so
libdnet_stub.so
libc.so
libXext.so

3. Get /etc/svc.conf from the OSF machine and put it in /etc on the
Linux box. chown and chgrp this to bin and chmod to 755.

4. Download from ftp.netscape.com the OSF/Alpha version of navigator
(ie. netscape-v301-export.alpha-dec-osf2.0.tar.gz)

5. Install it

6. Run it!!

Let me know how this works,

Gerald Anderson
