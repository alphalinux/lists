>>> Eric Smith said:
>
> I don't think you'll find a 4M card that can do 1280x1024x24; most cards
> need 32bpp to do 24-bit color, because they don't do packing. Even if
> you find a card that can do packing, I don't think XFree will support it.

Actually, there are some XFree86 servers that support 24-bit packed format,
on the cards whose hardware supports it. Refer to the XFree86 docs for an idea
as to which those might be.

HOWEVER, none of them will work on the Alphas, as the 24-bit packed format
code is *highly* unclean WRT 64-bit-ness. I mean, we're talking man-months
of effort, here, and it doesn't all work on the Intels, either... :-( :-(

So, buy a 4Mb card that'll do 24-bit packed, run it in 16-bit depth for now,
while you have a go at fixing the 24-bit packed code... :-) :-)
