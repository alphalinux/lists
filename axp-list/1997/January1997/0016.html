On Wed, 1 Jan 1997, Steve Giles wrote:
> I am having a problem getting Red Hat 4.0 on my UDB to boot from
> partition 1. I have used dd to move the MILO floppy to a 1.5 meg
> partition (as per instructions). I can boot from the floppy but
> booting from partition 1 says "Bad device or invalid partition..."
> or something like that.

Are you sure you have setup the boot option in ARC to use the
correct SCSI ID for you hard drive? The instructions I
received for installing Red Hat on my UDB assume a SCSI ID of 0.
In my case my hard drive is set to SCSI ID 6, so when setting up
autoboot, I had to use:

SCSI Bus 0 Hard Disk 6 Partition 0
