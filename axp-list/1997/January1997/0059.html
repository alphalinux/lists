I'm struggling with building glibc on the Alpha as well. I believe the
latest source package is libc-1229-tar.gz (or whatever) in /pub/glibc at
redhat.com. Make sure you also grab the latest crypt and threads which
reside in that directory!

Note that glibc expects you to have the header files from a later 2.1.x
kernel available as /usr/include/linux and /usr/include/asm. I spent
hours chasing compile problems before figuring this out.

I'll post to the list if things succeed.
