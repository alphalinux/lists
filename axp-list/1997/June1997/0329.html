Hi to all DEC fans out there !

I have some occasional X server crashes with my Digital AlphaPC164
motherboard (64MB mem) and DECchip 21164 CPU at 433 MHz with
Matrox Millennium 4 MB WRAM display adapter.

I'm running XFree 3.3 from redhat-4.2 upgrade rpms.

My kernel is compiled from redhat-4.2 sources ( 2.0.30 )

Crashes are seen often when using X programs from another computer
to my Alphas X display.

X server reports signal 11 and kernel as follows:

So is this kernel, memory or Xserver problem ?

The system is very stable and this is just happening when using X !
