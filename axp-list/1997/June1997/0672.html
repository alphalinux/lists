I own a 233mhz Multia I bought for about $800 new. I added 64 MB of RAM,
bringing the total to 88, added an external SCSI chasis with a 3GB drive
and 8X CDROM. Total cost including computer and RedHat disk: $1800. I
don't kow why this neat little box has been so maligned.

I've been nothing but pleased with it. It's small, fast, very well
constructed, and configured in a snap. (Well, maybe a couple of snaps
after tracking down the necessary references :) )
Since I use it as an internet web server doing medium-duty, I haven't
had any issue with the speed or the lack thereof. I think DEC should
produce a similar box with a CDROM but w/o the audio as a good web
server value.
