>>

>
>The 'system' routine on DUNIX expects to have /sbin/sh available.
>make a link thus: ln -s /bin/bsh /sbin/sh and it should work.

Another thing to consider is to check the documentation on
the `system' library function provided by your Fortran environment
(presumably part of libU77). I suspect it returns an INTEGER
status argument that _might_ have, if tested by your code, alerted
you to the problem more quickly. E.g. this might be better than
your "call system(...)" code:

integer status
status = system(...)
if (status.ne.0) print *, 'system call #1 status is ', status
...
