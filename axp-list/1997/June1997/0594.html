I've gotten Chimera-1.70p0 to compile and run pretty well --- it has
limited table support and doesn't do frames, etc., etc., --- but at
least I can mouse through links and get inline graphics. Check:

http://hasse.mathematik.tu-muenchen.de/~nikl/chimera-1.65p/features-1.70.html

if you're interested. There seems to be a problem with some of the
entry boxes (e.g., the URL: box) accepting typed characters in reverse
(!) -- I don't know whether this is an X or a Chimera problem (doesn't
happen on my intel box), and not having an excess of free time, I
haven't investigated (instead, I cut and paste from an emacs scratch
buffer or a xterm --- ugly :-)).

BTW, Chimera uses the Athena widgets, so you don't even need Motif. If
you don't have access to the OSF libs to run a native Netscape, it
might be an option for a native graphical browser.
