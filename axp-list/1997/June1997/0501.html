In my personal experience (a number of machines) I never had any
problems with the other, so called "BSD", driver for NCR cards.
I heard reports about boards working better with this "7,8" driver,
but I still have to really see one.

This does not help very much if you try to boot for the first time
from a distribution image which has only one driver, but once past
that hurdle I would test the other driver immediately.
