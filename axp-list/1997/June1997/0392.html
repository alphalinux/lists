Jim Paradis wrote:
> Kelvin Lunsford wrote:
> >
> > Does anybody out there know of a C compiler (DEC/GCC/NT) for the
> > Alpha processor which will let you set the default int size to 64
> > bits. This is for an embedded application, the resulting binaries
> > would not be running under an operating system.
>
> How about:
>
> % gcc -Dint=long "-Dunsigned=unsigned long"

This would convert 'unsigned int' into 'unsigned long long'
(or we would recurse on unsigned? but this is easily avoidable)
not something you likely would want to see. You can probably
control a code you write to avoid such surprises but headers
will be a fun.

Also all libraries would have to be likely recompiled from
scratch or every second function call would be a challenge. :-)
