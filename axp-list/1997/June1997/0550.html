I'd like to test a S3 Trio64 card in a UDB166 now that I got a NCR
controller with PCI riser (thanks again to Uli Kallenback from IKE in Stuttgart!)
but I don't know how to disable the TGA chip.

looking closely at this riser card, I realize that both the plain NCR controller
as well as this riser card with additional PCI slot have a soldered passive
termination resitor pack (yellow DIP 220/330 chip) which is directly connected
with the SCSI connector, so there is no way to disable termination
if I want to use both internal and external devices.

Am I missing anything obvious? I never saw a controller with soldered termination before,
looks like a pretty bad design decission to me ?!
