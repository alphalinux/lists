At 19:52 16.06.97 +0800, you wrote:
>Hi,
>
>I have inherited an Alpha 1000 Mikasa machine, without manuals.
>Linux is already working fine via MILO on the floppy.
>As per the instructions we tell SRM to do a "boot DVA0" at the prompt.
>
>How I can tell SRM to run this command automatically without waiting
>for the prompt?
>

set bootdef_dev=DVA0
set autoboot=yes
set halt=reboot

type boot

at the next poweron it will automatic boot
