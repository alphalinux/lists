On Sun, 22 Jun 1997, Steve Oliver wrote:

> My lowly UDB arrived loaded with NT. I'm currently running Linux with
> Digital's X server. There seems to be quite a difference in graphics
> performance between the two. Under NT the windows came up faster and they
> didn't rob colors from competing applications. Are there any plans for
> improving the TGA performance ? I'd hate to add a VGA card to this box, its
> already generating enough heat to cook breakfast with.
>
> Thanks for any insight.

I'm working on general TGA optimizations for both the kernel and the
XFree server. This should result in WAY faster console speeds as well as
multiple-font handling. As for the color-robbing, that's a Windows thing.
Windows will fix up the palette for other apps, while X doesn't do that.
