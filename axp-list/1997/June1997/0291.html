First of all, please check if PPP is compiled in the kernel; otherwise
you may have to reconfigure the kernel with PPP installed.

I use the following command:

/usr/sbin/pppd connect '/usr/sbin/chat -v "" ATm1l3DT....... CONNECT "" \
ogin: ..... word: .......' \
/dev/cua1 28800 -detach debug asyncmap 0 crtscts modem lock defaultroute

I hope this will help.
