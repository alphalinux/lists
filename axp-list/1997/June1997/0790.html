On Thu, 26 Jun 1997, Sam Ockman wrote:

> When I run the XFree86 Configuration file, it says I need
> XFree86-VGA16...but this doesn't seem to exist for the Alpha. What's up?

There is no VGA16 driver for the Alpha. Try symlinking your driver to
XF86_VGA16. I.E.: ln -sf XF86_TGA XF86_VGA16 that works for me. You
don't need the VGA16 server if you are running XF86Setup while X is up.
