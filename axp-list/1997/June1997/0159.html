At 23:30 03/06/97 -0400, you wrote:
>How do I go about changing global and local environment variables? I'm
>used to the old .profile from Aix, which I use at school (this server).
>
>Specifically I'm trying to install XF86, and can by exporting a PATH
>statement that puts /X11R6/bin before /X11, but that is only a fix that
>works until I turn off the computer.
>
>What do I do? - I have looked at /etc/profile and it shows the exported
>$PATH statement, but I do not know where the prior PATH statement comes
>from.
>
>Also, how do I set up Linux to create a .profile for other users when they
>login, or when I create that account?
>
>Thanks
>
>David
>

If you want your modified PATH to stay after you logout, put the commands to
set and export it in etc/profile.

eg,
PATH="/X11R6/bin:$PATH"
export PATH

or whatever.

if you want a .profile for each user put a script in their home directory
called .profile. the user's profile is executed after the global one.

Hope this helps.
