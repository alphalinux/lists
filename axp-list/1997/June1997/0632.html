On Thu, 19 Jun 1997, Bob Bottomley wrote:

> I downloaded the updates to RH 4.2 and started installing them. All went well
> until:
>
> # rpm -Uvh XFree86-XF86Setup-3.3-1.alpha.rpm
> failed dependencies:
> XFree86-VGA16 is needed by XFree86-XF86Setup-3.3-1
>
> I cannot find XFree86-VGA16. It does not exist in RH 4.2 or 4.1. What is
> going on here?

There is no VGA16 driver for the alpha series. The only way to run
XF86Setup is after you have an X server up and running.

To install it simply use the --nodeps flag on rpm.
