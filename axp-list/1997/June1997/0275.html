>>> Dave Riesz said:
>
> I never saw a definitive resolution described in the list archive. Is
> there anyone successfully using a Matrox Millenium in a UDB?

I don't believe there is; this came up a while ago, and, IIRC, the person
who'd bought the Millennium traded it in on an S3-based card.

The problem is, the BIOS code on the Millennium is fairly sophisticated,
and the BIOS emulator in the console firmware of the UDB is *not* up to it.

And there's no chance of a firmware upgrade on the UDB; not now, not ever,
beyond what's currently available.

I hope some others make recommendations about what works well in a UDB,
assuming that deep color is what you're after.
