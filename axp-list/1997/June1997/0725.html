Juan Leon <leonj@psc.edu> writes:
> I have a Jensen with a Qvision. Is my only option buying a new video
> card for this ancient machine?

If you need X and Linux, yes. Since the card isn't even supported on
XFree86/Intel (except as a generic VGA with 16 color, but that driver
hasn't been ported to the Alpha jet), so there is no easy way to get it
supported.

I've heard that there exists a Digital Unix X-Server for the card,
though. So if you'd already got a DU License, you may want to use
that instead of Linux. And no, you cannot use the DU X-Server with
Linux, because it uses unsupported DU system calls.

If you do get a news S3 card, make sure you're able to test it before
buying. The SRM console seems to be very picky about the VGA cards it
accepts. Both EISA and ISA cards should basically work, but EISA will
give you better performance.
