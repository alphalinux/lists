I've got it working on 2 UDBs and 2 Intel boxes w/o problem. The only
issues were compile time mods because of some glibc inconsistencies. I am
using rsaref, and there is a mod that has to be made to rsaref for it to
run correctly. From INSTALL:

On alpha, one should edit rsaref2/source/global.h, and make UINT4
"unsigned int" instead of "unsigned long int".
