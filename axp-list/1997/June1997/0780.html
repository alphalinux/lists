I'm trying to put together an emergency boot disk to I can recover my system
from (SCSI) tape in the event of a complete wipe of the root filesystem.

I've examined the RedHat 4.2 installation disk (ramdisk image) in the hope
that I can use a modified version of that, but it appears only to have
binaries to support a new installation/upgrade, rather than unpack a .tar.gz
on my tape (i.e. I can't make tar work and a shell isn't available until
relatively late). Has anyone already put together such a thing, and if so,
could I ask for a copy? Alternatively, does anyone know whether it's
possible to fit a small ordinary system on a 1440 floppy as a gzipped
ramdisk image?

My other possibility is to install a complete system on a Iomega Zip disk
and use my parallel port zip drive from another machine. Unfortunately, I
can't get the Zip driver to work. I've compiled it as a module, and into the
kernel; as a module I get an Oops! when I insmod it and in the kernel is
gives an error about the base of my parallel port when I boot. Could anyone
tell me the correct config information for a UDB?
