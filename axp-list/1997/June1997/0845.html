>>> Lo Sheng said:
> Hi,
>
> I wonder if if anyone has tried FTP installation on the Personal
> Workstation 433a machine. I have successfully boot up through Milo, and
> boot and ramdisk images from 'miata' directory. However, the install
> program comes to a halt at the point when it tries to autoprobe the
> Tulip card that comes with the machine. A check through the ramdisk and
> I found that insmod, its dependent libraries, are all missing.
>
> Am I missing out something here? Or is it because the 433a can only be
> installed from CDROM?

No, it should be installable via FTP. Did you remember to use IP addresses
instead of names? This seems to be an Alpha idosyncracy, at the moment...

Also, make sure to use the latest MIATA kernel from gatekeeper.dec.com in:

pub /DEC/Linux-Alpha/Kernels/miata-kernel-2030-test.gz

This kernel, IIRC, has a recent TULIP driver from Donald Becker, which
is needed to support the 21143 Ethernet chipset that is appearing on the
newer MIATAs. Check out:

http://cesdis.gsfc.nasa.gov/linux/drivers/tulip.html

for more detail about the driver, including some possible boot-line
media options that might be needed for certain hardware connections.

Good luck.
