On 24-Jun-97 Rick Windham wrote:
>Hello,
> Can someone tell me how to know which .rpm is a more current
>version? Example: binutils-970127-1 and binutils-2.7.0.16-2. The
>numbering scheme is different so how can I figure out which is newer?

An 'rpm -qpi' will tell you when the package was built. I can't find the
particular version of binutils that you're refering to, but 2.7.0.2 is from
Sep '96, so it may very well be newer. However, I would suggest going with
a legitimate release and not a snapshot (the date format indicates a
snapshot). 2.8.1.0.1 is the latest, AFAIK
