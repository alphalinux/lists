OK. First step (which will probably cure several of you problems) is
to update the SRM to the latest version. If you already have a Linux
machine or some other Unix machine, I can build you a disk to install
the new firmware. Please give me a yell.

The second part deals with how you are specifying the boot string. It
is VERY important to put double quotes around the flags. This can not
be stated enough.

Also, you may find it very useful to take a look at the application
note that I am writting about installing Linux on an Alpha platform.
It is still not done, but it is to a level where you will get enough
information for the install. You can find the latest copy at

http://www.alphapower.com/Support/Appnotes/

Good luck....
