Hello all,

I've been scouring the list archives for any mention of using the Matrox
Millenium in a UDB and am still quite stumped. I've upgraded MILO and
linload to the most recent versions from Digital and I'm using what I
believe to be an adequate 2.0.29 kernel. I also installed Digital's UDB
firmware upgrade, but it only upgraded to what I already had.

When I boot into linux, linload shows a lot of the usual information on
the blue screen, but nothing else. Curiously, the blue screen very slowly
moves UP a bit at a time until it's halfway off the monitor. I can't get
anywhere beyond that.

Another, equally serious problem is that when the Matrox card is
installed, the machine hangs while still in the ARC console. This seems
like something the proverbial firmware upgrade might fix, if there is, in
fact, an upgrade.

I never saw a definitive resolution described in the list archive. Is
there anyone successfully using a Matrox Millenium in a UDB?
