>>> Avinash Shenoy said:
>
> Is there a list of supported video boards for the AlphaStation 255
> available anywhere? I'm trying to get my unsupported Powerstorm card replaced
> by the local DEC guys & would like to have a few options on hand.

If they're going to replace it with one which we (DEC) sell in our boxes,
your choices are pretty limited:

1. original TGA, aka ZLXp-E1, is supported by XFree86 in 8-bit depth only.
2. ATI Mach64, supported by XFree86 on the Alphas, 8/16/32 bit depths
3. Trio64, supported by XFree86 on the Alphas, 8/16/32 bit depths
4. Millennium, well-supported by XFree86 on the Alphas, 8/16/32 bit depths

These are all the ones I know about; the local DEC guys ought to be able to
tell you the currently supported set of cards, but the above have been ones
that were supported, AFAIK.

Go for the Millennium, if you've got a choice, it's a screamer with the new
XFree86 3.3 release... :-)
