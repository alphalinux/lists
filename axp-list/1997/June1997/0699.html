Two things: if you're running on an 8-bit display, turn on
(install-colormap) in ~/gimp/gimprc. Then, if you're using an older
X-server (such as the 3.1.2 TGA server), then be sure to use option
--no-xshm when invoking gimp.

That should give you a working GIMP.
