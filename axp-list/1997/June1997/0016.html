Yes, and no; Netscape does not exist for Linux alpha per se. I have seen
on this list that it is possible to run the DEC UNIX version if you can
get the appropriate libs. However, I have not done it myself. It is
possible to use the em86 libs (ftp.digital.com/pub/DEC/Linux-Alpha/em86/)
and run the ix86 version on the alpha. The readme can give more info on
this. We tried this here, but were having some problems with Netscape
complaining about a getsockopt() error that we have been unable to track
down as yet.
That said, there are other options:
ftp.azstarnet.com has a binary of Mosaic for the alpha,
arena which I think comes with the alpha distribution,
or Lynx, which is a command line browser, and can be installed along with
the distribution.
There may be others, but these are the ones I have knowledge of.

Hope this helps,
Patrick
