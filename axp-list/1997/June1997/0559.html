After many installations, I standardized this process on UDB:

1) if I am installing from scratch, I will do a ftp install.
I would install the base only. Then would I ftp linux site
to get all the other pkgs. Then install th rpms from there.

2) if I already have linux, then I will use the old system
to fetch at least the base rpms/dirs. then I can do an install
from local harddrive.

Can you also install the base first, and then try to use CD-Rom?
