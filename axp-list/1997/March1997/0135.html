Hmmm... I'm running an old version of Linux Alpha (kernel 1.3.89) and am
using loadable modules for slip, ppp, st (scsi tape), sg (scsi generic
for a scanner), mssound and appletalk. All of these modules work just
fine. I have had some problems with the PCMCIA modules since the insmod
version that came with this release does not suppport passing parameters
to the module at load time and I need to pass some timing parameters to
run a PCMCIA modem. The PCMCIA modules did work OK for a Busstoaster
PC-Card though.
