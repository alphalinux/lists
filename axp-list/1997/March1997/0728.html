> Would someone be so kind to explain how one achieves this? I'm a
> little clueless with this...

You can create a shared library from a static library like so:

gcc -shared -o libfoo.so -Wl,--whole-archive libfoo.a -Wl,--no-whole-archive

For the task at hand, you'll want to put this libfoo.so in
emacs' /usr/lib/emacs/alpha-linux/version/ directory and add

-Wl,-rpath=/usr/lib/emacs/alpha-linux/version

to emacs' link line.
