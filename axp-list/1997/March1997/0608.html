I don't have the documentation with me, but as I recall it states that you
need "matched pairs" of SIMMs due to the width of the address path. You should
be able to to use two 32MB and 2 4MB SIMMs as long as they're installed as
pairs. If there's any doubt, try removing the pair of 4MBs and see if the
problem goes away.
