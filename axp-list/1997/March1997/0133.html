raster@act.net wrote:
>
> Has anyone had any luck using the UFS filesystem?

Unfortunately there is no "the UFS filesystem" but a whole pile
variations on the theme which differ enough not to be totally
compatible. Whatever you will find in Linux sources was done
by people with some particular variant on hand and it may, or may
not, work "as is" with some other instance of the beast.

If you can dig out low-level information about what you are really
trying to read it should be possible for you to modify UFS file system
driver for what you have on hand.
