Upon resetting my PC164 system time and rebooting
this morning, the system fails to boot
at the point where the network is brought up,
and the 3C509 card broadcasts continuously,
consuming nearly 100% of the ethernet bandwidth.
I'm not sure these two events are connected.
The system was working over the weekend, with
an incorrect time setting.

This raises another question: Is there documentation
about how system time should be configured?
The RedHat install process gives two choices
with respect to setting system time.
Which one is appropriate for a PC164?
