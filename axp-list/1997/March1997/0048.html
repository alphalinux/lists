The gdb binaries shipped with Red Hat 4.1 and Red Hat 4.0 don't debug
shared libraries properly. The new version, available from ftp.redhat.com as
/updates/4.1/alpha/gdb-4.16-6.alpha.rpm fixes this problem. It should work
on both Red Hat 4.0 and Red Hat 4.1 machines.

This fix is only needed for Alpha machines.
