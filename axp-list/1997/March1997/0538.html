Robert Allen Barker <rab@np.ac.sg> wrote:
> I would appreciate it if someone could help clarify which of the new Alpha
> chips have what set of bytewise instructions. I had heard:
> DEC's new 21164 (533,566,600) for the new 21164 LX boards has a subset
> of the MMX insts. as documented in the Alpha Architecture Manual.
> and
> The Misubishi/Samsung chip (21164PC) has a set of instructions called MVI
> (motion video instructions).
> Q. Do both of these chips have some bytewise instructions?

Both of these references are to the 21164PC. This chip has the MVI
instructions (which are referred to in the Alpha Architecture Handbook Version
3 as "MAX"), and which are conceptually similar to Intel's MMX. The original
21164 and the 21164A do not have these instructions. I suspect that the 21264
has them, but haven't verified it since I haven't been able to get 21264 docs
yet.

However, if you just need byte access to memory, that was originally added
as part of the Byte/Word eXtension (BWX) to the 21164A, and is supposed to be
in all Alpha chips following it, including the 21164PC.

The AMASK instruction can be used if it is necessary to determine at runtime
which architectural extensions are supported by the processor.
