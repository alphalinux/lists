> Anyway, aside from the ghostscript problems, I'm getting an aweful

Can't help you with ghostscript problems, but Printing-HOWTO
has a pretty long section describing how to avoid staircasing.

> I'm using the deskjet as the device since
> it's 100% compatible with the HP Deskjet

In that case this simple filter installed on a printer queue should
do just fine:

#!/bin/bash
#
# Prevent "staircase" effect when printing on HP printer
#
if [ "$1" = -c ] ; then
cat
else
echo -ne \\033\&k2G
cat
echo -ne \\033\&k0G\\f
fi

A need for a final form-feed depends on your printer and/or printcap.

This part really has nothing to do with axp.
