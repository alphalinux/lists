>>> "Scott A. Barker" said:
> Can someone please give precise instructions for setting up X windows
> with RedHat 4.1. I'm lost. The files called for in the older
> instructions don't exist on the 4.1 cd-rom. RPMs are there but the text
> accompaning the cd doesn't specifically address the TGA setup.

If I remember correctly, the install screen that lists the large number
of
graphics devices has, at the very bottom, a "MISC" or "OTHER" category.
If
you select that, it gives a much shorter list, one of which is TGA.
Selecting
*that* causes XF86_TGA to be installed. Answering the other questions
did
leave me with an XF86Config file that worked. I think you can initiate
the
whole X install by running "Xconfigurator".

Good luck.

--Jay++

I just now kinda hacked my way through it. I went to the RedHat/RPMS
directory on the CDROM and rpm'd various X* files until it was satisfied
that I had installed all the dependencies (libs, config, etc).

When I ran the configurator, I made sure that I only selected an 8bit
resolution (leftmost column) and when asked for the video chip selected
"Unlisted" then TGA on the next screen.

I know it's not much to go on but it seemed to work.
