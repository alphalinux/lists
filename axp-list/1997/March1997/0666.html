Paul,

Something I found with the RH install on my AS255/233 - If you've
used fdisk and installed a pc style partition table the RH install
process ignores the BSD disklabel. When I tripped over this I used dd
and wrote a single 512 byte block of zeros to block 0 and wiped out
the fdisk partition info. RH install then found the BSD disklabel and
installed quite happily.
