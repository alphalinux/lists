Subject: Will POVRay work on an Alpha?
Author: <axp-list@redhat.com> at INTERNET
Date: 3/14/97 4:25 PM

POVRay is a freeware raytracing package, very popular and,
so I hear, very sophisticated as well. Source code
available on the net (www.povray.org). It runs on a number
of platforms, including DOS and Linux/Intel. On an given
Intel machine, POVRay renders graphics about 25% faster
under Linux than under DOS. This is significant: a complex
graphic can take days to render.

I'm close to buying an Alpha, but before I do I need to know
that POVRay will run on it. There are apparently some
issues about 32-bit vs 64-bit word size, etc, things that I
don't really understand.

Do any of you on the list run POV under Linux on an Alpha?
What might be the issues when trying to compile such a
package under 64-bit Linux?

I'm sorry for asking such a broad question here, but if
anyone can help, I'll appreciate it very much.

Pete
