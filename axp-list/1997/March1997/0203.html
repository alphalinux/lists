I still do not expect surprises. The problem is that 'autoconf.h' header,
as generated by 'configure', is totally out-to-lunch. I did not check
if this is by a design or whatever glimpse is using to generate this
file is simply broken on Linux/axp. After I edited this file by hand
glimpse compiled to something. :-) Below is included result of my
edits.

Michal
