I'm trying to install RH on an Alphastation 500 (400 MHz). I'm making
some progress; I got an Alcor aboot disk (for 4.1), and a 4.0 root
disk. (My CD is 4.0.) I'm trying to install on /dev/sdb (dka300);
/dev/sda (dka0) has (had?) Digital UNIX on it. I'm using SRM.

I boot the system, and it takes me through the fdisk process. I
create sdb1 as type 83 (regular Linux), and sdb2 as type 82 (Linux
swap). When the install process asks for the swap partition, it only
offers sda2 as a choice. Disabling the DKA0 drive (making DKA300 the
only hard drive) runs me through fdisk with no problems, but then
tells me that there is no swap partition available.

If I try to go without a swap partition (128MB RAM), it goes to "Setup
filesystems" for a second or two, then reboots.

What undocumented step am I missing???
