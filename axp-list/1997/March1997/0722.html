On Tue, 25 Mar 1997, Michael Champigny wrote:

> Can anyone from Red Hat tell me whether OSS/Free and SoftOSS will be
> included in the next RH release? I didn't see any licensing
> restrictions on their web site.
OSS/Free is included in kernel source distribution (in 2.0.x it's under
another name) so it's automaticly included in all Linux distributions.

SoftOSS is only available as a part of the commercial OSS package. Source
code of it is not available.

> I know that 4-Front does support a commercial version of this software
> for Alpha, but I'm inquiring about the free version with the identical
> API.
The API will be identical between OSS/Free and the commercial OSS. However
some low level drivers (such as SoftOSS) will not be included in OSS/Free.

SoftOSS doesn't define a new API. It just makes it possible to play wave
table quality MIDI without having a wave table card.
