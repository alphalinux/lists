Hello;

I have two observations to share concerning running Insure++ under Linux.

1. You will need to learn patience when using Insure++.
POV has been running for 6 hours and is still parsing the test input
scene.
2. You will need either a large amount of real memory or increase your swap
space through the use of swapfiles.

I have it running on a machine where it took POV only 6 hours to render the
test scene.

I am not sure that the advantages that Insure++ provides are worth the extended
run time required by it.
