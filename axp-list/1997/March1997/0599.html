Perl on RH4.1 was compiled with a library that did not include the crypt
function. In the process of getting the SRPM and recompiling it I found
that I had to modify the spec file.

The following is for those folks at RedHat.

Description: Practical Extraction and Report Language
Name: perl
Version: 5.003
Release: 7
Copyright: GPL
Group: Utilities/Text
Source: ftp://ftp.funet.fi/pub/languages/perl/CPAN/src/5.0/perl5.003.tar.gz

%prep
%setup -n perl5.003

%build

# Perl doesn't find dlopen() in /lib/libdl.so, so we have to force it to.

%ifarch i386
sh Configure -des -Dprefix=/usr -Darchlib=/usr/lib/perl5/i386-linux/5.003 -Dd_dosuid
%endif
%ifarch sparc
sh Configure -des -Dprefix=/usr -Darchlib=/usr/lib/perl5/sparc-linux/5.003 -Dd_dosuid
%endif
%ifarch alpha
sh Configure -des -Dprefix=/usr -Darchlib=/usr/lib/perl5/alpha-linux/5.003 -Dd_dosuid -Ud_setruid -Ud_setrgid
%endif
%ifarch m68k
sh Configure -des -Dprefix=/usr -Darchlib=/usr/lib/perl5/m68k-linux/5.003
-Dd_dosuid
%endif

make

# Strip binaries (done now rather than at install)
strip perl
strip suidperl
strip x2p/a2p

%install

rm -rf /usr/lib/perl5
make install
install -m 755 utils/pl2pm /usr/bin/pl2pm
#install -s -m 6755 suidperl /usr/bin/suidperl
(cd /usr/include ; h2ph *.h sys/*.h linux/*.h asm/*.h)

# Fix the header file that declares NULL...
%ifarch i386
sed 's/( \&void \*) //' \
/usr/lib/perl5/i386-linux/5.003/linux/posix_types.ph > /tmp/$$.change
mv -f /tmp/$$.change /usr/lib/perl5/i386-linux/5.003/linux/posix_types.ph
%endif
%ifarch alpha
sed 's/( \&void \*) //' \
/usr/lib/perl5/alpha-linux/5.003/linux/posix_types.ph > /tmp/$$.change
mv -f /tmp/$$.change /usr/lib/perl5/alpha-linux/5.003/linux/posix_types.ph
%endif
%ifarch sparc
sed 's/( \&void \*) //' \
/usr/lib/perl5/sparc-linux/5.003/linux/posix_types.ph > /tmp/$$.change
mv -f /tmp/$$.change /usr/lib/perl5/sparc-linux/5.003/linux/posix_types.ph
%endif

%files
/usr/bin/sperl5.003
/usr/bin/perl5.003
/usr/bin/suidperl
/usr/bin/perl
/usr/bin/a2p
/usr/bin/c2ph
/usr/bin/pstruct
/usr/bin/s2p
/usr/bin/find2perl
/usr/bin/h2xs
/usr/bin/h2ph
/usr/bin/pl2pm
/usr/bin/perldoc
/usr/bin/pod2man
/usr/bin/pod2html
/usr/bin/pod2latex
/usr/lib/perl5
/usr/man/man1/a2p.1
/usr/man/man1/s2p.1
/usr/man/man1/perl*.1
