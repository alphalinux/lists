Hi all,

I just wanted to let you people know that our Alpha Server 2000 is now
booting automagically. Because I didn't want to repartition to bsd disklabel,
I set the environment variables to booting from the provided boot-floppy,
with the appropriate file/flags.

>>> set bootdef_dev dva0 #floppy
>>> set boot_file "vmlinux.gz"
>>> set boot_osflags "root=/dev/sda3"

This works a little slower as from harddisk, but it's no pain.
As the machine is in experimental status now, we propably have to reinstall
it anyhow in a week or two, and then I will do it with disklabeling.

So thanks are going to:
The RedHat Team, I guess you have another supported system,
Jay Estabrook, for providing the working boot-floppy,
All the other people who have contributed to linux, axp-linux, redhat-axp-linux.

A small question at last, when starting up, I get a message form syslogd:
unknown facility name "ws"
Is this something to worry about?
