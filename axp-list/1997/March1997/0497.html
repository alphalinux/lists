How are your disk partitions setup??
Installing NT/Linux on the same disk can have some pretty weird things
happen. You must make sure that the Linux partition is a Primary
partition. I have an UDB (166) that is setup dual boot with NTFS and NT,
and Linux. It took forever to get it installed correctly because the NT
install had changed my partion to an extended partition, and when ever I
installed Linux it would always fail.

Just a thought, hopefully it may help you.
