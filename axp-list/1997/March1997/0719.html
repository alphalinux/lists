On Tue, 25 Mar 1997 JAMES.JOHNSON@bge.com wrote:

> I have a Digital AXP PCI33 "NoName" motherboard and I would like to
> know if the onboard IDE controller is supported under RedHat Linux for
> both booting and installing using IDE hard drives and IDE CD-ROM's.
>
It does - the first week or so after I got my board (AXP PCI33), I used a
Wester digital eide drive, both to install from and boot on.

I also had a 6x wearnes eide cdrom drive - I couldn't use it
to install from, but after I compiled a new kernel it worked fine.

The interfase is only ide and not eide though, and the performance of
the drive was rather slow. But it worked until I got a scsi disk.

I hope this is any help.
