> I have the file installed, and all of the appropriate stuff. Here is a
> snippet from the error I was receiving when trying to compile msql 2.0B5.
> I have seen the exact thing with a number of other packages when trying
> to compile. Thanks for any help.:

<kotter><arnold><Ooh!>

OK, finally one I've figured out!

> In file included from /usr/local/include/sys/resource.h:27,
> from msqld.c:54:

This is the first hint of trouble, so, look in msqld.c, on line 54 and you
should see #include <sys/resource.h> or something similar (working from
memory).

Before this line, insert #include <asm/resource.h.orig> and it should compile.
You may run into this in other files in the same "package" as well.

Hope this helps.....
