On Thu, 13 Mar 1997, Matthew Geier wrote:

> 1.2.29 will boot into single user mode fine. Every thing seems to run
> (except that even with modutils-2.1.28 , I still cant get modules to work.)

1.2.29, have you been digging deep? :)

Anyway, when I had trouble with compiling modules on new kernels
(>=2.1.18) this was solved by upgrading binutils. I don't know if this is
your problem with modules, but if you like to give it a try, I have an rpm
with a new binutils in:

ftp://wopr.campus.luth.se/pub/redhat/RPMS/alpha/binutils-970127-1.alpha.rpm

I am still using modutils-2.1.23 with linux 2.1.26, and it works just
fine. There is a modutils rpm in the same directory if you can't get your
modutils-2.1.28 to work.

Happy moduling!
