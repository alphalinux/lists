I'm delighted that a number of listmembers are interested in
running POVRay benchmark tests. I've uploaded a zipped
source file to my web page. There's also a link for anyone
interested in getting a copy of POVRay. The page is at

http://homepages.packet.net/schaeff/povmen.html

The zipped file includes source files and the necessary
GIF image maps. It's about a meg and a half.

Be sure to render the image at 800 x 600 pixels, with
antialiasing set to 0.3. (The correct command line is on
the web page). Send your results either here or via private
email: tell me your machine setup and the time to render.
I'll gather all the results together and put 'em on the web
page for everyone's future reference. Whatever platforms
you care to test, I'll be happy to post the results. I'm
most interested in Alpha performance, but it'd be great to
have others represented, too.

That's it. BTW this has also been announced in the POVRay
forum on Compu$erve.
