Hi,

Maybe a bit out-of-subject, but I wonder if someone manage to install the
Squid cache properly on an alpha. I succeeded on Linux-Intel and
Linux-Powerpc, but not on alpha.

The problem arises when I try to run it as root. There is a configuration
that lets you to give a UID/GID to run-as. But I dont succeed to make it
work on my UDB running dynamic libs (961212-3). I already post the same on
the squid list, but nobody seems to be on alpha-linux on there.

I see also that there seems to be problems when gethostbyname() is called.
Squid being unable to find what localhost means. I help it putting
straight 127.0.0.1. Could that be a clue?

Out of this Squid compiles with no problem on Alpha, and runs smoothly if
started as no-root and referencing localhost as 127.0.0.1.
