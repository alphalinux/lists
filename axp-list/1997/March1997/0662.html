Don't underestimate the "hidden costs" that can creap up and make a $500
do-it-yourself bargain turn into a $1300 project. Starting with a cheap
motherboard is one thing. Buying an acceptable amount of memory (32M
minimum these days, especially for an Alpha), keyboard, mouse, floppy,
CD-ROM, case, power supply, hard disk (SCSI probably for an Alpha which
gets expensive, the CD-ROM may also need to be SCSI)... it can add up.

Don't get me wrong, piecing together a bargain system can be a kick, but
if money is an issue, proceed with caution.
