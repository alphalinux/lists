On Thu, 29 Feb 1996, Chris Ingrassia wrote:

> #3: Are there any
> mpeg layer 3 players that work under Linux/Alpha?

You are in luck! :)

The guys at Fraunhofer IIS has released a free, working, inefficient
source when they made their technical report on mpeg 1 layer 3.
This source can be found in an archive called mpeg1_iis.tgz or
dist08.tar.gz (the latter is including more junk and an encoder which does
not seem to work at all). Anyway, this source is running about 5 times
realtime on my Noname 266MHz.

By the way, does anyone know about a free mpeg layer 3 encoder source
except the one in dist08.tar.gz?

Then, there is a guy here in Sweden which has made some optimizations and
released his work free. His name is Johan Hagman. The program has been
renamed to mpeg3play and can now play directly to the audio device. This
program takes about 85-90% cpu while playing realtime on my axp.

Now, I have looked into the source myself and I have done some
optimizations which on the same machine makes the cpu usage drop to about
50-55%. Unfortunately the source is pretty ugly right now and it doesn't
support any other layers than layer 3 at this point, so I won't release my
source just yet (I am planning to rewrite it pretty much when I get the
time). However, I can give you a binary which works.

Note that my cpu usage figures are true only for some mp3 files 44kHz, 16
bit stereo. There are others which take more, I think this might be
non-joint stereo modules or something like that.

A tip about playing is to only use mpeg3play to decode to standard out.
Then use bplay to do the actual playing. This program buffers up the data
so that the sound isn't chopped up every time you move a window and such.

Also, process priority can be used to get rid of this problam. I guess
it won't be a problem on a much faster machine though... Here is a simple
script for playing one mpeg layer 3 stream which naturally only works for
44100 kHz, stereo, 16 bits:

#!/bin/tcsh

if ($uid == 0) then
\nice -n -20 mpeg3play $1 -r | bplay -S -s 44100 -b 16
else
mpeg3play $1 -r | bplay -S -s 44100 -b 16
endif

Here are the pointers to my program and bplay:

ftp://wopr.campus.luth.se/pub/axp/mpeg3play
ftp://wopr.campus.luth.se/pub/redhat-contrib/RPMS/alpha/bplay-0.95-2.alpha.rpm
