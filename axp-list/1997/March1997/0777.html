At 02:55 hrs. 03-27-97 -0400, you wrote:
>Hi All,
>
>Has anyone heard about linux having problems on the alpha with file sizes
>over 2GB. Apparently this has something to do with the ext2 filesystem.
>Any info would be appreciated.
You can't boot the system from a partition over 2GB. You can create and use
non root partitions over 2GB, but may have troubles creating or formatting
them depending on what version of fdisk or mke2fs you are using.
It can be done.. with some qualifications..email private for details if
more info needed.
