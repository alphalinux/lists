Peter,
I have the same box and have had to do the by-hand install. Its
fairly easy to do, just follow these steps... (If there is a better way yet,
please let me know...)

1. Mount your RH CD, and navigate to the /X directory
2. Copy the axpbin-ELF-EVx-test.gz and the alpha-Xtga-BETA-0.0.README to
your home dir.
3. Gzip -d the *.gz file, this is the X server, then copy (or mv) it to this
file name...XF86_TGA.
4. Then, edit the *.README file (w/ vi or something), and delete the first
part of the file until you get to the line "# this file generated........."
then save this file as "XF86Config" in you home dir.
5. OK, your almost there... copy the newly named XF86_TGA to /usr/X11R6/bin
6. Create a symbolic link from the server to X thusly:
rm /usr/X11R6/bin/X <cr>
ln -s /usr/X11R6/bin/XF86_TGA /usr/X11R6/bin/X
7. Chmod the /usr/X11R6/bin/XF86_TGA file so you can use it.
8. Copy the XF86Config file from your home dir to /etc/X11/XF86Config (yes
overwrite the old one)
9. startx

Viola, you've got an Xwindows system. (unless you or I goofed somewhere)
Give this a try and you should be up and running. If not, take 20 paces and
draw!

Good luck, Tiger
