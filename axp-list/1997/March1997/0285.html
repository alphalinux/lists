> Does anyone have any experience using this card?
> (It's not listed on the Xconfigurator list but can I use
> another one?)

There should be the following entry in /usr/X11R6/lib/X11/Cards

NAME S3-864 with STG1703
CHIPSET S3-864
SERVER S3
RAMDAC stg1703
NOCLOCKPROBE

This entry should work with that card.
You may need to run xf86config to configure the Xserver correctly.
