> * where can I find the latest version? What's the latest version number
> (major, minor, plevel; not the date version)? :-)

ftp://prep.ai.mit.edu/pub/gnu/glibc-2.0.1*.tar.gz

> * are there special linux alpha patches to go with it?

Not that I know of.

> * are there redhat spefic modules that I should include?

No. Do be sure to get the linuxthreads package with that. You should
get crypt from ftp.ifi.uio.no due to stupid US export laws.

> * something else I should know?

If you are using the binutils from Red Hat, you need to add

ifeq ($(subdir),elf)
sysdep-CPPFLAGS += -DNO_AXP_MULTI_GOT_LD
endif

to a file configparms in the build directory. You'll also
likely want a line

default_cflags = -O2 -Wall -pipe

in that same file. Finally, do read all of the README and INSTALL
files -- glibc set up is a little more than "./configure;make".
