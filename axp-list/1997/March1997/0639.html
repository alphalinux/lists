Actually, it became possible just yesterday... :-)

I was (finally!) successful in booting LINUX via AlphaBIOS/linload/MILO
on a DPWa-433 yesterday afternoon; I ran X11 and did a kernel compile with
no instability.

However, problems still remain with the IDE support (hangs the machine doing
a MILO "show"), and the fact that the built-in Ethernet chip is not (yet)
supported by a LINUX driver.

I hope to get the IDE problem fixed very soon, but don't know when a driver
for the Ethernet chip will be available. Temporarily installing another
supported Ethernet card would certainly be a viable workaround. And if you
only intend to have SCSI peripherals, the IDE problem disappears... :-)

I'll try to make appropriate MILO/kernel available in the next few days;
watch this list for the announcement.
