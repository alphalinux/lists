I agree here, we have a PC164 and a few UDB's which we use for various
tasks. The PC164, when I can keep it stable, is hands down the best
machine we have (Intel,Sparcs,etc.). However, stability is a bit problem
with these boxes, at least for some people. I think many of the problems
will be resolved by using Jay Estabrook's new milo's and redhat 5.0, as
glibc2 solved a lot of problems for me.

However, that said, if you are planning to use this as a mission critical
device, or put it into production right away, I would probably think
twice. You might be in for more than you bargained for. If you are
looking to tinker for a while, and be patient learning the quirks of the
platform, then go for it. They are great machines, and once running
properly nothing else you can buy can come close for the money.
