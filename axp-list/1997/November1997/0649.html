On Sat, 22 Nov 1997, Duane Sessions wrote:

> Is Linux now stable enough on an Alpha to run an ISP? Is anyone one the
> list using Linux on an Alpha as a Server?
>
> I am considering using two DEC Alpha Multia 166MHz systems to run an ISP. I
> would appreciate any comments, good or bad.

DEC AlphaStation 4-233, 233MHz, 96MB, Linux 2.1.24, RedHat 4.2.

10:22pm up 97 days, 3:17, 1 user, load average: 0.43, 0.37, 0.33

This system is used daily as a fileserver for a number of Win95 computers,
does e-mail for 60 people, runs a webproxy, and serves homepages.
Furthermore, people run X on it (Netscape DU, Lyx) all day. It's quite
busy.

With this configuration, it just doesn't crash. Choose your Linux kernel
with care however, other kernels have been less satisfactory.
