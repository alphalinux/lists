Hi.

I am interested in the performance of fortran code on DEC
Alpha/Linux machines generated by different compilers.
Did anyone do a comparison of say,
g77 / Microway NDP F77 / DEC F77 ? (anyone else?)
On our DEC Workstation (@ 400 MHz) I find the performance of my
computational chemistry (Molecular Dynamics) code rather disappointing:
it runs at aboud 60% of the performance of an SGI R10k @ 195 Mhz.
This probably has to do with a sqrt operation which the R10k has in
hardware, but the silly DEC compiler just puts a function call in my code,
probably inhibiting any pipelining...

Will my code improve if I compile with NDP fortran?
Does NDP inline intrinsic math functions?
Does G77?

Groeten, David.
