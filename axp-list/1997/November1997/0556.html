Enough of all this nonsense about ftp release dates. Mustang (pre
5.0) is visible on Redhat's ftp server, and the devel tree is visible
too. Are people really complaining that they don't have a perfect ftp
mirror of the CD before it comes out?!?

Here are some *real* questions about Redhat 5.0:

1. What kernel does it include?
-What kernel version?
-What patches are they applying?
-Are they applying alpha-patches-2.0.30-0.2a.gz for SYSV IPC
fixes & em86?
-Are they applying ap-0.2-fixes-1.diff to fix the trap handling
in the above patch?
-Are they applying any weird home grown patches that haven't
been on this list?

2. What gcc does it include?
-Does it include Richard Henderson's large stack frame fix?
-Does it include the profiling patches that have been floating
around?

3. What binutils does it include?
-Does it include David Mosberger's gprof patch?
-Does it fix the header file conflict with glibc-headers?

4. What libraries does it include?
-Does it include the binaries from David Mosberger that fix
profiling?

5. Did they fix the netstat alignment problem?
