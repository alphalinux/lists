On Wed, 12 Nov 1997, Harvey J. Stein wrote:

> yue-feng sun writes:
> > Wes's results show that Alpha 533 MHz 21164 with
> > ALphaPC 164LX is 3.5 times faster than a 200MHz PPro ( so 4.5 times
> > the speed of the PPro), so one Alpha 533 MHz has the speed about a
> > 300 MHz Quad Pentium II.
>
> In theory. In practice there is no Quad Pentium II. The Pentium II
> can only run dual.

I must disagree. Just like i486s, Pentiums and PPros, the PII is able to
run in SMP in amounts up to sixteen without special trickery and up to
sixty four with the so called clustering. It is only the matter of used
chipset. Popular chipsets are not designed to carry the maximal number of
processors. But it is only the matter of cost to get an appropriate
motherboard which supports the necessary amount of processors.

As with the PII it may seem ridiculous to use large number of processors
in SMP, though, as this processor does not really support more than 512 MB
of memory.
