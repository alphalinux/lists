Yes I have profiled my app, 80 % of the time is in one loop that has
the sqrt, or actually 1/sqrt which the DEC F77 compiler recognizes
and puts in a call to rsqrtf. However, the compiler should inline the
code for the rsqrt and then pipeline my loop, which now it doesn't.

So I will probably write F77 code for the 1/sqrt (I have C-code for it)
unless I can find a smarter compiler
