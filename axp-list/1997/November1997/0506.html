% More like they don't want to kill their CD market by having it
% available for FTP before. Both the FTP and CD version will most likely be

It doesn't kill any market, and I've never seen ANY other vendor pull
this. You can get Slackware, Debian, FreeBSD, etc.. from FTP sites
about a month before you can buy a CD. Ideally FTP versions should be
released one week + replication time before a CD is ready so that the
authors can fix any initial bugs that pop up. (No matter how much
beta-testing, some little slip-up always gets through).
