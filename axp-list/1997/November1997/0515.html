Jeremy Sieminski <devil@hell.roc.servtech.com> writes:

handy util if you haven't used it. Anyways, my question is why am I using
so much ram? Below I have a screen capture of my stats sorted by memory
usage and everything seems ok, but below that, I have another one where I
am using practically all of my ram.

You should feel lucky---otherwise it would be wasted money :^)

Normal:
Mem: 94152K av, 25424K used, 68728K free, 15592K shrd, 6696K buff

Abnormal:
Mem: 94152K av, 91904K used, 2248K free, 15584K shrd, 58440K buff

This is as it should be: buff refers to buffer cache, which stores
disk data in your RAM, speeding the disk access enormously. When
memory is needed for running programs, buffer cache is turned over to
them immediately, so there is practically no downside to it.

Note that you don't use your swap---an indication of ample physical memory.
