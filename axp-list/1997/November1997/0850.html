Hi,

Add me to the seekers for good, fast compilers for Linux/Alpha.

I do programming and CFD related work and speed on the Alpha is very important
for my applications. The current compilers for Linux are far inferior to the
DEC f77/f90 compiler for computing applications. I actually use DEC Unix
rather than Linux most of the time (although I would rather run Linux) as a
result of the speed penalty on Linux. I tried the early Microway f77 compiler
and found it slow to compile and not as good at optimizing code as the DEC
f77/f90 compiler, also it was not compatible with the rest of the Linux system
libraries at the time. I have not tried the latest Microway compiler but I
doubt that things have caught up with the excellent DEC compilers.
