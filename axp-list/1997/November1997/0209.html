>>> set console serial

By doing this the Digital Unix install program will consist of
a text based install (through a serial console) instead of an
X based one (in which a serial mouse may/will not work)

This is enough to get started. Now I have not seen any
problems with my AT keyboard and Digital Unix thus far.

As for the serial mouse and X I feel sure that this can be.
If you check out /var/Xserver.conf (I believe that's it)
in there you will find what appears to be a commented out
option to define the mouse as a serial MS compatible mouse.

I have not yet set this option (properly) and gotten X to work
so if anyone has any information on exactly what the line
should say, I would be greatful.

Hope this helps someone
