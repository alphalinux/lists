Dave Wreski wrote:
> I just wanted to make sure I understand this. There is a Digital version
> of netscape, I believe. But we can't run it because the Digital libraries
> needed are copyrighted?

That is correct. In other words, it it *technically* possible to haul
the
Digital UNIX libraries over to Linux/Alpha and run Digital UNIX binaries
that way. Whether you can do so *legally* is another matter. Right
now,
you can only do so if you have a Digital UNIX license for the machine
you're
running Linux/Alpha on.

EM86 is one way of making certain pieces of software available on
Linux/Alpha.
There is another way that I thought of, which unfortunately I'd need
lots
of help with to bring to fruition: implementing clean-room clones of
the
Digital UNIX a.out libraries. This is a large undertaking, but the
result
would be access to all the software that currently runs on Digital UNIX,
running at full native speed.

Any volunteers? 8-)
