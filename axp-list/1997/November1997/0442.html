Hi there,

it seems the IDE controller (the CMD646 chip) on the LX won't be supported for a
while, so has anyone tried using an external IDE controller on an Alpha?

Especially interesting would be something like the Promise ultra33 or fasttrak.
The ultra33 spec says it automagically configures the drives to ultra/dma, so
could that mean that it isn't necessary to have a special driver for the onboard
chipset as the driver just sees it as any IDE controller sitting on the PCI bus
and defaults to standard ATA, but with the drive correctly configured for
maximum transfer rate. Could this work?

Of course, any autoconfiguration-stuff usually means something gets fucked up
under Linux...

Anyway, has anybody out there tried an external IDE controller and did the
performance get anywhere near what it should be (>10Mbytes/sec)?
