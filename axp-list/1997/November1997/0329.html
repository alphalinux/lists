Yes, attached to a PC164. No problem, except the port address (or
whatever it's called...) in the source is wrong for the alpha. If you're
using ppa as a module, try "insmod ppa ppa_base=0x3bc". If ppa is
compiled into the kernel, boot it (from MILO) something like:
...vmlinux.gz root=/dev/sda3 ppa=0x3bc

This should do the job. Please specify more details when you ask!
