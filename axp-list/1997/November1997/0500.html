For RedHat 4.1, I needed to apply two patches posted by David
Mosberger-Tang to this list:

http://www.redhat.com/support/mailing-lists/archives/axp-list/1997-March/0463.html
fix for gprof,

http://www.redhat.com/support/mailing-lists/archives/axp-list/1997-March/0464.html
fix for three object files in libc_p.a.

Someone from the list told me that these are applicable to
RedHat 4.2, too (hope this is of any help for you).

I never had problems with the math library.
C++ is no problem.
I always use static linking for profiling.
