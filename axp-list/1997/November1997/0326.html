We just got some new RPX164-2 based alpha systems (boards from
deskstation). I'm just curious if anyone else out there has one of
them too or if we're the first ones (well, I had this impression from
some difficulties in getting things working and all that).

The machines:
RPX164-2 500Mhz
256Mb
4 Gb disk
SPEA V7 Mercury P64V (4Mb)
Eizo FlexScan T67S
RedHat 4.2

We got them working now more or less with NFS, NIS, amd and all
that. There are still minor problems in parsing OSLOADOPTIONS from the
ARC BIOS to milo and the workaround hack for using NIS maps with amd
isn't as nice as I want it to be.

We're planning in installing DQS 3.1.* on them: any experience with
that?
