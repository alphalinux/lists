If it's not shipped as a module, you'll have to recompile. ZIP could be

> Disk Druid and what advantages does it have?

I tried to install 4.9 but it wouldn't let me do it via FTP. Disk Druid
is a horrid piece of, uh, software. I suppose I shouldn't be so critical
becuase I was having other problems at the time with my hard drive, but
Disk Druid set up a lot of overlapping partitions and a bunch of other
nasty problems. Maybe it knew something fdisk didn't, I don't know, but I
can't say that Disk Druid is so great. The only advantage I see upfront
is that it moves partitions around (you don't specify which parition
number to create). What this does aside from that I don't know.
