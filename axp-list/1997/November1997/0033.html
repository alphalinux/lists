rrsantos@inf.ufrgs.br (Rafael Ramos dos Santos) writes:
> IS there anybody which can tell me where I can find bootdisk and MILO for
> jensen machines? I have RH 4.2 Linux for alpha and as described in
> the user's guide I could need jensen.img to make a bootdisk for dec
> axp 150, 2000 model 300. So, there aren't any files with similar
> names in the distribution CD.
>
> I'm looking for in the web and now I already know that jensen images
> can't be found in the 4.1 and 4.2 because it still missing on this versions.
>
> Can someone guide me to find these files in a public ftp site?
> Currently I already search ftp.redhat.com and on infomagic site. I
> didn't find anything.
>
> Is it possible to use old version of jensen.img (RH 4.0) ?

Yes. But the Jensen usually isn't able to boot from the builtin floppy
drive (You could boot with an SCSI floppy drive, but those are rare
...) So, you need the bootdisk only to keep the installation process
happy. So any other bootdisk image will do at that stage as well.

But you need a Jensen kernel to install it on your hard drive first.
There's more information available from my home page, see below.
