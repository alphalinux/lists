An 21164A does not have a hardware sqrt instruction, so changing
compiler will not improve the performance of your code, if sqrt
is really the critical part of it (profiled your app?).

So you have the options
- to write code that approximates sqrt
- to use a CPU with sqrt instruction (I think both the 21164PC
an the still not released 21264 do have one)
