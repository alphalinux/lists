Hi folks,

as far as I am new in the alpha-linux world but somehow familiar with
intel-linux, I have some questions regarding redhat and alpha-linux.

1) On the intel platform, I used fvwm2 as window manager. My redhat installation
(version 4.2) only gave me binaries of fvwm and fvwm95 (very scary...) to work
with and no trace of fvwm2. Because I want to run the alpha in a network and
under a common user interface, I am strongly interested in running fvwm2, for I
have configuration files for this one, prepared for the local network and the
environment (e.g. buttons to start jobs on other machines in the local network).
Does a fvwm2 exist for alpha-linux, can it be easily obtained and installed (via
glint or rpm) or will I have to compile the sources taken from intel-linux?

2) Surely this is a beginner's question, but regarding alpha-linux I am in fact
new: I haven't yet understood how to perform intel-linux binaries on the alpha.
I heard of em86, but what is it exactly? Is it a kernel patch? And talking of
kernels: can I take every kernel I get (e.g. 2.0.32) and simply compile it on
the alpha, or does this not work?

3) I read some mails dealing with g77 that really concerned me: are there known
problems running g77 (or any other fortran compiler) under alpha-linux? All the
programs i intend to run on the alpha are written in fortran, so it is of great
importance that there is an existing and running compiler for them. I saw
benchmarks for the alpha (I have a PC164LX with a 533 MHz processor) saying that
it should be at least three times as fast regarding floating point performance
as a pentium pro 200. So I hope that I can reach this goal with my machine,
otherwise the investment would have been in vain... On the other hand, if there
are problems with the g77, are there any commercial fortran compilers for
alpha-linux (there seem to be several for intel-linux).

4) Last question: many of the above questions may be obsolete with the
appearance of redhat 5.0 . Does this version already exist or is the date of
publication already known?

Thanks for answers,

Dirk !!
