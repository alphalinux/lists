There were, at least last spring when I purchased my millenium,
three different versions:

1. OEM version. This is the version that many (most?) oem
vendors put in their machines unless told differently. The ramdac
runs at 175 MHz.

2. The retail version. This is what you usually get when you
buy it in a store. But check to make sure it's not the oem
version. The ramdac on this one runs at 220 MHz.

3. The PowerDoc version. The ramdac on this one runs at 250
MHz.

The only difference that I'm aware of on these boards is the
ramdac speed. The higher ramdac speeds allow you to run the
boards at higher refresh rates at a given resolution. For
example, let's say you've got the board set at 1024x768x24 with 4
MB memory. You could run the oem version at max 60 Hz refresh,
the retail version at max 72 Hz, and the PowerDoc version at max
90 Hz. NOTE: all of these numbers are fictional - I'm just using
them as an example, but you should get the idea. Check your
board's docs for actual numbers.

Adding more memory only allows you to run at higher resolution
and/or colours.
