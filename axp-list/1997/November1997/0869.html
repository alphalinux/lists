Hugo,

Om 23:31 op 29/11/97 schreef je:
>
>I do think that the current information regarding Linux on the Alpha
>platform could be improved.
>
>I therefor suggest we create a new document which will have the following
>information:
> - Describtion of the various types of the Alpha systems. Preferably a
>list of machines for each group.
> - Describtion of the required changes, etc to get the boot process
>working with Linux.

The FAQ (at <http://www.azstarnet.com/~axplinux>) contains a list of Alpha
platforms that will run Linux. There's also a reference to a document which
gives a more general description of the Alpha architecture (This last
document is, admittedly, somewhat outdated.)

It also has references to milo.html and srm.html (at the same location)
which describe the boot process.

>We may add other chapters to the document as needed. The document will be
>GPLed and I would also suggest it will be written in SGML.

The FAQ is in fact written in SGML, following the linuxdoc conventions, and
is available in a number of other formats as well.

>I will try to give this a head start next week but need lot's of input on
>the various types.
>
>I strongly feel such an document is not difficult to write if enough
>people make their small contributions. But I know that plenty of people
>would save lot's of times if this documents lives up to it's expectation.
>
>I will let you know where you could find a HTML version of the concept as
>soon as I have it written and online.
>
>Hugo.

I do agree that there is room for improvement, but in my opinion it would
be better to update the existing docs, rather than start yet another one.
