> Is Linux now stable enough on an Alpha to run an ISP? Is anyone one the
> list using Linux on an Alpha as a Server?

Let me say that I love Red Hat and I love my Alphas ( I have quite a
collection.) That being said, I would not use Linux for an ISP on an
Alpha. There are too many pieces missing for me (NFS would be one, for
instance). I use Linux on a PC for firewalls at many clients and have
run it back when I first started for my server. Linux Alpha has grown
tremendously and might warrant another look in 5.0. I have Alpha
dedicated to running the Linux and we have even done some development
work to mold into what we need.
