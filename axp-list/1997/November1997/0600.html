> > 2.7.2.3, and I honestly don't remember which fixes are in there. rth

>
> Does it include g77?

No, because it changes gcc's backend and isn't well tested. People have
found bugs in gcc which resulted frmo g77, and I'm not going to break
the widely used C compiler for a seldom-used fortran compiler.

The good news is the egcs group, which is working on extending gcc in
various ways, incorporates g77 in their standard release. Once egcs
is released (it's in alpha test right now), both egcs and egcs's
fortran support should end up in the standard Red Hat release.
