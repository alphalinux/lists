On Wed, 26 Nov 1997, David Cabrero Souto wrote:

>
> Is there any FAQ, doc, ... on how to write 64 bits clean code ?
>TIA

I would say that reading any good text on programming will cover proper
portability issues. The key is not to make any assumptions about the size
of various variable types and definitely not the size of pointers. Also
be careful about typecasts, alot of problems with the Alpha misalignment
code seems to be typecasts of for example a pointer into a long integer
which are not the same size on the Alpha. They tended to be the same
size on 32 bit processors.
