I've just got my hands on a Digital Personal Workstation 433a, but I
have problems running RedHat 4.2 on it.

I use the miata milo/kernel from ftp.digital.com, which recognizes my
qlogic scsi-card. I installed RedHat 4.2 from another harddisk because
I didn't get the ethernet card to work, and can now boot from sda1.

The problem is these weird IO errors, ie:

[root@pcf10 /root]# clock
clock: unable to get I/O port access

I get something similar when trying to start X (it has a Matrox
Millenuim card so I use XFree86-SVGA-3.3.1-1):

xf86EnableIOPorts: Failed to set IOPL for I/O

Has anyone else had these problems, and been able to solve them? I
have read about people running RH on 433a's so it should be possible.
