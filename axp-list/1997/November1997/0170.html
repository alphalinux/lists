Joo NamSoo wrote:

> I have a problem when I install Digital 500 network card.
>
> I got the message as follows:
>
> "I cannot run /bin/insmod:No such file or directory"
>
> I'd like to know what insmod do.
>

안녕하세요...
insmod라는 것은 커널중에 모듈로 설정한 것을 필요할 때 찾아서 커널에 넣어 주는
것입니다.
위의 메시지를 보지 않으려면, 그리고 어떤 네트웍 카드인지 확실히 안다면 커널 컴파일을
다시 하셔서 모듈이 아니라 커널내에 확실히 속하게 만드는 것입니다.
커널 컴파일에 관한 자세한 문서는 한글 리눅스 그룹에 커널에 대한 HOWTO가 이미 번역이
되어있습니다.
