Actually information on the embedded storage shelf in the Alphaserver 1000
is likely to be found on the Alphaserver page as it is not a separately
marketed storage product. My point about the autodetection and disk BIOS
was simply that the DEC specific firmware in the drives may have a problem
with some non-DEC controllers. In any case you raise a good point about
the shelf firmware. If this is truly an AS1000A then the both the RZ29B-VW
and the DS-RZ1CB-VW WILL work fine with the shelf as they are the standard
drives that ship with that system, however if this is the older AS1000 (non
'A') which was designed with a narrow SCSI controller then it may have some
problems with the RZ29B-VW. I can try to verify this through tech support
next week but I suspect that we will receive a more authoritative answer on
this list before then.
