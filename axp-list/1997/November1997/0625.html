With regards to your message at 09:20 AM 11-22-97 -0800, Duane Sessions.
Where you stated:
>Is Linux now stable enough on an Alpha to run an ISP? Is anyone one the
>list using Linux on an Alpha as a Server?
>
>I am considering using two DEC Alpha Multia 166MHz systems to run an ISP. I
>would appreciate any comments, good or bad.
>
>Thanks,
>Duane Sessions
In answer to your questions:
Yes, it is certainly stable enough ( although not "out of the box" from a
Red Hat install) Some tweaking is definitely required. Also making the
right hardware option choices is pretty important, in areas like disk
interface and ethernet.
There are several ISP's running Alphas, especially as web servers running
Apache.
Using 2 UDB/Multias would work, but not for many users. Remeber you are
basically proposing to use 4 year old tech for the project. Kind of like
saying your are going to use DX2/66 or DX4/100 Intel technology. With the
lack of expansion slots, it is also a bit limited.
