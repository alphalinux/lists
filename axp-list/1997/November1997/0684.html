With regards to your message at 05:41 PM 11-19-97 -0500, Christopher
Curtis. Where you stated:
>
>Okay, time to install.
>
>I bought an IBM 4.3GB disk but don't know how to format it. I'm in the
>process of going though the RedHat install and entered fdisk. It can't
>recognize the number of heads, cylendars or sectors automatically, so I'm
>on hold with tech support. Their spec sheets say 8025 cylendars, 16
>heads, and 171 sectors. However, fdisk only allows me to have a maximum
>of 63 sectors. :(
>
>Tech support didn't help. Anyone?
Multiply 8025x16x171 to get: 21956400
Divide this by 64 to get:343068
Divide this by 32 to get: 10720
Guess where you can put those numbers?
I fdisk type x for expert mode.
Input heads, cylinders, and sectors per track using 64H, 32SPT, 10720C

return to main menu. Partition disk.
SCSI drives use logical translation.
Any combination of numbers that add up to a total less than the real are
legal...
