There is also another way to get Digital-Unix apps to run on Linux.
If you have the source, you can compile on the DU machine with
-non_shared flag enabled to force a a statically-linked binary to be
created. Then you can copy the binary over verbatim to the Linux
machine and it will run. I can do this for FORTRAN apps since I also
have the Digital-Unix FORTRAN compiler.

Of cource, this gets to be a problem with Netscape since the source is
not available. What we need to do is get the Netscape folks to
compile a version for us or perhaps loan them an account if they don't
have an Alpha with Linux.
