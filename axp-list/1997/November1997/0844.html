On Fri, 28 Nov 1997, Keith Baker wrote:

> I am trying to install ncftp. It fails dependancys on libnsl.so.1 (I think
> thats it...) How can I figure out which RPM that library is in? Is there

I don't know. http://www.rpm.org/ However, if you install the ncftp RPM,
it should tell you which dependency you need. I'm sure you have nls on
your machine. Add "-lnls" to the LIBS line of the Makefile.

> a "cute" graphical ftp program for the alpha under X?

Look for ftptool. I know of no others offhand.

> Also, how can I launch an xapp from an xterm and make it create a seperate

Type:
xapplication &

> process (so my xterm is still useable?) Where can I edit the FVWM popup
> list of programs?

/usr/X11R6/lib/X11/fvwm2/system.fvwmrc (may vary by system)

Also look for a program called "dotfile". Editing FVWM options by hand is
not really a trivial task.

> These are probably all silly questions...

Rather basic ones, yes...
