On Nov 18, Amer M Mallah wrote:

> tar fc /dev/nst0 -V "Volume name" -g /var/backup/systemA.inc .
>
> for each machine. The problem is restoring: How do I restore the SECOND
> tar file. For example,
>
> tar -xvf /dev/tape
>
> only restores the from the first tarfile made from the method above. But I
> have files on the second. Any ideas what I can do?

depends on where /dev/tape points to.
IMHO it should point to /dev/nst0 and _not_ to /dev/st0.
then you always have to explicitly rewind using `mt rewind'
but you can do

tar -xvf /dev/tape # extract saveset 1
tar -xvf /dev/tape # extract saveset 2
tar -xvf /dev/tape # extract saveset 3
mt fsf 2
tar -xvf /dev/tape # extract saveset 5
mt rewind
