tfreeman wrote:
>
> I've been asked to compile a program, call it "program.cpp" with the
> g++ compiler, and I have little idea of the command line switches -
> and no idea where to look at the moment.

$ gcc program.cpp -o program
tries to produce a binary 'program' from source file 'program.cpp'.
You can give more than one source file to gcc at a time.

For further compiler options look through info files, e.g. in
emacs type C-h i (Control-h followed by i) to get in info mode
and look for gcc.
