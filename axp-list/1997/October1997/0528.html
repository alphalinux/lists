It really depends on the application. Under NT and Lightwave rendering is
(surprised) 10~20% faster. The 211PC processor has 8kb 2nd level cache and
1mb 3rd level running at 7.5ns. Some applications slower but not by much.
Please note up until now all the motherboards we've seen lately use a 9ns
3rd level cache. I believe the 7.5ns picks up a wait state which accounts
for some of the speed.

Best we can tell Lightwave does not sit in cache so its not dependent on its
size. However, anything running on FX!32 are so cache size and speed is a
big factor.

Later this week I'll have a few benchmarks posted on our web site for your
viewing.
