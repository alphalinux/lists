Try to compile the file with -O2 or less. I had the same problem
while compiling SSLeay... Aparently certain optimizations trigger
that behaviour. I could identify -funroll-loops to be the culprit in
my case. I think that -O6 includes that option.

My recommendation is to never use anything higher than -O for
reliable code.
