I just installed Linux RedHat 4.2 on my newly acquired UDB (Alpha).
Its got the 166MHz processor in it. I installed 32MB of memory in it
and a 2GB Fujitsu SCSI 5400RPM external hard drive.

This machine is, to put it humbly, slow as hell. I was hoping to replace
a Pentium 75 (Linux) machine with it. But the Pentium machine with less
memory and a slow 3600RPM IDE drive is screaming fast compared to this Alpha.
I am using the stock kernel that came with RedHat because I can't get anything
else to compile right for me.

Does anybody know what the problem could be? Has anybody put up one of the newer
versions of compiled kernel and modules for UDB that I can download to experiment
with? While I am at it, asking for help, can anybody email me their XF86Config file
for UDB?
