>>> "bdwheele@indiana.edu" said:
>
> I've got an XLT-300, but it seems that *ONLY* AlphaBIOS is available
> for it. If I could get SRM for it, could I not boot Digital Unix or (gasp)
> OpenVMS?

There is no *official* SRM console for the XLT; it was designed/sold strictly
for WindowsNT. That's not to say that that hardware doesn't strongly resemble
another Alpha platform that *does* have both ARC and SRM. However, hardware
details may *not* be identical, and console code is *VERY* sensitive to such.

In other words, we don't recommend running anything but what was intended
to be run on the platform, WRT installed console code.
