> I'm looking for a backup utility that can backup across the network to a
> single Colorado Jumbo 1400. Basically what I have is 3 UDB's with
> 3 hard drives totaling 4.5 GB running Redhat 4.2 (2.0.30) and 1 Intel PC
> with 2 hard drives totaling 3 GB running Slackware 97 (2.0.30).

I don't think you can put a tape drive in a UDB, so I'm assuming the drive is
on your i386 machine.

If you want reliable backups, and a program that supports all tape drives,
check out BRU, from www.estinc.com.

I think the network version is a little more expensive than the single-machine
version, but its still only like $200 or so..

It really is a great program. I just never could understand using tar to
backup my tapes, which is what most of the linux-based programs (shell
scripts?) use.

You could also use amanda for network backups, and you can use dump/restore
with that..
