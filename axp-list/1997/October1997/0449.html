At 01:10 PM 10/27/97 -0500, you wrote:
>I have a 166mhz UDB ( soldered on cache and processor ) running RHL 4.2
>with X and want to get 800 X 600 hi color out of my TGA video. It was
>preset ( I guess, because this is the only resolution it works in ) 640 X
>480 256 color. I have scoured the FAQ's and archives, but can't find any
>pictures or explanations of locations for the video setting jumpers. I
>have multiple FAQ's regarding what jumper goes where for what resolution,
>but I have 2 separate groups of jumpers on my mlb, neither of which are
>labeled. I dunno what to change!

Unless the previous owner of said UDB messed with the pins you should not
have to set anything. In configX if you don't setup the refresh freq
exactly correct. I have RHL 4.1 and I was able to get 1024x768 right
out of the box, I just ran Xconfigurator and let it set things up.

Did you run Xconfigurator?
Did you select "unlisted card" and then the TGA xserver?
Are you sure you put the correct vertical/horiontal refresh values in
for your monitor?

If you still need the pinouts I cutout a graphic from the UDB manual
(which I found somewhere in PDF format, I can't remember where). I will
