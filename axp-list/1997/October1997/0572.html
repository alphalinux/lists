Stephen Tow writes:
> Howdy,
>
> I have a problems that I have not been able to figure out for the life
> of me...
>
> Why doesn't rsh/rlogin work when logged in as root? It works fine for
> normal users, but not for root. I'm sure this is some sort of security
> measure, but is there some way around this? Without it it's pure hell
> to try and propagate a common file across a dozen machines.

It's a security measure. If you figure out how to use PAM, you can
probably figure out how to enable it. One way around it is to nfs
mount the dozen machines onto one machine.
