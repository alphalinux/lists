It is not a problem of Alpha Chip but a problem of Fortran compiler
(not very optimized). You will have the same problem (and more) with a
Pentium Pro 266, do not forget that Pentium is only 32 bits processor
(bad performance in double precision computes) and has not secondary
memory cache.

The only advantage of linux Alpha System is when you have already
bought an Alpha station with Digital Unix because you can compile
with the DU Fortran compiler (which is very performant) and execute
your program on the Alpha Linux system.
