Hello everyone...
i was able to compile red hat linux 4.2 on Alpha (21164, 64MB RAM)
only with your help.
i performed the following steps:

make mrproper
make config
make dep
make boot
make modules
make modules_install

the kernel image is kept in the file vmlinux under
/usr/src/linux.
i did a gzip to this file in vmlinux.gz and put it under '/'
replacing the old toy one.
now, when boot i get messages like:

".....
Unable to handle kernel paging rquest at virtual address
0000000000000178
swapppr(0): ops 1
..."

thanks for all the helpful minds out there...
