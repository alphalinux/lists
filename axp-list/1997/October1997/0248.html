>> a Pentium 75 (Linux) machine with it.

>It can run database queries (Mini-SQL) on very large
>database (>5000 rows) faster than my P100 or my P133 (laptop).

This is (IMHO) the point. The Alpha architecture seems to be optimized for
maximum data throughput, using twice the bus width of a Pentium (64/128 bit).
So everything that operates on a large amount of data, e.g. the typical
system load of a file server, database applications or similar, should
perform much better on your Alpha than on a comparable Pentium platform.

Tight instruction loops however, that are executed within the internal
CPU caches, perform better on equaly clocked CISCs as they need less
instructions to do the same thing (roughly spoken).
