On Wed, 1 Oct 1997, M.M. SPITTERS wrote:

> Can somebody please help me with my problem?
>
> When I want to add a new user (adduser name) my system replies:
>
> "/etc/passwd is locked. Try Again Later."
>
> Rebooting my system has no result. And passwd is readable and
> writable for me, being root. What's wrong?

Remove the passwd lockfile:

rm /etc/.pwd.lock

This file is there to ensure mutual exclusion to the password database,
but sometimes it's left there by mistake after the password is already
changed.
