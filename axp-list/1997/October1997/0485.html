On Tue, 28 Oct 1997, Keith Griffith wrote:

> I just started using Linux on the alpha, and am not familiar with some of the subtle differences with the alpha version and the Intel version. I am trying to compile a custom kernel as follows:
>
> I run make confide and set the kernel up.
>
> I used to compile as follows "make zlilo"
> "make modules"
> "make modules_install"
>
> Of course this won't work. I use milo. What to do?

Easy (once you are told ;-) as they told me just last week.

cd /usr/src/linux
make config (if needed)
make depend
make clean
make boot
cd arch/alpha/boot
mv /vmlinux.gz /vmlinux.old.gz
cp vmlinux.gz /vmlinux.gz

Hugo.
