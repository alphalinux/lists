> I have tried few days ago to download the new version of realaudio player
> beta 5.0, specifying platform DEC Alpha and OS Linux 2.x ELF, and it did not
> tell me that a corresponding version does not exist, but then I found that
> the executable is anyway for Pentium. Does anybody know whether it's likely

If I was Real Com. I would not implement a native plugin for an OS that has
no native browser to use it for. So, since there is no IE/Netscape for
Linux/alpha native, I would not implement a native Real player. After all,
how big is a market that runs emulated software mixed with native software.
It is reasonable to expect that if the browser is emulated, that the plugins
are emulated as well.
