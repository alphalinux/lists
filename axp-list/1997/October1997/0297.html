Just some more info on the subject.

This is the number of iterations/minute on a small program that
makes simple calculations and constantly sends the results via UDP
datagrams. I ran both the client and server on the same machine, thus no
real network involved. Hardware: Sun SPARCstation 10, dual SuperSPARC
@40MHz, 32MB RAM and DEC UDB 21066A @233MHz, 32MB RAM.

SS10, Linux 2.0.27: 66000 iterations/minute
UDB, Linux 2.0.27: 144000 iterations/minute.


These are the memory copy bandwidth results from the stream
benchmark:

SS10: 43.7 MB/sec
UDB: 29.8 MB/sec
SS2: 29.4 MB/sec

The SS2 is an old headless SPARCstation 2 with 32MB IIRC.

According to the CPU Infocenter homepage
(http://infopad.eecs.berkeley.edu/CIC) the 21066 @233 is rated at 94
SPECint92 (no SPEC95 numbers listed). There are not numbers for the
SuperSPARC @40, however linearly downscaling the results for the 60MHz
version gives 60 SPECint92. The same linear downscaling gives 68 SPECfp92
for the SPARC while the 21066A performs at 110 SPECfp92.

So, the UDB's Alpha is faster than the SuperSPARC on the SS10,
however the memory bus on the Sun is substantially better. The SS10 is a
very low-end one, sure, but IPX, IPC, SS2 and family won't be any faster,
I suppose; at least the SS10 is a sun4m (before we completely forget, this
thread was about old Suns versus UDBs)

On the other hand, I've received a couple emails regarding Linux
versus Solaris on the same machine (SPARC). I know this is somewhat
offtopic here, but anyway: this is from a couple of benchmarks I run quite
a while ago. Let's complete the first test:

SS10, Solaris 2.5: 65000 iterations/minute
SS10, Linux 2.0.27: 66000 iterations/minute

Now, results from IOZONE (throughput on a 1MB file):

Solaris 2.5 on UFS:

4673880 bytes/second for writing the file
8324541 bytes/second for reading the file

Linux 2.0.27 on ext2:

6553600 bytes/second for writing the file
9532509 bytes/second for reading the file

(I know, both Solaris 2.5 and Linux 2.0.27 are old, but they were
quite current when I ran the tests :)
