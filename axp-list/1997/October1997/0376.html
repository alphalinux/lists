Hi, we have a simple program that crashes our alpha linux boxes
(Redhat 4.2) when run. The program is a Fortran program
translated by f2c and statically linked. Bellow is the program
and its translated c code. Any help, suggestions or ideas would
be greatly appreciated.

translate and link:
f2c testf1.f
gcc -o testf1 testf1.c -lf2c -static

When run testf1, it will crash our alpha machines. If you need
the binary file or additional information, I will email it to you.

Thanks.

Ming

Fortran program testf1.f:
============================================================
program test
c-----------------------------------------------------------------------
integer i
integer*2 i2
real*4 r4
real*8 r8
c-----------------------------------------------------------------------
do i = 1, 10
i2 = i
r4 = i
r8 = i
write(6,*) i,i2,r4,r8
enddo

end
