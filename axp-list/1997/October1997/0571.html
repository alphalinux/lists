hello all,

I just wanted to get some opinions on how other clusters are
run/suggestions on what I should do with mine..

I have several alpha machines (and a few x86 machines too) I want as
part of a cluster. How does everyone handle home directories for
users?

My thoughts were to put them all on the server machine and have each
machine run amd and mount from the server as necessary. However, I
suspect that the hard drives may be too small individually. In that
case, I thought it might be a better idea to put several home
directories on each machine. This would mean each machine would have
to be an nfs server. Is this decentralization reasonable? Does it not
defeat the purpose of having one "server"?

Finally, I wasn't too clear on whether or not this had been solved,
but does NIS work on alpha Linux yet? The latest news from the
mailing list seemed to indicate that it may work if a significant
amount of effort was put into it (not a problem) but it wouldn't
really work (a problem).
