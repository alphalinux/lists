Huw:
I ran into the same thing. Apparently the dump version wan't updated
when the file system version was. Make the following change to the dump.h
file and recompile it. (the value was 0)

#define DUMP_CURRENT_REV 1

As far as I can tell, this version of dump works fine with the current
ext2fs. It was just complaining for no good reason (?).
