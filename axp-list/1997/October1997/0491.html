I did put a full load of binary RPMS for the Alpha platform on
/pub/incoming. I guess they will be moved to /pub/contrib/alpha in due
time.

Here follows the whole bunch of headers off all packages uploaded. I added
my name as the packager so if this packages breaks you know how to
contact. Off course the only garantee I can give is that it will consume
diskspace. It did work for me but your mileage may vary.

AfterStep-1.0-3.alpa.rpm:

Summary: AfterStep Window Manager (NeXTalike)
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Ximenes Zalteca <ximenes@netset.com>
Copyright: GPL
URL: http://www.afterstep.org/
Group: X11/Window Managers

AfterStep is a continuation of the BowMan WM. It uses modified FVWM code
to bring you a very NeXTStep-ish experience in X Windows. However, it
doesn't stop there, and can be extended beyond just being a simple
NeXTStep clone.

This version includes Rob Malda's cool3.diff

WindowMaker-0.6.1-1.alpha.rpm:

Summary: WindowMaker WM
Copyright: GPL
Group: X11/Window Manager
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>

The WindowMaker window manager for X11

ascd-0.7-2.alpha.rpm:

Copyright: distributable
Group: X11/Applications
URL: http://www.cs.hope.edu/~malda/linux/mystuff.shtml
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Ximenes Zalteca <ximenes@mythic.net>
Summary: The AfterStep/WindowMaker CD Player

This program is a thermal detonator. Use it to either create
vast devastation over small countries, or else you could
use it to play CD's. It's your choice.

This puppy works best with AfterStep or WindowMaker.

ascdc-0.3-1.alpha.rpm:

Copyright: distributable
URL: http://www.cs.hope.edu/~malda/linux/mystuff.html
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Ximenes Zalteca <ximenes@netset.com>
Group: X11/Applications
Summary: The AfterStep CD Changer

Use this bad boy to swap CD's graphically under X. It is designed to work with
4 CD changers (My NEC 4x4 Specifically) but should work with any drive that
uses ATAPI swapping properly. Of course, the icon will seem wierd if you have
a 6 or 3 changer...

This puppy works best with the AfterStep Window Manager.

asclock-1.0-1.alpha.rpm:

Copyright: Freeware
URL: http://afterstep.edoc.com/
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Ximenes Zalteca <ximenes@null.net>
Group: X11/Utilities
Summary: The AfterStep Clock

A clock written to emulate the date/time application on the NeXTStep OS. asclock
supports multiple languages, military and AM/PM time, program execution, shape
extension, and multiple color depths. Goes best with AfterStep, but can be
coaxed into running under any WM. This version is compiled with English language
support.

asmail-0.50-3.alpha.rpm:

Summary: Mail notification program (much spiffier than xbiff).
Copyright: distributable
Group: X11/Utilities
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Ximenes Zalteca <ximenes@netset.com>

A graphical mail notification program for X11. Best used with the
Afterstep window manager.

asmixer-0.5-1.alpha.rpm:

URL: http://www.cs.hope.edu/~malda/linux/mystuff.html
Copyright: distributable
Group: X11/Applications
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Ximenes Zalteca <ximenes@netset.com>
Summary: The AfterStep Mixer

This program is designed to rain fire and brimstone upon a thousand lands
crushing an infinite number of souls with devestation never before
experienced by man. Either that, or it might just be a simple
Mixer Utility for Linux systems. Requires /dev/mixer to work. The
3 buttons are quite simply Volume, CD, and PCM.

And of course, asmixer looks best under the Afterstep WM!

asmodem-0.6-1.alpha.rpm:

Copyright: distributable
Group: X11/Utilities
URL: http://www.cs.hope.edu/~malda/linux/mystuff.html
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Ximenes Zalteca <ximenes@netset.com>
Summary: Afterstep/WindowMaker Modem app

Afterstep modem status indicator. Capable of running commands when
it detects that the modem has taken a bad turn. WindowMaker is also
supported.

The three modem lights function as TX/RX/CR indicators, with the
TX/RX lights able to blink when data is transfered.

bash-2.01-2.alpha.rpm:

Summary: GNU Bourne Again Shell (bash)
Group: Shells
Copyright: GPL
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>

Bash is an sh-compatible command language interpreter that
executes commands read from the standard input or from a
file. Bash also incorporates useful features from the
Korn and C shells (ksh and csh).

Bash is ultimately intended to be a conformant implementation
of the IEEE Posix Shell and Tools specification (IEEE
Working Group 1003.2).

This package is compiled with default support for cd spelling, i.e. minor
errors in the spelling of a directory component in a cd command will be
corrected in interactive shells.

kdebase-beta1-3.alpa.rpm:

Summary: The base applications that form the core of the KDE Desktop Environment
Group: X11/KDE
Copyright: GPL
Distribution: KDE
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Kalle Dalheimer <kalle@kde.org>
Vendor: The KDE team

This package contains the base applications of the KDE Desktop

kdelibs-beta1-2.alpha.rpm:

Summary: KDE Desktop Environment (KDE) - Libraries
Group: X11/KDE
Copyright: LGPL/GPL
Distribution: KDE
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Kalle Dalheimer <kalle@kde.org>

This package contains the libraries that all KDE applications build

Group: X11/KDE
Copyright: GPL
Distribution: KDE

kdesupport-beta1-2.alpha.rpm:

Summary: KDE Desktop Environment (KDE) - Support Libraries
Group: X11/KDE
Copyright: LGPL/GPL

Summary: Utility programs or the KDE

magicfilter-1.2.alpha.rpm:

Summary: MAGICFILTER is a customizable, extensible automatic printer filter
Copyright: GPL
Group: Utilities/System
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>

A customizable, extensible automatic printer filter.
Lets you automatically detect and print just about any
data type you can find a conversion utility for. This
filter is written in C and is controlled completely
from an external printer configuration file.

qt-1.31.alpha.rpm:

Summary: The Qt GUI application framework: Shared library
Copyright: distributable
Group: X11/Libraries

Contains the shared library needed to run Qt applications, as well as
the README files for Qt.

qt-devel-1.31.alpha.rpm:

Summary: Include files and documentation needed to compile
Group: X11/Libraries

Contains the files necessary to develop applications using Qt: header
files, the Qt meta object compiler, man pages, HTML documentation and
example programs. See http://www.troll.no for more information about
Qt, or file:/usr/lib/qt/html/index.html for Qt documentation in HTML.

remind-03.00.16-3.alpha.rpm:

Summary: REMIND is a sophisticated alarm/calendar program
Copyright: GPL (almost)
Group: Utilities
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>

Full-featured calendar/reminder program featuring sophisticated date
calculation, moon phases, sunrise/sunset, Hebrew calendar, alarms,

proper handling of holidays. Available for UNIX, MS-DOS, OS/2, and
other platforms.

sox-11gamma_cb3-1.alpha.rpm:

Summary: general purpose sound file manipulation tool
Copyright: distributable
Group: Applications/Sound
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Manoj Kasichainula <manojk@io.com>

The self described "swiss army knife of sound tools", sox can convert
between many different digitized sound formats and perform simple
sound manipulation functions. With the included script, sox can also
play sound files.

xmp3player-0.02.1-2.alpha.rpm:

Summary: mp3 browser
Copyright: GPL
Group: Applications/Sound
URL: http://cray.tuug.utu.fi/~phoenix/xmp3player/
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; nathan.hand@anu.edu.au

This application uses QT to allow you to interactively select
MP3 files. It has an mp3 browser and queue and also features
a useful volume control. By default it looks for MP3 files in
/tmp but this location can be overridden on the command line.

xperfmon-1.1-2.alpha.rpm:

Summary: A performance monitor for Linux with X Window System and /proc
Copyright: GPL
URL: http://pioneer.arc.nasa.gov/~rsmith/xperfmon.html
Packager: Hugo van der Kooij <hvdkooij@nllgg.nl>; Michael J. Long <mjlong@mindspring.com>
Group: X11/Utilities

A performance monitor for Linux with X Window System and /proc.
Shows selected machine load statistics, and can show
warnings when statistics go above or below settings.
Should work with Linux kernels 2.0.x, 2.1.x, and 2.2.x.
This version uses /proc.
