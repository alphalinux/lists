I have an AS500 on which I run redhat 4.2. It was a challenge to get
started but finding
ftp.digital.com:pub/Digital/Linux-Alpha/Kernels/milo-2030-alcor
was the key I needed. After that it was as straight install.

I believe there is a tulip chip in the onboard ethernet controller.
The DE4x5 driver does not work for me but the tulip driver does. The
trouble is that it doesn't work too well. The best I can get is
50KB/sec on a 10Mb/sec ethernet. Have others done better?

We have another alpha system from microway. This contains a DE500
ethernet board and does run the DE4x5 driver quite well. I tried
installing that board in the AS500 but then no driver would work.
Quite possibly the two ethernet controllers conflict. I looked in the
"service manual" located on the web (the windows program). It listed
jumpers but I could not find a jumper for disabling the onboard
ethernet.

Any help would be appreciated.
