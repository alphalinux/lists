> Does anyone have staroffice running on Linux/AXP? My requirements

> for under $100 that runs relatively well. I've tried LyX and other
> similar WYSIWYG TeX programs, but they are sufficiently strange that
> my users have problems. I'd appreciate any suggestions; I'm
> willing to try Applixware with em86, but I doubt that this will be

Well, I haven't tried this, and I doubt that the $100 requirement would hit,
but WordPerfect for Java may be ok (assuming it's been released finally).
Also, if you have an X68 machine laying around, you can run from that,
and export the display if you're apt. I tend to do that to use the large
monitor hooked up to my SparcLinux box, and Applix seems to run quite well off
of my 486 over a 10Mb/s Ethernet, however there's only one of me, so I
can't comment on scalability.
