You have probably mounted your ext2fs file system in such
way than on disk errors it remounts itself automatically
into a "read only" mode. This is actually a good thing.
Linux, instead of trashing your file system, gives you a chance
to recover gracefuly and investigate the problem. The first
thing I would check is if your log file do not contain
error messages which may indicate disk troubles. There may
not be there if they would have to be written on an already
remounted disk. Checking cables and termination is also
a good idea.

Buggered file system is even more uncomfortable.
Remounting it read/write without first checking
what is really going on may be not so hot idea.
