I would suggest using LyX. It is a graphically front-end to LATeX, uses the
XForms library (available for Linux/Alpha), and is a very nice package. As I
remember, it compiles cleanly on Alpha. Of course, if you have an Intel box
that you have access to on a network, you could install Applixware on that
machine, and using remote X and exporting your home directory to the Intel
box, run Applixware in a completely transparent fashion to your Multia.
Applixware appears to be well optimized to run on X-terminals, so the result
is more than satisfactory. I have this setup and am very pleased.
