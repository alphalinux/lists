>Hello,
> I'm having trouble figuring out how to get TheNextLevel to work
>with my .xinitrc file. If I have an .xinitrc file I can do either fvwm or
>fvwm95-2, if I don't have a .xinitrc then TheNextLevel runs.
>
> How can I get TheNextLevel to run with a .xinitrc file? Sorry if
>this is a dumb question...
>

TheNextLevel stuff is started by .Xsession or .Xclient in your home directory.
If .xinitrc is found, then that is run instead. To configure the TheNextLevel
stuff you need to decide what it is you wish to configure and copy the
appropriate file from /usr/lib/TheNextLevel to your home directory. The
files in that directory are all dotted so you need to do ls -a.

Anyway, If you wish to do some initialization of your X session, such as
starting up your favourite calendar program or modifying X so you can use
your backspace key, then you wish to copy over .fvwm2rc.init. If
you wish to modify the menus then its .fvwm2rc.menus and so on. My
details may be somewhat off, ie the /usr/lib might be /etc/lib but I am
home on my Windoze 3.1 machine and can't verify currently. I typically
power down on the weekend.
