I have just finished putting together some patches to allow four
programs to compile (and work) under Linux/Alpha that would not otherwise.
The four programs are ce-4.3, glimpse-3.6, netshade-0.0.4, and urt-3.0p2.
The patches can be gotten from my web site, in the software section
(http://www-ugrad.cs.colorado.edu/~rkirkpat/software/).
Knowing how fustrating it can sometimes be to get programs that
compile fine on Linux/Intel to compile on Linux/Alpha, I decided to make
these patches so other people would not have to go through the same stuff
I had to get these apps to compile.
These apps were not too difficult to get to compile. Most of the
problems were either missing #include statements (string.h mostly), or the
program trying to supply its own version of a libc function that
conflicted with the glibc of Linux/Alpha. Beyond that, there were no code
reworks required, so one could consider these rather superficial patches,
that tweak a few things to get programs to compile.
More information about these patches and programs can be had on my
web site. I hope these help someone out there! :)
