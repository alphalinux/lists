>I'm interested in getting the hfs_fs module working on
>my alpha, so that I can use a zip drive to shuttle stuff
>back and forth between a mac and the alpha (I can use dos, but
>hfs would be more elegant). having file system access is

Not that yours is a bad idea, but you should be aware that you can use
netatalk on your alpha to transfer files between it and a mac. This is
working just great for me, selected drives and directories on the Alpha
simply show up on the mac desktop and you can copy files to and from them
by drag-and-drop.

I am using appletalk over ethernet, I'm not sure if netatalk can do
appletalk over a serial link or not, maybe if you set up the Alpha as a
PPP server and made a PPP link from the mac? OTOH ethernet cards are
cheap and it's a lot faster than serial.

Anyway, just an idea and one you may not have considered.

I'm not sure about your modules problem, I have read of a lot of folks
having similar troubles but do not remember seeing an answer. Modules
work for me but I am running an ancient ECOFF version of Linux AXP. I
seem to recall something about stripping the module but I'm not sure.

Good luck.
