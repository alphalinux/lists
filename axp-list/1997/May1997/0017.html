> I have tinkered around with i386, Alpha and SPARC Red Hat Linux
> machines, and I have found them to be VERY consistent. I have also
> found the Users Guide to be very good, there is mre that could be added

The three versions are very consistent from an end user standpoint out of
the box. As development platforms though the alpha/intel versions differ
because the alpha version uses the gnu c library, rather than the standard
libc which the intel version uses.

> I have also found that the Aeleen Frisch (Sorry, I don't have

Again, from an end user/administration standpoint you are dealing with the
same OS/distribution. Its very similar to Solaris sparc/x86.
