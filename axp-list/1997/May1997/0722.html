>>> Rick Windham said:
>
> I tried mkdosfs on a floppy and it said "too few blocks for a
> viable file system." Is there a way to dos format a floppy under Linux?

Did you say:

mkdosfs /dev/fd0

instead of:

mkdosfs /dev/fd0H1440

and had you done a format before it:

fdformat /dev/fd0H1440

The system may get confused by a simple "/dev/fd0" because it erroneously
probed it and thought it was a 2.88M floppy.
