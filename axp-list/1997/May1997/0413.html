Carlos Alberto Carvalho wrote:
> Why isn't -mieee the default?

On Alpha, generating code that complies with the floating-point
exception recovery rules exacts a small but noticeable performance
penalty. The default is for maximum performance, and you have to
*ask* to sacrifice a little performance for correctness...
