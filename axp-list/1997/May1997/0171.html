At 04:51 PM 5/5/97 -0500, Adam Biskobing wrote:
> I searched the archives but didn't come up with any thing on if the 3com
3c905 100mbit card is support by linux yet. Anyone know?

It is, but you have to use the driver for the 3c59x. Make sure its an
older version of this driver, however, because the one included in the more
recent kernels don't work correctly with the 3c90x cards.

I use version that is included in the RedHat 4.1 distributions(kernel
2.0.27) and just copy the drivers/net/3c59x.c to the same path in a newer
kernel.

It works great with 2.0.30.

Hope this helps.

BTW: I've heard rumors of a patch to the kernel for the 3c90x, but have yet
to find it.
