Hello all,

I have a Fortran routine that I link to my C code. I've compiled it
and run it successfully on Linux/Intel using gcc and g77 as well
as on a Sun using their C and Fortran compilers. On my Alpha machine
with gcc and f2c, though, it doesn't work.

The problem is in the Fortran routine as converted by f2c. I can't post
the whole thing, but the offending line is

epsmch = dpmeps_();

where epsmch is defined to be a doublereal.

For testing purposes, I've set dpmeps_() to be simply

doublereal dpmeps_()
{
return 1.0;
}

where the prototype created by f2c is

extern doublereal dpmeps_();

and is located within the function that calls it.

For some reason epsmch is always set to 0. Any idea why this is happening?
