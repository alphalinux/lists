I have the same kind of problems on my RH 4.1 box w/ 128 Mb. The 'setenv
MEMORY_SIZE 127' did _not_ work. I also have the feeling that it is due to
something in the networking code. When I let the machine run a large fp
simulation (which uses nfs) everything seems to run ok for days. When I do
