Ciro Cattuto <ciro@stud.pg.infn.it> writes:
> I ran some numerical code on a 500 MHz 21164 Alpha system (Microway Screamer,
> 2 MB cache) and on a P166. Both systems are running Linux and the code
> was compiled using GCC. Most surprisingly, the P166 seems to be twice
> as fast than the Alpha workstation!! How can this possibly happen?
> Should I use some special optimizazion when compiling Alpha code
> with GCC? What else should I check?

Try '-funroll-loops'. This often helps, and it isn't turned on by any
-O<x> level. Also avoid unnecessary float/integer conversions.
