I am developing a CT (computed tomography) application, which runs
on *THIS* special computer (Highscreen Alpha 5000, PC164 board,
1 MB 3rd level cache, 128 Bit Memory access) about 2.5 times faster
than on a PPro 200 (both with Linux 2.0.X, and gcc 2.7.2.1).

Anyone made performance measurements before/after upgrading *THIS*
special computer to 256 Bit memory access?

My application is floating-point intensive and occupies about
8 MB of memory using a typical set of input data (so it would surely
run much faster on those expensive Alpha machines with 8MB 3rd
level cache).
