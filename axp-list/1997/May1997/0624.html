We are trying to set up virtual ftp servers on our PC164 running 4.1.

We have the following entry in our ftpaccess file:

virtual x.x.x.x /home/server/htdocs/rrs /home/ftp/etc/banner.msg

where x.x.x.x is the IP number of the server,

and the following in our passwd file:

test:password:521:523:RHSLinuxUser:/home/server/htdocs/rrs/./test:/bin/false

/bin and /etc are in /home/server/htdocs/rrs. /bin contains ls and the
other approriate files. /etc contains a passwd and group file with the
approriate entries.

When we ftp in as test, we get to their home directory, but it's like a
normal ftp; we can move about directories, all the way up to top.

What are we missing here?
