>Erik -- is there a list of supported applications? Like Netscape 4.0? I
>tried the OSF2 version but it doesn't like Linux even with the DEC
>libraries.

Depending on what it is that you want to do there are a couple of
programs out there that WILL emulate the x86 processor.

Pcemu compiles and runs OK on the Alpha but AFAIK it only emulates an
8086 chip.

Bochs also compiles and runs, and is supposed to be good enough to let
you run Windows 3.1, at least in standard mode.

Another approach is to run DOS and/or Windows on a real x86 and use
something like Desqview/X to export them to your Alpha as X apps, that
way they at least appear to be running on the AXP machine :-)
