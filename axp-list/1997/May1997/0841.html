# hit alt-fX, where X is the virtual terminal *after* the last one running
# a getty. When you start X, it should report what VT it's using for the
# graphics display.

Alternatively, you can use Alt+Left-Arrow and Alt+Right-Arrow to page
through your VC's. If you hit one of these keys a few times, you'll get
back to your X session.
