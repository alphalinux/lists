Well, I have X11R6.3 (libraries, headers, binaries except twm and X
servers) compiled with glibc 2.0.2 (so must work with any 2.0.x and
2.1.x) using fixed binutils-2.7.0.10 and gcc-2.7.2.2. Available by ftp
from genie.ucd.ie:/pub/alpha/X11R6.3.bin.tar.gz.

Old binaries still seem to work despite that they link old glibc while
X libraries link the new one. This may cause problems, but I haven't
discovered any yet.
