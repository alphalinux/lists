On Sun, 11 May 1997, Murray Todd Williams wrote:

> Heya, anyone got a zip drive to function properly?
>
> I've got a UDB, and I recompiled the kernel to include the parallel-port
> SCSI stuff and I included the generic SCSI support. When I boot the
> machine I get:
>
> PPA: unable to initialise controller at 0x378, error 1
>
> which I really don't understand. Can anyone think of some diagnostic
> things I can do to further diagnose the problem? Thanks.

I had this same problem when I first started using the Zip drive.
Manual says the first parallel port is 0x378, on mine, and this probably
holds true on yours as well, it is actually at 0x3BC (if I remember
correctly) if using PPA compiled into the kernel, you can use an option at
the milo prompt "ppa=0x3BC" (I believe, I am not at my machine right now),
But the exact syntax can be found in the README.PPA file included with the
kernel sources.
