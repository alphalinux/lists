Rick Windham wrote:
>
> Hello,
> I remeber all the discussion of running OSF/1 Netscape
> under Linux and I put the libraries and loader in the correct
> place, installed Netscape. But when I run it it says
> ./netscape: Permission denied.
> I have checked all of the permissions and they seem fine.
> I also have svc.conf and nsswitch.conf in /etc.
> Anyone have any ideas?

Make sure all of the shared libraries that you installed have
execute permission.

In general, whenever you run an executable that uses shared
components, any errors in loading the shared components are
reported against the main executable. F'rinstance, if a shared
library is *missing* and you run a program that uses it, you'll
get back "no such file or directory" even though you can *see*
the executable plain as day right in front of you.
