Lagodzinski, Andrew wrote:

<snip>

> Can anyone tell me the difference in performance between 233 and 266MHz?

Um, 33 Mhz?! Seriously, My machine is overclocked and I have never run
it at "only" 233 Mhz, so I am unable to give you a valid "hands-on"
evaluation, but since we are only talking about a one seventh increase
in processor speed, it probably is not worth the risk. Also, consider
these two recent posts:

- Steve at DCG Computers has sold "hundreds" that were over- clocked
and has not had any returns (This is where I got
my system - thanks Steve, it runs great!)

- I seem to recall a recent thread that discussed the
performance bottlenecks in Alpha systems, and I *think*
that the biggest limiting factor for the UDB was the
bus-width (my recollection may be wrong, but a small
increase in CPU speed will not help in such a situation)

So I would say, if your machine is already overclocked, leave it
alone, if you open the box (to add a card, increase HD, RAM, etc.)
I would probably reset the clock speed to 233 Mhz.

Good luck,
