I also struggled with this Multia. After fighting with memory msg
and other strange error msg. Finally I installed redhat. What I did is:
(1) in a pc,
copy milo a:\fwupdate.exe
copy linload.exe a:\
(2)in arc :
use update firmware
(3)from milo prompt:
milo> boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1 prompt_ramdisk=1
