On Thu, 29 May 1997, Avinash Shenoy wrote:

> Novice question....I'm trying to install RH 4.1 on a AlphaStation 255
> 300 Mhz. It runs the SRM console. I'm able to complete the install fine...
> however, on rebooting, the hard disk is identified as having a unknown "magic".
> Any ideas as to why this might be happening ?
Does the AS255 have the ARC console? MILO gets information about memory
size from ARC. I would think it would be easier.

> PS : I'm using BSD style partition tags as normal fdisk is unable to see
> the head/sector/cylinder nos of the SCSI HDD.~
I've never tried BSD partition tags, I thought Linux supported them. The
reason fdisk can't see your geometry is because it's a SCSI hard disk.
The formula that most people have been using is h = 64, s = 32, c = <size
of disk in megabytes> My Seagate Barracuda is 64,32,4095 in fdisk.
