ximenes@netset.com said:
> Alphas are 64-bit machines, therefore the OS is 64-bit code :)

It's a matter for the compiler, actually. Windows-NT on the Alpha
is a 32bit O/S, as is the MSVC++ compiler - RISC Edition. Linux/alpha
is 64bit.

The other numbers floating around, line 128bit and 256bit, are actually
the width of the memory path. The cache is for example loaded with a
256bit wide access to memory, but the CPU generates 64bit accesses.
