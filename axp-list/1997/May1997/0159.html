On Mon, 5 May 1997, Tim Miller wrote:

> Does anyone here think the bandwidth has gotten large enough
> to warrant a digest list for the axp-list?

Yes.

I subscribe, at the moment, because this list is the only interactive
Internet source for information on Linux-Alpha. I understand an RFD is
currenly underway for a newsgroup, which I regard as a Good Thing. I
presently subscribe only to ask occassional questions as I'm just
starting out on Linux. I hit the delete key for almost every message
after a quick skim. I expect this to change once I have a little more
experience and can answer questions in addition to asking them, but for
now, responses from me would usually just decrease the S/N ratio.

Because this is the sole source of information, I'll continue to
subscribe, but I sure would like it if all the messages were digested so I
could read the batch all at once.

I would find a digest useful for that reason, among others.

(I know, I should get around to using procmail.)
