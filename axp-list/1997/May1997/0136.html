for SYM53C875 in Alpha. anyone with such a configuration ?

-----Forwarded message from Gerard Roudier <groudier@club-internet.fr>-----

From: Gerard Roudier <groudier@club-internet.fr>
Subject: Re: NCR53C8XX (Bsd ported) driver latest versions.

It should be interesting to try a SYM53C875 controller under Alpha.
Is it possible? It seems to me such hardware configurations exist.
I've started a development version of the driver in order to take
into account all features that need large rewrite without breaking
the 1.18X driver serie that seems to be stable enough.

All materials about the driver and especially the patches for the
development version are available at linux.wauug.org/pub/roudier.
Latest stable version is 1.18h. Latest development version is 2.1a.
Obviously, I use 2.1a.

Regards, Gerard.
