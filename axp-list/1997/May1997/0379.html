Hi Gang,

Scenario:

You have a UDB with one SCSI drive. The formatted size of the drive is
345,090,000 bytes. You want to use it as a firewall and IP masquerade
platform. Knowing that those two activities are more memory intensive
than disk intensive, you put in 40 MB of RAM. You are using RH 4.1. You
will connect to the Internet via your ISP with a US Robotics 33.6
Sportster. The local LAN (2 more UDBs and a PC) is 10BaseT.

This machive will do nothing else except firewalling and IP masquerading.

Question:

What is the best partitioning for this drive? Bear in mind that another
drive is NOT an option.

Current config:

Device Filesystem Type Size
--------- ----------------- ------
/dev/sda1 16-Bit MSDOS <32M 2 MB
/dev/sda2 Linux Native 101 MB
/dev/sda3 Linux Swap 40 MB
/dev/sda4 Linux Native 194 MB
------
237 MB

What say?? Any options??

TIA!
