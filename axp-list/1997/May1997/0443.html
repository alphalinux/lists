I have had problems like this and found the problem to be that my milo was
not built with the kernel I was trying to boot. My suggestion would be
get a new kernel along with the milo from the same place, either from
redhat or from gatekeeper or build a new kernel and milo on you own from
source. The second of the 2 options is not too easy and not for the faint
of heart. It must have taken be weeks to do it the first time.

kevin
