>running RH4.2 .
>After I did 'make configure' (I chosed kernel name 'avanti', with SRM boot
>loader), then I executed 'make dep' and 'make clean'.
>So I build all with a single 'make', and obtained the new kernel vmlinux (not

This will leave a compressed kernel in the arch/alpha/boot directory
under where you compiled from. You should be able to boot that kernel
