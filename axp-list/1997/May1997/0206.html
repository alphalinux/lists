Hi,
Is there a gcc 2.7.2.2 for alpha linux? I'm having the following problem:
#include <stdio.h>
#include <math.h>

int main(argc,argv)
int argc; char *argv[];
{
double ti;
double sti;

ti = 2.22507385850720138e-308;
sti = sqrt(ti);
printf("tiny = %e\n", ti);
printf("sqrt(tiny) = %e\n", sti);

ti = 2.22507385850720138e-308 * 2.0;
sti = sqrt(ti);
printf("tiny*2.0 = %e\n", ti);
printf("sqrt(^) = %e\n", sti);
}

yields:

tiny = 2.225074e-308
sqrt(tiny) = -3.408875e-154
tiny*2.0 = 4.450148e-308
sqrt(^) = 2.109537e-154

I was having other floating point problems with small numbers that
going from 2.7.2 -> 2.7.2.1 took care of. The only 2.7.2.2 rpm's I could
find were for i386.
Thanks for any pointers.
