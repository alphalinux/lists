Rick,

You should be able to see a series of jumpers.

CPU speed jumper settings (VX41/VX42 only)

W9 W8 W7 speed (MHz)
-- -- -- -----
1 1 1 66
0 1 1 100
1 0 1 133
0 0 1 166 (standard setting)
1 1 0 200
0 1 0 233
1 0 0 266 (unsupported)
0 0 0 300 (unsupported)

Note: 1 is jumper removed, 0 is jumper installed.

Hope this helps:
