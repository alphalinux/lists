On Tue, 6 May 1997, Britt Bolen wrote:

> My old high school is thinking about putting in a Linux based email/web
> server system, they would have about 325 total users with probably no more
> than 30-50 logged in doing pine at a time, they would also be serving some
> web pages from the system, but not with much load (the current webserver
> is a Mac LCII :) The majority of users would be using POP mail from the
> local network. Would an alpha be better than an Intel box for this kind
> of setup? Thanks

We have an intel based box that does about the same thing.
It's a p166 with 64MB ram. plenty of HD space. It's main role is that of
mail/pop3 server. It's passwd file holds over 9000+ entries, and about
3000-4000 active accounts. It also doubles as an ftp server for the
university. We have very few problems with this machine, and runs with a
load under 0.50. The main reason I'm saying this is that, yes, the Alpha
would be a MUCH better performer... but at about 2-3x the cost of an intel
system that would be able to perform amply.
