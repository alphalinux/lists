> Does the alpha's FPU not support infinities and NaN's ?

Not directly. You must compile with -mieee to get them which
turns on software completion. Incidentally, libm must be
built with -mieee as well in this case.

I'm thinking of making some changes to ld to change the library
search algorithm when -mieee or similar is present. And making
glibc build libm.{a,so} and libm_ieee.{a,so}.

I've been mulling this problem over for several months and this
is the only solution I can see that is workable.
