>>> Bill Broadley said:
>
> Is 2.0.29 on the PWS a bad idea?
>
> Does the redhat-2.0.30 with 4.2 include the needed 21142 support?

Unfortunately, the RH 4.2 miata.gz does *not* include the latest
tulip.c driver, which is the only driver (currently) working with
the 21142. That kernel is sufficient to install from a local CD, but
not via NFS or FTP, obviously. For that, use the latest miata-kernel.gz
from off of gatekeeper.dec.com:pub/DEC/Linux-Alpha/Kernels/ directory.

Please refer to one of my previous axp-list postings re: where to get
Donald Becker's latest tulip.c driver, and what patch to apply to prevent
most unaligned accesses in that driver.
