This seems to be becoming something of a FAQ. Search for relocation
in the redhat archive and you will find the answer. The short
answer is that you need break things up into shared libraries:

ld -shared -o libfoo.so --whole-archive libfoo.a --no-whole-archive

Then be sure to put the library somewhere where the loader will look
for it, or set the LD_LIBRARY_PATH env variable to include the path
to the shared lib.

Good luck!!!
