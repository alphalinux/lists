Do you have ethernet (BNC) actually connected to the box? Linux 4.1
has a terrible bug that occurs when you start up your computer and it
tries to initiate the eth0 and there isn't actually any there it will
hang upwards of 20-25 minutes. Let the thing go (walk away and watch a
show or get something to eat) it will eventually start up. Log onto the
machine and the disable the network config and it will boot up normally.
