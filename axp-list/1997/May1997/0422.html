Hello Kirk

> Has anyone had any success installing Redhat Linux 4.1 on an
>Alpha with an IDE bus? I have an AlphaPC164 Motherboard and it has 2
>onboard IDE controllers. I have installed a Mitsumi 4X CD-ROM (IDE)
>and a Samsung HDD (IDE), however the system keeps hanging on me during
>the installation procedure when it tries to scan the hard drives.
>
> Is it possible that this is IDE related? Should I consider
>going SCSI instead? If so, does anyone have some recommened SCSI
>hardware for me? (Controller, HDD, CD-ROM Drive)? Any input would be
>very appreciated.

The IDE does work on the PC164 motherboard. However, if your trying to load
RH 4.1 there is a bug with any Alpha and IDE. What you need is a new
"ramdisk" image. This can be found at the following location:

ftp://ftp.redhat.com/pub/redhat/updates/4.1/alpha/

The problem we saw is when trying to run fdisk the system hangs on the IDE
CD-Rom drive.
