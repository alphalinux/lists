>>> Vincent Stoessel said:
> Has anyone tried using a Paradise Pipeline 64
> PCI Video card? Can any one point me to a URL with a list of video
> cards that alpha linux currently supports?
> Thanks.

The current XFree86 servers that have been ported to Linux/Alpha
and are available as part of the 3.2 source release or 3.2A binary
release, are:

S3 most of the older S3 chipsets, up to 968, IIRC
S3V the S3 ViRGE chipsets
Mach64 ATI Mach64 of various flavors
I128 Imagine-128-based cards
SVGA currently supports only Matrox Millennium
P9000 P9000-based cards
TGA DEC's TGA (non-VGA) chipset, 8-plane only
Mono for curiosity only :-)
