Hello;

Typing on a standard keyboard with only one hand is getting
really old. I have been looking at this one hand keyboard alternatives.
They work on a 'chording' concept. There is one which has three microswitches
at each finger tip, and based on the switches you press indicates the character
typed. Has anyone seen anywhere on the net, or some Linux page, whether there
is a Linux driver for these alternate keyboards?

The next question is would anyone know of a way to connect both
a standard keyboard and one of the alternate keyboards at the same time.
I would need someway to be able to have both connected so that one of the
other consultants could use the computer, or one of my children when they
help me with typing.
