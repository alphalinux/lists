> I am having troubles getting the ps2 mouse port on my noname working. Are
> there any known problems? should I be configuring somthing special?

Make sure you have a PS/2 mouse. :-)

ln /dev/psaux /dev/mouse

Tell XF86 you have a PS/2 type mouse.

My mouse locks up when static zapped. A quick Ctrl-Alt-F1 then back to
the X server clears that up.
