This is one of those "If a tree falls in a forest...." posts. If it
shows up on the list (like it's supposed to do), then folks will read it
and I appologize in advance if anyone considers this a waste of
bandwidth, or for any other mayhem(sp?)

I'm (hopefully finished) standing up the mail2news gateway software on
my UDB. No real surprises so far, thanks to the excellent HOW2 written
by Robert Hart. This is being posted to my local news server (on the
UDB) and should be forwarded to the list.

My only departure from Robert's Excellent Directions was to use my
existing subscription to the list instead of creating an alias on my box
and subscribing the alias. This makes it easier for me and my ISP, and
lets me use my existing .procmailrc regex's to gate the incoming list
mail to the local newsgroup.

I'll be glad to answer questions via email.
