On Tue, 1 Apr 1997, Stephen M. Jones wrote:

> Looking for a description of how to configure/setup a DEC Multia (Alpha
> chip - 21066), aka Universal Desktop Box, to dual boot to NT4 and Redhat
> Linux 4.1. Any suggestions will be greatly appreciated.

Use the ARC (NT) console and it's a breeze. One boot option calls
linload.exe which loads milo. Set up another boot option to boot to the
NT loader then.
