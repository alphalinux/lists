Hi.

FWIW, em86 seems to work fine on ddd, as well as on netscape.

Regarding the revised kernel ...

I obtained the patch tarfile, and applied same (in /usr/src/linux).
Among other things, binfmt_em86.c and binfmt_em86.c.orig (the latter
being empty) were created in the root of the /usr/src/linux directory,
rather than in /usr/src/linux/fs, where the other binfmt_*.c files
reside.

I moved binfmt_em86.c to the fs directory, and did the
make mrproper; make dep; make clean; make boot
Oddly enough, I don't remember being asked about enabling the '86
emulation. I'm fairly sure I answered all the questions as per the
old .config file (a copy of which I had saved, to be safe). I then
set
CONFIG_BINFMT_EM86=y
in the .config file, by hand.

The make proceeded OK. I installed the kernel.

The kernel wouldn't boot. Suggestions?
