At 04:13 PM 4/1/97 -0700, you wrote:
>At 03:00 hrs. 04-01-97 -0500, you wrote:
>>Looking for a description of how to configure/setup a DEC Multia (Alpha
>>chip - 21066), aka Universal Desktop Box, to dual boot to NT4 and Redhat
>>Linux 4.1. Any suggestions will be greatly appreciated.
>Easy. Set up NT first, make the partitions you want forr NT, leave part(s)
>of your disks unpartitioned for Linux.
>Next install Linux, use it's fdisk to make the needed partitions, REBOOT,
>carry on with install.
>Next, make yourself the second ARC boot entry for Linux, verify it, and you
>are off to the races.
