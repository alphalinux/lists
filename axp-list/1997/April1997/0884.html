The SANE (Scanner Access Now Easy) project has a QuickCam backend that
is known to work on Linux/Alpha. The b&w code hasn't been fully
tested and I'm sure there are a handful of bugs left, but if you don't
mind submitting bug reports, I'd be happy to help get the bugs ironed
out. For more info on SANE, see:

http://www.azstarnet.com/~axplinux/sane/
