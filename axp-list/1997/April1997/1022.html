On Wed, 23 Apr 1997, SurferBoy wrote:
> I'm looking for a POP3 client that I can retreive my mail from my main mail
> server while running Linux. Is there a package out there that will do this?

I've seen a program called "fetchmail" that works quite well under
Linux/Intel. I don't know how it will work on an Alpha, though, but it's
worth a shot. It's pretty flexible, but what I've seen work real well is a
daemon mode: It checks every so often on your POP3 server for new mail,
downloads it, and hands it off to a local agent (like procmail, for example).
You get all the benefits of local mail without the need to setup sendmail and
company. It was found on sunsite somewhere, under mail utils, IIRC.

Hope this helps! :-)
