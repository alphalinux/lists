Hello;

I have received the patch files from the POV team member, and my son and
I have spent the weekend so far debugging the patches.

POV versions 3.00b-d all work on the Alpha.
POV version 3.00e fails on the Alpha while rendering the test scene.

We have isolated the problem down to six files which where patched.
Each file had changes made to array members of structures. It looks
as though one or more are "walking" off the end of the array, thereby
trashing memory. The problem only shows up while rendering at a high
picture size.

Once we have the problem child isolated, we will be working with the
POV team member to generate a fix.
