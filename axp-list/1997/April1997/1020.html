At 01:12 AM 04-24-97 -0400, you wrote:

>First of all, my (biased) opinion is that a 433MHz Alpha will easily give
>a PPro 200 a run for its money 8-)
>
>As for coexistance, this is not a problem at all. I'd suggest that you
>make three partitions; NT (FAT), Linux (EXT2) and Linux swap. Install
>NT first, then copy milo and linload.exe to the NT C: partition. This
>will set you up to install Linux on the second partition (while you can
>install Linux anywhere, it defaults to looking for root on /dev/sda2;
>if that's where you put your root, things go that much easier).
Actually Jim, I would suggest 1 modification to that:
If there is sufficient disk space, such as 2GB or more, he probably would
want to make a 4th partition, prep it as NTFS from NT, and put most of the
NT stuff other than boot on it. This circumvents the shortcomings of the
regular FAT filesystem on NT. As you probably knowe, FAT partitions of any
substantial size really waste diskspace due to the fact that they allocate
very large clusters for all files. My recommendation for the 2.5GB disk is
as follows:
partition 1 - 200MB FAT for NT root
partition 2 - 1000 MB NTFS for the rest of the NT stuff.
partition 3 - 1000 MB E2fs for Linux /
partition 4 - 256 MB for Linux swap
