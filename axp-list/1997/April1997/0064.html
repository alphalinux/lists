Hello Everyone;

Good news concerning POV version 3.00b.
It works!

I found the source in my dad's archives. (He keeps everything.)

I moved it to the Alpha, and after help from my sister, we got it
to compile.

Using POV version 3.01 test scene fails on arithmetic error.
Using POV version 3.00b test scene renders to completion.

My dad suggested 'diffing' the POV source directories and looking
at the differences between the source files. My sister and I have done
this and hear are our results.

We looked at each file, then sorted them based on our impression of the
amount of code changes.

add_logic.dir == 14 <- Only plus marks in the files
bcyl.dir == 2 <- These do not exist in 3.00b
change_logic.dir == 3 <- My sister decided on these
debug.dir == 4 <- Either added or removed debugging statements
major.dir == 6 <- More changes than my sister or I know about
medium.dir == 1 <- Not to many changes
memory.dir == 1 <- Changed a memory function
minor.dir == 12 <- Style and Macro changes
zero.dir == 93 <- diff file size is zero

I was wondering if there are any other versions between 3.00b & 3.01.
If there are other versions it would be easier for my sister and I to
see the gradual changes instead of mass changes.

Would those interested in POV, and that have the source handy check to see
what version they have. Please send me the version information.

You need to look in povray/source/optout.h
Search for 'POV_RAY_VERSION'.

My sister and I will work on finding out what change broke POV.
As dad has said "Just make one change at a time til it breaks, the
last change you made is what broke it."
