First of all, the ownership of media does not constitute a license for
Digital UNIX. Licenses are pieces of paper which are "attached" legally to
a CPU. They contain the information which you use to "install" the license
on the machine. I will gladly sell you a set of our media which we no
longer use. In doing so, I would be selling it as used, and as a
"replacement".

However, using it without a license won't help you much. First, it's
illegal. Secondly, once you do the software install you can log in as root,
and do only limited things. You have to install the license key using the
license manager before you can do anything useful. Only then do you get a
full operational OS.

Hope this clears things up a bit for you.

On the other hand, if you want a license for an AXPpci33, we can sell you
one, along with a motherboard and 233MHz CPU for $ 1015 U.S. This will give
you a legitimate license, CPU, and a motherboard.
Better yet, you can buy a complete computer, with legal Digital UNIX,
based around an AXPpci33 266MHz, for as little as $1990.
This would include a 1.2GB Quantum hard drive, 32MB of RAM, 256K cache, a
desktop case, 266MHz CPU, floppy, 12x CD-ROM, 2MB S3 video, mouse,
keyboard, 270W power supply, ethernet card, and full warranty and support.
This price includes a full 2 user Digital UNIX license, and media on CD for
UNIX 4.0. You can then sell the UDB, or better yet use it as an X terminal
if you have another user in your location. As the UNIX license is a 2 user
one, this is perfectly legal and legitimate.
