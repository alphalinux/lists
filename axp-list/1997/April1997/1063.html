Jeff Reiter wrote:
>
> hmmm...
> I recently tried to install WinNT 1st (on a DEC Multia) and had some
> difficulty (1st time with this type of machine and all), so I installed
> Linux 1st with this config:
>
> 2 - 1GB SCSI drives
> Disk 1, 1GB
> partition 1 - 10MB FAT milo, boot for NT root
> partition 2 - 815MB E2fs for Linux
> partition 3 - 175MB for Linux swap
>
> Disk 2, 1GB
> partition 1 - 10MB FAT for NT root
> partition 2 - 990 MB FAT for the rest of the NT stuff.
>
> Can you advise how this might be improved? Maurice, would recommend
> doing what you described below?
>
Convert disk2/par #2 to NTFS. after 512mb on a FAT you get (maybe) 32
blocks per cluster - which means that every file is rounded up to an
even 16k.
