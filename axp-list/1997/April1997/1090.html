Hello!

A mail from Stephan Schroepfer inspired me to state for another time how to
install RH4.1 on the PC164 from Vobis (a large german computer discounter).

First you need linload.exe and milo (from either CD or gatekeeper.dec.com).
Put them both onto one MS-DOS formatted floppy disk and create a boot selection
on Floppy Disk #0 in the ARC console with linload.exe as OSLOADER specified.

To boot the installation you need another two disks: the boot disk (with the
kernel) and the ramdisk. For the boot disk you can try the kernel pc164.img
from directory kernel/ which is on the CD. the Do *not* use the ramdisk image
from CD. Instead download a fresh ramdisk.img from

ftp://ftp.redhat.com/pub/redhat/devel/axp/images/

Please try appropriate mirrors first. This install program on that ramdisk
image will not try to fdisk your CD.

Now you will have three disks and you should boot MILO, the boot disk and the
ramdisk image in this order. Do not forget to put the CD in its drive ;-).

Good luck,
Stephan
