I've got this one everytime I try to install RedHatLinux. I usually am able
to solve it by doing the copy manually from another virtual console.
However, the real reason is a problem in the RedHat install scripts, I'm
surprised they still haven't changed this.
Earlier in the procedure you were asked where to boot from, and that's
where the script will try to get the kernel to copy to your harddisk. You
said:
kernels/avanti-s.gz
try:
/kernels/avanti-s.gz
Just this extra / should make the difference.
