You can use the 4.0 libs that are included in em86. It installs them under
/usr/em68/lib - when you run something through em86, it knows to look
under that directory for x86 libs. Unless some application specifically
requires newer libraries than was was included in 4.0, you shouldn't have
a problem. Just for reference, I couldn't get Netscape 4.0b3 to run though
em86 (I rebuilt a kernel with both the em86 binfmt patch and the memory
size patch); once I ran netcape, it kept spawning netscape processes
untill I finally did a killall and got rid of them all. I was able to get
various other binaries to run with few errors, although extremely slow on
my UDB at 266 MHz and 56 megs ram.
