Hello Everyone;

After talking with my dad about EM86, the thought occurred to me that
with EM86 I should be able to take a POV version 3.01 Linux/Intel binary
and move it to the Linux/Alpha and be able to run it.

The question though is that if the test scene would render to completion
using the POV Linux/Intel binary would this prove anything or provide
any further information that would allow us to run POV version 3.01 native?

If the POV version 3.01 Linux/Intel version did fail in rendering the
test scene would that indicate that the problem was with one the the
shared libraries that POV version 3.01 was using?

Since POV version 3.00b running under Linux/Alpha does render the test scene
to completion, does that not rule out the possibility of it being a shared
library problem, and indicate that it is still a problem with the POV version
3.01 source code?
