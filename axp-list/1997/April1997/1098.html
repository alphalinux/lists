Random note:

I managed to install off the RH 4.1 media onto an IDE hard drive using
an IDE CD without using the fixed ramdisk.img . It's nowhere near
as simple as just getting the new image, but I offer in in any case
for whatever it's worth.

Go through the `first stage' install as usual. That's everything up to
and including identifying the type of CD drive you're using. The
important point here is that as soon as you start the `second stage'
of the install, you can [alt]-[F2] to get to a shell prompt.

[Alt]-[F2] to get to a shell prompt.

Get a copy of the mount binary from the CD. I.e.:
# cp /tmp/rhimage/live/bin/mount /tmp

Unmount the CD.
# umount /tmp/rhimage

[Alt]-[F1] to get back to the install script.

[Return] to continue to the next step of the install. Continue until
you're at the swap setup page.

[Alt]-[F2] to get back to the shell prompt.

Make the block special file for the CD drive. In my case, this was hdb, so:
# mknod /dev/hdb b 3 64

Remount the CD using the mount you copied from the CD. In my case:
#/tmp/mount -t iso9660 -r /dev/hdb /tmp/rhimage

Continue and finish the install as normal.
