You can get the binary from:

ftp://franck.pc.uni-koeln.de/pub/xemacs-19.15-alpha-unknown-linux2.0.29.tar.gz

Size: 2.297.384 bytes

This tar-file just carries the binary files. You will additionally
need the xemacs-19.15-common.tar.gz file from the usual xemacs ftp
sites. My experience is that the XEmacs binary is *very* sensitive
against libc changes. So I suppose that it will not work on any box
with another glibc installed. That's why I decided to post the
previous message with the patches.
