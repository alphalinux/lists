At 01:01 hrs. 04-05-96 +0000, you wrote:
>> Okay, this one's a tad more challenging. Does anyone know how to set up a
>> Digital Unix-Linux dual boot? The partitioning seems fundamentally
>> incompatible, unless I'm missing something...
>>
>> Dave.
>
> I've never tried this, but something tells me it would be incredibly
>easier if you had two physical HDD's, and that installing Linux FIRST
>would be much easier. What I'd like to do in the future, is have my
>box triple booting to Linux, Windoze NiTwit, and DEC UNIX, but I'm
>going to wait until I get another HDD to even try and mess with it.
>You would however, need to switch firmware everytime you booted to
>NiTwit (and perhaps linux, depending on which loader you wanted), and
>Digital UNIX, assuming anyone wants to attempt what I'm suggesting :)
>
It depends on what Alpha you are doing this on. With some machine types
there is enough flash to hold both SRM and ARC, so for those the switching
is relatively minor. On others it's a pain. For example the nonames only
have enough flash to hold one or the other, so you would want to boot Linux
from SRM. This however still leaves you stuck for NT.
