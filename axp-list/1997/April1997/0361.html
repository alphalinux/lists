Hello all,

We recently installed RedHat Linux 4.1 on a DEC AlphaStation 250 4/266.
It's generally worked well with the stock kernel included with the RedHat
install. However, I wanted to recompile the kernel, to streamline it for this
particular computer, and include several things not available in the stock
kernel. Problem is, I've had little to no luck getting a viable kernel to
compile. If I type "make boot" one more time, I'm going to be sick. :) I'm
posting these questions in the hope that some forgiving guru can assist us
clueless newbies. :)

First of all, we've been building a monolithic kernel because we're under
the impression that loadable modules don't work on an Alpha. Is this right?

Second: This computer has an NCR SCSI chipset of some sort; however, I'm not
really sure which of the several NCR SCSI drivers in "make xconfig" to pick.
Nor what options to use. Any suggestions or information here? One thing we
generally need to work is SCSI disconnections. We're trying to use a CD
writer with this machine, and as it has only one SCSI controller, we really
need SCSI disconnects to work. At the same time, one of our more workable
build attempts did get us sort-of working SCSI, but we got various fun
"unexpected disconnect" kernel errors when trying to use the CD writer. :(

Third: We've been picking just "DEC Tulip" for Ethernet adapter, is that
right? Because (when it gets this far), every time the Ethernet driver inits,
we get this stuff:

> eth0: enabling 10baseT port.
> kernel: unaligned trap at fffffc0000388d20: fffffc00006a35f1 29 2
> kernel: unaligned trap at fffffc0000388d58: fffffc00006a35f1 29 1
> kernel: unaligned trap at fffffc0000388d20: fffffc00006a37e9 29 2
> kernel: unaligned trap at fffffc0000388d58: fffffc00006a37e9 29 1

The hex numbers seem to be the same, always. Once that is done, the net
generally works okay, but randomly spews this:

> eth0: enabling AUI port.
> eth0: enabling BNC port.
> eth0: enabling unknown type port.
> eth0: enabling 10baseT port.

... over and over again, sometimes messing up the net, sometimes not.

Lastly, does anyone know how to get the AlphaStation's built-in sound card
to work with Linux? Is it even possible?

Once again, sorry to come into the list crying for help. Any assistance on
any of these questions will be *greatly* appreciated. :) Thanks and TTYL..
