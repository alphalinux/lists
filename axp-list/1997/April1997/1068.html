Hello,
The Bochs x86 emulator is freely available for personal use ($25 for
non-personal (impersonal? :-), is quite portable, and is reported to run
DOS, Windows 3.1, and Minix. The following is a snippet of the features
(NB - although the goal is complete i486 emulation, several features are
missing; it's free, it's an early version...) listed on the Bochs homepage
@ http://world.std.com/~bochs:

What bochs has/runs currently:
80386 instruction set. (still lacking tasking)
I have virtual-8086 and primitive paging support, but it's not
ready to be released
DOS 5.0/6.22
Windows 3.1 in standard mode
Bochs can also run Windows 3.1 in enhanced mode with virtual-8086
mode & paging features not yet ready for release.
Minix-386 1.7.x
HGA monochrome Hercules Graphics adapter, using an X window
1.44M 3&1/2" floppy disk
1.2M 5&1/4" floppy disk
720K 3&1/2" floppy disk
via soft-int 13h and minimal programmed DMA IO
10M, 20M, or 30Mbyte hard drive via image file
.......... and more.
Good luck,
Daniel
