On Mon, 21 Apr 1997, Nils Faerber wrote:

> Hello all!
> I recently purchased a GUS classic soundcard and want to install it in my
> Jensen but have a big problem now:
> GUS in DSP mode works fine but when using the native GUS mode all samples
> are much TOO fast! Using a mod player like gmod a song normally taking
> about five minutes finishes in 5 seconds!
> Is this an AXP problem?
> Or is the Voxware driver broken for Alpha?

The problem is in gmod. It assumes that the system clock is running at 100
Hz as in i386. In AXP it's running ten times faster. I'm sure this problem
gets fixed very quickly if you report it to the author of gmod.
