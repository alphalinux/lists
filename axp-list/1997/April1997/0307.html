>I am willing to make a bet that you have an old version of SRM. I
>seem to remember having the same problem with an AlphaStation 200
>with SRM 5.3 on it. I upgraded the SRM firmware to 6.4 and everything
>works like a champ.

>It looks like SRM 6.4 is the latest for the AlphaStation 255 too. I
>would suggest that you upgrade and try again.

I had the same problem, under srm, with revision 7.9, the latest known to me.
Try :
>>> boot dka400 -fl i
... blah blah blah
aboot> kernels/avanti-s.gz root=/dev/scd0
...
and you're done ! (it works at home ...)
For the firmware, check periodicaly ftp.digital.com, there a firmware directory
somewhere under that root.
Have fun !
