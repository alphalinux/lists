>>> Nils Faerber said:
>
> I recently purchased a GUS classic soundcard and want to install it in my
> Jensen but have a big problem now:
> GUS in DSP mode works fine but when using the native GUS mode all samples
> are much TOO fast! Using a mod player like gmod a song normally taking
> about five minutes finishes in 5 seconds!
> Is this an AXP problem?
> Or is the Voxware driver broken for Alpha?
> Or does the GUS not work in Jensen's ISA slots?

I don't believe there should be a problem with either the hardware or the
VOXware; I've successfully used a GUS ACE before with an early 2.0 VOXware
and was pleased with the results.

I have, however, found a number of software packages that were written
only with the PC in mind; this one sounds like a problem with the expected
number of timer interrupts per second. On the PC, timer ticks are 1/100th
of a second, on the Alpha, 1/1024th. "gmod" may be expecting the former
but receiving the latter... :-(
