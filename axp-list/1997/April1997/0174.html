> The include file timespecs.h was introduced by the axp-kernel patch.
> There is really no longer any need for this patch on the later Linux
kernels.

> The choices are either apply the axp-kernel patch to the 2.0.29 kernel

> patch being there, but if we do not do it sooner it will just keep
getting
