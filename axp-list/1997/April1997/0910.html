Ben Schumacher wrote:
>
> Alright, quick question here. Please tell me that Linux is able to handle
> a RAID controller(not sure make or model).
^^^^^^^^^^^^^^^^^^^^^

No one is going to be able to provide a clear answer without knowing
which controller you are planing on using. Check through the HOWTOs
(Hardware and SCSI if memory serves) to see what is specifically listed
as supported and unsupported (Compaq raid controllers for example) and
procure / decide as appropriate.

I would advise (in all seriousness) performing this analysis _before_
the PO gets cut, not after, like my current customer did.

Hope this helps.
