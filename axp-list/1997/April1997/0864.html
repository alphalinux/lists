I have a UDB with a 3 button Digital Mouse. Part No. PC7XS-CA, M/N M-S30,
S/N 50806xxx Its a PS/2 Mouse as well. Yes 3 button. However I can not
get Linux or NT for that matter to recognize the fact that it is a 3 button
mouse. Any ideas? So far DEC has not been exactly helpful at least I
havn't yet talked to the right group. I would VERY much like to use the
middle button.

I don't know NT but I think that Microsoft simply never uses
the middle button (all MS mice are 2-button). As for Linux,
I figure the fastest way to test mice is to run 'gpm -t <mousetype>'
and try it out with 'mev -i'; moving/clicking the mouse should
print reasonable values if you got <mousetype> right. There is
only a limited number of mouse protocols, so try them all.

Unless, that is, your mouse is a dual-mode one, perhaps switched
between modes via the RTS or DTR line; if the default is MS, your
third button would simply not generate anything.
