Miguel Barreiro Paz <enano@ceu.fi.udc.es> wrote:
> Isn't the Alpha fully virtualizable, unlike the x86? Then it
> would be theoretically possible to run NT/Alpha under Linux/alpha (just
> wondering... time to sleep :))

This subject came up some months ago.

In principle, yes. But to do so you would have to write non-standard
PALcode. Basically every PALcode entry would have to check whether it is being
invoked from a virtual machine, and trampoline into the VM handler if so.

Then you could run the NT PALcode on the VM.

Unfortunately I don't have time to work on it.
