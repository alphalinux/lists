> however, I'm currently stuck into the following problem :
> install with no floppy suport went fine untill I was prompted for root password
> then, next step was to copy kernel
> at this point, system say : kernels/avanti-s.gz not there, and ???
> what can I do ?

Go to the second console (Alt-F2), check - with 'mount' - what is
mounted where and copy a required kernel by hand. Go back to install
dialog and skip that step. These install scripts are not bugless
in all situations.

Alternatively you may boot for the first time your freshly installed
system using a kernel on CD, root on your hard drive, and sort
things out after you booted. I was using a similar way of booting to
get out of some emergency situations.
