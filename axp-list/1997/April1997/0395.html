Paul Chapman wrote:
>
> Hi,
>
> I can't seem to convince em86 to do anything. Regardless of what
> file I give it, all I get is "Error loading <file>". This happens
> whether or not the file is an x86 executable, or even whether or
> not the file exists!

Hmmm... three thoughts:

(1) Make sure you have access to your "new" /usr/x86 tree
(2) Make sure the files you're trying to run have execute
permission (if you ftp'ed them over to your udb, they'll
NOT have execute permission by default).
(3) If you have access to a Linux/Intel box, build a "hello,world"
program statically linked (gcc --static hello.c -o hello) and
see if *that* works.
