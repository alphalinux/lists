At 16:21 4/04/97 -0500, Jim Paradis wrote:
>
>
> April 4, 1997: The Linux/Alpha team at Digital Equipment Corporation
> today is releasing a developers' beta version of EM86, a Linux/x86
> emulator for Linux/Alpha. Using components of the award-winning
> DIGITAL FX!32 technology, EM86 is a software emulator that enables
> Linux/Alpha systems to run Linux/x86 software without modification.

FX!32 not only emulates the x86 environment, but also can do a binary
translation, resulting in much better performance for future runs of the
same program. Is this functionality planned for EM86 as well ?
