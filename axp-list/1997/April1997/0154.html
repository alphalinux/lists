Hello Pete;

Dad and I ran several test today and here are the results.
These were all run on a Alpha pc164 433Mhz, 128MB, 2MB cache,
Linux 2.0.27, gcc 2.7.2.1. POV version 3.00b

Dad did point out several area's where the 3.00b source code could be improved
for speed improvments.

We did receive e-mail today from a POV team member, concerning the problems with
POV version 3.01. He is going to send us the various patch files to patch POV
version 3.00b to POV version 3.01. That way we will know which patch breaks it.

Dad and I are going to try and make some changes to get 3.00b running faster.
There are several points in the code, where math is coded that Dad says should
really to a constant.
