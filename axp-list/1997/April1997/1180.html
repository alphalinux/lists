SurferBoy wrote:
>
> Hello,
> I'm installing em86 and it says it comes with libraries
> that are imaged from Red Hat 4.0. Will these work on my 4.1
> install?

Absolutely.

> If not where can I get the 4.1 i386 libraries? I would think
> if I just got the RPMs and over rode the architecture it would
> over wright my AXP libs.

Right; you will overwrite Alpha libs this way. In order to avoid
conflicts between x86 and Alpha libraries, em86 looks for x86
shared objects in the "/usr/x86" hierarchy. I *THINK* you can
use the combination "--ignorearch --root /usr/x86" to install
x86 rpm's for shared libraries, but I haven't actually tried this.

But yes, the x86 4.0 libraries should work just fine.
