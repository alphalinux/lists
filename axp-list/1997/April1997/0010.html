aka ZLXP. The server is under the X directory in the RHL CD. You'll hqve to :
- copy excutable and readme files in /usr/X11R6/bin
- uncompress them
- change the X link in ??? (do a file to find where it is) to the new image
- edit the readme file to remove comments and save as XF86Config
- define the XF86CONFIG variable to locate the XF86Config file
