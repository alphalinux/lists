First reaction: Boy, you guys work fast!

Second reaction: That's what you GET for playing around with
pre-beta software before it's officially announced! 8-)

Seriously: here's a little pre-announcement for you: I've been working
for the past few months on a system called "em86", which is an
emulator that lets you run Linux/Intel programs on Linux/Alpha.
The version that you guys downloaded is no doubt the "shakedown"
version that I put out to test the distribution packaging. Yes,
you found a bug, as did I. If you want to play around, get the
very latest em86-bin.tar.gz from /pub/DEC/Linux-Alpha/em86 on
ftp.digital.com. That one ran netscape just fine. FYI, I've run
it with netscape on a 32Mb 166MHz UDB. It's a bit pokey, but it
runs just fine.

Note that the current public version is a developers' beta.
There are unimplemented features and maybe even bugs 8-) Also,
I've got several performance-optimizing tricks up my sleeve
that I haven't deployed yet...

"Official" announcement is going to be on Friday, to coincide with
Linux Expo. I won't be down there, but Jay will, and he'll be
demonstrating it. Bring a couple floppies, and you can take away
a copy of your very own 8-)
