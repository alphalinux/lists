On Fri, 25 Apr 1997, Tim Miller wrote:

> I would very much like to install Linux on the ALPHA in
> my group. However, a critical software application we
> use (our experiment analysis code) can not be compiled
> with g77. (Yes, it's in Fortran).
>
> I have read (although now I can't remember where) that
> the Linux/Alpha kernel will run native Dig Unix binaries.
> Is this true? (I *beg* that it is). Anyone have any
> direct experience with Linux/Alpha and Digital binaries
> or, more specifically....with the DEC Fortran compiler
> and Linux/Alpha.

Yes.... as long as you compile your fortran apps STATICALLY linked, you
will have no problems running them in linux (though make sure a.out format
is enabled in you kernel).

OR... (I'm not 100% sure of legality of this solution) copy the shared lib
loader for DUNIX to the linux box, and any required shared libs to the
linux box.
