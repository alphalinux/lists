Adam,
The answer lies in /etc/securetty - the man page is REAL
brief, but I think if you remove this file, you will be able to
login as root from anywhere.

My reading of the man page tells me that securetty is a
file that lists the line that root can login on. On my Sun LX
(the only Red Hat installation available in the office) it contains
the following entries:

tty1
tty2
:
tty7
tty8

YMMV, but Red Hat Alpha should be similar, if not the same.
