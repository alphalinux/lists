Hi, I was able to boot an Alpha Server 2000 using the following:

boot dva0 -file vmlinux.gz -flags "load_ramdisk=1 root=/dev/fd0"

You need a floppy with aboot and the kernel on it, I guess all the boot-
floppies have this.
dva0 stands for the first floppy drive
vmlinux.gz is the file to boot
load_ramdisk=1 root=/dev/fd0 are the arguments to the kernel

This got me in the install program. However, this was on and Alpha Server 2000.
I think it should work for you either.

Good luck,
