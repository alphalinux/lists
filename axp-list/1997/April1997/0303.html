Just a few corrections to the list of Alpha processor types...

> The UDB (21066) is an EV4. It is dual-issue

Actually, the 21066 is what's known as an "LCA4" It was planned as
a cost-reduced version of the EV4. It takes an EV4 processor core,
removes half of the on-chip L1 cache, and adds a PCI interface on-chip.
These two modifications tend to really bottleneck the memory access.
The net result is that overall LCA performance tends to be worse than
EV4 performance at equivalent clock speeds.

> the 21064 is EV45. It is also dual-issue, but faster, and more
> streamlined.

The 21064 is EV4. The 21064A is EV45, which is EV4 implemented in
CMOS-5 technology (die shrink -> faster clock speeds)

> the 21164 is EV5. It is quad-issue. FAST
> the 21164A and PC are EV 56. Quad-issue and more streamlined and with
> MMX-like instruction subsets.

21164A == EV56. This does not have the VMI (MMX-like) extensions.
21164PC == PCA56. This is the first implementation that does have VMI.
