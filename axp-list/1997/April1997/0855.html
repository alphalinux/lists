Hello all!

Although we are testing a PC164 for over one week now we still have some
problems running RH4.1 on it. The machine is PC164/500Mhz/1MB L3 cache,
128MB main memory, a WD EIDE harddisk and an ELSA Winner 2000 AVI 3D
(S3 Virge/VX).

Now the problems:

- although the system runs fairly stable we get a lot of Ooops 1 messages from
the kernel whenever a memory vialation occurs.
- the X-server starts but hangs the machine. The usual default X-background
already appears, but as far as X wants to place the first window the system
stops. We tried two XFree-RPM-packages, the one from RH4.1 and the one from
devel-RH4.2. Both servers have the same bahaviour.

We would appreciate any solution to these problems.
Thanks,

Stephan
