Hi,

Finally got some time to dig into SQUID sources (which are incredibly
well coded) and discovered that the source of the suid problem is the
function resuid() that returns a wrong 'ilegal operation' in alpha-linux.

I changed it to esuid() and now all is ok. Didnt have the time to see
what exactly goes on in the resuid() function...

hope this helps
