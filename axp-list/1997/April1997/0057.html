Michal Jaegermann wrote:
>
> Does that mean that we will be able to run DOSemu, and Wine, on em86
> running on Alpha? Boy, what a way to convert a decent processor into
> a toaster. :-)

Well, I haven't investigated this, but I don't think this will
be possible (at least not with the stock Wine/DOSemu) because
em86 does not (and will not, unless someone pays me VERY well)
implement the vm86 system call.

Also, for efficiency's sake the x86 execution engine that I use
implements *only* a 32-bit non-protected-mode memory model. This
means that 16-bit code or code that does funky things with protected
mode on the 386 won't work...
