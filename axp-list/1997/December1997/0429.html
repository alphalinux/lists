using the egcs 1.0 compilers. In particular, I want to compare the
results to an Alpha running OSF1 4.0.

I'm using the /usr/bin/time command on both systems for comparison.

same or even better on the Linux system. BUT:

the real time is about a factor of 10 slower on the Linux system.
