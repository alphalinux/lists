For what it's worth, I have a Cybernetics CY-8000 tape drive on my
Microway alpha machine running Redhat 4.2. This tape drive is based on
the Sony 300SX (I guess that's the model) drive, uses an 8mm, 170M cartridge
that holds 25GB uncompressed. Cybernetics adds some kind of RISC processor
that does data compression and a few other things. I'm running it with the
compression on and it works great.
