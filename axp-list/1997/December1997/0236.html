On Sat, 6 Dec 1997, Jacques Leisy wrote:
% Which format do you use for the CD. I downloaded the files from
% ftp.cdrom.com. I would like to burn a CD. What should be the root.
% Which format should I use. What do you do with long file names
% A lot of questions...

Download all of /pub/redhat/redhat-5.0/alpha into say, /a/redhat

mkisofs -A Linux -P "Murray Stokely" -p murray@cdrom.com -V
"RH-5.0" -R -T -N -d -D -o /a/alpha.cd0 /a/alpha

Then just burn the image with burncd, cdwrite, or whatever you use.

worked fine for me.
