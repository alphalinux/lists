Hello Abdallah!

AHT> i have just read about NE2000 LAN card that
AHT> ISA network cards should NOT be used if the
AHT> computer is faster than 166MHZ.
AHT> is that true? i need to know if anyone
AHT> out there is using (el cheapo) NE2000 ISA card
AHT> for Alpha under RedHat linux 4.2.

I do, because I've got a Jensen (without PCI) and an EISA card is rather
expensive. By the way, the Jensen is running at 150 MHz, I think an EISA card
wouldn't be worth all the money. If you've got PCI, I recommend trying to get a
cheap PCI NE2000 clone. I paid 37,50 DM for a 10Mbit TP/BNC board a few weeks
ago, the ISA version was at 35 DM, now PCI has become 50 Pf more expensive, 38
DM, a similar ISA board costs 29 DM. (Prices in Deutsche Mark; Aachen, Germany)
