In /etc/sysconfig/network, put NISDOMAIN=foo.nis or whatever your NIS domain
name is,

install the ypbind rpm and make sure it is started at boot time,

Edit /etc/yp.conf and add "ypserver foo" to tell things like glibc where
the server is.

If your alpha is a server, things are a little bit more complicated, but
essentially reduce to installing the ypserv rpm and configuring it.
