The Redhat supplied kernel (and kernel source) is a patched version of
2.0.30. It has broken trap handling. I'm not sure if that's exactly
your problem, but you should at least build a better kernel & see if
the problem goes away. I wrote a document describing how to do this.
It's called "Linux Alpha Patches and Bug Fixes - Kernels, GCC, and
Profiling". It's available in the archives for this mailing list at:

http://www.redhat.com/support/mailing-lists/archives/axp-list/1997-November/0734.html

An older version is available at

http://users.bart.nl/~geerten/patches.txt,

but it, but it should suit your needs.
