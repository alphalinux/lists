At 09:43 AM 22/12/97 -0700, you wrote:
>hello there...
>i have installed an NE2000 ether16 LAN card into an Alpha 21164.
>i compiled my RHL 4.2 with support to this card and rebooted
>the machine.
>it detects the card at I/O 300, but then says that
>'...failed to detect IRQ line'.
>
>i use another pc to set the IRQ and it can allow me to set it on
>certain IRQs. now it is set on 3.

I used to use IRQ 5, and had to tell my kernel so by adding something
similar to "eth0=5,0x300,0,0" to the boot command. That was an NE2000 clone
(ACCTON). It broke down, I got another NE2000 clone but never got it to
work, even with this boot parameter, so I'm using 3Com 509 now.

>any rays of hope?

If yours is a clone you may experinece some trouble, but if its the real
thing you should be able to get it to work.
