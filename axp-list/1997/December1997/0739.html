>Looking for 64 more meg to upgrade the memory mode to 256 bit.
>I need 4 more 16 meg simms...dram...60ns 72pin 36bit tin(silver?)
>connectors. Siemens in the name on the ic's of the simms I have...Has
>anyone already done the research into a great price on these? Is anyone

I've just taken out 64 meg, which I plan to put in my new dual pentium
system. In my case the upgrade to 256 bit wide (PC164 366MHz) wasn't worth
it. Kernel compiles became noticably faster (10%), but other programs
gained hardly anything, some (PovRAY) even became slower. (Note that the
cache on PC164 is not upgradeble: if you widen the memory path, you also
double the size of the cache line, so only half as many lines will fit in
the cache. With certain patterns of memory access this may turn against you.)

If, on the other hand, the _size_ of your memory is a problem (noticed any
swap activity lately?), then adding simms is definitely _the_ way to
improve performance.

The simms I used were Kingston (KTM0160), which worked fine, but weren't
particularly cheap IIRC.
