I got a complete set of sources posted here earlier by yue-feng sun
and run this test after compiling with g77-egcs compiler. The program
reported

User time: 39.21680 seconds

Real time: 42.00000 seconds

Then I added, rather at random, without even looking at these sources,
"FFLAGS = -O6 -funroll-all-loops -fschedule-insns". Times
reported changed to:

User time: 6.69336 seconds

Real time: 7.00000 seconds

6-fold improvement for a few lousy flags? This should teach you
to use an optimizer in the future. :-)

After throwing into a fray also "-fschedule-insns2
-fexpensive-optimizations -frerun-cse-after-loop" I got the same
looking output but times reported changed to

User time: 6.35254 seconds

Real time: 6.00000 seconds

"Real time" shorter than "User time"? Oh, that is an optimization!
