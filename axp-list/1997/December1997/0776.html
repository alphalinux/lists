> On Mon, 17 Nov 1997, Linus Gasser wrote:
>
> > I'm wondering whether somebody of u already got a working em86 together
> > with RedHat5.0... all em86-rpm's I find demand Libc.so.6, but RH5.0
> > delivers libc.so.6.1, and em86 insists on libc.so.6. Does it mean, I
> > have 2 install RedHat4.2, or is it possible 2 get it running with a
> > RH5.0?
>
> UTSL

The sorce of problem is libc. One way to use em86 is this. Copy the files
in /lib of RedHat4.2 to some place like /usr/alpha-linuxlibc5. Add this
directory to /etc/ld.so.conf at top position. Do ldconfig. This way is
described in Glibc2HOWTO.( You need some more files than written there) I
do not know you can install the RPM files after this.( I made a dircet
copy from my RH4.2.). But many programs from x86 (like netscape) works
well as in the case RH4.2.
