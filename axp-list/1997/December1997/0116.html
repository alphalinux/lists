On Tue, 2 Dec 1997, Keith Baker wrote:

> Does SMBFS not exist for alpha? And if so why not? All of the other SMB
> support exists as does smbfs for intel...
> I yearn for the ether

Some time ago I´ve set up two Linux/Alpha boxes doing filesharing via SMB
for demonstration purposes. -- It didn´t work with stock RHL 4.0 Kernels
(it was state of the art back then... ;-)), but upgrading the kernel to a
newer version solved that problem.

The Alpha box set up as SMB server used SAMBA as SMB server software. As
far as I can remember it compiled cleanly.

In order to import SMB shares on your Linux box you have to compile
support for SMBFS into your kernel and use a tool called "smbount" to
actually mount the shares. "smbount" is part of the SAMBA package, btw.

So if nothing has been broken in the kernel since then SMB should work
quite fine on Linux/Alpha. ;-)

Hope this helps...
