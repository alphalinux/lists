According to the X server message, there are only 4 chipsets that it 
recognizes:
a) Matrox Millennium
b) Matrox Mystique
c) Matrox Millennium II
d) S3 Virge


Your Cirrus-based card will not ever work with a Digital Alpha system.  
Various cards require quite a bit of close-to-assembler fiddling around 
typically with 16 bit values.  This is death to performance on an Alpha system 
which prefers nice 64 bit values.


The only cards I am aware of that have been successfully used with the 
Multia/UDB use the S3 968 chipset.  (I've tried S3 Virge with partial success 
but some *massive* lockups; the Matrox cards apparently have color map 
problems on UDB systems.)


Look for a Diamond Stealth 3200 series board with 2MB of memory.  See: 
<http://www.hex.net/~cbbrowne/alpha.html> and/or <http://www.hex.net/~cbbrowne/
hardware.html#DANTZIG> for more details.
