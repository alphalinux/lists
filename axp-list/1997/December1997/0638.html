One other consideration you might want to look into is using QMail which
might help reduce the CPU problems youre having. Or have multiple machines
handle mail -- e.g. have one machine handle incoming mail, another handling
outgoing mail, etc.

You might want to go with RAID-1 instead of RAID-0 as that is striping.
RAID-1 is mirroring, so you have redundancy (e.g. if one hard drive dies,
you dont lose any data) and when doing reads, it can read from both drives.
0+1 might be a better choice as well.
