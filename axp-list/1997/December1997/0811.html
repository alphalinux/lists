I use minicom, its pretty self explainitory, and the ppp script below,
run as root. put it in /root

once connected

ctrl a q

leave without reset "yes"

then

./ whatever you call the below script
like

./pppstart

<begin script>

#!/bin/sh

/usr/sbin/pppd -d -detach defaultroute 38400 /dev/modem &

</ script>

should do it

good luck
