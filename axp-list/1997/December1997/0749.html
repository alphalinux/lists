At 03:59 PM 12/22/97 -0800, you wrote:
>
>I have a few quick questions concerning an unlikely beast, the Alpha 2100
>
>1) Does anyone know what a fair used marked price for this beastie is?

As already mentioned, it depends a lot on the configuration. There are two
basic configurations of 2100:

2100 (aka SABLE) 3 PCI slots and 8 EISA slots.

2100A (aka LYNX) 8 PCI slots and 4 (?) EISA slots.

Each of the two can have 1 to 4 EV4 or EV5 based CPU cards (all EV4
or all EV5, not mixed), and 1 to 4 memory cards (total of 6 cards max).

>2) Does anyone know how it's performance compares to a 533Mhz LX?

Fastest EV5 CPUs I've heard of are 300 MHz; I don't know if EV56 CPUs
were ever used (check the AlphaServer webpages under digital.com). So
that's about an EB164-style performance, ie slower than PC164, which is
slower than LX164.

>3) Does anyone know if linux will work on it?

LINUX currently works on both EV4 and EV5 versions of SABLE (I believe),
but it does NOT work on LYNX (yet).

>4) does anyone know if multiple processers will work on it under linux
>or not, and if there are any plans to support such?

The LINUX SMP code has not (yet) been ported to the Alphas, but stay tuned...
