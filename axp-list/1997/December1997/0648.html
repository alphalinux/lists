On Wed, 17 Dec 1997, Aaron M. Gowatch wrote:

> I've had a bit of trouble finding information pitting these two platforms
> against each other in real world, non-fp applications. Does anyone have
> any opinions or experience with both platforms?
>
> Any info would be greatly appreciated.

I run multiple servers, both on Alpha and on i386. Though my Alpha server
is the oldest of 'm all, it still performs best. Responsivity is great,
and when the system is forced to use swap, it's still fast. The Alpha is a
DEC AlphaStation 233MHz, 96MB, the i386s are a diverse bunch, mostly fast
pentiums with either (E)IDE or SCSI technology.

I would advise an Alpha if you're in for some adventure, but would like to
get great performance. The Alpha architecture seems better suited for this
task than most off-the shelf i386 products.
