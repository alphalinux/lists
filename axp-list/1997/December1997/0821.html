On Sat, 27 Dec 1997, Kevin Gross wrote:

> I want to hook up my alpha to my internet service provider.

I assume you run RedHat. Then, if you have X running, use the control
panel to set this up. It is very easy; the only trick is to get the script
right for your local ISP. If you havn't got X running, you can use some
other scripts until you find a better sollution. This is how I did:

My script /etc/ppp/pppswipnet
=============================
#!/bin/sh
/usr/sbin/pppd -d file /etc/ppp/swipnetoptions connect '/usr/sbin/chat -vf
/etc/ppp/cswipnet' /dev/cua2 57600
=============================

My options /etc/ppp/swipnetoptions
==================================
noipdefault
netmask 255.255.255.0
crtscts
asyncmap 0
ipcp-accept-local
defaultroute
==================================

My ISP-script /etc/cswipnet
===========================
""
at&f1t
OK
atdt2704900
ABORT "NO CARRIER"
ABORT "NO DIALTONE"
ABORT BUSY
ABORT WAITING
TIMEOUT 42
CONNECT
""
sername:
[your username]
assword:
[your password]
ipnet>
ppp
======================

Here, you change 'sername:', 'assword:' and 'ipnet>' to the prompt your
ISP gives you. In my case, I have

Username:
Password:

swipnet>_

Hope you get this working temprarly.
