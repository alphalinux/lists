>Is there already a port of G77 to 64 bit machines running linux?

Try g77-0.5.21.tar.gz (and gcc-2.7.2.3.tar.gz) at a GNU distribution
site (such as <ftp://prep.ai.mit.edu/pub/gnu>), or, for a
more recent version that might perform better (but might have
a few more bugs, certainly a few bug-fixes vs. 0.5.21), see
egcs at <ftp://ftp.cygnus.com/pub/etcs> (or some such thing,
I don't have the actual URL handy, but it's there somewhere).
