I only have RH Linux running on a lowly UDB and a lowly '486-33Mhz. The
differences I have seen to date for myself stem from a bunch of software
not being 64-bit ready. I don't know about RH 5.0, but under 4.2 some
software which uses tcl/tk seems broken (I think I'm accusing the correct
library), some video games I'd like to investigate are broken on axp, and
so forth.

Boot procedures between the two archetectures are different also, but I
_really_ like the fact that MILO can find the kernel _anywhere_ on my
harddisk. That certainly simplifies my life.

Hope this helps.
