>Hi,

>You may remember me from the Unaligned Kernel Traps on my Alpha XL-233.
>I figured out that it had to do with my IPX protocol. I have built a
>kernel without IPX and I don't get any kernel traps anymore.

IIRC this is a "known problem". The layout of the IPX packets is such that
you cannot get at the individual fields without unaligned accesses.
However, the access should be fixed in the kernel. The message is just
informative: everything should work fine.

>The problem
>is that I need IPX for Samba.( which I really need to run!).

I run SAMBA/smbmount in both directions between Linux (alpha or intel),
Windows NT (alpha or intel) and win95 (intel), all without IPX.

>I even
>tried to built the IPX protocol as a module. When I insterted it with
>insmod it wouldn't work though it was loaded. When I used rmmod to
>remove it, I got an OOPS error!!

Building it as a module will not solve the unaligned accesses.
