>>> RHS Linux User said:
> Will Linux/Alpha support Alpha server 1000 4/233's?

Yes! These boxes, aka MIKASA, have been supported for a while, at least in
their EV4 incarnation (thats the "4" in "4/233" :-). There's a MIKASA kernel
as part of Red Hat 4.2 in the "images/" directory; use the mikasa-s.img file
for booting from SRM console (the only way currently recommended).

The latest alpha-patches for 2.0.30 and 2.0.32 support MIKASA with either
EV4 or EV5 CPU.

Also supported by those patches is the AlphaServer 1000A, aka NORITAKE, also
with either EV4 or EV5 CPU.
