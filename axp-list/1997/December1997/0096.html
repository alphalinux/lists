AlphaPC164-SX, aka SX164, requires its own MILO and kernel, which you can
find on gatekeeper.dec.com:

/pub/DEC/Linux-Alpha/Kernels/Test-Milos-2.0.30+/sx164 # MILO
/pub/DEC/Linux-Alpha/Kernels/sx164-31oct-2030+.gz # kernel

Note that support for the SX164 is included in Red Hat 5.0, as well.
