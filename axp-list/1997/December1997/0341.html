Since many people downloaded XEmacs binaries for Linux Alpha from my
ftp site, I thought I should announce a new 20.3 binary running on top
of a RedHat 5.0 based system.

David Lebel "<lebel@socom.com>" and I struggled a little with some
patches, but finally we got it going.

As with any previous distributions this binary is extremely dependent
on the installed libraries (you'll get a segfault if you have other
library versions installed).
