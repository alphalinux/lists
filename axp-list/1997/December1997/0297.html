sights on Certicoms ECC-161.

> Alpha Linuxers everywhere!
>
> We're a fairly small community but we've got some awesome machines and

>
> The first Certicom challenge, ECCp-79, is just finished. The solution
> was found by Wayne Baisley and yours truly, using nothing but Alphas.
> See the contest pages [1] and our submission [2]. <p>
>
> Let's do the next one, ECC2-79, using 100% Alphas running Linux! It

>
> Some people have already contacted me to say they're in.

> Instructions for use:
>
> * Get your hands on an Alpha running Linux permanently connected
> to the net set up so that system("/usr/sbin/sendmail ...") and
> system("./wget ...") can work.
>
> * mkdir /tmp/ecdl
>
> * cd /tmp/ecdl
>
> * Get these files via Lynx or Netscape or whatever:
>
> http://pauillac.inria.fr/~harley/ecdl/ecdl.gz
> http://pauillac.inria.fr/~harley/ecdl/wget.gz
>
> * gunzip ecdl.gz
> chmod u+x ecdl
> gunzip wget.gz
> chmod u+x wget
> (NB: wget must be in the current directory)
>
> * nice +10 ./ecdl > log.`uname -n` &
>
> * Read the log.machine file occasionally to check for warnings e.g.,
> grep for the word "sendmail".
>
> * Not much work is lost if you stop the process and restart it,
> so you can run it at night and stop it in the morning if you prefer.
>
> * Email Robert.Harley@inria.fr if there are any problems with these
> instructions or with running the stuff on your setup.
>
> * Note: wget is subject to the GNU GPL. The source is at:
>
> ftp://ftp.gnu.ai.mit.edu/pub/gnu/wget-1.4.5.tar.gz
> http://pauillac.inria.fr/~harley/ecdl/wget-1.4.5.tar.gz
>
> I will release the source of ecdl *after* we solve it!
