Hi murray,

A few months ago I dived into Alpha assembly coding only to find out
that my carefully designed and optimized line-draw routine was exactly
as fast as the one I coded before in C and optimized with -O2...
Checking the output revealed almost identical code. So I see no use in
doing things the hard way if you can do it lots and lots easier with
C... I even printed out them manuals from Digital :)

Assembly on Alpha is no harder than anything else like 680x0 or Intel
stuff but since C compilers are mighty clever these days...

Well, all IMHO offcourse :)
