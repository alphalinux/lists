Hy folks...

i just tried out kernel-2.1.1 from ftp.azstarnet.com in order to be able to
compile milo ( with old kernels it gives me a lot of sintax error in the
headers files of kernel ). The first think i noticed was that in the file
linux/drivers/char/scsi/53c7,8xx.c there is in lines 250/251:

#include <linux/malloc.h>
#include <linuxv/malloc.h>

obviously there is a superflous double #include directive and the second one
causes the kernel not to compile.
