On Fri, 21 Feb 1997, Ken Serack wrote:

> In reading the IP Masquerading HowTo, I see that it seems to rely on
> Modules. Is this a problem, or does this work?

It should work fine on alphas by now. I haven't tried it personally,
though. And as for modules, the only modules you use are if you want to
run ftp, real audio, or irc through the masquerading firewall. and with a
few tricks, modules will work fine on the alphas.

To set it up, all you really need is the "ipfwadm" tools. and some
knowledge of firewalls and masquerading to set it up!
