Well that's a real problem ;)
Mesa is a library. You have to program it and if you know nothing about
make I suppose your programming skills are not much better.
I have Mesa 2.1 successfully compiled with RedHat 4.0 just by typing
make linux-elf
All demos ran well and quiet speedy on my old Jensen (150MHz).
After compilation you will find the libraries in the lib/ subdirectory. You
will then have to copy them to some place where you want to keep them,
probably /usr/local/lib/. Then edit /etc/ld.so.conf and append that path and
run ldconfig. All demos should run now.
