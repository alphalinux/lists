At 07:42 hrs. 02-10-97 -0500, you wrote:
>hi folks,
>
>i've got a seagate barracuda st32550n drive i'm trying to connect
>to a multia. i go into fdisk to set the drive params, and it won't
>accept values great than 63 for number of sectors. the geom for this
>drive is:
>
>cyl: 3510
>heads: 11
>sectors: 108
>
>i don't need to put milo on this drive, but i would like to be able
>to put the os on it. any help would be greatly appreciated!
Don't worry, be happy.
In fdisk go to expert mode (x) and set heads to 32, sectors pers track to 63
and cylinders to 2068.
Use it.
Why does this work?
Because this is all logical remapping anyway. Any combination that is within
actual total capacity is fine.
So, we do this:

3510x11x108=4169880
4169880/32/63=2068.39
round down to 2068
