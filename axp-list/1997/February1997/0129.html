Hi all,
I have tried to boot linux on an Alphastation 500/500 and failed :-(
David MT asked me to compile a kernel with serial console support and mail
him the boot messages. I thought someone here might also be interested in
this. The machine has Powerstorm graphics and the kernel can print messages
to it. The console eventually gets screwed up enough so you cant read
anything. I did not realize it was making it all the way up to trying to
mount the root file system, or I would have put a CD in the drive. The
CIA machine check is the only thing I think looks really bad. I think that
is also about the point where I loose the console.

This is an Alcor kernel compiled with serial console and VGA support.

Thanks,

Jim
