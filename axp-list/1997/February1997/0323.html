Bob Abbott wrote:
>
> Having some experience in both platforms and considering your problem
> domain; Alpha. No question. Large matrix FP ops is an area in which the
> Alpha particularly excels (compared to the Pentium-Pro in SMP.)

There is one big caveat, there is no optimized matrix algebra
package for the alpha running Linux. DEC's DXML (extended math lib)
runs matrix calculations as much as five times faster than
the public domain BLAS and LAPACK, and almost always at least
twice as fast. The DXML cannot be run on Linux as other AXP
libraries because the licensing code built in to DXML requires
some awareness in the kernel. It may be possible to obtain
optimized BLAS and LAPACK for the intel architecture, but I
haven't look at that.
