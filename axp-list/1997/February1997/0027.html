If you read this list, or the archive of this list on www.redhat.com, you will
discover the following fairly quickly:

1. There is no Native Linux/AXP binary of Netscape.

2. You cannot run the Linux/Intel version, it is for a different architecture.

3. You can run the Digital Unix version of Netscape, providing you have a
license to DEC Unix, and install the neccessary libraries and loader to the
Linux system. For support for DEC Unix 4.0, you need to run a development
kernel equal to or greater than 2.1.24.

4. Mosaic 2.7beta, Grail, Lynx, Chimera are browsers that are available and
work on Linux/AXP. There are probably more, but these are ones I have personal
experience with.

Locate the list archive and bookmark it, it is a mother lode of information on
running Linux on Alpha.
