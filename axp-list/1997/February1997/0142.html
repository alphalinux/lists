of memory and Redhat 4.0 Linux.

I've managed to get milo to boot, although the screen is solid light
green with some white and blue streaks on it (looks like the text that
should be there is messed up). I can talk to milo via the serial
port, and can get it to boot the avanti.img file that I copied from
the Redhat CD to a floppy. The kernel starts up and mounts the root
filesystem. Then it says:

Any help would be appreciated.
