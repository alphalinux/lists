Hello all, just announcing it again in case somebody missed it
:-)...

Version 1.0 of the Linux/Alpha port FAQ is here ! New features :

* Updated glibc information
* fully translated to french (though I doubt it will be useful to
anybody :-)
* Some words about the boot process under Alpha

The URL is :
http://www.eleves.ens.fr:8080/home/quatrava/AXP-port-FAQ/index.html.en

For people seeing this announce for the first time :

This FAQ deals with the Linux basic software (bootloaders, kernel,
runtime C library, compilation tools) as seen from the Alpha side. Its
initial goal is to solve common problems encountered when trying to
build the Linux kernel under the AXP architecture; it also contains a
brief history of the port. Please notice that this is not the official
Linux/Alpha history, maintained by Dave Rusling : the one here is only
intended to make things clearer about the real questions in the FAQ.
