I am trying to install Red Hat 4.1 on a EB164 with the following specs....

128MB RAM
Adaptec 2940UW with a SCSI 3.4GB hard Disk
Diamond Stealth
Hitachi CDR 7390 8X IDE CDROM

When I run Milo, the first possible error message is

dev_config: don't know how to set hostid of SCSI device (9004:8178)

I tried setting the environment SCSI0_HOSTID 7, but get the same message. If I run show, it said it has loaded the drivers, shows all correct information.

It does not appear to see the IDE controller, it comes up with

hdb: IRQ failed (-14)
hdb: IRQ failed (-14)

but later it does come up with

ide0 at 1f0 3F6 at IRQ14

Eventually, it comes up with the message

Unable to open an initial console

It appears that I can not access any IDE device, if I try to a ls command, Milo hangs. Unfortunatly, I do need to use an IDE but for the time being I borrowed a SCSI CD, (from a Mac!) and now I can start the installation.

I can now partition the SCSI drive, set up the Linux swap and native partitions, it now hangs trying to initalising swap space.

I have looked in the FAQ's, archives and Erratas, but there is nothing there.
