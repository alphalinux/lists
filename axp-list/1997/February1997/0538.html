On Fri, 14 Feb 1997, Adriano Nagelschmidt Rodrigues wrote:

> * The kernel is only recognizing 32MB of memory, even when I use the mem
> parameter (eg boot fd0:vmlinux.gz root=/dev/sda1 mem=256M).
>
>
I had a similar problem, which I just reported to the list. But I've got
no answer. But today I found the simple solution to the problem:
At the MILO prompt just set the Variable MEMORY_SIZE as explained in the
help after typing 'help' to the correct memory size. Now Linux should also
detect the whole installed memory.
