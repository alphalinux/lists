I just placed amd-920824upl102-7.alpha.rpm on ftp.redhat.com in
/updates/4.1/alpha. It fixes the problems w/ amd hanging which many
people have encountered.

If you cannot get your system to boot because of this problem, add
single to the end of the MILO command you are using to boot the
system. This will boot your system w/o networking. From there, remove
amd with "rpm -e amd" and reboot normally. When the system boots up
in normal mode you can install this new amd package to get amd
functionality.

I hope this helps -- sorry it took so long.
