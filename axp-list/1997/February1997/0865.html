Gee, ain't it fun being on the cutting edge??

I probably won't spend a lot of time on this, as I'm only "playing" around
with this machine (it was purchased to run NT 4.0, we just aren't ready to set
it up yet).

So far I haven't gotten very far :-(... I will try hooking up a serial
terminal to COM1 to see what MILO has to say. I'm using MILOs and LINLOAD.EXE
downloaded from gatekeeper.dec.com earlier today. If there's anything newer,
I'd appreciate a pointer to it.

I can get the machine to load ABOOT and it has no problem switching PALCodes;
I can even get ABOOT to load a kernel into RAM. Unfortunately all the
distributed kernels with RHCL4.1 are _not_ configured to boot from SRM, so I
don't think that will work. If some kind soul would like to e-mail me a
compiled kernel that's configured for booting from SRM, I could give that a
shot... If you're interested let me know and I can send you the specs the
kernel needs to have.
