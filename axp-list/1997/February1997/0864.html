>>> Kevin P Fleming said:
> Anyone out there have RHCL running on an AlphaServer 4x00 machine? I can't
> seem to find a MILO image that will get past the "Swapping to PALCode at
> ..." message. I know this machine is not specifically listed in the
> supported systems list, but then the mailing list archives contained some
> discussions of people trying this a few months ago...
>
> In case it matters, my machine has the latest firmware updates (V3.8) and
> runs AlphaBIOS, not ARC. It's a single-processor 5/300 machine with 256MB of
> RAM.

Sorry, but AFAIK, you're the first to try this, and you're pushing the heck
out of the envelope... :-) :-(

Chances are *extremely* poor that any of the existing MILOs will work on that
guy, and even if you were able to boot from SRM, there'd still be problems,
most likely with interrupt delivery but possibly even with address spaces.

If you hook up a terminal to COM1, do you see any messages there from MILO?
It actually dumps a few more debug-style messages there than to the console,
so this may indicate what's going wrong, or at least the last thing that went
right... :-(

Also, you need to be *sure* you're using the very latest "linload.exe", which
has a bugfix for memory configurations larger than 128Mb.

Good luck, keep us posted on any developments.
