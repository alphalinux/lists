On Tue, 11 Feb 1997, Andrew D. Isaacson wrote:

> The web page is at http://www.fcc.gov/isp.html. Please read the web page
> before submitting comments; according to the submission rules, you must
> include your postal address (among other things) with your comments.

Note in particular that the deadline for filing comments in the first
instance was January 27. The January 13 deadline is for filing reply
comments, i.e., to respond to the views expressed in the comments filed by
the Jan. 27 deadline.

So if you want to send in a reply comment, first read the comments at
http://www.fcc.gov/Bureaus/Common_Carrier/Comments/access_reform/ , and
select a comment to which you want to reply.

If you want to comment intelligently, you should also read the Federal
Register notice in the January 31 Federal Register,on pages 4670-4717,
which provides the proposed rules and a more-or-less English description
of them.

There's no simple URL for the FedReg document, but you can go to
http://www.gpo.ucop.edu/ and search the federal register for "Access
Charge Reform" to find it. It's way slow today, though.

> Note, further, that isp@fcc.gov does not appear to be the correct address
> to which to send comments about the rate increases. But, if you read the
> web page, you'd know that.

Well, that's the address for the Internet Access and Information
Service Provider topic, which has some overlap to this. But the deadline
for comments in the first instance isn't until Feb. 21, and replies aren't
due until March 24.

Is this on-topic for this list, though?
