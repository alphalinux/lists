This is for the people using alpha machines that are trying to get
radius to work with them. Thanks to Dave Thurston (dthurstn@digimag.net)
on getting this critter to work.

> Eric here are the changes that I made to get radius to
> work...
>
> (1)
> In radiusd.c and radpass.c change definition of variable
> salen from int to long
>
> (2)
> In conf.h and md5.c comment out the following code:
>
> #if defined(__alpha) && defined(__osf__)
> typedef unsigned int UINT4;
> #else
> typedef unsigned long UINT4;
> #endif
> Insert in its place :
> typedef unsigned int UINT4;
>
> This is all, it compiles cleanly after that...
>
>
> Dave
>
