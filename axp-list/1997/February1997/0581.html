I just checked and Linux and Digital Unix both do the same thing. Heres
the program:

#include <stdio.h>
#include <stdlib.h>

int main(void)
{
int i;
for(i=0; i<3; i++) {
FILE *f = fopen("/dev/null","r");
printf("f = %x\n", f);
fclose(f);
}
return 0;
}

and here is the Linux output:

jlnance-pc> a.out
f = 8049650
f = 8049650
f = 8049650
