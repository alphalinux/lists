> I have the following layout:
>
> DEC axp 150 PC (Jensen)
> 64 MB RAM
> Compaq Qvision 1024 video board.
> Yggdrasill Linux Internet archives (Winter 1997)
[..]
> from my old VMS days, I know an access violation when I see one!
> and it does not install any Xserver that I can use. I have tried to find and install one either on the CD-ROM or the net, but have been unsuccessful.
>
> I have also replace the EISA Qvision card with a plain Paradise ISA VGA card, with the same results.
>
> Q: Is there a way to get this setup to work?

Not easily. Yes, if you'are willing to add some patches to the VGA
servers and drivers (and I mean writing them). They generally havn't
been ported to the Alpha platform. And the QVision isn't even
supported by XFree/Intel.

However, the S3 server has been ported to the Jensen. You'll find an
early version on the "Yggdrasill Linux Internet archives" last disk.
Look at the FAQ, somewhere below /azstarnet.

The latest XFree beta (3.2A) also supports the Jensen with the S3
server.

> or should I scrap the idea of using RedHat and try something else?

X with Linux on the Jensen only works with S3 cards for the time
being. However, the Jensen's firmware is somewhat picky about graphic
cards, so you'd better test one before buying it.
