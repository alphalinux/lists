Til today I thought a special ROM was needed to use the aha1742 with
the Jensen. I heard or read that somewhere, maybe as a rumor. OTOH,
since ARC and SRM console are two different things, I tried another
aha1742 anyway, since I assumed it might work on one of those two.

Well, it turned out, that Linux works well, and it is possible to
invoke the NT installation utilities from CD! (Didn't have an NT to
test.)

So I guess it was just a rumor? Any comments?

Martin

PS.: The aha1742 contains a BIOS version (c) 1991, and a revision
ahead (G) of the one that came with my Jensen (H), so it's pretty
sure it just contains an Intel BIOS.
