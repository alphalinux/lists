Chris Ingrassia <chrising@erols.com> wrote:
> When I get into the install program for RedHat 4.1, it does everything
> fine until it trys to actually install something, at which point it
> says: "ERROR: RPM Database Not Found"

Sounds like you're trying to do an upgrade. The RPM database is in
/var/lib/rpm; do you have /var symlinked to somewhere else? For instance, I
used to have /var symlinked to /usr/var. If you do that, you need to use a
relative symlink instead of absolute. In my case this was as simple as
changing the link from /usr/var to usr/var (note the removal of the leading
slash).
