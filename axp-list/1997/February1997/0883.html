That's one reason why I want to get Linux running in the interim... maybe I
can get this machine putting all its muscle into the RC5 cracking contest
for a few weeks to atone for the fact that it will be running at 70%
efficiency for the remainder of its life.

Has anyone else noticed how inexpensive this caliber of machine has gotten?
We purchased ours about 45 days before (of course, that's the way it always
works) DEC's big memory price drop, but even so this machine is very well
priced. You can now buy a single processor EV56 400MHz machine with 4MB
cache, 128M RAM, SCSI, 100baseTX, decent video (for a server anyway), etc.
for well under $30K. Couple that with the expandability to as many as 4 466
MHz CPUs, 8G RAM, etc, and it's a hell of a deal. Certainly no Intel-based
machine can even compare.
