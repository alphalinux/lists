In your previous message Tony Galan wrote
>
> Hello,
>
> I installed red hat linux 4.0 in my alpha udb, but I don't know how to set
> up the video. I know udb's have TGA (21030) graphics card, but which
> Xserver do I use? Can I use Xconfigurator to set up my video? What specs
> do I need to input?
>
> Thanks.
>
> Anthony Galan

I copied Xtga-ELF-EV4-test.gz from the cdrom drive and named it XF86_TGA
in the /usr/X11R6/bin directory. Chmod 4711 XF86_TGA.

Copy axpbin-Xtga-BETA-.02.README from the cdrom and edit out the comments
from Jay Esterbrook and modify the monitor section for your monitor.

Rename the changed file to /etc/X11/XF86Config

ln -sf /usr/X11R6/bin/XF86_TGA /etc/X11/X to relink the server

startx

This worked for me, but I had to play with the monitors for a while and I
still don't have all the resolutions worked out yet.
