On Sat, 22 Feb 1997, Chris Ingrassia wrote:

> Ok, I've finally managed to find the correct boot image for my machine
> (Alphastation 200 4/233, I'm using avanti-s.img, i think that's the
> right one), and it boots to linux from the floppy, but right after
> detecting all of my SCSI devices, it says something about the MS-DOS
> filesystem (some sort of lengthy error), and gives me an analigned
> access kernel trap, and then says it can't mount the root filesystem.
> What am I doing wrong? I at first assumed it might be because I made the
> disk with dd and not rawrite, so I went upstairs and rawrited it, but
> still the same result... please help!

I'm guessing you're trying to boot to the ramdisk install image? Use

boot dva0 -flags "root=/dev/fd0 prompt_ramdisk=1"

and try again. Make sure you rawrite ramdisk.img to a floppy as well.
