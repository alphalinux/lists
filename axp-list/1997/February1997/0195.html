> Hi!
>
> I asked some 10 days ago about installation when floppy
> drive refuses to work, but got no answers...
> So has anyone managed this? Any ideas how to do it?

Ummm... it depends: are you using SRM or ARC? If you're using
SRM, then you can use bootp to boot over the network, but if
you're in ARC, the only way I know to get into SRM is to use
the floppy (although I think you can get the required SRM update
stuff on a CD.) Of course, if you're not stuck with a "half-
flash" machine like I am, then getting to and from SRM should be
much easier.
