I've made a patch that allows Linux/Alpha users to cross-assemble i960
assembly code using binutils-2.7. The patch fixes some inapropriate casts
in the i960 support code. With this patch I've successfully assembled
non-trivial hand-assembly and compiler generated assembly for the i960.

The compiler is going to take a bit more fussing before I get something
clean enough to release. Mostly, the problem there is configuration files.

Anyhow, the binutils-2.7 patch lives here:

ftp://ftp.picturel.com/pub/source/README.html
ftp://ftp.picturel.com/pub/source/binutils-2.7.alpha.patch

Feedback is welcome, I intend to use my new Linux/Alpha for real work and
would like to get any problems hashed out early.
