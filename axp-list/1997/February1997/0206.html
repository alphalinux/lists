There are some jumpers on the mainboard that you may need to set. Here is a
table that gives some of the modes you may need for a NEC 3FGe

Vert Pixel
W6 W5 W4 W3 Standard Resolution Refresh Freq(MHz)
0 0 0 0 VGA 640x480 59.94 25.175
0 0 0 1 ERGO VGA 640x480 72.79 31.5
0 0 1 0 SVGA 800x600 60.32 40.00
0 0 1 1 ERGO SVGA 800x600 72.19 50.00
0 1 0 0 VESA 1024x768 70.07 75.00
0 1 0 1 DEC 1024x768 72.033 74.367

0 = installed, 1 = not installed

If you hunt around, there is a WWW resource pointing out the location of the
jumpers. As I remember, they are not always well marked. Check through the
list archive over the past couple of weeks, the address has been posted
recently.
