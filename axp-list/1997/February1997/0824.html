bob@bob.usuf2.usuhs.mil said:
> That would be fine if I could boot the system and log on.

Boot into single user mode. At the MILO> prompt type something like

boot sda2:vmlinux.gz single

where sda2:vmlinux.gz is replaced with the path to your kernel. Once
you do this, you will get a bash# prompt, and enough of a system to
remove amd with the `rpm -e amd' command.
