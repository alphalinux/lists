I have a Sun GDM-1662B monitor on my UDB with TGA. All that was necessary
to get the thing up and running under X was convert the Hsync and Vsync to a
Composite sync signal. Although the GDM-1662 is one of Sun's old multi sync
monitors one should be able to get the fixed frequency one working as well.

- Some helpful info -
Biorobotic Vision's web page on hsync and vsync to composite sync
http://cvs2.anu.edu.au/monitorconversion/monitorconversion.html

For modeline starting points look at Vigra's web page
http://www.vigra.com/manuals/vs10/man-vs10.html
http://www.vigra.com/manuals/tigrix/userman.html

BTW: I only have X working haven't yet modified tga.c in Milo and the kernel to
sycn in text mode.
