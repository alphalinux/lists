To properly drive those monitors you will need a video card that
can generate sync-on-green or build external circuitry to get the monitor
to sync. For instance, the S3-968 chipset can generate sync-on-green but
the Mach64 series cannot. Most of the higher end graphics cards can
provide this signal. If one is only interested in running X and not the
console no additional programming is necessary other than finding the
correct modeline. If you need the console you might want to look at
SVGAtextmode, a program that can change sync signals and fonts using a
configuration file similar to XF86Config. I don't know if this program
will complile, run or fry the monitor or card under AXP. I use the TGA
server with a single freq. monitor just fine( no console). Let me know if
you want to unload any of those monitors or if you have anymore questions.
