Howie Schneider wrote:
>
> Hello all:
>
> I have Redhat 4.0 on a 166. I would like to upgrade to the new version
> (4.1). 4.0 came on a cd when I bought the unit. I have no idea how to
> upgrade from ftp. Is there some documentation somewhere or a faq on how

Did this last night. Create the root and boot disk from ramdisk.img
and noname.img from the images directory under the RedHat directory.
You can do this with dd, see man dd or do dd if=<image>.img of=/dev/fd0
with an empty disk in the floppy drive. You might have to be root
to do this step.
Stick the boot disk in the floopy drive, thats the noname.img disk.
Have all the IP adresses ready, including the IP adress of the
ftp server you are going to use. And the redhat directory with the
axp distribution.
You already have milo installed so on the milo prompt just type
boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1 prompt_ramdisk=1
and follow the instructions on the screen.
