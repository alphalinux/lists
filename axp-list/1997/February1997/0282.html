>>> David Paris said:
> Hi. I have a UDP that I want to stick my Matrox Millennium Power Desk 4mb
> card in so I can run at high color depths in X. I saw some traffic on this
> mail list about it and tried to follow all the steps but it just wont
> work.

Unfortunately, the BIOS emulator in the UDB ARC/SRM consoles is old, and
doesn't properly handle the BIOS code on the Millennium. However, the latest
MILOs *do* have much better BIOS emulation, and should work better.

This is true of *all* Alpha platforms: the latest versions of the ARC/SRM or
AlphaBIOS consoles have much better support for the Millennium, so if you're
having problems, grab the latest version and upgrade. Likewise for MILO.

Now, on the UDB, you have no newer firmware to try, but the latest MILO
should help a lot.

> Hmm I just went

That was your monitor protecting itself from a bad signal from the Millennium,
