Ran into a little problem while attempting to install RedHat 4.[01]
on a VX40 Multia. I attached an external Seagate Hawk 2LP to the
the little guy, and fdisk didn't seem to be able to pull the drive
geometry off of him. No problem, sez I...I'll just bop up to the
Seagate website and get the geometry and enter it directly, using
the advanced mode in fdisk. Unfortunately, fdisk only allows a max
of 63 sectors; the Seagate has 103.

I seem to have finessed the problem by varying the number of heads
and cylinders, but I was wondering if I'd missed a more elegant
solution.
