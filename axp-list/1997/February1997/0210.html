> I asked some 10 days ago about installation when floppy
> drive refuses to work, but got no answers...
> So has anyone managed this? Any ideas how to do it?

If you are using the SRM console, I think you can boot directly from
the RedHat CDROM. The command should be something like this:

>>> b dka400 -fl i

This will put you at the aboot prompt where you enter the name of the kernel
you want to boot and any flags you need.

If you are using the ARC console, you may still be able to boot from the CD,
but since I have never done this I can't be much help to you.
