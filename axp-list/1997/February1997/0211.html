I guess the first question is are you going to be running more
than 1 job at a time? If you are not then the 2 processor system
is not going to help you much, but you probably know this anyway.

> that the alphas are very fast machines, but also that Linux

At this point for your type of application I think the PPro machines
are going to be better for you. The compilers/libraries for the Alpha
are not very mature yet and this will show up for the type of task
you are going to be running. Digital Unix will work much better for
you than Linux for this type of application, you might consider it.
