I sent this message to Erik at RedHat and thought if there was something
obvious that the other folks on the list may see, it would be nice to get
this going...
I'd installed RH4.1 from the Official CD and got hangs when going
multi-user. Got a couple of helpful hints from Erik then ran into brick
wall. So...
--------------------------------------------------------------------------------
1. I decided to try and start over from scratch. Downloaded new glibc,
ypbind, amd, (and everything other than XF86 & tetex which would have taken
forever).

2. Did the complete, from scratch install (from external hd with old rpms
removed, updates copied in). Official RH4.1/axp from cd with updates.
Install completed successfully.

3. Machine rebooted, hung right after the 'eth0: media is TP' message.

4. Rebooted in single user mode, applied single line fix to
/etc/rc.d/init.d/network to add './ifup ifcfg-lo boot' before for-next loop.

5. Rebooted to multi-user, ypbind gave error message about 'YP_DOMANINAME'
not found and machine hung right after 'amd'.

6. Rebooted into single user mode. Removed nsswitch.conf (that is installed
now, wasn't before).

7. Rebooted multi, got ifcfg bad messages.

8. Rebooted single, took S55named.init out of /etc/rc.d/rc3.d (BTW,
named.init is NOT in rc5.d)

9. rebooted multi, everything OK except that I dont have a nameserver
running. I can telnet/ftp to myself and everybody that is in my hosts table
on my local net. Can't try ppp to my ISP because no pcmcia on RH4.1/axp (pet
peave - it's in the intel distrb. and I personnally helped Dave Hinds get
the makefiles to work for UDB's - which is (I think) the only alpha boxen with
pcmcia slots. My UDB's modem is pcmcia. Furthermore, I'd like to use a Sony
CDROM that has a pcmcia Slim-Scsi as my system cd-rom (so I can, like, install
RedHat from cd. Again, Intel install allows pcmcia installs, why not alpha?)

Where to go from here?

P.S. Where is the default window size set for Grail? If I use 800x600 in X,
the grail window goes above the upper edge and I can't resize it (without
going to Window ops - max/resize).
