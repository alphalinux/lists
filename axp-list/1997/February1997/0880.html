One newbie to another, dude. You have choices. If you buy from DEC
then you
are on your own re Linux. There are a fair number of reputable
companies that
buy DEC built Alpha based mother boards (mine came with a DEC keyboard
and mouse)
and build a system around this. These vendors will load and test both
operating
systems for you. At least two such companies regularly participate in
this forum.

I also had to bring an older DEC built Alpha machine up on the same
Linux as
the above mentioned machines. It went 1-2-3 up and running. I can not
see how
a x86 based machine could install any quicker, unless you like
_transported_
the data onto your drives.

MHO.
