>I looked at the BISS bits this morning. With the following patch, I was
>able to run all the test programs that come with their regular (as
>opposed to kaffe) distribution. It was dreadfully slow, but it did work.
>
>I'll see about getting some of Sun's JDK tests and running those this
>evening.

I am (as are a couple of friends) considering getting a UDB running Linux.
At current prices, they're a very attractive option. However, one of the
primary uses would be for Java development. Is there an official JDK port
to Linux/Alpha yet? I've not been able to find any information about one
on either blackdown.org or any of the Linux/Alpha web pages or FAQs.

I appreciate Kaffe/BISS, but I suspect for the moment, they'll lag behind
the offical JDK by a few months. If there isn't a Linux/Alpha JDK, is the
Digital Unix emulation complete enough to run the DU JDK under Linux?

As a side note, how good is the DU emulation in general? On the Sparc port,
SunOS emulation is pretty much complete -- I'd hope DU emulation is at the
same stage. Presumably I'd need a licenced copy of DU to get the necessary
shared libraries.
