upgrading) 4.1. The install seemed to go fine and the system works for the
most part. But it won`t die! Whenever I try to shutdown the system it
broadcasts it's warning message and things continue to run. Reboot, halt,
telinit, and shutdown with the -n switch are all similarly inefective.

way to use the concole is to alt-Fn to a not-yet-used VC.

Anyone have any similar problems? Solutions?

My machine is a UDB and it's only odity is an additional PCI network card.
(I can get both cards and the network to work by removing the S10network
link in /etc/rc.d/rc3.d/ and running my own script.)

everything".
