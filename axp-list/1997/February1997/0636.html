>Multia that we've purchased.
>
>I'm not really even certain where to start - if I "boot dva" from the
>SRM console with a supposed boot image in the floppy drive, I boot for
>a bit, and them am asked for the location of vmlinux.gz - I type in
>vmlinux.gz, and it boots the old RH4.0 install on sda1[2?]..
>Basically, ARC is vaguely confusing, and I'm not entirely certain I

You should install from the ARC console.
Type ARC at the SRM prompt to get into ARC console mode on reboot.
This should get you started...
