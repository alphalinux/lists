I have seen a lot of these errors but only after an RH4.0 or RH4.1 install
that has not been completely successful (which for me is often). The only
successful formula for an install is to remove the two 32M simms I bought
for the 3rd and 4th slots and just use the 24M that came in the machine.
Then, after the install is complete, I can put the other SIMMS back in for
88M total. For some reason an install does not work with a lot of memory.
