On Fri, 14 Feb 1997, Rob Montjoy wrote:
> There is yp support glibc but there seems to be no NYS stuff
> compiled into it. It is the NYS stuff that allows you do without
> a ypbind on the SPARC and X86 ports of RedHat.
>
> Anyhow, I grabbed ypbind-3.0.tar.gz from the net and compiled it.
> I had to comment out the IPC (shared memory stuff) in slave.c
>
> Here is what is in my /etc/nsswitch.conf
>
> passwd: files nis
> group: files nis
>
> I have a /etc/yp.conf file but I do not seem to need it.
>
> Make sure that you set your NIS domain using the domainname command
> before running ypbind.

This worked for me, too, but remember that you need to get
nis-tools-1.0.tar.gz as well, and apply the included patch to the ypbind
directory (otherwise it won't compile.)

And, you need to make a /var/yp directory, so that ypbind
can store its binding information.
