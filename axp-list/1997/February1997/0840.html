> And oh, since I'm using the mouseport mouse that came with my Alpha,
> what special settings do I need to use? There seems to be no option for
> mouseport mice anywhere, and I do not know which COM port to tell it the
> mouse is plugged in on, since it seems to ask no matter what.

You should try to make a link from /dev/modem to /dev/psaux. Psaux is
the correct device, this isn't a COM port. If it doesn't work, you
will perhaps have to start gpm manually with the option ps/2 for using
the mouse with TTY based programs, X works without gpm.

If this doesn't work as well, you will probably have to recompile the
kernel with PS/2 mouse driver enabled.
