Open Sound System for Linux on Digital-Alpha Workstations

LOS ANGELES: 4Front Technologies is announcing the availability of the
the Open Sound System(TM) sound drivers for Linux (OSS/AlphaLinux)
running on
Digital Alpha workstations. OSS/AlphaLinux is a commercial
implementation of
the OSS/Free drivers (the free soundcard drivers provided with the Linux
kernel distribution).

OSS/AlphaLinux provides a user-friendly GUI which makes the installation
of
sound drivers and configuration of sound cards extremely simple!.
OSS/AlphaLinux
even supports Plug-n-Play (PnP) soundcards for which there is no support
in OSS/Free. Finally, OSS/AlphaLinux supports many soundcards which are
not
supported in the OSS/Free sound drivers.

4Front Technologies has developed OSS/AlphaLinux to be compatible with
OSS/Free for base audio functionality, but it also provides additional
audio
interfaces such as support for 4Front's revolutionary new SoftOSS -
software
emulation of wave-table synthesis hardware. SoftOSS on the Alpha can
deliver
crisp MIDI music rendered at 48KHz-16bit-stereo sound, using wave-table
instrument
samples. For more information on SoftOSS please visit 4Front's web site
at:
http://www.4front-tech.com/softoss.html.

AlphaLinux users can visit 4Front's Web site at
http://www.4front-tech.com
to download a "free" 5-day trial version of OSS/AlphaLinux. The software
is
currently available for beta testing. It will be commercially avalable
after
two weeks.
