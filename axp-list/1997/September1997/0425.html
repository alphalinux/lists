>>> Ronny Ranerup said:
>
> anyone else had problems with an Alpha dying when switching between consoles
> (Alt-[f1-..]). I can just do it a few times before the Alpha hangs completely.
> Very nasty.

It'd be helpful if you'd said which server and which version, but...

Are you using the latest XFree86 release, 3.3.1, available as RPMs on
Red Hat's website, in the updates/alpha/ directory, for RH 4.2?

If so, then you should file a bug-report at xfree86@xfree86.org.

Also, if you're using an S3 Trio64-based card, try adding:

Option "no_pci_disconnect"

to the Device Section of your XF86Config file.
