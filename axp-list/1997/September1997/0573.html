It would be helpful to see the messages, but they might be from "loop" or
"md" devices. which are are *not* supported as part of MILO. Build your
kernel with only the standard devices (floppy, IDE and/or SCSI, Ethernet
even though its not used).

Also note that later MILOs build better against a kernel source tree that
has been patched with alpha-patches-2.0.30-0.[12].gz.

> plus i have 64 megs of ram and milo says i only have 24 :(
> how can i change this?

This sounds like a mismatch between MILO and linload.exe; make sure you're
using the latest linload.exe as found at:

gatekeeper.dec.com:/pub/DEC/Linux-Alpha/Miniloader/arc
