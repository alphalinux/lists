>Any comments on what this workstation would be worth? and will Linux run on it?

hi,

this issue has been discussed *many* times, but what the heck :-)
a multia will definitelly run linux (at least Red Hat, never tried (yet) Debian
or Craftworks), but with relatively poor performance (between P90 and P150,
depending on the type of software), due to small cache and slow
memory access ... if you really want, you can overclock it, but cooloing may
become a real problem, and the result is maybye not worth the pain ...
the video board will allow you to run the Xfree86 X server
only (Metro X not supported, and likely won't ever be), and the 340Mb HD (I'd be
for a SCSI device, but check) won't allow you to install all the software
packages. don't even think installing an additional drive inside, there's no
room and the power supply won't make it. so you'll have to get an external HDD
plus a CD. 32M ram sounds ok for a general purpose use.
if you want to install a video board, the (supposed) SCSI interface (a NCR
53C810) offer you *one* PCI connector (but there're room in the box for a small
board only)
the multia (aka udb) also feature two pcmcia connectors (maybye not on all
models, check first, where you may plug a modem and ???)
imho, go if it's *cheap* (and depending on where you are cheap may means
different things, in France i consider <200USD), but check first how expensive
an AS200/250 might be ... you could be surprised
good luck (and bravo for joining the Alpha Powered fans club)
