I had the same problem with the new em86 0.2 and 2.0.30 kernel. The
memory base address changed with the em86 patches. So just look at
/proc/pci to make sure that the memory base that you are setting in
XF86Config is correct.

Note: with the new em86 patches I get this memory base address from doing
a cat /proc/pci.

Prefetchable 32 bit memory at 0xa000000.
