I've seen those ``No such file or directory'' error messages from
mv during the stage1 compile stage when (successfully) building g77
for Intel and the Alpha under DU. I wonder if perhaps during stage
1 only the C compiler is built, hence the Fortran object files do
not yet exist, which is no problem, but generates harmless error
messages from mv. In other words, I don't think the mv messages
are related to your problem.
