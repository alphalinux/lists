Roman Martonak wrote:
>
> I have successfully compiled and installed g77-0.5.21 on my friend's UDB with
> 40 MB RAM and 96 MB swap, it took 3 3/4 hours and went smoothly. However, when
> I compile the following simple program and try to run it (no optimization),
>
> program test
>
> real*8 a
> integer*4 i
>
> do i = 1,4000000,1
>
> a = a + dlog(dfloat(i))
>
> enddo
>
> write(*,*)'a = ',a
>
> stop
> end
>
> the program crashes with the message Floating Exception. Any idea what might
> be wrong ?
>

You have to initialize a. The f77 standard does not specify
what value should be given to an uninitialized variable.
