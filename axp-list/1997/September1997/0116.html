On Mon, 8 Sep 1997, Benoit E. Donnette wrote:

> | My X server keeps on crashing when there are 20 to 25 X clients open

> I beleive that one of the following should work:
>
> -More swap,
> -After reconfiguration of the swap parameters,
> -More RAM.

No, this has nothing to do with it. You don't even have to run the clients
on the same machine. I tried right now with XFree 3.3 and this was the
memory status just before X was crashing:

ajo@wopr ~ >free
total used free shared buffers cached
Mem: 62120 61600 520 37760 9704 23520
-/+ buffers: 28376 33744
Swap: 102384 3688 98696

I don't see any problem with these figures.

I have seen this problem for a long time (and been annoyed about it).
There was one server which didn't suffer from this, but I don't remember
the exact version. I was using the S3 server at that time.
