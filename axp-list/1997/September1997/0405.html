I just got MILO up on a Personal Workstation 500au. I am using the AlphaBIOS
to boot from. I created an entry in the graphical setup for booting MILO from
a floppy. The floppy is a MSDOS formated disk with "linload.exe" and "milo".
Make sure you have the latest versions... I think it is MILO is 1.5.
(miata version of milo)

I have been able to boot the Linux Kernel from the Redhat 4.2.1 CDROM using:

MILO> boot -t is09660 hda:/kernels/miata.gz root=/dev/hda load_ramdisk=1

where hda is the IDE CDROM. I'm still having trouble formating the SCSI
hardrives when I am in the Install script.
