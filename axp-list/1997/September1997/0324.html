Hi,

I was trying to use modules on my Alpha PC164, but a very weird thing
is happening.

The compilation goes fine, I reboot the machine, but when I try to
load the module ...

[root@alpha /root]# cd /lib/modules/2.0.30/misc
[root@alpha misc]# modprobe cyclades
bash: /sbin/modprobe: No such file or directory
[root@alpha misc]# ls -l /sbin/modprobe
-rwxrwxr-x 1 root root 29160 Sep 17 10:39 /sbin/modprobe
[root@alpha misc]# insmod cyclades.o
bash: /sbin/insmod: No such file or directory
[root@alpha misc]# ls -l /sbin/insmod
-rwxrwxr-x 1 root root 28728 Sep 17 10:39 /sbin/insmod
[root@alpha misc]#

This seems to happen because the modutils version that I have
installed in my machine is not ported to Alpha.

Does anyone know where I could get a module utilities package to work
with kernel 2.0.30 in Alpha ???
