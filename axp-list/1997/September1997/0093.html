I'm not sure from the wording of the original question, but he might have
been asking if the SRM console supports the Ethernet card, ie can you use
it to BOOTP a kernel over the net? ARC console can't do this, and AlphaBIOS
is going to at some future time.

I've found being able to BOOTP a kernel (built for direct boot from SRM, of
course :-), to be a great labor-saving technique when debugging kernel code.

Others may want to use BOOTP to load a kernel when there's no local disk
at all.

So, do you know if the 3C509 supports this? I'm betting not...
