Does anybody know how the amd distribution is created?
What does the upl102 mean? Are they bugfixes/updates for the official
amd last updated in 92? Where can one find info on the Linux distributed amd?

According to the amd info pages, the latest amd should be at ftp://usc.edu/pub/amd/,
but that one doesn't even have a working Makefile for Linux and it isn't called
anything like the amd-920824upl102-8 that seem to be the latest for redhat.

I think I found a much newer debian amd (i.e. a higher last number) but I can't find
it anymore.

The problem is that the amd I've got is seriously buggy. I think I found
a way to make it more stable though, instead of:

/defaults type:=nfs;opts:=rw,hard,intr,nosuid,nodevs

I use:

/defaults opts:=rw,hard,intr;type:=nfs

The first one I use on all intel-linux-boxes but that fails miserably on Alpha. The second one
seems to work much better, but I really would like to have the latest amd before
going any further.
