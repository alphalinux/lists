On Wed, 10 Sep 1997, Rick Windham wrote:

> Hello,
> I just put in new memory 32Megs and about 5 days later I'm getting
> LCA machine checks. This memory was not shipped in a static bag. :-( How
> can I tell from the syslog what bank the LCA Check was in? And does this
> absolutely mean the memory is bad? I've had the machine for a 6 months
> and this has never happened before; but one of the power connectors on the
> motherboard is a little loose and the machine reboots peridoically on it's
> own. I'm getting a new motherboard shipped to me...should I try it out
> before I send the memory back? It's a UDB 166Mhz soldered in CPU.

The memory is not necessarily bad. I have examined an axppci motherboard
which gave more LCA checks with more heat (and it also froze sometimes if
it got hot enough). So I tried to move the ram, and there was no problem
with the ram in another machine.

If your new motherboard is close I recommend waiting for it.
