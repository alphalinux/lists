On Fri, 12 Sep 1997 lebrun@lasa13.astrsp-mrs.fr wrote:

> Hi,
> thanks for your prompt answer,
> excuse my ignorance, but how do I check the 'ecoff' compatibility of my system ?

I must say I was hoping for a file in /proc which contained a list of
supported binformats, but apparently there is no such file :(

On the other hand, it seems you have already found out with strace that
this is not your problem. If I strace an ecoff binary without the ecoff
support this happens:

execve("./linpack_dp_roll", ["./linpack_dp_roll"], [/* 21 vars */]) = 0
strace: exec: Exec format error

This was not your error with strace so I guess you have ecoff support.

ecoff support (or a.out support) is by the way configured when the kernel
is compiled.
