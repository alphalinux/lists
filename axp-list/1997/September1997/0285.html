Hello,
I have heard some people were trying to use ide on the 164LX series
and I thought I might mention that the ide controller on this board is
a cmd 646. The cmd640 had serious problems with linusx. So far in
fact there are special compile options int the kernel for it.
I thought this might help some one out.

On a side note : Does anyone know if the new bios ofor the 164 LX fixes
the missing irq on the parrallel port??
