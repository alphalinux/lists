I am almost sure I did not use the right configuration options. Before,
when I was compiling the kernel in order to use the sound, I installed the
sources from the Redhat rpm package, and used mostly the default options, and
everything worked fine. Now, starting with the unmodified 2.0.30 source tree
and patching it with the alpha patches that come with em86, the configuration
dialogue was slightly different, and I was not sure at several points what
to choose. Could somebody send me the .config file appropriate for UDB
(for completeness, I am attaching my, probably wrong one) ?
