OK,

I took the same pre-built version from RedHats site, but I also got
xemacs common files, as if I were going to compile it. These files also
need to be placed in your usr/local area in there respective positions. If
I recall I had to place them there myself, after they were automatically
untarred into the root area (which is where i guess the xemacs people
thought we would be keeping this stuff).

You can get the common files from www.xemacs.org. No compiling needed,
just a bunch of info type files for the pre-compiled alpha stuff. Maybe
redhat should keep a copy of these files on there site too?
