Hi everyone!!!
In my Case I used to DS10
One disk Alpha Linux the other Tru64 UNIX.
First I installed Tru64 UNIX and then ,, install Alpha Linux
Modify os_flags, boot file in the SRM mode
When boot linux, I just power up
When boot unix, >>> boot -file vmunix -flags a dka0 ( device )

It is no problem . It is very simple and easy.
