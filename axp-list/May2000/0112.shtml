  I just built Motif 2.1.30 from the sources provided by the open
  group. I build for both Intel and Alpha, but to make the alpha work I
  had to change the linux section of Imake.cf file as show below. Was
  there another way to solve this?

  motif/config/cf/Imake.cf

         #ifdef linux
         # define MacroIncludeFile <linux.cf>
         # define MacroFile linux.cf
         # undef linux
         # define LinuxArchitecture
         # ifdef i386
         # define i386Architecture
         # undef i386
         # ifdef __i386__
         # ifndef i386Architecture
         # define i386Architecture
         # endif
         # undef __i386__
         # endif /* __i386__ */
         # endif /* i386 */
         # ifdef __alpha__
         # define AlphaArchitecture
         # undef __alpha__
         # endif /* __alpha__ */
         #endif /* linux */
