The FAQ at alphalinux.org can help you answer most of these. I know for sure
that the sx164 will work (I'm writing this on one
right now :-), and I believe a good SCSI card can be had for $50-100. Good
luck.

Carl Aron wrote:

> I run a small web-hosting and development company, and I am considering
> building my new web server using Alpha hardware and linux.
>
> My current server is an Intel clone AMD K6, running RedHat 5.2, and is
> chugging along nicely with about 50 domains (admittedly, some of these
> are very small, not very active web sites). I figure I'm OK for a while,
> but I want my next server to be good enough to host 400 or 500 sites if
> possible.
>
> I've been looking at several AlphaServers on eBay that look like
> possibilities, but I don't know what's a good model for installing Linux.
> Some of the descriptions are a bit short on detail, and I often can't
> find exactly what they describe in the Digital web site's Retired
> AlphaServer section.
>
> If anyone can comment on the various models available and problems or
> benefits with running linux on any of them, I'd greatly appreciate it.
>
> What I hope for:
> Hardware: at least 256 MB RAM, at least two drives, preferably SCSI, with
> software mirroring (RAID?), so that I can just swap drives in the event
> of a failure. Tape backup, too. CPU fast enough to stand up to multiple
> domains using Perl, PHP, and SQL.
>
> Software I want to run: Apache web server and Redhat or other SSL web
> server, Perl, PHP, MySQL, PostgreSQL, POP, IMAP, Sendmail or similar,
> FTP, Secure Shell, PGP. (ASP if I can become convinced it won't choke my
> machine or open security holes).
>
> The auctions that look like possibilities are:
>
> DEC4000 model 610 -- Big monster with tons of RAM, 6 SCSI drives and bays
> for 10 more drives, DAT and DLT tapes. Attractive because it is very
> complete with lot's of room to grow, but possibly an ancient model,
> either slow or or not usable with linux?
>
> Alpha Server 2000 P233 -- A complete system, with drives and tape backup,
> but I don't know how fast it is or if it will boot linux.
>
> Alpha 21164SX 533 MHz Server -- A nice complete system, but has a single
> EIDE drive rather than multiple SCSI. Can SCSI be added for a reasonable
> price?
>
> AlphaServer 3300R -- This looks like a real nice one. No Hard drives and
> not enough RAM, but that can be fixed.
>
> Any advice you can give me would be greatly appreciated. Thanks!
