I'm having trouble getting IP masquerading working on Alpha
under RH6.2. I don't know if it's Alpha specific or not.

Has anyone gotten IP masquerading working in RH6.2? I had it
running perfectly in RH6.1 but after installing 6.2 I can't
get it going. :-( And I used the same procedure.

I am using the Linuxconf firewall tool. I have a standard
setup with one gateway and several machines connected to it
on a private network.

On the firewall host I enable only forwarding rules, enable
masquerading, and set it to route from my private network
address 192.168.0.0/24 to 0.0.0.0. This worked fine in RH6.1.
I should say that I'm using the default route, I've enabled
IP forwarding, and I can successfully connect via ppp from
my firewall machine to the net. I also have NFS running between
the gateway machine and my other internal machines, so it's not
a network problem. I've also enabled ipchains and firewall through
Linuxconf to automatically start. /sbin/ipchains -L looks OK to me.

Any ideas????
