Dumb Kid wrote:
>
> On Intel platform, I just need to pop in a boot floppy and type
> "rescue" at the lilo prompt.
>
> How to one do the same on Alpha machine which is using AlphaBIOS ?

Take your installation media (floppy or CD) and start an "update". Do
not install anything but after an initial setup switch to a shell on the
second console (Alt-F2). You will end up with in an environment which
is probably (I did not compare step by step ramdisks from RH 6.2) richer
than on Intel.

AlphaBIOS has nothing do to with this save minor details of an inital
boot which will be a bit different with other types of a firmware.
