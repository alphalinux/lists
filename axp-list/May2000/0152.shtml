> What type of architecture is in the
> DEC ALPHA 3305?
> Will this machine run Linux? Tru64Unix?

http://www.compaq.com/alphaserver/archive/3300/digitalserver3300.html

I just got a 3300R this week and am beginning the process of installing an
OS and setting it up. So far it looks great. It'a very heavy-duty
professional server. The memory is a little pricey (about $2/Mb EDO ECC
DIMMs). It's still available -- it's used in high-end servers like these.
You need to install the DIMMs in sets of FOUR (256-bit data path!). SCSI is
Ultra-Wide and used standard 80-pin SCA2 hot-swap drives. You can also add
regular 50-pin or 68-pin SCSI drives.

I don't know for a fact, but I think Tru64 is a no-go on these. There was
some mail this week and last on the FreeBSD-Alpha list about possible boot
problems with the 3300s. I'm not sure how that panned out.

Good Luck!!!
