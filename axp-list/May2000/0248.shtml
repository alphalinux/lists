> How does one measure this on the UP2000 ?

Hold up a thermometer to the air exhaust external to the case, as we have
been advising for a while.

> It seems that lm_sensors don't compile on the Alpha, did someone
> suceed ?

Not all chipsets have lm sensors. The UP1000 does, the UP2000 does not.

> The room is cooled at about 30 - 35 C, it is quite crowded with
> computing stuff.

That's extremely hot. A normal room should be closer to 20C ambient. As Tom
said, they have tested the UP2000 at 40C ambient and it was OK. But you
certainly don't want to run it that way in reality.

> The system is usually stable, but had an unexplained crash two weeks
> ago. It is working most of the time with both CPU. However, the system
> disk is not used very much but an external disk array.

I have yet to met an external disk array that would function correctly at an
ambient temperature of 30-35C.
