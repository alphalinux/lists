> things using ccc/cxx when I use libraries bulit with them and link them
> with to other thinks like mod_php I'll get unresolved symbols unless I use
> ccc/cxx exclusivly.

This is normal -- different C++ compilers usually don't mix. With different
C compilers you can often get away with it, but it depends. I think Compaq's
goal is to make that work, but you'd have to be sure you didn't use any C++
if you filed a bug report.
