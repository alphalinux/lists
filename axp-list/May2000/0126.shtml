Your problem is not a Linux problem, but a 64 bit architecture problem.

... Check out the Tru64 Unix Alpha porting guides.

http://www.tru64unix.compaq.com/faqs/publications/pub_page/porting.html

The full Tru64 unix docs are on-line at:

http://www.tru64unix.compaq.com/faqs/publications/pub_page/doc_list.html

The Compaq (DEC C) compilers are avaialble at:
http://www.unix.digital.com/linux/compaq_c/index.html
If you haven't tried them instead of GCC, you will experience a 15-30%
performance improvement for most code and possibly a 200% improvement
if you do a lot of floating point.
