On Wed, 10 May 2000, Arkadiy Chapkis - Arc wrote:

> I have a small question. How can I set a date on my Alpha AS200 4/233 so
> it'll affect it permanently? Right now when I change the date or time using
> date -s
> command it changes it, but after reboot it comes back as it was. Only option is
> to change it in ARC, but I have to bring the machine down.

After you set the date, you have to write it in the hardware clock. Do
that with 'hwclock --systohc' (I usually add --utc so that hardware clock
is stored in GMT).
