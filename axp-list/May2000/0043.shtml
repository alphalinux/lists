The UX or "Ruffian" machine type uses a slightly different milo than some
other models.
Be sure to use a ramdisk image compatible with this model.
It uses a different loader called "ldmilo"

One needs to create a small ( say about 3MB) FAT partition at the beginning
of the hard disk to store ldmilo and your kernel on.

As there is no utility in the ARCSBIOS on this machine to create it you may
use a program called "arcinst.exe" which may be downloaded from:
http://www.compaq.com/support/files/alphant/drivers/index.html

This will allow you to easily create and format that partition.

There are utilities in the ARCSBIOS console that allow you to copy files to
that partition.

Good luck, and let us know if further is needed.
