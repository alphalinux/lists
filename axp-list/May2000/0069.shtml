On Tue, 9 May 2000, Carl Aron wrote:

> Hardware: at least 256 MB RAM, at least two drives, preferably SCSI, with
> software mirroring (RAID?), so that I can just swap drives in the event
> of a failure. Tape backup, too. CPU fast enough to stand up to multiple
> domains using Perl, PHP, and SQL.

software raid works. Hardware raid is also available, take a look at the
DAC960.
 
> Software I want to run: Apache web server and Redhat or other SSL web
> server, Perl, PHP, MySQL, PostgreSQL, POP, IMAP, Sendmail or similar,
> FTP, Secure Shell, PGP. (ASP if I can become convinced it won't choke my
> machine or open security holes).

thats all supported you're just going to have to do alot of compiling. I
wouldnt bother with ASP if you're already using PHP.

> The auctions that look like possibilities are:
>
> DEC4000 model 610 -- Big monster with tons of RAM, 6 SCSI drives and bays
> for 10 more drives, DAT and DLT tapes. Attractive because it is very
> complete with lot's of room to grow, but possibly an ancient model,
> either slow or or not usable with linux?

Old system. It looks like a turbochannel and thus is unsupported by
Linux. Also compared to the rest of them it's quite slow.

> Alpha Server 2000 P233 -- A complete system, with drives and tape backup,
> but I don't know how fast it is or if it will boot linux.

It looks like a sable and there are sable variations that are supported
but im not sure about this one. It's not a very fast system though i
believe it is SMP.

> Alpha 21164SX 533 MHz Server -- A nice complete system, but has a single
> EIDE drive rather than multiple SCSI. Can SCSI be added for a reasonable
> price?

yeah but you'll spend the same money bidding on the AS3300R and that
already has scsi and alot of other things

> AlphaServer 3300R -- This looks like a real nice one. No Hard drives and
> not enough RAM, but that can be fixed.

This is a sure thing. it has lots of memory bandwidth unlike the SX. Not
to mention the 4 bay hotswap and it's a rack so you have room to
grow. They usally go for around $500 on ebay, a steal for a server that
has a cross bar switched memory architecture. They run Linux great with
two cavets.You have to use atleast kernel 2.2.14 and it must be compiled
as generic.it has SRM available, same thing as a AS800 but a white box.
Also this is new in box so all you have to do is send in the warrenty card
you get 3 years onsite from Q.
