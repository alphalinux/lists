I had RH6.1 working properly on the Jensen. I following the advice on
this list (in particular there was a special Jensen image I had to use
since the generic kernel image never worked). I had no trouble at the
root password part of the install. It did hang on boot though. I had
to replace the gawk package with mawk after booting into single user
mode. gawk was (is?) broken on Jensen due to a library issue. Check
the list archives for "Jensen" and you should find what you need. My
Jensen is now in my "museum" but it may boot again someday. :-)

/Michael
