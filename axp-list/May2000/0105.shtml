If compaq really wants to win over linux users, they should add high
quality optimization for alpha directly into gcc. Intel is already
doing this for their cpu's. I'd hate to see compaq/alpha squander the
potential gains it could make in linux mind share by not fully
supporting open source tools. (I do appreciate their releasing
free-as-in-beer compilers! It just seems that there is a lot of code
duplication in maintaining separate compilers rather than beefing up the
alpha code generation in gcc.)

T.
