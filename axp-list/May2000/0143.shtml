It will run Linux and FreeBSD/Alpha. it uses SRM to boot linux. It's the
same thing as a AS800. Tru64, You would have to try it.

The 3305 is a 500MHz EV56 server with a cross-bar switched memory
architechture, 256 bit datapath, on board scsi, and hot swap drive bays.
Makes an excellent server with it's aweseome throughput.
