> running RH5.2 on an alphastation 200 4/233. The package name is

> it's not like it's that old. I've been looking around for a couple

Good in a sense. There is a new Kmail out which fixes of annoying bugs. Its
version is 1.0.29.1, and I have been running it on all of my Linux boxes
with no problems (Alpha and Intel).

You can get it here:

<http://devel-home.kde.org/~kmail/kmail-1.0.29.1.tar.bz2>

bzcat kmail-1.0.29.1.tar.bz2 | tar -xvf - (use bunzip2 if you don't have
bzcat)
cd kmail-1.0.29.1
./configure
make
make install (as root)

Enjoy,
