There's a good README page when you download the CCC that explains some of
the difference between GCC and CCC. They are not 100% compatible, but they
shoudl be close. I'd like to start doing some tests to compare some
real-world things (e.g. MySQL, APache, etc...).

You can't share *.so files between progs compiled on different compilers, so
if you have a suite of apps that you compile together (e.g. PHP & Apache),
*both* need to get compiled with CCC to work together.

I'm not a guru on kernel compiling. After reading the article, I wondered if
the kernel could be compiled with CCC...
