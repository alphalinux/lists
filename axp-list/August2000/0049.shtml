Hi Martin, hi Alex!

The 3com 509B does work in my jensen. It is not compiled as a module, but
'monolithic' into the kernel. But the first time I installed rh 6.2 after
the initial boot the system hang while it tried to bing up eth0. But this
problem is NOT related to the network card but to the rh network setup. As
the exact same kernel flawlessly worked with SuSEE 6.3 and 6.4. That is
networking worked.
        I guess my problem was, that I gave the IP addresses of DNS-servers that
are only temporaily connected. The second time over rh 6.2 I omitted to
specify DNS servers and booting works OK.

Good Luck!

        Sebastian Moeller
