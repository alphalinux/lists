Plugins for Netscape from Tru64 are in
"COFF format alpha paged dynamically linked not stripped shared library"
format while what you will get just running 'make' on the sources
mentioned above most likely will be
"ELF 64-bit LSB shared object, Alpha, version 1".

It should be possible to convert the second format to the first one
(maybe using 'objcopy' - but possibly after recompiling it
appriopriately, I am not sure) or coach binutils to produce the required
format in the first place. After all gcc and associated tools run on
Tru64 as well. But likely the simplest would be for someone with an
access to Tru64 to produce plugins in an expected format and leave them,
for example, on ftp.alphalinux.org. Still, any 'objcopy' wizzards
around?
