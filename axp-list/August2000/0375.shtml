Uh, no there isn't. The legacy (4.x) Netscape Communicator/Navigator
sourcebase is not 64 bit clean, so building a native Linux version is
impossible. The Tru64 binaries, to my knowledge, use a flag to the
Compaq compiler to generate 32 bit code. From what I've been told,
this switch doesn't work on the Linux port of the compiler, so we're
stuck with Tru64 binaries until Communicator 6.0 (Mozilla).

Of course, you can run Mozilla releases now...
