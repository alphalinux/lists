for a 5300 the manual says this. I don't think it is correct.
especially the speed. 

---
Minimum: 64/256MB
Maximum: 2GB ECC
Speed/Type: 70ns ECC synchronous DIMM
Mode: Two-way interleaved SDRAM
Configuration: Sixteen 168-pin DIMM sockets
Upgradable: With 64 and 256MB EDO ECC DIMM memory kits
---

What I do know is that the latest options guide for this system lists
the same part number for memory that is used in an AS1200.  And that ram
is "SDRAM DIMM 100MHZ UNBUFFERED. (PC66) 16MX72 3.3V 168"   Actually
come to think of it the 5300 is the so called "nt only" version of an
AS1200.  So all you need are standard ECC dimms.
