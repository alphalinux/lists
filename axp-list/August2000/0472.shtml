I'm already using a beta version of Compaq C++ V6.3 for Tru64 which also
supports the "-std gnu" switch to port KDE2 to Tru64. For KDE2 a good and
stable C++ compiler is essential. Currently I can compile it under Tru64
with both, egcs 1.1.2 and Compaq C++ V6.3 beta with the "-std gnu" switch.
There were still some GNUish features left which were not covered by "-std
gnu" but Compaq Compiler folks are very helpful in sorting that out.
