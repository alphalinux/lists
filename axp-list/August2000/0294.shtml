Hello,

I was wondering if anyone has any experience with development tools for
large programs on Alpha Linux. We are currently using Solaris/SPARC
systems, and the development environment is the SUNWspro products that
include compilers, a debugger, and libraries (including their high perf
math library, similar to DXML). Looking at Compaq's web site, I see
products such as LaDebug, which works only for FORTRAN programs (but we
have a lot of C++ code), the Enterprise Toolkit, which, while touting
multi-platform capabilites, appears to be intended to run on Tru64
UNIX. Other packages I've seen (but have no experience with) are
Windriver's IDE and something called Code Warrior.

If anyone has any experience with these on Alpha Linux, I would love
to hear about them.
