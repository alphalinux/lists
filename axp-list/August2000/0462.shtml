The 5300 is the white box AS1200.
The 3300 is a white box AS800.

I've got 3 3300's and, yes, they all use ECC EDO unbuffered DIMMs with out
a problem.

now the 5300 on the other hand uses SDRAM DIMMs. I've tried a whole bunch
of different kinds of SDRAM in it and none seems to work. I borrowed some
RAM from an actual 5300 where I work and the 5300 inits fine, but trying to
actually find the specs on this RAM is proving to be extremely difficult.

The messages that I've posted here before are captures from ttyS0 durring
bootup.

It would have been great if it did come w/ memory.... then I wouldn't be
going through this headache =)

110698I003 01
90000-20658-04B
second dimm:
110698I003 00
90000-20658-04B

those don't look like any standard DEC (Compaq) part number though.

well at least I know that the machine works now..... so it is worth
