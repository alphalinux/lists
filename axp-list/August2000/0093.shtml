W Bauske wrote:
>
> I see there is a file called /etc/security/limits.conf.

This is used by PAM libraries and is described in PAM docs.

> How do the shell settings interact with what's there?

Your 'login' program is supposed to consult that, via pam stuff,
and set assorted limits.

This is really a mechanism by which you can adjust these
values for non-root accounts. The catch is that your
/etc/pam.d/login configuration file should include a line like

   session required pam_limits.so

before this will work and I am not sure if a default one from
any distribution does this.
