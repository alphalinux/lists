> Does anyone know how to adjust the key repeat rate under KDE? I have a client
> who wishes to not have a pause when she hits a key before it starts repeating.
> We used to have this set in solaris by passing a flag to the window
> manager. Is
> there something similar I can do with KDE? Is there something in RH6.2 that

I believe that the X server mechanisms for setting key repeat haven't been
implemented until XFree 4. With XFree 4 you should be able to configure
this in the X server configuration and xset r
