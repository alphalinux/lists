Compaq DS20E is a 264DP-based system anyway so yes you should be able to
use their 833 MHz 8MB DDR cache daughtercard expected for DS20E after
Christmas (all 833 MHz CPU before that are reserved for the ES40). Of
course, neither Compaq nor API will officially support using this on 264DP.

Compaq has concluded that beyond 833 MHz, the existing Tsunami chipset,
even in the full-bandwidth config present in DP264 / DS20E / ES40, provides
diminishing returns for faster CPUs (imagine than that the returns would be
even lesser in the UP2000+ which only has half the mem bandwidth of DP264).
Cpq will use a new chipset with AGP Pro and triple 64-bit 66 MHz PCI buses,
as well as 60% higher mem b/w using PC133 CL2 at 512-bit width, for 1+ GHz
21264s from next spring...
