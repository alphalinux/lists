"John Eshleman" <jeshleman@hpti.com> wrote:
> > I assume
> > that this would require me to use UFS on the DU side, but what would be
> the
> > mount command for the RH side? Is there anything special I have to
> install
> > in order to allow RH to read the BSD UFS partition from DU?
>
> mount -o ufstype=sun /dev/sdb3 /mnt
> mount -o remount /mnt
>
> The "sun" ducks a byte-ordering problem, and the remount gets it to
> read-write instead of read only. cat /proc/mounts to figure out what's
going
> on.
>
> also...you need to compile the kernel to support UFS and UFS read/write.
