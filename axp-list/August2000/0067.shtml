On Fri, 4 Aug 2000, Bob Taylor wrote:

> Attempting to install openssh and am getting a dependency on this.

Many of the rpms from Alphalinux.org use the DEC (Compaq) C compiler, of
which this is a dependency. While it may seem like a bit of a headache,
it is definitely a good thing. :-) Applications compiled on Alpha Linux
using the DEC compiler are typically significantly faster than the same
app compiled with gcc.

There are links to the libots rpm as well as the others at
www.alphalinux.org.
