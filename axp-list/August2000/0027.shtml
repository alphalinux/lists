Hi Alex,

This is a long shot...is your 3Com 3C509B configured for EISA mode? If the
card came out of an EISA machine(e.g. Compaq Deskpro 486/M series), it will
be in Eisa configuration to match the EISA bus in those machines. Only way
to get back to ISA is to put the card in an EISA machine and run the 3Com
EtherDisk Diagnostics and Auto Configuration diskette, manually select ISA
mode and save.

Give me a shout if I'm on the right track and you need some help.

Ed Schalk
