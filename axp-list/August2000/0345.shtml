To all:

I'm setting up a 3 node cluster and I seem to be having a problem with the
hosts.allow and hosts.deny file.

Basically, I want to allow only certain services to certain hosts.

My hosts.deny file looks like this:

ALL: ALL

My hosts.allow file looks like this:

in.telnetd: master, compute1, compute2

If I remove both the hosts.deny and the hosts.allow file, I'm able to telnet
into the master from either compute01 or compute02. Once I put in
hosts.allow and hosts.deny, I can not telnet into the master from he compute
nodes.

Does anyone have any idea why? As far as I can tell, I should be able to
telnet into the master using the files above. Did I edit the files wrong?

Any help would be greatly appreciated.
