Dave Gilbert wrote:
>
> Hi,
> I've spent the day messing about with KDE2-beta4 and had quite a bit of
> success. I should say I actually run Gnome but pick and choose interesting
> KDE apps.
>
> In particular I've been messing about with Konqueror; KDEs web browser;
> and I'm starting to think that it is getting stable enough to be more
> stable than Netscape on Alpha (by whichever means the other parallel
> argument decides exists.....).
>
> I built all the Qt/KDE stuff from source (there is a kde-rebuild script
> somewhere on their servers - can't remember where - which takes a lot of
> the pain out of it).
>
> Konqueror manages to render most of the pages I've given to it;
> including one which repeatedly killed Netscape. Having said that I found
> one which repeatedly killed Konqueror - but the KDE guys seem to be nice
> enough about it.
>
> It will manage https (get yourself a recent version of OpenSSL installed
> first) and will kind of manage Java. For the Java I installed a Kaffe
> pre-built RPM I found on rpmfind.net. It starts applets up but they open
> in there own window rather than in the browser - but I guess that means
> it is getting there.
>
> Oh, Kpresenter seems to SIGFPE in this version - but I haven't tried
> -mieee'ing it yet.
