David Webb wrote:
>
> I'm running RH6.2 on a pc164lx. I wanted to try
> using the Compaq compiler to compile the
> 2.2.16 kernel

Linux is not written in Standard C, never was, and nobody ever claimed
that this is the case. If you peek into kernel sources you will find
out very quickly that they heavily depend on a C dialect and constructs
which are gcc specific. You cannot even drop an optimization and
correctly compile a kernel.
