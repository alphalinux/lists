"William H. Magill" <magill@isc.upenn.edu> writes:

> I believe that there was only ever one box which had NO SRM console
> accessable by setting the CMOS "console selection". That was the
> "white NT" box - I forget it's name. To the best of my knowledge, all other
> boxes which shipped with the ARC console showing could flip to SRM by
> simply changing the CMOS setting.

The XL series never had a (publically released) version of SRM.

Also, I seem to remember reading in the 21064/66 manual that some EV45
chips lacked hardware support for 64-bit memory management. The 21064
manual says:

   Except for its memory-management functions, the 21064A-275-PC is
   functionally identical to the other four 21064A microprocessors. The
   21064A-275-PC will only support the memory-management functions
   necessary for the Windows NT operating system and other operating systems
   that use the Windows NT memory-management model.

Were these chips used in any Digital machines?
