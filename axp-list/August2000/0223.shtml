Bingo! Thanks!

The jumper was the trick to the whole thing. The jumper in question is J2 -
it's in front of the box, near the CPU. The normal state of the jumper is 2-3;
to boot in fails safe mode, select jumper 1-2.

Next, I created a floppy using mkboot in the utilities directory
(ftp://ftp.digital.com/pub/DEC/Alpha/firmware/utilities/) based upon the file
found in
ftp://ftp.digital.com/pub/DEC/Alpha/firmware/v5.8/alpha200/. The "winning"
file that I used was as200_v6_9.exe - the "losing file" (and I'm not sure
why...) was as200_v7_0.exe. But, I'm a little ahead of myself.

I booted the the as200_v6_9.exe floppy, did a full flash (or whatever the menu
said.) Held my breath - rebooted, and was happy to see the >>> prompt.

I'm guessing it was one of two things. I pulled down the wrong firmware image
or I don't have enough memory (I only have 16M in the box.)

Thanks for everyone's help! Should be an (official) member of the AlphaLinux
crowd in a few minutes ;)
