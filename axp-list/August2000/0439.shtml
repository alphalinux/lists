Mark Arrasmith <arrasmith@math.twsu.edu> writes in axp-list@redhat.com:

> ... short-circuit of rpm -bb. If the package is already compiled,
> is there any way to jump past the prep, compile, and install stages
> straight to building a binary?

We patch this into RPM. It is a very small patch.

--- rpm-4.0/rpmqv.c.orig Mon Jul 10 00:49:36 2000
+++ rpm-4.0/rpmqv.c Mon Aug 7 01:15:43 2000
@@ -1280,6 +1280,8 @@
           case 'b':
             ba->buildAmount |= RPMBUILD_PACKAGEBINARY;
             ba->buildAmount |= RPMBUILD_CLEAN;
+ if ((ba->buildChar == 'a' || ba->buildChar == 'b') && ba->shortCircuit)
+ break;
             /*@fallthrough@*/
           case 'i':
             ba->buildAmount |= RPMBUILD_INSTALL;
