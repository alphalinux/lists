Has anyone had luck with an Actiontec PCI call-waiting modem
on Alpha? I have one installed in an XP1000. lspci shows the
modem is detected and gives me the I/O and IRQ info. But I
simply cannot get a dialtone from it. :-(

I used setserial to set it up:

setserial /dev/ttyS2 uart 16550A port 0xa000 irq 28 autoconfig

But this doesn't seem to have any effect. I tried getting seyon
and minicom to dial but it doesn't work. KPPP returns "Modem
Busy" when I try to query it.

Is this just an incompatibility? I know these modems work in the x86
world.
