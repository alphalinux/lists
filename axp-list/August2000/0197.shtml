  We had problems making CDs that could be read on DPW500s under Digital
  Unix using mkisofs and cdrecord, though the CDs works fine on other
  machines.

  The mount would work, but the top level directory would be
  invisible. Very odd behavior.

  We fixed this by using the newest mkisofs (mkisofs 1.13) and the
  following flags:

     /opt/schily/bin/mkisofs -nobak -iso-level 3 -untranslated-filenames -J -r -l -D -f -m RCS

  These disks seem to be ok on all the systems we have ( Digital Unix
  4.0e on DPW433au, 500au and 255/233, Linux on both intel and alpha and
  various windows 95/98/ny/2K boxes)
