On Thu, 10 Aug 2000, Iwao MAKINO wrote:

> I agree with Maurice's comment.... And you may not know this fact, but that
> currently shipping Alpha products only support SRM.

Exactly why a Linuxbios is so attractive. NO SRM anomolies, no need to
have to use BSD disk label etc. Ideally something very simple. SRM works
fine, and in truth so does the old ARC console. Both have their
detractions (video cards, Raid cards, etc that don't work with one bios or
the other) The idea is something very simple, easy to use, and let the
Linux kernel figure out the rest. Ideally if you have a linux driver that
functions correctly (big assumption) for your great XYZ attachement, it
should work regardless of BIOS.

So far on the x86 machines the bonnie numbers are better under Linux bios.
Go figure, but wouldn't it be nice if we saw a performance increase to add
to a easier to use BIOS? Here's to wishful thinking.
