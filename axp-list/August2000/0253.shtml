It should complie OOTB. Did you try to build in the source directory
(this is not recommended)? What configure line did you use? Did you
"make bootstrap" or simply "make"?

Normally, I do something like...

gzip -dc gcc-2.95.2.tar.gz | tar xvf -
cd gcc
mkdir build
cd build
../configure --prefix=/opt/gnu --enable-languages=c++,java
--enable-threads --enable-shared
make bootstrap

...and wait an hour or so. If it completes without errors it is ready
for "make install".
