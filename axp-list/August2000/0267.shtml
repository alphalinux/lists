>>From the context I understand that you are talking about Linux console?
Try (this is 'echo' from bash):

echo -en "\\033[1;1;44m"
echo -n and now for something totally
echo -en "\\033[0;39m"
echo " different"

In general for things like that consult Keyboard-and-Console-HOWTO.
If you just want boxes with a colour on a console type 'man dialog'
and see examples which come with it.
