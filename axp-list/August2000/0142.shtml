what I've found works really well is this:
select fdisk and write a BSD disk label to your drive(s)
once that's done, hit the back key.
then you can use RHat's nice graphical installer and it will respect the
BSD partitions and put aboot on the disk for you.
my guess is the disk(s) you have don't have a BSD disklabel. RHat's
installer is set up to check for the existance of one. If it's there, it
uses BSD, if not it uses DOS, shich sounds like your problem

lemmie know what you find
