Personally, I find all of the Linux disk tools very primitive -- forcing me
to do things that I gave up doing back in Ultrix days (like calculating
cylinders and the like).

However, Since I dual boot with Tru64, I find that using disklabel and/or
the diskconfig GUI gives me a much cleaner way of dealing with the
partitioning, then I simply use fdisk to change the type fields. It solves
all the problems that fdisk has in dealing with large disks.

It also allows me to mount the Linux partitions on Tru64 - but since I use
AdvFS on Tru64, not the reverse.
