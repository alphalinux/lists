Compaq's Alpha Technology Solutions Group is pleased to announce the
availability of Linux JumpStart! v1.0 for AlphaServers and AlphaStations as
an open source project.

Now you can get involved with development of JumpStart! for Alpha. Visit
http://www.linuxalpha.compaq.com/sourceforge and go to the JumpStart!
Homepage.

The JumpStart! open source project uses SourceForge software on our site so
you can download copies of the JumpStart! source and provide useful updates
back to us. There are bug tracking forums, message forums, a patch manager,
news etc. You can also provide us with inputs on features you would like and
we can see if we can add it for a future release.

Linux JumpStart! for Alpha is a multipurpose bootable CD to make installing
and running Linux on Alpha systems easier. JumpStart! enables you to
automatically set up your Alpha system to install Linux, rescue an Alpha
Linux system or act as a bootable Alpha Linux CD. It can automatically
partition your hard drive, set up environment variables and prepare to
install many Linux distributions. Advanced users can use it like a rescue
CD.
