frampton@mail.flarc.edu.on.ca said:
> I was hoping to find an "updates" directory on the redhat.com FTP site
> which I could just download and upgrade. Alternatively I could order
> an "update" CD but not at the $50 price...

There *is* such a thing, at ftp://ftp.redhat.com/updates. There is also
an errata page that can be found by following the support trail from
www.redhat.com.

gatekeeper.dec.com has an ftp mirror that is loads faster and more
accessible. (Must be running on an alpha:-)

The updates fix many things, to be sure. Do load them. After that is done,
you will need to install a non-redhat-patched kernel to get gtk based
programs like GIMP and control-panel working.

(This last thing is an irritating problem, it's been around for a while,
and the workaround has been around for a while. If nothing else, RedHat
should report the hard workaround on the errata page. It took me a *long*
time to figure out that this was necessary.)
