If you got that far then you are done as far as kernel and hardware are
concerned. The rest is a question of setting properly your software
support. Check files in /var/log/. Usually there are some informative
messages there if something failed in a print chain.

I've never been a fan of this "automated" control panel printer setup;
it is hard to be sure what it is doing and what went wrong. But this is
just me. :-) Besides you can always barge in with an editor into
/etc/printcap and print filters listed there.

Consider "Magic Filter" software.
