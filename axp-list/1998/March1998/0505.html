> the build-in ethernet chip, but fails to initialize the network if I add

I had quite similar one, SMC Etherpower using a 21040 chip, in my
UDB. I think I used de4x5 drivers... I didn't need any statements
to the MILO prompt (I had the driver compiled as module, if it matters).

For some time, I had also an PCMCIA card in my Multia, no problems
with it either.

Kernel version was probably 2.0.30 at the beginning and later 2.1.xx
(I've since bought an XL300 so I don't remember all the little details
from UDB days).
