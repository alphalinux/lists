I found a security hole in my RH 4.2 system (164LX, Millenium II).

In rare cases, while the screen is locked, pressing <Ctrl> and
then <Enter> gives me access to the system without entering a
password. I don't know how to reproduce this reliably but this
problem definitely exists.

Remarks:
- I am running XFree86 Version 3.3.1, which is known to have only
experimental support for the Millenium II (but I don't know if
this problem is either specific to Version 3.3.1 and/or the
graphics hardware used.
- I don't know if this problem is specific to axp-linux.

Has anyone else seen that?
