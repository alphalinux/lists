These are shared libraries. Did you rerun ldconfig after you put
all these libraries there? If some of that went into /usr/local/lib
you have to make sure that this path is included in /etc/ld.so.conf
before that run.

ldconfig runs every time from a startup scripts when you boot but if
you installed new shared libraries somewhere you have make sure that
re-running ldconfig is a part of a startup process.
