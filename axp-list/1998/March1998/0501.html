I got an Enorex PC164 system with a 3com PCI Fast Etherlink XL 10/100
ethernet card and a cheap video card that has the Cirrus 5446 chip in it.
I installed RedHat 4.2 for alpha on this machine and built a kernel for
the 3com 59x (9xx) ethernet interface. The kernel boots and it
finds the device: 3Com 3c905 Boomerang 100base-TX at 0xb400, IRQ 17
If I try to configure the ethernet device:
ifconfig eth0 10.5.5.3 netmask 255.255.255.0 broadcast 10.5.5.255
my machine hangs. I want to run the card at 10Mb, not 100Mb and it
works OK on Windows NT. Anyone know what to do to get this to work ?

2nd problem is the Cirrus 5446 video card. I selected what seemed
appropriate during install but when I try to startx, I get:
(--)SVGA: PCI: CirrusLogic GD 5446 rev 69, Memory @ 0x02000000

*** None of the configured devices were detected ***

Anyone know how I can get this card to work ?
