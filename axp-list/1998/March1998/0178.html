Cornelius Creedon wrote:
>
> speaking of overclocking multias:
> any success/failure stories on overclocking the 166 MHz cpu?
> (a little wary to toast my only alpha...)
>

Well IMHO, don't.

I'm not even sure if the VX040 (with the soldered-in CPU) has the
appropriate jumpers. Even if it did, the next available speed would be
200Mhz. which is more than the 10-15% which is considered safe to
overclock. In addition, if you did pop the CPU you would have to replace
the entire motherboard (unless you have surface-mount solder/de-solder
equipment around).

The VX-41 (with the socketed CPU) I think has the jumpers but I would
still not try it unless I had a spare CPU on the shelf. If you have one
of these you might try to obtain a 233Mhz. CPU for it though.

Again I think that faster Alpha machines are getting very affordable.
The 275Mhz. board and CPU mentioned on this list recently for $250 can
be clocked to 300Mhz. and would make the core of a nice machine that is
MUCH faster than any flavor of Multia/UDB (due mainly to the wider
memory buss of the 21064 and the 2MB. cache).
