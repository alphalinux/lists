I had the same problem and found this as a cure:

Start with a fresh 2.0.33 tar.gz
Install the symlinks as described in the Readme of the 2.0.33 kernel.
(replacing i386 with alpha ofcourse)
patch this kernel with the 2.0.32-0.1 patch
(to be found at gatekeeper.dec.com somewhere in Alpha-Linux/Kernels)

make config & all that stuff..

The patch updates resourcebits.h

Have fun!
