On 11 Mar, Joao Rochate wrote:
> I want also to use it with the 21'' monitor.
> So, there is a PowerStorm 4D20 with 16Mb of VRAM (??). But I can't use it
> with Linux. I've tried to use it as a TGA server, but "no can do".
>
> Is it really true that I can't use the PowerStorm?

Yes, it's really true. Of course, if you'd like to write a drive for it
and donate it to XFree86, I'm sure they'd be happy to accept it.

> Well, since I can't, wich videocard should I buy (not directly to Digital)
> and install it on a PCI slot? Should I buy S3 based? Matrox Millenium?

I believe that the Mach64, S3, and Millenium drivers have all been
ported over to Linux/Alpha. I'm not terribly impressed with the S3
server in 3.3.1 myself, so I'd recommend a Millenium card.

Note that you can most likely use the PowerStorm under Digital Unix or
NT.
