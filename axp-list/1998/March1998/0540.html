On my UDB the memory base that worked for me was 0x02000000, which
I got from a sample XF86Config file or a HOWTO, I don't exactly remember
which. When using this memory base address everything seemed to work fine
with the TGA card and Xwindows. The Xserver may require a different memory
base than the one that /proc/pci states. Though I just looked at my
/proc/pci for my UDB, and the only address it gave was 0x0200000!?! Is
this a standard UDB (i.e. 166MHz), and is this TGA board what I think is
called a TGA-1 board? There are a two and three that support higher bit
depths that are not currently supported by the TGA Xserver. (Sorry, I
can't remember the technical name of the video board at the moment.)
If you want a copy of my XF86Config file, just ask. Hope this is
of help and good luck!
