On 9 Mar, Rick Sharpe wrote:
> Forgive me if this is an old one but.. I have an Alphastation 500/400 that
> I recently installed Redhat Linux 5.0 on. I then tried to add some extra
> packages and updates with rpm. Most installed OK but several complained
> that they required file libc.so.6 and a couple wanted file libdl.so.2. So
> what did I miss during installation or where do I get these files from?

These packages are probably for RH 4.2, which had libc.so.6. 5.0 has
libc.so.6.1 due to incompatibilities.

You'll need to recompile these packages from the source RPMs by
downloading the corresponding src.rpm and running rpm --rebuild
file-1.0-1.src.rpm and then installing the resulting binary RPM in
/usr/src/redhat/RPMS/alpha/.
