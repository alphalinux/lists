Poor Philippe ...

I "solved" this Problem :
I waited 20 (!!!) minutes. Then Linux managed creating a 800 MB Partition !

Then i ran into an other Problems, namely the configuration of the X Server (TGA).

I "solved" this Problem in exchanging a 2000$ graphic Card (16 MB Memory)
with a Matrox Card (200$). (Is there a driver for PowerStorm on Windows/NT ?)

Then i started the SVGA Server. Then i configured it.

Then i waited again and again ...

To come to a conclusion, the system(433au) now is terrible slow, slower
than a 486/66 Mhz Linux System. I don't know why.

The access to the hard disk is also extremely slow.
I measured 0,53 Mbyte/sec. (hdparm -t)
(On my AMD K6/200Mhz System with Adaptec Controller 2940 it is 4MByte/sec on Linux.)
The network link is very slow (btw.: how to configure the eth0 with 100MB full duplex ?)

I will make a Summary and put it on our Web Server because
many Students of our University were curious about the Performance
of Linux on an Alpha System.
