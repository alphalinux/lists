On Tue, 17 Mar 1998, Uncle George wrote:

> does anyone know how to add a multicast address to the route tables?
> a route add -net 230.0.0.0 netmask 255.0.0.0 dev eth0
> gives a net not reachable error.

Uhm, is there already an IP address on this card? Sorry, I'm not fully
sure what you mean by 'multicast', but if this is the case, you'll have to
assign a second IP address to the card. I don't know how to do this
offhand, but RedHat has a set of files in /etc/sysconfig/network-scripts
for configuring such a thing. I'm sure they have nice little tools that
do it, too. You'd have to check the rc files to see what to do, but I
think you just create the file ifcfg-eth0:1 (for example).

Hope that helps,
