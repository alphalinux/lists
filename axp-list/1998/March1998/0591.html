On Wed, 25 Mar 1998, Sheer Panic wrote:

> yes, there is a flaw: most computers don't draw 200W.. some will push 100W
> when they are spinning up, but usually they pull about 50ish.
>
The specs on the 164LX motherboard says that it draws 100 W.
The CPU itself draws ~ 40W (and since they are never idle, you can
bet they are drawing the max).
On top of this, you have several 2-3 W fans going, the power supply
isn't all that efficient so it's using some watts, surely you have
multiple 7200 RPM drives drawing 10-15 watts apiece... etc.
NIC, HA, all drawing more power.

so ~150-200 is a good figure for total power draw.
