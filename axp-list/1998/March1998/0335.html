Keith,

Here's my printcap for DJ540. I'm using 550C filter w/o any problems. You'll
need to make proper changes to your settings like printer name and lp1 to
lp0 etc. also make sure filter is pointing to your particular filter
location. Here's my printcap entry. I'm sure you can get it to work with
proper changes ;)

#
#
crayon|dj550c|color|HP DekJet 550C:\
:lp=/dev/lp1:sd=/var/spool/lpd/crayon:sh:mx#0:\
:if=/usr/local/bin/dj550c-filter
#
#
