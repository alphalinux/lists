On 12 Mar, Hiren Hindoccha wrote:
> 1. I've an opportunity to buy a Digital Alphastation 200 4/166
> for $699.00 (including the monitor which looks pretty generic).
> This station comes without any manuals and I have no idea
> what is in it (i.e hardware). The vendor (used computer store) claims

The manuals aren't a big help, IMHO (I have them).

> that it will run all windows applications . It currently has

Well..I believe that FX!32 will run a significant portion of Windows NT
programs compiled for NT/Intel, however this is emulation and will be
slower than native code (there's some native Alpha NT stuff, too).

> Windows NT4.0 WKS installed on it. Is $699.00 a good price to
> pay for it ?

Yeah, that's not bad. This one was purchased from DCG for something
around $1k sans monitor.

> Memory installed on the Workstation shows up as 56 M ( I looked at the
> setting from the NT Control Panel)

That should be acceptable for workstation usage, although you might
want to bump it up a little higher (64-72).

> 2. If it is a good deal and I buy it, I currently have a RH4.2 Linux
> running on a Pentium 100 MHZ, 64 MB , 2 GB HD which acts as an
> internet gateway, caching DNS server, dial on demand , packet filtering
> firewall, SMB machine, Fax server (hylafax) machine. Would it be
> adviseable to change this to an NT machine and change the Alpha to
> be a Linux machine ?

You could run Linux on both of them ;) But yes, I would personally
recommend Linux over NT on the Alpha as you'll have to run a
significant amount of things under FX!32 anyhow.

Of course, this has a problem: there's no decent web browser for
Linux/Alpha. The new version of lynx that I've RPM'd up is much
improved, but is still lynx. The new version of Grail that came out the
other day is also improved, but is still awful. There's very little
else, although I assume that porting Netscape will become a top
priority once the source is available March 31st.

> I'm reluctant because I don't have the specs for the machine. I've searched
> the digital web sites and looked at the FAQs on the RH site and all I could
> determine is that the Alphastation has an NC ??810 chip.

DEC hasn't disowned this system, and you should still be able to find
information about it somewhere on their website. (They have, for
instance, disowned the UDB).

> How can I find out which sound card it has ?

It's a MSS compatible card.

> Which Ethernet card ?

21040, 10BaseT/2/5 autosensing. Same as the UDB has, I believe.

> What are the SCSI settings ?

Fast SCSI-2 via an NCR810 chip, again like the UDB.

> How can I find all the above information ? Is there a Digital Web site
> where I can download the specs from ?

Look around the website some more, I seem to recall seeing it. Or, you
can ask me and I'll get you whatever information you need.
