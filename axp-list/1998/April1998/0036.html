Hi,

i just got Suns HotJava Browser to run with my RedHat5 and jdk-1.1.5 from
uncle g.

To save some people some work i made the jdk115 with classes and HotJava
one big tarball.

You can download it from

ftp://alpha.tabu.uni-bonn.de/pub/Linux-AXP/
ftp://z40456-2.tabu.uni-bonn.de/pub/Linux-AXP/
or
ftp://data.tops.net/pub/linux/axp/

the first two URLs are faster (10mbit).

The file is called "RH5-alpha-jdk115+HotJava.tar.gz" (9.8MB)

If you extract it into /usr/local it will run out off the tarball.
I included one lesstif lib (libXm.so.1.2).
The package is not jet well tested, but until now it worked for me.
You may want to install lesstif if you will run into trouble.
If you experience errors or it shuld not work, drop me a line (also
do so if it works :), please.

best regards,
--armin

PS: thanx uncle for the new jdk :))
