I put the 'date' binary on a floppy, mounted the floppy as soon as
the shell on the second virtual console was started, and set the date
correctly. In this way, I installed 5.1 with all the time stamps correct.

Unfortunately, I was so excited to have found 5.2 in the local bookstore
that I forgot to do this on the upgrade.
