The wrap around is caused by incorrect display timings. And incorrect
display timings will often also cause the screen to blank. Select a
different resolution and see if it works better. Also if you have XFree86
configured correctly you can use ctrl-+ and ctrl-- to switch resolutions.
If all else fails, edit the timings directly in /etc/X11/XF86Config.

And it doesn't matter if you run the X-server as root or a user. It has
excessive privelages... (come on FB support...)
