I had the same problem when I was doing this. As far as I could tell, your
milo is fine, it's the Ruffian kernel image that is lacking - no driver for
the on-board SCSI (doh!).

You can get around this by using a different Ruffian kernel image - for
example I took one from an older Red Hat distribution. The 5.2 ram disk
image with a 5.1 ruffian kernel produced no trouble for me.
