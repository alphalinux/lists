Compaq releases Linux Prototype Driver for Compaq PCI-based Array
Controllers

HOUSTON, December 11, 1998 - Compaq Computer Corporation (NYSE: CPQ), the
world's leading PC server vendor, today released worldwide a prototype
driver for Compaq PCI-based Array Controllers to address the requirement for
customers that use the Linux operating system and require leading edge RAID
subsystems and large amounts of external storage. The Linux Driver for
Compaq Smart-2 PCI Disk Array Controllers is beta software that enables
developers by providing them a source code base and specifications.

Developers interested in the Linux Driver for Compaq Smart-2 PCI Disk Array
Controllers are open to further develop, test, and support it. Compaq does
not provide any warranty or technical support. The Compaq Linux Driver for
Compaq Smart-2 PCI Disk Array Controllers is provided under the terms of
Version 2 of the GPL General Public License as published by the Free
Software Foundation. See the GNU General Public License for more details.

For more details please refer to the notice and documentation provided with
the release on http://www.insync.net/~frantzc/cpqarray.html. The driver will
also be made available on www.kernel.org.
