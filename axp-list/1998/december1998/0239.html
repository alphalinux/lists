Which alpha? I think some images have trouble. If you absolutely need to
upgrade, you can ftp download the latest packages of the programs that you
use and install those. 5.2 monkeys with the .confs of the computer, so,
only upgrade if you care to bother fixing them back to your old configs.

I keep a local copy of 5.2 around, so if I screw up some package I can
quickly reinstall it.
