If you've got variation 0, you've _not_ got the latest MILO. Note
that the latest MILO does _not_ live, as one would expect, in the
Miniloader subdirectory, but rather under Kernels.

Please clarify "generic 2.1.131 kernel". Is that CONFIG_ALPHA_GENERIC?

> What's the difference between Generic and (say) EB164 or LX164?

Generic supports all supported systems in the same kernel. There
is a minor performance penalty, and the kernel is about 200k larger.

EB164 and LX164 are "variants" of the same motherboard. Their main
I/O chipset is different (CIA and PYXIS, respectively), so kernels
built especially for one will not run on the other.
