I'm not using a 56k modem (USRobotics V.Everything). It's been working
since I first installed Linux (Red Hat 3.0.3).

        /usr/sbin/chat \
                ABORT BUSY \
                ABORT "NO CARRIER" \
                ABORT "NO ANSWER" \
                ABORT "NO DIALTONE" \
                ABORT ERROR \
                REPORT CONNECT \
