Anybody successfully using BRU to back up to a SCSI Zip drive
on an alpha?

At home I have an intel/linux system running 5.1 with an
internal ATAPI zip drive, and I'm able to use BRU to back
up to the zip drive. But on my alpha/linux system at work
(also running 5.1) I get lots of checksum errors when doing
an integrity check on any file that's over a few KB. The
only differences are intel->alpha and atapi->scsi. It's
not a media problem, and I have no problems whatsoever
using the zip drive to mount filesystems.

Any ideas about what the problem is?
