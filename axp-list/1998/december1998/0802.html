>
> Bob Segrest wrote:
>
> > I have a SCSI DAT tape transport that I would like to use with my Alpha
> > Linux system. I have the hardware installed with a SCSI id of 6, but
> > haven't figured out which device in the /dev directory points to it or a
> > command that makes it do anything. I found the HOW-TO for ftape, but have
> > not found anything for SCSI tape operations.
> >
> > Any hints greatly appreciated...
>
> Here's one, 'man tar'.

SCSI tape works nice on my Alpha, I use /dev/st0 (or better /dev/nst0,
when you want to put more than 1 file on the tape !!), tar or taper for
archiving and when using tar you likely also need the mt utility to
position/rewind/retension/erase etc.the tape.
