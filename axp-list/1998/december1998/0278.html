The latest pcmcia-cs package has always worked out-of-the-box for me.

1) untar
2) make config, hold enter until prompt returns
3) make all
4) make install
5) /etc/rc.d/init.d/pcmcia restart

Enjoy.

(No, I've never tried 2 modems, and I've never tried EQL pcmcia or
otherwise.)
