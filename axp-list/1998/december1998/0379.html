Thanks to all those that replied to my question about the
new location of the AlphaBIOS site;
http://ftp.digital.com/pub/Digital/info/semiconductor/alpha/dsc-software-alpha.html
had what I was looking for.

I should remark that version 5.66-1, which is indicated for
an AlphaPC 164LX, did *not* work in my Durango II LX system
from Aspen. The upgrade completed successfully, but MILO
hangs while computing the BogoMIPS. Cycling the power has
no effect in this instance. According to Aspen tech support,
this indicates a scsi problem. I noticed that someone else
(Peter Petrakis, I believe) reported in this forum the same
problem. I've backed out to 5.62-1a, and all is well again.

Thanks again,
