> Hello (writes he sheepishly):
> I screwed up my profile file. I added a path that unfortunately recycles
> any login to the login prompt. I tried to edit the file through ftp and
> telnet, but to no avail.
> Is there a way I can bypass the current profile file and login to my
> machine to edit it back to the way it should have been?

Try using ssh or rsh to login to yourself. Do the following...

$ ssh localhost /bin/sh -i

And that should give you a pristine prompt. Should. No guarantees.
