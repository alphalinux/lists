This is a VERY complex issue, HOWEVER, for the 21164(A) you should get
the 21164 hardware reference manual from and read it very carefully:

http://gatekeeper.dec.com/pub/digital/DECinfo/semiconductor/literature/21164hrm.pdf

You should also take a look in the "Alpha compiler cookbook", that
explains basics of mutiple instruction issuing and code optimization,
too. It's at:

http://www.partner.digital.com/www-swdev/pages/Home/TECH/documents/alpha_cookbook/cook.htm
