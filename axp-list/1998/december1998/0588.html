> this works fine as long as I boot from floppy:
>
> LOADIDENTIFIER=Linux
> SYSTEMPARTITION=multi(0)disk(0)fdisk(0)
> OSLOADER=multi(0)disk(0)fdisk(0)\linload.exe
> OSLOADPARTITION=multi(0)disk(0)fdisk(0)
> OSLOADFILENAME=\milo
> OSLOADOPTIONS=
>
> But now I would like to boot directly from harddisk. So I created a
> FAT partition (sda1) and put linload.exe and milo there. I changed
> the boot menu definition to
>
> OSLOADER=scsi(0)disk(0)rdisk(0)\linload.exe
>
> but it does not work. Does anybody know how I have to define the boot

change the following, assuming kernel is on 2nd partition:
SYSTEMPARTITION=scsi(0)disk(0)rdisk(0)partition(0)
OSLOADER=scsi(0)disk(0)rdisk(0)partition(1)\linload.exe
OSLOADPARTITION=scsi(0)disk(0)rdisk(0)partition(1)
OSLOADOPTIONS=boot sda2:vmlinux.gz root=/dev/sda2
