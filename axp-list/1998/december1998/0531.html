Jeff's got it. The install kernel defaults to root=/dev/sda2.
If your root partition is sda6, you need to tell it so. Otherwise
it'll panic every time with an 'unable to mount root filesystem'.
Once you're up and running and build a new kernel, this won't
necessarily be a problem anymore.
