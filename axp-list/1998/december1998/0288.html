On Sat, 5 Dec 1998, Bibek Sahu wrote:

> There are upsides and downsides. The upside is that it's native
> alpha code. The downside is that it's a memory hog: although it uses
> shared libraries, those libraries are only shared by other DU programs.
> For someone like me, whose only major DU program is netscape... it often
> seems like it's using a very large amount.

        Just tried it on my XL366, and Netscape 3.x ate up about 11M,
probably more with the libs. That is indeed a memory hog, but that is the
same as Netscape 4.08 on my P100, which is also at 11M at the moment. Just
the nature of the beast (netscape) no matter the platform. Just glad my
XL366 has 196MB of RAM! :)
        Can't wait for Mozilla to get stable, and then a "lite" version
come out, so maybe it will only use 5-8MB! :) Actually, I hope a lot
less... Anyway, just some data points.
