On 6 Dec, David Spellman wrote:
>
> My AS200 runs at 233 Mhz, and the chip is mounted in a ZIF socket. I was
> just curious if it was possible to replace the current chip with a faster chip.
> And if it's possible, does it make sense to do so?
>
> david
>

It probably *is* possible, but you would also have to upgrade the
socketed 466MHz oscillator. The AS200 is severely limited by the 512k
cache (not upgradable, AFAIK), so a faster CPU would not really speed
things up much, since the machine would be memory bound.
