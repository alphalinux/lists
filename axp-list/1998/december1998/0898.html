Hello,
  I just recently installed Red Hat 5.2 for Alpha and have found
a SEVERE error in the X server. (I mean SEVERE ERROR). from what I have
been able to research so far it is caused by the acceleration code in
the Matrox Millennium code. This code is shared so I am saying if you
use the SVGA driver I would turn off acceleration.

What Happens:
*I start X up.
*I usually start Netscape. ( it has happened without netscape).
*The X display wraps about 6 inches to the right
 around the display. So the 6 inches that wrapped of
 the right side shows up on the left side.
*Nothing else happens. The kernel is gone system experiences a total
lockup.
*please note:L I was a user doing this not root.
