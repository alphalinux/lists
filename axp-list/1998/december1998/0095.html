On 02-Dec-98 Philip Blundell wrote:
>>Could the 233MHz 21066A be dropped into a Multia? (Speaking of Onsale)
>
> Some Multias had those processors as standard if I remember right. I think
> the 166MHz versions had the CPU soldered down so the answer is probably no.

The VX-40 had a soldered-in 166Mhz. CPU. The VX-41 had a socketed 166Mhz. CPU
and the VX-42 had a socketed 233Mhz. CPU.

Having said that, I di not know if the motherboard of the VX-41 supports
233Mhz. clock speeds (ie: does it have the speed jumpers, the VX-40 does not).
