I've noticed on my LX164 using rc5des (http://www.distributed.net, DU4.0
version v2.7100.414) that the performance steadily decreases with time.
When I first start the thing up it gets about 535kkeys/s, and now, with an
uptime of 1.5 days, the performace is down to 480kkeys/s. If I scroll
through the logs, I can see it steadily decreasing overnight, when there
should have been *no*load*. I've noticed this several times.

Has anyone else noticed anything similar? top shows that rc5des is getting
~98% of the CPU (rest going to top and X).
