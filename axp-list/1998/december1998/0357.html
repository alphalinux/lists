I'm not sure fully sure about 5.2 (do not yet use it), but I think this
is very likely the old problem with the non-modularized kernel that RH
ships:

The stock 5.2 kernel only supports very few NI cards already compiled in
and NE 2000 is not one of those. Get your system up running, make a new
kernel that includes NE 2000, install it and you should be all set.
