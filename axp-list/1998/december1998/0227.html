I do not advise removing the supplied aliases.

The "problem" with unix like systems is that until you figure out
where/how/why/when of anything, you have no clue where to start. If it is
a configuration problem, look in the directory tree /etc, or the 'dot'
files '.anyfile'. I like Linux, it is just too time consuming to get
everyting to work exactly the way I want it. But, after a while you either
get used to it and/or find the configuration options that you want. If you
like your configuration, back it up 1) to a different partition, 2) to a
floppy, 3) to a CD-R or tape backup.
