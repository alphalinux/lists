David:

not at my AXP box,but a recent intel install ran into the same problem...
see if you've got isapnptools installed, which you can use to disable
Plun-n-Play and set the IRQ and io parameters (if there aren't jumpers on
the card or you can't do this from windows.) one pnpdump and a slight
config edit and everything is going fine now with my cheapo NE2K clone.

see man pages and http://www.roestock.demon.co.uk/isapnptools/ for info
