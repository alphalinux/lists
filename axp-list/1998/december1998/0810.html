> SCSI tape works nice on my Alpha, I use /dev/st0 (or better /dev/nst0,
> when you want to put more than 1 file on the tape !!), tar or taper for
> archiving and when using tar you likely also need the mt utility to
> position/rewind/retension/erase etc.the tape.
>

        I like to use dump, it has all the features I need to do either
complete backups or incrementals. I built a simple menu interface in perl
to control which tape to insert (ie: press 1 for tape 1,.... etc..). Works
awesome. Also allows me to retrieve one file, or one set of dierctories
from tape (I am not saying that other packages can/cannot do this, just my
only experience is with dump)..
