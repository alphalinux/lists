You have some other options if you *just* want a graphical browser (no
mail, no news).

Mozilla has been compiled for Linux/Alpha and works reasonably well if
you don't need java. I'm at work right now so I don't have the location
for it handy, mail me privately if you can't find it in this list's
archives and I'll dig it up.

HotJava also runs well if you have a fast machine and it does do java
(but no javascript). It's reasonably fast on my 533Mhz. box but I once
tried it on a 166Mhz. Alphastation and it crawled.

The KDE file manager, KFM, does a credible job of displaying basic web
pages if you want to install KDE. I don't use it for that because it
won't do authentication of private pages (or if it does I haven't
figured out how).
