> Could someone let me know which 2.1.xxx kernel is the most stable and
> troublefree to install/upgrade for an AS200 4/233, internal SCSI, cable
> modem, generic video card (S3). . . any other advice would be appreciated.

Hello Joseph,

        I've had some success with 2.1.130 on my AlphaStation 200, but I
warn you there are some problems. The biggest is that I've had frequent
Kernel panics (every few days) which appear to be network related. I've
also had a few random kernel deadlocks. :(
        Otherwise, it runs pretty well. You will need to get several
upgrade packages from rawhide.redhat.com to support the new 2.1 kernel
semantics.

Good luck,
