One minor point of clarification...

The names Cabrieolet, EB64+, EB64P, Alpha PCI64-275 and AlphaPC 64, all
refer to the same ATX form factor motherboard.

The EB64 was a full size AT style motherboard designed and sold principally
as an engineering evaluation board for the first generation Alpha CPU. The
EB64 has no PCI slots and preceded the first generation Alpha core logic
chip sets.
