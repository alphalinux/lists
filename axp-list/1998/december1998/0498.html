On Fri, 11 Dec 1998, Roberto D'Amico wrote:

> Hi,
>
> I've installed RedHat 5.1. I was wondering if it was possible to attach my
> SyQuest removable drive and how would I go about it.

Yeh it should be possible; Lets say its an IDE version (you don't say) and
lets say its the second drive on your second interface; that would make
it /dev/hdd (a=1st drive, 1st interface, b=2nd drive 1st interface etc).

Say the disc had already been used under Win95 then you could probably
do:

mount /dev/hdd1 /mnt

(If it was /dev/hdd) If that doesn't work then try /dev/hdd2, /dev/hdd3
and /dev/hdd4 until you find one which works.

If its come off a Mac or something then you will probably need to build a
kernel with support in for the Macs file system.
