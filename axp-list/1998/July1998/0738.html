On Fri, 10 Jul 1998, Mailing List Agent wrote:

> I have yet another of the AS200 4/233's from onsale... I finally got 5.1
> to go into the install process (using avanti-s kernel and ramdisk images
> from SRM console) and I go into ftp install (I have an NT server on the
> network I will pull from) and go to fdisk to drive and it error 11's on
> me... Halts the system and everything... Any ideas?

That's SIGSEGV. It's possible that part of the FTP install is broken. I
did a CDROM install myself (same system, same setup). You could try a
different install method. Another possibility might be heat or a
bad/loose SIMM? Does it always fail in the exact same place? If so, it's
more likely the install (and NOT the hardware).
