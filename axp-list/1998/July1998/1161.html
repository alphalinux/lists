On Mon, Aug 03, 1998 at 03:35:13PM -0700, LSL Information wrote:
>
> Folks!
>
> Linux/UNIX ferver aside, there was never any real doubt that Compaq would
> push NT over UNIX - there is almost certainly more *money* in it for them.
> So let them push NT. Maybe even encourge them to push NT. Because
> when/(if/whatever) NT is proven worse than UNIX the Compaq *CUSTOMERS*
> will move to UNIX/Linux.

Well, but the hardware has to support it.
You can put up boxes with NT in mind which have VGA / SCSI / ... which won't
work with Linux/DUX. (Most hardware does, but you still have to check.)
So at least, the people putting together these should at least have all of
those (or two of the three) in mind, so users are able to change.
I wouldn't buy an expensive alpha for NT, if nobody tells me, that it will
also run Linux/XFree86. I'd also prefer to buy it without subsidizing
Microsoft. but that's another question.
