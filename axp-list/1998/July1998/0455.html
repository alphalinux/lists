Brian Craft <bcboy@dorothy.wanglab.brandeis.edu> writes:

>
> ARC: Won't boot a floppy, much less the hard drive. In both cases it
> returns a file system error. I'm using the avanti MILO image.
> I've written it to several floppies with dd, and I always get
> this result. I wrote it to the hard disk with
> "dd if=/dev/fd0 of=/dev/sda1 bs=1440k". I also tried ftping the
> file and doing "dd if=avanti of=/dev/sda1", etc.

After installation, the boot image should have already been copied to
the hard disk. Did you notice a messages saying "put your boot disk
into the floppy drive if you have not already done so," or something
to that effect? After that the screen displays "copying boot image to
...". If you got MILO booted, you can try

boot sda2:vmlinux.gz root=/dev/sda2

where sda1 is the 'root' file system you selected to install the
system.

After successfully doing the above, you can format your 4MB partion
into DOS fs, copy the MILO image to that partition. And in the ARC
menu, make appropriate changes.

Good luck
