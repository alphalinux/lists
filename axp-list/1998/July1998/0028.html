Section 6.1.1 (page 90) of the "Official Redhat 5.0 Linux Installation
Guide" says:

"Only Red Hat Linux/Intel and Red Hat Linux/SPARC support
modular kernels; Red Hat Linux/Alpha users must build a
monolithic kernel (see Section 6.1.3 below)."

which is plain enough. However, I find that the kernel distributed
with Red Hat Linux/Alpha has modular support compiled in, e.g., the
file /proc/modules exists. In addition, I can find no mention of the
above restriction in the files in the source RPM,
kernel-source-2.0.30-2.

On the negative side, I've built a module for my sound card, a SB 16,
following the directions in the Readme files in
/usr/src/linux-2.0.30/drivers/sound/ but cannot get it to function.
The program "modprobe -a" returns nothing, and "sndconfig" produces an
error related to "modprobe" when it tries to test the card. I've
submitted a ticket on the problem to Red Hat weeks ago and received no
response (as with the previous support tickets, which in anyone's book
is equivalent to no support at all, but this is off the subject).

Does Linux/Alpha support modules? If so, I can supply more details to
anyone interested in helping. In addition to the sound module, I'd
like to install the SCSI tape module, but have started with the sound
module since it appears to have better documentation.
