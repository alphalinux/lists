For those who are having problems with an SX motherboard and the 5.1
ramdisk, my advice is to use the 5.0 boot image then use the 5.1 ramdisk.
This worked for me.

For some reason, the 5.1 boot disk will not read the 5.1 ramdisk.
Of course, you'll boot with the default 2.0.30 kernel
from 5.0, but just compile the new 2.0.34 kernel later...
