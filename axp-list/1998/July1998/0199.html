This version adds better guessing of videoram sizes and attempts
to run on the ZLXp-E3. Like the previous patch, this is a patch
against a vanilla XFree86 3.3.2 source tree.

NOTE: Driving an -E3 is totally untested and based on a couple
assumptions about that card without actually seeing it -- only
experiment if you're feeling brave. I would appreciate feedback
regarding whether the -E3 is successful.

A binary compiled against RedHat 5.1 libraries and a copy of the
patch are available at:

http://www.cs.brown.edu/~tor/xfree86/XF86_TGA-980703.gz
http://www.cs.brown.edu/~tor/xfree86/tga-patch-980703.gz

-tor
