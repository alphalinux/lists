"AARON MARKS" <amarks@sarnoff.com> writes:
> I've seen a lot of guesses on how to install RH 5.1 on a headless Alpha
> (via the serial console). Has anyone actually done it? When I try it,
> it disconnects the serial console at the ramdisk prompt.

I tried it in the past, with an earlier RH version. It was not
possible with the normal boot floppy, because this contains a link from
/dev/console to /dev/tty00 (or similar), the first virutal console.
Thus I first installed RH using normal VGA and keyboard, and later
changed to a serial setup, by linking /dev/console to /dev/ttyS0.

So you'd need to link /dev/console to /dev/ttyS0 on the boot floppy,
then it *might* work.

HTH, Martin
