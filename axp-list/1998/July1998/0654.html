Richie,

do
locate libXm.so.1
that'll should you where the file is, say '/usr/lib/X11/libXm.so.1'.
Then do a
rpm -q -f /usr/lib/X11/libXm.so.1
it'll tell you which package provides it. Then you can get it from,
say, ftp://ftp.redhat.com.

Hope this helps.
