On Sat, 25 Jul 1998, Tony McNeil wrote:

> Here is the problem. When I compile the kernel source
> that came with the RH 5.1 dist. and select the NCR53c8xx

You shouldn't use that.
If you run make xconfig, and choose the NCR53c8xx-option, you will se that
CONFIG_SCSI_NCR53C7xx will be set. This works. So, I use

# CONFIG_SCSI_GENERIC_NCR5380 is not set
# CONFIG_SCSI_NCR53C406A is not set
CONFIG_SCSI_NCRC7xx=y
# CONFIG_SCSI_NCRC7xx_sync is not set
CONFIG_SCSI_NCR53C7xx_FAST=y
# CONFIG_SCSI_NCRC7xx_DISCONNECT is not set

Works great.
