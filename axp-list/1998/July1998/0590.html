Hey Joe, did you try upgrading the firmware already? If not, download the
latest version of the firmware from Digital's FTP server and do an update
as described in section E of your AS 200 manual (did you get one of those
with your Alpha?). You'll basically want to download the BIOS update file
and the "make-a-boot-image" program (unfortunately, a DOS .EXE file), make
a bootable disk image using the two, then boot the Alpha with it. At the
Apu> prompt, you run "update arc". Also, there is a website that you might
want to look at; it has some more information for AS 200 users.

I'm not really an ARC/MILO user; I'm more of an SRM/aboot person, so I
don't know if I can help much further on ARC. If you have SRM questions
though, perhaps I can help. :)

http://www.toast.net/~strycher/AS200.html

Hope this helps,
