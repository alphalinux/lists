Here's my README file:
This is my 'port' of SUN's HotJava 1.1.4 Browser to my RedHat 5.1 Linux
Alpha.
Please note that HotJava will crash with lesstif version < 0.85.3, when
downloading a file via ftp or http.
Therefore i did compile 0.85.3 and put the motif-1 lib (libXm.so) into the
jdk/lib/alpha/green_threads/ dir. I setup the LD_LIBRARY_PATH in the
'hotjava' shell skript, to check that dir first, to avoid conflicts with
already installed older lesstif libs.

I found two bugs in hotjava.

1. If you start Hotjava as root icons (gifs) are broken and it does not
look neat at all :^|

2. Do not use the right mousebutton in HotJava !
This hangs my X forever and i have to login via telnet from another
box to kill java. :^((

This is a nasty bug, since i have to walk across some rooms.

!!! Many, many many many thanks to Uncle Georges jdk !!!

This package contains

1. jdk-1.1.5v10 bins and classfile from Uncle Georgs
2. HotJava-1.1.4-generic from SUN
3. lesstif 0.83.3 from hungry.com

To install please extract the tarball into the /usr/local/ dir
(tar xvzf HotJava-1.1.4+jdk-1.1.5v10.tar.gz -C /usr/local)

and type /usr/local/bin/hotjava

Enjoy.
