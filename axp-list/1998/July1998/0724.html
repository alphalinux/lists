Tibbetts, Richard wrote:
>
> I'm going to throw a totally ignorant question in on this topic:
>
> When I load a new Intel box, the included Netscape runs just
> fine without the need for installing Motif.
> Why then does it not run on Alpha?

Beacuse Netscape does not provide a compiled/executable version of
Nescape for Linux on the Alpha processor. Netscape DOES make available
a version of Netscape compiled/executable on the Linux/i386 platform.

The only Alpha native port available is the DU port, and that was
compiled with linked-libraries, which need to be installed on your
machine to allow Netscape to access them.

You need a license for the libraries.

www.lsl.com is trying to come up with a version of Netscape that
can run natively on Linux/Alpha and not require any additional
software (motif, DU libraries, etc.)... Until then, the DU version
of Netscape is considered by many to be the best solution to the
problem "How can I spend more time in front of my ALpha?"
