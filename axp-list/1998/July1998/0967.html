Hi,

There are several files that actually need to be editied.
(/etc/hosts.allow, /etc/hosts.deny...couple of others.)
Then you will need to write a "chat script" these files will communicate
with your ISP when you log in. There are too many different files to
write individually here. One link to look at for a guide is:

Http://www-callug.cs.berkeley.edu/ppp/primer.html

If using RedHat 5.0 it should be a snap in the control panel if that is
installed.

Other distributions vary, without knowing your system it's hard to tell
you what files need edited or created.

Hope this link helps
