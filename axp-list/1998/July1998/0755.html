Below is a message about EGCS performance - however I haven't sent
it to the EGCS list yet because I have an unexplained performance
descrepency.
In particular I can not recreate the fact that 19980627 had a better
Dhrystone score; I was trying to do this by passing -V .... to the
compile flags; i.e. using the new driver but the old cc1 etc.

Can anyone explain why this might be - I keep reasonably detailed
records of the speeds of each version, and am worried why I can't recreate
the previous one.

Diffind the assembly output of the two versions for Dhrystone
doesn't seem to show many differences.

I'm totally mystified about this; why should it be slower today than
it was last week, same kernel, same libraries and as far as I can tell
the same compiler (with the -V flag that is).

Dave

Hi,
Every time I ftp a new version of EGCS I run a set of 4 benchmarks shown
below. Included in the table below is the current version of EGCS I am running
with the results it gives and the best performance that have been achieved
and the version of EGCS on which they occurred.

At the moment the performance of Dhrystone seems to be dropping a lot,
Mesa is going down gently - but Povray suddenly got a lot better in the previous
version with a slight slip this week.

Dave (dg@treblig.org)

11/7/98

EGCS Current version: egcs-2.91.47 19980707

Benchmark Current Best - on egcs version
Dhrystone 885K/sec 961K 971225
Mesa Trispd 191K tri/s 202K 19980129
Povray drums 240 sec 236sec 19980627
Povray fish 89 sec 88sec 19980627

Descriptions:
Dhrystone: V2.1 without register attribute compiled with -m6 -DTIME
(A very small, simple, artificial benchmark, integer only)

Mesa Trispd: Mesa-2.5 compiled with -O6 -mcpu=21164a -DSHM
running the gdemos/trispd program on an 8 bit X server (Matrox)
(3-D graphics system, trispd just renders same size triangles
as fast as possible).

Povray : Povray3 compiled with -O6 -mieee
Ray tracing program.

povray drums: povray/povscn/level3/drums2 run with the flags:
+Oout.tga +W640 +H480 +V -D +X +L../../../include/ +D1
A drum kit with heavy lighting effects

povray fish: povray/povscn/level3/fish13 run with the flags:
+Oout.tga +W640 +H480 +V -D +X +L../../../include/ +D1
A fish

System
------
DEC Alpha (EV56) running on an LX164 (2MB third level cache)
running at 533MHz.
Linux (currently 2.1.98); installation from RH5.0.
