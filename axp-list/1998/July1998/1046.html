This one drove me crazy also.

You probably don't have the link for localtime setup.

Check this link and if not setup make sure it exists

lrwxrwxrwx 1 root root 32 Jul 28 23:19 /etc/localtime ->
../usr/share/zoneinfo/US/Central

Next make sure this file exists.

/etc/sysconfig/clock this file has one line
CLOCKMODE="ARC"

That should get you back to normal.

to set your clock from a time server you can use ->
rdate -s time_server ; clock -A -w

That should get you back to the 20th century. :-))
