Dear Onsale As233 owners =],

  If you are wonder what to do as far as redhat for this machines here it goes.

1) need arc installed

2) grab the test milo avanti image from digital.

ftp.digital.com/pub/DEC/Linux-Alpha/Kernels/Test-Milos-2.0.30+/milo-avanti-980606


copy that to a fat disk as avanti and then linload.exe also

next goto the arc console and make a new boot item, set the osloader to \linload.exe and then the boot image to \avanti

next set the boot options to boot fd0:xl.gz root=/dev/fd0 prompt_ramdisk=1 load_ramdisk=1

3) snag the redhat xl.gz image and just copy it to a fat disk

4) snag the ramdisk.img and rawrite that to a disk.

5) boot the redhat, as soon as milo is done loading insert your disk with xl.gz on it....

6) after xl.gz goes though it should prompt you for a root floppy, pop that sucker in and boom the menu should come up

NOTICE how i said should =]


P.S. Thanks to everyone who pushed me thought it =]
