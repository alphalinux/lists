I recently moved from Redhat 5.0 on my AlphaStation 200 to DEC Unix 4.0D,
solely for the purpose of having a stable machine to run Netscape on. However,
DU is less to my liking - RAM hog, mediocre video card support, etc.

How stable is Netscape for OSF when running under DEC UNIX compatibility? I've
(legally) got the libs and everything (since DU4 is installed and I'm running
it right now), so I can copy the libs to a 'safe place' and then whack this
install and put 5.1 on.
