>
> I need some help with XServer installation on a AlphaStation500.
> The grahpics device is a DEC TGA2 (this is what /proc/pci reports).
>
> I think, the XF86_TGA (3.3.2 resp. 980706) should support it, but I'm not
> sure. After using Xconfigurator (RH51), the XServer won't start. It fails
> with a message: "no configured devices found"
>
> So: Is the server unable to drive the card or is there some error in my
> XF86Config? If you suspect the latter: Can you provide me with a sample
> XF86Config for the TGA2?

The 3.3.2 release XF86_TGA only supports 8-plane original TGA; the version
made available by Tim Rowley, based on 3.3.2, also supports the 24-plane
orignal TGA.

TGA2 is NOT supported at this time by any XFree86 server, and it is NOT
compatible with the original TGA.

I'm working on getting the TGA2 programming docs made available, if not
publicly, at least with a reasonable NDA, to Tim, so that he can do the same
for TGA2, and contribute the code to XFree86. Cross your fingers...
