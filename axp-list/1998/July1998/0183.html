>>Can SRM be used to replace AlphaBios/ARC? I thought there had to
>>be more flash memory to do that which means only certain systems
>>can have SRM on them. If it fits, can one swap between them some
>>way?

OK, it depends..... IF your talking about the Digital Personal WorkStation
433a,500a,600a or a OEM motherboard.

The DPW433a(500,600-etc) from Digital can switch between SRM UNIX, SRM VMS
and AlphaBios (from mid 97). If your are in AlphaBios, Hit F2 during the
powerup init, and get into setup. select change cmos settings, select
Advanced settings. You will see the console selection about 2/3d's of the
way down the screen in a pull down menu. Select the console you want then
save the settings. Power cycle the machine and it will start the console you
selected. To switch back do a "set os_type NT" then "init" and power-cycle.

Most (Not all) systems from Digital switch between AlphaBios/Arc and SRM in
this manner.
