> The question is: What would it take to make this run on the AS200? I
> know very little about the differences between Alpha architectures, other
> than those of the 3000 series and (learning) AS200... Are they completely
> different, or just relatively minor things?
I would suspect that they work similar but for sure something is different.
The LEDs on the AXPpci33 are connected to the memory controller and toggled
when data is written to two special addresses (one address for data and one
for something like enable). The documentation was wrong so Thomas
Bogendoerfer did disassemble the SROM to find the correct addresses.
If the LEDs in the AS200 work in the same way just with different addresses
this should be no problem at all. I have file 'hardware.h' in the module's
packages containing the address definition. This would have to be changed.
