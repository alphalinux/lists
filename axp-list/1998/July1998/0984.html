This is is just a guess, but

The egcs compiler distributed with RH-5.1 had the Haifa scheduler enabled.
There are some problems that have been found since that have the symptoms
(on alpha) of "virtual memory exhausted" (which you might be seeing as
"machine just locks").

Installing egcs w/o the Haifa scheduler, or disabling any optimization on
files that trigger the problem are possible work arounds.

I don't know of any problems specifically with ssh on alpha.
