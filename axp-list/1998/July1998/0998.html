you should be able to type

set os_type=NY <enter>

when you re-boot should be in arc with a more intuitive menu selection.
i had lots of trouble trying to install from srm and never quite
succeeded, but arc was pretty easy; follow the manual. somehwere in the
arc menus you can set it back to srm/vms. i can also set mine (as400
-4/233) to a dec unix.

before you exit srrm, type

show config<enter>

here you'll see lots of nifty info on srm settings. a little extra
knowledge can't hurt.
