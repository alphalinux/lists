>>> Michel LESPINASSE said:
>
> Well, Thinking again about this, I think that this is probably that my
> video card is not supported by MILO.... I still use the VGA card supplied
> by digital and I have no clue what its name might be :) in Display System
> Configuration / PCI Configuration, the VGA device is reported as beeing
> Device Name : VenID=10BA, DevID=301
> Device Type : VGA
> Revision : 3
> Physical Slot : 4

That VendorID indicates a Mitsubuishi card? Never heard of one...

And yes, that could be the problem; at that point in the boot sequence,
MILO is running a BIOS emulator on the VGA's BIOS ROM to initialize it.
If it has problems, which it obviously is, that's a typical failure
indication, too.

Try something like a Diamond or #9 S3-based card, ot ATI Mach64, or Matrox
Millennium or Mystique.
