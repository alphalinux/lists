Mainly untested code for the 8-plane TGA (UDB) people this time:

* Slight change of ramdac mode for both 8- and 24-plane TGA

* Added sync_on_green option for 8- and 24-plane TGA (untested)

* Changed 8-plane code to perform card state save/restore (untested)

* Added 16- and 32-bit modes for the 8-plane TGA (untested)

Unsurprisingly, I would like feedback about the success of these
additions.

Binary and patch available as:

http://www.cs.brown.edu/~tor/xfree86/XF86_TGA-980705.gz
http://www.cs.brown.edu/~tor/xfree86/tga-patch-980705.gz
