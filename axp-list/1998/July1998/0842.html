Don't install the latest ncurses update from the RedHat 5.0 update site. It
brings back the "size" bug that earlier updates of this package fixed. Now my
mutt window thinks that it has room for showing my entire mbox (50+ messages
on a typical morning) instead of just the 24 lines that are really visible.

The only "work around" seems to be to type "eval `resize`" before starting
programs like mutt and ncftp.
