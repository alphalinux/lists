I haven't used the OSS drivers in quite some time, primarily
because they didn't work in some of the latest 2.1.x kernels. I'd
reccommend the ALSA drivers / utilities. They have OSS emulation, so
there's no need to go out and build a whole new set of programs.

Besides, ALSA has support for full duplex on SB cards, which OSS
does not. ('course, you need a card that does full duplex for that...
awe/64/Gold does, 64 Value does not) Check out http://alsa.jcu.cz/.

I have an AWE/64 Gold on a PC164sx, so I can guarantee that this
configuration works. ;-)
