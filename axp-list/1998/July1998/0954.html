This is not a good thing. Simply increasing the number of files is NOT a
good idea. This sounds like it could be some sort of real kernel problem.
More likely, some other progam is opening files and never closing
them... Do you have some app which is spawning itself over and over? THe
no free inode seems like the disk may be full... I'm not exactly
sure... first guess would be to make sure nothing is spawning over and
over again... You shouldn't just keep increasing the number of files you
can have open.
