>>> "B. James Phillippe" said:
>
> BTW, do you know of any way to change the solid block cursor to, say, a
> flashing block (or bar)?

The cursor for both the 8-plane and 24-plane TGAs is done using the hardware
cursor of the particular RAMDACs, and is limited to the capabilities of those
RAMDACs; blinking is NOT one of (either of) them, AFAIK... :-\

Also, this is why the cursor for the 8-plane TGA can be "seen through" (you
can see the character "behind" the cursor), and the 24-plane cursor cannot;
it's the RAMDAC capabilities/limitations (8-plane has Bt485, 24-plane Bt463).
