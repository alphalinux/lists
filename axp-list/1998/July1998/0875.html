On Fri, 24 Jul 1998, Eric Hilton Jones wrote:

> I was curious, in the past I have just created a directory for new
> versions of egcs to run from to keep the gcc installed with redhat and the
> new versions seperate. Has anyone had any problems upgrading the egcs
> compiler that comes with 5.1 to a newer version?

You may always install several versions of gcc in it's usual place
('/usr/lib/gcc-lib') and select any of them using gcc's '-V' option. Note
that gcc ('/usr/bin/gcc') is only a driver; it's version doesn't matter.
