On Mon, 6 Jul 1998, George wrote:

> So I did it. I wacked the passwd file and I cant get in.

Boot from a floppy like you were going to install/upgrade. Switch to
another terminal (alt-f2) and there should be a login prompt. Mount your
root filesystem manually (mount -t ext2 /dev/sda2 /mnt) and edit the file
/mnt/etc/passwd to create an entry that looks like this:

root::0:1:superuser:/:/bin/bash

Then unmount the filesystem (umount /mnt) and cycle the machine. You can
now log in as "root" with no password.

If you have shadow passwords installed, you'll have to create a similar
entry in /mnt/etc/shadow:

root::0::::::

You may need, if using /etc/shadow, to put an 'x' is the password field
entry in /etc/passwd. I'm not sure how Linux deals with shadow files.
