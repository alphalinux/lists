On Sat, 11 Jul 1998, Thunderhawk Alpha wrote:
> The point of this whole thread (and this mailing list) is that most of us
> want the same software availability on the ALPHA platform as the Intel one,
> RedHat's bundled software included....

And if you were thirsty, would you refuse the water I offer you simply
because you'd like to have a beer?

The fact of the matter is, more commercial software is available for
Linux/Intel. If your main concern in computing is commercial support,
than you will use Windows, absolutably.

If you can get away with less commercial support for applications, and
need a stable UNIX-like OS, but still need occasional office or word
processing type stuff, then you might run Linux/Intel with Wabi,
Executor and probably Applixware.

OTOH hand, if you need top-of-the-line performance, with a stable OS,
don't need commercial stuff, and need it all at low cost, then I'll
bet you're probably running an Alpha, with Linux.

> I'm glad the Los Alamos guys didn't take the advice above when they made
> the 'flea market supercomputer' composed of 68 DEC Alpha processors. See

This fits in the last category above. Avalon isn't gonna be used for
word processing, for crying out loud.. it's running PVM and specialized
parallel code.. if you think these guys cared about whether or not
Applixware runs on it natively, you're mistaken.

I'd love to see more support for the Alpha from Linux vendors, hell..
I'd like to see lost of support for Linux on any platform, but that just
isn't reality.. and likely won't be the case for a long time .. and,
may never be the case at all. Until then, we have to make reasonable
choices.. that might be Linux/Intel.. that might be Wintel..

I have a certain freedom that most people don't have when selecting
platforms.. I use computers for the sake of using them.. so long
as I can write C code for the machine, it's a possibility...

So long as I have a machine for CAD, everything else is basically
an expensive toy.

For those setting up machines and networks for corporations, selecting
Alpha's for the secretaries may not be best move.
