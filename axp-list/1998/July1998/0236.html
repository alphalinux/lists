Dumb question but have you installed the run-time QT libraries? (although this
does not look like a missing QT) There is a HOWTO in the same directory as
the RPMs which tells you what you need to install in addition to the KDE RPMs.

Try ldd kfm to see what it is missing.

Did you re-run ldconfig if you added libs?

Those Beta-4 RPMs are working fine for me although I did not use the QT RPMs
but compiled that myself.

BTW - you only need egcs if you want to compile the KDE stuff, you shouldn't
need it if you are installing the RPMs (but it is nice to have).

Let me know if you still have trouble.
