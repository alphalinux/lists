First:
Did you get the binary, or the patch?
If you go the binary, did you uncompress it (I believe it's in
.gz format).
If you got the patch, go get the binary, uncompress it, and put
it in place.
Then be sure to set the permissions:

cd /etc/X11/bin
chmod 755 XF86_TGA

and you're on your way.
