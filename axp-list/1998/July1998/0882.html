I've a set of patches to build a kernel that can be run on any supported
system, while retaining the option for building a kernel optimized for a
specific system.

I would like to see it tested on as many systems as possible, and as
quickly as possible. I'd like to see the patch go in for Linux 2.2.

Please see

ftp://ftp.cygnus.com/pub/home/rth/d-axp-gen-3a.gz
ftp://ftp.cygnus.com/pub/home/rth/genvmlinux-3.tar.gz

The first is a patch against 2.1.110 (I'll update it for .111 soon),
the second is a kernel binary containing all of the drivers that were
shipped with RH 5.0.

As a sanity check, I'd like everyone to try my kernel binary before
you play with the patches too much -- I want to verify that the one
binary really will run everywhere.

thanks,
