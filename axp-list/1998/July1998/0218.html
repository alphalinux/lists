I compiled the 3.0.1 distribution yesterday, and it pretty much works.
Note that the version checking is a little broken when it comes to alpha
kernels if you've applied patches to your source. For example, I had to
remove the definitions of ioremap and iounmap from k_compat.h, as the em86
patches define these in the standard headers, and twice defining them
will cause parse errors when compiling.

There were a few warnings in the fdomain and qlogic (related to the
definition of the Scsi_Host_Template stricture definition) code if I
remember correctly, but as I don't have those, I ignored them. Serial
module (modem) works fine, but I had to decrease the serial speed to
38400, and change the memory allocation in /etc/pcmcia/options -- before
doing that it would incorrectly identify the modem as a memory card.
