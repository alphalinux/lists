> Seems like I missed the beginning of this thread, so I
> have a few questions related to this GLINT X server.
>
> - How are XFree and this server related?

This server was based on internal code from the development base of
XFree86. There was a misunderstanding that caused this server to be
publicly announced which shouldn't have happened.

> - Is there a relation to the XFCom_3DLabs glint server
> from S.u.S.E (which is i386 only)?

The XFCom_3DLabs server from S.u.S.E. is based on the XFree86 sources
(actually, it is the other way round, S.u.S.E. started this development
and donated the code to XFree86, XFree86 developers helped to continue
this project).

The AXP version of XFCom_3DLabs is almost ready. 8bpp works, I am still
fighting higher color depths.

> - Is this beta version stable enough for everyday work?

The XFCom_3DLabs Intel server is rather stable, I would say. I know of
lots of people using it day to day.
