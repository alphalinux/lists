My machine is an AlphaPC 164LX running at 533 Mhz. The original installation
is Red Hat 5.0. I'm running Linux 2.0.32 . My SCSI card is an ITIpci SCSI
and Ethernet host adapter. I have two SCSI hard disks and a SCSI CD-ROM.
The machine was bought from Microway.

My machine is always running. I've noticed a "surprise bug" that seems to
happen about once every two weeks. It always happens in the morning after
the machine had been idling all night. Either the X-windows will crash the
system after initial Startx or the system will hang when I try to login.
To me it looks like a kernel "memory leak" but it might simply be a hardware
fault. Has anyone else been experiencing this?
