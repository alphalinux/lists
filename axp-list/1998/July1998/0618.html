> > OSLOADOPTIONS boot fd0:vmlinux.gz
>
> Hmm.. Don't you have to set more options, like root=[some device] and
> stuff, or is this new for the newer linux versions?
> I don't have my alpha in front of me, but I know I have
>
> boot sda3:vmlinux.gz root=/dev/sda3
>
> for my ordinary hd-based system, but I don't remember how I set it up for
> the floppy.

boot fd0:vmlinux.gz root=/dev/fd0 boot_prompt=1 load_ramdisk=1

This will allow booting off of the floppy, when it is time to mount root,
it prompts you for a new floppy, which gets loaded onto the ramdisk.

The kernel disk is the one for your motherboard, and one that gets put
in for root is ramdisk.img
