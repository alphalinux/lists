I have an HP DAT running on DEC Alpha, kernel 2.0.33,
using Adaptec 2940UW SCSI host adapter. There are
occasionally some SCSI hangup problems (requiring a reboot),
but Doug Ledford is solving them. I'm using his 5.0.10 verion
of the aic7xxx driver for the 2.0.33 kernel. The driver
is available at:

ftp.dialnet.net/pub/linux/aic7xxx
