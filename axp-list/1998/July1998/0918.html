I myself run a Alphastation 200 4/166 Avanti system with a TGA/2MB
(ZLXp-E1) with none of these messages.

I'm running a RHL5.1 as a base with kernel upgraded to 2.1.104. As a
general rule of thumb, the Alpha generally runs better on the 2.1.*
development kernels as all the bugfixes and Alpha-specfic support will
be implemented first.

Try the latest development kernel and you should be free of these
problems....
