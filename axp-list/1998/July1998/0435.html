> got the kernel from redhat cd for alpha. I compile with no problems.

Did you used "make xconfig" or "make menuconfig" ?
This doesn't work correctly on Alpha.
Use "make config" only. ("make oldconfig" works too, if
you manually edit your .config after running "make config")

(BTW: Jay's Alpha-Patches is more up to date then the RH5.1 stuff.
Go for a clean 2.0.34 kernel and apply
/pub/DEC/Linux-Alpha/Kernels/alpha-patches-2.0.34-0.1.gz (ftp.digital.com))
