Wow!! I am impressed. This server seems to work very well on my FireGL100Pro.
I used to have Gavin's 3.9 version of the Server and it only worked without
acceleration.

The only benchmark I did until now, was with xquake timedemo demo[1-3]
and I got the following (approx.)

Ati Rage II +DVD: ~45fps
FireGL1000Pro with the old X-Server (unaccelerated): ~26fps
FireGL1000Pro with new XFCom Server (accel): ~63fps

I think that's a fine result.
BTW, this was tested on an LX164 motherboard with 128Megs RAM and
600MHz CPU Clock.
