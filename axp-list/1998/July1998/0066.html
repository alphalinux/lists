Hi Ken,

We will make you whole again! Here's the deal;

We removed the Axp CD's from the Intel set for a number of reasons; Size -
it was not going to fit onto 2 CD's and the Intel folk didn't want the Axp
stuff (tsk tsk ;-)

We watched the Red hat 5.1 Axp debates, trials and tribulations with mixed
interest and concern. We didn't want to commit to a large run of CD's that
could (would?) be rendered useless within days and we also wanted to add
value to the set as well.

Bottom line; we will be releasing a CD set shortly which should keep
everyone happy. Anyone who feels "incomplete" should receive a CD or two
from us at no extra charge. Watch http://www.lsl.com and this group for
more details.

We remain committed to Linux on the Axp!

Warm regards,
