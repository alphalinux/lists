On Tue, Jul 21, 1998 at 12:39:43PM -0500, Greg Mader wrote:
> Folks,
>
> Okay, newbee question. I am interested in running linux on
> an alpha, but I need to run oracle on it. I am aware of the port of oracle
> to linux now in the works, but I suspect that it will be an intel version.
> Can I run oracle for digital unix on a linux alpha, and how do I go about it?

Oracle: Oracle8 is ported to Linux (don't know about what hardware platforms
are supported) and will be available by the end of the year, acc. to a
german computer mag (c't). Threre will be a test version available via
Internet. Check Oracle's web pages ...

Linux is able to run DUX binaries. You need to have the DUX shared libraries
however, so you need a DUX licence. If you want to run DUX4 binaries, you
need a 2.1.x kernel.
You might run into problems, though. The DUX X-Server, e.g., is reported not
to run on Linux, because it needs some kernel functions not provided by the
Linux kernel. Most applications should run, as they don't use this sort of
low-level functions. There have been success reports for Netscape, MatLab,
Mathematica, IIRC.
