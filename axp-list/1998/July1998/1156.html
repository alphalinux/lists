The NT Posix "subsystem" (a hack iam shure like the rest of NT) is
almost impossible to "exploit". MS will not make any docs about it
publicly available. Just take a look at the NT Filesystem support in
Linux. The NT Filesystem certainly is a complex thing to implement, but
would be no 'big deal' if, yes *if*! they would give us some documentation
about it. The point is that NT is as closed as a piece of software can be.
Hence forgett about things like binary compatibillity and other stuff. It
will not work. Wine does not work, nor will it work in the future.
If it *would* work MS would change things to make it not work again.

weeks ago.
