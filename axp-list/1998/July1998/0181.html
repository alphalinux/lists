There has been much talk about the relative merits of UltraDMA IDE disks,
SCSI disks, and busmastering.

While there is no doubt that UDMA is a vast improvement over the older PIO
mode IDE interface, there is still much information coming to light.

The fact is that UDMA gobbles up system memory to achive its results, not
to mention CPU cycles. An interesting article on this can be found at:

http://www.tomshardware.com/busmaster.html

Happy benchmarking!
