Quoting Karrazzi, Zakk (Karrazzi@pbworld.com):
>
> How do you I configure my modem for SLIP/PPP.
> host server uses DHCP for IP and WINS.
>
> I have configured my loopback adapter using ifconfig lo, but I'm lost
> now.
> I know that I have to create some sort of script, but how???

Are you using RedHat? Simple network configurations are easy using the RedHat
program "netcfg". I found it easy to set up a normal PPP dial in, even with a
configuration that needed a couple of extra prompt/responses in the dial in,
and doing IP masq through the PPP port.

Unfortunately, as with most unix (and Windows) system administration tools,
once you want to do something slightly complicated the tool screws up
big-time. Fortunately, unlike Windows, you can always go back to the raw
command line tools to add the missing bits. netcfg fell down badly when I
wanted to add specific routes to specific hosts through the ppp connection
when it was up, and otherwise to route everything through my new cable modem.
Fortunately, this could be handled through doing "route add" commands in the
/etc/ppp/ip-up.local.
