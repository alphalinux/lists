On Fri, 10 Jul 1998, Mike Sullivan wrote:

> I would like to set up a web server at home. Can this be done
> using a cable modem. Do you actually get a valid internet address and if
> so how does this work as my understanding was that there are not enough
> to go around so you had to be special to get one.

Yes, you have to have a valid internet address to talk on the internet.

The cablemodem company COULD do address-translation and try to force
everyone to compress into a very few IP addresses. But since this breaks
a lot of things, ICQ and Quake among them, they'd get some very unhappy
customers. :) You will get a valid IP address, although there's no
guarantee that it will never change.
