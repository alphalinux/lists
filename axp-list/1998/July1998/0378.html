I removed the video card I was using since it only
worked in the ARC console and not the SRM console.
I put in the ZLXp-E2 card that came with the system
(AS200 4/233). I now have a problem that I can't
read the screen when MILO loads. After Linux starts
to boot the screen is readable. I'm using 1280x1024
with 60Hz, but changing to 640x480 doesn't help.

Another thing I've noticed is the console is very
slow with this card. Is there a reasonable card
I can use that is supported under Linux/XFree86,
OpenVMS/DECwindows, and NT? I've tried a Diamond
Stealth 64 Video 3240 (S3 968 chipset), but DEC-
windows doesn't support it. I've also tried an
ATI Mach64 GT, but that isn't supported in SRM.
The S3Virge cards I've tried (325 and 988) don't
work under DECwindows or NT.
