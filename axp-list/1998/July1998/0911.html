Get the binary:
ftp://ftp.azstarnet.com/pub/linux/axp/misc/alpha-ELF-xquake.gz

Toss the binaries contained therein in /usr/local/games (well,
anywhere, technically... but this works well for me).

Get the necessary pak files ... from a dos version, I guess.
mkdir /usr/local/games/id1
mv *.pak /usr/local/games/id1
(NOTE: id1 has to be in the same directory as the xquake binary)

Run the program:
cd /usr/local/games
./xquake (or ./xf86quake)

There's also a README included with the xquake binary that goes
into more detail, and I think describes a slightly different way to do it.
<shrug> This works for me. Note that the X server has to be running in
256-color (8-bit pseudocolor) mode for it to work right.
