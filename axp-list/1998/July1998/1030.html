Hello!

I set up a "FAQ-O-MATIC" here at NYU for alpha-linux today.

http://www.nyu.edu/cgi-bin/cgiwrap/jjv200/axp-faq

Or hit the link at: http://www.nyu.edu/~jjv200/

It is going to be a user maintained system, so it's accuracy and whether
or not it is up-to-date is up to you....

I think this type of system would be better and more thorough than one
where one person is in charge of publishing the FAQ. Here, if you learn
something, you can instantly let others learn about it.

Linux/PPC has used a system similar to this as their main FAQ, and it is
enormous and well maintained. I think the Alpha community can do at least
as well.

I am going to try and set up the various sections, but anyone interested
is welcome to try, as well. The system is automatic and is open to
everyone. To add a "question" (please, don't add questions without
answers), all you need to do is give your email address and password for
accountability.

If you are interested, it might be worthwhile moving info from the old faq
to this one.

Please send me any comments and questions.

Enjoy!
