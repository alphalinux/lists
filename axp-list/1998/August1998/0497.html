Ah, yes, the infamous page_alloc problem. We've narrowed it down to be a
bug in the Haifa scheduler in egcs (pretty vague, I know). I'm testing
the latest snapshot right now to see if it exhibits the same behaviour or
not. For now, recompile egcs with the --disable-haifa switch and it
should compile just fine.

Someone on the list has revised RPMs for this, fyi, but I'm not sure who.
