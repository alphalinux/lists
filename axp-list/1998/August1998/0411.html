Prob 1)
Is IPV4 forwarding enabled? in /etc/sysconfig/network, there should be a
line like FORWARD=1. When the system starts, you should see a line
"Enabling IPV4 Forwarding".
If that is set, you may be using a kernel with the masq/forwarding disabled.

Prob 2)
If I'm not mistaken, the TGA driver only supports 8bit color depth.
