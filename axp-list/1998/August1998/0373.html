one thing you can try, when you use the Y adapter it normally says where
to plug in keyboard and mouse. On alpha that will be reversed. So that
you plug keyboard in mouse spot and mouse in keyboard. If you get it
wrong you will NOT get any video. Also when you have different memory
you can use it but you have to interleave it. That is to say that you
put one memory in slot 1 and then another in slot 2 and the same type as
in slot 1 in slot 3 and so on. Another thing to watch for, if there is
one flash on your keyboard, but not 2 flashes then you might be in the
debug mode. What that means is that you will need a serial connection
to an outside terminal to boot your computer again.

Hope this helps
Marek
