Mark Dohojda wrote:
>
> We Carrera computers configured those systems and I can tell you how it
> is done.
>
> First you install as usual and then after everything is install you set
> up the system so that you can use the serial console (in the
> etc./inittab file) and then make sure that you have root access in the
> /etc/securetty file (ttyS0, ttyS1 etc..) after that you can boot the
> machine using the serial connection to a monitor (a debug monitor).
>
> You can boot the system without keyboard or mouse (although you can't go
> into the X without them) however in order to by pass the BIOS check you
> have to have cable plug into the serial port first.
