Did you compile it with -mieee flag? I would expect Mozilla (and Java)
to do all kind of dirty floating point tricks - like taking square
roots of really small numbers if not simply dividing by zero.
These are deadly on Alpha if a generated code is not prepared for them.
It is possible that you may need also some libraries compiled -mieee.
