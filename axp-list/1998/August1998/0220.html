On Mon, Aug 03, 1998 at 08:05:24PM -0600, brinkmann@nmsu.edu wrote:
> I've been experimenting with the "poll" system call using the appended
> test program. It works well on our Solaris machines, but fails with an
> "Operation not permitted" error on my Alpha running RH Linux 5.0, both
> using stdin and file pointers. The RH errata makes no mention of any
> problems. Can anyone be of assistance?

Well, I don't know anything about poll().
I remember though there being a discussion about select() and poll() system
calls in the kernel list. From what I remember, poll() is emulated by
select() in 2.0.xx kernels, which cannot be done properly. In 2.1.x this
changed: select() now is emulated by poll() which can be done properly.
So, IIRC, switching to 2.1.x kernels could help you. Better check the
linux-kernel archives yourself ...

Hope, I could give you a little assistance.
