As part of a project for my University's Open Day, I'm trying to set up a
number of QuickCams around the University along with a web page for the
unwashed masses to go ape over :-) Hopefully I'll also be able to set up a
demo linux system with a nice flat panel display to impress them even more.
I'd love to go with the big 42" plasma but it might just be a 20" LCD....

I've already set up my Intel Multia (NT) to do this using Webcam32 but I
thought for balance I should set up the Alpha Multia (Linux) to do the same
thing. I haven't got very far yet (this is a background project and OpenVMS
support/general management comes first). I downloaded what looks like
appropriate software from illtel.denver.co.us and although it compiles,
qcread doesn't see the camera. The error message is "qcread: Initialization
failed: Quickcam not detected or not supported type". I have changed the
config file to point to the right address for the lpt port and the kernel
has been built with LPT support (2.0.34).

Any pointers to getting this to work or pointers to other software that
will allow me to upgrade my test web page
(http://lucifer.its.latrobe.edu.au/~mark/webcam) would be appreciated.
