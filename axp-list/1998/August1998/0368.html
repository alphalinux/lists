Unfortunately, RH 5.1 didn't include the correct support for RPX164-2, aka
RUFFIAN.

The correct way to build a MILO floppy for it is to use the following
file from off gatekeeper.dec.com:

/pub/DEC/Linux-Alpha/Kernels/Test-Milos-2.0.30+/ruffian.img

Take that file and RAWRITE it to a floppy, or use:

dd if=ruffian.img of=/dev/fd0H1440 bs=10240

The floppy will then contain all the components needed to boot on a RUFFIAN.

Good luck.
