Nah I fixed mine just last night, same deal, and nobody here help they
just said SRM does it which I gathered...but I wanted milo to work,

Do an isacfg -all and note the settings of all your isa stuff (just in
case you loose it, the new ones not supposed to but who knows, mine was
ok but Murphy's Law is one to watch for).

Go get the file as200_v6_8.exe from
ftp://ftp.digital.com/pub/Digital/Alpha/firmware/v5.2/alpha200/
renamed as200__8.exe to FWUPDATE.EXE

I then got mkboot.exe from
ftp://ftp.digital.com/pub/Digital/Alpha/firmware/utilities/
Which is a msdos program

And I did (at the windows 95 dos only boot on my PC), should be a dos
screen cause mkboot.exe is a mickeydos only program (based on
rawrite.exe). There are linux instructions but I found them later, so I
used msdos to make it.

1. Formatted Floppy with MSDOS partition (to make sure there are no errors
did unconditional)
format a: /u

2. ran mkboot
mkboot c:\FWUPDATE.EXE a:

This made a bootable firmware update disk.

3. booted up to SRM console, popped in the new floppy,ran this
boot dva0

This booted the firmware update, read the info from the same ftp site in
order to have a clue, but its simple to figure out really.

My SRM firmware went from 6.1-2 to 6.8 and I added ARC firmware 4.5 (mine
had no ARC firmware (I got it from www.onsale.com last week).

4. Follow the ARC instructions in the redhat manual, it now works like its
supposed to.

5. Duh, boot linux and install it of course, if its installed just put
your milo on the small dos partition, set your parameters and enjoy.

If you need more help/details I got four more to do tonight (for someone
else), but one I get them done, I will be checking the email. I figure
this will help... down SRM, this is much more kewl.

Oh yea ignore the error message about Hey do NT partitions, its just a MS
ploy to piss off non-nt users, course if you also install NT it shuts up
:).

This was a 25 minute redhat install with about 6 hours of digging for a
clue...no one help me so I figure I will try to help you, hope it helped.
