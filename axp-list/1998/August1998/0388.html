Always get the latest stuff from Becker's site. Don't trust
that RH has the latest correct drivers. Doesn't hurt to try
theirs, but if it doesn't work, go to the source.

It works in a PC164 and a PC164LX for me. As always, YMMV.
Also works in my PII/PPros. I only use 2114x cards for
all my Linux boxes (6 misc). Did you try both the de4x5.c
and tulip.c?? I've used many versions of tulip.c and it
basically always worked except for one old card that was
non-MII. Had to make a minor mod to tulip.c to get it
working to auto-detect switch/hub speed.

I've also seen other brands at similar prices so the
FA310TX is not unique in terms of price.
