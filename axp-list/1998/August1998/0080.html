:) Keep the faith. NT really, really stinks. Any admin with any sort of
load on the hardware will be forced to use UNIX. NT just isn't reliable
enough. If the admin has no experience with UNIX, they may choose NT,
but when NT fails, they will need to search for something else. Any
sane admin with UNIX experience will stay away from NT.

So, I think we have the advantage.. but we still need to promote UNIX.
The main argument I always hear is that since UNIX is so old, it must
not be up to the task for newer technology type stuff. People are used
to MS software rolling over on itself every few years, they don't know
what to think. We need to keep telling them that if the software is
put together well the first time, you can expect it to stay around.

UNIX is older than I am, damnit! :)
