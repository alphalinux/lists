This directory contains the files necessary to install Windows NT on an
EB64+ or an AlphaPC 64 system.

To update the firmware on an AlphaPC 64, use the provided fwupdate.exe
by choosing "Install new firmware" from the firmware's Supplementary
Menu.

In order to update the firmware on an EB64+, you will need to obtain an
updated firmware ROM from your system vendor.

--------------
New features
--------------
4.49-7 - Disk partitioner properly aborts formatting of 4GB+ partitions
Several disk partitioner problems with logical partitions fixed
Removed latent NTFS support
4.49-6 - Disk partitioner problem with large drives fixed
4.49-5 - Serial mouse support added to firmware
4.48 - Disk Partitioner
Supplementary Menu...
\->Setup the System...
\->Machine Specific Setup...
\->Partition Disks...
The disk partitioner can create and format partitions on disks
connected to firmware supported disk controllers.

--------------
ATAPI CD-ROM
--------------
If you are installing from an ATAPI (IDE) CD-ROM Drive, when prompted for
your mass storage devices:
- choose 'S' to configure additional mass storage devices
- insert this floppy and choose 'Other'
- choose 'ATAPI CD-ROM'
Note that in the firmware, ATAPI CD-ROM Drives are listed as scsi devices.

------------------------
Supported boot devices
------------------------
With the firmware on this diskette, Windows NT can be booted from an IDE
disk or a disk connected to an Adaptec AHA-154x, Adaptec AHA-294x/394x,
NCR 8xx, or QLogic ISP1020 SCSI controller. Windows NT can be installed
from IDE, NCR 8xx, or QLogic ISP1020, but can not be installed from an
AHA-15x. Windows NT 4.00 can also be installed from an Adaptec AHA-294x
or AHA-394x.

Once Windows NT is running, any SCSI controller on the HCL can be used for
non-boot disks.

--------------------------------------------------------------------------
DC21040 Ethernet and upgrading from versions of Windows NT prior to 3.51
--------------------------------------------------------------------------
If you have a network adapter based on the DEC DC21040 chip, the
existing DC21040 driver is not automatically updated when upgrading from
versions of Windows NT prior to 3.51. The updated driver must be
installed manually after upgrading Windows NT. Follow these steps to
upgrade Windows NT to and update the network driver:

1. Insert the Microsoft Windows NT CD-ROM into your CD-ROM drive.

2. From the ARC Boot menu, select Supplementary menu
and press Enter. When the Supplementary menu appears, select Install
Windows NT from CD-ROM and press Enter. Windows NT Setup begins.

3. Follow the prompts to complete Windows NT Setup.

4. Double-click the Network icon in the Control panel.

5. If you are running TCP/IP protocol, double-click TCP/IP Protocol in the
Installed Network Software box. Write down all of your TCP/IP configuration
information. This information must be entered after the driver update.

6. In the Installed Adapter Cards list, select the card(s) for which the
driver is to be updated, then click Remove. Do NOT click OK to exit the
Network Settings dialog at this point.

7. Open a Command window, and type the following commands:

cd /d %windir%\system32

dir oemnad?.inf

ren oemnad?.inf oemnad?.inx

8. Return to the Network Settings dialog and click Add Adapter.

9. Select your card(s) from the list. Follow the prompts to install the
card.

10. If you have TCP/IP running, enter the settings you saved earlier.

11. Shutdown and restart your system.
