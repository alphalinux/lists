I've tried ssh 1.2.26 and 1.2.22 under RH 5.1 and I have the same trouble
with both of them when I try to access remote machines running Solaris
2.5.1 with ssh 1.2.20. Using telnet into the same systems works fine.
Also, ssh works okay when I access a remote machine running SunOS 4.1.4
and ssh 1.2.20. It also works okay when I access a PC on the local LAN
running RH 4.2 and ssh 1.2.26.

When I run RH 3.0.3 with ssh 1.2.17, 1.2.22 or 1.2.26, I have no problems
with slowness when accessing the Solaris 2.5.1 systems. Is anyone aware
of problems with ssh 1.2.20 and/or Solaris 2.5.1? Has anyone else
experienced this sort of problem?

The slowdown has happened every time I've tried accessing the Solaris
2.5.1 systems from RH 5.1, but never from RH 3.0.3.
