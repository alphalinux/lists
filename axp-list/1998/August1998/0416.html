MpegTV announces mtv-1.0.6.10. This release of the MpegTV
Player now works well on Linux Alpha glibc systems (RedHat 5.x).

This version uses a version of libforms.so.0.88 compiled for glibc.

Until now, libforms.so was not available for Linux Alpha
glibc systems, and the libc5 version was not working properly
on glibc systems.

The MpegTV Player for Linux Alpha (glibc) can be downloaded from:

ftp://ftp.mpegtv.com/pub/mpeg/mpegtv/player/alpha-unknown-linux-glibc/

Installation instructions:

http://www.mpetv.com/download.html

Make sure that you download and install the libforms.so.0.88
file that is provided with in the mpegtv download directory.
The RPM package available from the RedHat site does not work
well with glibc systems.

MpegTV Player is an MPEG Video and VCD Player (with audio).

MpegTV Player is a $10 Shareware for personal use.
