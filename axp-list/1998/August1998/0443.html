My attempts died at a unaligned memory access when the RedHat installation was
reading the ramdisk. Upping the memory 16->24 didn't help any, but as soon as
I put 64 in it it started working just fine. Perhaps you can tell MILO
(there's an option for MILO that's different from the kernel flag) how much
memory the machine has, since I believe it defaults to 32 megs.
