You might want to hook up a monitor/keyboard to install linux, but I don't
think they are needed for normal operation. If your computer(s) don't have
a video card, you could put a cheap PC video card in there. We're talking
$30-50 here.

Graphics are not necessary to run linux or administer it. The whole redhat
install is text based, and all administration and changes can be done text
based, though there are X-windows based things for novices and
windows-weenies.
