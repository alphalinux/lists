Missed the first part of the thread (sorry). My UDB has the following
address 0xa000000
this information can be found in the /proc/pci file look for a prefetch
address and it should
be in there. this is all in the README.DECtga file. After running
XConfigurator (make sure
you dont run the probe option skip it then select your 8 bit resolution). of
course will generate the XF86Config file. Close to the end under the
graphics section there should be an entry under there saying "Generated or
inserted by Xconfigurator".
I removed the comment line preceding the ram statement indicating 2 meg of
video.
I then added the following lines:

Option "dac_8_bit"
MemBase 0xa000000

Besure not to quote the numeric address. Then if you chose your monitor
correctly and
your monitor will support the mode lines generated, it should start just
fine. Sorry again
for elemental instructions, but missed the thread.
