>>> Dave Wild said:
>
> I have access to an Alphaserver 2000 4/200. I would love to get Linux
> running on it. Has anybody tried this? Any clues on which machine type
> (EB164,PC164,etc) to try?

AlphaServer 2000, aka AS2000, aka Demi-Sable, should be able to use a SABLE
kernel, of which there is one on Red Hat 4.2 (not 5.0, but you could use the
kernel from 4.2 to install 5.0... :-). Look for images/sable-s.img. Boot it
from SRM console following the guidelines in the FAQ at:

www.azstarnet.com/~axplinux/

Note that current Linux/Alpha code does NOT support SMP.

Good luck.
