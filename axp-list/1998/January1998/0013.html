I'm sorry, I cannot provide any information on your particular compile
problem, but I'm rather familiar with PostgreSQL on Linux/intel and on

Generally, I'm afraid you'll run into much bigger problems with PostgreSQL
on Linux/Alpha. From what I've seen from the PostgreSQL developer mailing
lists, die PostgreSQL Hackers seem to be rather sure that PostgreSQL is
not 64bit clean right now. (There is no 64bit port of PostgreSQL currently
existing).

But I would really _love_ to be proven wrong by someone telling me that he
as a working PostgreSQL server running on Linux/Alpha... ;-)
