Jurriaan Saathof wrote:
>
> ...Linux/alpha?? I already downloaded the OSF/1 libraries, but they were
> version 2.0. When I run netscape there are some libs it can't find. Does
> anyone know where to get these libraries.

Strictly speaking, you have to have a DEC Unix license
to use the libraries and you must use them on the
specific machine for which you have a DEC Unix license.
You probably need libraries from v 3.2, but
it may be that you just don't have all the libraries:
/usr/lib/nls
/usr/ccs
/usr/lib/cmplrs
