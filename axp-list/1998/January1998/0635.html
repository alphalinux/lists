Stephen Wolff wrote:

> Hi,
>
> I am curious to know whether anybody has
> managed to install and run any quantum chemical
> packages under Alpha/Linux?
>
> In particular: Gaussian, ADF?

It is possible to run DEC Unix Gaussian executables
on an Alpha/Linux system. This is not,
strictly speaking, legal, since you must
dynamically load several DEC libraries,
which one is permitted to use only on the DEC Unix
machine for which one has a license.

It is also necessary to "make" Gaussian
on your DEC Unix machine using the public
domain linear algebra General Matrix Multiply
subroutine. DEC's DXML library is very clever.
It looks (I think) at the kernel of the machine it
is to be loaded to for license information, and
won't run under Linux. DEC's GEMM is
about four times as fast as the public domain
version. Surprizingly, the performance is still
quite good. A profile of the system shows that
GEMM is important but not so much as to
make it uncompetitive with other alternatives
at the same price. Brute force wins here.

We would like to get DEC to sell versions of
fortran, math libraries, DXML, ported to Linux,
and Jon Hall has been working on that, but...
