hmmm,

i don't know how to compile it into the kernel... but what i did was set
the keyboard rate upon start-up in my rc.local file with the command:

kbdrate -r 30

this will set it as fast as it'll go. there is a 'delay' option to kbdrate
also, but if you don't speciy one, it is automatically set to 250ms (the
lowest).

works for me! :)
