Hello everybody,

Our calculation to solve ECC2-89 is well under way.

The idea is to calculate a "discrete logarithm" in the group of points
on an elliptic curve. There are 2^88+12377169451261 points which
means that we need to do about 26 trillion iterations of a
pseudo-random function until we find the same point in two different
ways, and then we can compute the final answer.

If you would like to soak up some spare Alpha cycles on this project,
you can head over to:

http://pauillac.inria.fr/~harley/ecdl3/

You just need to download the source, compile it and run on Linux,
Digital Unix or NetBSD. There are ReadMe and ReadMe.html files with
full details. There is also an up-to-date counter and list of
contributors. Currently 31 people have done one trillion iterations.

See you there soon!
