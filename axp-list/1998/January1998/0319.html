Joakim Roubert <jokke@df.lth.se> writes:

> I have a 2.1.67 and a 2.0.30 working. I usually run 2.0.30. The thing is,
> key repeat (what is it called?) is so _slow_ in 2.0.30, but nice and fast
> in 2.1.67. So I'd like to compile a 2.0-kernel with a faster one.

man kbdrate:

KBDRATE(8) Linux Programmer's Manual KBDRATE(8)

NAME
kbdrate - reset the keyboard repeat rate and delay time

SYNOPSIS
kbdrate [ -s ] [ -r rate ] [ -d delay ]

DESCRIPTION
kbdrate is used to change the IBM keyboard repeat rate and
delay time. The delay is the amount of time that a key
must be depressed before it will start to repeat.

It works under linux/alpha too.
