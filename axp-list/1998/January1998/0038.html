I've just finished setting up a PWS 433a with RedHat 5.0.
This is a fresh install - Linux was never installed before,
and the OEM NT was never even set up. I removed NT :) and set
the workstation up for Linux only. I used the miata milo and
kernels from the CD.

For the most part everything works O.K, (considering that RedHat 5.0
broke xview and em86) except for 2 things.

A. This workstation has an Adaptec 2940 Wide SCSI adaptor. Disk
access seems slow. For example, fsck on a 500M partition takes an
uncomfortably long period of time (10 minutes +) and periodically
an "ls" in a directory with about 10-15 files is noticeably long.
Also, emacs takes about a year to load, which I am assuming is a
related problem. Is their a problem with this SCSI driver? Is there
something that can be tuned?

B. Glint does not work. Python complains about "resource
temporarily unavailable". I see a simialr complaint in the
list archive back in May. Has this been fixed?
