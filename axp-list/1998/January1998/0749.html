>
>
> This reminds me of something, how on earth can there be a SCSI-BIOS when
> running on an Alpha? Is there a CPU in the Adaptec SCSI controller, or
> does it contain a ROM with Alpha machine-code or what?

I *think* that those messages just indicate that the BIOS code on the
Adaptec card has been run through the x86 emulator that is built into
the consoles (ARC/AlphaBIOS/SRM).

This is the way VGA BIOSes have been run for a while; it's only recently,
IIRC, that the SCSI BIOSes have joined them...
