---Stephen Williams <steve@icarus.icarus.com> wrote:
>
> Has anybody tried this? I'm thinking of giving the recent development
> kernels a try on my alpha and was wondering if others have any
experience.
> (Before I go downloading and compiling.)
>
We had an alpha running Red Hat 4.2 on which there were a lot of
arithmetic traps and exceptions coming from both system software
and software we wrote. The problems didn't go away until someone
on this list recommended the 2.1.x kernels. I tried the 2.1.72
kernel, but then discovered the software that came with 4.2 were
too old for 2.1.72. I upgraded the alpha to Red Hat 5.0 last
Wednesday and so far the only problem was I needed to reverse
the order nfs and portmap are loaded in /etc/rc.d/rc3.d.
Red Hat 5.0 and the 2.1.72 kernel appears to have solved the
traps and exceptions problems we were having.
