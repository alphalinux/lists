>>> Benjamin Lagunilla said:
>
> I am trying to install redhat 5.0. unfortunately, there seems to
> be no kernel image for mikasa. What other kernel image would suffice?
> Or do I have to build it myself? How?

Assuming you're booting from SRM console (there's no recommended MILO for
MIKASA at the moment), grab the "mikasa-s.img" from the Red Hat 4.2 archives
on gatekeeper.dec.com:

/pub/linux/redhat/redhat-4.2/images

This should work, so long as your MIKASA has an EV4 CPU daughtercard, which
is the case if the model designation is 4/??? (that's what the "4" stands
for). If it's an EV5 based box (ie 5/???, aka MIKASA Pinnacle or Primo),
let me know, I'll build one for you...
