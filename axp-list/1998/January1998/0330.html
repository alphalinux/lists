Alessandro Mirone <alex@greco2.polytechnique.fr> wrote:
>
> I get "floating exceptions" (and core dump) from some programs
> that gave NaN when compiled on a PC-Linux.
> Could someone give me somt tips about how
> to deal with it? a solution of course would be checking the
> values before doing mathematical operations.
> But apart that could I use some compilation switch to get NaN
> and avoid crashing the programs?
> Any help is appreciated
> bye
You must compile with ieee. The Alpha chips generate an exception on several conditions such as NaN, Infinities, denormals. These must be handled in an exception handler. Use the -mieee when compiling with GCC.
