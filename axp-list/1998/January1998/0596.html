I built and installed egcs into /usr/local and then created links
/usr/local/bin/gcc -> /usr/bin/ecc and /usr/local/bin/g++ ->
/usr/bin/ECC for the egcs versions of gcc and g++ respectively. This
way I have both versions installed just in case and they both can run
unhindered. There are some path things you will need to change like
includes, but nothing too serious.
