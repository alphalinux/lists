On Mon, 12 Jan 1998, Michael Fonte wrote:

>
>
> Okay I just came from the Linux/Alpha FAQ and followed the directions
> for getting DEC Unix Netscape to run under Linux/Alpha.
>
> Netscape is running fine but it won't resolve and domain names.... The
> DNS works fine under xterm's, etc. What else do I need to do?
> We are running RH 5.0, kernel 2.0.30, and using the OSF/1 libs,
> netscape-v304-export.alpha-dec-osf2.0 (not 3.0.1 which was listed in
> the FAQ but I could not find this version on netscape's server)

In the file /etc/svc.conf, the line that begins with hosts= needs to be
hosts=bind (Possibliy ,local after that, but bind's the bit that you
need...)
