On Wed, 31 Dec 1997, Adam Fenn wrote:

> Has anyone got Postgres SQL working on the alpha linux? I am running
> stock RedHat 5.0 and I get the following errors. Any help is
> appreciated.

I am sorry to report that Pgsql does not as yet run on
Linux/Alpha. The port to this platform is still in progress, but head way
is being made, and it should not be too long until it is running smoothly
on Linux/Alpha. The error you experienced has been fixed in the CVS
tree for Pgsql, but it will probably not appear in a formal release for a
while (since no formal releases are planned for at least a short while, as
far as I know). The current problem being hit is in the initialization of
the databases after the build of the program. It is a rather complex
problem even I don't understand, so I will leave the description at that.
In the end, have hope, as there will soon be a version of pgsql
that will run on Linux/Alpha! And when that happens, we will make sure
that the Linux/Alpha world knows! :)
