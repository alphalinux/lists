'-lht' means that it wants to link with a library named 'libht.so'
or 'libht.a' A similar deal for this '-lrx'.

The complete line is :
>
> c++ -o htnotify -L../htlib -L../htcommon -L../gbdm-1.7.3 -L../rx-1.5/rx
> htnotify.o -lcommon -lht -lrx

It is pretty obvious that these libraries are expected in ../htlib
and in ../rx-1.5/rx directories. Either Makefile has problems or you
did some "shortcuts" and libraries were not compiled.
