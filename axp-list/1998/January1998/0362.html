> docs; the original poster didn't.
>
> If you have a "virgin" NT installation and you want to keep it the
> simplest method is to reinstall it from CD an newly created partitions.
> It is not very trying.
> - make NT partitions with NT fdisk
> - boot Linux and make remaining partitions with Linux fdisk;
> do not follow with the rest of installation - reboot
> - install NT
> - install Linux
>
> > I have experience installing Linux on x86
> > platform,
>
> Beyond initial steps to boot your machine, and an absence of something
