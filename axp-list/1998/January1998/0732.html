> Would it even be possible to do something like this? IE: by running our
> imaginary program as follows:
>
> ix86toalpha binary-ix86-src binary-alpha-dest
There is a group doing this sort of thing for Windows->OS/2 translation.
The last report i saw said that they could completely translate simple
programs like minesweeper and solitare. I don't have the address at hand,
but I'll get it out of my bookmarks when I get home. Of course, this
doesn't involve a change of architecture, but it couldn't hurt to see what
they're doing. On the whole, this sort of project certainly sounds
doable. The only problems I see are in picking out the library calls and
kernel calls from straight assembly. This is already done by em86, so
maybe a hack of em86 from "interpreter" to "compiler" is in order?
