I have a recipe for installing em86 ...rpm packages
on a RedHat5.0 for Alpha.
Now, I'm able to run SOME program but not ALL
(i.e. communicator4.04 gives me memory violation)

That's how I did it:

1) The packages I have installed successfully are the following:
em86-0.2-3.alpha.rpm
em86-baselibs-0.2-1.alpha.rpm
em86-x86-0.2-1.i386.rpm
em86-X11libs-0.2-1.alpha.rpm

2) First patch the kernel, compile and install (as usual)

3) make as 'root' the following symlinks BEFORE step 4)!!!!
ln -s /lib/ld-2.0.5.so /lib/ld.so.1
ln -s /lib/libc.so.6.1 /lib/libc.so.6
ln -s /lib/libm.so.6.1 /lib/libm.so.6
ln -s /lib/libdl.so.2.1 /lib/libdl.so.2

4) Unpack rpm packages with:
rpm -ivh --nodeps em86-0.2-1.alpha.rpm
(--nodeps is necessary because RedHat5.0 does NOT install
the files libc.so.6 and libm.so.6 that is why we made
symlinks at step 3))
rpm -ivh --ignorearch em86-x86-0.2-1.i386.rpm
rpm -ivh em86-baselibs-0.2-1.alpha.rpm
rpm -ivh em86-X11libs-0.2-1.alpha.rpm

an the other libs you need.

Hope will be useful to somebody.
