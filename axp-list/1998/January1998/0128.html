Vladimir Kamenov wrote:
>
> Hi all,
>
> I want to buy an Alpha computer and to use it for numerical calculations.
> The reports for the slow C/C++ compilers under Linux confuse me. As I know
> gcc gives very slow performance. What is the situation with the new egcs?

I don't know about numerical work, but for plain integer stuff I get between
10-20% better performance from the 971225 EGCS snapshot than I do from
the GCC 2.7.2.3 with RedHat.

That is on a 21164a on an LX board with EGCS configured to optimise for the
21164a.

What I really need to do is to recompile glibc with the new compiler to
optimise things like printf and all the file I/O code - but I don't
fancy ftp'ing 13MB of source over my poor 33K6 modem.
