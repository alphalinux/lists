Why? This is, obviously, a user space function and kernel only provides
hooks to implement it. If you want to see if you really have it you
should try something like that:

$ nm /lib/libc.so.6 | grep ieee
00000000000878e0 T __ieee_get_fp_control
0000000000087920 T __ieee_set_fp_control
00000000000878e0 W ieee_get_fp_control
0000000000087920 W ieee_set_fp_control

So it is there. Beyond that it is trivial to write a two line C program
which uses the function and try to see if it links. Faster and more
reliable than bothering the whole world. And apart of that I wrote
explicitely that the function is available and even provided, guessed,
prototype for it.
