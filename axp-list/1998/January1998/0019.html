>

They are reported to be slightly delayed due to a minor chipset glitch.
You can see info on them at:
http://www.sec.samsung.com/Products/alpha/mb164ux.html

BTW, perfoprmance-wise they should be extremely similar to the LX, as they
use the same chipset. One advantage is that they are being factory built
with optional 4MB cache. This would give them up to a 10% performance
advantage. There ARE some 4MB LX boards, but these are all retrofitted by
vendors, and as it is a board modification voids the original warranty.
