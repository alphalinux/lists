Has anyone successfully compiled a kernel with ftape support? I have
tried but ran into problems with LATCH and CLOCK_TICK_RATE.
/usr/src/linux/drivers/char/ftape/calibr.c needs these to be defined, but
/usr/include/linux/timex.h only defines these #ifndef alpha. timex.h
specifically states that the calculations to derive these constants are
invalid for the alpha. It also states that this is o.k. because none of
the kernel source uses it anyway! Since these two constants are used for
timing in the ftape driver, can anyone suggest how I might derive suitable
replacement values?
