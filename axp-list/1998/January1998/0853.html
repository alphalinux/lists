tommy,

very quick answer:

- Compile your soundcard as a module, configure it with isapnp
(man pnpdump, isapnp),
then load the module, viola (you will *need* isapnmp cause the
alpha-bios has no such thing as 'pluck and brai')

- XFree86 has a SVGA server that *does* run on (at least my) alpha (maybe
RH did not install it ?? (look at the RH ftp site in the RPMS dir) )

- Your linux will never ever run the Intel Eexpress without problems,
if you need a cheap card, buy a NE2000 clone, Eexpress is for use with
MSwindows-products only.

- i run 2.0.33 without problems.
