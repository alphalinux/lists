>>> Christoph Wegener said:
>
> I've just tried to install RedHat Linux on a Alpha600. I probed both -
> the alcor and alcor-s BOOT iamge with the following command line at the
> console prompt:
>
> boot dva0 -file vmlinux.gz -flags "boot_ramdisk=1 prompt_ramdisk=1".

It appears you're booting from SRM console, so you'll want the alcor-s image.

> The crux is, that the machine does NOT ask me for the ramdisk, it tries
> to mount the hard disk root-partition all the time and there for I get a
> kernel panic.
>
> Did I use a wrong command line?!?

Make the flags argument "load_ramdisk=1 root=/dev/fd0" and it should work.
