If it runs on RedHat Alpha, is it not likely that it will also run on
Red Hat Intel?

It is required to use C, and be otherwise pretty portable, according
to the set of required specifications.

There is nothing that requires that the resulting programs *only* run
on Intel; it is only required that they *DO* run on Intel...

Go ahead, write it on the Alpha box. And if it doesn't work on a
P100, then show off the code, and we'll try to see why it's not
portable.

I would presume that the results would be turned into a nice SRPM
file which could be compiled using Red Hat 5.0 Alpha...
