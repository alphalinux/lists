in linux-include/netinet/tcp.h, tcp_seq ( SEQUENCE NUMBER ) should
always be 32 bits

defining it as a u_long on an linux/alpha makes it into a 64 bit entity
- which is not what u want in a tcp header definition.

ALSO
in ip_icmp.h, the n_time value should also be 32 bits long.
