On Thu, 15 Jan 1998, Huff, J. Taylor wrote:

> My problem is that when I put my MILO disk in, boot the machine, and
> select boot Linux from the boot menu. I don't get the milo> prompt. It
> just locks up on me. "Output from monitor displayed below". I've tried
> sever different Milo images, and even downloaded the "AlphaStation 200"
> milo image from ftp://gateway.dec.com but I am still having problems.

So you have the Avanti boot disks. Ok. Then:

> LOADIDENTIFIER=Linux
> SYSTEMPARTITION=multi(0)disk(0)fdisk(0)
> OSLOADER=multi(0)disk(0)fdisk(0)\linload.exe
> OSLOADPARTITION=multi(0)disk(0)fdisk(0)
> OSLOADFILENAME=\milo
> OSLOADOPTIONS=

I don't have OSLOADFILENAME=\milo.
I don't think I have anything there. The only thing you have to do when
creating a new one is to set the name, \linload.exe (like you did), and to
set the load directory to \, and then not to set any OSLOADOPTIONS (like
you did).

It works on mine.
