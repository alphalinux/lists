On Thu, 21 May 1998, Bruce, Stephen wrote:

> Is there an equivalent of MS pptp VPN software for
> Linux? If so is it available for the alpha RH5?

Equivalent? Not really, however the new kernels do support
(experimentally) GRE tunneling (the MS/Cisco PPTP deal). If
you want some more secure, read the Linux-VPN howto. It's
pretty straightforward -- open a PPP session to the remote
host and connect with SSH.

http://sunsite.unc.edu/pub/Linux/docs/HOWTO/mini/VPN
