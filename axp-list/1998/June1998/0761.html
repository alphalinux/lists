I installed milo in a small partition (FAT 1440 k) and I want to boot
linux (RH 5.0) from first partition (root partition) of the first SCSI
with:

boot sda1:vmlinux.gz root=/dev/sda1

or,

boot -t ext2 sda1:vmlinux.gz root=/dev/sda1

linux can't boot

With:

boot fd0:avanti.gz root=/dev/sda1, linux is booting from a FAT floppy
disk.

I think that my milo version can't recognize SCSI.
