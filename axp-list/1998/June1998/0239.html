One of the more common causes of this that I've seen is that you put
support for the ext2 filesystem as a module. This doesn't work. It needs
to load that filesystem so that it can get at the module definition
(kind of a chicken & egg problem).
Recompile it with the ext2 support compiled in, and put the rest
of them as modules, and you should be fine.
