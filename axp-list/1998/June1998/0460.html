On Tue, 12 May 1998 Jay.Estabrook@digital.com wrote:

> > Does anybody have any negative experiences with
> > an ASUS SC875 SCSI controller (SYMBIOS Logic
> > 53C875) on an Alpha (LX or Miata)? I asume that
> > this SCSI controller should work without any
> > problems.
>
> Not the ASSI specifically, but the 875 is well-liked and the driver
> very stable at this time.

Well, I have an ASUS SC875 controller. No problem with my narrow disks,
narrow cd or narrow dat. However for some reason the controller is unable
to write on my ultra wide Barracuda. It works fine if I switch to
asynchronous transfers, but writes always leads to tonnes of error
messages if I use synchronous transfers. I have one internal narrow chain,
and one internal wide chain. No external units. The disk has been verified
with the same type of scsi controller in a NetBSD/Intel machine, no
problem.

Any ideas?
