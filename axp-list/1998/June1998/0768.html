> the LX motherboards (and vice versa)??? I've got a cheap source and need
> to know if it is compatible *before* buying :)..

This is what the manual of the 433a/au says...

The DIGITAL Personal Workstation au-Series
system supports up to three banks of memory.
Each bank contains two identical DIMMs
(SDRAM JEDEC 21-C conforming, 168-pin
with ECC, 83-MHz, 3.3V). To ensure a match,
the POST (power on self test) code use the I2C
bus to read the characteristics of each DIMM
from a ROM on the DIMM. If identical pairs are
not installed, the POST code may reject the
DIMMs and flag the memory as unavailable.

Memory banks can be populated in any order.
The POST code automatically maps the largest
banks to the lowest addresses starting at
00.0000.0000. This prevents holes in the memory
address space.

Memory Bus: The memory bus connects the switch
and the core logic to the main memory DIMMs.
Width to/from memory: 128 data bits plus 16 ECC
bits.

Memory: The main memory system has the following
features:

Unbuffered 3.3V SDRAM chips mounted on
168-pin dual inline memory modules (DIMMs).
Six DIMM sockets on the MLB arranged in three
banks: 0, 1, and 2.

The two DIMMs in a bank must be identical.
Total size ranges from 32 MB to 1.5 GB (when
256-MB DIMMs are available).
Each bank (two DIMMs) is 128-bits data plus
16-bits ECC wide.

My conclusion is: 100MHz (10ns or faster)
unbuffered SDRAM with ECC with and one of
those EEP eprom's on it... The same stuff
they also use in Pentium ][ servers when
they use ECC.
