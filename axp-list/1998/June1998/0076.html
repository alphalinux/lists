OK OK OK ... don't kill me ;)
Last night I wanted to change my networkcard's settings in my Noname PCI
Alpha. I use a Microdyne NE2000 clone card which uses a software setup :(
So what to do with those cards?
I had to get it out of the Alpha, put it into a 'normal' PC, boot DOS and
set it up.
My idea then was:
We have a great x86 emulation at hand right built into the Alpha BIOS!
So what would be the deal to enable this emulation to boot a primitive DOS
in order to setup cards or do any other primitive things? I know that it
cannot emulate the 386 or upards commands but that should be no problem for
those setup programs.
A free DOS implementation is there from Caldera (OpenDOS) which could be
used and eventually modified.
Just my thought's ;) I would love to have this!
Any ideas how to do it?
