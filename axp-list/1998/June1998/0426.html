I've been porting the glint/permedia server from Suse. The source is
not yet releasable, due to it's alpha (and I don't mean the processor)
status. The source will be released someday by XFree86, but not by
me... I'm only doing the AXP port.

Anyway, try the version at: ftp://ftp.altatech.com/axp-linux/xglint

There are two files in there:

XF86Config 20 Kb Mon May 11
17:07:00 1998
XF86_GLINT.gz 1479 Kb Mon May 11
17:24:00 1998

The first is my /etc/X11/XF86Config file, the second is the server
/usr/X11R6/bin/XF86_GLINT (gzipped). Make sure /etc/X11/X is a
logical link to the server.

Suse, at http://www.suse.de/XSuSE/XSuSE_E.html, has an XF86Config
generator, that may run under the emulator (if my XF86Config file
doesn't work for you, and you can't hack it to make it work).

It works fine for me. Your mileage may vary. It is very young code.
If you break it, keep both pieces; Alta (or anybody else) doesn't
guarantee it, etc...

If you have a problem, feel free to ask me about it; but I can't
guarantee I can answer it (the Linux Expo is in a week and I have
negative time left for anything else).
