Check /etc/fstab on the client for the nfs entries. You should see
something like:

server:/export/local /opt/local nfs hard,intr,nodev 0 0

The last two numbers should be zeros. They are (man 5 fstab) a tag
telling dump that this filesystem needs to be backed up and a fsck pass
number (what order they are checked in). The second number should be 1
for / and 2 for everything else local. It should be 0 for any removable
media and remote filesystems (incl. / if its NFS served).
