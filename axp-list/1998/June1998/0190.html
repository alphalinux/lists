This MAY be related to a name-resolution issue on your box, I've seen
it before. This can happen when it cannot locate its own display by
hostname, and there is no reference to the display as localhost.

Check your /etc/nsswitch.conf and make sure that the naming order is
set to:

files dns

(if you are also using NIS:
files dns nis

Also, make sure that your machine has its hostname/IP pair entered in
/etc/hosts. If you are not using static networking, you can set this
IP equal to the loopback address.

This should allow the config file to work its 'get by hostname' stuff...
