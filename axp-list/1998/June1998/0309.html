Jay Estabrook,

> 164LX usually comes as 466/533/600; PWS as 433/500/600. The PCI bus will
> run at its max speed if the processor MHZ is a multiple of 66MHz, so all
> those speed of 164LX will be fastest, only the PWS 600 runs PCI at max.
Hmmm. Strange... they could have used a *6.5 for the 433, and a *7.5 for
the 500MHz. It seems odd (to me) to throw away 7.7% (433) and 6.7% (500)
possible PCI performance on such expensive machines. The jumper setting of
the PWS aren't documented either. They show a certain jumper setting for
the 433MHz, not saying by how much the clock is multiplied.

The 164LX, however uses a binary system for the CPU freqency switch
(0111 = 7, 1000 = 8, 1001 = 9).

Does the memory on the PWS need an eprom like the SRAM's for Intel's
440LX chipset? Or do normal ECC SDRAM's without eprom do fine??
