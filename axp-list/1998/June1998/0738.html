peter@peterw.demon.co.uk (Peter Watkinson) writes:

> Hi,
>
> Has anyone had any success with any DAT and CD RW on alpha linux?
> Particularly 21164 models.

I have had success with a Ricoh MP6201S CD RW using Cdrecord 1.6 and
XCDroast 0.96d. Just be sure to run Cdrecord with fs=4000k and
xcdroast with the environment variable CDR_FIFOSIZE=4000k. I have
written over 70 cds with no problems, and performed multiple
record-erase-records on re-writable media. It is connected to a
Symbios 825 SCSI controller. This is on a 21164PC with RedHat 5.0 and
kernel 2.1.x.

I have not bothered to plug a DAT into this machine yet, but if it
helps, DATs work fine with an alpha under Digital Unix and with Linux
on Intel so they ought to work on an alpha with Linux...
