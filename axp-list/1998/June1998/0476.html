On Tue, 12 May 1998, Robert-Velisav MICIOVICI wrote:

>
> Hello guys, I have some installation problems please read/help:
> I managed to install MILO on Alpha Server 400 and I started the
> AXP Linux RedHat 5.0 installation. With:
> "boot fd0:avanti.gz root=/dev/fd0 load_ramdisk=1" I managed to start
> installation. After I selected the type of monitor, keyb..., I formated
> the hardisks... I've configured the X Windows (that one failed) and at the
> end of the installation the system asks for "boot disk". I know that,
> this disk must be with ext2 filesystem, but really I don't know where and
> wich are this "boot disk".
>
Just use the boot disk you booted with. When you booted Linux, you used
two disks: first the boot disk, then a root disk. Stick the boot disk
back in the diskette drive and press Enter. It will be (relatively)
smooth sailing from then on! The reason the installation program asks
for the boot disk again is to copy the kernel image to the hard disk so
you can boot from the hard disk. You will need to install milo and
linload on a small FAT-formatted partition in order to boot without using
the floppy drive, though.

Good luck.
