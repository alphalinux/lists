I have pcmcia running flawlessly on my udb w/RH5.0. Which kernel did you
recompile?

You need the latest pcmcia sources.
Turn on modules in the kernel. (2.0.30 has been working fine...newer
versions have kernel trap thing..very annoying)

Compile pcmcia modules...install (pretty straight forward, read the readme
files that come with sources).
Reboot.

If you need specific help...let me know!
