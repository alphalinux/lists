>>> Stefan van der Eijk said:
>
> What is the difference between a 164LX and a Miata
> (Personal workstation). Which one preforms better,
> and why?

Both are based on the same CPU (EV56 aka 21164A) and core logic (PYXIS),
So MHz of the processor would contribute to performance differences.

164LX typically has 2Mb of cache, but can have 4Mb.
PWS can have 0, 2Mb, or 4Mb. Obviously the 0Mb would preform slowest.

164LX usually comes as 466/533/600; PWS as 433/500/600. The PCI bus will
run at its max speed if the processor MHZ is a multiple of 66MHz, so all
those speed of 164LX will be fastest, only the PWS 600 runs PCI at max.

> Does anybody have any negative experiences with
> an ASUS SC875 SCSI controller (SYMBIOS Logic
> 53C875) on an Alpha (LX or Miata)? I asume that
> this SCSI controller should work without any
> problems.

Not the ASSI specifically, but the 875 is well-liked and the driver
very stable at this time.
