>>> Dan Roscigno said:
>
> I am trying to install RedHat on a PC164LX system. I have
> the latest SRM console installed. I would like to be able
> to install RedHat 5.1 to this system.

Unfortunately, you're out of luck unless you switch over to AlphaBIOS
console, as there are no SRM-bootable kernels available for the LX164,
AFAIK. Or, you could build such a kernel if you already have LINUX running
on an Alpha available for such a task.

> I am trying to
> use the serial port as the console as I have no graphics
> card installed on this machine (it is one of several
> machines for a research project here that runs headless).

Another problem: I don't believe that Red Hat LINUX will install via
serial console. You may need to temporarily install a VGA in order to
perform the install. Also, you would need a kernel capable of supporting
serial console, and, IIRC, this means it has to be a 2.1.xx kernel.

Sorry...
