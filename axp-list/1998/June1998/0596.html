> Hello:
Hello!

> I've got a RH 5.1 system on an Alpha XL 233 system. Over the weekend I
> tried to do a tape backup of my entire system. I received two sets of
> errors. One, for the tape. This is *probably* caused by running out of
> tape (right?)
> st0: Error with sense data: Current error st09:00: sense key Medium Error
> st0: Error with sense data: Current error st09:00: sense key Medium Error
> st0: Error on write filemark.
> st0: Error with sense data: Current error st09:00: sense key Medium Error
> st0: Error with sense data: Current error st09:00: sense key Medium Error
> st0: Error on write filemark.
This looks more like a general tape problem. See later...

> The second one appears to be a disk error.
> scsi0: MEDIUM ERROR on channel 0, id 0, lun 0, CDB: Read (10) 00 00 32 8e 20 00
> 00 06 00
> Current error sd08:06: sense key Medium Error
> Additional sense indicates Unrecovered read error
> scsidisk I/O error: dev 08:06, sector 1659832, absolute sector 3313186
> scsidisk recovered I/O error: dev 08:06, sector 1659830, absolute sector 3313184

OK. Check your SCSI-bus !!!
Check termination and cabling. Do you have at least the minimum cable length
between your devices? Are all connector connected correctly? How long is
your SCSI bus in total?

> By the way, which device is '08:06'?
Have a look in /dev/ and look for the device with major number 8 and minor
number 6. This should be sda6 ...
