> Currently the equivalnet to MILO for the UX/BX/Ruffian boards is the

> file.

> Jay Estabrook has been doing some work towards updating these, and to make
> them "in synch" with the current MILO for LX and other board types.
>
> How's it going Jay??

The "ruffian.img" floppy image file I pointed to earlier has the original
(Deskstation) versions of ldmilo.exe and osfpal.nh on it; however, the
milo.ful is one built with more recent alpha-patches-2.0.30 than were used
in the original version. Along with those patches came the DEC BIOS emulator,
which allows *much* better support of the PCI VGA cards than the original.

The Deskstation files are available at:

http://www.deskstation.com/linux/install.html
