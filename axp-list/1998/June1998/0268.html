Hello Nils .. i wrote about this problem a while ago .. the fix is :

Quote

It's more a question of which patch *I* missed... :-\

I believe that the alpha-patches-2.0.30-0.2.gz, that Jim Paradis made,
have a needed patch for include/asm-alpha/a.out.h which somehow slipped
through the cracks when I made subsequent alpha-patches releases... :-(

Make sure the following replaces the appropriate stuff at the end of
that
file, and it should work a bit better...

#ifdef __KERNEL__
#define STACK_TOP ((current->personality & ADDR_MAX_32BIT) ? 0x100000000UL : \

((current->personality&ADDR_MAX_31BIT) ? 0x80000000UL :
0x00120000000UL))
#endif

end quote
