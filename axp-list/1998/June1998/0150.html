I have FORTRAN filter applies that do 110MFLOPS
on PPro 200, 150MFLOPS on PII400, 300MFLOPS on Alpha 533
with egcs and 500MFLOPS with DEC FORTRAN on Alpha 533.
Your code needs a little tuning. The x86 egcs code
is pretty well tuned I think but I'm not happy the
PII400 only improved performance 40MFLOPS over a PPro 200.

Also, that same code runs 618MFLOPS on an RS/6000, 64 bit
enabled for those folks in that other thread on AIX.

YMMV.
