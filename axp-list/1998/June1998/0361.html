On Sat, 27 Jun 1998, Robb wrote:

> I have a newbie question concerning source RPMS. I am trying to upgrade
> to the latest development kernel on my XL300, and so need to upgrade a
> variety of packages. Most of the latest stuff is only available as .tgz
> or SRPMS. I have RTFM and whatnot, but cannot figure out the correct way
> to install a SRPM. Would someone be kind enough to give me the procedure
> and commands, or a pointer to some information?

First install the SRPMS by doing a 'rpm -i package.src.rpm'
Then look in /usr/src/redhat in the various dirs, there is it installed,
and in the /usr/src/redhat/SPECS dir there is a file called
'package.spec'.
Do a 'rpm -bi /usr/src/redhat/SPECS/package.spec' to make & install the
package, or do a 'rpm -bb /usr/src/redhat/SPECS/package.spec' to make the
.rpm-file. Install this rpm-file and if it works, make it also available
for the rest of us....
