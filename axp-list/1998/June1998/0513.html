2.0.n kernels are NOT snapshots of 2.1.x kernels

the rules are:
a.b is stable if b is even
a.b is experimental if b is odd
a.b.c is a branch from a.b to add functionality or fix bugs
although for stable versions only bug fixes and new driver
releases are allowed
