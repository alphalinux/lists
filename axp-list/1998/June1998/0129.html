pancho> 1. Are there any tools to see whats going in/out of cache?

21164 has a counter which is called PMCTR(Performance Counter).
If you have 21164, 21164A or 21164PC, you can get various data about
cycle, miss predict and D/I cache load misses....
(Please refer to 21164 Hardware reference manual, P5-30).

I made a milo and kernel to deal with PMCTR.

ftp.eni.co.jp/.2/Linux-Alpha-JP/ftp.statabo.rim.or.jp/PMCTR

There some milos and kernels(EB164, PC164, LX164, SX164) for PMCTR and
sample program. These routines are experiment version, but it's
useful if you want to see cache behavior.
