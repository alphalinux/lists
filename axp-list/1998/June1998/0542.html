> ftp.uk.linux.org/pub/linux/alan
>
> Some alpha-related FP ieee fixes are included in these patches I believe.

Also, what version of the compiler are you using, and are you converting
floats to ints?

The reason I ask is that gcc has a bug in the conversion code such that
it does not generate the software-completion flags for "cvttq"
instructions. This means that even if you specify "-mieee" the
conversions can still generate SIGFPEs. This bug has been fixed in
later versions of egcs.
