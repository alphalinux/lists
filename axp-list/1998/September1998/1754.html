Check the time of the file and your system's time. Some Alpha systems still
have some trouble with setting the correct time and date.

Edit the toplevel Makefile and comment out the SMP=1 line.

> I know no one can tell me exactly what is causing this, but does anyone have
> any ideas as to how to get a kernel compiled already??? I'm dying here.
You will succeed, promised ;)

> I've never had a kernel crash out so many times. Is there a safe way to
> compile a kernel??? I just don't get it....
The safest way is to use a recent version of egcs (latest is 1.1b AFAIK) and
a kernel > 2.1.121. Comment out SMP=1 in linux/Makefile do
make clean depend boot modules modules_install
and you should have a read-to-go kernel in linux/arch/alpha/boot.
