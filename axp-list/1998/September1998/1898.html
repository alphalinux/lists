PHP would do it just fine, but that requires an actual submit and reload of
the page. I've had the best luck with JavaScript. Name your variables:

<INPUT TYPE=TEXT NAME="Dumb">

Then, don't put a submit button at the bottom of the page, just put a plain
old button that calls a JavaScript function called validate() or whatever.

function validate() {
if document.formname.dumb = "dumb"
alert("Dumb");
else
alert("Not dumb");
formname.submit;
}

This is a good way to do limited input validation. If you have to do a
lot of it, it's good to write an actual include file with a validate
function. The advantage is that you don't have to reload the page, and, you
have to learn some JavaScript in order to do it. Netscape's
developer.netscape.com has a good piece on input validation with JavaScript.
