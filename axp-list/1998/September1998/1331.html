HI all!

Just a little problem with postgresql from redhat 5.1/Alpha:

I just installed the packages and installed a webinterface.

But whenever the webserver (or someone else using psql) connects to
the database, the following message is displayed at the console 4 times:

postgres(3610): unaligned trap at 0000000120116e00: 000000011fffd93a 28 1

The entries in the dbase are not corrupted or anything.
It is possible to write some data in the base and to read it out again.

The message is only displayed when the connection is made.

It is not a very big problem, but when the dbase is frequented more
often, there is no chance to work at the console because of this
message, which comes up every time someone connects to postgresql.

Used Hardware:

AlphaPC 164-UX-Board, 533Mhz CPU, S3-Graphics-Adapter,
On-Board Networkdevice (10BaseT), Additional 3COM905 Network interface.
