>>> Kai Schmidt said:
>
> I am just trying to install Red Hat 5.1 on a XL266. I can boot
> the kernel, but when I am asked to insert the ram disk the
> system falls over with a kernel panic VFS: Unable to mount root
> fs on 02:00.

Your boot command line to MILO should look like:

boot fd0:vmlinux.gz load_ramdisk=1 root=/dev/fd0

I don't seem to ever need "prompt_ramdisk=1" as well, but YMMV...

Something else if the above doesn't work: try using the *updated* ramdisk.img
from RH 5.0, instead of the 5.1 version. It'll still work, and seems to work
much better in some instances.

Good luck.
