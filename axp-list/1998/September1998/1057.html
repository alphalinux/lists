On Wed, 30 Sep 1998, Steph wrote:

> Has anyone tried to install RH 5.1 via ftp ?
> I can boot into the installation program and select the ftp method.
> After that I choose to partition my disk (ide) which is detected by
> the kernel ... but the program tells me that there are no available
> hard disk on my alpha ?!?!?!
>
> Someone has an idea ?

I installed RH 5.1 about a week ago this way, and had no problem. Check
that your drive is partitioned properly (did you let the install program
format it?) You can always type alt-F2 and play around with the shell to
see if you can fdisk and mount the drive.
