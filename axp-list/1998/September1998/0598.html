You don't need to disable the onboard TGA card, I have tried a number of PCI
cards in my UDB and had varying success/problems. I did manage to get
as far as having X running with an S3 Trio64 v2 card but the machine would
lock up tight after some time. I'm tempted to try that one again with the
no_pci_disconnect config set to see if that solves the problem for the UDB. If
that works then great, that card was nice and small and would fit in the UDB
without having to lose the internal HD. With 2MB onboard it could manage 16
bit colour at 1024x768 and when I had it running before it made the machine
feel very much faster than with the current slow TGA driver, and of course, no
more colour flashing problems :-)
