>>> Steph said:
> Has anyone tried to install RH 5.1 via ftp ?
> I can boot into the installation program and select the ftp method.
> After that I choose to partition my disk (ide) which is detected by
> the kernel ... but the program tells me that there are no available
> hard disk on my alpha ?!?!?!

Are you booting from SRM console, and setting up that disk with BSD-style
partition table? If so, that's your problem (actually Red Hat's); the 5.1
RAMdisk scripts do NOT look for BSD-partitioned disks.

However, there is a solution; simply use the RH 5.0 RAMdisk (look for the
updated ramdisk.img to be sure). It seems to work fine with the 5.1 CD, so
I'd expect it to work for FTP as well, but, as always, YMMV... ;-}

Good luck.
