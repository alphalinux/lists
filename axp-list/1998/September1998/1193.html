I did nothing apart from rpm-ivh the package and everything run smoothly
apart from entering a few mount point for Appletalk to see. If you are
running 5.1, you need netatalk-1_4b2+asun2_0a18_2-phh4_i386.rpm, any other
version, at least in my experience; won't work. You certainly don't need to
configure interface if you only have one, and usually there ain't much you
can do on AppleTalk anyway.
