All of your x86 shared libraries to be used with em86 should go in
/usr/x86/lib. The problem you'll likely find with netscape is that em86
doesn't like using ld-linux.so.2 and libc.so.6 for intel. I fought with
it a while back; I could probably hunt down my modifications to the code
for you if you'd like, but I never did get it working.

- Bob
