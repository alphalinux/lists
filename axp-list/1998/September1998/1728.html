Has anybody tried to compile a kernel on an alpha with egcs 1.1b? I tried
compiling the 2.0.33 source tree that compiled just fine with egcs 1.0.3a and
it failed with the following:

gcc -D__KERNEL__ -I/usr/src/linux-2.0.33/include -Wall -Wstrict-prototypes -g
-O2 -pipe -mno-fp-regs -c -o ppp.o ppp.c
/usr/src/linux-2.0.33/include/linux/symtab_begin.h:46: warning: This file
contains more `{'s than `}'s.
/usr/src/linux-2.0.33/include/linux/symtab_end.h:16: warning: This file
contains more `}'s than `{'s.
gcc -D__KERNEL__ -I/usr/src/linux-2.0.33/include -Wall -Wstrict-prototypes -g
-O2 -pipe -mno-fp-regs -c -o slhc.o slhc.c
gcc: Internal compiler error: program cc1 got fatal signal 11

Guess I should go back to 1.0.3a?
