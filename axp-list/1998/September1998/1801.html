I had some trouble to have RH51 on a AS500/266 with a QLogic ISP installed.
In particular: MILO and kernel did report a lot of unexpected interrupts but
did work on the first try. I then upgraded the AlphaBIOS/ARC as suggested by
the docs and afterwards it hang on SCSI probe. I tried some MILOs from Jay,
but neither helped.

Using SRM/aboot (from floppydisk, as I have a PC style partition table) to
boot the kernel is working eithout any problems though. Note that you need
aboot-0.5-7 in order to boot recent (2.1.1xx) kernels built with recent
binutils (2.9.x). I reecently installed 2.1.125ac2 w alpha.irq.patch
(from RTH?) and it works as expected until now.

I'm still interested in getting ARC/MILO boot procedure to work, so if
anybody has some new suggestions ...
