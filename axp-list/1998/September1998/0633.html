Hi,

We faced the same problem (scsi adapter (ncr53c810) detected by `MILO
image' but deleted by boot image --> no disk --> no /bin/insmod) on an
Avanti AS200 4/166, using :
MILO image : avanti
boot image : xl.img

Using avanti.img as boot image automagically solved the problem.
