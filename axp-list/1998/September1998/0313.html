On Tue, 15 Sep 1998, B. James Phillippe wrote:

> The command for installing aboot is "swriteboot /dev/sda bootlx". I don't
> know what effect this will have on your partition scheme because of the
> first partition starting at offset zero. I suspect that it will be okay as
> long as you don't ever try to use /dev/sda1.

Well, I find it useful to create a one cylinder long partition at the
beginning of the disk. I always mark it as "boot" type (type "d"). I
make it to remind whoever might mess with the disk the free space at the
beginning is intentional.

One has to use `swriteboot -f1' in this configuration.
