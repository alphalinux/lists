Quoting Peter Petrakis (peterpan@wn.net):
> I couldn't send messages to the list for some reason that I just figured out.
> The list server is case sensitive. I had PeterPan@wn.net as my email addrss and
> it didn't work. When I changed it to peterpan@wn.net the message was sent to
> the list and i recived it in like 5 seconds. Is it supposed to do that?

According to RFC-822, the standard for internet addresses, some mail servers
may consider the userid part of the address case sensitive, and others may
not. So it would be a good idea to always treat it as if it were case
sensitive. The address part is NOT case sensitive, and sites that treat it as
if it were (like some Microsoft products) are seriously broken.
