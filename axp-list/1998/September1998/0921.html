Bob McElrath wrote:
>
> I seem to have nuked my flash rom, and can't get it to boot. (4 beeps ->
> "No valid header found in ROM; loading entire ROM". Anyone else done this
> before? Anyone know how to fix it? (Adding CF7 jumper to load the
> "failsafe loader" has no effect)
>

The failsafe loader takes a long time to access a floppy. If it won't
(and it has to like the floppy format), you'll need to remove/reprogram
and replace the FLASH. By definition you need the FSB in block/sector
0.

I did this one one of my LX boards and damaged a land pad under the
flash (pin 6 if memory serves). It has an ugly green wire to an
adjacent via now.
