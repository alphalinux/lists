On 02-Oct-98 Michal Jaegermann wrote:
>>
>> What is "Project.tmpl" part of?

> Project.tmpl is something which is a given project specific
> and is defined conditionally. Sounds like some particular Imake
> you have on hand defines it even if it is not present.

While seemingly off-topic the answer might be of interest to anybody installing
SwIM Motif for axp. Apparently SwIM Motif 2.0.1 installs an Imake.tmpl file
that wants Project.tmpl. The installation re-names the original to
Imake.tmpl.old and restoring that solved the problem. Motif 1.2.4 seems to have
a Project.tmpl but 2.0.1 does not.

I was trying to compile SIAG on my Alpha when I encountered this problem. It
compiles without errors but the PW word-processor dumps core if you are not
very careful.

Has anybody had success with SIAG on the Alpha platform or, (the perennial
question) know of any WYSIWYG WP that will compile on an Alpha now that I have
Motif? I'll buy Applix when it comes out for the alpha but would like something
in the meantime.
