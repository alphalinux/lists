Your problem is probably that your root partition
name has changed because of the new partition you
have created. Try to say mount root=/dev/<old partition name + 1>
to MILO and it will mount the correct partition. Then you can
edit /etc/fstab. From what I know in Linux x86, the root filesystem
is mounted on read-only first in order to be checked. Then it is
remounted RW.
