OK; the first thing to ask is which version of the kernel did you build?
Some pointers which may help:

1) Depending on the kernel version, there is either a pair of options
for TGA and VGA support or an option to support TGA. If you have the
first type then make sure you enable VGA; if you have the second type
make sure you DISABLE TGA. A sign of this is that although you don't get
anything else on the screen you can still here the disc crunching.

2) Did you set the machine type at the beginning of the configuration to
LX?

3) Some versions of the kernel just don't like being configured with
make xconfig or make menuconfig. So after you've done make xconfig
to set it up, go through and check it with make config.

4) On Alpha you want the file arch/alpha/boot/vmlinux.gz - thats the one
you need to boot (don't decompress it or anything by hand).

5) What other hardware do you have in the machine?

Well see how you go with that and let us all know how you get on.
