> The package is only present in the RedHat5.0 distribution.

RedHat switched from gcc to egcs in going from 5.0 to 5.1.
The related RPMs are
egcs-1.0.2-8.alpha.rpm
egcs-c++-1.0.2-8.alpha.rpm
egcs-g77-1.0.2-8.alpha.rpm
egcs-objc-1.0.2-8.alpha.rpm
however 1.0.2 is now considered obsolete.
I would recommend downloading egcs 1.1b (see
http://egcs.cygnus.com/ for more information).

You can also install the gcc RPM from the 5.0 distribution:
it works also under 5.1. If you do not have any compiler now,
I would suggest install gcc from RPM, and then use that to
compile egcs 1.1b. Then you have two compilers ;-)
