Bob McElrath wrote:
>
> BTW, I discovered that egcs doesn't install the cpp binary by default.

This is not really correct but maybe cpp is not the place you expected
it. Look

$ locate /cpp | grep -v cpp.
/lib/cpp
/usr/lib/cpp
/usr/lib/gcc-lib/alpha-redhat-linux/2.7.2.3/cpp
/usr/local/egcs-1.1b/lib/gcc-lib/alpha-unknown-linux-gnu/egcs-2.91.57/cpp
/usr/local/egcs-snap/lib/gcc-lib/alpha-unknown-linux-gnu/egcs-2.92.13/cpp

Preprocessor is installed in a compiler specific library although
with cpp is likely of no consequences which preprocessor binary you are
really using. I'd rather suspect that your PATH is not correct.

OTOH quite a few other programs, various X-windows stuff in particular,
expect to find /lib/cpp and some RH sets did not put it there.
Make yourself a link if it is missing.
