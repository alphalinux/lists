Hello,

I've just recieved an ALPHAstation 500/333MHz at work and I'm having
some trouble getting Linux installed on it.

I'm using alcor.img for my boot disk. I use ramdisk.img for the
ramdisk. I get started using MILO and then the boot disk.

In the installation, I get right up to the point where it tries to mount
the root disk it just created.

I have a RZ29B that I'm creating 3 partitions on. One 10MB MILO
partition, one 256MB (I have 128MB ram) swap partition, and the rest is
just an ext2 partition.

It creates the filesystem and then when it tries to mount it, I get a
'directory not found' error from the mount attempt.

When I got that far, I was doing an FTP installation.

Oddly enough, when attempting an NFS installation, while I could NFS
mount the RedHat directory from other locations, I got the same
'directory not found' error while trying to mount the NFS filesystem as
I got when trying to mount the ext2 filesystem (the above problem).

Does anyone have any suggestions.
