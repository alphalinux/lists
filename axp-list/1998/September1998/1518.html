Um. I assembled my UX (running debian, not redhat) using the Linux tools (I
don't have an NT CD) - I made sure that the filesystem was 16 bit FAT using
some option to mkdosfs (and the appropriate parition type under fdisk) and then
I mounted it and copied ldmilo.exe and friends to it. I suspect you need to
simply recreate the dos partition and recopy the files. Note that I mounted it
as an msdos filesystem, not as a vfat one; this may or may not make a
difference, but it certainly can cause the ARCSBIOS to produce odd looking
directory listings if you mount it as vfat or use mtools (I suspect there
arguments to mtools to avoid creating vfat directory entries, which would
probably do just as well).
