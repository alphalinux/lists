On Thu, 8 Oct 1998, Jochen Kuepper wrote:

> On Thu, 08 Oct 1998 Dam Thanh Son wrote:
>
> >Notice that I use the old DEC TGA server (accelerated, version 3.1.2).
> >With the XFree86 TGA server (which I don't normally use because it is not
> >accelerated), I need to tune more (can be done with xvidtune).
>
> What server is that ? Is it a DU program ? Does it run TGA2 cards as well ?

No, it's a Linux program that used to come on Redhat 4.0 CD. You can ftp
it from gatekeeper.dec.com/pub/DEC/Linux-Alpha/X11, I use the file
XF86_TGA-3.1.2-bugfix-EV4. I know nothing about TGA2. You need libc from
RH 4.x to use this server. Also, I found that it cannot read *.gz fonts,
so be sure to have only *.Z fonts. On my UDB, it feels a lot faster than
the non-accelerated servers.
