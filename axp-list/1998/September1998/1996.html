If your at the SRM console (>>>)
type set os_type NT
power cycle the system
Note: Highlight the Menu Items using up-down arrows, then execute by typing
a CR.
Then from the ARC console, select "Run a program"
enter A:arcinst.exe

after your done, to get back to SRM... if you want to..

Follow the following menu path, Highlight the field using up-down arrows,
then execute by typing a CR.

Supplementary menu
Set up the system
Switch to OpenVMS or Digital UNIX console
"Switch to Digital Unix"

then SELECT
Setup menu
You should see a banner telling you to Power Cycle the system, it will
re-init and start the SRM console.
