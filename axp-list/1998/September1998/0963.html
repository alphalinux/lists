Sure. The first thing you'll want to do is read all the nice documentation
that folks have put together. Here is a shopping list. I'd recommend reading
each before starting with the installation.

Alpha-HOWTO (see http://www.azstarnet.com/~axplinux)
AS200 Support Page (see http://www.toast.net/~strycher/AS200.html)
RedHat-5.1 Users Manual and Alpha Addendum
Alphastation Users Manual

I'll give you a hint now; if you decide to use the SRM console, make your
boot disk from the avanti-s.img image on the RedHat CD. If you decide on
ARC, use the avanti.img image.

Good luck,
