>
> Hello,
>
> I am experienceing problems with installing the milo onto the first
> partition. I installed RH5.1 and divided my hard drive into four
> partitions.
>
> sda1 MSDOS partition 2mb
> sda2 linux swap 64mb
> sda3 linux native 2gig
> sda4 linux native 2gig
>
> The installation goes fine. I copy the milo image from the floppy
> to my first partition and then I set up my AS250 to boot off of the hard
> drive. But when I try to boot Linux from the first partition I get a
> 'file system unrecognized' error message.
>
> Any idea's would be apreciated. Thanks.

>From what I've seen/read, you have to create a 16bit FAT
file system, not a 12 bit one. Seems ARC/Alpha bios
can't read 12 bit FAT. I believe 16bit is used if you
make a partition 32MB or more, or, force it to 16 bit
when you format it.
