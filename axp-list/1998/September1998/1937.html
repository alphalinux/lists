> On Wed, 21 Oct 1998, Franco Tassone wrote:
>
> > I need some help in setting the ftp server in our alpha xl266 RH5.1, in such a
> > way that all non-anonymous ftp logins are definitively chrooted in users home
> > directory.

Very simple actually, using WuFTPd ... make the passwd entry:
user:pass:UID:GID:GECOS:/path/to/chroot/./path/to/home:/bin/shell

Just remember that /path/to/chroot needs /lib, /usr, /bin, etc. I find it
easiest to create hard links to the needed files if you're on the same
partition.
