You have to have some way of creating an ISO9660 CD image with Rock Ridge extensions. This is the key to getting the long filenames correctly written. The CD mastering software that I have access to for Win95 will write an ISO9660 CD but not with Rock Ridge extensions, it supports only the Romeo and Joliet extensions, so the CD that I made and tried to install from didn't work. The mksiofs command for linux will create a proper CD image, so if you have access to another machine with linux, that is the way to go.
