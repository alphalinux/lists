> > > On Tue, Aug 31, 1999 at 11:53:09AM +0000, Shane Steven Sturrock wrote:
> > > The SX has a 16k/16k L1 cache, and no onboard L2. The LX has 96k
> > THIS IS WRONG!
> NO IT IS RIGHT :) Onboard means IN the cpu package in this case.
> > I own an SX and I know that it has 1MB of external 2nd level cache.
> Yes this is true, but its not onboard the cpu, its external!
OK ok ... it is also called 3rd level cache or backup cache, not 2nd level
cache.
The 21164PC has for that reason no 2nd level but backup cache (maybe called
3rd level).

> The PC on the CPU part number denotes the absence of the L2 cache on the
> SX cpu...I suspect that an SX is a 21164 on which they disabled the
> onboard L2 cache cause it failed passing manfacturing tests and stamp it
> 21164PC and sell it with no L2
Err ... I do not think so. AFAIK the 21164 is a completely different CPU.
The PC version also has some "new" instructions which the "pure" 21164 does
not have. Digital claim those functions to be something MMX equivalent
able to help encode/decode MPEG1 and things like that.
But who uses those instructions anyway?
