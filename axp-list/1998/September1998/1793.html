>SO, has anyone been able to install Linux (using MILO or the console) on
either of this systems USING the QLogic controller? Any hints, tips, things
to look for? Is it worth to get an Adaptec controler, and if so, can I use
one (PCI) straight from the Intel PCs?

I had an ISP 1040B that came stock with my system. It didn't like my SCA
Seagate Hawk - it locked up when MILO tried to read it. I didn't know how
to build a MILO and had no source access, I just called up the company and
asked them to switch it for a 2940UW - which is what it should have included
in the first place as it was advertised. It fixed my troubles. If I
remember right, the ISP didn't like a CD-ROM of mine either. A card switch
fixed that too.

And I had a Miata like yours, ex it is a 500a.
