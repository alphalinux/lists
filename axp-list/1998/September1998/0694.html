Hmm; I don't know, usually a TGA1 card is called ZLXp-E[123], I've never
heard of ZLXp2-E... :-\

But assume that its the ZLXp-E2; there is a hacked up server for it;
look for the latest server at:

http://www.cs.brown.edu/~tor/xfree86/

Tim Rowley (tor) hacked the 3.3.2 XF86_TGA code to support 24-plane,
albeit somewhat slowly (treats it only as dumb framebuffer, ie no
acceleration to speak of).

gunzip it, move it to /usr/X11R6/bin/XF86_TGA, make sure /etc/X11/X is
a symbolic link to it, and go...

You may want to refer to the axp-list archives, see if someone there has
contributed an XF86Config file that would be appropriate...

Good luck.
