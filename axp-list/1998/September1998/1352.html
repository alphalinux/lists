Hi,

I'm wanting to translate (using a script) X-window dump files into
other formats, specifically postscript. On non-Alpha platforms,
I can do this using the netpbm package--use xwdtopnm to generate a
"portable anymap file" and then I can use pnmtops to translate to
Postscript. The netpbm package is installed on the Linux Alpha's,
but I can't get xwdtopnm to work: I get thie message:

xwdtopnm: unknown XWD file version: 33554432

Does anyone know of a version of xwdtopnm (or another command-line
type method) that will work on Alpha's? (The problem is also seen
on Digital Unix). I suspect that there's something unique about
.xwd files on Alpha-powered machines.
