Can you be a little more specific about your problem?

1) Get the latest version and compile it.
2) Go through your smb.conf and look for mistakes, then do it again.
3) If you Linux machine does not show up to Win95/NT machines, is there a WINS
server setup, if not is the Linux machine in the lmhosts file of the Windows
macine.
4) Make sure all machine are running with the same subnet mask.

I have found the whole Winows networking to be VERY fragile and even when
evetyhing is setup up right it only works most of the time.

If you still have trouble feel free to e-mail me. I've done quite a bit of
Samba work over the last few months on a variety of UNIX boxes.
