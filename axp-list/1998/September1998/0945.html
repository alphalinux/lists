OK, in reference to the mindspring docs,section by section
Need Section

1. Good section, correct and all, although I used 6.8 update not 6.7
from ftp://ftp.digital.com/pub/DEC/Alpha/firmware/v5.2/alpha200/ :)

2. Correct...I used the standard milo.

3. The readme in redhat says use xl.img NOT avanti.img for the kernel
image. In fact if you try to use the avanti.img, it will not boot, the
xl.img image however does work great.

Under BEGIN Section
1. Sounds good

2. boot floppy
is sufficient here, the rest is useless :)
3. OK
4. OK
5. OK

Under Install MILO,
yes make sure you made a milo boot partition during the install(dos
partition at least 1Meg, you can make it larger, in fact it will be in
most systems anyways). Fdisk the rest for unix use...not dos!
mkdosfs /dev/xxxx where xxxx is the dos partition, the mount the dos
partition under say /mnt and mcopy a:* /mnt

Then do the rest stated there...dont forget to point your boot to the scsi
dos partition instead of the floppy in ARC.
