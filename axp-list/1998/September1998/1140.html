Greetings,

On a related note, does GPL prevent someone from modifying and using the
code if they don't distribute it?

For example, suppose Fred writes the awesome biology code "Gene+" in
school and distributes under GPL and lots of people use it and
contribute and love it. Then NewBioCorp modifies Gene+ and gets some
amazing new research done with their new version, and makes lots of
money consulting with it, but doesn't distribute it. (Suppose for the
argument that even a lawyer can be shown from the nature of NewBioCorp's
results that they are using Gene+ code, and that it has been modified.)
This is really against the spirit of GPLing the code and Fred's
intentions in doing so, and Fred and the whole Gene+ mailing list want
NewBioCorp to release the modifications.

Does GPL require them to make their source modifications freely
available? If not, is there a license which does?
