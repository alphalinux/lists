Looks like you have a problem in your header files.
/usr/include/linux/errno.h is not part of egcs.
Maybe you need to re-install the kernel header rpm.
You didn't happen to overwrite the orignal /usr/src/linux
directory at some point? On my system, here's how the
include is set up:
