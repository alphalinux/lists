On Tue, 29 Sep 1998, Stephen J. Gaudet wrote:

> >Thanks to everyone who convinced the AlphaBios team in adding this
> >very basic feature!!
>
> It is a great feature but...it has no serial control via com1. Which means
> you can set it to count down and boot headless. However, can't stop it or
> make any changes via console. Good but, still needs work.

Doesn't LX support SRM? As SRM is targeted to server systems and not
only workstations, it has serial console support since day one. It would
be an ideal solution for you, I suppose.

Adding serial support to AlphaBIOS would be a great pain, if at all
possible, I'm afraid, due to it's full-screen, semi-graphics concept.
