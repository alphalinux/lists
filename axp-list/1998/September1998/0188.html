> Does anyone have the Summasketch running in Linux? This is both on alpha and
> Intel if you know... seems to be the most standard one so I figured it might
> have a driver... is it allways absolute in X? Aka no pick up and drag type
> stuff...
> Thanks,
> Keith
> P.S. Where do I find drivers? Is it a kernel level thing? Or an Xdriver...

It is part of the XInputExtensions (a module). RedHat includes it with
every X-Server (/usr/X11R6/lib/modules/xf86Summa.so). It's a standard
part of XFree86

I've heard some bad reports, but the SummaSketch III Professional works
here. absolute && relative modes are supported.

Here my related setup (XF86Config)
I can switch between mouse and tablet by
executing "xsetpointer psaux" and "xsetpointer tablet"

-------------
Section "Module"
Load "xf86Summa.so"
EndSection

Section "Pointer"
Protocol "PS/2"
Device "/dev/psaux"
Emulate3Buttons
Emulate3Timeout 50
EndSection

Section "Xinput"
SubSection "Mouse"
Protocol "PS/2"
Device "/dev/psaux"
DeviceName "psaux"
Emulate3Buttons
Emulate3Timeout 50
EndSubsection

SubSection "SummaSketch"
Port "/dev/ttyS0"
DeviceName "tablet"
# Keith, ever tried man XF86Config ??
Mode absolute
Cursor puck
EndSubSection
EndSection
-------------
