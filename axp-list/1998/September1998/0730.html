On Tue, 22 Sep 1998, Scott Lewis wrote:

> Anybody tried Debian on an alpha yet? I'm pretty used to RH5.1, but word
> on some of the groups is that even in beta, debian is more stable than RH.
> Perhaps sacrificing usability for stability? I know which one I will
> take...

I have been running Debian 2.0 for Linux/Alpha on my XL366 and UDB
for about two month now with hardly any problems. They are running almost
as well as my Intel boxes running the Debian 2.0. For overall stablity,
they are as stable (if not more so) as RH4 (don't know about RH5, skipped
it in favor of Debian). One I am using as a devel server for compiling and
testing databases and such (UDB), the other (XL366) is the main
file/web/database server for my personal network. Neither take a heavy
load, but they haven't been idle a lot either. Overall, I am very happy
with Debian.
The only sticky point was installation of some packages were a bit
tricky. Just make sure you build the system with the newest copy of
Debian from ftp.debian.org that you can get. Things are improving
everyday. Also, subscribe to the debian-alpha list (see www.debian.org)
for more advice/help.
My two cents.
