Do it the easy way.
Keep the older one until you prove that the newer one works! To
use the new one, just run:

export PATH=/usr/local/bin:$PATH

To set /usr/local/bin first in the PATH. Then it will pick up
the new version. Once it's stable, then delete the old one, and
re-install the new one in the proper location (optional!). You can
change where it will install to using the --prefix modifier when you
configure it.
