>>> Tomas Macak said:
>
> I just tried 2.1.123 kernel on AS1000A aka noritake 4/233. But after:
> Console: colour VGA+ 80x25 it freezes.
>
> I have problems with 2.0 also: I cannot boot after hard reset noritake
> kernel, but mikasa (AS1000) works. When I once boot mikasa, noritake
> also works ...?

for the very latest stuff, try using 2.1.124 plus Richard Henderson's
patch set: ftp.cygnus.com/pub/home/rth/d-axp-gen-19.gz. There have been some
loose ends WRT Alpha support for a number of releases, and only with that
patch set should everything be reasonable.

Also, for a 4/xxx model AS1000A, make sure you do NOT choose the PRIMO option,
since that depends on an EV5 CPU, and yours has only an EV4.

Good luck, and please report any problems with those patches.
