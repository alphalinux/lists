On 19-Oct-98 Steve G wrote:
> I just pulled egcs-1_0_3a.tar.gz from egcs.cygnus.com. I've never installed
> a new gcc compiler before and I'm not sure where to start. What directory
> should I un-tar this into? Also is tar -xzvf sufficient for un-compressing
> it? I'm sure there's a readme file in there somewhere, but I want to know
> where to extract it to so I can read it. Any help on problems I might have
> would be greatly appreciated, also.

Yes there is a README or an INSTALL or the like. Basically you un-tar the
archive with tar xvzf. Cygnus suggests building from some place other than the
source directory.

I just made a directory /root/egcs. From there you type
/usr/src/egcs-1.1b/configure (or whereever your source tree is). Then type
"make bootstrap" and go have a beer or two, it takes a while.

Note that 1.1b is the latest version of egcs, not 1.0.3a.
