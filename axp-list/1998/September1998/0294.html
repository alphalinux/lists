On Tue, 15 Sep 1998, Jonathan Vafai wrote:

> What is the most recent kernel that works out of the box?
>
> I am trying to get 1.1.121 to work. It compiles, but will not boot after
> it unzips.

If you get "aboot: block-size smaller than anticipated!" error then you
might try the fix to aboot I posted to this list some time ago. If you
don't have the patch, you may get a source RPM package that contains it
and recompile it (using `rpm --rebuild'). It is available at:
'ftp://ftp.amg.gda.pl/pub/OS/Linux/people/macro/SRPMS.glibc21/aboot-0.5-6.src.rpm'.
There is also an Alpha binary package nearby but it requires glibc 2.1 so
unless you know what you are doing do not install it.
