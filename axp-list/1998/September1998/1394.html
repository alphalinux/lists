> I'm thinking about getting one of these as200 machines from onsale.com.
> It's an AS200 4/233 for 229 bucks. It is worth it?
>
> Eric

Once you get past the idiosyncracies of installation, yes. Damn, I paid
349 for mine. Oh well. Do be sure to add more memory, though. Mine came
with 16 which wasn't even enough to run the installation successfully. 24
is a minimum. They seem pretty happy with >48. Also, if it doesn't come
with a disk, get a Fast SCSI-II internal. The external bus is only SCSI
II. There is a noticeable difference between my two identical drives,
where one is external, and one is internal. I should put the CD external,
and put the other HDD internal. Project for this weekend, I guess. There
is a wealth of information on Linux on a AS200 out there, so don't worry
too much.
