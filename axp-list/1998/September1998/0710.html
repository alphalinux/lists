Hi Steph,
I did post a message dated 14 Sep on how to boot off the CD and get to an
installation screen
without using the ram diskette for a SX164 with a ATAPI CD_ROM and a IDE
hard drive.

I have sucessfully installed and am running RH5.1 Linux/Alpha on my
Deskstation 533Mhz with 21164PC processor.

Here is a summary of the previous post:

Details:
DeskStation Rebel 533 with AlphaBIOS 5.62-1
System Alpha PC164SX
21164PC 533MHz, 64MB RAM,
Generic DualChannel PCI IDE controller with fixed hard disk ST34342A 4GB
and ATAPI Sony CD-ROM CDU711

1) At MILO> prompt, type boot floppy. This will give some scrolling text
and will make floppy drive
LED light up indefinitely. Just ignore it, and take note how Linux
recognized your CD-ROM
drive. (eg, hdc for IDE ATAPI CD-ROM etc)

2) Now turn on the machine again with the floppy with milo and linload in
A: and CD-ROM1 of RH5.1 in CD- ROM drive.

3) When you get to the MILO prompt issue, boot kernel directly from CD-ROM
issuing this command

boot -t iso9660 hdc:kernels/sx164.gz root=/dev/hdc

4) After scrolling text,a RH5.1 installation screen should appear.

Yours Sincerely,
Govin C
