> I have an Alpha LX164, and for some reason the mouse is no longer working
> correctly. When I start up X-Windows everything works fine but the mouse.
> Upon exiting it I find an error message stating the mouse does not exist at
> /dev/mouse.

/dev/mouse is a symlink to the "real" mouse device; either /dev/psaux
for the PS/2 mouse port, or /dev/cua[01] for a serial mouse. Check
the symlink and see if it makes sense...
