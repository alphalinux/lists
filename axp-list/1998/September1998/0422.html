stefan.larsson@svt.se writes:
> Can RH 5.1 be installed on an Digital Alpha XL 300 ? I have been trying to
> get substantional info about this but havent been able to find anything so
> far. Does anyone have any url:s or something that i can have ?

Yes, that is possible by using MILO. The XL-300 is also know as
XL-Turbo, so the system type is 'xlt', NOT just 'xl'! This is
important to select the right MILO image and the kernel. You might
also use stuff that's refered to by the name of the chipset, which is
'alcor'.

Now, I recall that the kernel supplied by Redhat-5.1 isn't working
correctly. I found somewhere at ftp.digital.com an kernel named
'alcor-2030', which worked for me.

For URLs look at http://www.azstarnet.com/~axplinux/

You should also read the MILO-Howto.

HTH, Martin
