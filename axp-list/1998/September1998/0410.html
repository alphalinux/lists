>>> Don Myroon said:
>
> I've just recieved an ALPHAstation 500/333MHz at work and I'm having
> some trouble getting Linux installed on it.
>
> I'm using alcor.img for my boot disk. I use ramdisk.img for the
> ramdisk. I get started using MILO and then the boot disk.

You need to use the XLT MILO and kernel, if possible; that hardware is more
nearly like the AS500 hardware than the ALCOR is.

Your other problems may go away with the different MILO/kernel. Also, I've
found that I usual need to specify the numeric IP address, rather than the
name, fully qualified or not, of the server, in order to get NFS to work.

Good luck.
