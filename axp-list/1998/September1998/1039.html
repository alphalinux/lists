What you will need to do is to get DOS program called rawrite. Then you
will exit to DOS shell and run it. When it asks you for the source file
just say avanti.img and the a: drive as you destination. This will
create a disk for you. Also you will not be able to read the ramdisk at
all, so don't be scared if you can't access it after you use the
rawrite.

Hope this helps
