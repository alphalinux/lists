Yes. I would recommend this procedure:
- download the floppy image at
ftp://gatekeeper.dec.com/pub/DEC/Linux-Alpha/Kernels/Test-Milos-2.0.30+/ruffian.img
(this MILO is the same distributed with RH 5.1)
- copy the image into a real floppy (for instance using dd on another
Linux box)
- make a system partition using ARCINST from a Windows NT (sigh)
CDROM (alpha subdirectory)
- copy LDMILO.EXE, OSFPAL.NH, MILO.FUL directly from the floppy
to the boot partition using ARC (from the panel where the
machine components are represented in a tree structure).
- start Linux installation as described in
http://www.deskstation.com/linux/install.html
Never modify the system partition again (use fdisk only to
change the other partitions); never mount it as a msdos filesystem
from Linux, as you may render it unusable and then you would have
to redo everything from scratch.

things ;-)
