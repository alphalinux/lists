Hello!

> One of the last irritating buglets I have with my Alpha is that I can lock
> up the machine to the point where I need to go for the reset button, by
> using the "magnifying glass" feature in xdvi.
> I can obviously avoid the problem by not using this feature, but I wondered
> whether anyone has a fix.
[...]
> My machine is an SX146 currently running 2.1.123.
> I'd be grateful for any clues, or even reports of xdvi working perfectly on
> some other system.
I recommend using 2.1.121 since 2.1.122-123 are NOT stable.
With my SX I have the problem that modules are not cleaned up correctly and
that some bug causes programs to crash randomly. 2.1.121 does not show this
effect.
You might also have a look at the SX164 page at

http://www.unix-ag.org/Linux-Alpha/Architectures/SX164.html

and include the IRQ patch. You should also _disable_ the RTC in your kernel
in order to avoid IRQ problems...
