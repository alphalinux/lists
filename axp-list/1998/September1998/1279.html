well, there are a few ways to do it... if you already have an ARC console
try this:

1. get this file (note: there is also a version 6_8 somewhere but
i don't have the URL here at work):

ftp://ftp.digital.com/pub/DEC/Alpha/firmware/v5.1/alpha200/as200_v6_7.exe

2. copy it to a freshly made dos floppy as filename: fwupdate.exe

3. in your ARC, select "Install new firmware..." with this floppy in the
drive...

4. you will be given a choice to update arc ROM or srm ROM, choose srm...
and then all the obvious steps thereafter (exitting and such),
and viola!

to get back to the ARC from SRM follow the instructions at:

http://www.mindspring.com/~ccsf-lug/AS200.htm

hope this helps!

matt
