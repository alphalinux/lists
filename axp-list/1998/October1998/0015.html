I've compiled qt 4.0 with no problems, I haven't tried 4.1. I have
installed gcc and ecgs, I'm not sure which I used to compile with, but it
compiled clean. Remember to make the changes to your .cshrc or .login
file (or it won't work) also in the documentation on QT there is a
mistake. If you use the bash shell, you have to correct thier .login
script thingy (I found this for a friend, I use tcsh for a shell, so I
didn't see it when I did it for my machines.) As I recall the error was
pretty obvious.

If you need help I'll be happy to help you. It seems I've had to help
about 3 or 4 people recently with compiling from source. I think
trolltech needs to write some clearer instructions ;)
