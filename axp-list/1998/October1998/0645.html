Steve,

        You need 3 disks. MILO, boot and root

        On disk one (FAT format!!!!) you COPY the linload.exe and
udb-noname.img milo
        image (found in /milo/images).

        Disk 2 you use rawrite (or dd under UNIX) to place the noname.img (from
/images)
        file on it, the third use rawrite to put the ramdisk.img (get the new
one in the
        errata from RedHat) file on it.

        Now put the MILO disk in the floppy and start the process. Install docs
can
        be found at:
        www.alphalinux.org follow the link for docs then udb-multia

        FTP works fine, you may need to put the IP address of the FTP server
instead
        of the hostname that's all.

        Good Luck,
        rdp
