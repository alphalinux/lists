One problem, I believe the 500mhz is a PC164 that uses simms, not SDRAMS.
Another item worth mentioning, on the PC164 motherboard it ran one of two
ways. If you populated four of the eight slots you had a 128-bit data path,
populate all eight you have a 256-bit data path to cpu, cache and ram.

So Tom must make sure you have a PC164 first with 8 simms sockets and then
decide which memory configuration he wants next. If you have a LX then it
has 4 SDRAM slots.
