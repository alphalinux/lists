Bill,

I installed RH5.1 via FTP over the internet without any problems at all. I
updated the RPM's as detailed on the general and alpha erratas. This box is
my sendmail/http server. It's been up for over a month without a reboot.

Gotchas:

DNS does not work with the ftp install. You must use IP addresses for the
ftp server. And if you have a small disk drive, you probably cannot install
the X server.
