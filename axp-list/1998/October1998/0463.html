On Sun, 22 Nov 1998, Stephen Williams wrote:

> Does anybody know of a schematic editor that runs under Linux/alpha?
> I'm looking for a tool that generates net lists that I can feed to
> other EDA tools of my own making.
>
> I'm not seeing, so I must be looking in the wrong places. I'd be willing
> to put some porting effort into a decent tool.

Well, I can be of some service here, actually ..

I have started putting together links and information on EE and DSP
related applications for Alpha/Linux.

One of the packages that I am looking into is the Chipmunk distribution[1].
It is said to work on Alpha/Linux.

Within this distro is:

Spc-tools
        These tools, contributed by Harold Levy, support using Log as a
        schematic capture tool for SPICE. These tools work in conjunction
        with the logspc command in Log.

I have yet to grab all of the source and make it on my Alpha. I'm hoping
I'll have some time during the week for this.

My page will be on alphalinux.org, I have a few patches to stuff that I
did, and the only software links that I will make are to stuff that has
been reasonably tested on Alpha/Linux for functionality.. so as this
page grows, it will hopefully become a practical resource for EE on
Alpha/Linux.

1. http://www.pcmp.caltech.edu/chipmunk
