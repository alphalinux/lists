No need for a port, I figured it out
tripwire-1.2:
edit make file and uncomment the CFLAGS line with the -DTW_TYPES32='int'

CFLAGS = -DTW_TYPE32='int' # DEC OSF/1 Alpha (or any other architecture
                                # where int [but not long] is a 32 bit
quantity)

Then make the correct settings for your linux box in the Makefile, and it
should compile out of box.
