Some notes:
1) It's a LiteOn chip
2) The latest tulip.c ( V.90) works with it, but there are some rough edges.
3) Donald is supposed to be working on .91 this weekend..
4) The card, so far with .90, DOES not give as good performance as a real
21140 chip bsaed model. Whether this is driver or hardware we don't know
yet. We are only just getting going with this model of chip.
