On Sun, 22 Nov 1998, Jonathan L Dubois wrote:

> I have been using DU netscape for a while now without any problems..
> but today when I tried to run it I got the following error:
>
> /usr/X11R6/bin/netscape: /sbin/loader: Fatal Error: cannot map libXm.so
>
> To my understanding its staticly linked so why would it be looking for
> libXm anyway? Besides, I had it working long before installing
> lesstif...
>
> can anyone tell me what I might have done to make this happen?

Well, install libXm.so in /usr/shlib is the easy answer... Xm is the motif
library. It must not be a statically linked version of netscape...
