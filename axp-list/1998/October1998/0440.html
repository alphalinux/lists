A possible problem I'm having getting linux running on my LX164 (I
haven't received *any* responses from my recent post) *may* be a
TGA/VGA problem. When I make xconfig, I'm shown a line which gives me
an option for TGA. The *next* line gives me the option for VGA. What
I see happens is:

     1. If TGA is set to no, then the VGA option is dead.
     2. If TGA is set to yes, then the VGA option is alive and selecting
        VGA 'y' is not possible.

If TGA is set to 'y', then .config is:

CONFIG_TGA_CONSOLE=y
# CONFIG_VGA_CONSOLE is not set

If TGA is set to 'n', then .config is:

# CONFIG_TGA_CONSOLE is not set

and NO CONFIG_VGA_CONSOLE=y

Is this my problem? If so, should I edit .config adding
"CONFIG_VGA_CONSOLE=y"?

Thanks in advance!
