There is a difference in the architecture of the 2 processors.
The i586 is a more recent processor than the 21066.
You will find that floating point performance is better compared to the
i586, but for disk/memory I/O (esp. if you have an IDE drive)
and integer computations, sure, the i586 beats the old Alphas.
Also, how much memory do you have ? I find that Alpha processors need a
lot of memory in order to perform well. 64 Megs is quite frankly a
minimum.
