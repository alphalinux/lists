On Wed, 18 Nov 1998, Ron Farrer wrote:
>Hello all,

>Is there anyone out there running KDE on their AS200? What is the
>stability/performance like?

I have KDE 1.0 on this AS200 4/233 (80 megs of RAM). These are the binary RPMs
from kde.org (the older ones, as for some unknown reason the "base" package is
completely missing from the new set on kde.org.

Some notes: As you'll undoubtedly notice, kmail is bugged when it comes to
date stamps. I apologize to the list for not upgrading the mailer already,
but I just haven't had time. :)

As for stability, kaudioserver is quirky, but the other components are pretty
solid. I have KDE running during the part of the day I'm in the lab - I quit
it when I leave, so I don't run it 24/7.

Performance is something KDE lacks. If you don't have more than 64 megs of
RAM, forget about KDE unless you like constant swapping. With my 80 megabytes
of RAM it's tolerable, but when I had only 48 megs, it was much *much* slower.

Note this is on a 233 MHz Alpha. You lucky folks with newer systems probably
don't have to worry about KDE performance. :)
