I've got a Perl Module for the "FIG" format I've been playing with --
haven't released it to the public yet, but it handles everything I
need it to do.

I use "xfig" to sketch out the "background" static stuff, then draw
more objects on it through perl like this:

======================================================================
#!/usr/local/bin/perl -w

use strict;
use Image::Fig;

my $fig = new Image::Fig(Filename => 'foo.fig'); # Load background

$fig->set(Textsize => 24, Font => 'Courier');

$fig->Text(X => 1.5, Y => 1.5, String => 'some text');

$fig->Text(X => 1.5, Y => 2.5, Textsize => 12, Font => 'Helvetica',
           String => 'some more text');

$fig->write(Language => 'ps', Filename => 'output.ps');

======================================================================

It has all the basic FIG objects, text, lines, circles, arrows,
splines, colors, patterns, etc. Should be pretty easy to figure out
for someone with basic Perl knowledge.

Email me if anyone is interested in "Beta-testing" it.
