> On Fri, 27 Nov 1998, B. James Phillippe wrote:
>
> > The binary RPM requires libstdc++-2.9 (available from rawhide.redhat.com).
> > There is also a SRPM.
>
> I installed libstdc++-2.9, and compiled a blackbox out of the SRPM. When I
> try to start it, I get the following:
...
> blackbox.cc: signal 11 caught
> shutting down

I'll bet you compiled with an older C++ compiler, such as egcs-1.0.3. I
should have been more clear:

You need libstdc++-v2.9 if you use the binary RPM.
Do not install it if you build the SRPM, unless you also upgrade your C++
compiler as well.
