I just completed a very uneventfull install of RH5.2 on my Multia.
This is my fist look at an Alpha but I've around UNIX since the
early 80's. I downloaded a RH5.2 CD image from ftp.cdrom.com
then did an FTP install from a RH5.1 i386 system. Choose either
"workstation" or "server" install and let it run on automatic.
Costomize later. I made th milo/boot/ramdisk floppies using
dd onthe i386 system.
