Ian,
There is an X server on www.xsuse.com that is for the FireGL and other 3D
cards. Check it out. Once you have the file and install it, you will have
to run the following command to get it to run with X.

remove the file name X under /usr/X11R6/bin

then under the same directory run this command

ln -s /usr/X11R6/bin/XFCom_3DLabs /usr/X11R6/bin/X

(I think that is what the file name is called)
This should get you up and running.

I have run it and it seems to run pretty well. I think it suffers from the
same problems as the SVGA 25 window problem, but you should try it just the
same.
