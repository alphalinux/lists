Hi,
   I am running Redhat 5.1 on an LX164 with a Matrox Millenium II video
card. Yesterday we had a problem with xdvi causing X-windows to fail
catastrophically. The error message in the message file was: " kernel:X:
memory violation at pc=12004fc60 rp=12004fed0 (bad address =1ffff5d8)"
This was the first time we had tried to use the magnifier feature of xdvi
since we upgraded from 4.2. I upgraded XFree86 to version 3.3.2.3-25
which fixed the xdvi problem, but our X display is still distorted. The
highest resolution, (1280x1024) which is the one that was being used when
X died, has concave sides and is off center. The lower resolutions are
still fine.
   Did the crash somehow mess up my video card? I checked some of the
config files which look okay and I have the problem with different window
managers.
   Any suggestions would be greatly appreciated.
