On Tue, 17 Nov 1998, Scott Lewis wrote:

> Thanks for the input, but it is not the CD label that concerns me
> yet, it is the label for the jewel box. One battle at a time...
> As a side note, what would be involved with creating a postscript
> template with some bounding boxes, and a small application that would take
> simple text input and put it in the right place? Fonts, and sizing would
> be a 1.1 release, I would just like to get something on these that is
> neater than my ChickenScratch (tm).

There is a software package that might help you with that. It's called
'tgif' (I have it for i386 - ftp.x.org:/contrib/applications/tgif/ ) and
it can produce .eps files with 'variables' you can define later with text
editor or your own program. :)

I've been using it to create faxcover templates, where the fax program
simply inserts the data (like To:, From:, etc...).
