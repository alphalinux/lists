Jonathan L Dubois wrote:
>
> I have been using DU netscape for a while now without any problems..
> but today when I tried to run it I got the following error:
>
> /usr/X11R6/bin/netscape: /sbin/loader: Fatal Error: cannot map libXm.so
>
This is DU's loader telling you that it cannot find the motif library.
All the DU versions of netscape are dynamically linked. And lesstif
cannot help you as it is ELF-based. loader is looking for the motif it
can recognize, and of course that'll be ECOFF. Just copy the libXm.so
from DU to /shlib, and you'll be ok.
