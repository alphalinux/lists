Ken Marin wrote:
>
> Where can I get the 2.1.129 for the as200 Alpha (RH5.1)?
Err... there is no such thing as a "2.1.129 kernel for the as200" !!
All kernels are generic and is applicable to all architectures and
platforms! Just:

cd /usr/src
wget http://sunsite.unc.edu/pub/Linux/kernel/v2.1/linux-2.1.129.tar.gz
tar -zxvf linux-2.1.129.tar.gz
cd linux
make menuconfig

<choose all your favorite kernel options>

make dep;make clean
make modules;make module-install
make boot

cp arch/alpha/boot/vmlinux.gz /vmlinux.gz-test

reboot to MILO prompt (assuming you're on MILO)..
MILO> boot sda1:vmlinux.gz-test root=/dev/sda1 (assuming your root is
on sda1)

HTH.
