Hi all,

I tried running Netscape Navigator 4.5 (complete install) under em86
emulation using the following script, as provided on this list:

#!/bin/sh
 # check for lock file
if [ -f $HOME/.netscape/lock ]
then
echo "A lock file $HOME/.netscape/lock exists"
echo "If netscape is not running, Remove this file, then try again"
exit 0
fi
export MOZILLA_HOME=/usr/local/netscape
export MOZILLA_NO_ASYNC_DNS=True
exec ${MOZILLA_HOME}/netscape -no-about-splash $*
exit 0

communicator opens to my home page, consumes 100% of the CPU, and locks. I'm on
an ethernet link behind a university firewall, if that helps. Netscape 4.04 for
Alpha runs fine under NT4.

em86 v0.4 was installed from binary from gatekeeper.

Attempting to run Netscape Communicator 4.5 on an LX164 533MHz with 128 MB RAM,
under RedHat Linux 5.1 with all appropriate updates.

Can't kill em86 either, have to reboot to regain control of my CPU.

Any ideas?
