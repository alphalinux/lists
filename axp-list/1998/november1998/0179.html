From: Dragos-Anton Manolescu <dragos.manolescu@acm.org> 16 Nov 1998

> Problems that I have noticed:
>
> i. netscape starts spawning one process after another. I don't know
> what happens, but disabling the asynchronous DNS solved it
> (MOZILLA_NO_ASYNC_DNS).

Try adding the following to your shell:
# check for lock file
if [ -f $HOME/.netscape/lock ]
then
  echo "A lock file $HOME/.netscape/lock exists"
  echo "If netscape is not running, Remove this file, then try again"
  exit 0
fi

Also, do not run the exec in background - remove the & at the end,
then add "exit 0" as the last line of the script.

This works for me on RH 4.2, I don't have 5.2 installed yet.
