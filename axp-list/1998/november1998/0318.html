If you want performence a single CPU system is always better
up to the point where you need more then one of the fastest
CPUs can do, then you buy two of the fastest CPUs you can get.

Also, you can look at the "top" display. Is the CPU pegged
at 100% utilization? If not, what good will adding a second
CPU do you?

You have to state the problem you are trying to solve. Depending
on what it is you could gain more by going to UWSCSI disks or
adding more RAM.

This machine, the one I am writting this on has two CPUs, 512MB RAM
and a pair of UWSCSI disks. I can disable one of the CPUs and
notice no diference most of the time. When it make a diference
is if I run a DBMS server and load it up with work. You
need a bunch of CPU intensive tasks all running at once to
make an SMP system worth the price. For a single user workstation
your money is better spent on one fast CPU.
