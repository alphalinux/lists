> Does anyone have the sites where I can locate Linux Quake/Quake2 and the
> patches required to run it on the Alpha. I'm curious to see how it runs.
> If anybody has any information on where to get it and any little hints or

AFAIK only a binary version is available that Linus himself ported some
years ago. For me this version does not very stable anymore since I upgraded
from libc5 to glibc. The binary should be simple to find on any Alpha-Linux
ftp server. We have it also available at
  ftp://ftp.unix-ag.uni-siegen.de/pub/os/linux/alpha/games/
  xquake-1.06-alpha-bin.gz

If you should find any other resources, i.e. for example the source ;)
please let me know ASAP ;)
