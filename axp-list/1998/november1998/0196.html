ok,
yesterday i decided to install the new rh 5.2 on my alpha. Since i dont
believe in any upgrade machanism i tried a new installation on another hd
and then copy some valuable config files by hand.
Here're comes the bug parade:

- The milo, boot and ramdisk worked fine, disk partitioning also worked

- I was not able to load a network card module to install via network
  The error was "/bin/insmod not found"

- So i switched to the shell mounted my old system and did copy
  insmod to /bin

- I was *still* not able to load a module the error was
  "/modules does not exist" or such

- I mkdir /modules and did copy the appropriate module to /modules

- Hurray! The install programm now was able to load a network module

- After i picked the packages the install programm
  "Terminated on signal 11"

- I did all this again with another set of floppies, but with the same
  result.

Also, kernel 2.0.34 was very stable and i never had a problem with it.
2.0.35 crashes at boot time when a slice was not cleanly unmounted while
fscking. 2.0.35 does not run longer than 2 hours when the box is under
load.

Any suggestions ?

best regards,
