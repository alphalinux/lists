On Sun, Nov 15, 1998 at 07:07:28PM -0800, Steve Proctor wrote:
> Anyone had any luck installing Linux (or any unix os) on the little Dec
> Multia? I just picked up one of those $99.00 specials and want to install
> RedHat on it. Whats the best way to do it? It keeps locking up when I try
> to to the install? I have tried it both ways, per the FAQ's I have found.
> Just wondering if it really works?

I had no problem at all. What kind of installation are you doing? If it's
FTP or NFS, be forewarned that the installation kernel in RedHat 5.0 (and
maybe 5.1 -- don't know) is broken WRT Ethernet drivers, so you'll need to
use the 4.2 kernel. The 5.2 kernel is OK with the network installs....
