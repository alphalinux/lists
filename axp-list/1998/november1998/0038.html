Hello

>Enter the Samsung UX....

NT works 100%. Under linux not ready for prime time yet on Linux unless
your comfortable with the following issues:

P=problem
S=solution

P)initial load on a virgin disk

S)must have a source diskette/CD of arcinst.exe to prep your hard disk.
Reason, the UX BIOS from Samsung/Deskstation does not format the drive such
that Linux can see it.

P)ethernet runs only 10MBit from a new CD install

S)must rebuild a new kernel tulip.c 090 driver. Next, edit this driver to
force it into 100MBit mode. If your on a 100Mbit network this means you'll
have to somehow get the source from another system.

P)won't run headless, (for most not a problem) if building a beowulf the UX
is not a solution unless you want to install a graphics card in every system.

S) none

While the UX is a very good motherboard, a potential owner of this MUST be
aware of these issues. If you like hands on, then you'll be happy with this
motherboard. If not wait, a new BIOS is being worked on.

One last note, if you were to purchase a complete system from a vendor then
it should be ready to plug and play thus avoiding the above listed problems.
