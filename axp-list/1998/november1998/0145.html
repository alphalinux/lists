Thanks to all who offered to run my software. I do understand
that SPEC benchmarks are not all that helpful and only testing
_MY_ code is truly meaningful. I agree, and will take up the
offers later, maybe some months from now.

The Image processing software I plan to use is based on IRAF.
This is the "800 pound gorilla" of astronomical IP packages.
To download and install this package would take the better
part of a day. I assume most of you did not include an IRAF
install in your offer. See http://iraf.noao.edu/ for more
info on IRAF.

I knew Alphas are the fastest uP chips. What I was wondering
was if they came at a premium price. For example in the
Pentium world, clearly the 400Mhz PII is quicker than the
350Mhz PII but if you divide each by the cost you will find
that the 350 has much better performance __per dollar__.

I bought a $99 multia. Slow I know, but I can use it to learn
about Alphas and develop code that others can run on their
faster machines. (If that is, I can get the darn thing to work
but that's the subject of another e-mail)
