I noticed this thread in my inbox and I thought I'd comment. Is there a
way that we can come up with a hardware / software compatibility list or
database?

Winehq seems to have a pretty good system. This would allow people to
submit what hardware is KNOWN to work on what - we can state an relavant
info such as brands / machine type / drive / bus / arch / chipset / video /
kernel / OS revision and so on, plus any code or setup 'jiggling' that
needed to be done to get it to work, and maybe even a rating on how well
the setup works and what needs to be improved. It would be cool if Open
Source Software or such can come up with a database that can shame the MS
HCL. Also this way we can say, if someone selects 'turbochannel', the
database can spit out - 'not yet supported, try BSD', or if someone makes a
patch for it - where to look for more info.

I'd certainly contribute data, but I'm not a database programmer type,
don't know how. Are there any takers? Should this suggestion be made to
the wider Linux community - call it the LCL?
