Here is a small fix to xpdf - free PDF viewer. The current
xpdf-0.7-2.src.rpm, which can be found on powertools-5.0,
is unfortunately useless due to off-by-one bug which prevents
loadinf of any file on Alpha (on Intel maybe one gets lucky
and stack survives).

Instructions:

- get xpdf-0.7-2.src.rpm from powertools-5.0
- do 'rpm -i xpdf-0.7-2.src.rpm' and go /usr/src/redhat directory
- uudecode and feed into 'patch' diff file below; it should apply
cleanly
- type 'rpm -ba SPECS/xpdf.spec'
- go to RPMS/alpha directories and install resulting .rpm files.
- save these, and a corresponding .src.rpm from SRPMS for a future
reference
- get some .pdf file to test your efforts.

Cheers,
Michal
