Jeffrey N Woodford wrote:
> I now have another question, concerning disk partitioning. The Linux
> setup program requests that I create a separate Linux partition and a
> swap partition. However I already have Windows NT 4.0 on this
> computer and I don't want to lose any of the information that's
> already on there. I tried to create a new partition using the NT Disk
> Administrator program but it wouldn't allow me to create a new
> partition (the option was always greyed out in the menu). The help
> files say that I have to "select free space" on the drive and create a
> partition from that, but it doesn't say how I go about selecting free
> space using the Disk Administrator program.

I'm willing to bet the "create new partition" option is grayed out
because you are already using the entire hard disk for your NT
partition.
There's no more free space on the drive for you to create an additional
partition.

There's only two choices for you in this situation:

(1) Back up the drive, nuke the NT filesystem, then
re-partition the drive and reinstall NT

(2) Install a second drive just for linux.

If I were you I'd try to go with option (2). Drives are pretty cheap
these days 8-)
