I have a bunch of PC164LX's with DE500 ethernet cards that I need to
get working at 100mbit and RH 5.0.

The tulip-bugs mailing list archive tells me that I need to upgrade
the driver and make one small change. That's easy, but what I don't
have is a /usr/src/linux/.config which matches the kernel that RedHat
provides. I attempted to make one up; it works with the driver
(tulip.c 0.83) distributed with RH5.0, but not with the new (0.87)
version -- I get crashes with "cannot fulfil kernel paging request"
when the driver is initialized.

working, so perhaps I didn't build a kernel properly. Can anyone send
me a working config for this system?
