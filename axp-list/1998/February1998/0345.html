On Sun, 22 Feb 1998, Joakim Roubert wrote:

> I got the jdk src-rpm, and wanted to build it with the normal
> rpm -ba /usr/src/redhat/SPECS/jdk-1.1.5.spec
>
> But, I got an ugly message:
>
> =====
> octavia:~# rpm -ba /usr/src/redhat/SPECS/jdk-1.1.5.spec
> Package lookup failed: -p /sbin/ldconfig
> Build failed.
> octavia:~#
> =====
>
> I have the /sbin/ldconfig, but I can't find any documentation; neither
> man-pages nor doc-files. :(
>
> In the SPEC-file I find the following lines (the last ones):
>
> ======
> %post -p /sbin/ldconfig
>
> %postun -p /sbin/ldconfig
> ======
>
> What is this supposed to do?

Initiate the libraries. If you have a normal (redhat) system this file
MUST be present.

Make sure you have the dynamic library stuff or suffer with things like
this.
