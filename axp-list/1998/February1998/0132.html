> After numerous hours and finally getting em86 installed, kernel
> patches, etc. I did it. At least I thought I did it. Netscape comes
> up until I touch it. I can move the pointer over it and the decriptions
> in that status window at the bottom change but as soon as I click,
> poof! Netscape is no more :(

This is a problem with Netscape that em86 can't fix. What happens
is that when you touch the netscape window, it tries to connect to
the home page. By default, this is www.netscape.com. The netscape
home page uses java. Java crashes netscape under em86.

One workaround is to give netscape a specific url when you first
start up (e.g. "% netscape http://www.redhat.com") then go into
"Options|Network Preferences", select the "Languages" tab, and
disable Java and Javascript.
