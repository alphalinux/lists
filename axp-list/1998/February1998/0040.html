Hello,
I'm trying to install RedHat Linux 4.2 on a Digital Alpha PC164LX
machine (21164 chip, AlphaBIOS 5.60) using MILO. However, nothing I
try seems to work. Below is what I've tried so far. If you have any
suggestions, could you please e-mail them to me? Thanks in advance!

Here is what I've tried:

1. Used the MILO diskette provided by the manufacturer.

Result: Computer hangs, never got to MILO prompt

2. Installed the MILO image pc164.img from the RedHat CD onto floppy
using rawrite (according to directions in manual).

Result: Garbage at the MILO prompt, unable to execute instructions.

3. Downloaded a new version of "milo" from the DEC web site.
ftp://ftp.digital.com/pub/Digital/Linux-Alpha/Miniloader/latest-Images/AlphaPC164LX/milo
Also downloaded the latest version of linload.exe (version 1.5)
ftp://ftp.digital.com/pub/Digital/Linux-Alpha/Miniloader/alphabios/linload.exe
Copied them onto floppy manually (not using rawrite).

Result: Garbage at the MILO prompt, unable to execute instructions.

4. Downloaded "milo.dd" from the DEC web site.
ftp://ftp.digital.com/pub/Digital/Linux-Alpha/Miniloader/latest-Images/AlphaPC164LX/milo.dd
Installed this image onto floppy using rawrite.

Result: AlphaBIOS gives "Invalid Operating System Selection" error.

5. Downloaded the PC164LX MILO image file for RedHat 5.0.
ftp://ftp.digital.com/pub/linux/redhat/redhat-5.0/alpha/milo/images/lx164.img
Installed this image onto floppy using rawrite.

Result: Garbage at the MILO prompt, unable to execute instructions.

6. Used linload.exe and milo from #3, and also copied milo.dd and
fmu.gz obtained from #4.

Result: Garbage at the MILO prompt, unable to execute instructions.

7. Formatted unconditionally the diskette and tried #3 again.

Result: Garbage at the MILO prompt, unable to execute instructions.

8. Downloaded another version of milo. (an older version)
ftp://ftp.digital.com/pub/Digital/Linux-Alpha/Miniloader/v2.0/2.0.30/lx164/milo
Copied it manually onto floppy (no rawrite).

Result: Garbage at the MILO prompt, unable to execute instructions.

9. Downloaded a new MILO image from the DEC web site.
ftp://gatekeeper.dec.com/pub/DEC/Linux-Alpha/Kernels/milo-2031-lx164
Installed this image onto floppy using rawrite.

Result: AlphaBIOS gives "Invalid Operating System Selection" error.

Thanks for your help!
