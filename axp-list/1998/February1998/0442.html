as of yet there are no char fetch/store instructions for the alpha gcc compiler ( for the
21164 processor there are instructions, but compiler hasn't been educated yet )

u in essence have to:
1) fetch a long
2) isolate the char in the long
3) test the character.
4) do ur branch
( it is much better written in the assembler book )

for storing u have to
1) fetch the long
2) mask out the char in the long
3) or in the new value
4) store the long
