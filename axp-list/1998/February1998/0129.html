The problem you might be experiencing and the reason why Netscape is exiting
could be due to the fact when you start up Netscape, it tries to connect to
www.netscape.com, and load java applets there. Java does not currently work
with em86 (or AFAIK) so when you start up netscape, you need to immediately
disable java.

If you have a fast link, you might need to unplug your ethernet to be able
to get into the options and disable java applets.

Hope this helps!
