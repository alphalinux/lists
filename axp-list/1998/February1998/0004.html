>
> Compilation on Alphas for Alphas seems to be slow. Thus it takes
> my 21164a/533MHz on LX about 6-7 minutes to compile a Linux kernel,

Hm, your machine seems to be doing quite a few other things as well.
On few various LX machines I tried a compilation time for a kernel
consistently varied between 4 and 4 and half minutes - depending
on options and few details. Do you have IDE drives?

'make -j 3 boot' may help a bit.

> which is about the same time it takes to compile on a Pentium-II 266MHz.

You are comparing (compiling :-) apples and oranges. Compiling a kernel
for a 64-bit RISC machine is a totally different animal from making such
kernel for a 32-bit CISC. And optimizer is far from the same.

> Other native compilation tests are similarly disappointing.

I would rather worry about a quality of a produced code. This may
use some improvements.
