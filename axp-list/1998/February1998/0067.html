I think you're being a little unfair here. A check of past posts on this
list will reveal a LOT of interest in getting a modern browser working
under Linux/Alpha. I would go so far as to say it is the main thing
preventing many folks from moving to the Alpha platform.

DU Netscape is not a great solution as few of us have licenses, but it
does work if you do. Em86 solves not only the browser problem but helps
people who want to run things like applixware as well, but it is not
(yet) working under RH 5.0. The speed penalty is probably intolerable on
a slow machine like a UDB or noname but with affordable 533Mhz. machines
available it becomes more practical.

If the ultimate goal here is to help RedHat sell more product,
discussion of how to run popular programs under their distribution is
certainly appropriate (IMHO).
