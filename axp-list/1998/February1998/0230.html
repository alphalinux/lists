If you need/want run executables from 4.2 under 5.0 you have to keep
a set of old shared libraries around. Drop them into a separate
directory on its own, add this directory to /etc/ld.so.conf and
rerun ldconfig. It is possible that you will need to keep the previous
loader as well.

Check executables with ldd and you will see what they need for libraries.
Unfortunately Alpha distribtion does not include an old set. I think
that Intel does. Recompiling can be a better option anyway.
