> GTK? what this/that

GIMP ToolKit (I think.) GIMP uses it. control-panel also uses it to make all
the pretty little buttons you see. Well, you would see them if it didn't
crash.

It is a library that does X stuff. It seems to determine that the X server
can do MIT-SHM, but gets an error when it tries to use it. The kernel update
causes the SHM to work correctly, allowing all the GTK-base programs to
start running.
