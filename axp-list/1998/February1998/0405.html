"Brian W. Taylor" <taylor@bohr.phy.nau.edu> writes:
> RedHat-5.0 on a Jensen box. The error occurs on while installing
> the basesystem packages as is the following:
>
>
> Error installing package: package install of basesystem failed: package
> basesystem-4.9-1 is for a different architecture

'basesystem' is a noarch package. Make sure you're using the latest
installation disks, i.e. 'ramdisk.img' from the update directory.

The 'noarch' classification was only introduced lately, prior to
this only 'alpha' and 'axp' classifications were used.
