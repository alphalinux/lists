I have at this moment Netscape running on pc164lx under em86.
This is version 3.0/export - just because it works for me (tm). :-)

System - RH 5.0, with updates; patched kernel 2.0.33, unannouced
em86-0.4. I have all required library files and netscape executables
sitting in /usr/local/lib/netscape and I really run the following script,
called /usr/local/bin/netscape:

#!/bin/bash

NETSDIR=/usr/local/lib/netscape
export X86_LD_PRELOAD=$NETSDIR/gnumalloc.so
exec $NETSDIR/netscape $@

#well, we never should reach here
exit

Note use of 'gnumalloc.so' library. Netscape has assorted problems
with memory routines and an old 'gnumalloc' seems to be more tolerant
about its shenanigans then stricter default library routines.

All of this is not too speedy but it seems to work fine. The machine
in question so far is only on a small local net so I did not try
anything from a wide world, but I do not expect real problems.

I did not try Netscape email. I dislike that mailer in any case. :-)
