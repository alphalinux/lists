I just got through doing this myself....

You need to run genhdlist, in .../misc/src/install/genhdlist (where
"..." is the path to your CD image's root). It takes one argument,
which should be the same as "...".

So, if your CD image is mounted at /mnt/cdspool, run this:
/mnt/cdspool/misc/src/install/genhdlist /mnt/cdspool

What this does is to examine .../RedHat/RPMS and generate a table of
contents of all RPM files installed there. It stores this TOC in the
../RedHat/base/hdlist file in a format similar to that of
/var/lib/rpm/packages.rpm. RH's install2 program (I think) uses the
hdlist to determine which RPMs to offer you.

Or so I gather from reading code. I could be quite wrong.
