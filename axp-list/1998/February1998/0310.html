The JDK and HotJava browser are commercial, of course. But HotJava 1.1.2
appears to be free for personal use, and the binary JDK has always been
free.

I installed the Linux Alpha JDK ported from Sun's sources by George
Baeslack:

http://www.voicenet.com/~gatgul/JDK/java-alpha.html

along with LessTif 0.82 as a Motif substitute. It works pretty well,
but there are a few quirks, for example TextArea garbles its display.
I am using it on RedHat 4.2. I don't know if it runs at all on 5.0.

I downloaded HotJava directly from Sun:

http://java.sun.com/products/hotjava/1.1.2

However they only supply binaries for Win95/NT and Solaris. Both versions
include a JRE. I downloaded the Solaris version and extracted it on a
Sun Ultra/1, then picked out the platform-independent components. You could
probably do the same with the Windows distribution, except it wouldn't
contain the "hotjava" shell script which is a convenient way to start
the browser. All of HotJava seems to be written in Java, except for the
SSL part, which I think won't work on Linux.
