I have a Dec Alphastation 200 4/100 that came with a DEC PS/2 Mouse
part number 30-46117-01.

I replaced the hard disk that came with the machine with a new bigger
one on which I installed Red Hat 4.2. I defined the mouse as a PS/2
mouse and double checked that /dev/mouse is linked to the PS/2 mouse
port. When I fire up X, the mouse acts really strange. The mouse
cursor jumps to the top line and kinda stays there but it will move
horizontally.

I put the original hard disk with DEC UNIX back in and the mouse works
fine!

I was able to get a serial mouse to work on the serial port, but I'd
really like to use that port for other things and use the PS/2 port
for the mouse.

Has anyone else seen this problem? Does anyone have any suggestions as
to how to go about fixing this problem?
