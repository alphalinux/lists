Please use the following for building the latest MILOs:

1. Red Hat 4.2 (it's possible under 5.0, but you have to fiddle a bit).

2. pristine linux-2.0.30 sources plus the patches from:

gatekeeper.dec.com:/pub/DEC/Linux-Alpha/Kernels/alpha-patches-2.0.30-0.4.gz

3. MILO sources from:

gatekeeper.dec.com:/pub/DEC/Linux-Alpha/Kernels/milo-sources-2.0.30-0.4.gz

4. Answer YES to the "Rebuild the PALcode from source" question.

Good luck.
