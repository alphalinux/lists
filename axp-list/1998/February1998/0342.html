> Is anyone out there burning CD's off a RedHat 5.0/Alpha 164lx system?
> If so what software are you using (will the cdwtools package work?) and
> what hardware?

I've been burning CD's with Teac's 4x/12x CD-R (CDR-4120 series)
drive in a RH 5.0/164LX box. My SCSI-controller is Adaptec's
3940UWD, and the software I've been using is Joerg Schilly's
cdrecord 1.6a10. No problems so far.
