We want to upgrade the RAM in our Deskstation Ruffian system.
The motherboard is RPX 164-2, processor is 500 MHz 21164.

We want to add 128 MB DIMM modules.

Can someone recommend suppliers and specific DIMMs to order?
Can you use the same SDRAMs you put in an x86 motherboard?
The RPX 164-2 MoBo Manual sort of recommends 72-bit DIMMs
(vs. 64-bit), since these allow ECC -- anyone has any experience
with 72-bit vs. 64-bit?

The MoBo Manual does contain a list of suppliers, but we wondered
if there are others out there, maybe cheaper? Please note that
we're only interested, at this point, in the 128 MB DIMMs, not
the smaller size ones.

This question was kind of asked before, but it (a) concerned a
different MoBo, and (b) no clear answer seems to have been posted.
