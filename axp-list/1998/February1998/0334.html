The kernel at gatekeeper sx164-31oct-2030+.gz works fine for
my Alpha PC164SX, but now I want to compile such a kernel my self
(I need an ethernet driver, apparently not compiled in).

The kernel seems to compile cleanly, using RH4.2 with GCC 2.7.2.1.
