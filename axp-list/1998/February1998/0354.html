You may be out of luck. This sounds like a close relative of the
PowerStorm 3D30 that I have, which seems to be TGA2 rather than TGA.
Check /proc/pci to see if it says TGA or TGA2.

If it is TGA, there is a TGA server with the version of XFree86
distributed with RH5. If it is TGA2 then it seems not to be supported
by XFree86 unless someone has other info.

Note: there is a VGA enable jumper on my card which does NOT seem to
make it look like a VGA device so I am not sure what the jumper is
meant to do?
