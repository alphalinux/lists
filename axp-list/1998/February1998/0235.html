Hi !

Some people do not trust in menuconfig.
The only reason why this is not working on alpha is the "unset"-series
in arch/alpha/config.in. make menuconfig simply _must_ reread the file while
running and this overrides some selections.

menuconfig is not broken; config.in is heavy broken.

Who ever had the fancy idea with those unsets should remove them and
accept that other people (like me) don't like "make config".

If the unsets are really important try it this way

if [ -z ${xx:=} ] ; then # found the smiley ?
xx=blah
# put unsets here
# if they are really required
fi

Also replace xx, I suggest "thanks_stefan" ;-}
