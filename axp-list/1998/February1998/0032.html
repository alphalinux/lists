I got the KDE beta 3 running on my RH 5.0 system and it seems to run OK.
As a desktop it is a very nice package.

It is not, however, a solution to the "lack of a decent browser" problem
with Linux/Alpha. The browser feature works reasonably well but locks up
occaisionally (getting quotes from nasdaq.com for example) and,
obviously, it lacks java support.

Is there any solution other than Netscape for a java-enabled browser
under Linux/Alpha?
