>>> Mark Brown said:
> >
> > What I ask is: Is there any official or non-official graphics suport under
> > Linux for PowerStorm 4D20??
>
> You may be out of luck. This sounds like a close relative of the
> PowerStorm 3D30 that I have, which seems to be TGA2 rather than TGA.
> Check /proc/pci to see if it says TGA or TGA2.
>
> If it is TGA, there is a TGA server with the version of XFree86
> distributed with RH5. If it is TGA2 then it seems not to be supported
> by XFree86 unless someone has other info.

Nope; TGA2 is not currently supported under XFree86.

> Note: there is a VGA enable jumper on my card which does NOT seem to
> make it look like a VGA device so I am not sure what the jumper is
> meant to do?

Uh, actually, that "enable" jumper is there so that you can install multiple
cards, and *disable* the VGAs on all but one. The VGA (simple Cirrus with
little memory) is there primarily for console text mode support, and is NOT
intended to run X. It *may* be able to run the MONO XFree86 server, but the
real Cirrus support in XFree86 has not been ported to Alpha, AFAIK.
