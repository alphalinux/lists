Christian Worley wrote:
>
> What's the milo boot command for booting over the net from a BOOTP server?

There is none. MILO currently doesn't support network boot.

I'm currently working on adding bootp support to aboot. Adding such
support
to milo is problematical because of the issue of network drivers. There
are none currently in milo, and adding them would only bloat things up.
In aboot, I can rely on the SRM console for low-level device support.
