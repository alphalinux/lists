In case you already got a working system, including aboot, it's the
most easieast thing to install a new version. Install 'jensen.gz'
somewhere on you filesystem e.g. '/jensen.gz', on the same disk aboot
is installed on.

Use aboot to load and start it. At the aboot prompt type

<p>/jensen.gz root=/dev/fd0 load_ramdisk=1

where <p> is the partition number of the /-filesystem.
Insert a disk containing 'ramdisk.img' if being asked for instering
a ramdisk.

> There is no pseudo-boot image :( or is jensen.img ?

Yes, that's it.
