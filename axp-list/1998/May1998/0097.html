It seems that the reason the network card won't work in the avanti
kernels I compile is that the base address is not getting set correctly.
The avanti-s kernel from RedHat uses a base address of 0x8000 while the
ones I compile wind up trying to use 0x10100.

I think this behaviour is controlled by pci.h but it is not obvious to
me how to correct the problem.

Any suggestions would be greatly appreciated.
