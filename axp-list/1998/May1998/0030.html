I still have those strange problems using PLIP on a AlphaStation 200 4/166.
As far as I can see the root of the problem is the assigned IRQ to the plip
module.

If I insert the module without parameters it tells me that it takes the
"assigned IRQ 5". After playing a while with ifconfig, route, ping (without
any succesful connect) the IRQ counter in /etc/interrupts is still 0.

Then tried "isacfg --all" on the Digital Unix console and it tells me, that
the lpt1-port has IRQ 7.

Back in Linux I tried
insmod plip io=0x3bc irq=7
ifconfig ....
and after these two commands my Alpha completely locks up. The only option
is to use the power switch.

Is there anybody on the list who knows more about the IRQ settings on a
AlphaStation?
