>>> Harvey J. Stein said:
>
> Is anyone selling 21264 based systems?

None that I know.

> Do they run Linux (yet)?

I have a set of 2.0.32-based patches that would work single processor and
single PCI bus. I believe the most recent 2.1.xx kernel support for the
DP264 is still limited to single PCI bus (actively being worked) but would
prolly run SMP, albeit a bit unstably.

> How good is gcc or egcs on them?

Richard Henderson worked out some 264-related scheduling patches for EGCS
back in the January/February timeframe; I would expect they've made their
way into their sources by now, but have no idea if they've been tested.
