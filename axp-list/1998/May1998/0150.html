Try resetting your memory size a little smaller in milo before booting and
see if the paging errors go away:

milo> set MEMORY_SIZE xxx

Where xxx is your memory size in MB minus 2-4MB. I had this problem in RH4.2 but
not in RH5.0 and used 252 for a 256MB system and it cured the problem. Since
it's also gone in RH5.0, if you're not already there, might try that. Using
a newer milo is also supposed to fix it.
