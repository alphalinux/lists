This problem come from broken (or shorten?) alpha/RedHat/base/hdlist.
If possible, try to remake hdlist by using alpha/misc/src/install/genhdlist.

To create hdlist.

$TOP means redhat top directory (ex. redhat-5.1/).
# chmod u+x $TOP/alpha/misc/src/install/genhdlist
# $TOP/alpha/misc/src/install/genhdlist $TOP/alpha

To check the hdlist
# chmod u+x $TOP/alpha/misc/src/install/dmphdlist
# $TOP/alpha/misc/src/install/dmphdlist $TOP/alpha

======
FTPed.
-rw-r--r-- 1 xxx xxx 268673 May 25 15:22 hdlist

Newly created.
-rw-r--r-- 1 xxx xxx 2698944 May 28 10:38 hdlist
======

Using this file, we succeed to install redhat 5.1.
