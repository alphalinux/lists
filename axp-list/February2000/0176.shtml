just copy linload.exe and the appropriate MILO image to your FAT partition.
This can be done using the cp command from linux, or with a drag&drop from
NT. Then set the boot parameters as shown in
http://www.redhat.com/mirrors/LDP/HOWTO/MILO-HOWTO-5.html
On my 164SX that I installed RH 6.1 on in December, I just copied the files
to the FAT partition ( /milo ) from NT, added a new boot selection in
AlphaBIOS, and installed WITHOUT having to use ANY floppy disks.
Make sure that you DO NOT put the linload.exe or MILO image in a
subdirectory on your /milo partition.
