On Tue, 22 Feb 2000, Andreas Vetter wrote:

> To switch permanently from SRM to AlphaBios type in SRM:
>
> >>>set os_type nt
>
> Was this your question?
>

On an LX you can type that as long as you like, but if SRM is loaded it
won't make any difference. The LX only has room for 1 firmware image, if
you wish to change to AlphaBIOS, do as you did below then use the Upgrade
AlphaBIOS feature (or, if you like you can use the Failsafe bootloader,
just cap CF7 on the motherboard).

--rdp
