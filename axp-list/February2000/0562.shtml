Yes, Linux runs just fine on a 7x00 white box, its a NT only version of the
4x00 rawhide. I would recommend you use the SRM console. Which will
support running Linux on the white-box systems. To switch,

1. On power-up, hit F2 to get setup menu's
2. select CMOS Setup,
3. Hit F6 for advanced CMOS setup.
4. 1/2way down the screen are the console settings
5. select SRM Unix.
6. Hit F10 (save)
7. follow on screen instructions then power cycle.

(NO It will not run Unix or VMS, just the SRM console)

To switch back to AlphaBios (if the need arises)
at the SRM console prompt ">>>"
type "set os_type nt" without the quotes and power cycle. the machine will
then power up in AlphaBios.

NOTE.. that switching the console messes up the toy-clock and DEC in their
wisdom to save flash space, removed the
DATE command from SRM, so you have to set the current date the first time
you boot into Linux.

You may be able to reset the clock during the install by ALT-F#ing a virtual
console and issuing the command
/sbin/hwclock --set --date "MM/DD/CCYY hh:mm:ss"
(Path may not be correct during the install-- look for it)
