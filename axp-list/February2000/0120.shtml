OK,
# mkisofs -a -r -J -l -L -o cdrom.iso /download
is the command line I use.
that makes an image that has RockRidge and Joliet extensions, is ISO9660
Level 3 and includes "hidden" files.

The only advantage to adding Joliet directory extensions is that it is
readable on a windows system, however it is very little overhead to make a
disk that is both, and if you HAVE to read it on a Windoze machine, you will
be able to get your entire filenames.
