> If you want to boot directly off the Adaptec , you should have kept your
> AlphaBIOS on it.
To go to the AlphaBIOS on PWS from the SRM one need only type 'ARC' and it
will load the AlphaBIOS stuff. To permanently have it boot to AlphaBios you
have to change the 'set_OS=NT'. MILO works fine with the Miata, although I
have Qlogic so I can (and have used both) methods of booting.
