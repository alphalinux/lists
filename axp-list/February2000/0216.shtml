> I had already been able to boot MILO, it's the installation images that
> are confusing---not that I cannot tell which one to use, but rather, how
> to launch the install.

Oh, OK, which version of RH are you using? 6.0 and newer have just one
generic kernel image. So you can use dd (or rawrite under dos/win) to
write the images/generic.img file to a floppy, you'll need to do the same
for the images/ramdisk.img file.
 
After that, start MILO as you have been. Then at the MILO prompt:
MILO>boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1

with the kernel floppy in the drive, when prompted put in the ramdisk
floppy...and away you go!

> I think---from what I can gather from the documentation---I will have to
> add a boot option to the boot menu (this machine has the ARC console mode
> enabled right now) which will boot the installation image. Does this
> sound correct? At any rate, I think you have given me enough direction
> that I can get this going. Thanks for the help.

ARC will start linload.exe that's loads MILO, you can use the Options line
of the ARC boot selections to pass parameters to MILO. This is most
usefull after the install.
