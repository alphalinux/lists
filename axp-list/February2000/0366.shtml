I've been searching online archives for a straight answer and haven't found
one, so I'll ask a (probably) stupid question:

I've just acquired a DEC Personal Workstation 433a with 128MB of RAM and an
Adaptec 2940UW SCSI card (and two 2Gb SCSI drives). I've upgraded the
firmware to the latest version and switched from AlphaBIOS to SRM. My
reading indicates a lot of people have a lot of problems because the 2940UW
is not recognized by SRM.

[N.B. I've got an older PW433a that should exhibit the Pyxis chip problem.
That's not what this e-mail is about. I'm not experiencing any errors with
the current PCI cards and don't plan on adding any more.]

Here [at last] is the question: "Can I actually get this thing to work?" or
in other words: "Do I have a lovely DEC doorstop here?"

I don't need to know exactly *how* to get it to work: I can figure that out.
I just want to know if I'm wasting $79.95 buying RedHat Deluxe 6.1 for a
machine that will never be able to run it.

My reading tells me I might be able to boot from an IDE hard drive (with the
latest version of SRM). I could pop an IDE drive in it, would that work (I
know I can boot from floppy and IDE CD-ROM)? I could yank the SCSI system
altogether (for use in an Intel PC) or use it as non-bootable data space.

I've also read that IDE performance might not be great (DMA not
implemented?). Is this true?

I know there are a lot of these machine out there. How have some of you
handled this.

Thanks for your patience with this tired line of questioning.
