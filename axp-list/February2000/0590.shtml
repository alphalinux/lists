> On sx164, what is the advantage switching to SRM ?

I have an sx too.

With the end of AlphaNT, AlphaBIOS is EOLed too, so whatever
new will come out requiring low end support from OS, I expect it
will only be able to get it with new versions of SRM.

Besides, Linux is the only OS which uses AlphaBIOS (except
AlphaNT) - all *BSD can only run with SRM, so if you'll want to
experiment with SRM, it's better to install it from the beginning.
