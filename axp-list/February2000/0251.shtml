>Hi all.
>
>If someone has recommendable web server benchmark program for
>Alpha-Linux, please let me know that is most popular & useful
>except SEPC95..
>Actually, I would like to get a compare test data between Alpha (new
>designed by Samsung) and x86 (P-III) machine.

Try ApacheBench (ab), which is a part of the Apache distribution.
Alternatively, try httperf
(http://www.hpl.hp.com/personal/David_Mosberger/httperf.html)

I have done a benchmark using httperf:

Xeon 500 cabrio - 1128 req/s
UP2000/667MHz/2M - 1536 req/s

The above probably isn't a fair reflection of the current state of
affairs. You should benchmark with dynamic content to get more
useful results.
