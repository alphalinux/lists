The "PWS 433a" is mentioned in the list of supported platforms
for both SuSE 6.1 and SuSE 6.3,
I cannot tell about RedHat.
The "Adaptec AHA 294x" is mentioned as a supported SCSI adapter,
no special reference to "wide" vs. "narrow"
(SuSE 6.3 - did not check 6.1).
SuSE supports both SRM and ARC console.

Recently, there were several postings here regarding install
problems with RedHat 6.1 - if that worries you, try SuSE.
 
My own machine is completely different (AStation 200 4/166 using ARC,
NCR53C8xx SCSI adapter; running SuSE 6.1), so this is from the
book only (and not from personal experience).
