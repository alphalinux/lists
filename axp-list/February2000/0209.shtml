On Sat, 12 Feb 2000, Tereknor wrote:

>
> Is there a way of getting SSH1 to talk to SSH2? I have a windows ap
> that talks SSH - but only SSH1. I would like to use a better version.

Nope, all you can do is have both ssh1 and 2 installed on the server, with
sshd2 running. If configured for ssh1 compatibility, sshd2 will spawn a
version 1 daemon when it sees an incoming version 1 connection. BUT--you
do need ssh1 to be installed as well for that to work.

Kinda hokey in my opinion--version 1 has always done what I needed it to
do, so I've been sticking with it.
