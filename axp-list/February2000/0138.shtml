I finally got this one working...you'll need the newest stable kernel on
rawhide (2.2.14-1.3.0), which has the necessary fixes for the ES1371.
Kudzu will recognize it on boot and will add the necessary alias
conf.modules.

Very nice audio, but to get MIDI support, download the timidity package
while you're at it.
