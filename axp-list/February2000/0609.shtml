The X server can blank the screen by itself.

The xset program allows you to change the parameters that will invoke the
screensaver (e.g. how many minutes, etc.). It just runs long enough
to send the parameters to the X server: the X server itself then does
video blanking.

There is also a screen saver extension, which I believe allows
client programs to be notified if the screen saver is invoked; this
allows for fancy screen savers with fancy graphics.
