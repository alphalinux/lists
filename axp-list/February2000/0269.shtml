> Thanks for the reply

> I am trying to install MILO from floppy. Used rawwrite from Linux cd to copy
> linload.exe and milo image to floppy. This is machine that I have 2 ether
> net cards that I can connect to network.

> venu

Are you using rawrite to write a MILO disk image to a floppy? (this image
file would probably have a file size like 1474560)

You don't want to use rawrite to put linload and milo files themselves on
the floppy--instead, what you need to do is (assuming a Windows machine)
open a command prompt and type "format A: /u" and then "copy linload.exe
milo A:"
