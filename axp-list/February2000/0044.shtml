I had an Acrobat Reader on my Tru64 box (Acrobat3 was the directory
I had installed in /usr/local). I tar'd up the directory and unpacked
it on my Linux/Alpha box where I had installed the Netscape RPM.
Running the 'acroread' script produced and error about not being
installed on 'intellinux'. I modified the script to use 'alphaosf'
instead of 'intellinux' when setting the ACRO_CONFIG and up came
Acrobat Reader! I'm reading a PDF file now with it!
