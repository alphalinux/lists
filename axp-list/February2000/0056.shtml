Hey there everyone,

I finally have the KICKSTART working on the Alpha! Figured if anyone
else was looking to see how it was done, I would fill them in. The
complete document will be posted to our web site at www.dcginc.com if
you need more details.

Needed:

1. Redhat Ramdisk
2. Floppy with a kernel on it

To start:

Unzip a RedHat ramdisk to /dev/ram1 with the command:

dd if=/dev/fd0 | zcat | dd of=/dev/ram1

Then mount up ram1 with the command:

mount /dev/ram1 /mnt/ram1

Then copy a ks.cfg to /mnt/ram1/tmp.

umount /dev/ram1

Rezip the ramdisk back onto the floppy:

dd if=/dev/ram1 | gzip | dd of=/dev/fd0

Then from the SRM/MILO prompt type the command

boot dva0 -file vmlinux.gz -flags "root=/dev/fd0 load_ramdisk=1"

Insert the ramdisk when requested and you should be off and running.

Hope this helps someone out.
