> There is a utility for handling tape libraries. I think that it
> is called 'mtx' (extended mt?) although I am not 100% sure. It should
> be easy to find with a help of various search engines or maybe on
> freshmeat. SCSI tape changer commands are standard AFAIK.

mtx is for very small tape changers (1 device, 6 slots).

The program for real tape libraries is changer, written by Gerd Knorr
http://www.in-berlin.de/User/kraxel/linux.html
Version 0.14

It should work well with real libraries (4 MO's, 88 slots)..
