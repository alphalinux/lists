On Mon, 7 Feb 2000, Wingell, Joshua wrote:

> The only problem is that I have 2 devices on the computer that I
> am not certain have great support on Linux.
>
> I have an HP ScanJet 4P SCSI scanner and a Yamaha 4260 SCSI CDRW
> drive.
>
> These are both plugged into a SCSI UW Adaptec crontroller.
>
> Are there programs that will allow me to scan?

Yep - you need the 'sane' package - see http://www.mostang.com/sane;
that will allow you to scan (I use it with a ScanJet on my parallel port).
There are also plugins to go and pull stuff straight into the GIMP image
firtling program.

> Are there programs that will allow me to use my CDR (preferably
> with a GUI - I'm lazy)?

I have heard of things called cdwrite and the like - but I've not got a
burner; search out www.freshmeat.net and have a wonder around there - it
has loads of Linux software pointers.
>
> This computer is mainly used as a gateway for my 2 PCs. I have IP NAT
> setup on Windows 2000 (it is built in). Linux has IP NAT as well,
> right?

Yep, I use it as home; look for 'IP Masquerading'.

> Oh, I will also need support for my Ethernet card. It has a Digital
> networking chip on it, so I am pretty sure it should be well supported.

Yep, the Tulip drivers probably.
