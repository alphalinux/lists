Hi,

Praveen Srinivasan wrote:
> It sounds like you might not have the expat stuff. If you put the expat stuff
> (available at abisource.com) in the same directory as the regular abiword stuff, it
> should compile it automatically.

Just a nit... expat should be a directory that is on the same level as
abi.

> I've succeeded in getting Abiword to compile and
> work, although it has some problems (spell check doesn't work).

Yes. This is because of 2 reasons:
1. the dictionary is a i386 version - by this I mean it seems
   to be built with all sorts of i386 specific data size
   assumptions like the size of int and char * being 32 bit.
2. the ispell code also makes these assumptions and is reasonably
   suspect when it comes to working on a 64-bit machine. I am
   trying to work on it and I also know that ispell has is available
   on Linux/Alpha.
