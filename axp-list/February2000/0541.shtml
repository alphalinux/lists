I gotta wonder if you have a problem that's more scsi than disk.

Wait a minute...bright flash...if you're running NT from another
disk, you must be using ARC f/w and MILO to load linux. The primary
loader and MILO live on a tiny dos partition in that case. e2fsck can't
check that, and would rightly gripe if asked to. If you can boot even
single-user, you should be ok there.
