Has anyone had success at installing 6.1 on a jensen?

I've gotten to the point of running the 6.1 installation. to get there:
- I've rummaged through many different web sites and archives.
- discovered JENSEN_INSTALL moved on dec's site to ARCHIVES, this
  broke a lot of good jensen web pages.
- got aboot installed and working with 1.7 firmware upgrade, using
  the great minlable and copyboot utilities for ARC.
- using the ramdisk.img from the alphalinux page jensen instructions I can
  get redhat-6.0 to seem to install (it's running the install right now).

Before that, I retrieved ramdisk.img from the 6.1 distribution and
burned the 6.1-alpha ISO image to a cd -- I tried installing from
ramdisk.img and from the cd itself. The install loads and I get through
the fdisk portion and the install completely hangs just after the "set root
password" screen. Nothing in the syslog to suggest anything is wrong
or what it's up to.

I did notice a few things about the 6.1 install:
- dec422 ethernet driver doesn't work. it does work on the 6.0 ramdisk.
- X install doesn't work, this is mentioned on this or other lists, had to
  install 'expert text' in the boot flags.
