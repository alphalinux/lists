On Mon, Jul 09, 2001 at 05:43:03PM +0000, mlandrier@ifrance.com wrote:
>
> Is there anybody who can help me to install Linux on
> ES40 ?

Probably.

> I have tried to use jmpstart (a package from compaq)
> with the result : kernel panic.
>
> When i run jmpstart from alphabios, jmpstart say that
> he can't find a milo configuration for the ES40.

THere is no MILO for ES40 - you MUST use SRM console.

> When i boot with RedHat 7.0, Redhat can't find my
> disks ?

As mentioned by someone else already, if you have a DAC960, then it
must have firmeware rev 3.0 or greater to work with Linux.

> My ES40 have 4 processor and 4GB of memory.

There's your main problem - too much memory... :-)

RH 7.0 will NOT install with more than 2GB of memory, unless you
add "memsize=2G" to the boot command flags.

Or you could try RH 7.1; it uses a 2.4 kernel which will handle
as much memory as you can install.

Good luck.
