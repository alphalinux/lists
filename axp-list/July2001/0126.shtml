If the main program you use is already a Tru64 executable, it may "just
work"
by installing the 'netscape rpm' package of Tru64 libraries/loader/etc.
stuff.
One way to get a fair idea about this is to check the libraries which the
executable requires on Tru64:

        rlogin tru64_box

        $ odump -Dl foo

        <list of libraries needed by the executable appears>

If your list doesn't include things like threads, or DECwindows code, you
are probably in good shape.

I hope this helps.
        Rick
