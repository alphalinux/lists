>> You can find a number of Permedia 2 based cards
>> for around $40 on Pricewatch. VooDoo 3 PCI cards

>
>However, there are problems with some combinations with the
>Voodoo3-3K PCI card (and the TNT2 Vanta card) with XFree 4.
>At least in my case, UP2000 running RH7.0, I have yet to
>get 1600x1200 at 24-bit color working on either of the

>addressing correctly, resulting in a garbled display but
>with the mouse cursor displayed flawlessly. (The mouse
>cursor's flawless display and the fact that 16-bit color
>works perfectly with the TNT2 card imply the monitor is
>fine with this resolution.)

>Disclaimer: The most recent XFree 4 package I have attempted
>to use is 4.0.3-9. There are rumored to be later versions of
