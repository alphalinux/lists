The relevant code is in arch/alpha/kernel/traps.c in the linux kernel
tree. It should be easy to modify, if you want to try that. Expect the
logging flood gates to open, though. I've crippled a few systems by
enabling the printing of every unaligned access (driver debugging...no
fun).
