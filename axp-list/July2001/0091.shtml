Hi,
I am having a problem with the UP2k+ board (SRM 5.8). With the following
cards are installed on the system: Oxygen VX1 / 32MB, DE500-BA ethernet,
KZPCA U2SCSI, the system boots fine and I can get to SRM prompt. When I add
a DEGPA-SA 1000BaseSX card, the machine does not boot. I have tried other
(known good) DEGPA cards, but the problem persists. The same card also
works on other UP2k+ boards. I have also tried vaious card/slot
combinations, but no improvement.
The system will need to run Linux and Tru64.
Additional symptom:
Normally after power-on, the SROM LED comes on for an instant, goes off,
then comes on for about 12 seconds while the SROM loads (I think). When the
DEGPA card is inserted, only the first short SROM LED pulse is observed.
