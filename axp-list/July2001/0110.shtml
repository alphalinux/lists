Dear Alpha-Linux users,

I have one of the "standard" questions which I could not solve
by checking the Web sites (SuSE hardware list, API):

  Which (currently available) graphic cards are supported?

  (environment: AS 200 4/166 "Avanti", using ARC-bios and Milo,
  currently running SuSE 7.0 with XFree 3.3.6;
  SW development, text and office use only - neither CAD nor games)

(If necessary, I can upgrade the software, but I prefer to stay
in the SuSE line. SuSE-AXP list seems to be quite empty, maybe
due to the summer season - no answer to this question till now.)

Until recently I used a 17" monitor (Digital VRT17-HA, 1024*768)
with the original TGA 21030 card.

   Question aside: Any way to determine its RAM size?
   (The boot messages do not list that info.)

I now got a 19" monitor (Eizo F730, 1280*1024 or 1600*1200)
and will probably need a better card (4 - 8 MB RAM) to increase
the resolution.
The main problem is of course that most current offerings, even of
supported chip sets, are for AGP slots, which the machine does not
provide.

I read that Matrox Millennium I is heavily recommended,
but this seems impossible to get by now.
 
   I found an offer for a Matrox Millennium G200
   (250 MHz, 8 MB, PCI version) - any comments / experiences?

   Any hints for other sources / other cards?

Second-hand is no good alternative, because this is a company
machine; German sources are preferred by our bookkeepers.
