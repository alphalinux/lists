Hello. I just installed RH7.1 last night on PWS. It wouldn't boot
afterwards either. I tried booting off the floppy (boot dva0 -fl
root=/dev/sdb2). that didn't work. I cam in this morning and using:
'boot dva0 -fi vmlinux.gz -fl root=/dev/sdb2' it worked. This is using
the RH7.1 install floppy. From there I am setting up the aboot partition
(aboot is still present from my RH6.2 installation of it) and my little
kernel/boot partition (sdb1 for me) to boot from the harddrive. This
should work for you (it is working here). Try it and let me know if you
have problems.
Bon CHance,
James
