I haven't tried it on alpha yet, but rh 7.1 for x86 has a screen during
installation asking if you want to configure no security, medium security,
or high security.

Medium and above will shut off connections to all inbound ports below
1024, which includes rsh, unless you specifically said to do otherwise.

Barring that, I'd check the xinetd.conf file and so forth to make sure
that rsh is enabled.

Then I would check /etc/hosts.(allow|deny)

Then I would route through /etc/pam.d/r*.
