On Mon, Jul 16, 2001 at 10:55:13AM +0200, Joerg Bruehe wrote:
>
> Which (currently available) graphic cards are supported?
>
> (environment: AS 200 4/166 "Avanti", using ARC-bios and Milo,
> currently running SuSE 7.0 with XFree 3.3.6;
> SW development, text and office use only - neither CAD nor games)

> Until recently I used a 17" monitor (Digital VRT17-HA, 1024*768)
> with the original TGA 21030 card.
>
> Question aside: Any way to determine its RAM size?
> (The boot messages do not list that info.)

If it is a small card, 2MB and 8-bit depth only. A full-size card
would be 24-bit depth only, and 16MB.

> I now got a 19" monitor (Eizo F730, 1280*1024 or 1600*1200)
> and will probably need a better card (4 - 8 MB RAM) to increase
> the resolution.

TGA can do 1280x1024 at 72Hz or thereabouts, IIRC. 1600x1200 is possible,
but prolly only at 60Hz...

> I found an offer for a Matrox Millennium G200
> (250 MHz, 8 MB, PCI version) - any comments / experiences?

G200 is reported to work well.

Critical point for ANY card you try is whether the BIOS emulator in
the MILO you are using will handle the BIOS code on whatever card you
plug in.

> Any hints for other sources / other cards?

I tried a few of the cards I have with the latest SRM console for an
AS250 4/266 and found the following:

card SRM console XFree86 4.0.3
---- ----------- -------------
3DLabs Oxygen VX1 yes yes
3DLabs Permedia 2V yes yes
ELSA Permedia 2 (OEM) no no
DEC Permedia 2 (COMET) yes yes
ATI Rage 128 no no
3DFx Voodoo Banshee no no
3DFx Voodoo3 3000 no no
Diamond Stealth 3 S540 yes no (4.1.0 yes)
ELSA TNT2 no no

Again, since you are using ARC and MILO, those have different BIOS
emulators and may behave completely different (good or bad).

If at all possible, make sure you can return a card which doesn't
work out.

> Second-hand is no good alternative, because this is a company
> machine; German sources are preferred by our bookkeepers.

I'd think with as old and slow a machine as that, you'd *prefer*
second-hand (cheaper :-).

Stay away from S3 Trio64 cards, they've been problematic on AVANTIs.

Good luck.
