the only thing that you'd have to be really concerned about is booting the
kernel. Once it's booted, you can run just about anything for a controller
that's supported under alphalinux (IDE/ SCSI/ etc).

couple of obvious (but hardly practical) choices are to:
-put aboot and your kernel on a floppy (may not fit this way)
-burn a CD of aboot and a kernel and boot that way.

I forget which AlphaServer it was, but one of them REQUIRED you to use a
floppy to boot OSF/1 on it (not just for installs... EVERY time you booted
it).

another choice would be to get a small disk and connect it to the on-board
QLogic card. You'd boot off that drive, and then use the disk connected to
the 2940U2W to run the machine from. The only problem with this is that
there isn't anywhere to put a second 3 1/2" drive in the case unless you
take the floppy or the CDRom out.
