On 31-Jul-2001 Rick van der Linde wrote:
> Hi,
>
> I can buy an Adaptec 2940 U2W with a 18 GB disk, now I wonder:
>
> Will I be able to boot from the 18 GB drive connected to the Adaptec 2940
> U2W
> on the Alphastation 500/333?
>
> Probably I'll give it a try anyway, but if it's not possible it saves a lot
> plugging.
>
You cannot boot from additional IDE controllers. But it should be possible to
boot from an additional SCSI-Interface (at least at some systems).
When using the srm console a show dev will tell you if srm has recognized your
controller. If so you can boot from it otherwise not.
If done this with an AlphaServer2000 and it worked (but I don't know the scsi
controller i used).

But you can use your disk under linux in all cases (as long it's recognized by
the kernel - and the Adaptec 2940 is).
