Hi,

One of our UP2000 Dual Alpha machines just started
having probems getting into SRM console mode after
a power on.

Here's what happens:
  1) The machine is turned on.
  2) The CPUs go on
  3) The monitor sees a video signal.
  4) The monitor screen turns blue
     with a cursor in the upper left
     corner.
Then THATS IT!!!

  o No Testing the System message
    No Testing the Memory message
    No Testing the Disks message
  o No going into SRM console mode

I have found that if I wait two minutes
and then hit the Enter key on the keyboard
the P00>>> prompt will appear and the
machine will accept SRM commands such
as boot.

My question is: What do I do to get the
machine to boot into SRM console mode
again as it used to?

Thanks for any info. --Lew
