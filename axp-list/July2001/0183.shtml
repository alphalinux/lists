It's a normal XP1000 with 667MHz and 1GB RAM. It also has a SYMBIOS (or
similar) SCSI controller.

The problem with almost all distributions is it hangs after the message

aboot: starting kernel boot/linux with arguments root=/dev/ram ....

The Debian kernels go a little farther and say:

halt code = 2
not valid kernel stack
PC = .......

I have tried booting with CD, floppy and bootp. None of them got farther
than that.

Moreover, Tru64 boots OK. I have just got a Tru64 CD and it seems to work
fine.
