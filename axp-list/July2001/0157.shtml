I was in the process of setting up RedHat 7.1 on a 164LX Alpha. As this
machine will be our campus webserver ( e.g. attached to the Internet full
time ). I thought some intrusion detection software would save me
time/trouble later on. I downloaded Tripwire 2.3.0-50 and compiled.

The first sign that something may be wrong was the directory structure that
was being created... directories and files were being created with
"i686-pc-linux" in the name ( as opposed to maybe "alpha-unknown-linux" or
"alpha-ev56-linux"). Once the build was complete, all 4 executables (
siggen, tripwire, twadmin, and twprint ) all seg fault on me.

Has anyone successfully installed Tripwire on an Alpha?
Is tripwire "not 64-bit clean"?
What other intrusion detection software are people using?

Thanks in advance...
