"Niki W. Waibel" wrote:

> pc164 freezes when network traffic occurs.
> ping (low traffic) is okay, ftp is not.

This has long been a problem with the PC164.
It just can't handle 100 MB/s nics.
The eepro100 is probably the best card you
can use with the pc164, but you have to
run it through a 10 mb/sec switch or other throttle
device to make it reliable.
