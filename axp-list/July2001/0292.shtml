> Hello, I'm trying to do Linux RAID 5 over partitons of only one disk
>drive.Can I do this?

This sounds like a troll, but I'll bite.

Why do you want to do this? There are a number of reasons why you
would want to RAID a number of disks together: Gain throughput,
gain capacity for one single logical disk, gain reliability by using
redundancy. Doing this on a single disk (if it can be done, I don't
know) defeats all of them.
