Hi all,
as for all the poor folks who don't have a SRM-konsole (like me...), I
compiled a kernel with all the required options in it and copied it to the
boot-floppy. Together with an off-the-shelf ramdisk I managed to get RedHat
7.1 installed. And it's working quite good (writing this on it ;-)
 If you want to use it: I didn't use the milo and loadlin.exe from the
RedHat-CDs. The ones given with 7.1 don't work for me. So I took the ones
from 6.2 (or so I think. It's been a good while since!)
 I put an image of the bootdisk in
ftp.alphalinux.org/incoming/boot164.img
 Perhaps it might be worth putting up somewhere? I remember someone talking
about special tricks for special machines. Well, this one works for a PC164
with MLIO and an AIC7XXX. I think it should for others, too.
