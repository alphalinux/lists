Alan Skousen wrote:
>
> I must be missing something. No doubt something obvious. I installed 7.1
> on an Alpha machine and can no longer get rsh from w2k to connect as I
> could on 7.0. Any ideas?
> Alan
>

RH7.1 turns off rsh/telnet and various other insecure connections no
matter what security level you select at install. I always select
none for the firewall and it still leaves them turned off in the /etc/xinetd.d
directory's config files. You have to delete the "disable=yes" line from
the config file and restart xinetd. Apparently saying no firewall is
insufficent for RH's install to believe you want those services started.
They also turn off the ftp server I believe and if you want it on, you
have to edit that config file too. (this was on my P4's not Alphas but
I suspect it's the same)
