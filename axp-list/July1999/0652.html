I saw it yesterday, left a message requesting two, and provided all
neccessary purchase info (CC#, ship address, etc.) since I didn't
get a person on the line.

If I get my two, great, if I dont, that's fine too - I dont need them, but
for $30, I find them hard to pass up...

I'd say that $29.95 is about the right price, and it is worth it if you
have a well stocked parts bin:

    Parity RAM (4 16 Meg SIMMs is good enough for casual use)

    Left over IDE notebook drive (the bigger the better, SCSI is best,
    if you can find it), but network boot is possible too.

    Keyboard/Mouse *may* be included, if not, there is almost always
    a free one available at CompUSA (after rebates).

    A PCI/SCSI adapter card would be nice, but a 3.5" HD adapter
    would also have value (toss-up in my opinion).

The real challenge is the lack of a floppy drive, but once you flash the
latest SRM/ArcBIOS into the unit, the floppy is pretty much ignored.

I think these would make nice X Terminals, nice in-law web boxes
(but WinNT might be a better OS for such a use - better support for
the built-in graphics adapter), and a great source of spare PS and
other misc. parts when your "main" multia dies...

Just my $.02,
