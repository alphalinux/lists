Well since then I've gotten random crashes from 2.2.10 and have
reverted to unpatched 2.2.7. X sometimes crashes about once or twice
a day, but as long as I'm not using xdm or gdm, I can simply startx
again without disruption.

Back when I was using xdm or gdm, an X crash would often completely
take out the display. Which forced me to log in remotely to reboot.

So its not a perfect situation, but not using xfs, not using gdm, and
sticking with 2.2.7 is the most stable configuration I've gotten under
2.2 on this lx164 machine.
