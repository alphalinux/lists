I don't know... I haven't used Kaffe in a long time, since it hasn't
been really stable on Alpha in a long time. For that matter, why does
RedHat continue to distribute it? Because a broken VM is better than
none?

> What can I use to debug java code????

That's been an ongoing problem, for Alpha and Intel users alike... jdb
isn't very good and nothing else exists.

I have higher hopes for the gcj project from Cygnus. Gcj is a Java
compiler which is portable and debuggable, owing to its integration with
the gcc toolset. I believe patches are available so gdb understands java.
Imagine stepping into native code from java code with one debugger... at
last it is possible.
