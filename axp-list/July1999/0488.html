This is rather funny to me... I had the exact same
problem with RH 5.0 way back when...

First of all.. I think you can still get single user
mode.. by doing at MILO:

boot dev:vmlinuz single (other stuff...)

The problem I had with RH 5.0 was that something was
wrong with shadow passwords... hence, no matter what
the password was set to.. it wouldn't auth.

You might try re-installing without shadow and MD5...

However, since RH 5.0, I haven't heard of this problem
again... so I'm not sure if that is it.. but it sounds
so familiar...

D.
