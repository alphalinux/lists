Ruffian UX4, FireGL1000Pro. no other plug-in cards
Absolutely virgin RH6.0 install.

I've given up on trying to use other window managers briefly, it does seem
that at the moment I can at least get into gnome, even if I find it a bit
clunky. I've discovered several reasons why I get so many core files:

Gnome, configuration tool, screensaver setup
Firstly "Try" does nothing at all.
Then for each screensaver which has parameters, the button to configure the
screensaver's attributes closes down the screensaver configuration panel,
and drops core.

strace
I straced xgammon last night to see what strace is all about. About 1/2 a
second later I found out - it drops core!

It really does bring back memory of UAEs and GPFs...
