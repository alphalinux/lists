Hi,
1) use vmlinuz-2.2.5-16.gz, you can copy it to the root directory.
cp /boot/vmlinuz-2.2.5-16.gz /vmlinux.gz
2) Do you add "root=/dev/..." in the booting command?
It should be like: milo> boot sda2:vmlinux.gz root=/dev/sda2
