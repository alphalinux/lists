Hello,
>> + #elif defined(CONFIG_ALPHA_LX164)
>> + # include <asm/core_pyxis.h>

>Why do you need this? CONFIG_ALPHA_PYXIS should have been
>defined by arch/alpha/config.in.

The config.in defines the variable CONFIG_ALPHA_LX164. So here
in the IO section the correct glue chip needs to be included. This came
about when I was tryin to get the PCMCIA utilities to work on the
LX164.

Note: Pcmcia does work on the LX164 just remember to set:
cat /etc/sysconfig/pcmcia
PCMCIA=yes
PCIC=i82365
PCIC_OPTS="irq_list=5,9,10"
CORE_OPTS=
CARDMGR_OPTS=
