I have a PWS500a.. and my personal experience has been
that these machines are very picky as to what kind of
memory they can use. I had bought some 10ns SDRAM w/
ECC ... and tried a variety of high quality parts
(NEC, Toshiba, Micron, etc.).. none worked... I
finally went back to the guy who sold me the machine
and bought the memory that came with the original
machine.. which was Digital using Samsung chips.

In fact, if anyone knows of a place where I can buy
memory that will work with my PWS, I'd like to know
too.
