Anyone been successful in getting XFMail 1.3 to build on an Alpha? It seems
the problem is that the latest version of the xforms library I can find is
xforms 0.88.0, while XFMail 1.3 wants 0.88.1.

I've built XFMail 1.2 using xforms 0.88.0, and it works but is rather unstable.
 To read new mail, you have to check the mail, allow XFMail to dump core, then
restart XFMail to read said mail. No mail is lost, but it's obviously a PITA
to do this. Should anyone want to try out this PITA, I've made the RPMs
available at my usual place (FTP orangesherbert.ces.clemson.edu). :)

Failing that, any good Alpha-safe GUI email clients that:
* read mail from the spool (should be easy to do :) )
* Don't require an entire "desktop environment"
* Support folders and filtering mail into folders as well as XFMail?
