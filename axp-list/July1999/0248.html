 You should use the generic-smp boot image to make the boot disk, then
use your RedHat 6.0 CD as the root.
The command is like:
boot dkb0 -fi "vmlinux.gz" -fl "root=/dev/scd0 load_ramdisk=1"

You may find more information on my webpage.
