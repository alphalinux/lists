Hello all,

Was playing with a few systems using a large fortran fluid dynamics
code that uses 5 grids to model a blood flow, I was rather impressed
with the results (low is better, minutes:seconds):

Redhat-6.0/egcs-1.1.2, fort=tru64 fortran on linux.

PII-450-512k cache g77 -O3 75:02
Celeron 450A-128K cache g77 -O3 74:44
Alpha 21164-600 4 MB cache g77 -O3 29:27
Alpha 21264-500 4 MB cache g77 -O3 17:16
Alpha 21264-500 4 MB cache fort -O3 8:42
