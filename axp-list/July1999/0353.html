This does for shure *not* work. Look at the 2.2.5 source RPM.
Beside the ~20 patches (including raid-0.90) there are also HJ's
nfs-updates included which are not in 2.2.10.

The latest knfsd (1.4.6) includes important patches that are really required
for stable NFS operation.

----------------------------------------------------------------------
This is the Linux kernel NFS daemon 1.4.6. It is based on linux-nfs
0.4.22. It is tested on Linux/alpha and Linux/x86 running glibc 2.1.1.

WARNING: The NFS servers in Linux 2.2 to 2.2.10 are not compatible with
other NFS client implemenations. If you plan to use Linux 2.2.x as an
NFS server for non-Linux NFS clients, you should apply the patches
enlosed here.

linux-2.2.7-sunrpc.patch, nfsd-2.2.7-2.lockd.patch, nfsd-2.2.7-3.patch
and nfsd-2.2.7-nfsfh.patch are recommended for Linux 2.2.7 to 2.2.10.
For other kernel versions, they have to be applied by hand if they are
still needed.
