Has anyone had success with the stock RH6.0 kernel on the Multia regarding
sound?
The kernel *should* load in and correctly use the ad1848 sound module but I
tried adding
this to /etc/conf.modules to no effect:

options ad1848 io=0x530 irq=9 dma=3 dma2=1

This has no effect. I then loaded the module manually:

modprobe ad1848

It loads in fine as an lsmod shows but I still get no audio. What could be
wrong here?
Can anyone verify this or tell me how to get sound *without* recompiling?
The whole
point of having a modular kernel is that I shouldn't have to build by hand
anymore so
long as the module is prebuilt.

Also, it seems the joystick module hasn't been included with the stock Alpha
kernel.
