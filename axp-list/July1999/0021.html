On Thu, 1 Jul 1999, Barnett Hsu wrote:

> It already has MILO installed, so I tried to
> boot the Red Hat Linux 6.0 install boot floppy
> with "boot fd0:vmlinux.gz" at the MILO prompt.
> The kernel booted, but instead of asking for
> the ramdisk, it booted the Red Hat 5.0
> installation on sda2! Next I tried at the
> MILO prompt "boot fd0:vmlinux.gz loadramdisk=1",
> but that had the same result. I checked the
> Alpha Installation Addendum for Red Hat
> Linux 6.0 and it didn't seem to say to do
> anything different?

Try:

boot fd0:vmlinux.gz root=/dev/fd0 load_ramdisk=1

note the '_' in the load_ramdisk.
