I have a copy of SuSe 6.1/AXP and all my Intel machines run
debian-unstable. I'm not really a big fan of RedHat; if you can afford
the downtime (installation is a hassle) I would recommend you go with
Debian if you're looking to migrate. I've not had the opportunity to
evaluate SuSe, so if you're looking for a system with an easier
install/maintenance, I would say to wait out RedHat until 6.1 final. I
run RedHat 5.2 on my production machine as well - so you may also want to
consider downgrading (and applying the errata) or, again, waiting for
RedHat 6.1. Personally, I'll wait for x.2 +a couple months before
upgrading RedHat.
