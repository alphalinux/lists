>>> "Song Byung-Hun" said:
>
> I have Digital AlphaServer 1000A 5/333 model

The 1000A is also known as NORITAKE, and yours is the EV5 version.

> I wish install to RedHat6.0 Alpha version
> But, I don't know, What is my server Pre-Built Standard MILO Images ???
> I'm can not search..
> this model exist?? MILO Image.. if not, How install (possibly SRM ??)

There is no MILO for NORITAKE; you must install from SRM console.

Try booting the generic kernel directly from the RH 6.0 CD, like so,
assuming dka400 is the SCSI CDROM:

>>> boot dka400 -file kernels/generic.gz -flags root=/dev/scd0

Good luck.
