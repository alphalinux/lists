Since we're talking about PDF's and Alphas these days .. does anyone
know of a good tool to make them on Linux/Alpha ..

The only way I know of is to make a postscript file (or a collection of
gifs/jpgs 1 for each page) and use Image Magick's "convert" to put them
together into a PDF.

The drawback is that IM does PDF's as graphics-only (to be expected, I'm
not getting down on IM) .. that is, every page is a bitmap in the PDF
which gets large very quickly.
