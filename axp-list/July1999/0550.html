The DS20 will indeed run with 1Gig w/o any trouble. Firt I'd recomend
getting a later SRM console for it. 5.4-19 has worked for me:

ftp://gatekeeper.dec.com/pub/DEC/Alpha/firmware/interim/ds20/

There was at some time a problem with the DS20 that if you set the console
to UNIX the memory would not get initialized properly, but setting it to
OpemVMS seemed to solve the problem (this was only evident of larger
memory systems). You could also try that before upgrading.

I've also seen that message try to boot an SMP kernel on a UP machine with
that particular SRM version.

regards,
