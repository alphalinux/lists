Greetings,
        I recently purchased a 164SX/21164A combo, and I'm problems with it,
namely occasional kernel builds fail. Under Linux v2.0.36, 100% of my
kernels build successfully, under v2.2.4 100% build successfully, and
under v2.2.9, 70% build successfully. I've done a number of things in an
attempt to ``verify'' that it is indeed the kernel that is at fault:

        * Reseated my RAM.
        * Replaced my RAM.
        * Swapped the positions of my RAM.
        * Talked to people that have the same board (they have similar problems
          w/ this board and the later v2.2.x kernels).

The above has changed nothing w.r.t. the v2.2.9 (and later) kernels, I get
the same problems.

Has anyone seen this kind of issue w/ a 164SX?

At present I'm using the following configuration:

Hardware
--------
    * 164SX MB w/ a 21164A @ 533 MHz and 2 x 64 MB PC100 SDRAM
    * Mylex/Buslogic BT-958 UW SCSI-2 controller
    * Seagate ST34572W (4 GB UW SCSI-2)
    * Matrox Millennium II (4 MB)

Software
--------
    * Linux v2.2.9
    * Debian v2.2 (Potato)
        * egcs v1.1.2
        * binutils v2.9.1

Thanks.

Alexander
