> --- Wes Bauske <wsb@paralleldata.com> wrote:

> > Add root=/dev/fd0

>
> Now I get the prompt "Insert root disk and press Enter".
> So I put in the disk containing the ramdisk image.
> I get a kernel panic saying that it can't mount the
> root disk. ;-(

Did you really used the "root" and not the "boot" disk ?
Hmm, try root=/dev/scd0 (SCSI CD-Rom) or root=/dev/hd?
(for EIDE CD).

It should work.

If your MILO is too old (I'm not sure if such an old MILO even exists)
maybe you also to add also bootdevice=sr0 bootfile=kernels/[your system].gz".
This is normally only required for SRM because MILO should set this
automagically. I'm not sure if the very first MILOs did that. I even
can't verify this because the MILO sources are still not under
CVS [Hi Jay ;-} ;-}].
