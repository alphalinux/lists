Has anyone had any luck with this?

Very impressed with the performance, I'm pushing this into
production-test at work for some fairly large (80gb ea.) document-stores.

The only problem I've had so far is that the code isn't at all
alpha-friendly, which limits my choices in implentation.

So far, I've tracked down an unaligned trap in 'reiserfs_add_entry()'.
This alone is an alpha showstopper (not to mention other issues).
Right now, I'm in the process of adding 'printk()'s all over the
place...

Is anyone else working on this?
( even if you have a partial, semi-functional patch )
