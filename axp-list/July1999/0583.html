If you fill in the ram in 4s, you have to make sure the jumper on the
m/b is set to 128bit rather than 256bit. Other thing is I'm not sure
that you can mix 70 and 60ns rams within a single bank. May cause
problems.

You also have to be sure that the memory are fpms rather than edo. But
if you got the old ones off your alphastation, then I gather it should
be fpms.

Good luck!
