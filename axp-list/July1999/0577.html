This is an FYI.

If you edit, /etc/rc.d/init.d/halt

Change:

*halt)
        command="halt" to command="halt -i"

*reboot)
        command="reboot" to command="reboot -i"

This cleans up the shutdown and reboot for all the currently shipping
Alpha's I've tried it on (Miata-> 4100)
For the "reboot", you must set the console variable "boot_osflags" to where
your root filesystem is located, if its passed as a boot parameter.
example if you boot using boot -fl "root=/dev/sda3"
you should: set boot_osflags "root=/dev/sda3"
