> ok. Did that.

> Now I get the prompt "Insert root disk and press Enter".

> I get a kernel panic saying that it can't mount the
> root disk. ;-(
>
> I've tried two different floppy disks and both the
> original Red Hat Linux 6.0 ramdisk image as well as
> the ramdisk image in the 6.0 updates directory.
> I get the kernel panic.

Ok, I'm not sure about RH, but Debian uses the following MILO command to
boot the root floppy:

boot fd0:/linux root=/dev/fd0 load_ramdisk=1 prompt-ramdisk=1

This tells the kernel to load the rootdisk image into RAM and run it
completely in RAM, which is needed for our install disks. Hopefully, this
will work for you as well.
