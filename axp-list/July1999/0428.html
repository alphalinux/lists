>
> Hi,
>
> I tried to install RH 6.0 via CD install
> "unable to open initial console".
> Any idea ??

This looks like what I had when I used a "joliet filesystem" CD.
When you make a copy of a linux CD and do it via a harddiks
image , the filesystem is newly created.
MS uses Joliet extensions , which cannot handle the device special files
,
of which /dev/console is one , hence "cannot open initial console"
If you made a direct copy of an original RH CD, this problem should not
occur.

Make Boot / Ramdisk floppies and you should be allright.
