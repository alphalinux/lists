Champigny, Michael said:
> Has anyone gotten Multia sound to work with the stock RH kernel? I would
> like to use the ad1848 module (which is the driver for Microsoft Sound
> System) but it's not working.
>
> In /etc/conf.modules:
>
> options ad1848 io=0x530 irq=9 dma=3 dma2=1

I'm running 2.2.10, but this:

alias char-major-14 ad1848
options ad1848 io=0x530 irq=9 dma=3

in /etc/conf.modules works fine for me. (Well, it did when I was
running 2.2.1. I don't know that I've actually tried sound since I
upgraded to 2.2.10.)
