I searched the list and I didn't see this one as of yet. The machine is an
avanti with 96 meg of ram. Fresh redhat 6.0 install with the relevant
updates included. I recompiled the kernel to support my hardware. Apart
from all of the annoying unaligned trap errors that redhat seems to let
get by them, here's a funny core dump:
