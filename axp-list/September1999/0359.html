With RH6.0 I have now attempted to make the soundsystem on the Multia work, and
it almost does. Only I am not able to make the MIDI sub-system work, and I
wonder if anyone here has?

I am for now only able to use the default kernel that came with RH6.0 (2.2.5),
but as the kernel Documentation/sound/CS4232 file says that Linux's PNP
facilities can be used and isapnp comes with the distribution I tried to use
that. That was unsuccessful, and pnpdump told me that it couldn't find any
PNP cards, so that utility perhaps isn't at all intended for the Multia?

Anyway, I have the following in /etc/conf.modules:

alias sound cs4232
pre-install sound insmod sound dmabuf=1
alias midi opl3
options opl3 io=0x388
options cs4232 io=0x534 irq=9 dma=3 dma2=1 mpuio=0x330 mpuirq=10 synthirq=-1 synthio=-1

Normal sound will install, but MIDI wont. The following is /dev/sndstat:
