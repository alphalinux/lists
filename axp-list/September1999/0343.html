Kevin Mocklin wrote:
>
> If you don't care about performance, you can run any 875 based adapter in
> Tru64, but it will run with the psiop (written for Symbios 810 chip) driver
> at SCSI-2 speeds, not Ultra SCSI speeds. Only IntraServer and Compaq 875
> based boards will run with the itpsa driver and give you Ultra SCSI
> performance.
>

Thanks for the clarification Kevin.

One question though. If it's UW do you get the 16 bit transfers
or is it 8 bit no matter what since an 810 is 8 bit? If psiop does
only 8 bit, that's a factor of four degradation, but if psiop can
do wide, it's only a factor of two.

I went back and reread our previous discussion and it is
the U2W's that don't work at all without the proper licensing.
