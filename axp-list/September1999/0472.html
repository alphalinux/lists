The JDK1.1.7v2 version on the blackdown.org mirrors is not compiled
against the more recent glibc libraries. Is there a mirror or ftp site
with the more recent build?

This error was fixed for the i386 (not the Alpha!) in jdk1.1.7v3 for i386.
The same thing needs to be done for the Alpha.

The following error is still present in the JDK1.1.7v2 version of the
blackdown JDK as ported by Uncle George and built on a RH5.x libc machine:

The error output from "java -version" is:

/usr/local/jdk118/bin/../bin/alpha_21164a/green_threads/java: error in
loading shared libraries:
/usr/local/jdk118/bin/../lib/alpha_21164a/green_threads/libjava_dl.so:
undefined symbol: _dl_default_scope

I appreciate the effort put forth by the JDK/Linux/Alpha porters.
However, the JDK 1.1.x series has to come up to par with the i386
releases. This effort is critical for ApacheJserv as well as other
products. Thanks for your assistance and quick response.
