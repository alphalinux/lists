I can't answer the 'out of memory' problem you have, but I can explain why
your 'free' command changes over time. It is infact normal for your
physical memory to be used by the system like that.

The system will use unused physical memory to cache i/o. But it will be
freed up in the event it is needed for process memory utilization.
