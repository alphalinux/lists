> working fine with my NT4 installation.
> Now wanted to switch to Linux (SuSE 6.1 for AXP) and have the
> following problem.
> After i had installed the new firmware (5.4) for the alpha i managed to
> start

> installation, because my hdds (or the better the controller) are not

Did you check if the module is available for this controller (as soon as
the installation starts there's sth like Kernel modules)? If that
doesn't help, than you'll have to build a kernel yourself (on another
machine) and include the Mylex support. If none of this works for
you, then it would probably be best to turn to SuSE and ask them to
produce a proper kernel for you. Also, check out the
http://www.dandelion.com/Linux/ for usefull notes on Linux/Mylex.

BTW, wasn't this a RAID controller? I'm looking forward to hear how it
behaves under Linux.

Good luck.
