On Wed, 15 Sep 1999, Hyung Min SEO wrote:

> Is it possible to remote boot the Linux with SRM Console?
> Where can I get the documentation for it?

There is a section (3.7) in the SRM-HOWTO that describes how to boot
Linux/AXP via bootp using the SRM console. One divergence is that in my
case, I used "make bootpfile" in the kernel directory and booted the output
bootpfile, instead of doing the steps in the HOWTO which say to do "make
netboot" in the aboot directory.
