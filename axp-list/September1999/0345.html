Sorry for the off-topic discussion folks, I'll try to keep it brief...

RE: 8-bit vs 16-bit...I'm not sure on that. I know it will only negotiate
FAST-10 (SCSI-2), but not sure about the wide/narrow. The other side of
this to consider is that the psiop driver was never intended to run Ultra
SCSI adapters (i.e. 875). The support was added to allow boot strapping of
an 875 back when our itpsa driver was not integrated as part of Tru64 and it
had to be loaded via floppy. Beyond boot support, the psiop driver was not
qualified/tested by IntraServer or Compaq for 875 support. Hence the "using
unsupported adapter" message that appears when the psiop driver runs an 875.
It seems to work ok from my experience, but it's use is definitely
considered unsupported and *is* untested.
