On Sun, 12 Sep 1999, Stephen Hitchner wrote:

>Also is there a telnet program that allows, you to run X??? I'm looking
>for an Intel and/or Alpha version, I would like to be able to access my
>machine from school, but with X????

Use ssh. It does automatic, encrypted X11 forwarding.

Barring that, you can use X11's client-server architecture to run the
server on one machine, and the client on another.

On the remote machine (in Bash) type: export
DISPLAY=your.localmachine.name:0.0

On the local machine: xhost +your.remotemachine.name

P.S. Hope you have a fast internet connection.
P.P.S this is not an Alpha-Linux question.
