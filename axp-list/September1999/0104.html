Hi everyone

OK, I have had good success with Netscape Communicator 4.6 and success
with Mathematica 3.0 (it ran perfectly). Mathematica4.0 dies with a
siogctl problem (front end only).

I haven't managed to get acroread going correctly yet. It loads up and
displays a document. However, when I pull down a menu, it causes my
xserver to go CPU bound and then it eventaully crashes. These even
occurs when I bring the display up on my Tru64 machine (so acroread
seems to destroy the xserver).

Unfortunately I can't get em86 to work. I get the error message:

BUG IN DYNAMIC LINKER ld.so: ../sysdeps/alpha/dl-machine.h: 493:
elf_machine_rela: Assertion `! "unexpected dynamic reloc type"' failed!

which I just don't understand. When I install em86-2.2 I get an error
message saying

/var/tmp/rpm-tmp.73223: /usr/bin/ldconfig86: cannot execute binary file
execution of script failed

It is all a bit confusing...

Stu.
