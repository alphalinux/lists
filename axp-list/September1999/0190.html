>>> Designer Seven said:
> Well, to add more info to my problem,
>
> The machine is a Miata PWS500a
> I'm using AlphaBIOS (5.67 I think) and MILO to boot.
>
> and again, the problematic cards are:
>
> STB GLyder

According to their webpages, GlyderTX Gold is based on GLINT 500TX GOLD plus
Delta, and there's an onboard VGA for text-mode purposes.

> ELSA Gloria XL

>From the ELSA website, this card is based on GLINT MX plus Delta, but
no mention of what RAMDAC manufacturer. Note that it also has an S3
Trio64V2/DX VGA onboard for text-mode console work.

> both work fine for console... but when I start X, I
> get no GLINT/Permedia based card found.
>
> So what is the problem?>

>From the XFree86 3.3.5 README for 3DLabs support:

"This server supports the following 3DLabs chipsets:

    GLINT 500TX with IBM RGB526 RAMDAC
    GLINT MX plus Delta with IBM RGB526 and IBM RGB640 RAMDAC
    GLINT MX plus Gamma with IBM RGB526 and IBM RGB640 RAMDAC
    Permedia with IBM RGB526 RAMDAC
    Permedia 2 (classic, 2a, 2v)"

So, I'd guess the problem may be one of the following:

1. both these cards use a non-IBM RAMDAC, as XFree seems rather specific
   that only the IBM RAMDAC versions are supported.

2. the server is getting confused with the onboard VGA; what does /proc/pci
   look like for those cards?

I'd suggest asking the XFree86 folks for help.

Good luck.
