Hello all..

I know this isn't completely Alpha-specific, but its about software
available for Linux/Alpha.

I have an x86 notebook, an x86 desktop and an Alpha desktop all running
Linux.

I have several files that get shared amongst these machines (mostly xfig
drawings, and applixware stuff) and I'd like all of them to have the
latest version of my drawings and what not.

This is solved on the desktops via NFS: my Alpha's drive holds the work
and the x86 box imports the file system, so either machine can get to
the files.

My notebook needs those files too, and if I'm out and I update a drawing
or something, when my notbook goes back on my network, I'd like the
machines to sync up their directories. Any newer revisions on my notebook
should be brought over to the desktops, and vice versa.. and backups
of the old revisions should be made, just in case.

Does anyone know of any software that is capable of doing this (commercial
or otherwise?) and that is available for Linux/Alpha?
