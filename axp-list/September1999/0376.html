On 20-Sep-99 Gary Masters wrote:

> I encountered the same troubles, and ended up nuking xfs and placing
> FontPath statements in XF86Config. I don't think this xfs problem is
> limited to Alpha, though.

It's terribly slow on my Sparc. KDE runs *much* faster if I don't try to use
xfs and instead use FontPath statements. The X server seems more stable as
well. I haven't loaded RH6 on my Alpha yet (don't plan to at the moment -
what's on there now is rock stable and I have no intention of changing that
until I don't need to do work on it every day).

Mind you, this is a slow Sparc (IPX) in the first place, but the speed
and stability difference was quite noticable.
