I just tried gzilla and found it to be quck (which is good), but most web
designers use tables to make for a nice page layout, and it's table
support as you mention, is seriously lacking. Just hit slashdot or yahoo
and you'll see what I mean.

If not for the that, I would definately consider it. Maybe later releases
will be better.

I just pulled out the latest version of amaya and tried it out. It works
quite well, though I'm not sure I like 'double-clicking' to activate a
link. (amaya was designed more as an editor than a browser, but it does
browsing quite well). Maybe switching from edit mode to browse mode in
amaya should turn on single-click link following.
