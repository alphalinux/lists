There was a note recently about problems getting the Tru64 version of
Mathematica to run on Alpha Linux, despite the use of the libraries (looked
like an unsupported ioctl() call perhaps) which made me wonder how many
different applications folks have tried to do this with.

I'd be interested in hearing from anyone who's run a commercial Tru64
application on Alpha Linux successfully. Send me mail with the name of the
app and the versions and I'll summarize for the list. Conversely, if you've
tried an application unsuccessfully, I'd be interested in hearing about that
as well.

Send you mail to tom.morris@alpha-processor.com so we don't bog down the
entire list.

Thanks,
