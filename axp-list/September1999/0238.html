Hi,

My system is a Ruffian UX 600MHz
kernel 2.2.11 (de4x5 module version 0.544)

I use the de4x5 ethernet module and it works fine since the last months
on our 10Mbit network with the following options in conf.modules

options de4x5 args='eth0:autosense=TP'

But now we have switched to a 100Mbit network
So i changed my conf.modules to

options de4x5 args='eth0:fdx autosense=100Mb'

But i get no connection :-( (ifconfig -a shows eth0 up but a ping hangs)
When using the old options it connects fine but with only 10Mbit

Any Ideas ?
