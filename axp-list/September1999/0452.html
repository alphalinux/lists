hurlebau@hurlebaus.de writes:
>
> While installing Compaq CCC on my axp I get the following message:
>
> using this kit with /usr/lib/gcc-lib/alpha-redhat-linux/egcs-2.91.60 is
> completely untested, good luck
>
> Which egcs is Compaq CCC tested with??
>
I guess with 1.0.3a. If I remember well, I had a similar message when
installing ccc since I had a self-compiled egcs-1.0.3a installed in
/usr/local while the installation procedure of ccc assumes an RPM
installed egcs (not in /usr/local). I just had to set some symbolic link and
reinstall and the warning did not appear any more and ccc works (of course
I had to set rpm -i --nodeps to install).
