On Wed, Sep 08, 1999 at 04:31:37PM -0400, Eran Guendelman wrote:
>
> I didn't compile the kernel. I'm using whichever one came with the Red
> Hat distribution. How do I find out if it was compiled with CONFIG_SMP=y?
> In general, how do I find out the options the kernel was compiled with,
> because if I have to recompile it I want to make sure I only change
> this specific setting.
>
Install the kernel-smp (i forgot the exact rpm package name but it's on the
redhat cd, just do ls kernel* on RPMS directory on the CD) package. And then
make the links on /boot directory points to the smp kernel (After you install
the package, it will put the kernel and system map in /boot). And if you boot
from /vmlinux.gz then just copy the one on /boot to /.

I think the default from redhat is not SMP. But i could be wrong.

Hope that helps.
