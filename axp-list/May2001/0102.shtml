>the board has an alpha 166 processor, which came pre-mounted. i have four
>ecc simms installed. i am using an s3-based PCI video card (recommended,

>motherboard doesn't appear to have any onboard cache...meaning, the slots

The cache chips are not necessary, but certainly have an impact on performance.
The chips used to install 256K cache are likely fairly easy to find, but
the bigger ones used to install 1MB of cache are virtually impossible to
find now..

>that might send me in the right direction? any help would be greatly
>appreciated. thanks.

If not email me privately and I can send them to you as PDFs.

Also, does the power supply you are using have the 2 extra power connectors?
This is pretty well necessary..
