if my memory serves me right the old jensen type machines require:

1. jensen as specific option ) at least from 2.2.12 to 2.2.18 generic
kernels did hang at boot up)

2. Adaptec AHA 174X scsi driver (NOT as a module)

3. SRM as boot option

BTW all of you recently converted jensen users, could you please check,
whether there are more requirements in newer kernels.
