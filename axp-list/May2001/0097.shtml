Hello all!

Using RH 7.0 (kernel 2.2.17) I just get trubble when mounting CDs:
System reports "wrong major or minor number" or "invalid filesystem"
sometimes.
Interesting, if I insert one of this "illegal" disks into CDdrive, then
power recicle the machine, boot Linux - then I can mount this disk without
errors.
No such problems with "really" ISO9660 CD disks (such as RedHat installation
disks or like it).
It seems, my "invalid" disks has "Joliet" type, supported by Linux.
Any suggestions?

My system is Jensen, SCSI CD Drive DEC RRD44, RedHat 7.0.
No such problems with Tru64 4.0F on this machine.
So this is no hardware problem :-).
