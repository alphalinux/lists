Hi Mark,

TP probably means Twisted Pair, a type of network cable. What connection are
you using?
Mostly RJ45 is used, which looks like a square plug, usually made of clear
plastic. Twisted Pair
is connected using round connectors, needing T pieces in the back of the
network card, and terminators
on the cable, whereas RJ45, or UTP, or Ethernet (all the same thing) plugs
straight into the card.

Your NIC card must have a setting somewhere which enables you to set which
type is is to use.
