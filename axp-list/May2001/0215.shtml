> I'm using an old 10 base hub. The NIC card I belive can be configured to
> run
> at auto, 10, or 100. I'm a bit new to linux configurations. How would I
> force the NIC card to run at 10 base?

I don't know which NIC you've got but
on my internal NIC de4x5 (RJ45) this works fine:

kernel parameters for booting: eth0:fdx autosense:10Mb

Look in /usr/src/linux/drivers/net/de4x5.c for more
options...
