On Tue, May 15, 2001 at 07:02:08PM -0400, Will Woods wrote:
> On Tue, 15 May 2001, Graeme Hogg wrote:
> > has ayone here used a cd writer with a jensen? i get shmget errors with
> > cdrecord.
>
> This happens (to me) a lot - usually you can make it go away by increasing
> the max shared memory. I use 12MB.

As it was said many, many times - likely the simplest thing is to
set in a cdrecord configuration file a size of a shared memory segment
to 4000k instead of a default 4M and forget about it. Works even
if /proc/sys/kernel/shmmax is not present.
