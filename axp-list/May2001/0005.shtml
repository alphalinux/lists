I am successfully using a Mylex DAC960PDU in my Digital Alphaserver 3300
running Redhat 7.0. The controller has PC firmware Rev 3.51. SRM doesnt
recognize the controller, so I can't boot to it. I can however boot to a
SCSI device on the onboard Qlogic controller and once linux is up it detects
the Mylex RAID controller just fine. Also, I had to create the RAID set
with a x86 machine and the Mylex DOS tools. Hope this helps.
