I also see flickering lines and flickering parts of windows
copied or moved about 150 pixels to the right of where they
should be, but only on some configurations. My system is a
UP2000, RH7, XFree 4.0.3-9, TNT2 (Video-88PCI-16) graphics card.
I see the flickering shifted image problems when I try to do
24/32 bit color. With 8-bit or 16-bit color, it appears to
work correctly.

It might help to switch from XFree 4.0.1 to 4.0.3-9 or other
more recent X version. Given the similarity between symptoms
with different graphics cards, it might be caused by an X bug,
maybe one that only manifests itself on Alpha.
