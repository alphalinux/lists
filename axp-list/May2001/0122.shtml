On Tue, May 22, 2001 at 11:26:39AM +0200, MILLET Mathieu wrote:
>
> I'm having trouble using XFree 4.0.1. From time to time, the screen and the
> keyboard lock (I don't have any access to any terminal). When the screen
> locks, it gets a little bit fuzzy (some lines are drawn everywhere on the
> screen).

One solution may be to upgrade to the latest 4.0.3 servers; that is,
to XFree86 4.0.3 plus some additional Alpha patches, which you can find
on gatekeeper.dec.com:/pub/DEC/Linux-Alpha/X11/diffs-403-01apr2150.gz.

These primarily fix problems found on pre-EV6 CPU machines, though some
patches fix generic problems.

Red Hat, SuSE and Debian were all sent copies of these patches, and I
believe they are part of the latest binaries available from those
sources (RH 7.1 beta, SuSE 7.1, Debian ???).

Good luck.
