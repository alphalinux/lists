On Monday, 28 May, Vegt, Henry van der wrote:
> Is there anybody who is able to read the standard ODS2 (VMS) CD on a linux
> machine ?

I've come across a few references in the past:
 ftp://ftp.canberra.edu.au/pub/ise/linux-vax/vmsfs/inprogress/vmsfs.tar.gz
 http://www.mssl.ucl.ac.uk/~atp/linux-vax/
 ftp://ise.canberra.edu.au/linux-vax/vmsfs/inprogress/
which all seem to contain pieces of the abandoned Linux-VAX project.
In vmsfs.tar.gz there are utilities that can supposedly read files off
(not mount) a Files-11 ODS-2 volume. I wasn't particularly impressed when
I tried them out, but can't remember why.
