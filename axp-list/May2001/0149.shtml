kbagel@yahoo.com said:
> upon attempting to fetch them from compaq I constantly get
> error server is down or not responding.
> can anyone else get to it?

Most of Compaq appears to be behind a broken firewall. If you try to connect
with ECN enabled, you won't get through.

If the box from which you're trying to connect is a Linux 2.4 box, then
echo 0 > /proc/sys/net/ipv4/tcp_ecn and see if that helps. It makes it
work for me.

See http://www.gtf.org/garzik/ecn/ for more information.

Compaq were informed of this problem some time ago, but evidently haven't
seen fit to fix it yet.
