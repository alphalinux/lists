Hi,

This is a kernel(2.2.17) patch for EV6.

 1. EV6's PAL code issues MB instruction in TLB fill routine due to
    SMP system. This patch suppresses these overheads(thus, this
    patch can be enable only single processor).

 2. Using wh64 instruction, the speed of "clear_page" and "copy_page"
    routines are about twice faster than before.

If you have any problems, please let me know.

P.S.
  Optimized BLAS routine are also available. This time, all Level 3
routines are supported and some of Level 3 routines(GEMM, TRMM, TRSM,
SYMM) are parallelized by pthread. And you can complile and use this
library on Tru64 UNIX box.

  ftp://www.netstat.ne.jp/pub/Linux/Linux-Alpha-JP/BLAS
