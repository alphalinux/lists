Hi,

I made vectorlized sin/cos routine. It runs 28(sin)/30(cos)
clocks/each. I do not know these routines are useful...., I'll make
faster code if someone is interested in.

There are some variations,

      -- C Version --
             dsinv(double *src, double *dst, int size);
             dcosv(double *src, double *dst, int size);
             ssinv(float *src, float *dst, int size);
             scosv(float *src, float *dst, int size);

      -- FORTRAN Version --
             call dsinv(dsrc, ddst, size);
             call dcosv(dsrc, ddst, size);
             call ssinv(ssrc, sdst, size);
             call scosv(ssrc, sdst, size);

                     dsrc, ddst:double precision matrix
                     ssrc, sdst:single precision matrix

and there are NO exceptional handling, and I've only checked
dsinv/dcosv routine.
