When reading some documentation on Alpha programming and architecture one
can learn that "The Alpha architecture does not provide an integer
division instruction" (quoted from David Mosberger article on Alpha
performance tuning).

But, when reading Digital Unix Assembly Programmers Guide (1996) one can
learn about several integer division instructions (divl, divlu etc) on
page 3-15. What is going on?

On this list there were several postings on optimal SQRT routine...

But reading Alpha Architecture Hadnbook (version 4 , October 1998) one can
learn about SQRT instruction.

How can I test whether my computer (LX 21164A/533 MHz system from
Microway, based on Samsung motherboard) have such instruction?
I noticed AMASK instruction but I am not expert in alpha asembler (yet :-)
but I can use C programm.

Any suggestions?

And if my computer has sqrt instruction why my compiler (egcs 1.1) does
not use it (or how to insist on using it)?

Generally, how (where) can I obtain a complete list of CPU instructions
for my system? I plan to start assembler programming of some crucial
routines of my computational applications (for example random number
generator).
