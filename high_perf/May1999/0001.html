Hi,

I made vectorlized single sqrt/inverse sqrt routine.

sqrtf/sqrtfi runs 66/62 clocks.
ssqrtv/ssqrtiv runs only 12/11 clocks/each.

for C
            float sqrtf (float);
            float sqrtif (float);
            void ssqrtv (float *src, float *dst, int count);
            void ssqrtiv(float *src, float *dst, int count);

for FORTRAN
            FUNCTION REAL sqrtf (REAL)
            FUNCTION REAL sqrtif (REAL)
            PROCEDURE ssqrtv (src, dst, count)
            PROCEDURE ssqrtiv(src, dst, count)
                                src, dst : real array
                                count : integer
