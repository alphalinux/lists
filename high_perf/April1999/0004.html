Johannes G.D. Hausmann wrote:
>
> So if Joachim promises this, all one would need to do is take out the
> -mcpu=ev56 in the .rpmrc, recompile and that's it.

Just for the record. To modify flags used by RPM you do not have
to touch .rpmrc - which may not even exist. It is enough to set
RPM_OPT_FLAGS to desired values and this can be done in a spec
file; the same way as RPM_BUILD_ROOT, for example. We are talking
about creating rmp archive, of course, or otherwise this does not
make any sense.

> Probably one should
> rename the spec file and the binary, like fastmath-ev4-something.

And that, i.e. this modified spec file, is an excellent place for
something like '-mcpu=ev4' as one of options in RPM_OPT_FLAGS.
Or you may set CFLAGS for 'make' in spec file directly. Keep in mind
that somebody else may use this spec file in the future.

  Michal
